# Earthquakes and Volcanoes — Seismic Waves, Mw Scale, Eruption Types

## The Big Picture

Earthquakes and volcanoes are surface expressions of tectonic processes. Earthquakes release stored elastic strain; volcanoes release heat/volatile energy from depth. Both are most concentrated at plate boundaries, but both occur intraplate as well.

```
+---------------------------------------------------------------+
|        EARTHQUAKE + VOLCANO GLOBAL DISTRIBUTION               |
|                                                                |
|  SUBDUCTION ZONES (Ring of Fire):                             |
|    - Greatest earthquake hazard (megathrust M9+)              |
|    - Explosive arc volcanoes (andesite/dacite/rhyolite)       |
|    - Deep Wadati-Benioff seismicity to 700 km                 |
|                                                                |
|  DIVERGENT MARGINS (Mid-ocean ridges):                        |
|    - Shallow (< 30 km) earthquakes, M ≤ 7                     |
|    - Effusive basaltic eruptions                              |
|                                                                |
|  TRANSFORM FAULTS:                                            |
|    - Shallow seismicity, can be large (M7–8)                  |
|    - No associated volcanism                                  |
|                                                                |
|  INTRAPLATE:                                                   |
|    - Less frequent, but large events possible                 |
|    - New Madrid Seismic Zone, Yellowstone caldera             |
+---------------------------------------------------------------+
```

## Earthquake Mechanics — Elastic Rebound and Fault Types

**Elastic rebound theory** (Reid, 1910): rocks accumulate elastic strain as plates move. When stress exceeds rock strength (or friction), rapid slip occurs → stored strain released as seismic waves.

```

                        NORMAL FAULT        REVERSE FAULT       STRIKE-SLIP
                        (extension)         (compression)       (lateral)

    Hanging wall        ↙ HW down           ↗ HW up             HW and FW
    (HW) over                                                    move sideways
    footwall (FW)       |  /|               |\ |                | |
                        | / |               | \|                | |
                        |/  |               |  \                | |

    SETTING:            Rifts, normal       Subduction,         Transform
                        faults in           collision zones     faults
                        Basin & Range       (thrust faults)     (San Andreas)

    FOCAL               Tension (T)         Compression (P)     Strike-slip
```

**Blind thrust faults** — no surface expression; responsible for some damaging surprises (Northridge 1994, M6.7).

```
             transverse)                       through outer     outer core
                                               core)

Love wave    Horizontal shear    ~3–4.5 km/s  Surface only      Damage, fault
(surface)    (side-to-side       (slower than  (ground motion)   mapping
```

**Shadow zones** — S-waves cannot travel through the liquid outer core → S-wave shadow zone starts at ~105° from epicenter. P-waves refract at CMB → P-wave shadow zone from ~105°–140°. These shadow zones confirmed the liquid outer core (1906, Oldham; 1914, Gutenberg).

**Travel time curves** — Seismographs worldwide record P and S arrivals. Difference in arrival times = distance to epicenter. Three stations minimum to locate earthquake (trilateration).

**Energy equivalents:**
- Mw 5.0 ≈ 32 TJ ≈ Hiroshima bomb
- Mw 6.0 ≈ 1 PJ ≈ small nuclear test
- Mw 9.0 ≈ annual global human energy consumption
- Mw 9.5 (1960 Valdivia, Chile) ≈ 2.5 × 10¹⁸ J

**Intensity vs Magnitude** — Magnitude is intrinsic to the earthquake source. Intensity (Modified Mercalli, MMI I–XII) describes shaking at a specific location. Same earthquake → different MMI at different distances/ground conditions.

## Volcanic Explosivity Index (VEI)

Logarithmic scale 0–8 based on ejecta volume:

```
VEI   VOLUME          PLUME HEIGHT   EXAMPLE               RECURRENCE
---   --------------  -------------  --------------------   -----------
0     < 10⁴ m³        < 100 m        Mauna Loa (effusive)  Continuous
1     10⁴–10⁶ m³      100–1000 m     Stromboli             Daily
2     10⁶–10⁷ m³      1–5 km         Galeras 1993          Weekly
3     10⁷–10⁸ m³      3–15 km        Ruapehu 1945          Yearly
4     10⁸–10⁹ m³      10–25 km       Eyjafjallajökull 2010 Decades
5     10⁹–10¹⁰ m³     20–35 km       Mt. St. Helens 1980   Centuries
6     10¹⁰–10¹¹ m³    30–40 km       Pinatubo 1991         ~100 yrs
7     10¹¹–10¹² m³    > 40 km        Tambora 1815          ~1000 yrs
8     > 10¹² m³        > 40 km        Yellowstone ~640 ka   ~100,000 yrs
```

**Tambora 1815 (VEI 7):** 70,000+ direct deaths; 1816 "Year Without a Summer" (frost in July in New England); global famine.
**Pinatubo 1991 (VEI 6):** Aerosols cooled Earth ~0.5°C for ~1 year.

## Decision Cheat Sheet

| Situation | Interpretation |
|-----------|---------------|
| Swarm of small earthquakes at volcano | Magma movement; precursor to eruption |
| Deep (>300 km) earthquakes | Subduction zone; slab seismicity |
| Shallow (<35 km), strike-slip focal mechanism | Transform fault |
| No S-wave received at seismograph | P-wave in shadow zone; liquid in path |
| VEI 4+ with SO₂ spike | Significant stratospheric aerosol loading |
| Long-period (LP) seismicity at volcano | Fluid (magma or hydrothermal) movement |
| Caldera depression at large volcano | Previous magma chamber collapse |

## Common Confusion Points

**Richter scale is obsolete** — Media still reports "Richter scale" but scientists use Mw (moment magnitude). For large earthquakes, Richter underestimates; only Mw correctly captures the total energy of M7+ events.

**Magnitude is logarithmic in amplitude but the energy scales as 10^(1.5M)** — Each full unit of Mw = 10× larger amplitude but ~32× more energy (10^1.5 = 31.6). M9 releases ~1000× more energy than M7.

**Not all volcanoes are at plate boundaries** — Yellowstone, Hawaii are intraplate (hotspot). Intraplate volcanoes can produce large events: Yellowstone's last supereruption was VEI 8.

**Lahar vs lava flow** — Lahars are cold mudflows (rainwater + ash). They travel far, fast, and can happen long after eruption ends (rain remobilizes ash deposits). Lava is molten rock. Lahars historically kill more people (Nevado del Ruiz 1985: 23,000 dead from lahar, not lava).

**Tsunami from earthquake** — Not all earthquakes cause tsunamis. Tsunamis require: large Mw (≥7.5 typically), shallow focal depth (<50 km), significant seafloor displacement (vertical), ocean above. Strike-slip faults produce little vertical displacement → minimal tsunami. Reverse/thrust faults (Sumatra 2004, M9.1) → catastrophic tsunamis.
