## The Big Picture

Minerals are the atoms-to-molecules level of geology. ~5,700 known minerals; ~30 are "rock-forming" (>95% of Earth's crust by volume). The critical insight: Earth's crust is dominated by silicate minerals because silicon and oxygen are the two most abundant elements in it.

```
+---------------------------------------------------------------+
|               MINERAL HIERARCHY                                |
|                                                                |
|  ELEMENT:   Si, O, Al, Fe, Ca, Na, K, Mg  (Big 8 of crust)  |
|                          |                                     |
|                          v                                     |
|  ION/CRYSTAL:  [SiO₄]⁴⁻ tetrahedra link into structures      |
|                          |                                     |
|                          v                                     |
|  MINERAL:   Quartz, Feldspar, Pyroxene, Olivine, Mica...     |
|             (defined composition, crystalline structure)       |
|                          |                                     |
|                          v                                     |
|  ROCK:      Granite (quartz+feldspar+mica)                    |
|             Basalt (pyroxene+plagioclase)                      |
+---------------------------------------------------------------+
```

**Big 8 elements of Earth's crust by weight %:**
O (46%), Si (28%), Al (8%), Fe (5%), Ca (4%), Na (2.4%), K (2.3%), Mg (2.3%)

## Crystal Systems — The Seven Symmetry Families

All crystals fall into one of 7 systems based on unit cell geometry (3 axes, 3 angles):

```
SYSTEM       AXES              ANGLES           EXAMPLE MINERAL
-----------  ----------------  ---------------  ------------------
Cubic        a = b = c         α=β=γ=90°        Halite (NaCl), Garnet, Pyrite
Tetragonal   a = b ≠ c         α=β=γ=90°        Zircon, Rutile (TiO₂)
Orthorhombic a ≠ b ≠ c         α=β=γ=90°        Olivine, Sulfur
Hexagonal    a = b ≠ c         α=β=90°, γ=120°  Quartz, Calcite, Ice
Trigonal     a = b = c         α=β=γ ≠ 90°      Calcite (subsystem of hex)
Monoclinic   a ≠ b ≠ c         α=γ=90°, β≠90°   Pyroxene, Biotite mica
Triclinic    a ≠ b ≠ c         α≠β≠γ≠90°        Plagioclase feldspar
```

The crystal system determines cleavage directions, optical properties, and habit (external form). Cubic minerals (halite, galena) cleave into perfect cubes. Hexagonal minerals (calcite) produce rhombohedral cleavage fragments.

## Silicate Structure — The Fundamental Framework

The silicate tetrahedron [SiO₄]⁴⁻ is the basic structural unit. How tetrahedra link together determines mineral class:

```
STRUCTURE          TETRAHEDRA     Si:O RATIO   EXAMPLE
-----------------  -------------  -----------  ---------------------------
NESOSILICATES      Isolated       1:4          Olivine (Mg,Fe)₂SiO₄
(island)           (no sharing)                Garnet, Zircon

SOROSILICATES      Pairs          2:7          Epidote
(double island)    (1 corner)

CYCLOSILICATES     Rings          1:3          Beryl (Be₃Al₂Si₆O₁₈)
                   (2 corners,                 Tourmaline
                    ring)

INOSILICATES       Chains         1:3          Pyroxenes (single chain)
                   (2 corners,    4:11         Amphiboles (double chain)
                    chain)

PHYLLOSILICATES    Sheets         2:5          Micas (biotite, muscovite)
                   (3 corners)                 Clay minerals, Serpentine

TECTOSILICATES     3D network     1:2          Quartz (SiO₂)
                   (all 4         (+ Al subs.) Feldspars, Zeolites
                    corners)
```

**Key pattern:** More sharing = lower Si:O ratio = more Si relative to framework oxygens = higher polymerization = generally higher melting temperature, more resistance to weathering.

## The Rock-Forming Minerals

### Feldspar Group — Most Abundant Mineral in Crust (~60%)

```
FELDSPAR
   |
   +-- ALKALI FELDSPARS (K, Na end members)
   |      Orthoclase (KAlSi₃O₈)  — pink, opaque, common in granite
   |      Sanidine — high-T volcanic equivalent
   |      Albite (NaAlSi₃O₈) — technically also plagioclase end member
   |
   +-- PLAGIOCLASE (Na-Ca solid solution series)
          Albite (Na end) ←———————→ Anorthite (Ca end)
          more felsic            more mafic
          (granite/rhyolite)     (basalt/gabbro)
```

### Quartz — SiO₂
- Pure tectosilicate, extremely resistant to weathering
- Hexagonal; hardness 7 (Mohs); conchoidal fracture (no cleavage)
- Common in granite, sandstone; missing from basalt (too much Mg/Fe ties up Si)

### Mica Group — Sheet silicates
- Muscovite (K, Al) — white/silver, felsic rocks
- Biotite (K, Mg, Fe) — dark, common in granite

### Pyroxene and Amphibole — Chain silicates
- Pyroxenes (augite, diopside) — single-chain, common in basalt/gabbro
- Amphiboles (hornblende) — double-chain; distinguished by 60°/120° vs 90° cleavage

### Olivine — (Mg,Fe)₂SiO₄
- Nesosilicate; forsterite (Mg) to fayalite (Fe) solid solution
- First mineral to crystallize from basaltic magma (Bowen's series)
- Abundant in upper mantle; weathers rapidly on surface

### Calcite / Dolomite — Carbonates (non-silicate)
- Calcite (CaCO₃): hardness 3, reacts with HCl (fizzes)
- Dolomite (CaMg(CO₃)₂): weaker HCl reaction
- Limestone and marble are dominated by these

## Mineral Identification Properties

```
PROPERTY       WHAT IT MEASURES              DIAGNOSTIC VALUE
-------------  ----------------------------  -----------------------------------
Hardness       Resistance to scratching      Mohs 1–10 (definitive for field ID)
               (atomic bonding strength)

Cleavage       Preferred fracture directions Crystal structure; count planes + angles
               (weak bond planes)

Fracture       Non-planar breakage patterns  Conchoidal (quartz), fibrous, uneven

Luster         How light reflects off        Metallic vs nonmetallic (vitreous,
               surface                       resinous, pearly, silky, earthy)

Color          Variable (impurities);        Unreliable alone (quartz = any color)
               use streak instead

Streak         Color of powdered mineral     Consistent; hematite streak = red-brown
               on unglazed porcelain         regardless of specimen color

Specific       Density relative to water     Sulfides/oxides = 4–8 g/cm³
Gravity        (water = 1.0)                 Silicates = 2.5–3.5 g/cm³

Crystal Form   External expression of        Often visible in good specimens
               internal crystal system

Special:       Magnetism (magnetite),        Useful shortcuts
               reaction with HCl (carbonates),
               double refraction (calcite),
               taste (halite), smell (sulfur)
```

### Mohs Hardness Scale

```
1  Talc         (Mg₃Si₄O₁₀(OH)₂)  — scratched by fingernail
2  Gypsum       (CaSO₄·2H₂O)      — fingernail barely scratches
3  Calcite      (CaCO₃)           — copper coin scratches it
4  Fluorite     (CaF₂)
5  Apatite      (Ca₅(PO₄)₃)       — knife blade scratches it
6  Feldspar     (KAlSi₃O₈)        — scratches glass barely
7  Quartz       (SiO₂)            — scratches glass easily
8  Topaz        (Al₂SiO₄(F,OH)₂)
9  Corundum     (Al₂O₃)           — Ruby, Sapphire
10 Diamond      (C)               — hardest natural substance
```

Reference points: fingernail ~2.5, copper coin ~3, glass plate ~5.5, steel file ~6.5

## Economic Minerals — Ores vs Gangue

An **ore mineral** contains the target metal in extractable concentration. **Gangue** is the worthless host.

| Metal | Primary Ore Mineral | Formula | Crystal System |
|-------|-------------------|---------|---------------|
| Iron | Hematite, Magnetite | Fe₂O₃, Fe₃O₄ | Hexagonal, Cubic |
| Copper | Chalcopyrite, Malachite | CuFeS₂, Cu₂CO₃(OH)₂ | Tetragonal, Monoclinic |
| Gold | Native gold, Calaverite | Au, AuTe₂ | Cubic, Monoclinic |
| Aluminum | Gibbsite (bauxite) | Al(OH)₃ | Monoclinic |
| Lead | Galena | PbS | Cubic |
| Zinc | Sphalerite | ZnS | Cubic |
| Lithium | Spodumene, Lepidolite | LiAlSi₂O₆, K(Li,Al)ₓSi₄O₁₀ | Monoclinic |
| REE | Bastnäsite, Monazite | (Ce,La)(CO₃)F, (Ce,La,Nd)PO₄ | Hexagonal, Monoclinic |

## Decision Cheat Sheet

| Field Situation | Best Identification Approach |
|----------------|------------------------------|
| Unknown dark mafic mineral | Check cleavage angle: pyroxene ~90°, amphibole ~60°/120° |
| Unknown white/clear mineral | Hardness: talc (1), calcite (3), feldspar (6), quartz (7) |
| Suspect carbonate | Drop HCl: calcite fizzes strongly, dolomite weakly |
| Metallic luster, heavy | Check streak: hematite = red-brown, pyrite = greenish-black |
| Black mineral in granite | Biotite (sheet cleavage) vs hornblende (two cleavage planes) |
| Shiny cubic crystals, heavy, lead-gray | Galena (PbS) |
| Glassy fracture, hardness 7 | Quartz |

## Common Confusion Points

**Color is unreliable for minerals** — Quartz can be colorless (rock crystal), purple (amethyst), pink (rose), black (smoky), yellow (citrine), or white (milky). Impurity elements cause color. Always check streak for ID.

**Cleavage vs fracture** — Cleavage produces flat, shiny planes (reflecting light uniformly). Fracture produces irregular, often conchoidal (curved, glass-like) surfaces. Quartz has no cleavage; it fractures. Calcite has three perfect cleavages.

**Crystal form vs cleavage** — Crystal form is the external habit during growth (how crystals look). Cleavage is how they break after formation. A well-formed calcite crystal (scalenohedron) breaks on different planes than it grew.

**Mineral vs rock "by the same name"** — The mineral "olivine" is a specific formula (Mg,Fe)₂SiO₄. The rock "dunite" is made almost entirely of olivine. Limestone is a rock; calcite is the mineral it's made of.

**Feldspar is the most abundant mineral but rarely what people picture** — Everyone pictures quartz (glassy, clear). But feldspar (pinkish-white, often blocky) is twice as abundant in the crust. Granite's white/pink coloration is mostly feldspar, not quartz.
