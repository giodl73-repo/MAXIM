# Aromatics & Herbs — Lamiaceae and Mediterranean Culinary Botany

## The Big Picture

```
+------------------------------------------------------------------+
|              THE HERB LANDSCAPE                                  |
|                                                                  |
|  LAMIACEAE (Mint Family) — DOMINANT CULINARY HERB FAMILY         |
|  Rosemary, Thyme, Oregano, Basil, Sage, Mint, Lavender           |
|  Square stems | Opposite leaves | Bilateral flowers              |
|  Mediterranean origin (most culinary species)                    |
|  Glandular trichomes = essential oil production                  |
|                                                                  |
|  OTHER FAMILIES:                                                 |
|  Bay laurel (Lauraceae) | Parsley/cilantro (Apiaceae)            |
|  Tarragon (Asteraceae) | Alliums — garlic/onion (Amaryllidaceae) |
|                                                                  |
|  FUNCTIONAL CHEMISTRY:                                           |
|  Phenolics (thymol, carvacrol): antimicrobial membrane disruptors|
|  Terpenes (camphor, 1,8-cineole, pinene): aroma + mild bioactivity|
|  Organosulfur (allicin, diallyl disulfide): allium heat          |
|  Ketones (thujone, camphor): neurotoxic at high doses            |
+------------------------------------------------------------------+
```

## The Herb vs Spice Distinction

```
CONVENTIONAL DISTINCTION:
  HERB = green, fresh plant parts (leaves, stems)
  SPICE = dried parts (seed, bark, root, flower bud, fruit)

  This is CULINARY CONVENTION, not botanical taxonomy.

THE FUZZY CASES:
  Same plant, both herb and spice:
  Coriandrum sativum → coriander seed (spice) + cilantro leaf (herb)
  Foeniculum vulgare → fennel seed (spice) + fennel frond (herb)
  Anethum graveolens → dill seed (spice) + dill weed (herb)
  Trigonella foenum-graecum → fenugreek seed (spice) + methi leaf (herb)

  Fresh vs dried versions of herb:
  Fresh rosemary ≠ dried rosemary (different compound profile)
  Fresh basil ≠ dried basil (drastically different)
  Fresh vs dried: heat drives off most volatiles; dried can develop
  different character via Maillard reactions, oxidation

PRESERVATION NOTE:
  Most culinary herbs = cold-tolerant temperate zone plants
  Most culinary spices = tropical plants that must be dried to ship
  The distinction partially reflects shipping logistics of pre-refrigeration era
```

## Lamiaceae — Family Profile

```
DIAGNOSTIC FEATURES:
  Square cross-section stems (distinctive)
  Opposite leaf arrangement (pairs)
  Bilateral (not radial) flowers
  2 stamens or 4 (in pairs)
  Aromatic: volatile oil in glandular trichomes on leaf surface

GLANDULAR TRICHOMES:
  Microscopic hair-like structures on leaf epidermis
  Two types: peltate (secretory, oil-containing) + capitate
  Oil stored in subcuticular space under a cuticle dome
  Physical damage (crushing, bruising) ruptures dome → release aroma

  WHY THIS MATTERS:
  → Always bruise, tear, or chiffonade leaves just before use
  → Dried herbs: crushing reactivates vs preground that has lost volatiles
  → Fine chopping > large pieces for aroma extraction in cold applications
  → Whole sprigs in long-cooked dishes: add at end for fresh aroma
```

## Rosemary (*Salvia rosmarinus*)

```
TAXONOMY NOTE:
  Reclassified from Rosmarinus officinalis to Salvia rosmarinus (2017)
  Based on molecular phylogenetics — it IS a sage, not a separate genus
  Both names still used commercially

ORIGIN: Circum-Mediterranean, especially coastal cliffs
GROWTH: Perennial woody shrub, to 2m; narrow, needle-like leaves

KEY COMPOUNDS:
  Carnosic acid (~1.5–2.5% dry weight): diterpene phenol
    → Potent antioxidant — approved food preservative (E-392)
    → Converts to carnosol and other compounds on storage/heat
    → Historical use: rub rosemary on meats before refrigeration
    → Modern: rosemary extract in commercial food preservation
  Camphor: ~15–25% of EO → pungent, medicinal note
  1,8-Cineole (eucalyptol): ~30–50% of EO → eucalyptus freshness
  α-Pinene: ~10–25% → clean pine note

COGNITIVE CLAIMS:
  1,8-Cineole as acetylcholinesterase inhibitor
  → Inhibiting AChE → more acetylcholine available → cognitive effects
  Studies: aromatherapy with rosemary essential oil → modest speed/accuracy
            improvements in cognitive tasks (Mark Moss, Northumbria Univ.)
  Mechanism: systemic absorption of 1,8-cineole via inhalation → measurable
  Status: preliminary; not clinically actionable but not fabricated
```

## Thyme (*Thymus vulgaris*)

```
ORIGIN: W. Mediterranean (Spain, Provence), naturalized widely
KEY COMPOUNDS:
  Thymol: ~30–50% of EO (varies by chemotype — see below)
    → Monoterpene phenol; powerful antimicrobial
    → Disrupts bacterial cell membranes (gram+ and gram−)
    → Active ingredient in original Listerine formulation (1879)
    → Still in some antiseptic throat preparations
  Carvacrol: ~2–10% in common thyme
    → Similar structure + activity to thymol (both monoterpene phenols)
    → Dominant in oregano
  p-Cymene: precursor to both thymol and carvacrol; ~10–25%
  Linalool: ~5% in some chemotypes → adds floral softness

THYME CHEMOTYPES:
  Same species (T. vulgaris) produces different chemical profiles
  depending on genetics + growing conditions:
  thymol chemotype: high thymol (dominant in French thyme)
  carvacrol chemotype: high carvacrol (related to oregano character)
  linalool chemotype: floral, gentle (preferred for cosmetics)
  geraniol chemotype: rose-like
  Chemotype is agriculturally important for essential oil industry

LEMON THYME (T. × citriodorus):
  Hybrid: T. vulgaris × T. pulegioides
  Key shift: geraniol + linalool dominant → distinctly lemony
  No thymol-dominated antimicrobial potency
  Different culinary use: seafood, poultry, gentler applications
```

## Oregano (*Origanum vulgare*)

```
ORIGIN: Mediterranean (particularly Greece, Turkey, Middle East)
KEY COMPOUNDS:
  Carvacrol: ~40–80% of EO (high-carvacrol chemotype → pungent/hot)
  Thymol: ~5–15% (some overlap with thyme — these are related)
  p-Cymene: ~5–10%

REGIONAL VARIATION:
  Greek oregano (O. vulgare subsp. hirtum): highest carvacrol → most pungent
  Italian oregano (O. vulgare subsp. vulgare): lower carvacrol, gentler
  Marjoram (O. majorana): entirely different — milder, floral (linalool-dominant)
    Marjoram ≠ oregano despite same genus

MEXICAN OREGANO (Lippia graveolens, Verbenaceae):
  DIFFERENT FAMILY — not Lamiaceae
  Convergent flavor profile (carvacrol-based aroma)
  Stronger, more citric than Mediterranean oregano
  Traditional in Mexican cuisine (chili con carne, mole)
  Not interchangeable with Mediterranean oregano in fine cooking

THE PIZZA HERB:
  Oregano + tomato sauce = Italian-American food tradition
  Adopted in US after WWII via returning soldiers from Italy
  Oregano went from ~0 to #1 herb import in US: 1948–1956
  "Pizza was the spice that introduced Americans to oregano"
```

## Basil (*Ocimum basilicum*)

```
ORIGIN: Tropical South Asia (India/SE Asia), naturalized Mediterranean
KEY COMPOUND VARIATION BY CULTIVAR:

Sweet basil (Italian):
  Linalool: ~35–45% → floral, slightly floral-spicy
  Eugenol: ~10–20% → clove warmth
  Methyl chavicol (estragole): minor in sweet basil

Thai basil (O. basilicum var. thyrsiflora):
  Methyl chavicol (estragole): ~75–85% → distinctly anise-like
  Eugenol: minor
  → completely different sensory profile from sweet basil

Holy basil / Tulsi (O. tenuiflorum):
  Eugenol: ~70–80% → heavy clove character
  Hindu ritual plant; entirely different culinary use
  Stir-fried in Thai cooking (pad krapow) → intense, spicy-clove

Lemon basil (O. × citriodorum):
  Citral dominant → lemony
  Less common; specialty use

STABILITY PROBLEM:
  Basil volatile oils = exceptionally unstable
  Dried basil loses most volatile character within weeks
  Difference between fresh and dried basil: profound
  Dried basil = useful in long-cooked applications (released into sauce)
  Fresh basil = finishing herb only (add last, don't cook)
```

## Bay Laurel (*Laurus nobilis*)

```
ORIGIN: Mediterranean coastal regions
KEY COMPOUNDS:
  1,8-Cineole: ~40–50% → fresh, eucalyptus-clean
  α-Pinene: ~10–15%
  Sabinene: ~10–15%
  Eugenol: minor

CULINARY ROLE:
  Used whole (dried): French bouquet garni, mirepoix, court-bouillon
  Long cooking releases compounds slowly → background complexity
  Remove before serving (mouthfeel issue + concentrated flavor)

HISTORICAL SIGNIFICANCE:
  Greek/Roman laurel wreaths (corona laureata): THIS PLANT
  "Baccalaureate," "poet laureate" — etymological origin in Laurus nobilis
  Apollo's sacred plant; symbol of victory and knowledge

CALIFORNIA BAY (*Umbellularia californica*):
  DIFFERENT FAMILY (also Lauraceae but different genus)
  Contains umbellulone (unique compound) → much stronger, pungent
  Can cause headaches with concentrated exposure
  NOT the same as culinary bay; use 1/4 of recipe quantity if substituted
```

## Sage (*Salvia officinalis*)

```
ORIGIN: N. Mediterranean coast (Dalmatia/Adriatic historically prized)
KEY COMPOUNDS:
  Thujone (α + β): ~15–35% of EO — CAUTION COMPOUND
    → Monoterpene ketone; GABAA receptor antagonist → neurological effects
    → At high doses: convulsions, hallucinations
    → Same compound responsible for absinthe neurotoxicity concerns
  Camphor: ~10–30%
  1,8-Cineole: ~10–20%
  Borneol: ~10–20%

THUJONE CONCERN IN CONTEXT:
  Culinary sage use: no meaningful health risk
  The thujone content of normal cooking quantities is tiny
  Sage essential oil supplements: risky at high doses
  Absinthe: reformulated in 1990s; EU limits thujone to 10mg/L
             Modern absinthe well within limits

TRADITIONAL PAIRINGS:
  Pork, sausage, stuffing (European tradition)
  Brown butter + sage: classic Italian (pasta sage butter)
  Explains: sage's slight preservative function historically
  → used in fatty meats that spoil easily

SALVIA GENUS CONFUSION:
  S. officinalis = culinary sage
  S. hispanica = chia (seeds) — completely different plant
  Pineapple sage, clary sage, white sage (ceremonial): different species
  Genus Salvia: ~1,000 species with wildly different chemistry
```

## Fenugreek (*Trigonella foenum-graecum*)

Botanically Fabaceae (legume family) — not Lamiaceae:

```
FENUGREEK:
  Origin: Mediterranean/SW Asia; major spice + legume + herb
  Three culinary roles:
  1. Seed (spice): bitter, maple-syrup aromatic
  2. Fresh leaves (methi): slightly bitter, used in Indian cooking
  3. Sprouted seeds: milder, used in salads/garnish

KEY COMPOUND: SOTOLON
  The maple syrup aroma compound
  4,5-dimethyl-3-hydroxy-2(3H)-furanone
  Also responsible for "maple syrup urine disease" smell
  (a metabolic disorder where branched-chain amino acid processing fails)

  FENUGREEK BODY ODOR EFFECT:
  Sotolon absorbed from GI tract → excreted in sweat, urine
  Some people produce maple-syrup-scented sweat after eating fenugreek
  Documented, mild, harmless — can confuse metabolic disorder screening

DIOSGENIN:
  Steroidal saponin (~1.5% of seed)
  Historical: major starting material for partial synthesis of progesterone
  (Marker degradation → early hormonal contraception development)
  Precursor to synthetic steroid hormone industry

INDUSTRIAL IMPORTANCE:
  Fenugreek = flavor in commercial "maple syrup flavor" used in pancake syrups
  Much cheaper than real maple syrup
  Pure sotolon or fenugreek extract in imitation maple products
```

## Alliums — A Different Mechanism Entirely

```
AMARYLLIDACEAE (formerly Alliaceae):
  Garlic (Allium sativum), Onion (A. cepa), Shallot, Leek, Chive
  Not Lamiaceae — mechanistically distinct from other herbs

THE ALLIINASE MECHANISM:
  Cells contain:
    COMPARTMENT A: Alliin (flavorless, S-propenyl cysteine sulfoxide)
    COMPARTMENT B: Alliinase enzyme (in vacuoles)

  When tissue is DAMAGED (cutting, crushing, chewing):
  Compartments merge → Alliinase cleaves Alliin → ALLICIN

  Allicin (diallyl thiosulfinate):
  → Highly reactive organosulfur compound
  → Broad antimicrobial (disrupts bacterial thiol enzymes)
  → Decomposes to: diallyl disulfide, diallyl trisulfide, ajoene
  → These are the "garlic smell" compounds

  COOKING CHANGES:
  Heat: inactivates alliinase → less allicin formed
  → Cooked garlic is milder than raw (enzyme denatured)
  → Crush before cooking to allow allicin formation,
    then cook = intermediate character
  → Raw garlic pressed into dish at end = maximum pungency

WHY GARLIC SMELL LINGERS:
  Diallyl disulfide absorbed through gut → bloodstream → excreted via lungs
  Garlic breath = volatile organosulfur in exhaled air
  Not from mouth bacteria; cannot be fully masked by brushing
  Parsley anecdote: weak chlorophyll interaction; mostly folk remedy
```

## Volatile Oil Preservation

```
DEGRADATION MECHANISMS:
  Oxidation: terpenes react with O₂ → peroxides → off-flavors
             limonene → carvone (good) or oxidized off-notes (bad)
  Volatilization: compounds simply evaporate at room temperature
  Photodegradation: UV light accelerates oxidation
  Thermal: heat drives off even stable compounds

STORAGE HIERARCHY (best to worst):
  1. Live/fresh plant (oils still in intact cells)
  2. Freshly harvested, refrigerated
  3. Dried whole herbs in sealed, dark, cool containers
  4. Dried whole herbs at room temperature
  5. Ground herbs (maximum surface area, maximum loss)
  6. Pre-ground, stored in clear containers on bright shelf (worst)

PHENOLICS AS NATURAL PRESERVATIVES:
  Thymol + carvacrol (thyme, oregano): disrupt bacterial membranes
  Eugenol (cloves, basil): antimicrobial
  Carnosic acid (rosemary): antioxidant

  This is WHY Mediterranean cooking is herb-heavy:
  Pre-refrigeration food preservation → herb-rubbed/marinated meats
  The antimicrobial chemistry was discovered empirically over millennia
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| What defines the Lamiaceae family? | Square stems, opposite leaves, bilateral flowers, glandular trichomes with essential oils |
| What's in Listerine that's from thyme? | Thymol — a monoterpene phenol with broad antimicrobial activity. Active in original formula (1879). |
| What's the difference between oregano and marjoram? | Same genus (Origanum) but different species and chemistry. Marjoram (O. majorana) = milder, floral (linalool). Oregano = pungent (carvacrol). |
| Why is dried basil so much weaker than fresh? | Basil volatiles (linalool, eugenol, methyl chavicol) are unstable — dried basil loses most within weeks |
| What's the "maple syrup" compound in fenugreek? | Sotolon — absorbed from gut, excreted via sweat/urine. Same compound that gives maple syrup urine disease its name. |
| What's the alliinase mechanism? | Physical damage merges compartments: alliin + alliinase → allicin → diallyl disulfide/trisulfide (the actual garlic aroma) |
| Why doesn't cooking garlic produce as much allicin as raw? | Heat denatures alliinase → enzyme can't convert alliin → milder flavor |
| Why is sage a concern in supplement form? | Thujone content (15–35% of EO) — GABAA antagonist; safe in culinary use, risky in concentrated supplements or essential oil |

## Common Confusion Points

**Mexican oregano ≠ Mediterranean oregano:** Different plant family (Verbenaceae), different flavor character. Not interchangeable in precise cooking. Mexican oregano stronger, more citric — designed for bold Mexican flavors (mole, chile).

**Bay laurel vs California bay:** California bay (*Umbellularia californica*) contains umbellulone, which is absent in culinary bay. It's much more pungent. Using California bay where recipes call for bay laurel will overpower the dish.

**Thai basil ≠ Italian basil:** Methyl chavicol dominant in Thai basil → anise/tarragon note. Eugenol + linalool dominant in Italian basil → floral-clove. These are genuinely different flavor profiles, not just intensities.

**"Herb" = mild, "Spice" = strong — false:** This stereotype doesn't hold. Fresh horseradish and fresh chili are more intense than many dried spices. Dried rosemary and sage are more intense than fresh dill. Character and intensity are compound-specific, not herb-vs-spice.
