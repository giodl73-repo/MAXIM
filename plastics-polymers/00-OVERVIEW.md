# Plastics & Polymers — Landscape and Taxonomy

## The Big Picture

```
+------------------------------------------------------------------+
|                    THE POLYMER UNIVERSE                          |
|                                                                  |
|   NATURAL POLYMERS              SYNTHETIC POLYMERS               |
|   ---------------               -----------------                |
|   Cellulose, Starch             +-------------+                  |
|   Silk, Wool, Lignin            |             |                  |
|   Natural Rubber (NR)       THERMO-       THERMO-    ELASTO-    |
|   DNA, Proteins              PLASTICS       SETS       MERS      |
|                                  |             |          |       |
|                              Melt &       Cross-    Stretch &    |
|                              reprocess    linked    recover      |
|                                  |             |          |       |
|                          PE,PP,PET      Epoxy,PU   NR,SBR,      |
|                          PS,PVC,ABS     Phenolic   EPDM,         |
|                          Nylon,PC       Melamine   Silicone      |
+------------------------------------------------------------------+
```

Polymer = Greek *polymeros*: "many parts." A macromolecule built by linking
small repeat units (monomers) via covalent bonds. Everything from grocery bags
to jet engine housings to DNA is some version of this architecture.

## The Three-Way Split

```
+---------------------+---------------------+---------------------+
|   THERMOPLASTICS    |     THERMOSETS      |     ELASTOMERS      |
+---------------------+---------------------+---------------------+
| Linear / branched   | Cross-linked 3D     | Lightly cross-      |
| chains — no bonds   | network — permanent | linked or strongly  |
| between chains      | covalent bonds      | entangled chains    |
+---------------------+---------------------+---------------------+
| Melt on heating     | Cannot melt —       | Large reversible    |
| Resolidify on cool  | char/decompose      | deformation (~500%) |
+---------------------+---------------------+---------------------+
| Recyclable by melt  | NOT recyclable by   | Partially recyclable|
| process (in theory) | melt process        | (ground rubber)     |
+---------------------+---------------------+---------------------+
| ~80% of global      | ~15% of global      | ~5% + rubber sector |
| plastic production  | plastic production  | (tires, seals)      |
+---------------------+---------------------+---------------------+
| PE, PP, PVC, PET,   | Epoxy, polyester,   | NR, SBR, EPDM,     |
| PS, ABS, Nylon, PC  | phenolic, PU rigid  | NBR, silicone, CR   |
+---------------------+---------------------+---------------------+
```

## Five Numbers That Govern Behavior

```
+--------------------------------------------------------------+
|  CRITICAL PARAMETERS FOR ANY POLYMER                        |
|                                                              |
|  Tg  — Glass Transition Temperature                         |
|         Below Tg: glassy / brittle / stiff                  |
|         Above Tg: rubbery / leathery / ductile              |
|         PET: 80°C   PC: 147°C   PMMA: 105°C                |
|                                                              |
|  Tm  — Melt Temperature (crystalline polymers only)         |
|         Amorphous polymers have no sharp Tm                 |
|         PP: 160–170°C   HDPE: 130–135°C   PET: 260°C       |
|                                                              |
|  Mw  — Weight-average Molecular Weight                      |
|  Mn  — Number-average Molecular Weight                      |
|         HDPE: Mw ~ 100,000–500,000 g/mol                   |
|         Ultra-high: UHMWPE ~ 3,000,000–6,000,000 g/mol     |
|                                                              |
|  PDI — Polydispersity Index = Mw/Mn                        |
|         PDI ~ 1.0: perfectly monodisperse (theoretical)     |
|         PDI ~ 1.1: living polymerization (anionic, ATRP)    |
|         PDI ~ 2.0: free radical (most commodity plastics)   |
|         PDI > 5: controlled degradation or branching        |
+--------------------------------------------------------------+
```

## The Major Polymer Families

```
POLYOLEFINS  (C-C backbone, no heteroatoms)
  PE   — polyethylene       LDPE / LLDPE / HDPE / UHMWPE
  PP   — polypropylene      Isotactic / syndiotactic / atactic
  PB-1 — polybutene-1       Pipe, packaging

STYRENICS  (benzene ring pendant)
  PS   — polystyrene        GPPS (general purpose) / HIPS / EPS (foam)
  ABS  — acrylonitrile-butadiene-styrene   (rubber-toughened)
  SAN  — styrene-acrylonitrile

VINYLS  (halogen or polar groups)
  PVC  — polyvinyl chloride         Rigid / Flexible (plasticized)
  PVDC — polyvinylidene chloride    Barrier film (Saran)
  PVDF — polyvinylidene fluoride    Chemical resistance, piezo
  PTFE — polytetrafluoroethylene    Teflon — extreme chem/temp resistance

POLYESTERS
  PET  — polyethylene terephthalate  Bottle, fiber, film
  PBT  — polybutylene terephthalate  Engineering connector, auto
  PC   — polycarbonate (bisphenol-A) Optical, impact, medical

POLYAMIDES (Nylons)
  PA-6    Caprolactam ring opening     Fiber, film
  PA-66   Adipic acid + hexamethylene  Fiber, engineering
  PA-12   Long-chain, low moisture     Fuel tubing, flexible
  Aramids — Kevlar (PPTA), Nomex (PMIA) — fiber only, high-strength

POLYURETHANES
  Flexible foam  (cushions, mattresses)
  Rigid foam     (insulation boards)
  Elastomer      (shoe soles, wheels)
  Coating/adhesive  (two-part systems)

THERMOSETS
  Epoxy          Structural adhesive, PCB substrate, aerospace matrix
  Phenolic       Bakelite — first synthetic plastic (1907)
  Unsaturated PE Fiberglass boats, consumer parts
  Melamine-F     Laminates, dinnerware
  Cyanate ester  High-Tg aerospace (>250°C)

ELASTOMERS
  NR   — natural rubber (cis-1,4-polyisoprene)
  SBR  — styrene-butadiene rubber  (tire tread)
  EPDM — ethylene propylene diene  (roofing, auto weather seal)
  NBR  — nitrile rubber            (oil resistance, gloves)
  CR   — neoprene                  (wetsuits, hose)
  VMQ  — silicone                  (high-temp, medical)
```

## Common Confusion Points

**"Plastic" vs. "Polymer"**: All plastics are polymers; not all polymers are
plastics. DNA, cellulose, and silk are biopolymers. "Plastic" implies synthetic
origin and ability to be molded — a subset of the polymer world.

**Thermoset harder than thermoplastic?** Not necessarily. UHMWPE and PEEK are
harder/stronger than many soft thermosets. The thermoset/thermoplastic split is
about cross-link architecture, not hardness ranking.

**PET bottle vs. PET fiber vs. PET film**: Same repeat unit
(–OC-C6H4-CO-O-CH2CH2–), radically different properties from orientation and
crystallinity control during processing.

**Resin identification codes (1–7)**: These identify the polymer type, not
curbside recyclability. #1 (PET) and #2 (HDPE) have actual infrastructure. #3
PVC, #6 PS, and #7 "other" are rarely accepted in practice.

**Vulcanization ≠ cross-linking in general**: Vulcanization specifically means
sulfur cross-linking of rubber. Thermosets also cross-link but via different
chemistry (epoxide ring-opening, condensation, etc.).
