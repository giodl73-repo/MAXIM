# Health Psychology & Stress

## The Big Picture

Health psychology applies psychological science to physical health outcomes.
The core insight: stress is not just "feeling bad" — it's a physiological
cascade with measurable immunological, cardiovascular, and neurological
consequences. Cohen's cold studies gave us the cleanest human data. McEwen's
allostatic load gave us the cumulative wear-and-tear framework. Together they
explain why ACEs (adverse childhood experiences) shorten life expectancy.

## Section 1: Stress Response Physiology

```
  TWO SYSTEMS — DIFFERENT TIMESCALES:
  ┌────────────────────────────────────────────────────────────────┐
  │ FAST: Sympatho-Adrenal Medullary (SAM) axis                    │
  │ Timeline: seconds                                              │
  │ Pathway: Hypothalamus → Sympathetic NS → Adrenal MEDULLA       │
  │          → Epinephrine (adrenaline) + Norepinephrine           │
  │ Effects: heart rate ↑, blood pressure ↑, glucose ↑,           │
  │          digestion suspended, immune function temporarily      │
  │          boosted (short-term), pain suppression               │
  │ Purpose: immediate mobilization for fight/flight/freeze        │
  │                                                                │
  │ SLOW: Hypothalamic-Pituitary-Adrenal (HPA) axis               │
  │ Timeline: minutes                                              │
  │ Pathway: Hypothalamus (CRH) → Anterior Pituitary (ACTH)       │
  │          → Adrenal CORTEX → Cortisol                          │
  │ Effects: sustained energy mobilization, immune suppression     │
  │          (chronic), anti-inflammatory (acute), hippocampal    │
  │          remodeling (chronic)                                  │
  │ Purpose: sustained stress response; resource mobilization     │
  └────────────────────────────────────────────────────────────────┘

  HPA AXIS — ANATOMY IN DETAIL:
  ┌────────────────────────────────────────────────────────────────┐
  │ Hypothalamus                                                   │
  │   ↓ CRH (Corticotropin-Releasing Hormone)                     │
  │ Anterior Pituitary                                             │
  │   ↓ ACTH (Adrenocorticotropic Hormone)                        │
  │ Adrenal Cortex                                                 │
  │   ↓ Cortisol (glucocorticoid)                                  │
  │                                                                │
  │ NEGATIVE FEEDBACK: Cortisol → hippocampus + hypothalamus      │
  │ → suppresses CRH and ACTH → cortisol returns to baseline       │
  │                                                                │
  │ DIURNAL RHYTHM:                                                │
  │ Peak: within 30-45 minutes of waking (Cortisol Awakening      │
  │ Response, CAR) — prepares body for day's demands              │
  │ Trough: late evening, promoting sleep                          │
  │                                                                │
  │ DYSREGULATION IN DEPRESSION:                                   │
  │ HPA hyperactivity → elevated baseline cortisol                 │
  │ Blunted negative feedback (DST suppression test)               │
  │ Hippocampal atrophy (volume reduction, especially CA1/CA3)    │
  └────────────────────────────────────────────────────────────────┘

  THE ADAPTIVE → MALADAPTIVE TRANSITION:
  Acute stress → adaptive: mobilizes resources for short-term threat
  Chronic stress → maladaptive: same systems remain activated
  → Immune suppression, cardiovascular damage, neural remodeling

  The stress response evolved for acute physical threats.
  It is poorly suited to chronic social stressors (job insecurity,
  relationship conflict, financial worry) that never physically resolve.
```

## Section 2: Allostatic Load

```
  McEWEN & STELLAR (1993): Allostatic Load

  ALLOSTASIS: achieving stability through change
  (different from homeostasis — a single set point)
  Multiple mediators work together to maintain stability across
  changing demands → requires active regulation, not passive

  ALLOSTATIC LOAD: the cumulative cost of repeated physiological
  activation and failure to shut down adaptive responses

  ┌────────────────────────────────────────────────────────────────┐
  │ TYPE 1: Repeated hits — same stressor recurs frequently       │
  │ → Cardiovascular system re-activated repeatedly               │
  │ → Endothelial damage accumulates                              │
  │                                                                │
  │ TYPE 2: Inability to habituate — normal stressor doesn't shut │
  │ down → Prolonged cortisol → hippocampal damage                │
  │                                                                │
  │ TYPE 3: Inadequate response — depleted system can't mount     │
  │ adequate stress response → Different pathology                │
  └────────────────────────────────────────────────────────────────┘

  ALLOSTATIC LOAD BIOMARKERS:
  Primary mediators: Cortisol, DHEA-S, norepinephrine,
                     epinephrine, inflammatory cytokines (IL-6, TNF-α)
  Secondary outcomes: Blood pressure, waist-hip ratio, total cholesterol/
                      HDL, HbA1c, immune function (NK cells)
  Tertiary outcomes: Actual disease (CVD, diabetes, cognitive decline)

  ADVERSE CHILDHOOD EXPERIENCES (ACEs):
  Felitti et al. (1998): retrospective study, 17,000+ Kaiser Permanente patients
  Categories: abuse (physical/sexual/emotional), neglect, household dysfunction
              (substance abuse/mental illness/domestic violence/incarceration/
              divorce in household)
  ACE Score (0-10) → dose-response with health outcomes

  ┌────────────────────────────────────────────────────────────────┐
  │ ACE Score ≥ 4 vs 0:                                           │
  │ Alcoholism: 7× more likely                                    │
  │ Depression: 4.6× more likely                                   │
  │ Suicide attempt: 12× more likely                               │
  │ IV drug use: 10× more likely                                   │
  │ Heart disease: 2.2× more likely                                │
  │ COPD: 3.1× more likely                                         │
  │ Life expectancy reduction: ~20 years for ACE score ≥ 6        │
  └────────────────────────────────────────────────────────────────┘

  MECHANISM: ACEs activate stress response systems during sensitive
  developmental periods → epigenetic changes → altered HPA set point
  → lifelong increased stress reactivity
```

## Section 3: Stress-Illness Pathways

```
  NEUROIMMUNOLOGICAL PATHWAY:
  ┌────────────────────────────────────────────────────────────────┐
  │ Stress → SNS activation → pro-inflammatory cytokine release   │
  │ (IL-6, IL-1β, TNF-α, CRP)                                     │
  │ → Systemic inflammation → atherosclerosis, cancer proliferation│
  │   depression (inflammatory depression subtype), dementia      │
  │                                                                │
  │ Paradox: Acute stress → immune boost (fights infection)        │
  │          Chronic stress → immune suppression + dysregulation   │
  └────────────────────────────────────────────────────────────────┘

  CARDIOVASCULAR PATHWAY:
  Repeated BP spikes → endothelial shear stress → lipid infiltration
  → arterial plaque formation → atherosclerosis
  → Coronary artery disease, stroke
  Hostility/anger as the dangerous Big Five-adjacent trait for
  cardiovascular risk (Type A research; Dembroski)

  COGNITIVE/NEURAL PATHWAY:
  Cortisol → glucocorticoid receptors in hippocampus
  → Dendrite retraction, neurogenesis suppression (CA3 especially)
  → Memory impairment (especially declarative/contextual)
  → In chronic depression: measurable hippocampal volume reduction
  Reversible in some cases: antidepressants + exercise restore BDNF
  and hippocampal volume

  BEHAVIORAL PATHWAY:
  Stress → sleep disruption → more stress → more cortisol → immune down
  Stress → substance use (alcohol, tobacco) → direct health effects
  Stress → dietary changes → cardiovascular/metabolic consequences
  Stress → exercise avoidance → fitness decline → health decline

  TELOMERE SHORTENING:
  Blackburn & Epel (Nobel 2009 for telomere/telomerase work):
  Telomeres: protective caps on chromosome ends; shorten with cell division
  Psychological stress → oxidative stress → accelerated telomere shortening
  → Cellular aging proxy
  Nurse-caregiver study: caregivers of chronically ill children had
  significantly shorter telomeres (Epel et al. 2004)
  Effect size: ~10 years of additional biological aging
```

## Section 4: Cohen's Cold Studies

```
  COHEN ET AL. (1991, 1997, 2003, Carnegie Mellon):

  WHY THESE STUDIES MATTER:
  Most stress-illness research is retrospective or correlational.
  Cohen's studies are PROSPECTIVE — measure stress BEFORE illness.

  PROTOCOL:
  1. Assess participants' psychological stress (questionnaire) + behavior
  2. Quarantine participants
  3. Administer rhinovirus (common cold) or flu virus via nasal drops
  4. Monitor viral replication AND symptom development over 5-7 days
  5. Measure: (a) viral replication (infection), (b) clinical cold symptoms

  KEY FINDING:
  ┌────────────────────────────────────────────────────────────────┐
  │ Dose-response: higher Psychological Stress Index (PSI) →       │
  │ higher rate of clinical cold development                       │
  │                                                                │
  │ PSI 1: 27% infection rate                                      │
  │ PSI 5: 47% infection rate                                      │
  │ Effect persists after controlling for: age, sex, education,   │
  │ weight, smoking, alcohol, exercise, diet, pre-existing         │
  │ antibody levels, season                                        │
  └────────────────────────────────────────────────────────────────┘

  EXTENDED FINDINGS:
  • Chronic stressors (>1 month) more immunosuppressive than acute
  • Social ties: diverse social network → lower infection rates
    (protective independent of social support quality)
  • Sleep: <7 hours/night → 3× more likely to develop cold after
    viral exposure (Cohen 2015 — controlled nasal inoculation study)
  • Positive emotional style: independently protective after
    controlling for negative affect
```

## Section 5: Resilience

```
  BONANNO (2004): Four Trajectories After Adversity:
  ┌────────────────────────────────────────────────────────────────┐
  │ RESILIENT: ~65% of bereaved/trauma survivors                   │
  │ Minimal sustained distress; stable functioning throughout      │
  │ NOT absence of negative emotion — normal grief that doesn't   │
  │ become disorder; not pathological denial                        │
  │                                                                │
  │ RECOVERY: ~25% (after bereavement/loss)                        │
  │ Initial high distress → return to pre-trauma functioning       │
  │ Takes 1-3 years; previously called the "normal" pattern        │
  │                                                                │
  │ CHRONIC DYSFUNCTION: ~10%                                      │
  │ Persistent elevated distress; doesn't remit without treatment  │
  │ → PTSD, MDD, complex grief; predicted by: prior mental health, │
  │   low social support, severe/prolonged stressor               │
  │                                                                │
  │ DELAYED ONSET: small minority                                  │
  │ Initial coping → delayed deterioration months/years later      │
  │ Less common than clinical folklore suggests                    │
  └────────────────────────────────────────────────────────────────┘

  RESILIENCE vs RECOVERY:
  Resilience = maintained functioning through adversity (NOT absence of pain)
  Recovery = return to prior functioning after distress period
  Most people are resilient — this contradicts the stage model assumption
  that grief must pass through predictable stages (Kübler-Ross)

  PROTECTIVE FACTORS FOR RESILIENCE:
  ┌────────────────────────────────────────────────────────────────┐
  │ Social support: most robust predictor across studies           │
  │ Self-efficacy: belief in own ability to cope                   │
  │ Cognitive flexibility: ability to reframe and find meaning     │
  │ Positive emotions: broaden-and-build (Fredrickson) — positive  │
  │   emotions undo physiological effects of negative emotions     │
  │ Meaning-making: finding significance in experience            │
  │ Physical health: fitness and sleep buffer against stress        │
  └────────────────────────────────────────────────────────────────┘

  POST-TRAUMATIC GROWTH (Tedeschi/Calhoun 1996):
  Some trauma survivors report positive change following struggle
  Domains: personal strength, new possibilities, relating to others,
           appreciation of life, spiritual change
  Controversy: is reported growth accurate (vs rationalization/bias)?
  Research: growth and distress can coexist; growth doesn't require
  absence of suffering; growth is more common than PTSD in most populations
```

## Section 6: Positive Psychology & Well-Being Science

```
  SELIGMAN (2011): PERMA Framework

  THE POSITIVE PSYCHOLOGY PROJECT:
  Psychology's disease model → treatment of pathology → 50/50 effort
  Seligman's argument: ~50% of effort should go to positive functioning,
  not just reduction of illness

  PERMA:
  ┌────────────────────────────────────────────────────────────────┐
  │ P — POSITIVE EMOTIONS: not just happiness but the full range  │
  │     Joy, gratitude, serenity, hope, awe, elevation            │
  │     Broaden-and-build (Fredrickson): positive emotions → broad │
  │     thought-action repertoires → build durable personal resources│
  │                                                                │
  │ E — ENGAGEMENT: flow states (Csikszentmihalyi); fully absorbed │
  │     in activity; challenge/skill match; time distortion        │
  │                                                                │
  │ R — RELATIONSHIPS: positive connections with others            │
  │     Social support is the strongest predictor of well-being    │
  │     and resilience across all studies                          │
  │                                                                │
  │ M — MEANING: belonging to and serving something larger than    │
  │     yourself; purpose; coherent life narrative                 │
  │                                                                │
  │ A — ACCOMPLISHMENT: goal pursuit and achievement for its own   │
  │     sake; agency; growth                                       │
  └────────────────────────────────────────────────────────────────┘

  EVIDENCE BASE FOR POSITIVE PSYCHOLOGY INTERVENTIONS:
  ┌────────────────────────────────────────────────────────────────┐
  │ Three good things / gratitude journaling: d~0.3-0.5            │
  │ Best possible self (write about future ideal you): d~0.3       │
  │ Signature strengths use: moderate effects; depends heavily on  │
  │   individual engagement                                        │
  │ Savoring / mindful appreciation: small effects                 │
  │ Kindness acts: short-term but inconsistent long-term effects   │
  └────────────────────────────────────────────────────────────────┘
  Overall: interventions work but effects are modest and often
  short-lived; intervention quality matters enormously; WEIRD
  problem is acute here; commercial wellness applications overstate

  HEDONIC ADAPTATION:
  We return to a baseline happiness set point after positive/negative
  events (Brickman et al.: lottery winners vs accident victims)
  Implication: income beyond ~$75K (or $100K in 2021 revision by
  Killingsworth) shows diminishing returns to happiness;
  Kahneman/Deaton found plateau; Killingsworth found continued
  positive correlation — ongoing debate at high income levels
```

## Section 7: Mindfulness

```
  JON KABAT-ZINN: Mindfulness-Based Stress Reduction (MBSR)
  Developed at UMass Medical (1979) for chronic pain patients
  Defined: "paying attention in a particular way: on purpose,
  in the present moment, and non-judgmentally"

  CLINICAL DERIVATIVES:
  MBSR: 8-week program; chronic pain, stress, anxiety
  MBCT (Mindfulness-Based Cognitive Therapy): MBSR + CBT for
    recurrent depression; prevents relapse (3 or more episodes:
    relapse rate 66% → 43%); guideline-recommended in UK (NICE)
  DBT: mindfulness as first module; built for BPD (see 03-CLINICAL)
  ACT: mindfulness as defusion from thoughts; context of values

  META-ANALYTIC EVIDENCE:
  ┌────────────────────────────────────────────────────────────────┐
  │ Anxiety: d~0.5-0.7 (moderate-large)                           │
  │ Depression: d~0.4-0.6 (comparable to antidepressants)         │
  │ Chronic pain: d~0.3-0.5 (pain intensity AND acceptance)       │
  │ Largely non-inferior to active treatments (CBT, medication)    │
  │ for most conditions — not clearly superior                    │
  │                                                                │
  │ Best evidence for: recurrent depression prevention (MBCT),    │
  │ chronic pain, anxiety, caregiver stress, substance cravings   │
  │ Weaker evidence for: immune function, "performance", longevity│
  └────────────────────────────────────────────────────────────────┘

  NEURAL CORRELATES:
  Default Mode Network (DMN): active at rest; mind-wandering
  Mindfulness practice → DMN deactivation during tasks → less
  mind-wandering → greater present-moment focus
  Anterior cingulate cortex: attention regulation, reinforced
  Insula: interoceptive awareness (body-awareness emphasis in MBSR)
  Amygdala reactivity: reduced with long-term practice

  COMMERCIAL DISTORTION:
  Buddhist origin of mindfulness practice largely stripped away
  → "Mindfulness" applied to everything: products, apps, "mindful eating"
  → Poorly standardized practice quality in many studies
  → McMindfulness critique: individualizes social problems
  → Performance mindfulness (Goldman Sachs, NFL): outcomes unclear
```

## Section 8: Sleep

```
  SLEEP ARCHITECTURE:
  ┌────────────────────────────────────────────────────────────────┐
  │ N1 (NREM Stage 1): Light sleep; hypnic jerks; easily woken    │
  │   ~5% of night; theta waves                                   │
  │                                                                │
  │ N2 (NREM Stage 2): True sleep onset; sleep spindles + K-      │
  │   complexes; ~50% of night; memory consolidation starts       │
  │                                                                │
  │ N3 (NREM Stage 3 / Slow Wave Sleep / Deep Sleep):              │
  │   Delta waves; most restorative; growth hormone release;       │
  │   ~20% of night; immune function; declarative memory          │
  │                                                                │
  │ REM (Rapid Eye Movement): Dream sleep; muscle atonia;          │
  │   ~25% of night; emotional memory processing; creative        │
  │   association; first REM ~90 min after sleep onset            │
  └────────────────────────────────────────────────────────────────┘

  SLEEP CYCLE:
  ~90-minute cycles through N1→N2→N3→N2→REM
  Early cycles: more N3 (deep sleep)
  Later cycles: more REM (dreaming)
  → Cutting sleep short disproportionately cuts REM

  CIRCADIAN RHYTHM:
  Suprachiasmatic Nucleus (SCN) in hypothalamus = master clock
  Light → retinal ganglion cells → SCN → suppresses melatonin
  Melatonin (pineal gland): not a sleep drug; a darkness signal
  Blue light (screens) = short wavelength → powerful SCN signal
  → Evening screen use delays circadian phase (social jet lag)
  Chronotypes: genetic variation in circadian timing (CLOCK genes);
  adolescents biologically shifted to later sleep (~2 hours)

  SLEEP DEPRIVATION EFFECTS:
  ┌────────────────────────────────────────────────────────────────┐
  │ Cognitive: sustained attention impaired first; resembles       │
  │   intoxication; 24h awake ≈ 0.10 BAC impairment              │
  │   Working memory, executive function, decision quality        │
  │                                                                │
  │ Emotional: amygdala reactivity increased ~60% (less PFC       │
  │   regulation); mood lability, stress intolerance              │
  │                                                                │
  │ Immune: <7 hrs → 3× more susceptible to rhinovirus           │
  │   (Cohen et al.) NK cell activity reduced                     │
  │                                                                │
  │ Metabolic: insulin sensitivity reduced; hunger hormones        │
  │   shifted (ghrelin ↑, leptin ↓) → increased appetite and     │
  │   caloric intake; long-term → T2DM and obesity risk           │
  │                                                                │
  │ Cardiovascular: 1-hour sleep loss (spring clock change) →     │
  │   24% increase in cardiac events next day (Sandhu 2014)       │
  └────────────────────────────────────────────────────────────────┘

  "SLEEP DEBT" MYTH:
  Can you recover lost sleep? Partially: performance bounces back
  after recovery sleep. But not fully: cognitive deficits persist
  beyond apparent subjective recovery. Chronic sleep restriction
  creates a stable setpoint of impairment that users adapt to
  (and underestimate).

  TREATMENT — CBT-I vs Pharmacological:
  CBT-I (Cognitive-Behavioral Therapy for Insomnia):
  Components: sleep restriction, stimulus control, cognitive
  restructuring, relaxation training, sleep hygiene
  Efficacy: superior to medication in long-term maintenance;
  effects persist at 1-year follow-up; first-line per AASM
  Pharmacological: benzos, z-drugs (zolpidem) — effective short-term;
  tolerance/dependence; rebound insomnia; falls risk in elderly
```

## Section 9: Occupational Health Models

```
  KARASEK DEMAND-CONTROL MODEL (1979):

  Two dimensions:
  ┌────────────────────────────────────────────────────────────────┐
  │             │  Low Control   │   High Control                 │
  │  High       │  HIGH STRAIN   │   ACTIVE (healthy challenge)   │
  │  Demands    │  (burnout risk)│   Learning + motivation        │
  │─────────────┼────────────────┼──────────────────────────────  │
  │  Low        │  PASSIVE       │   LOW STRAIN (least stress)    │
  │  Demands    │  (skills decay)│                                │
  └────────────────────────────────────────────────────────────────┘

  High Strain = high demands + low control = highest CVD risk,
  psychological distress, sick leave
  ISO STRAIN = high strain + low social support: most dangerous

  Extension — Johnson/Hall: added SOCIAL SUPPORT as third dimension
  Iso-strain (high demands + low control + low support) → highest
  cardiovascular disease risk in large longitudinal studies

  SIEGRIST EFFORT-REWARD IMBALANCE (ERI, 1996):

  ERI = Effort (demands + obligations) / Reward (income + esteem + status)
  When effort >> reward → threat to self-regulatory resources → strain

  Components of Reward:
  • Financial compensation
  • Esteem (recognition, respect)
  • Job security and career opportunity (status control)

  Over-commitment (personality factor): tendency to over-invest
  regardless of reward → amplifies ERI effects

  EVIDENCE: ERI predicts CVD, depression, alcohol use beyond
  demand-control model; validated in Europe/Japan/US/Asia

  JD-R MODEL (Bakker/Demerouti) — see 04-ORGANIZATIONAL.md
  Most flexible model; applies to any job; resources buffer demands
```

| Health Psychology Question | Evidence-Based Answer |
|---|---|

## Common Confusion Points

**Acute vs chronic stress are physiologically opposite in some effects**: acute
stress boosts immune function; chronic stress suppresses it. The same system
that protects you from the immediate infection can cause the vulnerability if
chronically activated. The "stress = bad" heuristic is too simple.

**Allostatic load ≠ cumulative stress exposure**: it's cumulative PHYSIOLOGICAL
wear, measured biologically. Perceived stress and biological load can diverge.
People with high perceived stress but strong coping resources may have lower
allostatic load than those who deny stress but show elevated biomarkers.

**Resilience is not the absence of grief**: Bonanno's resilient trajectory still
involves sadness, pain, and difficulty. The distinguishing feature is that these
do not reach clinical levels and do not persist. Calling someone "not resilient"
because they express grief is a misuse of the construct.

**MBSR is not meditation generally**: MBSR is a specific, structured 8-week
program with defined practices. "Mindfulness" apps and corporate programs are
often much weaker versions. Study quality in mindfulness research is highly
variable; many trials lack active control groups.

**Sleep debt is partially real but overrecoverable in people's minds**: you can
recover some sleep debt, but cognitive performance doesn't fully bounce back
even after subjects report feeling fine. The subjective adaptation to chronic
restriction fools both the person and naive self-assessments of performance.
