# Persuasion & Influence

## The Big Picture

Persuasion research sits at the intersection of social psychology, marketing,
and political science. Cialdini's six principles are the practitioner's toolkit;
the Elaboration Likelihood Model explains WHEN each technique works; inoculation
theory tells you how to build resistance. These are not soft ideas — they're
mechanisms with identified neural correlates and measurable effect sizes.

```
+----------------------------------------------------------------------+
|                    INFLUENCE LANDSCAPE                                |
|                                                                      |
|  INDIVIDUAL LEVEL          SYSTEMIC/ARCHITECTURAL LEVEL             |
|  ────────────────          ────────────────────────────             |
|  Cialdini 6 principles     Nudge theory (Thaler/Sunstein)           |
|  RECIPROCITY/COMMITMENT    Default effects                          |
|  SOCIAL PROOF/AUTHORITY    EAST framework                           |
|  LIKING/SCARCITY           Choice architecture                      |
|  + Pre-suasion (2016)      Dark patterns (deceptive UI)             |
|                                                                      |
|  COGNITIVE ROUTE MODEL     DISINFORMATION RESISTANCE                |
|  ──────────────────────    ────────────────────────────             |
|  ELM (Petty/Cacioppo):     Inoculation theory (McGuire)            |
|  Central route: effortful  Prebunking vs debunking                 |
|  Peripheral route: cues    SIREN/inoculation vaccine model         |
|  Motivation × Ability      Illusory truth, firehose effects        |
+----------------------------------------------------------------------+
```

## Section 1: Cialdini's Six Principles

Robert Cialdini's *Influence* (1984) identified six universal persuasion principles
through field research. Each has a mechanism — not just a label.

### The Six Principles

```
  RECIPROCITY:
  ┌──────────────────────────────────────────────────────────────────┐
  │ Principle: We feel obligated to return favors, gifts, concessions│
  │                                                                  │
  │ Mechanism: Evolutionarily adaptive norm enforcement — reciprocity│
  │ enforces cooperation in repeated-game environments               │
  │                                                                  │
  │ Exploitation techniques:                                         │
  │ • Uninvited gifts: Hare Krishna flowers; Amway samples; free     │
  │   mints → obligation to tip (18% vs 3% increase per study)      │
  │ • Door-in-the-face (DITF): Large request refused → smaller       │
  │   request accepted; concession reciprocates concession          │
  │ • Personalization magnifies effect: gift addressed to recipient  │
  └──────────────────────────────────────────────────────────────────┘

  COMMITMENT / CONSISTENCY:
  ┌──────────────────────────────────────────────────────────────────┐
  │ Principle: Once we commit, we align future behavior to look      │
  │ consistent; identity-based pressure                              │
  │                                                                  │
  │ Mechanism: Cognitive dissonance (Festinger) + self-perception    │
  │ (Bem) — both drive consistency-seeking behavior                  │
  │                                                                  │
  │ Exploitation techniques:                                         │
  │ • Foot-in-the-door (FITD): Small request → large request         │
  │   Freedman/Fraser: agree to small sign → 76% agreed to large     │
  │ • Public > private commitment (written > verbal)                 │
  │ • Lowball: agree to offer → conditions worsen → still comply     │
  │ • Self-labeling: "You seem like the kind of person who..."       │
  └──────────────────────────────────────────────────────────────────┘

  SOCIAL PROOF:
  ┌──────────────────────────────────────────────────────────────────┐
  │ Principle: In uncertainty, we look to others' behavior as        │
  │ evidence of correct action                                       │
  │                                                                  │
  │ Mechanism: Genuine heuristic — others' behavior IS informative   │
  │ in complex environments; exploited when information is absent    │
  │                                                                  │
  │ Exploitation techniques:                                         │
  │ • Canned laughter: Audiences laugh more even at non-funny jokes  │
  │ • "Most popular" product labeling (hotel towel reuse: 26%→33%)  │
  │ • Testimonials, review counts, "bestseller" labels               │
  │ • Pluralistic ignorance (bystander effect: everyone looks calm)  │
  │ More powerful when: targets are uncertain, targets are similar   │
  │ to the social proof referents                                    │
  └──────────────────────────────────────────────────────────────────┘

  AUTHORITY:
  ┌──────────────────────────────────────────────────────────────────┐
  │ Principle: We defer to perceived experts and legitimate          │
  │ authorities to make correct decisions efficiently                │
  │                                                                  │
  │ Mechanism: Genuine epistemic efficiency (experts often ARE right)│
  │ + exploitation through symbols of authority (not substance)     │
  │                                                                  │
  │ Exploitation techniques:                                         │
  │ • White coat effect: nursing study — unauthorized medication     │
  │   dose followed if ordered by "doctor" voice on phone           │
  │ • Titles, uniforms, job titles in email signature               │
  │ • "9 out of 10 dentists recommend" (constructed authority)       │
  │ Milgram connection: legitimate authority is the mechanism        │
  └──────────────────────────────────────────────────────────────────┘

  LIKING:
  ┌──────────────────────────────────────────────────────────────────┐
  │ Principle: We comply with and are persuaded by people we like    │
  │                                                                  │
  │ Mechanism: Halo effect (positive trait → global positive        │
  │ attribution) + reduced cognitive scrutiny of liked sources       │
  │                                                                  │
  │ Predictors of liking:                                            │
  │ • Physical attractiveness (even in unrelated domains)            │
  │ • Similarity: shared demographics, opinions, interests           │
  │ • Familiarity: mere exposure effect (Zajonc) — repeated          │
  │   exposure increases liking even without memory of exposure      │
  │ • Cooperative context: working together → liking (contact hyp.) │
  │ • Flattery: works even when obviously strategic                  │
  │                                                                  │
  │ Exploitation: Tupperware parties; "warm up" scripts; sports      │
  │ team endorsements; political rally celebrity appearances         │
  └──────────────────────────────────────────────────────────────────┘

  SCARCITY:
  ┌──────────────────────────────────────────────────────────────────┐
  │ Principle: We value things more when they are rare or            │
  │ becoming unavailable                                             │
  │                                                                  │
  │ Mechanisms:                                                       │
  │ • Loss aversion (Kahneman/Tversky): losing access > gaining      │
  │ • Uniqueness motivation: rare = exclusive = status               │
  │ • Reactance (Brehm): restricting access makes desire stronger   │
  │                                                                  │
  │ Exploitation techniques:                                         │
  │ • "Limited time offer" countdown timers                          │
  │ • "Only 3 left in stock" — inventory artificially low            │
  │ • "Application deadline" for no-deadline products               │
  │ • Exclusive/waitlist framing                                     │
  │ Note: artificial scarcity still works psychologically            │
  └──────────────────────────────────────────────────────────────────┘
```

### Pre-Suasion (Cialdini 2016)

```
  PRE-SUASION: Priming the context BEFORE the message

  Core insight: What captures attention in the moment before
  a message is delivered shapes how that message is processed

  MECHANISMS:
  • Attention directs focus → focus signals importance → important
    things receive favorable processing
  • Priming associations: show patriotic images → more support for
    domestic products
  • Channel factors: small barriers removed before ask

  EXAMPLES:
  Asking "Are you adventurous?" before pitching new product → higher
  acceptance (they've self-identified as open to novelty)

  Showing pleasant nature images in background of ad → higher brand
  warmth ratings (affect transfers to nearby object in attention)

  PRE-SUASION vs PRIMING:
  Priming (subliminal, fast): associative spreading activation
  Pre-suasion: deliberate attention management + identity activation
  Both exploit the same basic mechanism: current focus = current value
```

## Section 2: Elaboration Likelihood Model (ELM)

```
  PETTY & CACIOPPO (1986):

  KEY QUESTION: When does message quality matter vs source cues?

  TWO ROUTES TO PERSUASION:
  ┌────────────────────────────────────────────────────────────────┐
  │ CENTRAL ROUTE (High Elaboration):                              │
  │ Effortful, systematic processing of argument quality           │
  │ Engage when: motivated + able to process                       │
  │ Requirements: subject involvement, sufficient cognitive        │
  │   capacity, time, relevant knowledge                           │
  │ Outcome: DURABLE attitude change, resistant to counter-       │
  │ persuasion, predicts behavior                                  │
  │                                                                │
  │ PERIPHERAL ROUTE (Low Elaboration):                            │
  │ Heuristic cue processing — shortcuts, not argument quality     │
  │ Engage when: not motivated OR not able to process              │
  │ Cues used: source attractiveness, expertise, number of        │
  │   arguments, audience reaction, music, etc.                    │
  │ Outcome: SURFACE change, fragile under counter-persuasion,    │
  │ less predictive of behavior                                    │
  └────────────────────────────────────────────────────────────────┘

  DETERMINANTS OF ROUTE:
  MOTIVATION: personal relevance (high stake → central route)
  ABILITY: cognitive load reduces elaboration → peripheral
           distraction, time pressure, complexity → peripheral

  THE ELABORATION CONTINUUM:
  Low elaboration ←──────────────────────────→ High elaboration
  Peripheral cues                          Argument quality
  dominates                                dominates

  KEY PREDICTIONS:
  • High-quality arguments work ONLY if audience is elaborating
  • Celebrity endorsements work for LOW-involvement products
  • For high-stakes decisions (health, financial), provide quality
    arguments AND engage motivation
  • Fear appeals: moderate fear → central processing of solution;
    high fear → defensive avoidance (Witte's Extended Parallel
    Process Model extends ELM for health communication)

  HEURISTIC-SYSTEMATIC MODEL (Chaiken):
  Parallel model with different terminology; HSM adds
  "systematic processing supplements heuristics" — they can
  run simultaneously; ELM implies more exclusive routing
```

## Section 3: Inoculation Theory

```
  MCGUIRE (1961): Medical analogy for belief resistance

  CORE INSIGHT: Resistance to persuasion is not just about
  confidence in your beliefs — it requires PRACTICE countering
  attacks on those beliefs

  TWO-COMPONENT INOCULATION:
  ┌──────────────────────────────────────────────────────────────┐
  │ 1. THREAT component: warn that an attack is coming; creates  │
  │    motivation to defend                                      │
  │                                                              │
  │ 2. REFUTATION component: expose to weakened form of the      │
  │    attack + counterargument; builds cognitive antibodies     │
  └──────────────────────────────────────────────────────────────┘

  PREBUNKING vs DEBUNKING:
  ┌──────────────────────────────────────────────────────────────┐
  │ DEBUNKING (fact-checking): correct misinformation after      │
  │ exposure; problem = impression perseverance (belief persists │
  │ after correction), cognitive effort, often too late          │
  │                                                              │
  │ PREBUNKING (inoculation): expose to manipulation TECHNIQUE   │
  │ before encountering misinformation; teaches recognition of   │
  │ rhetorical moves rather than correcting specific facts       │
  │                                                              │
  │ TECHNIQUE-BASED INOCULATION (Lewandowsky/van der Linden):    │
  │ Identify manipulation techniques (not content), then expose  │
  │ and refute — generalizes across topics because technique     │
  │ recognition transfers                                        │
  └──────────────────────────────────────────────────────────────┘

  INOCULATION TECHNIQUES (common manipulation moves to inoculate against):
  • Ad hominem attacks on scientists/consensus
  • Cherry-picking evidence
  • False equivalence / both-sidesism
  • Exploiting uncertainty to reject established consensus
  • Conspiracy framing (everything is coordinated deception)

  SIREN MODEL (Lewandowsky 2020):
  Source → Inoculation → Retrieval → Encoding → Neutralization
  Operationalizes inoculation for mass communication contexts

  EVIDENCE:
  • Van der Linden et al. (2017): one-paragraph inoculation
    reduced false balance framing effect on climate consensus
  • Games (Harmony Square, Bad News): play as disinformation
    creator → builds recognition and resistance
  • Effect sizes: moderate but consistent across topics
  • Inoculation effects DECAY over time → booster shots needed
```

## Section 4: Propaganda Techniques & Disinformation

```
  ILLUSORY TRUTH EFFECT:
  Repeated exposure to a statement → increased belief in its truth
  Even when flagged as false!
  Mechanism: processing fluency — familiar = easy = true (heuristic)
  Implications: repetition is a disinformation weapon;
  correction by repeating the false claim → backfire risk

  FALSE BALANCE (Both-Sidesism):
  Presenting two sides as equally credible when evidence is one-sided
  E.g., climate scientist vs skeptic given equal airtime = false balance
  Mechanism: exploits journalistic norms of "balance"
  Effect: manufactured controversy → public uncertainty

  FIREHOSE OF FALSEHOOD (RAND Cordesman 2015):
  Russian information warfare strategy: flood information space with
  multiple contradictory false claims simultaneously
  Goal: NOT to persuade — to confuse, exhaust, and create cynicism
  ("Nothing is knowable, everyone lies")
  Counter: point out the strategy, not each individual claim

  COORDINATED INAUTHENTIC BEHAVIOR:
  Bot networks, fake account networks amplify fringe content
  → Astroturfing (manufacture appearance of grassroots support)
  Platforms: Facebook's CIB policy; Twitter bot detection
  Research: bots account for ~2/3 of shared URLs for some topics

  DEEPFAKES / SYNTHETIC MEDIA:
  Video/audio manipulation accessible at scale
  Current limitation: detection still possible; future: deteriorates
  Liar's dividend: real evidence can be dismissed as deepfake
```

## Section 5: Online Influence & Algorithmic Amplification

```
  ALGORITHMIC AMPLIFICATION HYPOTHESIS:
  Engagement optimization → outrage/emotional content rewarded
  → polarizing content surfaces → beliefs radicalize

  EVIDENCE (more complex than the hypothesis):
  ┌──────────────────────────────────────────────────────────────┐
  │ Supporting: YouTube radicalization pathways (Ribeiro);       │
  │ Facebook "angry" emoji → increased inflammatory content       │
  │ (internal docs)                                               │
  │                                                              │
  │ Complicating:                                                 │
  │ • Most misinformation shares by small number of users         │
  │ • Exposure to cross-cutting content higher online than        │
  │   offline (Guess et al.)                                      │
  │ • Bail et al. (2018): exposure to opposing Twitter bot →     │
  │   increased polarization (backfire effect)                   │
  │ • Sunstein echo chamber claim: empirically contested          │
  └──────────────────────────────────────────────────────────────┘

  ACTUAL CONSENSUS (as of ~2024):
  • Algorithms amplify emotionally activating content (clear)
  • Direct causal path from algorithms to polarization: uncertain
  • Pre-existing partisanship predicts consumption patterns more
    than algorithmic exposure
  • Social sharing (people to people) > algorithmic recommendation
    for most misinformation propagation

  FILTER BUBBLES:
  Eli Pariser (2011): personalization traps users in info silos
  Empirical support: mixed; people DO seek out like-minded content
  independently of algorithmic sorting; the filter is mostly people
```

## Section 6: Nudge Theory

```
  THALER & SUNSTEIN (Nudge, 2008):

  CORE CONCEPT:
  Choice architecture — the way choices are presented — affects
  decisions without restricting options or changing incentives
  "Libertarian paternalism": preserve freedom, steer toward better outcomes

  DEFAULT EFFECTS:
  ┌──────────────────────────────────────────────────────────────┐
  │ Opt-OUT: to NOT be enrolled, you must take action            │
  │ Opt-IN: to BE enrolled, you must take action                 │
  │                                                              │
  │ RESULT: ~90%+ choose the default regardless of which it is  │
  │                                                              │
  │ Classic case: organ donation rates                           │
  │ Germany (opt-in): ~12% enrolled                              │
  │ Austria (opt-out): ~100% enrolled                            │
  │                                                              │
  │ 401k enrollment: auto-enrollment → savings rates triple      │
  │ (Madrian/Shea 2001)                                          │
  └──────────────────────────────────────────────────────────────┘

  MECHANISM: Status quo bias + loss aversion + cognitive effort
  avoidance; changing the default changes the path of least resistance

  EAST FRAMEWORK (UK Behavioural Insights Team):
  ┌──────────────────────────────────────────────────────────────┐
  │ E — EASY: reduce friction; default to desired behavior       │
  │ A — ATTRACTIVE: draw attention; make relevant, engaging      │
  │ S — SOCIAL: use social norms ("Most people in your area...") │
  │ T — TIMELY: intervene at right moment (at decision point)    │
  └──────────────────────────────────────────────────────────────┘

  NUDGE APPLICATIONS (with evidence):
  Energy conservation: neighborhood comparison on utility bills
    → 1.4-3.5% reduction; $350M saved in US annually (Opower)
  Tax compliance: "Most people pay on time" → higher payment rate
  Cafeteria healthier eating: placement, prominence, defaults
  COVID vaccination: proactive scheduling + default appointment

  LIMITS:
  Reactance: some nudges trigger psychological reactance (scarcity
  nudge feels manipulative → backlash)
  Effect size: often small; high-quality nudge RCTs show d~0.1-0.3
  Context dependency: nudge in one culture fails in another
```

## Section 7: Dark Patterns

```
  BRIGNULL (2010): "User interfaces designed to trick users into
  unintended actions"

  TAXONOMY:
  ┌──────────────────────────────────────────────────────────────┐
  │ TRICK QUESTIONS: ambiguous wording to get unwanted opt-in    │
  │ "Uncheck to not receive marketing" (double negative)          │
  │                                                              │
  │ ROACH MOTEL: easy to get in, hard to get out                 │
  │ Subscribe in one click; cancel requires 5 steps + phone call  │
  │                                                              │
  │ PRIVACY ZUCKERING: default privacy settings = maximum data   │
  │ sharing; opt-out buried in settings                           │
  │                                                              │
  │ CONFIRMSHAMING: opt-out button says "No, I hate saving money"│
  │ → emotional cost to declining                                 │
  │                                                              │
  │ MISDIRECTION: draw attention away from information you        │
  │ wouldn't want user to notice                                 │
  │                                                              │
  │ HIDDEN COSTS: low price displayed; shipping/fees added at     │
  │ checkout (commitment/consistency + sunk cost)                │
  │                                                              │
  │ NAGGING: repeat permission requests until user capitulates    │
  └──────────────────────────────────────────────────────────────┘

  REGULATORY TRAJECTORY:
  FTC (US): Dark patterns enforcement actions (Amazon Prime 2023
    — FTC sued for roach motel subscription cancellation)
  EU DSA / GDPR: explicit dark patterns provisions
  California: automatic renewal laws
  Direction: increasing regulatory scrutiny; not yet systematically enforced
```

## Decision Cheat Sheet

| Persuasion Goal | Best Tool |
|---|---|
| One-time behavioral compliance | Reciprocity (gift/concession) or FITD/DITF commitment sequence |
| Durable attitude change | Central ELM route: high-quality arguments + motivated audience |
| Quick peripheral compliance | Social proof + authority cues (works for low-involvement) |
| Resist misinformation | Inoculation/prebunking with technique-based approach |
| Population-level behavior change | Nudge defaults + EAST framework |
| Correct existing false belief | Debunking with fact + alternative narrative + emotional re-engagement |
| Identify manipulation | Identify the technique (reciprocity exploitation, false scarcity, etc.) |
| Detect dark patterns | Look for friction asymmetry — easy to accept, hard to decline |

## Common Confusion Points

**Persuasion ≠ manipulation**: persuasion with accurate information and genuine
argument quality is the legitimate form. Manipulation exploits heuristics to
bypass rational evaluation. The same mechanism (reciprocity) can be used
legitimately (exchanging genuine value) or exploitatively (uninvited gifts
engineered to create obligation).

**Nudging is not neutral**: the choice architecture always exists — there is
always a default. The question is whether it was designed thoughtfully or
accidentally. Libertarian paternalism is still paternalism — who decides which
default is "better" is a normative judgment, not a technical one.

**Inoculation requires motivation**: the threat component only activates
resistance-building if the person cares about the belief being attacked. For
low-involvement topics, inoculation effects are weaker. Technique-based
inoculation works better precisely because it doesn't require content-level
engagement.

**ELM peripheral cues don't just fool the dumb**: high-elaboration people still
use peripheral cues — but only when they LACK motivation or ability for a
specific topic. Even experts process peripherally outside their domain.

**Echo chambers are self-selected as much as algorithmic**: people actively
seek confirming information. Algorithms amplify and accelerate this tendency
but did not create it. The fix is not only algorithmic; it requires addressing
the underlying motivated reasoning.
