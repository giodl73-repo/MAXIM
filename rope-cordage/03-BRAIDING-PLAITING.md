# Braiding and Plaiting: Round Braid, Hollow Braid, Kernmantle

## The Big Picture

Braiding creates rope by interlacing strands at angles rather than twisting parallel helices around a central axis. The result is a rotationally balanced structure (no torque under load), better bending compliance, and a round cross-section without the groove channels of 3-strand rope. The trade-off: braided rope is harder to splice using traditional methods, though modern buried splices achieve equivalent efficiency.

```
BRAID vs. TWIST: STRUCTURAL COMPARISON
========================================

  TWISTED (3-strand laid):
  ┌──────────────────────────────────────────────────────────────────┐
  │  Structure: 3 helical strands around common axis                 │
  │  Cross-section: triangular void pattern between strands          │
  │  Torque: Yes — loads twist rope body (Z-lay torques CW)        │
  │  Splicing: Traditional tuck splice (intuitive, manual)           │
  │  Surface: Grooved, rough — higher friction, more abrasion wear   │
  │  Stiffness: More flexible (opens at lay under bending)           │
  └──────────────────────────────────────────────────────────────────┘

  BRAIDED:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Structure: Carriers interlace over-under in helical pattern     │
  │  Cross-section: Round, no voids in hollow/solid braid            │
  │  Torque: None (carriers balanced ±direction)                     │
  │  Splicing: Buried lock-stitch splice (requires technique)        │
  │  Surface: Smooth, round — handles easily, reduced abrasion wear  │
  │  Stiffness: Stiffer at small diameter; more kink-resistant       │
  └──────────────────────────────────────────────────────────────────┘
```

## Braiding Machine Geometry

A maypole braider moves bobbins (carriers) on two interlocking sets of tracks, each set rotating in opposite directions. The carriers snake around each other as they travel, depositing yarn in the diagonal interlacing pattern.

```
BRAIDER GEOMETRY (8-carrier example)
======================================

  TOP VIEW:
  ┌──────────────────────────────────────────────────────────────────┐
  │         4 CW carriers:  ○ → → → ↓ → → → ↑ (clockwise)        │
  │         4 CCW carriers: ● ← ← ← ↓ ← ← ← ↑ (counterclockwise)│
  │                                                                  │
  │         Braid point: center where carriers cross and braid forms│
  │                                                                  │
  │  8-CARRIER BRAID:                                                │
  │  ○ ● ○ ●    (alternating CW and CCW carriers — 4 each)         │
  │  ↓         ↑                                                     │
  │  ● ○ ● ○    (braid forms at center as carriers pass each other)  │
  │                                                                  │
  │  16-CARRIER BRAID:                                               │
  │  8 CW + 8 CCW carriers → tighter braid, rounder section        │
  │  Smaller braid angle per carrier (same total coverage)           │
  │  Smoother surface, less pattern visibility                       │
  └──────────────────────────────────────────────────────────────────┘

  BRAID ANGLE:
  The angle of individual strand helix relative to rope axis.
  Controlled by: carrier speed / take-up speed ratio
  Low braid angle (steep): higher strength, lower elongation
  High braid angle (shallow): lower strength, higher elongation, softer
```

## Round Braid Types

### Hollow Braid

```
HOLLOW BRAID
=============

  ┌──────────────────────────────────────────────────────────────────┐
  │  Construction: Braided sheath with hollow center                 │
  │  Typical carriers: 8 or 16                                     │
  │                                                                  │
  │  Cross-section:                                                  │
  │  ┌────────────────────────────────────────────────────────────┐  │
  │  │                  ╭──────────────────╮                      │ │
  │  │                  │  HOLLOW CENTER   │                      │ │
  │  │   BRAID WALL     │  (empty)         │  BRAID WALL         │  │
  │  │   (interlaced)   ╰──────────────────╯  (interlaced)        │ │
  │  └────────────────────────────────────────────────────────────┘ │
  │                                                                   │
  │  Properties:                                                    │
  │  • Lower strength than double braid (wall carries all load)    │
  │  • Very flexible — minimal resistance to crushing              │
  │  • Can insert core (makes it a "cover" for double braid)       │
  │  • Simple to manufacture on small braiders                     │
  │                                                                   │
  │  Uses: Utility rope, horse lead lines, pool rope, flag halyard │
  └──────────────────────────────────────────────────────────────────┘
```

### Solid Braid

```
SOLID BRAID
============

  ┌──────────────────────────────────────────────────────────────────┐
  │  Construction: Fully filled cross-section                        │
  │  Some carriers lock in place as core; others braid around      │
  │                                                                  │
  │  Cross-section:                                                  │
  │  ┌────────────────────────────────────────────────────────────┐  │
  │  │  ████████████████████████████████████████████████████████  │ │
  │  │  (fully filled — no hollow — all yarns interlaced)         │ │
  │  └────────────────────────────────────────────────────────────┘ │
  │                                                                   │
  │  Properties:                                                    │
  │  • Stiffer than hollow braid (resists crushing and kinking)    │
  │  • Harder to splice (cannot bury easily)                       │
  │  • Good surface uniformity                                     │
  │                                                                   │
  │  Uses: Clothesline, narrow diameter utility, braided fishing   │
  │        line, cotton clothesline                                 │
  └──────────────────────────────────────────────────────────────────┘
```

## Double Braid (Braid-on-Braid)

The double braid is the workhorse of modern marine and industrial rope. It combines a parallel-strand or braided core (high strength) with a braided cover (abrasion protection, handling).

```
DOUBLE BRAID CONSTRUCTION
==========================

  ┌──────────────────────────────────────────────────────────────────┐
  │  CROSS-SECTION:                                                  │
  │  ┌────────────────────────────────────────────────────────────┐  │
  │  │  COVER (braided):  ██████████████████████████████████████  │ │
  │  │  (braided sheath   ██████████████████████████████████████  │ │
  │  │   16 or 32         ██████████████████████████████████████  │ │
  │  │   carriers)        ██████████████▓▓▓▓▓▓▓▓▓▓▓▓▓▓███████   │ │
  │  │                    ███████████▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓██████   │ │
  │  │  CORE (parallel    ███████████▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓██████   │ │
  │  │   strands or       ███████████▓▓▓▓▓ CORE ▓▓▓▓▓▓▓██████   │ │
  │  │   braided):        ███████████▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓██████   │ │
  │  │                    ██████████████████████████████████████  │ │
  │  └────────────────────────────────────────────────────────────┘ │
  │                                                                   │
  │  LOAD DISTRIBUTION:                                             │
  │  Core: typically carries 50–60% of load (parallel fibers, high │
  │         tensile efficiency)                                     │
  │  Cover: carries 40–50% of load (braided at angle, lower        │
  │          efficiency, but high abrasion protection)              │
  │                                                                   │
  │  DESIGN PRINCIPLE:                                              │
  │  Core is optimized for strength (straight or low-angle braid)  │
  │  Cover is optimized for handling and abrasion resistance        │
  │  Together: higher efficiency than hollow braid alone           │
  └──────────────────────────────────────────────────────────────────┘

  SPLICING DOUBLE BRAID: eye splice requires separating cover and core,
  splicing each independently, then reassembling. Requires specific
  fids (hollow pushing tools) sized to each layer. Significantly more
  complex than 3-strand tuck splice.

  STRENGTH: Eye splice in double braid = 95–98% of rope MBS
  (vs. bowline knot = 70% of rope MBS)
```

## Kernmantle Construction

Kernmantle (German: kern = core, mantel = sheath) is the standard construction for climbing rope and technical rescue rope. The core (kern) carries the primary load; the sheath (mantle) provides abrasion resistance and handling properties.

```
KERNMANTLE ANATOMY
==================

  ┌──────────────────────────────────────────────────────────────────┐
  │  MANTLE (sheath):                                                │
  │  Braided, 16–48 carriers                                       │
  │  Nylon or polyester                                              │
  │  Braided at 30–50° angle (optimized for abrasion, not strength)│
  │  Carries 10–30% of load                                        │
  │  Provides: softness, UV protection, abrasion resistance        │
  │                                                                  │
  │  KERN (core):                                                    │
  │  Parallel fiber bundles (for dynamic rope)                     │
  │  OR twisted plied bundles (for static rope)                    │
  │  Carries 70–90% of load                                        │
  │  High tensile efficiency (fibers nearly axial)                 │
  │                                                                  │
  │  TYPICAL SPECIFICATIONS:                                         │
  │  Dynamic climbing (UIAA EN 892):                               │
  │   Diameter: 9–11 mm                                            │
  │   Impact force: ≤ 12 kN (single strand, standard factor fall)  │
  │   Elongation under load: 5–10% (designed for energy absorption)│
  │   Falls held: ≥ 5 (UIAA test: fall factor 1.77)               │
  │                                                                  │
  │  Static rescue (NFPA 1983 Technical Use):                      │
  │   Diameter: 12.5 mm (1/2") minimum                             │
  │   Breaking strength: ≥ 40 kN (4.08 metric tons)               │
  │   Elongation: ≤ 10% at 10% of MBS (stiff under working load)  │
  │   Label: NFPA 1983 Technical Use class                         │
  └──────────────────────────────────────────────────────────────────┘
```

### Dynamic vs. Static Kernmantle

```
DYNAMIC vs. STATIC CLIMBING/RESCUE ROPE
=========================================

  DYNAMIC ROPE (climbing rope, UIAA EN 892):
  ┌──────────────────────────────────────────────────────────────────┐
  │  Purpose: Absorb energy from a FALL                              │
  │  Behavior: High elongation (7–10% in use, 20%+ at break)       │
  │  Core: Loosely twisted bundles (elongation designed in)          │
  │  Falls arrested: ≥ 5 UIAA test falls                           │
  │  Impact force: LIMITED (≤ 12 kN) — force limitation is the    │
  │    key design parameter, not breaking strength                   │
  │                                                                  │
  │  PHYSICS: F = m × a = m × Δv/Δt                                │
  │  More rope elongation → longer Δt → lower peak force on climber│
  │  Dynamic rope trading some energy as elongation = less force   │
  └──────────────────────────────────────────────────────────────────┘

  STATIC ROPE (rescue rope, NFPA 1983):
  ┌──────────────────────────────────────────────────────────────────┐
  │  Purpose: Carry load at controlled descent; controlled ascent    │
  │  Behavior: Low elongation (≤ 6–10% at working load) — stiff   │
  │  Core: Parallel fiber bundles or highly twisted (minimal stretch)│
  │  NOT for dynamic falls: high elongation rope absorbs fall;       │
  │    static rope transmits full impact force → dangerous           │
  │                                                                  │
  │  CATEGORIES (NFPA 1983):                                         │
  │  General Use: 1-person and 2-person loads, 40 kN MBS           │
  │  Technical Use: 1-person loads, 40 kN MBS                      │
  │  Escape: 1-person self-rescue only, 13 kN min, single use      │
  └──────────────────────────────────────────────────────────────────┘
```

## Flat Webbing and Plaiting

Flat webbing (flat braid or woven flat strap) is used where width distribution of load matters — harnesses, slings, lashing.

```
FLAT WEBBING
=============

  TUBULAR WEBBING:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Two-layer flat braid that forms a tube                          │
  │  Cross-section:  ═══════════════════════                         │
  │  Crushes flat but springs back                                   │
  │  Use: climbing harnesses, anchor slings                          │
  │  Width: 1" (25 mm) standard; 9/16" (14 mm) common              │
  │  MBS: 17–22 kN for 1" nylon tubular webbing                    │
  └──────────────────────────────────────────────────────────────────┘

  FLAT (SOLID) WEBBING:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Plain weave or leno weave flat strap                            │
  │  Cross-section:  ════════════════════════  (solid)               │
  │  Stiffer than tubular; higher abrasion resistance                │
  │  Use: industrial lifting slings, cargo straps                    │
  │  Width: 1"–6" (25–150 mm) common                               │
  │  MBS: 1"×2mm polyester → 3,000 lb; 4"×6mm → 12,000 lb         │
  └──────────────────────────────────────────────────────────────────┘

  DYNEEMA WEBBING (high-tech flat):
  ┌──────────────────────────────────────────────────────────────────┐
  │  UHMWPE fibers woven flat → very thin, very strong             │
  │  1" × 1mm Dyneema sling: MBS ~30 kN                            │
```

## Braid Angle and Its Effects

```
BRAID ANGLE EFFECTS
====================

  Braid angle α = angle of carrier strands relative to rope axis

  ┌──────────────────────────────────────────────────────────────────┐
  │  Low α (steep strands):                                          │
  │  ╱╱╱╱╱╱╱╱╱  (steep — nearly parallel to axis)                  │
  │  → High tensile strength (strands nearly axial)                  │
  │  → Low elongation                                                │
  │  → Stiffer in bending                                            │
  │                                                                  │
  │  High α (shallow strands):                                       │
  │  ╱╱╱╱╱╱    (shallow — more diagonal to axis)                   │
  │  → Lower tensile strength                                        │
  │  → Higher elongation (braid must "straighten" before tensioning)│
  │  → More flexible in bending                                      │
  │                                                                  │
  │  TYPICAL VALUES:                                                 │
  │  Low elongation braids (Dyneema, Vectran rigging): α ≈ 10–15°  │
  │  General-purpose marine braid: α ≈ 15–25°                      │
  │  Dynamic kernmantle sheath: α ≈ 30–50°                         │
  └──────────────────────────────────────────────────────────────────┘
```

## Decision Cheat Sheet

| Situation | Braid Type |
|-----------|-----------|
| Marine sheets/halyards, general use | Double braid, polyester cover/core |
| Dynamic climbing fall arrest | Kernmantle, UIAA EN 892 certified |
| Technical rescue, static lowering | Kernmantle, NFPA 1983 Technical Use |
| Torque-free rigging | Any braid (no torque under load — vs. 3-strand) |
| High-strength, low-elongation yacht rigging | Dyneema core, polyester cover double braid |
| Lightweight emergency use | Hollow braid nylon or polyester |
| Industrial lifting slings | Flat polyester webbing (round slings also used) |
| Self-rescue, single-use descent | Kernmantle, NFPA 1983 Escape class |

## Common Confusion Points

**Hollow braid is weaker than double braid of equal diameter.** Hollow braid has a wall (the braid carries load) but air in the center. Double braid fills the core with load-carrying fiber. Same outer diameter, significantly different strength.

**Static rope is not safer than dynamic for climbing.** The opposite: for lead climbing, using static rope risks severe injury. The rope's elongation is what limits peak force on the climber's spine and harness attachment. A static rope gives no elongation on a fall → full impact force transmitted → potential death or critical injury.

**Braid does not prevent kinking.** Braided rope resists kinking better than 3-strand (no groove to open up), but it still kinks if coiled improperly or subjected to uncontrolled rotation. Especially UHMWPE braid — if allowed to form a loop that overruns and twists, the resulting kink dramatically reduces local breaking strength.

**Fid size is critical for double braid splicing.** The hollow fid (tool used to bury tucks in double braid) must be correctly sized for the rope diameter. Wrong size: either won't thread or too loose → tuck falls out. Manufacturers specify fid sizes by rope diameter range.
