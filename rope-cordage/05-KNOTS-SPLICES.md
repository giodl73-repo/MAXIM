# Knots and Splices: Ashley's Taxonomy, Strength Retention, Whipping

## The Big Picture

A knot is the simplest rope terminal but the most expensive in strength. A splice is the structurally superior alternative: it preserves 90–98% of rope breaking strength vs. a knot's 50–75%. Ashley's taxonomy provides the framework for understanding what every knot does and why it works.

```
KNOT CHOICE HIERARCHY (by strength retention)
===============================================

  100% MBS: Straight rope (no termination)
   95–98%:  Eye splice (3-tuck tuck splice or lock stitch)
   90–95%:  Back splice (terminates rope end — no loop)
   85–90%:  Long splice (joins two ropes — no diameter increase)
   80–85%:  Short splice (joins two ropes — diameter increases)
   75–80%:  Figure-8 knot loop (figure-8 on a bight)
   70–75%:  Bowline (most common loop knot)
   65–70%:  Double fisherman's knot (bend)
   60–65%:  Clove hitch, cleat hitch (hitches)
   50–60%:  Square knot, reef knot (should NOT be used for loads)

  WHY KNOTS REDUCE STRENGTH:
  ┌──────────────────────────────────────────────────────────────────┐
  │  At every bend in a knot, the rope is under combined:          │
  │  • Axial tension (primary load)                                  │
  │  • Bending stress (outer fibers more stressed)                 │
  │  • Transverse compression (from loaded rope parts pressing)    │
  │                                                                  │
  │  The failure point is at the first sharp bend after the knot  │
  │  (the "knot nip") where all three stresses peak simultaneously │
  │                                                                  │
  │  UHMWPE is especially bad in knots: low friction → slippage    │
  │  occurs before material failure → effective strength ~50–65%   │
  └──────────────────────────────────────────────────────────────────┘
```

## Ashley's Taxonomy

Clifford Ashley's "The Ashley Book of Knots" (1944) catalogs 3,900 knots and defines the structural taxonomy still in use:

```
KNOT TAXONOMY
=============

  STOPPERS — prevent rope from running through an opening
  ┌──────────────────────────────────────────────────────────────────┐
  │  Overhand knot:     Simplest. Jams when loaded. Use with care. │
  │  Figure-8 stopper: Larger than overhand; doesn't jam as tight. │
  │                    Standard stopper for climbing/rescue rope.    │
  │  Ashley stopper:   Triple bulge; best for fine rope/string.    │
  └──────────────────────────────────────────────────────────────────┘

  BENDS — join two ropes end-to-end
  ┌──────────────────────────────────────────────────────────────────┐
  │  Sheet bend:        Joins two ropes (same or different diameters)│
  │  Double sheet bend: More secure for very different diameters.    │
  │  Square knot:       Joins same-size ropes — NOT for heavy loads  │
  │                     (jams or capsizes if loaded incorrectly)     │
  │  Reef knot:         = Square knot; used for tying reefs in sails│
  │  Carrick bend:      Large diameter hawsers; unties after loading│
  │  Double fisherman's: Joining slippery ropes (UHMWPE, Dyneema); │
  │                      very secure; difficult to untie after load  │
  │  Water knot (tape): For flat webbing — loops and joins in tape   │
  └──────────────────────────────────────────────────────────────────┘

  LOOPS — create a fixed loop in the rope
  ┌──────────────────────────────────────────────────────────────────┐
  │  Bowline:           Standard fixed loop; easy to tie/untie;    │
  │                     70% efficiency; standard marine/rescue loop  │
  │  Figure-8 loop:     75–80% efficiency; preferred for climbing  │
  │                     (more secure than bowline when frozen/wet)   │
  │  Double figure-8:   Rethreaded through loop — for attaching to │
  │                     anchor (equalized two-leg anchor loop)       │
  │  Butterfly loop:    Fixed loop in middle of rope; load in any  │
  │                     direction; strand-efficient                  │
  │  Alpine butterfly:  High-load loop in bight; for rigging       │
  └──────────────────────────────────────────────────────────────────┘

  HITCHES — attach rope to an object (post, ring, other rope)
  ┌──────────────────────────────────────────────────────────────────┐
  │  Clove hitch:       Two half-hitches; quick; slips under shock │
  │                     load if not backed up                        │
  │  Round turn + 2 HH: More secure; distributes load over post    │
  │  Rolling hitch:     Attaches to another rope under load;       │
  │                     won't slide toward load                      │
  │  Prusik:            Friction hitch on another rope; climbs and │
  │                     locks in one direction                       │
  │  Munter hitch:      Belaying hitch on carabiner; variable      │
  │                     friction for dynamic belay                 │
  │  Timber hitch:      Attaching to logs/posts; self-tightening   │
  └──────────────────────────────────────────────────────────────────┘
```

## Knot Strength Retention Table

| Knot | Category | Approx. Strength % | Notes |
|------|----------|-------------------|-------|
| Eye splice (3-tuck) | Splice | 95% | Best terminal option |
| Figure-8 on a bight | Loop | 75–80% | Standard climbing loop |
| Bowline | Loop | 70–75% | Standard marine loop |
| Double fisherman's | Bend | 65–70% | Slippery synthetic rope |
| Sheet bend | Bend | 50–65% | Joining different diameter ropes |
| Clove hitch | Hitch | 60–65% | Quick attach; back up if critical |
| Square/reef knot | Bend | 45–55% | Symmetric loading only; low-load |
| Overhand loop | Loop | 55–65% | Simplest; jams under load |

Note: Values from laboratory testing of nylon and polyester ropes. Actual retention varies by rope diameter, fiber type, and knot tying technique. UHMWPE shows 10–15% lower retention due to low friction.

## Eye Splice (3-Strand, Tuck Method)

The eye splice is the definitive rope terminal — achieves 95% of rope MBS, creates a permanent loop, and adds minimal diameter at the splice.

```
3-STRAND EYE SPLICE PROCEDURE
================================

  SETUP:
  Rope end: strand 1 (top), strand 2 (middle), strand 3 (bottom)
  Form desired eye size; mark splice start point on rope body

  STEP 1: Unlay strands back approximately 4 rope diameters
  ┌──────────────────────────────────────────────────────────────────┐
  │  Rope: ═══════╮   2  ╮                                         │
  │                 3     1   ← unlay 3 strands                    │
  └──────────────────────────────────────────────────────────────────┘

  STEP 2: First tuck — strand 2 (center) goes under center strand
          of standing part, against the lay
  ┌──────────────────────────────────────────────────────────────────┐
  │  Standing part: ═══════[A][B][C]═══ (3 strands showing at tuck)│
  │  Tuck strand 2: under [B]                                        │
  └──────────────────────────────────────────────────────────────────┘

  STEP 3: Tuck strand 1 under [A] (next strand to left, against lay)
  STEP 4: Flip eye over; tuck strand 3 under [C] (remaining strand)

  REPEAT for 3 complete tucks (each strand tucked 3× total)
  (Marine standard: 3 tucks = ~90%; 4 tucks = ~95%; 5 tucks = standard)

  TAPER (optional): After 3 full tucks, thin each strand by removing
  half its yarns and do 1–2 additional tucks with tapered strands.
  Eliminates rope diameter step-change; smoother appearance.

  SEIZE or WHIP the splice end to prevent unraveling.
```

## Back Splice

The back splice terminates a rope end without a loop — prevents unraveling. It increases rope diameter at the splice (by ~25%) — won't run through a block or fairlead.

```
BACK SPLICE
============

  PURPOSE: Permanent end finish; ~90–95% of rope MBS
  WHEN NOT TO USE: If rope end needs to run through pulley or fairlead
                   (use whipping instead for running ends)

  PROCEDURE:
  1. Crown knot at end: Each strand tucks over next, under third
  2. Pull tight to form crown
  3. Tuck each strand 3× against the lay (same as eye splice tucks)
  4. Taper and whip
```

## Short Splice and Long Splice

Both join two rope ends. The difference is whether the resulting splice increases rope diameter.

```
SHORT SPLICE vs. LONG SPLICE
==============================

  SHORT SPLICE:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Unlay both rope ends 4–5 tucks' worth                         │
  │  Interleave ends → tuck each strand of rope A over/under       │
  │  strands of rope B (3 tucks each direction)                    │
  │  Result: Diameter increases ~25–30% at splice                  │
  │  Strength: 85–90% of rope MBS                                  │
  │  Use: Permanent field repair where increased diameter acceptable│
  └──────────────────────────────────────────────────────────────────┘

  LONG SPLICE:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Unlay both rope ends 10–15 rope diameters each                │
  │  Interleave ends; walk strands out along opposing strands        │
  │  Then tuck-splice at three evenly spaced points                │
  │  Result: NO diameter increase — looks like continuous rope       │
  │  Strength: 85–90% of rope MBS                                  │
  │  Use: Rope that must run through blocks, fairleads, or where   │
  │        uniform diameter required                               │
  └──────────────────────────────────────────────────────────────────┘
```

## Wire Rope Terminals

Wire rope cannot be spliced with the tuck method (strands are steel wires). Terminal options:

```
WIRE ROPE TERMINALS
====================

  SWAGED TERMINAL:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Wire rope end inserted into metal ferrule                       │
  │  Hydraulic press squeezes ferrule around wire                    │
  │  Result: Metal-to-wire friction + deformation lock             │
  │  Strength: 90–100% of wire rope MBS (depends on design)        │
  │  Use: Permanent, high-load, marine rigging, structural cable     │
  │  CANNOT be field-made without hydraulic press                  │
  └──────────────────────────────────────────────────────────────────┘

  FLEMISH EYE (mechanical splice):
  ┌──────────────────────────────────────────────────────────────────┐
  │  Wire rope unlaid; strands rewound around thimble in opposite    │
  │  direction; covered with swaged ferrule                          │
  │  Strength: 95–100% MBS                                           │
  │  Field-assemblable with proper tooling                         │
  └──────────────────────────────────────────────────────────────────┘

  WEDGE SOCKET:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Tapered steel socket; wire rope bent around wedge; pull         │
  │  seats wedge tighter                                             │
  │  Strength: ~75% MBS (not full efficiency — dead end loses load)  │
  │  Use: Temporary, field connections, elevator cables              │
  └──────────────────────────────────────────────────────────────────┘

  POURED SOCKET (spelter socket):
  ┌──────────────────────────────────────────────────────────────────┐
  │  Wire end opened (broom), inserted in socket, poured with zinc   │
  │  (or resin) to lock in place                                     │
  │  Strength: 100% MBS (highest efficiency)                       │
  │  Use: Permanent, critical-load applications                      │
  └──────────────────────────────────────────────────────────────────┘
```

## Whipping

Whipping prevents a rope end from fraying. It does not provide structural strength — it is a finishing technique.

```
WHIPPING TYPES
==============

  COMMON WHIPPING (simplest, no tools needed):
  ┌──────────────────────────────────────────────────────────────────┐
  │  1. Lay thread along rope end (pointing toward end)              │
  │  2. Wrap thread tightly toward rope end (20+ turns, 1 diameter)│
  │  3. Pass thread through loop left at start                       │
  │  4. Pull thread tight; trim                                      │
  │  Durability: Moderate (can slip if poorly seized)                │
  └──────────────────────────────────────────────────────────────────┘

  SAILMAKER'S WHIPPING (strongest, tool required):
  ┌──────────────────────────────────────────────────────────────────┐
  │  Needle-and-thread method                                        │
  │  Thread passes through strands at both ends of the whipping    │
  │  Locks whipping in place mechanically                          │
  │  Durability: Excellent — does not slip                         │
  │  Use: Permanent marine whipping on laid rope                   │
  └──────────────────────────────────────────────────────────────────┘

  WEST COUNTRY WHIPPING:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Half-hitch applied alternately on opposite sides                │
  │  Slower but very secure without needle                         │
  │  Result: zigzag pattern of thread on rope surface              │
  └──────────────────────────────────────────────────────────────────┘

  HEAT FUSION (synthetic rope):
  ┌──────────────────────────────────────────────────────────────────┐
  │  Hold cut end near (not in) flame → fibers fuse together       │
  │  Works only for nylon, polyester, polypropylene, UHMWPE        │
  │  NOT for aramid (chars, doesn't melt cleanly)                  │
  │  NOT for natural fiber rope (burns, doesn't fuse)              │
  │  Quick field method; adequate for moderate use                 │
  └──────────────────────────────────────────────────────────────────┘
```

## Decision Cheat Sheet

| Situation | Knot/Splice Choice |
|-----------|-------------------|
| Maximum strength permanent loop | Eye splice (3-tuck minimum, 5-tuck preferred) |
| Climbing top-rope or lead anchor | Figure-8 rethreaded or figure-8 on bight |
| Marine tow line attachment | Bowline (can untie after load) |
| Joining two slippery UHMWPE ropes | Double fisherman's (low friction requires more tucks) |
| Quick attach to post/bollard | Round turn + 2 half hitches (back up clove hitch) |
| Rope end that runs through block | Long splice (no diameter increase) |
| Rope end that doesn't run through blocks | Back splice or whipping |
| Wire rope permanent terminal | Swaged end or poured socket |
| Field wire rope terminal | Flemish eye with swaged collar |

## Common Confusion Points

**Bowline is not always the right knot.** The bowline is easy to tie and untie, which is why it's popular. But its 70% efficiency means 30% of rope MBS is lost every time you use it instead of a splice. For permanent fixed connections, always splice.

**Square knot ≠ load-bearing joining knot.** The square/reef knot is designed to be untied easily and is prone to capsizing (converting to a slip knot) when loaded asymmetrically. It is a parcel knot, not a structural bend. Use double fisherman's or carrick bend to join rope ends under load.

**UHMWPE requires doubled/buried splices for full efficiency.** Standard 3-tuck splice retains only ~80% for UHMWPE because the slippery fiber allows tucks to slide under load. Use lock-stitch buried splice (manufacturer-specified) for 95%+ efficiency. Standard tuck splicing technique is not sufficient.

**Whipping is not waterproofing.** Whipping prevents fraying and protects the cut end. It does not seal the rope against water infiltration, and it will not prevent swelling or water absorption in natural fiber rope ends. Use dipping or heat fusion (synthetics) if end sealing is the goal.
