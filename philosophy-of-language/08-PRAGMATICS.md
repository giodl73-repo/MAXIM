# Pragmatics: Grice and Implicature

## The Pragmatics Problem

Semantics deals with what sentences MEAN. Pragmatics deals with what speakers COMMUNICATE — which is typically much more than what sentences literally say. The gap between sentence meaning (semantic content) and speaker meaning (what's communicated in context) is the domain of pragmatics.

H.P. Grice (1913–1988) laid the foundations by identifying the inferential mechanisms by which hearers recover what speakers mean from what they say.

```
+-----------------------------------------------------------------------+
|                    THE SEMANTICS/PRAGMATICS DIVIDE                    |
|                                                                       |
|  WHAT IS SAID:                     WHAT IS COMMUNICATED:             |
|  (semantic content)                (speaker meaning)                  |
|                                                                       |
|  The LITERAL, TRUTH-CONDITIONAL    WHAT IS SAID +                    |
|  content of the sentence.          IMPLICATURES +                    |
|                                    PRESUPPOSITIONS +                  |
|                                    INFERENCES FROM CONTEXT            |
|                                                                       |
|  "Can you pass the salt?"          A REQUEST for salt.               |
|   literally: question about        Not really asking about ability.  |
|   ability                                                             |
|                                                                       |
|  "It's warm in here"               I want you to open a window.     |
|   literally: a statement           (hint / indirect request)         |
|   about temperature                                                   |
|                                                                       |
|  GRICE'S QUESTION:                                                   |
|  What REASONING allows hearers to recover speaker meaning             |
|  from literal meaning + context?                                      |
+-----------------------------------------------------------------------+
```

## Grice's Cooperative Principle

Paul Grice's 1967 William James Lectures (published 1975 as "Logic and Conversation") introduced the framework.

### The Cooperative Principle

```
  THE COOPERATIVE PRINCIPLE:
  "Make your conversational contribution such as is required,
  at the stage at which it occurs, by the accepted purpose
  or direction of the talk exchange in which you are engaged."

  In short: be cooperative. Conversation is a joint activity.
  Participants are presumed to be trying to communicate effectively.

  THIS PRESUMPTION is what enables implicature.
  If I say something that appears irrelevant, insufficient, or strange,
  the hearer SEARCHES for an interpretation that makes it cooperative.
  That search is what yields implicature.
```

### The Four Maxims

```
  Grice identified four categories of norms:

  MAXIM OF QUANTITY:
  1. Make your contribution as informative as is required.
  2. Do not make your contribution more informative than required.
  → Say enough; don't over-inform.

  MAXIM OF QUALITY:
  1. Do not say what you believe to be false.
  2. Do not say that for which you lack adequate evidence.
  → Be truthful.

  MAXIM OF RELATION:
  Be relevant.
  (Simplest maxim but hardest to define precisely.)

  MAXIM OF MANNER:
  1. Avoid obscurity of expression.
  2. Avoid ambiguity.
  3. Be brief (avoid unnecessary prolixity).
  4. Be orderly.
  → Be clear, efficient, well-organized.

  THESE ARE NOT MORAL NORMS:
  They are norms of rational cooperative behavior.
  Violating them has inferential consequences — it does not
  necessarily mean the speaker is being immoral.
```

## Conversational Implicature

The key concept: what is IMPLIED but not SAID.

### How Implicatures Are Generated

```
  MECHANISM:
  Speaker says p.
  Hearer infers: speaker intends to be cooperative.
  If p (literally) doesn't satisfy the CP, there must be MORE.
  The hearer finds q such that "p + q" is cooperative.
  q is the IMPLICATURE.

  EXAMPLE 1: Quantity implicature
  A: "Do you have a pen I could borrow?"
  B: "There's a shop around the corner."

  Literal: statement about a shop's location.
  Relevance violation? Unless...
  Implicature: "The shop around the corner sells pens."
  + possibly: "I don't have one to lend you."
  Hearer reconstructs the cooperative intent from the literal.

  EXAMPLE 2: Scalar implicature
  A: "Are you friends with all the department heads?"
  B: "I'm friends with SOME of them."

  Literal: B has at least some friendships.
  If B were friends with ALL of them, saying "all" would be
  more informative (Maxim of Quantity). B said "some."
  Scalar implicature: B is NOT friends with all of them.

  SCALAR TERMS (scales):
  <all, most, many, some, few>
  <always, often, sometimes>
  <none, some>
  <and, or>
  Asserting a weaker term implicates the stronger one is false.
  "I got some of the questions right" → not all.
```

### Types of Implicature

```
  CONVERSATIONAL IMPLICATURE:
  Arises from the Maxims + Cooperative Principle.
  CANCELABLE: can be explicitly denied without contradiction.
  "I got some — in fact, all — of the questions right."
  This cancels the scalar implicature without contradiction.
  The implicature was an inference, not part of what was said.

  CONVENTIONAL IMPLICATURE:
  Triggered by specific linguistic forms (not inferential).
  "He managed to open the door" → it was difficult to open.
  "even": "Even John passed" → John is an unlikely passer.
  "but": "P but Q" → concessive relationship between P and Q.
  NOT cancelable: "He managed to open the door,
                   though it wasn't difficult" → oddness.
  NOT at-issue content: "but" doesn't contribute to truth conditions.

+----------------------------------------------------------------+
| TYPE               | CANCELABLE | TRUTH-CONDITIONAL | EXAMPLE |
|--------------------|------------|-------------------|---------|
| Conversational     | Yes        | No                | Scalar  |
| (generalized)      |            |                   | (some)  |
|--------------------|------------|-------------------|---------|
| Conversational     | Yes        | No                | "There's|
| (particularized)   |            |                   | a shop" |
|--------------------|------------|-------------------|---------|
| Conventional       | No         | No                | "but",  |
|                    |            |                   | "even", |
|                    |            |                   | "managed|
+----------------------------------------------------------------+
```

## Flouting the Maxims: The Source of Figurative Language

The most important application: when speakers VISIBLY violate a maxim, they invite the hearer to search for a coherent interpretation that preserves the Cooperative Principle. This is where irony, metaphor, and hyperbole come from.

```
  VIOLATING QUALITY → METAPHOR, IRONY, HYPERBOLE

  METAPHOR:
  "Juliet is the sun."
  Literally false (violates Quality).
  Hearer: speaker is cooperative, so what are they trying to say?
  → Juliet is beautiful/warm/life-giving/central (metaphorical content).

  IRONY:
  The teacher arrives to a silent classroom and says:
  "What a noisy bunch you are!"
  Literally false (violates Quality).
  Context: classroom is quiet. Speaker obviously knows it.
  → Hearer understands the OPPOSITE is meant: irony.

  HYPERBOLE:
  "I've told you a million times!"
  Literally false (violates Quality, Quantity).
  → Implicature: I've told you many times. I'm exasperated.

  VIOLATING QUANTITY → UNDERSTATEMENT
  "It was not entirely unpleasant." (of a great event)
  Much less informative than "It was wonderful."
  → Understatement: extreme enthusiasm, or extreme politeness.

  VIOLATING MANNER → IMPLICATURE FROM FORM
  "He opened the door and came in."
  "He came in and opened the door."
  Same truth conditions (if doors can be opened after entering).
  But order implies: the order described IS the order of events.
  (Manner: be orderly → default = chronological order)

  VIOLATING RELATION → CHANGE OF SUBJECT
  "I see the government has raised taxes again."
  (Said when asked for reaction to embarrassing news)
  Irrelevance is deliberate: I refuse to discuss this.
```

## Relevance Theory (Sperber and Wilson)

Dan Sperber and Deirdre Wilson's "Relevance: Communication and Cognition" (1986) revised Grice's framework.

```
  GRICE'S PROBLEMS:
  Multiple maxims can conflict. How do you adjudicate?
  The maxim of Manner (be brief) conflicts with Quantity (be informative).
  The maxims seem ad hoc.

  RELEVANCE THEORY'S PROPOSAL:
  Reduce ALL maxims to a single cognitive principle:
  Relevance maximization.

  COGNITIVE PRINCIPLE OF RELEVANCE:
  Human cognition tends to be geared to the maximization of relevance.
  (We automatically attend to what seems relevant.)

  COMMUNICATIVE PRINCIPLE OF RELEVANCE:
  Every act of ostensive communication communicates a presumption of
  its own optimal relevance.

  OPTIMAL RELEVANCE:
  An utterance is optimally relevant when it achieves:
  - POSITIVE COGNITIVE EFFECTS: new information, strengthened beliefs,
    corrected beliefs, inferential conclusions
  In proportion to:
  - PROCESSING EFFORT: the cognitive cost of interpreting it

  More cognitive effects for less processing effort = more relevant.

  IMPLICATURES IN RELEVANCE THEORY:
  Not derived from maxims, but from searching for the interpretation
  that yields optimal relevance given the least processing effort.
  The hearer computes implicatures as part of finding the most relevant
  interpretation — not by checking maxims.

  EXPLICATURES:
  Relevance theory distinguishes EXPLICATURE (the developed semantic
  content of what is said, including disambiguation, reference
  assignment, enrichment) from IMPLICATURE (inferred propositions).

  "He's not going to die."
  Explicature: he is not going to die from this [illness/wound] [now]
  (reference assignment + pragmatic enrichment = explicature)
  Implicature: it's not as serious as it looks.
```

## Presupposition

Related to but distinct from implicature.

```
  PRESUPPOSITION: information that is taken for granted as background,
  not asserted.

  EXAMPLES:
  "John has stopped smoking."
  Asserts: John is not currently smoking.
  Presupposes: John WAS smoking.

  "The king of France is bald." (cf. Russell)
  Asserts: the king of France has the property bald.
  Presupposes: there IS a king of France.
  (Russell analyzed this as existential assertion;
   others argue it is a presupposition.)

  "Even Bill passed the exam."
  Asserts: Bill passed.
  Conventional implicature: Bill is unlikely to pass.
  Presupposes: others also passed.

  PRESUPPOSITION VS IMPLICATURE:
  Presuppositions SURVIVE NEGATION (survive the "negation test").
  "John has NOT stopped smoking." still presupposes John was smoking.
  Implicatures do NOT survive negation.
  "I got some of the answers right" implicates not all.
  "I didn't get some of the answers right" — no implicature.

  PRESUPPOSITION FAILURE:
  If the presupposition is false, the sentence may be odd/infelicitous.
  "The king of France is bald" — if no French king.
  Strawson: presupposition failure → sentence lacks truth value (gap).
  Russell: it's just false.
```

## Decision Cheat Sheet

| Concept | Definition |
|---------|-----------|
| Cooperative Principle | Conversations are presumed to be cooperative joint activities |
| Maxim of Quantity | Say enough, but not more than needed |
| Maxim of Quality | Say what you believe to be true and warranted |
| Maxim of Relation | Be relevant |
| Maxim of Manner | Be clear, brief, orderly |
| Conversational implicature | What is communicated beyond what is said, via CP reasoning; cancelable |
| Conventional implicature | What is communicated via specific linguistic forms (but, even); not cancelable |
| Scalar implicature | From "some" → not all; from "or" → not both |
| Flouting | Visibly violating a maxim to trigger implicature (metaphor, irony, hyperbole) |
| Presupposition | Background information taken for granted; survives negation |
| Relevance theory | Reduces maxims to one: optimal relevance (max effects, min effort) |

## Common Confusion Points

**Implicature ≠ entailment**: Entailment: "John is married" entails "someone is married" — true in all models where the entailing sentence is true. Implicature: "Some students passed" implicates "not all students passed" — cancellable inference, not a logical consequence.

**Cancellability tests for conversational implicature**: Conversational implicatures can be explicitly cancelled: "Some of the students passed — in fact, all of them did." If you can cancel it without contradiction, it's an implicature. If cancellation is contradictory ("he managed to open the door — and it was easy!"), it might be conventional implicature or presupposition.

**Flouting vs violating**: Flouting = DELIBERATELY and OPENLY violating a maxim to communicate something. The hearer recognizes the open violation and seeks an implicature. Violating = secretly failing to observe a maxim (deception, error). Flouting is where figurative language lives.

**The maxims are not MORAL norms**: Grice is not saying it is wrong to be non-cooperative. He is saying that the PRESUMPTION of cooperation is what enables pragmatic inference. You can be non-cooperative and simply communicate that. The maxims are rational norms of effective communication, not moral obligations.

**Relevance theory does not eliminate implicature**: Relevance theory provides a different theoretical account of how implicatures are generated — via relevance maximization rather than maxim violation. It still recognizes the same phenomena (scalar implicatures, metaphor, irony). The controversy is about the right theoretical framework.
