# Late Wittgenstein: Language Games and Meaning as Use

## The Turn

Ludwig Wittgenstein returned to philosophy at Cambridge in 1929 after nearly a decade away. Over the following decades, he dismantled his own Tractatus position and developed a completely different approach to language. The Philosophical Investigations (1953, posthumous) is the central document. It is written in a distinctive style: numbered remarks, thought experiments, dialogues, and questions — never a systematic treatise. The style is intentional: traditional philosophy's demand for systematic theory is itself what needs to be dissolved.

The core reversal: where the Tractatus sought the hidden logical structure beneath language, the Investigations insists there is no such hidden structure. Language is what it appears to be — a collection of overlapping practices, none of which is the "true" essence of language.

```
+-----------------------------------------------------------------------+
|                    TRACTATUS VS INVESTIGATIONS                        |
|                                                                       |
|  TRACTATUS (1921)                INVESTIGATIONS (1953)               |
|  -----------------               ------------------                  |
|  Language has ONE function:      Language has MANY functions:        |
|  picturing atomic facts          describing, questioning,            |
|                                  commanding, joking, greeting,       |
|                                  performing, praying...              |
|                                                                       |
|  Words get meaning by            Words get meaning by USE            |
|  NAMING objects                  in language games                   |
|                                                                       |
|  Meaning = the named object      Meaning = how the word is used      |
|                                                                       |
|  All propositions have           Different "regions" of language     |
|  the same LOGICAL FORM           have different grammars             |
|                                                                       |
|  Language mirrors                Language = diverse, embedded in     |
|  logical structure               forms of life (Lebensform)          |

|                                  language's surface grammar          |
+-----------------------------------------------------------------------+
```

## Language Games (Sprachspiele)

The central concept of the Investigations.

### What a Language Game Is

```
  Wittgenstein's first example (PI §2):
  A builder A calls out orders to assistant B: "block", "pillar",
  "slab", "beam". B brings the stone of the type called.
  This is a complete language game.
  Four words. No syntax. No nouns/verbs distinction.
  But it functions as communication.

  THIS IS A COMPLETE LANGUAGE.
  It is not a degraded or primitive form of "real" language.
  It works for what it needs to do.

  WHAT THIS SHOWS:
  We cannot say language "essentially" involves grammar,
  naming, or any particular structure.
  Every such claim is wrong because there are working
  language games that lack the feature.
```

### Wittgenstein's Extended List of Language Games

```
  PI §23: "Here we note the DIVERSITY of language games."

  Giving orders                  Reporting an event
  Speculating about an event     Making up a story
  Play-acting                    Singing catches
  Guessing riddles                Telling a joke
  Translating from one language  Asking, thanking, cursing
  into another                   Greeting, praying

  "...and it is interesting to compare the multiplicity
  of the tools in language and of the ways they are used,
  the multiplicity of kinds of word and sentence, with what
  logicians have said about the structure of language."

  The Tractatus (and Frege/Russell) assumed ONE structure.
  Wittgenstein II shows: there are indefinitely many.
```

## Meaning as Use

The most quoted thesis of the Investigations.

```
  PI §43:
  "For a large class of cases — though not for all —
  in which we employ the word 'meaning' it can be defined thus:
  the meaning of a word is its use in the language."

  NOT: the meaning = the object it names
  NOT: the meaning = a mental image accompanying it
  NOT: the meaning = a rule for its application

  MEANING = what competent speakers DO with words.
  = the role the word plays in our language games.

  THE TOOLBOX ANALOGY (PI §11):
  Think of the words in a language as tools in a toolbox.
  Hammer, pliers, saw, screwdriver, rule, glue-pot, nails, screws.
  "The functions of words are as diverse as the functions of these objects."

  When we see the word "word" we see a uniform appearance.
  When we see the tools, we see diverse functions.
  The GRAMMAR of use matters, not the visual form.
```

### What "Use" Means

```
  Use is not arbitrary:
  - It is embedded in PRACTICES (language games)
  - Practices are embedded in FORMS OF LIFE (Lebensform)
  - Forms of life are what human communities DO —
    shared activities, reactions, training, institutions

  "To imagine a language is to imagine a form of life." (PI §19)

  A word's use includes:
  - What follows what (inferential role)
  - What circumstances call for it
  - What it is a response to
  - What actions accompany it
  - How corrections are made

  USE ≠ meaning = frequency count.
  It is the RULE-GOVERNED practice, not just statistics.
```

## Family Resemblance

The Tractatus sought the ESSENCE of language — the one thing all languages share. Wittgenstein II attacks this.

```
  PI §66–67: Consider the word "game."
  Board games, card games, ball games, Olympic games...

  What do all games have in common?
  "Don't say: 'There must be something common, or they
  would not be called "games"' — but look and see
  whether there is anything common to all."

  BOARD GAMES: pieces, board, rules, winning
  PATIENCE (solitaire): no winner
  BALL GAMES: ball, but no board, no rules for all
  RING-A-RING-A-ROSES: fun, no winning, no ball

  No single feature is shared by all games.
  What we find is a network of overlapping similarities:

  A ── B ── C ── D ── E
  AB share: competitive structure
  BC share: pieces + board
  CD share: physical skill
  DE share: entertainment
  No feature runs from A to E.

  LIKE FAMILY RESEMBLANCE:
  Family members resemble each other — same nose, same eyes,
  same build — but no single feature is shared by all.
  Resemblance is overlapping and criss-crossing.

  CONSEQUENCE:
  The demand for a single defining ESSENCE of "game" (or "language",
  or "rule", or "meaning") is misguided.
  Concepts have fuzzy boundaries and family resemblance structure.
```

## The Private Language Argument

One of the most debated passages in 20th-century philosophy (PI §243–315).

### The Setup

```
  Can there be a language whose words refer to private sensations —
  known only to the speaker, not publicly verifiable?

  Example: You have a recurring sensation.
  You call it "S" and resolve to call it "S" every time it occurs.
  Is "S" a word? Is it meaningful?

  INITIAL APPEAL:
  My pain seems to be known directly by me.
  I can name it "pain" or use an arbitrary mark "E".
  This appears to be the foundation of sensation language.
```

### Wittgenstein's Argument

```
  PROBLEM: What makes your re-application of "S" CORRECT?

  You would check your memory: "yes, this feels the same."
  But this means: your memory seems to agree.
  And memory seeming to agree ≠ actually agreeing.

  PI §258:
  "Whatever is going to seem right to me is right.
  And that only means that here we can't talk about 'right'."

  IN PRIVATE LANGUAGE, THERE IS NO STANDARD OF CORRECTNESS:
  Without a public practice, there is no distinction between
  "correctly applying 'S'" and "it merely seems correct to me."
  The pseudo-rule "I call it S when I have this sensation"
  cannot serve as a real rule — real rules require an external
  criterion for following them correctly.

  REAL SENSATION LANGUAGE:
  Words like "pain" get their meaning from:
  - Public circumstances of use (pain behavior, injury, etc.)
  - Social training and correction
  - The role in a shared practice
  Not from a private ostensive definition.
```

### The Broader Significance

```
  THE ARGUMENT TARGETS:
  1. Cartesian picture of mind: the mind as a private theater,
     immediately self-transparent, whose contents are named
     in inner monologue.
  2. Mentalist theories of meaning: meaning = mental image
     or private mental state associated with a word.
  3. The Tractatus's own conception of names: names are logically
     attached to objects by private acts of meaning.

  IMPLICATIONS FOR PHILOSOPHY OF MIND:
  Mental states do not get their concepts from inner inspection.
  The criteria for "pain," "belief," "intention" are public criteria.
  This anticipates functionalism (mental states = functional roles)
  and anti-Cartesian positions in philosophy of mind.

  CS PARALLEL:
  A programming language where only the programmer who wrote a
  variable could ever check if it had the "right value" would be
  useless. Code works because it is publicly interpretable.
  Types, interfaces, contracts = the public criteria that make
  private computation shareable.
```

## Rule-Following

Closely connected to private language. The Kripke-Wittgenstein problem (Kripke's 1982 reconstruction).

```
  PI §185–202: A pupil learns the rule "+2":
  2, 4, 6, 8... 998, 1000, 1002...

  What happens when he gets to 1000 and writes: 1000, 1004, 1008...?
  You say: "That's wrong! The rule is '+2'."
  He says: "But I AM following the rule! It's '+4' after 1000!"

  How do you SHOW the rule uniquely requires the standard continuation?

  KRIPKE'S RADICAL PROBLEM:
  Any past application of a rule is compatible with infinitely many
  functions. How does past usage determine which function you meant?
  If you computed sums of small numbers, you might have been following
  "quus": x ⊕ y = x+y if x,y < 57; 5 otherwise.
  Nothing in your past behavior rules this out.

  WITTGENSTEIN'S RESPONSE:
  Rules are not objects in the mind that uniquely determine actions.
  Rule-following is a PRACTICE.
  "Following a rule" correctly means: acting as the community does.
  The normative dimension comes from the shared practice, not from
  a Platonic rule-entity or a private mental state.

  "Obeying a rule" is a practice, like obeying a law.
  A person alone in the world cannot be said to follow rules —
  only a person embedded in a practice can.
```

## Philosophical Therapy

The ultimate meta-point: what philosophy is for.

```
  PI §109:
  "Philosophy simply puts everything before us,
  and neither explains nor deduces anything.
  Since everything lies open to view there is nothing to explain."

  PI §126:
  "Philosophy simply presents everything and neither
  explains nor deduces anything."

  THE GOAL OF PHILOSOPHY (for Wittgenstein):
  Not to discover new truths.
  Not to construct theories.
  But to DISSOLVE confusions caused by language.

  THE SOURCE OF PHILOSOPHICAL CONFUSION:
  Language has one appearance (sentences look similar).
  But different uses work completely differently.
  "Time flies like an arrow" — grammatically like
  "Fruit flies like a banana" — same surface, utterly different.

  Philosophical problems arise when we take expressions out of
  their home language game and use them in other contexts.
  "What is time?" — the word "time" works in many language games,
  but when we demand its ESSENCE outside all context, it goes on
  philosophical holiday.

  PI §116:
  "When philosophers use a word... and try to grasp the
  essence of the thing, one must always ask oneself:
  is the word ever actually used in this way in the
  language which is its original home?"

  PHILOSOPHY'S RESULT:
  A fly-bottle has a fly trapped inside.
  The fly has found its way in through a small opening
  but cannot find its way out.
  Philosophy should show the fly the way out of the fly-bottle.
  (PI §309)
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| What is a language game? | Any of the diverse language practices embedded in forms of life — giving orders, praying, guessing |
| What is "meaning as use"? | The meaning of a word = the role it plays in language games, not an object it names |
| What is family resemblance? | Concepts held together by overlapping similarities with no single shared essence |
| What is the private language argument? | Without public criteria, there is no distinction between correctly applying a word and merely seeming to |
| What is the rule-following problem? | Past usage underdetermines any rule; rule-following is a social practice, not a mental state |
| What is philosophical therapy? | Dissolving confusions caused by language's misleading surface grammar |
| How does this differ from Tractatus? | Tractatus sought one hidden essence; Investigations accepts irreducible diversity of practices |

## Common Confusion Points

**"Meaning is use" is not a behaviorist claim**: Wittgenstein is NOT saying meaning = frequency of occurrence or that meaning is just behavior. He is saying the normative role of a word in a practice — what counts as correct use — is not a Platonic essence or a mental image. The practice is social, not physical.

**Language games are not games in the literal sense**: They are NOT recreational activities. The term is meant to highlight two things: (1) like games, they have rules; (2) unlike logical calculi, they are diverse, embedded in activity, and not reduced to a single underlying form.

**The private language argument does not deny private experience**: Wittgenstein is not saying pain doesn't exist or that inner experience is impossible. He is saying that a language picking out only the subjective feel, with no public criteria, would not be a language at all.

**Kripke's Wittgenstein is not Wittgenstein**: Kripke's 1982 book presents "the Kripkenstein problem" — a skeptical problem about rule-following that Kripke develops from Wittgenstein. Many scholars argue this is not Wittgenstein's own view, which is more deflationary (the "community view" in Kripke's version is attributed to Wittgenstein but is disputed).

**"Forms of life" is deliberately vague**: Wittgenstein uses "Lebensform" to point at the background context that makes language practices possible — human nature, institutions, shared reactions, training. He does not define it precisely. That vagueness is intentional.
