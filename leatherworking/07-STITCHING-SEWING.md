# Stitching: Saddle Stitch, Machine Sewing, Thread Selection

## The Big Picture

The stitch is the primary structural element of assembled leather goods. The saddle stitch — done by hand with two needles — is fundamentally stronger than a machine lock stitch because it has no single point of failure: if one thread breaks, the stitch holds everywhere else. Machine stitching is faster but mechanically inferior for high-stress applications.

```
SADDLE STITCH vs. MACHINE LOCK STITCH
========================================

SADDLE STITCH (hand):
  Two needles; one thread; each needle through same hole from opposite sides
  Thread path:
    ╔═══════╗
  ══╣ NEEDLE╠══ through hole from left
    ╚═══════╝
  ══╣ NEEDLE╠══ through same hole from right (passes through left thread loop)
    ╔═══════╗

  Result: each stitch interlocks with the next in BOTH THREAD PATHS

  FAILURE MODE: If thread breaks at one stitch:
    Adjacent stitches remain locked; the break stays isolated
    No unraveling; the seam holds everywhere else

MACHINE LOCK STITCH:
  One top thread + one bobbin thread; interlock through the leather
  Thread path: top thread loops around bobbin thread at stitch point

  FAILURE MODE: If thread breaks at one stitch:
    Chain reaction is possible as the interlocking point is gone
    Stitching CAN unravel from the break point
    Quality varies by thread tension and thread type

CONCLUSION:
  High-stress applications (holsters, saddlery, load-bearing straps):
    → Saddle stitch required; machine stitching not adequate
  Decorative seams; garment leather; non-structural:
    → Machine stitching acceptable and faster
```

## Tools for Hand Stitching

```
HAND STITCHING TOOL SET
========================

STITCHING CHISELS / PRICKING IRONS:
  Drive holes along a pre-grooved stitch line
  Spacing options: 3mm (tight), 4mm (standard), 5mm (heavy work)
  Count of tines: 1-tine (corners); 4-tine, 6-tine, 8-tine (production)
  Diamond vs. round: diamond cross-section tines → angled holes → diagonal
                     stitching pattern (most common; more professional look)

STITCHING GROOVER:
  Cuts a groove in the leather surface parallel to the edge
  Thread sits in groove (below surface) → protected from abrasion
  Width: adjustable with fence
  Depth: light pressure = shallow; heavier = deeper groove
  Use: always on items that will experience wear or rubbing

WING DIVIDERS:
  Two points; adjustable spacing
  Mark stitch-line distance from edge
  More accurate than ruler for consistent stitch placement

STITCHING IRONS / AWLS:
  Stitching awl: pointed blade; punch + widen holes; hold needle
  Harness awl: diamond-section blade; standard hand-stitching
  Hook awl: used in sewing-machine-like machines (not standard for hand sewing)

NEEDLES:
  Blunt or slightly rounded tip (leather already punched; no sharp needed)
  Harness needles: thick; blunt; standard for leather
  Size: match to thread (needle eye slightly larger than thread)
  Quantity: 2 required for saddle stitch (one per end of thread)
```

## Saddle Stitch — Step by Step

```
SADDLE STITCH TECHNIQUE
=========================

SETUP:
  Thread length: 2.5–3× the stitch length (for wrist-based measurement:
                 span your hand out; multiply the stitch length by 2.5 in
                 hand-widths = approx thread length needed)
  Thread one needle on each end of the same thread
  Wax the thread: run through beeswax block; pull through multiple times
  Waxing achieves:
    1. Reduce friction during stitching
    2. Makes thread slightly sticky (stitches stay where placed)
    3. Adds water resistance

STITCH SEQUENCE:
  1. Push needle #1 through first hole from left (e.g., leave 4" tail)
  2. Equalize thread: pull until equal length on both sides
  3. Push needle #1 through second hole from LEFT
     Push needle #2 through same second hole from RIGHT
     Critical: needle #2 must pass IN FRONT of the thread from needle #1
               (not behind; this determines stitch angle and lock)
  4. Pull both needles tight simultaneously (equal tension both sides)
  5. Advance to next hole; repeat

LOCKING THE END:
  At final hole: backstitch (go back through 2–3 previous holes)
  Or: at final hole, pass needle #1 through from right to left
      Needle #2 passes through same hole from left to right
      Tie off with square knot; pull knot into the leather; trim flush

TENSION:
  Even tension on both needles = even, flat stitch
  Too tight: puckers leather; cuts through thread over time
  Too loose: stitches not locked; visible gaps

STITCH LENGTH vs. LEATHER THICKNESS GUIDE:
  Leather weight    Recommended stitch spacing
  Light (2–4 oz)    3mm
  Medium (5–7 oz)   4mm (standard)
  Heavy (8+ oz)     5mm
```

## Machine Sewing for Leather

```
LEATHER MACHINE SEWING
========================

MACHINE REQUIREMENTS:
  Walking foot (triple feed): required for leather
    Standard presser foot: has no feed engagement on top surface
    → leather slides; stitches uneven; feed dogs grip bottom only
    Walking foot: adds top feed engagement synchronized with feed dogs
    → leather moves uniformly; even stitch spacing

  Needle type: leather (wedge-point) needle
    Wedge tip: cuts through leather (vs. standard round = tears)
    Sizes: #14–#18 (light to heavy leather)
    Change needle frequently: blunt needle = uneven holes + tearing

  Thread: upholstery thread (V-69, V-138) or heavy polyester
  Tension: adjust so knot buries at midpoint of leather thickness
           (not on surface; not visible from either side)

LEATHER-SPECIFIC SETTINGS:
  Stitch length: 2.5–4mm (longer than fabric; leather is thicker)
  Foot pressure: moderate (too much crushes grain; too little: skipped stitches)
  Speed: slower than fabric; controlled

COMMON MACHINES FOR LEATHER:
  Consumer: Singer Heavy Duty (225); limited to 3–4 oz combined
  Walking foot consumer: Juki TL-2010Q; Brother Quilt machines (upholstery)
  Professional: Juki TL-2000Qi; Reliable Barracuda (walking foot)
  Industrial: Consew 227; Cowboy CB4500 (heavy leather; saddle work)
              These handle 10+ oz total thickness; thread V-138+

NEEDLE PUNCH METHOD:
  For heavy leather on light machine: pre-punch all holes with stitching chisel
  Then run through machine with standard stitch length matching holes
  Machine pulls thread tight; mechanical combination = stronger than machine only
```

## Stitching Alignment and Quality Control

```
COMMON STITCHING ERRORS AND FIXES
=====================================

PROBLEM: Stitches angled inconsistently
  Cause: varying needle entry angle; not holding both threads consistently
  Fix: maintain consistent needle entry at same angle each stitch

PROBLEM: Uneven thread tension (one side tight, one loose)
  Cause: pulling one needle more than the other during tightening
  Fix: pull both needles simultaneously with equal force; use both thumbs

PROBLEM: Thread cutting through leather at holes over time
  Cause: thread too tight; hole size too small for thread
  Fix: looser tension; use next-larger pricking iron spacing;
       or use slightly larger punch diameter for the tines

PROBLEM: Skipped stitches (machine)
  Cause: dull needle; wrong needle type; incorrect timing
  Fix: replace needle (use leather/wedge needle); check machine timing
       if timing: professional service

PROBLEM: Puckering along stitch line
  Cause: thread too tight; leather not pre-punched cleanly
  Fix: reduce tension; ensure all holes are fully punched (not just pricked)

PROBLEM: Thread twisting / kinking (hand saddle stitch)
  Cause: thread rotating as it passes through hole
  Fix: rotate thread through hole; thread does not untwist naturally
       When thread feels kinked: let needles hang free and untwist
```

## Stitching Patterns and Decoration

```
DECORATIVE STITCHING OPTIONS
==============================

STANDARD STRAIGHT SADDLE STITCH:
  Two equal-length stitches; uniform appearance
  The baseline for all leather work

CROSS STITCH:
  Two passes of thread crossing at 45° in each hole
  Decorative; looks like × pattern
  Same strength as saddle stitch; more thread consumed

WHIP STITCH:
  Thread wraps around edge; not through the leather
  Used for: connecting two pieces edge-to-edge
            fringe attachment; decorative border

OVERCAST / BLANKET STITCH:
  Edge finishing; decorative; secures edge
  Not structural (no two-hole interlock)
  Common on: folded edges; decorative borders

LOCK STITCH (machine-only):
  Creates visible diagonal pattern on one face
  Standard machine stitch appearance

BRAIDED THREAD:
  Four-strand braid pulled through oversized holes
  High decorative value; not structural
  Used for: wallet spines; decorative accents
```

## Common Confusion Points

**"Hand stitching" does not mean slow or inferior.** In high-end goods (Hermès, Goyard), all stitching is done by hand. The saddle stitch is not a compromise — it is the superior stitch for durability. Machine stitching is a production efficiency choice, not a quality choice. For small batches or individual items, hand stitching is perfectly practical.

**The needle #2 passes IN FRONT of needle #1's thread.** This is the most critical mechanical detail of the saddle stitch and the most common beginner error. If needle #2 passes BEHIND the first thread, the stitch creates a different locking geometry — technically still a stitch, but with different tension characteristics. The standard saddle stitch specifies: #2 in front of #1's thread.

**Pre-punching all holes before stitching is not optional for thick leather.** Attempting to push a harness needle through un-punched 7–9 oz leather without pre-punching is both difficult and dangerous (bent needles, hand injuries from pushing). The stitching chisel or awl creates the hole; the needle just carries the thread through. These are separate operations.

**Thread tension from the machine is different from the needle tension you feel.** Machine stitching tension is set at the machine. Hand stitching tension is set by how hard you pull. A common beginner error is pulling the hand-stitch thread as tight as possible "for security." This is wrong — overtight stitching cuts through the leather fiber at the hole edges over time, especially with waxed linen thread.

**A walking foot is not optional for thick leather.** Standard presser feet only engage the feed dogs from below. Leather sticks to the bottom of the standard foot from above. The result: the bottom layer feeds but the top layer doesn't, producing skewed stitches and uneven pieces. A walking foot (also called dual-feed or triple-feed) solves this. Without it, you'll produce the consistent amateur result visible on mass-market goods.
