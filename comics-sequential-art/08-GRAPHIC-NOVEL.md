# The Graphic Novel as Form

## The Big Picture

```
+------------------------------------------------------------------+
|                   THE GRAPHIC NOVEL                               |
|                                                                  |
|  TERM ORIGIN: Will Eisner, "A Contract with God" (1978)          |
|  "Graphic novel" printed on cover as description                 |
|                                                                  |
|  WHAT IT IS (contested):                                         |
|  1. A comics work of book length (200+ pages)                    |
|  2. A complete narrative (not ongoing serial)                    |
|  3. Intended for bookstore (not direct comics market) distribution|
|  4. A marketing/prestige term: "comics for adults"               |
|                                                                  |
|  PROBLEM: The term is used for all of these + none of these      |
|  "Maus" = graphic novel; also = collected serial magazine strips  |
|  "Watchmen" = graphic novel; also = 12-issue superhero series     |
|  "Complete Peanuts Vol 1" = graphic novel? No one says this.     |
|                                                                  |
|  WORKING DEFINITION (pragmatic):                                 |
|  A single-volume comics narrative with literary ambition         |
+------------------------------------------------------------------+
```

## The Term's History

### Will Eisner's "A Contract with God" (1978)

```
EISNER'S CHOICE OF TERM
  Eisner: veteran cartoonist; The Spirit (1940-52)
  1978: wanted to make a serious artistic statement
  PROBLEM: walking into a bookstore saying "I have some comic books"
  would place the work in the children's section.

  "GRAPHIC NOVEL" ON THE COVER:
  Signal to booksellers: this is a "novel" (serious; adult; literary)
  that happens to be in graphic form.
  Not children's comics. For adults. Serious.

  WAS EISNER FIRST?
  Other claimants:
  Richard Kyle (1964): coined "graphic novel" in amateur press
  Jim Steranko: "Chandler: Red Tide" called a graphic novel earlier
  Jack Katz: "The First Kingdom" (1974-) described as graphic novel

  BUT: Eisner got the term into public usage via "A Contract with God."
  The subsequent cultural conversation credits him.
  The actual origin: doesn't matter much; what matters is: the term circulated.

A CONTRACT WITH GOD (Eisner, 1978, Baronet Press)
  Four interrelated stories; 1930s Bronx; Jewish immigrant experience
  Framing character: Frimme Hersh, who made a contract with God and was betrayed
  Art style: expressive; influenced by newspaper strip and EC comics
  Content: adult; death; sex; despair; no superheroes; no redemption guaranteed
  FORMAT: first page has printed "A Graphic Novel by Will Eisner"
```

## Eisner's Influence as Theorist

```
"COMICS AND SEQUENTIAL ART" (EISNER, 1985)
  First serious theoretical account of the medium
  Written for his class at the School of Visual Arts, NYC
  Predates McCloud by 8 years

  KEY CONTRIBUTIONS:
  Timing in comics: panels encode duration
  The panel as moment: frozen but implying motion
  Word-image interplay: systematic analysis
  Gesture and anatomy: the body as language
  "Sequential art" as the formal category (what McCloud built on)

"GRAPHIC STORYTELLING AND VISUAL NARRATIVE" (EISNER, 1996)
  The "how to" companion
  Narrative construction; storytelling craft; audience

EISNER AWARD
  Named for him; the comics industry's highest honor
  Annual; multiple categories
  Given at San Diego Comic-Con since 1988
```

## Landmark Graphic Novels by Type

### Original Graphic Novels (not serials)

```
WORKS CREATED AS A BOOK (not collected from serial)

A Contract with God (Eisner, 1978)
  First to use the term; adult Jewish life; four stories

The Complete Persepolis (Satrapi, 2000-2003 / collected 2007)
  Iranian childhood; political memoir; originally published in two volumes
  (Technically: French album format; L'Association; collected as OGN in English)

Blankets (Craig Thompson, 2003)
  600pp; first love; evangelical upbringing; faith
  One of the longest single-author graphic novels; drawn continuously

Logicomix (Doxiadis/Papadimitriou/Alecos/Annie, 2008)
  Bertrand Russell's life as comics; logic, set theory, foundations of math
  (With your TCS background: this one is worth reading specifically)

Building Stories (Chris Ware, 2012)
  Not a bound book: a box of pamphlets, books, newspapers, and fold-outs
  One apartment building; several residents; multiple time periods
  Each piece can be read in any order
  THE COMIC AS PHYSICAL OBJECT: the format is the content

Sabrina (Nick Drnaso, 2018)
  First graphic novel nominated for the Booker Prize
  Murder in the internet-conspiracy age; emotional flatness as style
```

## What "Graphic Novel" Claims (And What It Actually Means)

```
THE PRESTIGE FUNCTION
  "Graphic novel" in a bookstore = this is "serious"
  Same work called "comics" = children's; niche; not for you
  The term exists because there is no other term for
  "a comics work intended for adults without condescension"

  THIS IS A MARKETING DISTINCTION, NOT A FORMAL ONE:
  Watchmen (12 superhero comics) = "graphic novel" in bookstores
  Captain America #1 (same format) = "comic book" in comic shops
  Same physical form; different label; different audience signal

ART SPIEGELMAN'S AMBIVALENCE:
  He hated the term "graphic novel."
  Called it: "a bookshelf-friendly term for comics"
  Said: "I don't want to be the Thomas Edison of comics -- I want
         to be the Charles Darwin. Edison invented something;
         Darwin understood something."
  Meaning: he didn't want to be known for creating a genre (graphic novel)
           but for understanding the medium (comics as form).

GRANT MORRISON'S CRITIQUE:
  "Graphic novels" = "respectability politics"
  = apologizing for being comics
  = asking the literary establishment to approve
  = a symptom of the medium's lack of confidence in itself

THE RESOLUTION (practical):
  The term is imprecise but useful.
  It signals: book-length; bookstore; literary ambition.
  Using it: this is the category that most readers and booksellers
            understand as "comics that are also books."
  Criticizing it: valid; "sequential art" or just "comics" more precise.
```

## Comics in Literary Culture: The Canon Question

```
WHAT GETS CALLED "LITERARY"?
  Maus: Pulitzer; Time magazine; academic syllabi
  Watchmen: Time 100 novels; university literature courses
  Fun Home: National Book Critics Circle; Tony Award (musical)
  Persepolis: Angouleme Grand Prix; multiple film awards
  Jimmy Corrigan: Guardian "100 Best Books of the 21st Century"

  THE CRITERIA:
  Formal complexity (how it uses the medium)
  Subject matter (serious; personal; political; historical)
  Cultural conversation generated (reviews; awards; discourse)
  Durability (read years after publication)

  ABSENT FROM "LITERARY" CANON (typically):
  Most superhero work even when formally interesting
  Most genre manga (even when narratively rich)
  Much of what actual comics readers read

  THIS IS A CULTURAL POLITICS PROBLEM:
  What the literary establishment validates as "serious" reflects
  its own biases (toward memoir; toward European/American work;
  toward formally experimental work).
  The exclusion of mainstream manga from "literary" status
  is a cultural-institutional choice, not an aesthetic verdict.
```

## Production and Distribution

### How Graphic Novels Are Made and Published

```
FORMATS:
  Original Graphic Novel (OGN): created as book; no prior serialization
  Trade Paperback (TPB): collected serial issues; softcover
  Hardcover Collection (HC): collected serial; library quality
  Omnibus / Complete Edition: large collection; multiple volumes

MAJOR PUBLISHERS (2024):
  Scholastic Graphix: most successful US graphic novel publisher by volume
    (Raina Telgemeier's "Smile," "Sisters"; huge YA market)
  Pantheon: literary/memoir; adult (Maus, Persepolis)
  First Second: literary; wide range; children to adult
  Image: creator-owned; Saga, Walking Dead
  Drawn & Quarterly: Canadian; literary alternative
  Fantagraphics: alternative; art comics
  Viz: manga translation (dominant in that category)
  Yen Press, Kodansha Comics: manga translation competitors

THE YA GRAPHIC NOVEL EXPLOSION (2000s-2010s)
  Raina Telgemeier's "Smile" (2010): 10M+ copies
  Phoebe and Her Unicorn, Big Nate, Dog Man -- massive school market
  Graphic novel = primary format for children's publishing in 2020s
  This is unambiguously the largest market segment by volume
  More children read graphic novels than text novels in the US today
```

## Chris Ware: The Formal Peak

```
CHRIS WARE (b. 1967)
  The most formally ambitious American comics artist active
  "Jimmy Corrigan: The Smartest Kid on Earth" (2000)
    Collected from The Onion newspaper strip (1995-2000)
    Loneliness; four generations of a family; wordless sequences
    Extreme formal complexity; tiny panels; large negative space
    Architecture as character (the World's Columbian Exposition 1893)
    Won: Guardian First Book Award (2001); Eisner Award

  "ACME NOVELTY LIBRARY" (1993-ongoing)
    Each issue a different format; none the same
    Ware as book designer: every formal element is a narrative choice

  "BUILDING STORIES" (2012)
    Not a bound book: a box containing 14 different-format pieces
    A building in Chicago; multiple tenants; multiple timelines
    Reader constructs the narrative from the pieces in any order

  "RUSTY BROWN" (2019)
    An obsessive collector's life; suburban alienation; time layering
    Each chapter's cover is itself a wordless story

  WARE'S THESIS:
  Comics is a reading machine.
  Every formal choice (panel size, page layout, font) encodes meaning.
  The book as object is part of the work.
  Loneliness is structural; it is designed into the negative space.
```

## Common Confusion Points

**"Graphic novel = novel in graphic form"** -- A graphic novel is not a novel with
pictures. It is a comics work. The "novel" part is aspirational/marketing.
"Graphic" and "novel" are adjective/noun; together they form a new category
(graphic = visual; novel = book-length narrative). The confusion with "novel"
(literary fiction) is intentional and exploited.

**"Graphic novels are better than comics"** -- "Graphic novel" is a marketing
distinction. "Watchmen" was 12 comic books; as a collected volume it became a
"graphic novel." Same content, different label. The distinction encodes prestige
hierarchy, not a formal quality difference.

**"You have to read graphic novels in order"** -- Most yes; but "Building Stories"
(Ware) explicitly allows any order. The "reading order" of a graphic novel is
a design choice, not a physical necessity.

**"Comics for children vs. graphic novels for adults"** -- Scholastic Graphix's
YA graphic novels (Raina Telgemeier) are "for children" and are the largest
segment of the US graphic novel market. Plenty of graphic novels are for children;
the prestige use of the term doesn't capture the full market.
