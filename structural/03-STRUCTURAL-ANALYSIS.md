# 03 — Structural Analysis

## Matrix Methods, Influence Lines, FEM, Dynamic Analysis

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Determinacy and stability: degree of static indeterminacy (DSI), degree of kinematic indeterminacy
- Force method (method of consistent deformations): compatibility equations, redundants, flexibility
- Slope-deflection method: end moments, joint equilibrium, sidesway equations
- Moment distribution method (Hardy Cross): distribution factors, carry-over factors, iterative process
- Influence lines for beams: Müller-Breslau principle, ILs for shear and moment
- Influence lines for trusses: unit load traversal, maximum member forces under moving loads
- Moving loads: equivalent uniform load, absolute maximum moment, impact factor
- Matrix structural analysis: stiffness method formulation, element stiffness matrices
- Direct stiffness method: local vs global coordinates, transformation matrices
- Beam element: 4×4 stiffness matrix (Euler-Bernoulli), consistent mass matrix
- Truss element: 2×2 (local), 4×4 (global), assembly procedure
- Frame element: combined axial + bending, 6 DOF per element
- Boundary conditions in matrix method: partition, modified stiffness, support settlements
- Finite Element Method: displacement-based FEM, shape functions, Galerkin method
- 2D FEM elements: CST (constant strain triangle), Q4, isoparametric formulation
- Mesh refinement: h-refinement, p-refinement, convergence criteria
- Structural dynamics: equation of motion mẍ + cẋ + kx = F(t)
- Free vibration: natural frequency, mode shapes, orthogonality
- Forced vibration: frequency response function, resonance, damping ratio
- Multi-DOF dynamics: modal analysis, modal superposition, mode shapes
- Response spectrum analysis: SRSS and CQC modal combination
- Time history analysis: Newmark-β integration, seismic records
