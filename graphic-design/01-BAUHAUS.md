## What Made the Bauhaus Different

Before the Bauhaus, European art education split along two tracks:
- **Fine arts academies**: painting, sculpture, "pure" art
- **Technical schools**: applied craft, trades

The Bauhaus declared this split false and damaging. Gropius's founding manifesto
(1919) opened: "The ultimate aim of all creative activity is building!"

```
OLD MODEL                      BAUHAUS MODEL
---------                      -------------
Fine Art                       FINE ART
  |                               |
  v                               | (unified)
  [prestige, pure]                |
                                  v
Craft / Applied Art            CRAFT + INDUSTRY
  |
  v
  [low status, "mere" trade]

Designer =/= Craftsman         Designer = Craftsman = Thinker
```

## The School Structure

Every student at the Bauhaus took the same path:

```
+--------------------------------------------------+
|  VORKURS (Foundation Course)                     |
|  6 months                                        |
|  Taught by: Itten, then Moholy-Nagy, Albers      |
|  Content: Color theory, materials, basic forms   |
|  Purpose: Unlearn academic habits                |
+--------------------------------------------------+
                    |
                    v
+--------------------------------------------------+
|  WORKSHOPS (3 years)                             |
|                                                  |
|  Two masters per workshop:                       |
|  MASTER OF FORM (artist) + MASTER OF CRAFT       |
|                                                  |
|  Workshops:                                      |
|  - Typography / Printing                         |
|  - Weaving                                       |
|  - Metal / Furniture                             |
|  - Ceramics (Weimar only)                        |
|  - Wall Painting                                 |
|  - Theater                                       |
|  - Photography (added 1929)                      |
+--------------------------------------------------+
                    |
                    v
+--------------------------------------------------+
|  BUILDING WORKSHOP (advanced)                    |
|  Architecture + comprehensive design             |
|  Gropius's ultimate goal (arrived late)          |
+--------------------------------------------------+
```

The dual-master model was key: a practicing artist taught concept, a practicing
craftsman taught technique. Neither alone was sufficient.

## The Aesthetic Output

### Typography

Bayer's "universal" typeface (1925) was geometric, lowercase-first, functionalist.
The Bauhaus questioned the necessity of capital letters as aesthetically arbitrary.

```
BAUHAUS TYPOGRAPHY PRINCIPLES
------------------------------
- Geometric construction (circle, square, triangle as basis)
- Sans-serif favored (no historical ornament)
- Grid alignment
- Red + black two-color printing (Soviet-influenced)
- Text + image as unified design field

FAMOUS TYPEFACES FROM THIS TRADITION
- Futura (1927) -- Paul Renner, not Bauhaus but same moment
- Bauhaus (ITC Bauhaus) -- later revival
- Universal -- Bayer's lowercase experiment
```

## The Three Phases and Their Shift

```
WEIMAR (1919-1925)
  Tone: Expressive, arts-and-crafts inflected
  Itten's mysticism vs. Gropius's functionalism
  Klee, Kandinsky dominate painting workshops
  Output: Still handcraft-oriented

DESSAU (1925-1932)
  Tone: Rationalist, industrial, proto-corporate
  New building designed by Gropius himself
  Moholy-Nagy's machine aesthetic takes hold
  Bayer's typography; steel furniture
  Output: Industrial prototypes, mass-producible forms

BERLIN (1932-1933)
  Tone: Survival; politically beleaguered
  Private school (city funding cut)
  Closed July 1933 by vote of masters under Nazi pressure
  Output: 14 years of work; ideas scatter globally
```

## The Political Dimension

The Bauhaus was caught in Weimar-era politics throughout:

```
TENSION                           RESOLUTION
-------                           ----------
Art for art vs art for society    Never resolved; Gropius vs Meyer
Socialist collective vs market    School sold designs; industrialist patrons
German nationalism vs             Always suspect; Dessau city council
internationalism                  tried to cut funding repeatedly
"Degenerate art" accusation       Nazis targeted Kandinsky, Klee specifically
```

Hannes Meyer (director 1928-1930) was a committed Marxist who pushed the school
explicitly toward social housing design. Gropius's letters show tension; Mies
depoliticized the school again but too late.

## Why It Matters Today

The Bauhaus is present in every:
- Sans-serif UI font (Helvetica's spiritual ancestors were geometric sans)
- Grid-based web layout (the grid as universal structure)
- Design system thinking (systematic visual language)
- Industrial product designed for mass production
- Foundation course at any art/design school worldwide

The distinction between a graphic designer and "someone who makes things look
nice" is a Bauhaus-derived distinction: design is disciplined problem-solving
using visual means.

## Common Confusion Points

**"Bauhaus was a style"** -- It was first a pedagogy and thesis. The aesthetic
(geometric, sans-serif, primary colors) emerged from the ideas, not the other
way around. Many Bauhaus teachers (Klee, Kandinsky) made highly personal work.

**"Bauhaus = minimalism"** -- No. The school was interested in color, theater,
weaving, photography, ceramics. Mies van der Rohe's phase was minimal; the
overall school was not. "Less is more" was Mies's position.

**"Bauhaus = German"** -- Yes in origin, but the output was explicitly
internationalist. That was part of what made it suspect to nationalists.

**"The Foundation Course is just art class"** -- Itten's and Albers's foundation
courses were systematic investigations of perception, materials, and form. Albers's
"Interaction of Color" is still a reference text for anyone doing visual work.
