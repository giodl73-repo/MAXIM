## The Cultural Shift: What Changed After WWII

```
PRE-WAR AMERICAN DESIGN
  - Commercial art (illustrators, letterers)
  - Advertising = persuasion + decoration
  - Art director != designer (separate roles)
  - No systematic corporate identity programs

POST-WAR SHIFT
  US corporations going national / international
  Television creates visual mass-market
  Corporation as brand entity (not just products)
  Need: Systematic, repeatable, recognizable visual identity

THE OPENING
  Bauhaus methods + American corporate need
  = Professional "graphic design" as distinct field
  = Designer as strategic partner, not decorator
```

## Paul Rand (1914-1996)

The theorist-practitioner who defined American corporate modernism.

```
RAND'S THESIS
-------------
Design is not decoration.
Design is problem-solving with visual tools.
A logo does not "mean" the company -- it becomes the company
through consistent application over time.

"IBM" [striped rebus]  1956
"ABC" [circle with letters]  1962
"UPS" [brown shield]  1961
"Westinghouse" [W circuit motif]  1960
"NeXT" [cube at 28 degrees]  1986 (for Steve Jobs)
"Yale" [Rand's last institutional commission]
```

### The IBM Logo

```
ORIGINAL IBM WORDMARK (1924)
  Lettering, fine serif -- looks like a bank

RAND'S PROGRESSION
  1956: Slab-serif City typeface -- bolder, more mechanical
  1960: City Bold variant
  1972: 13-stripe version
  1981: 8-stripe version (still in use)

WHY STRIPES?
  Speed, dynamism (IBM was computers = computation = speed)
  Stripes add visual interest without changing letterforms
  Creates uniqueness while remaining legible
  At small sizes stripes disappear -- still works

THIS IS RAND'S METHOD:
  1. Start with a clear concept
  2. Express it through the simplest possible form
  3. System should work at all sizes and in all contexts
```

### The NeXT Logo

```
1986: Steve Jobs, fired from Apple, hires Rand to design NeXT identity.
Rand, 72 years old, presents ONE solution with a 100-page booklet
explaining the concept. Take it or leave it.

The cube logo:
- Cube = computer hardware, 3D = cutting edge
- 28 degree rotation = dynamic, not static
- Lowercase "e" in NeXT = distinctive, readable
- Black + white + yellow = sophisticated

Jobs: "He's one of the few people I've met who has real design taste."
Rand to any client who requested alternatives: "No."

This is Rand's philosophy made literal:
The designer understands the problem and solves it.
The client pays for that expertise.
Multiple options = admitting you haven't solved the problem.
```

### Rand's Books

```
"Thoughts on Design" (1947)       -- The manifesto; still current
"Paul Rand: A Designer's Art"     -- Comprehensive retrospective
"Design, Form, and Chaos" (1993)  -- Late essays; client critique
"From Lascaux to Brooklyn" (1996) -- History + aesthetics
```

## Saul Bass (1920-1996)

Two distinct contributions: film title sequences and consumer brand identity.

### Film Titles

Before Bass, film credits were text lists before the film started -- functionally
a pause before the content. Bass made them the first act of the film.

```
FILM TITLES AS DESIGN BREAKTHROUGH
------------------------------------

"The Man with the Golden Arm" (1955)
  - Animated lines forming a arm reaching down
  - Jazz score (Elmer Bernstein)
  - Establishes tone before first frame of film
  - Client: Otto Preminger

"Vertigo" (1958)   [Alfred Hitchcock]
  Spiraling concentric circles over female eye
  Mathematically generated (John Whitney's computer)
  Psychological dread established in 2 minutes

"Psycho" (1960)   [Alfred Hitchcock]
  Horizontal lines fragment and reassemble text
  Nervous energy, fragmentation = film's theme
  Text as subject, not label

"Anatomy of a Murder" (1959)   [Otto Preminger]
  Jointed paper-cut figure that comes apart
  Dismemberment as visual metaphor, pre-film

BASS METHOD:
  1. Read the script / understand the film's core theme
  2. Find ONE visual metaphor for that theme
  3. Animate that metaphor in 90 seconds
  4. Music and motion in sync
```

### Consumer Brand Identity

```
BASS LOGO WORK
--------------
AT&T "globe" logo        1984    Still in use (updated 2005, 2016)
United Airlines          1974    "Tulip" motif
Quaker Oats             1969    Simplified Quaker figure
Kleenex                 1955
Minolta                 1978
Continental Airlines    1968

BASS'S APPROACH vs RAND'S
  Rand: Design is intellectual problem-solving, present one solution
  Bass: Design is intuitive + systematic; research-driven; multiple options OK

Both: The logo is a mark that must work in all contexts,
      at all sizes, in black and white and color.
```

## Milton Glaser (1929-2020)

The humanist counter to Swiss rationalism. Illustration, decoration, and
emotion are valid design tools.

```
GLASER'S POSITION IN THE FIELD
--------------------------------
Swiss Style / Rand:   Remove personality; design = problem-solving
Glaser:               Personality and culture ARE the content
                      Illustration is not anti-modern
                      Design can be warm, playful, referential

Glaser founded Push Pin Studios (1954) with Seymour Chwast:
  - Counter to the Helvetica dominance
  - Used eclectic historical typefaces
  - Victorian, Art Nouveau, Deco references
  - Illustration as primary medium
  - Influenced every decorative trend 1960s-1980s
```

### The Bob Dylan Poster (1967)

```
CONTEXT: Dylan's "Greatest Hits" album insert
METHOD:
  - Silhouette of Dylan's profile (black)
  - Hair exploding in psychedelic colored waves
  - Simple hand-drawn look; Art Nouveau influence
  - Text in bold geometric sans

WHY IT MATTERED:
  One of the most reproduced posters in history
  Showed that modernism and illustration could coexist
  Psychedelic movement had a design language
```

### "I Love NY" (1977)

```
ORIGINAL CONTEXT
  New York City fiscal crisis, 1970s
  NYC begging tourists to come back
  State tourism commission hires Wells Rich Greene agency
  Glaser joins project; creates symbol on back of envelope
  in a taxi

THE MARK:
  "I" + red heart + "NY"
  Courier typeface (typewriter = democratic, not pretentious)

  Original: intended as temporary campaign
  Actual: permanent icon; most copied design in history

  The mark works because:
  - Scale-independent (heart shape reads at any size)
  - Substitution template (every city copies it)
  - Emotionally legible cross-culturally
  - Zero ambiguity: verb + place = direct statement
```

## The Corporate Identity Program

By the 1960s, the corporate identity program emerged as a discipline:

```
WHAT IS A CORPORATE IDENTITY PROGRAM?
--------------------------------------

NOT: A logo
IS: A complete system governing how an organization looks
    in every visual context

COMPONENTS
+------------------------------------------+
|  CORPORATE IDENTITY PROGRAM              |
|                                          |
|  Primary Mark (logo)                     |
|    - Horizontal / vertical / icon forms  |
|    - Color / black / reverse versions    |
|                                          |
|  Color System                            |
|    - Primary palette (2-3 colors)        |
|    - Secondary palette                   |
|    - Pantone specs for print             |
|    - RGB / HEX specs for screen          |
|                                          |
|  Typography System                       |
|    - Primary typeface(s)                 |
|    - Hierarchy (headline, body, caption) |
|    - Size/weight rules                   |
|                                          |
|  Grid System                             |
|    - Page/screen layout rules            |
|    - Image treatment (photography style) |
|                                          |
|  Usage Rules                             |
|    - Clear space minimums                |
|    - Forbidden uses (rotation, effects)  |
|    - Application examples (business card |
|      letterhead, signage, advertising)   |
+------------------------------------------+
      |
      v
  STANDARDS MANUAL (printed book)
  Before digital: 3-ring binders; books of rules
  Now: digital brand guidelines / design tokens
```

## Common Confusion Points

**"Rand = Swiss Style"** -- Rand was influenced by European modernism but was
distinctly American: more intuitive, more willing to use wit and playfulness,
less doctrinaire about the grid. His logos have personality where Swiss work
aims for neutrality.

**"Logo = brand identity"** -- A logo is one element. A brand identity is the
complete system. Rand understood this; his standards manuals show how the IBM
logo was intended to be applied across every touchpoint, not just stamped on
letterhead.

**"Corporate design = boring"** -- The best corporate design of this era
(IBM, CBS, NeXT) shows that constraint amplifies creativity. Rand's achievement
was solving real communication problems with extraordinary elegance.

**"Bass did movie posters"** -- Bass did titles and posters. His title sequences
are moving design -- time-based graphic design before anyone called it that.

## Decision Cheat Sheet

| Situation | Approach | Why |
|-----------|----------|-----|
| Open-ended brief, confident designer owns answer | Rand's single-solution presentation | Present one definitive option with conviction; don't dilute with alternatives |
