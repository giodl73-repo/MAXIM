# East Asian Religions — Daoism, Confucianism, Shinto

## The Big Picture

East Asian religious life does not follow the Western model of exclusive membership
in one tradition. The same person typically participates in multiple traditions
simultaneously — this is not inconsistency but reflects a different underlying
structure. Traditions are treated as complementary registers addressing different
domains of life.

```
+----------------------------------------------------------------------+
|              EAST ASIAN RELIGIOUS LANDSCAPE                          |
|                                                                      |
|  CHINA: THREE TEACHINGS (Sanjiao) + Chinese Folk Religion            |
|                                                                      |
|  +-------------------+-------------------+-------------------+       |
|  |   CONFUCIANISM    |      DAOISM       |    BUDDHISM       |       |
|  +-------------------+-------------------+-------------------+       |
|  | Social order.     | Natural flow.     | Suffering and     |       |
|  | Proper relations. | Cosmic process.   | liberation.       |       |
|  | Self-cultivation. | Non-action.       | Karma, rebirth.   |       |
|  | Government by     | Inner alchemy.    | Bodhisattva ideal.|       |
|  | virtue.           | Immortality.      | Pure Land.        |       |
|  +-------------------+-------------------+-------------------+       |
|            ↓ absorbed elements from all three ↓                      |
|  +----------------------------------------------------------+        |
|  |              CHINESE FOLK RELIGION                       |        |
|  | No founder, no scripture, no institution — yet arguably  |        |
|  | the largest religion by adherents (400–500M+)            |        |
|  | Ancestor veneration + local deities (Mazu, Guan Yu,      |        |
|  | Tudigong) + divination + qi cosmology + domestic ritual  |        |
|  | Structural position: the ground-level practice into which|        |
|  | the Three Teachings feed; most Chinese participate in    |        |
|  | folk religion regardless of which "teaching" they follow |        |
|  +----------------------------------------------------------+        |
|                                                                      |
|  A traditional Chinese scholar might practice Confucian ethics       |
|  in government service, Daoist meditation for inner cultivation,     |
|  Buddhist rituals for funerary rites, and folk religious practices   |
|  (ancestral offerings, Kitchen God, temple visits) year-round.       |
|  All four layers operate simultaneously without contradiction.       |
|                                                                      |
|  JAPAN:                                                              |
|  +-------------------+-------------------+                           |
|  |      SHINTO       |     BUDDHISM      |                           |
|  +-------------------+-------------------+                           |
|  | Kami, purity,     | Death, afterlife, |                           |
|  | festivals, shrines.| merit, funerals. |                           |
|  | This world.       | Transcendence.    |                           |
|  +-------------------+-------------------+                           |
|  (+ Confucian ethical framework in both)                             |
+----------------------------------------------------------------------+
```

Geography: Confucianism and Daoism originate in China; they spread through the
Sinosphere (Korea, Japan, Vietnam). Shinto is specifically Japanese. Buddhism
arrived in China via the Silk Road (~1st c CE), then spread to Korea, Japan, Vietnam.

## Daoism

### Philosophical Daoism — The Daodejing

Laozi ("Old Master") — probably a legendary figure, though tradition dates him
to the 6th-5th century BCE. The Daodejing (Tao Te Ching, "Classic of the Way
and Its Power") is ~5,000 Chinese characters, 81 short chapters. One of the most
translated books in the world.

The fundamental move: the Dao is what cannot be said.

```
  Chapter 1, opening lines:
  "The Dao that can be spoken is not the eternal Dao.
   The name that can be named is not the eternal name.
   The nameless is the beginning of heaven and earth.
   The named is the mother of ten thousand things."

  This is not mystical evasion — it is a logical claim:
  the Dao is the ground of all distinctions, including
  the distinction between what can and cannot be said.
  It precedes language and therefore cannot be captured by it.
  Comparable to: pointing at the class of all classes
  (Russell's paradox territory), or naming the frame within
  which all naming occurs.
```

Core concepts:

```
+-----------------------------------------------------------+
|                    KEY DAOIST CONCEPTS                    |
|                                                           |
|  DAO (Way):       The ultimate ground of all things.      |
|                   Not a god, not a creator, not a person. |
|                   The process/pattern underlying all.     |
|                   Operates through natural spontaneity.   |
|                                                           |
|  DE (Power/Virtue): The Dao as manifested in particular   |
|                   things. Each thing has its own de —     |
|                   its natural virtue, power, potency.     |
|                   A skilled craftsman has de. A good      |
|                   ruler has de. A tree has de.            |
|                                                           |
|  WU WEI (Non-action): Not passivity but acting in         |
|                   accord with the natural flow of things. |
|                   Yielding, non-forcing, effortless       |
|                   effectiveness. Water wearing away rock. |
|                                                           |
|  PU (Uncarved block): Original simplicity before          |
|                   culture, convention, artifice. The      |
|                   infant. The unhewn log. What we are     |
|                   before society shapes us.               |
|                                                           |
|  ZI RAN (Naturalness/Spontaneity): Following one's        |
|                   own nature. What happens of itself.     |
+-----------------------------------------------------------+
```

Political theory of the Daodejing: the ideal ruler governs so lightly that the
people don't know they are being governed. Anti-Confucian: Confucius says govern
through virtue and example; Laozi says the more you try to govern, the more
disorder you create. Small states, low population, minimal intervention.

Yin-yang: not a Daoist invention but a Chinese cosmological framework absorbed
into Daoism. Complementary opposites that generate each other and cannot exist
without each other. Not good/evil (as in Zoroastrian dualism) but complementary.
Yin: dark, yielding, receptive, feminine, moon, winter. Yang: bright, active,
generative, masculine, sun, summer. The taijitu (yin-yang symbol) shows each
containing a seed of the other.

### Zhuangzi — Perspectivism and Spontaneity

Zhuangzi (c.369-286 BCE), the Inner Chapters (Zhuangzi, Chapters 1-7) are considered
his authentic work. More playful, more radical, more philosophical than the Daodejing.

```
  The Cook and the Ox (Chapter 3):
  Prince Hui's cook carves an ox. His cleaver finds the natural spaces —
  he never cuts against resistance. "I glide through such great joints or
  cavities as there may be, according to the natural constitution of the
  animal." Nineteen years; the blade still like new.

  This is wu wei as skill: not forcing but finding the grain of things.
  Applicable to any domain where forcing produces damage and skill produces
  flow. Compare: expert programmers don't fight their language, they work
  with its nature. (Not forced — but the analogy holds.)

  The Butterfly Dream (Chapter 2):
  "Now I do not know whether I was then a man dreaming I was a butterfly,
  or whether I am now a butterfly dreaming I am a man."

  Not skepticism — a substantive claim about perspectives. Each perspective
  is real within itself. There is no view from nowhere that settles
  which is "really" you.
```

Zhuangzi's perspectivism: what appears as "this" from one perspective appears
as "that" from another. The sage doesn't try to adjudicate from a fixed viewpoint
but rides with the transformation of things. This is epistemological, not nihilist —
he is not saying there is no truth, but that truth is perspective-relative.

Zhuangzi on death: "The Great Clod burdens me with form, labors me with life,
eases me with old age, and rests me in death." Death is not to be lamented —
it is one transformation among many. His wife died; he was found singing. When
questioned: she was a transformation of qi; mourning would be to not understand
the nature of things.

### Religious Daoism

Philosophical Daoism (Laozi/Zhuangzi) and Religious Daoism are distinct but related.
Religious Daoism (Daojiao) developed from the 2nd century CE onward:

```
+-----------------------------------------------------------+
|              RELIGIOUS DAOIST TRADITIONS                  |
|                                                           |
|  Tianshi Dao (Celestial Masters, c.142 CE):               |
|  Zhang Daoling received revelation from deified Laozi.    |
|  Communal organization, confession of sins, healing.      |
|  First institutionalized Daoist church.                   |
|                                                           |
|  Waidan (External Alchemy, c.3rd c CE onward):            |
|  Elixirs of immortality from mercury, cinnabar, gold.     |
|  Many practitioners died from mercury poisoning.          |
|  Concept: body mirrors cosmos; correct the body's         |
|  processes to achieve longevity or immortality.           |
|                                                           |
|  Neidan (Inner Alchemy, c.8th c CE onward):               |
|  Alchemical process is internal — the body itself         |
|  is the crucible. Qi (vital force), jing (essence),       |
|  shen (spirit) cultivated through meditation and          |
|  breathing. The goal: merge with the Dao.                 |
|                                                           |
|  Quanzhen School (Complete Perfection, c.12th c):         |
|  Monasticism, celibacy, vegetarianism. Major school       |
|  today in China. White Cloud Monastery, Beijing.          |
|                                                           |
|  Zhengyi (Orthodox One):                                  |
|  Married priests, hereditary succession, ritual           |
|  specialists. Dominant in Taiwan and diaspora.            |
+-----------------------------------------------------------+
```

Qi cosmology: all of reality is constituted by qi (vital energy/force). Dense qi
= matter; refined qi = spirit. The universe itself is a process of qi transformation.
Five phases (wu xing): wood, fire, earth, metal, water — not elements but phases
of qi transformation cycling through seasons, bodily organs, musical notes,
directions. Medicine, martial arts, feng shui, and political theory all draw on
five-phase analysis.

## Confucianism

### Kong Qiu (Confucius, 551-479 BCE)

Confucius lived in the Spring and Autumn period — a time of political fragmentation
and interstate warfare in Zhou dynasty China. His project: restore the moral and
ritual order of the Zhou golden age. His method: self-cultivation of virtue, proper
observance of ritual, government by moral example rather than coercion.

The Analects (Lunyu) — short exchanges between Confucius and his disciples. Not a
systematic treatise; must be read as a record of a teaching relationship.

### Core Concepts

```
+------------------------------------------------------------------+
|                    CONFUCIAN CONCEPTS                            |
|                                                                  |
|  REN (benevolence/humaneness/authoritative conduct):             |
|  The supreme virtue. The full realization of humanity.           |
|  Ren is achieved through li, expressed in all relationships.     |
|  "Do not do to others what you would not want for yourself."     |
|  (Negative formulation of the Golden Rule — Analects 12.2)     |
|                                                                  |
|  LI (ritual propriety/rites):                                    |
|  The totality of proper behavior in social roles.                |
|  Not empty formalism — li is the form through which              |
|  ren is expressed and cultivated. Ritual makes virtue            |
|  habitual. Compare: etiquette as a system that externalizes      |
|  and trains inner virtue.                                        |
|                                                                  |
|  YI (righteousness):                                             |
|  Doing what is right because it is right, not for              |
|  profit. The superior person (junzi) acts from yi.             |
|                                                                  |
|  ZHI (wisdom/moral knowledge):                                   |
|  Knowing what is right in concrete situations.                   |
|                                                                  |
|  XIN (trustworthiness):                                          |
|  Integrity; matching words and actions.                          |
|                                                                  |
|  ZHENGMING (rectification of names):                             |
|  When names don't match realities, social order decays.          |
|  "Let the ruler be a ruler, the minister a minister,           |
|   the father a father, the son a son." (Analects 12.11)        |
|  This is not conservative stasis — it is a call to make        |
|  rulers actually rule well, ministers actually serve.            |
+------------------------------------------------------------------+
```

### Five Relationships (Wu Lun)

The social fabric is constituted by five dyadic relationships, each with specific
mutual obligations:

| Relationship | Superior role obligations | Subordinate role obligations |
|-------------|--------------------------|------------------------------|
| Ruler / Minister | Benevolence, good governance | Loyalty, remonstrance when ruler errs |
| Father / Son | Nurture, guidance, example | Filial piety (xiao), care in old age |
| Husband / Wife | Leadership, protection | Respect, household management |
| Elder / Younger brother | Guidance, mentorship | Respect, deference |
| Friend / Friend | (The only equal relationship) | Mutual faithfulness, remonstrance |

Note: four of five are hierarchical, but all involve mutual obligations — the superior
is not simply owed deference but must fulfill specific duties. The ideal is not
domination but reciprocal virtue. Women are notably absent from the public relational
structure — a major target of feminist critique.

### Mencius vs. Xunzi — Human Nature Debate

The most consequential debate in classical Confucianism:

- **Mencius (Mengzi, ~372-289 BCE)**: Human nature is originally good. Everyone has
  four "moral sprouts" (siduan): the sprout of compassion → ren; shame → yi;
  deference → li; moral approval/disapproval → zhi. Moral failure is failure to
  cultivate innate goodness, like neglecting to water a plant. The role of education
  is nurturing what is already there.

- **Xunzi (~313-238 BCE)**: Human nature is originally bad (evil). People are greedy,
  partial, pleasure-seeking. Left unchecked, conflict results. Ritual and education
  reshape human nature into virtue — nature is the raw material, ritual is the
  transformation. The role of education is transformation, not just cultivation.

Both agree that people can become virtuous through education and self-cultivation.
They differ on whether virtue is actualization of potential or correction of defect.

### Neo-Confucianism (Song-Ming, 960-1644 CE)

Neo-Confucianism emerged as a response to Buddhist and Daoist metaphysical sophistication.
Confucianism needed a robust metaphysical framework.

**Zhu Xi (1130-1200)**: The dominant synthesis. Two ultimate principles:
- **Li** (principle, pattern): the normative structure of all things. Every thing
  has its li; human nature IS the li given by Heaven. The Great Ultimate (Taiji)
  is the supreme li.
- **Qi** (material force): the stuff of which all things are made. Li and qi are
  always together — you can't have pure li without qi or pure qi without li.
  Moral evil arises from impurity or obstruction in one's qi, obscuring the li.
- Method: "investigation of things" (gewu) — attentive study of things in the world,
  including texts, to understand their principles. Extension of knowledge.

**Wang Yangming (1472-1529)**: The major alternative — the "mind" school (xinxue).
The mind is itself the principle (xin ji li). We don't need to investigate external
things; knowledge is already in the mind as innate moral knowledge (liangzhi).
"Unity of knowledge and action": to truly know the good is already to do it.

### Confucianism in the Modern World

Neo-Confucianism shaped Japanese, Korean, and Vietnamese intellectual culture deeply.
Modern relevance:
- East Asian developmental states (South Korea, Japan, Taiwan, Singapore) — Confucian
  emphasis on education, meritocracy, hierarchy, collective over individual as
  structural advantage in industrialization
- Corporate culture: Samsung, Toyota, Huawei — hierarchy, loyalty, group harmony
  have Confucian DNA even when not explicitly named
- Singapore's meritocracy as a deliberate neo-Confucian social design
- Mainland China: Xi Jinping's "China Dream" explicitly rehabilitates Confucianism
  (after Mao's Cultural Revolution attacked it) as ideological foundation

## Shinto

### What Shinto Is

Shinto ("the way of the kami") is Japan's indigenous religious tradition.
No founder, no single founding text, no systematic theology. Emerged as a
distinct tradition partly in response to Buddhism's arrival in Japan (6th century CE).

```
+----------------------------------------------------------+
|                    SHINTO BASICS                         |
|                                                          |
|  KAMI:                                                   |
|  Not "gods" exactly — spirits/forces that inspire awe.   |
|  Present in remarkable natural phenomena: mountains,     |
|  storms, rivers, exceptional trees, the sun.             |
|  Also: spirits of the imperial ancestors, notable dead.  |
|  Motoori Norinaga (18th c): "anything whatsoever which   |
|  was outside the ordinary, which possessed superior      |
|  power or which was awe-inspiring was called kami."      |
|                                                          |
|  SHRINES (Jinja):                                        |
|  Residence of kami. Torii gate marks the threshold       |
|  between ordinary and sacred space. Haiden (oratory),    |
|  honden (inner sanctuary, usually not accessible).       |
|  ~80,000 shrines in Japan.                               |
|                                                          |
|  PURITY (Kiyome):                                        |
|  Ritual purity (ke) vs. ritual pollution (kegare).       |
|  Harae: purification rites. Misogi: water purification.  |
|  Death, blood, illness = kegare. Not moral sin —         |
|  a physical/ritual state requiring purification.         |
|                                                          |
|  MATSURI (Festivals):                                    |
|  Ritual events maintaining the relationship between      |
|  community and its kami. Seasonal, agricultural.         |
+----------------------------------------------------------+
```

### The Kojiki and Nihon Shoki — Foundation Texts

Compiled in 712 CE (Kojiki) and 720 CE (Nihon Shoki) by imperial command.
Content: creation myth (Izanagi and Izanami create the Japanese islands), the
Sun Goddess Amaterasu (ancestor of the imperial line), and the descent of the
imperial family from the kami. These texts are political as much as religious —
they legitimate the imperial institution by grounding it in divine ancestry.

The creation narrative differs from Abrahamic or Hindu cosmogonies in being
generative rather than creatio ex nihilo: the kami emerge from primordial chaos
and generate the world through their interactions.

### Buddhism-Shinto Syncretism (Shinbutsu-shugo)

When Buddhism arrived in Japan (officially 552 CE), it did not replace Shinto —
the two traditions merged in a syncretic complex lasting until 1868.

```
  Honji suijaku ("original ground, manifest traces"):
  Japanese kami ARE Japanese manifestations of Buddhist deities.
  Amaterasu (Sun Goddess) = Dainichi Nyorai (Vairocana Buddha).
  Local kami = local manifestations of bodhisattvas.

  Result: Buddhist temples and Shinto shrines in the same complex.
  Monks maintained shrines; shrine priests recited sutras.
  Most Japanese funerals were Buddhist; weddings were Shinto.
  This is still largely true today.
```

### State Shinto (1868-1945)

The Meiji Restoration (1868): the new government needed ideological legitimation
and a basis for national unity. Shinto was separated from Buddhism (shinbutsu bunri),
the honji suijaku doctrine abolished. Shrines were organized into a national hierarchy.
Ise Shrine (imperial ancestor kami) at top. Shrine visits, participation in Shinto
ritual, and veneration of the Emperor were declared civic duties, not "religion"
— thereby claiming they could be required of all citizens regardless of their
personal religion.

The State Shinto apparatus:
- Yasukuni Shrine (Tokyo): enshrines the spirits of Japan's war dead. Created 1869.
  The enshrinement of Class A war criminals in 1978 made it politically explosive.
  Prime ministerial visits trigger diplomatic crises with China and Korea.
- Emperor as living kami: State Shinto doctrine made the Emperor a divine being.
  The 1946 declaration (Ningen-sengen) had Hirohito formally deny his divinity.

Post-WWII: Supreme Commander Allied Powers (SCAP) directive (1945) ordered
disestablishment of State Shinto. Shrines became private religious corporations.
Yasukuni remains a nationalist flashpoint.

### Contemporary Shinto

Post-WWII Shinto is largely non-doctrinal and non-exclusive. Most Japanese participate
in shrine visits (hatsumode — New Year visit) and festivals without considering
themselves "Shinto" in an exclusive religious sense. Shinto is deeply embedded
in Japanese seasonal life and tied to local community identity.

New religions (shinshukyo): emerged from 19th century onward. Tenrikyo (1838),
Konko-kyo, Soka Gakkai (Buddhist-derived but organizationally distinct). Soka Gakkai
is Japan's largest religious organization (~8 million families) with a affiliated
political party (Komeito).

## Chinese Folk Religion

Often excluded from standard taxonomies because it has no institutional form,
no single founder, no unified doctrine. Yet it may be the world's largest
religion by adherents (400-500M+) if counted separately.

```
+------------------------------------------------------------+
|                  CHINESE FOLK RELIGION                     |
|                                                            |
|  Ancestor veneration:                                      |
|  The dead remain morally present. Ancestral tablets        |
|  in home altars. Qingming festival (grave sweeping).       |
|  Joss paper and ghost money burned for the dead.           |
|  Hungry Ghost Festival: dead who lack living               |
|  descendants need offerings.                               |
|                                                            |
|  Local deities:                                            |
|  Tudigong (Earth God) — every neighborhood has one.        |
|  Mazu (sea goddess) — 400M+ devotees, Taiwan, Fujian.      |
|  Guan Yu (war god, also god of business integrity).        |
|  City gods, trade gods, medicine gods.                     |
|  Promoted and demoted based on efficacy (divine            |
|  bureaucracy mirrors human bureaucracy).                   |
|                                                            |
|  Divination:                                               |
|  I Ching (Yi Jing) — hexagram casting.                     |
|  Kau cim (fortune sticks at temples).                      |
|  Feng shui: spatial arrangement for qi optimization.       |
|                                                            |
|  Integration with medicine:                                |
|  Five-phase theory underlies Traditional Chinese           |
|  Medicine. Body = cosmological microcosm. Qi blockage      |
|  = illness. Acupuncture, herbalism as ritual-medical.      |
+------------------------------------------------------------+
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| What is the Dao? | The ineffable ground and process of all reality. Not a god, not a person. The way things naturally flow. |
| What is wu wei? | Non-forcing action — acting in accord with the natural grain of things. Effortless effectiveness. Not passivity. |
| What does Confucius actually teach? | Virtue through proper relationships and ritual. Government by moral example. Self-cultivation of ren through li. |
| What is ren? | Benevolence/humaneness — the supreme Confucian virtue. Realized through proper relations and ritual. The fullest expression of what it means to be human. |
| What is a kami? | An awe-inspiring spirit/force. Can be in natural phenomena, places, or persons. Not "gods" in the Western sense — more like sacred presences. |
| Why does Shinto matter politically? | State Shinto (1868-1945) used kami veneration and emperor worship as nationalist ideology. Yasukuni Shrine remains a live diplomatic issue. |
| What is the "three teachings" idea? | Confucianism, Daoism, Buddhism as complementary rather than competing. Each addresses different aspects of life. The same person can practice all three. |

## Common Confusion Points

**Philosophical vs. Religious Daoism**: Laozi and Zhuangzi (philosophical Daoism)
are foundational texts. Religious Daoism (Daojiao) is an institutionalized practice
tradition from the 2nd century CE onward with priests, rituals, liturgy, and
community organization. The two are related but distinct. Scholars distinguish
them; practitioners may not.

**Confucianism is not a "religion" in the Western sense**: It has no creation myth,
no soteriology (path to salvation), no afterlife doctrine, and no concept of God.
Its focus is entirely on this-worldly social ethics and self-cultivation. Some
scholars classify it as a "civil religion" or "ethical philosophy." Yet it functions
socially as religion does: shaping identity, providing moral framework, legitimating
authority.

**Neo-Confucian li vs. Daoist li**: Both words are transliterated "li" in romanization
but are different characters. Daoist li (理): pattern, principle — same as Neo-Confucian.
But also: Confucian li (禮): ritual propriety, rites. Context determines which li
is meant.

**Shinto "polytheism"**: Shinto has eight million kami (the phrase yaoyorozu no kami —
"eight million kami" is an expression for "innumerable"). This is not polytheism
in the Greek sense of a structured pantheon with specific personalities and domains.
Kami are more diffuse — presences in things, not persons above things.

**"Confucianism and women"**: Classical Confucian social structure is organized around
five relationships, four of which are male-male. Women's roles are defined within
the household. Contemporary feminist scholarship critiques this structure heavily.
The tradition is not static — Korean Neo-Confucian debates about women's learning
were lively. But the baseline structure is patriarchal.
