# Abrahamic Religions — Judaism, Christianity, Islam

## The Big Picture

Three traditions — Judaism, Christianity, Islam — share a genealogy, define themselves
partly against each other, and have profoundly shaped global history. The shared lineage
is real but contested: each tradition reads the common heritage differently and claims
to be the correct development or supersession of what came before.

```
+-------------------------------------------------------------------+
|              ABRAHAMIC GENEALOGY                                  |
|                                                                   |
|  Abraham (patriarch, ~2000 BCE — legendary, not historical)       |
|      |                                                            |
|      +-- Isaac --> Jacob (Israel) --> 12 Tribes --> ISRAELITE     |
|      |                                              RELIGION      |
|      +-- Ishmael (Islamic tradition: ancestor of Arabs)           |
|                                                                   |
|  Israelite religion (Mosaic law, Temple cult, prophets)           |
|      |                                                            |
|      +--> Second Temple Judaism (after Babylonian exile,          |
|      |    ~6th-2nd c BCE) — Torah canonized, synagogue emerges    |
|      |         |                                                  |
|      |         +--> JUDAISM (rabbinic revolution after 70 CE      |
|      |         |    Temple destruction — Talmudic tradition)      |
|      |         |                                                  |
|      |         +--> Jesus of Nazareth (~4 BCE - ~30 CE)           |
|      |               --> CHRISTIANITY (Paul's universalization,   |
|      |                   Gentile mission, Greek conceptual        |
|      |                   framework, Roman empire)                 |
|      |                                                            |
|      +--> ISLAM (Muhammad, 610 CE Arabia) — claims to restore     |
|           original Abrahamic monotheism, correcting corruptions   |
|           in Jewish and Christian transmission                    |
+-------------------------------------------------------------------+
```

Not a simple linear progression: each tradition defines itself partly against the others.
Christianity claims to fulfill and transcend Judaism; Islam claims to correct both
Christianity (Trinitarian distortion of monotheism) and Judaism (neglected prophecy).
Judaism does not accept the Christian fulfillment claim or the Islamic supersession claim.

### The Rabbinic Revolution

The Temple's destruction by Rome in 70 CE was a civilizational crisis: the entire
sacrificial cult (the ritual core of Israelite religion) was gone. The rabbinic
movement's response was the most consequential religious transformation in Jewish history:

- Temple → synagogue (any minyan of 10 can constitute a congregation)
- Sacrifice → prayer, study, and good deeds as equivalent acts of worship
- Priests (Cohen lineage) → rabbis (scholars, not a hereditary class)
- Centralized cult → portable religion, viable in diaspora anywhere

This is why Judaism survived the loss of its homeland: the rabbinic revolution created
a diaspora-portable religion. The Talmud became the "portable homeland."

### Halakha — Jewish Law

Halakha (from halach, "to walk" — the way to walk) is the body of Jewish law governing
daily life. Source hierarchy:

```
  d'oraita (biblical law) -- highest authority
       |
       v
  d'rabbanan (rabbinic enactments)
       |
       v
  minhag (custom) -- lowest but often practically decisive
```

The 613 mitzvot (commandments) tradition: 248 positive (you shall), 365 negative
(you shall not) — Maimonides' count, widely accepted. Halakhic reasoning process:
apply Torah principle → through Talmudic precedent → through responsa literature
→ to new circumstances (technology, bioethics, etc.). The question "may I use
electricity on Shabbat?" was decided by responsa literature in the 20th century.

### Denominations (from most traditional to most liberal)

| Denomination | Scripture authority | Halakha | Key features |
|-------------|-------------------|---------|--------------|
| Haredi (Ultra-Orthodox) | Torah divine, literal | Fully binding, all 613 | Segregated communities, Yiddish, rejection of secular modernity |
| Modern Orthodox | Torah divine | Fully binding | Engagement with secular world; "Torah u'Madda" |
| Conservative (Masorti) | Torah divine but historically conditioned | Binding but can evolve by rabbinic consensus | Committee on Jewish Law and Standards |
| Reform (Liberal) | Torah human document guided by God | Ethical core binding; ritual optional/voluntary | Informed choice, autonomy, social justice emphasis |
| Reconstructionist | Torah as civilization, not revelation | Folkways with democratic community decision | Founded by Kaplan; Judaism as evolving civilization |

### Key Theological Concepts

- **Covenant (berit)**: The central category. Sinai covenant obligates Israel to Torah in
  exchange for God's protection and presence. Not just rules — a relationship.
- **Tikkun olam** (repair of the world): originally a kabbalistic concept about cosmic
  restoration, now commonly used in liberal Judaism for social justice work.
  The semantic shift is significant — the original meaning is theurgic (human acts
  affect divine realm), not merely ethical.
- **Messianic expectation**: Traditional: a human king-figure from Davidic lineage who
  restores Israel, rebuilds Temple, ushers in universal peace. Not a divine figure.
  Reform: the Messianic age (human progress toward justice), not a personal Messiah.
- **Kabbalah**: Medieval mystical tradition (major texts: Zohar, 13th c Spain, attributed
  to Simeon bar Yochai but likely written by Moses de León). Sefirot (10 divine
  emanations from Ein Sof — the infinite). Lurianic Kabbalah (Isaac Luria, 16th c):
  tzimtzum (divine contraction to make space for creation), shevirat hakelim (shattering
  of vessels), tikkun (repair). Deeply influenced Hasidism (18th c revivalist movement).

### Historical Periodization

| Period | Key events | Religious significance |
|--------|-----------|----------------------|
| Biblical/pre-exilic (~1000-586 BCE) | United monarchy, Temple built (Solomon), division, Assyrian conquest | Prophetic tradition, Torah composition begins |
| Babylonian exile (586-538 BCE) | Temple destroyed, exile | Monotheism consolidates; synagogue emerges |
| Second Temple (538 BCE - 70 CE) | Return, Second Temple built, Hellenistic period, Roman period, Jesus, Paul | Pharisees, Sadducees, Essenes, LXX (Greek translation), apocalypticism |
| Rabbinic (70-600 CE) | Temple destroyed again, Mishnah, both Talmuds | Rabbinic Judaism becomes normative Judaism |
| Medieval (600-1800) | Diaspora, Islamic Golden Age, Crusades, expulsions (Spain 1492) | Maimonides (philosophy + law), Kabbalah, Shulchan Aruch |
| Modern (1800-present) | Haskalah (Jewish Enlightenment), Zionism, Holocaust, State of Israel | Denominational split, existential crisis, sovereignty |

## Christianity

### The Historical Jesus Problem

Scholarship distinguishes the Jesus of history from the Christ of faith.
The "quest for the historical Jesus" has three phases:

1. First Quest (19th c, Schweitzer): Liberal Protestant attempt to recover
   the "real" Jesus behind the dogma. Schweitzer's "The Quest of the Historical Jesus"
   (1906) concluded that Jesus was an apocalyptic preacher expecting imminent
   end of world — not the liberal moral teacher the questers hoped for.

2. Second Quest (post-Bultmann, 1950s): Bultmann had argued the historical Jesus
   is largely inaccessible and irrelevant — what matters is the kerygma (proclamation).
   His students (Käsemann, Bornkamm) pushed back.

3. Third Quest (1980s-present): Crossan, Meier, Vermes, Sanders. Places Jesus
   firmly in 1st century Jewish context (eschatological prophet, social reformer).
   Criteria: multiple attestation, criterion of dissimilarity, historical plausibility.

Historically defensible claims: Jesus was baptized by John, gathered disciples,
was crucified under Pontius Pilate (~30 CE). The resurrection is a theological
claim, not a historical one (cannot be assessed by historical method).

### Christology — The Central Dispute

How is Jesus related to God? This question drove three centuries of dispute.

```
+---------------------------------------------------------------------+
|               CHRISTOLOGICAL SPECTRUM                               |
|                                                                     |
|  HIGH CHRISTOLOGY                    LOW CHRISTOLOGY                |
|  (Jesus = divine)                    (Jesus = human)                |
|                                                                     |
|  Logos Christology (John 1:1):       Adoptionism:                   |
|  "In the beginning was the Word."    Jesus was a human adopted      |
|  Jesus = pre-existent divine Logos   as Son of God at baptism.      |
|                                      (Theodotus, Ebionites)         |
|                           |                                         |
|                    NICAEA 325 CE                                    |
|                    Debate: homoousios vs. homoiousios               |
|                    "same substance" vs. "similar substance"         |
|                    (one iota of difference — literally)             |
|                    RESULT: homoousios wins. Arianism condemned.     |
|                    Jesus is "of one substance with the Father."     |
|                                                                     |
|  Nestorianism (condemned 431 CE, Ephesus):                          |
|  Jesus has two natures SO distinct they're nearly two persons.      |
|  Mary = Christotokos (Christ-bearer), not Theotokos (God-bearer).   |
|                                                                     |
|  Monophysitism (condemned 451 CE, Chalcedon):                       |
|  Jesus has ONE nature (divine absorbed human).                      |
|  (Still held by Coptic, Ethiopian, Armenian churches)               |
|                                                                     |
|  Chalcedonian formula (451 CE, still normative for                  |
|  Catholic, Orthodox, Protestant):                                   |
|  "Two natures — divine and human — without confusion,               |
|   change, division, or separation. One person."                     |
+---------------------------------------------------------------------+
```

The Trinity: Father, Son, Holy Spirit — one God in three persons (hypostases).
Not three gods (tritheism). Not one God in three modes (modalism/Sabellianism,
condemned). The Nicene Creed (325 CE, revised Constantinople 381 CE) is still
recited by ~2 billion Christians.

### Canon Formation

The New Testament did not fall from the sky as a bound volume. Key process:

- Paul's letters (~50s CE) — earliest written documents in NT
- Mark (~70 CE) — earliest Gospel; Matthew, Luke (~80-90 CE) use Mark as source
- John (~90-100 CE) — most theologically developed, different tradition
- Muratorian fragment (~170-200 CE) — earliest known canon list
- Eusebius (4th c) — distinguishes homologoumena (accepted), antilegomena (disputed),
  notha (rejected). Disputed: James, Jude, 2 Peter, 2-3 John, Revelation
- Athanasius's Easter Letter (367 CE) — first list matching modern NT exactly
- Councils of Hippo (393) and Carthage (397) — Western ratification

The "Gnostic Gospels" (Nag Hammadi library, discovered 1945): Gospel of Thomas,
Gospel of Philip, etc. Not suppressed conspiracies — they circulated, were argued
against, and excluded by developing ecclesial consensus.

Key Reformation theological claims:
- Sola scriptura: Scripture alone as authority (vs. Scripture + Tradition)
- Sola fide: Justification by faith alone (vs. faith + works + sacraments)
- Sola gratia: Grace alone (vs. merit)
- Luther's catalyst: indulgences (certificates of remission of purgatorial punishment)
  sold by Johann Tetzel. 95 Theses (1517) were primarily about indulgence theology.

### Atonement Theories

Why did Jesus die, and how does it save? No single theory is normative for all Christians.

| Theory | Proponent | Mechanism | Critique |
|--------|----------|-----------|----------|
| Satisfaction (Anselmian) | Anselm (1097) | Jesus satisfies the infinite honor-debt owed to God by human sin | Medieval feudal metaphysics |
| Penal Substitution | Calvin, Protestant orthodoxy | Jesus takes the legal penalty (punishment) humans deserve | Makes God's mercy subject to God's justice; violence at the heart |
| Moral Influence | Peter Abelard | Jesus' death demonstrates God's love, inspiring human moral transformation | Says nothing about why death was necessary |
| Christus Victor | Aulén (1931) | Jesus defeats the powers of sin, death, and devil | Mythological framing; what powers exactly? |
| Participatory | Recent scholarship | Union with Christ in his death and resurrection | Less forensic, more ontological |

## Islam

### The Foundational Sources

```
+------------------------------------------------------------------+
|              ISLAMIC SOURCE HIERARCHY                            |
|                                                                  |
|  QURAN                                                           |
|  The direct speech of God (Allah) transmitted through            |
|  Gabriel to Muhammad (610-632 CE). NOT Muhammad's words —        |
|  God's own words. Uncreated (in mainstream Sunni view).          |
|  Arranged in 114 suras (chapters) by length, not chronology.     |
|  Arabic is the sacred language — translation is interpretation.  |
|  I'jaz (inimitability): cannot be matched in eloquence/style.    |
|       |                                                          |
|  SUNNA (the Prophet's practice)                                  |
|  Transmitted through hadith (individual reports of Muhammad's    |
|  words, actions, tacit approvals). Six canonical Sunni           |
|  collections (kutub al-sitta), most authoritative:               |
|  Sahih al-Bukhari and Sahih Muslim.                              |
|       |                                                          |
|  HADITH SCIENCE (Ilm al-hadith)                                  |
|  Sophisticated authentication methodology:                       |
|  Isnad (chain of transmission): A heard from B heard from C...   |
|  Matn (content): the actual report.                              |
|  Grades: sahih (sound), hasan (good), da'if (weak), mawdu' (forged)|
|       |                                                          |
|  SHARIA (Islamic law)                                            |
|  Derived from: Quran + Sunna + ijma (scholarly consensus)        |
|              + qiyas (analogical reasoning)                      |
|  + ijtihad (independent legal reasoning — contested scope)       |
+------------------------------------------------------------------+
```

### Five Pillars (Arkan al-Islam)

| Pillar | Arabic | Practice | Significance |
|--------|--------|----------|-------------|
| Declaration of faith | Shahada | "There is no god but God; Muhammad is His messenger" | Entry into Islam; repeated in prayer |
| Prayer | Salat | Five daily prayers (fajr, dhuhr, asr, maghrib, isha); facing Mecca | Direct relationship with God; no priestly mediation |
| Charity | Zakat | 2.5% of accumulated wealth annually to defined categories of recipients | Social solidarity; purification of wealth |
| Fasting | Sawm | Ramadan — complete fast dawn to sunset for entire month | Spiritual discipline, solidarity with poor |
| Pilgrimage | Hajj | Once in lifetime if able; Mecca, Dhul Hijja month; circumambulate Kaaba | Ummah (worldwide Muslim community) made visible |

### The Sunni-Shia Split

The split is political in origin (succession) but has developed into theological
and legal divergence over 14 centuries.

```
+-------------------------------------------------------------+
|              SUNNI / SHIA SPLIT                             |
|                                                             |
|  Muhammad dies 632 CE without designating a clear           |
|  successor.                                                 |
|         |                                                   |
|         +-- SUNNI position: Abu Bakr (father-in-law)        |
|         |   elected as first Caliph through community       |
|         |   consultation. Legitimate succession: Abu Bakr   |
|         |   → Umar → Uthman → Ali (fourth Caliph).          |
|         |                                                   |
|         +-- SHIA position: Ali ibn Abi Talib (cousin        |
|             and son-in-law) was the designated rightful     |

|             usurpers.                                       |
|                                                             |
|  Battle of Karbala, 680 CE:                                 |
|  Ali's son Husayn ibn Ali refuses to pledge allegiance      |

|  Commemorated in Ashura — annual mourning ritual.           |
|                                                             |
|  THEOLOGICAL DIVERGENCE:                                    |
|  Sunni: Caliph is political/administrative leader.          |

```

### Jurisprudence (Fiqh)

Four Sunni legal schools (madhabs), each associated with a founding jurist:

| School | Founder | Geographic base | Methodology |
|--------|---------|----------------|-------------|
| Hanafi | Abu Hanifa (d.767) | Turkey, South Asia, Central Asia | Most use of ra'y (opinion), analogical reasoning |
| Maliki | Malik ibn Anas (d.795) | North/West Africa | Heavy weight on Medinan practice (Sunna of Medina) |
| Shafi'i | al-Shafi'i (d.820) | Egypt, Southeast Asia | Systematized usul al-fiqh (legal theory) |
| Hanbali | Ahmad ibn Hanbal (d.855) | Arabian Peninsula | Strictest adherence to hadith; least ra'y |

Shia: Ja'fari school (after Imam Ja'far al-Sadiq) — allows mut'a (temporary marriage),
differs on inheritance and some ritual law.

### Sufism

Sufism (tasawwuf) is the mystical dimension of Islam — the inner path (tariqat) to
direct experience of God (fana, annihilation of self in God).

- Tariqas (orders): institutionalized Sufi lineages with distinctive practices,
  dhikr (rhythmic remembrance of God — chanting divine names), sama (spiritual
  listening/music, context of Rumi's Whirling Dervishes)
- Ibn Arabi (1165-1240): greatest systematic Sufi metaphysician. Wahdat al-wujud
  (unity of being) — everything is manifestation of the one Being. Profoundly
  influenced later Islamic thought and arguably neoplatonism.
- Rumi (1207-1273): Masnavi, the Divan-e Shams — poetry expressing divine love.
  Most widely read poet in the United States today. Mevlevi order (Whirling Dervishes).
- Tension with legalistic Islam: Sufis have faced periodic persecution from Salafi/
  Wahhabi reformers who view saint veneration as shirk (associating partners with God).

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| What is the Talmud? | Mishnah (oral law codified ~200 CE) + Gemara (rabbinic commentary) = Talmud. The Babylonian Talmud (~500 CE) is authoritative. |
| Why did Judaism survive the Temple's destruction? | Rabbinic revolution: replaced Temple/sacrifice with synagogue/study/prayer. Portable religion. |
| What was decided at Nicaea? | Homoousios: Jesus is of same substance as the Father. Arianism (Jesus is lesser, created) condemned. |
| What is the Chalcedonian formula? | Jesus has two natures (divine + human) in one person, without confusion or separation. ~451 CE. |
| Why Sunni vs. Shia? | Succession dispute after Muhammad's death (632 CE). Sunni: elected Caliph. Shia: Ali (Muhammad's cousin/son-in-law) as rightful Imam. Karbala (680 CE) sealed the split. |
| What is sharia? | Islamic law derived from Quran, Sunna (hadith), ijma (consensus), and qiyas (analogy). Not a single code — a tradition of legal reasoning. |
| What is the difference between Quran and hadith? | Quran = God's direct speech. Hadith = reports of Muhammad's words/actions. Different authority, different textual status. |

## Common Confusion Points

**"Old Testament" vs. Tanakh**: Not the same. The Christian Old Testament (in Protestant
versions) contains the same books as the Tanakh but in different order and grouping.
Catholic and Orthodox OTs include deuterocanonical books (Maccabees, Sirach, etc.)
rejected by Protestants as Apocrypha. Jews do not use the term "Old Testament" —
it implies supersession by a "New" one.

**Halakha is not Christian legalism**: In Christian theological polemic, "law" is
contrasted with "grace" (Paul's argument in Galatians). This frames Jewish law as
a burden. From the inside, halakha is not a burden but a covenant privilege — the
framework of relationship with God.

**Sharia ≠ penal code**: In Western media, "sharia" connotes amputation and stoning.
In reality, sharia covers all of life — prayer times, dietary law, business ethics,
family law. Criminal penalties (hudud) are a small subset, applied rarely even in
traditional Islamic law, requiring extremely high evidentiary standards.

**"Muhammad as Prophet"**: In Islamic theology, Muhammad is fully human — not divine,
not the Son of God. Calling Muslims "Muhammadans" (old Western usage) misrepresents
Islam: Muslims worship God (Allah), not Muhammad.

**The Trinity is not three gods**: The technical claim is one God in three hypostases
(persons/subsistences). Tritheism is a heresy within Christianity. The Quran's
rejection of the Trinity (Surah 4:171) may be partly targeting a popular misunderstanding
(God, Jesus, Mary as the three) rather than the technical Nicene formulation.
