# Comparative Religion — Field Overview

## The Big Picture

Academic religious studies is descriptive/analytical, not normative/confessional.
The goal: understand religion as a human phenomenon — its origins, structures, functions,
transmission, and variation — not to adjudicate between competing truth claims.

```
+-----------------------------------------------------------------------+
|              RELIGIOUS STUDIES AS ACADEMIC FIELD                      |
|                                                                       |
|   ADJACENT FIELDS          RELIGIOUS STUDIES         NOT THIS         |
|   ----------------         -----------------         --------         |
|   Philosophy of        --> Descriptive, historical,  Theology:        |
|   Religion                 comparative, analytical   normative,       |
|                            What do people believe?   confessional     |
|   Anthropology         --> Why? What social          What should      |
|                            function? How trans-       we believe?     |
|   Psychology           --> mitted? How does it                        |
|                            evolve and change?                         |
|                                                                       |
|   Cognitive Science    --> Religion as natural human                  |
|   of Religion              phenomenon: byproduct of                   |
|                            cognitive architecture                     |
|   Sociology            --> Religion as social fact:                   |
|                            solidarity, meaning,                       |
|                            legitimation of order                      |
+-----------------------------------------------------------------------+
```

Methodological key distinction — emic vs. etic:
- Emic: the insider view. What practitioners believe and mean.
- Etic: the analyst's view. Structural/functional categories imposed from outside.
Good religious studies uses both and is transparent about which it is doing.

This file: field overview. Adjacent coverage: `philosophy/05-PHILOSOPHY-OF-RELIGION.md`
covers God arguments and theodicy. This directory covers living religious traditions,
their sociology, psychology, and the science interface.

## The Founding Figures — Origin Theories

The 19th and early 20th century established the discipline's categories. Understanding
them means knowing what each theorist got right and where the critiques land.

### Tylor: Minimum Definition and the Animism Hypothesis

Edward Burnett Tylor (1832-1917) defined religion minimally as "belief in spiritual beings."
The minimum definition strategy avoids begging questions about God, truth, or salvation —
you can apply it cross-culturally without assuming Christianity is the prototype.

His evolutionary schema: animism (spirits everywhere) → polytheism (spirits consolidate
into named gods) → monotheism (gods unify). Problems:
- Teleological: implies monotheism is most advanced, most evolved
- Colonial: non-Western religions treated as earlier stages of European development
- Ignores function: what religion does socially and psychologically
- Empirically wrong: "evolution" doesn't track actual history (polytheistic Greeks were
  chronologically after monotheistic Akhenaten)

Still useful: minimum definition remains a tool for comparative work. The animism
category, rehabilitated by Harvey's "new animism," remains analytically alive.

### Müller: The Disease-of-Language Hypothesis

Max Müller (1823-1900): gods arose because language over-extended metaphor. "Dawn" becomes
a personal agent (Daphne), "sun" becomes Apollo — poetic language mistaken for literal
personhood. Religion as accumulated metaphor crystallized into myth.

Critique: explains nothing about why these specific metaphors took religious form;
ignores function entirely; exemplary "nothing-buttery" (reduces religion to a mistake
of language). But: Müller's comparative philological method was rigorous for its time
and his translations of the Rig Veda brought Sanskrit literature into Western scholarship.
His Sacred Books of the East (50 volumes) remains a foundational reference.

### Durkheim: The Social Fact Theory

Emile Durkheim's "The Elementary Forms of Religious Life" (1912) is the most
influential sociological account of religion. Key argument structure:

```
+----------------------------------------------------------+
|  DURKHEIM'S ARGUMENT                                     |
|                                                          |
|  Step 1: To find religion's essence, study the           |
|          simplest form available.                        |
|          --> Australian Aboriginal (Arunta) totemism     |
|                                                          |
|  Step 2: Sacred/profane is the formal category that      |
|          defines religion. NOT "belief in gods."         |
|          Sacred = extraordinary, set apart, powerful     |
|          Profane = ordinary, everyday, instrumental      |
|                                                          |
|  Step 3: The totem (clan emblem) is simultaneously:      |
|          - symbol of the sacred force (mana)             |
|          - symbol of the clan itself                     |
|          Therefore: god = society                        |
|                                                          |
|  Step 4: Collective effervescence — intense group        |
|          ritual generates emotional energy that feels    |
|          like contact with the sacred. The sacred is     |
|          real: it IS the social group's power.           |
|                                                          |
|  Step 5: Religion's function = social solidarity.        |
|          Shared beliefs + shared rituals = moral         |
|          community. That's what religion does.           |
+----------------------------------------------------------+
```

What Durkheim got right: sacred/profane distinction is genuinely formal and cross-cultural.
Collective effervescence is a real phenomenon (crowd psychology, ritual emotion).
The social function of religion (solidarity, moral community) is real.

What Durkheim got wrong: functionalism doesn't explain content — why THIS totem?
The god = society reduction is too strong (Otto's response: the numinous resists
social reduction). The Arunta ethnography was second-hand and partially inaccurate
(Spencer/Gillen's data had errors). Malinowski's critique: if religion = solidarity,
why do people pray alone?

### Weber: Rationalization and the Verstehen Method

Max Weber (1864-1920) approached religion through Verstehen (interpretive understanding
— grasping the actor's own meaning framework, not just external causal explanation)
and ideal types (analytical constructs that clarify by exaggerating salient features).

Key contributions:
- Protestant Ethic thesis: Calvinist predestination anxiety → systematic work discipline
  → capital accumulation. See `06-SOCIOLOGY-RELIGION.md` for full treatment.
- Rationalization (Rationalisierung): modernity as progressive disenchantment of the world
  (Entzauberung) — removal of magic, systematization, bureaucratization, formal rationality
  displacing substantive rationality. Religion either survives as a private meaning system
  or gets absorbed into ethics and bureaucracy.
- Three types of authority: charisma (personal gift of grace), tradition (it has always
  been done this way), legal-rational (rule-governed office). Routinization of charisma:
  the founder's personal authority gets institutionalized, bureaucratized, diluted.
- Theodicy as the central religious problem: why do the righteous suffer? Each tradition's
  answer to this shapes its sociology. "Theodicy of disprivilege": subordinate groups
  receive consolation (the meek shall inherit the earth); dominant groups receive
  legitimation (wealth signals divine favor).

### Eliade: Hierophany and the Homo Religiosus

Mircea Eliade (1907-1986): phenomenological approach. Sacred breaks into the profane world
through hierophany — any manifestation of the sacred in a mundane object or event.

```
  Hierophany:    Any manifestation of the sacred. A rock, a tree, a person,
                 a moment in time. The sacred is not a property of the object
                 but of its relation to the sacred order.

  Axis mundi:    World-pole connecting heaven/earth/underworld. Manifests as
                 sacred mountain (Meru, Olympus, Sinai), tree (Yggdrasil,
                 Bodhi tree), pillar, temple. Every sacred space is a center.

  Eternal return: Ritual does not merely commemorate primordial events —
                  it re-enacts them, making them present again (in illo tempore).
                  Sacred time is cyclical, not linear. The calendar re-creates
                  cosmogony (New Year = recreation of the world).

  Homo religiosus: The religious human as the natural, universal mode. Being
                   human means orienting toward the sacred. Secular modernity
                   is historically anomalous — modern people suppress religious
                   orientation but it re-emerges in secular forms (nation,
                   ideology, sports, celebrity).
```

Critiques of Eliade (substantial):
- He was a Romanian nationalist with Iron Guard connections — his universalist
  "perennialism" (all religions are variations on the same sacred) served
  ideological functions (subsumes particulars under a universalist haze)
- Flattens historical specificity: treating Amazonian ritual and Chartres cathedral
  as variations on one theme loses everything important about both
- Circular method: defines religious experience by phenomenological content, then
  uses that content to identify what counts as religion
- Crypto-theology: the homo religiosus assumption smuggles normative claims into
  an ostensibly descriptive framework

### Otto: The Numinous

Rudolf Otto's "Das Heilige" (The Idea of the Holy, 1917) argued that religion
has an irreducible experiential core: the numinous (from Latin numen).

```
  Mysterium:    That which is wholly other — beyond rational categories,
                not merely unknown but structurally different.

  Tremendum:    The overwhelming aspect — awe, dread, "creature-feeling"
                (Kreaturgefühl): sense of being nothing before the absolute.

  Fascinans:    The attractive aspect — fascination, allure, drawing power,
                the magnet of the holy.
```

Otto's contribution: against reduction of religion to ethics (Kant), social function
(Durkheim), or language mistake (Müller). The holy has its own a priori category.
Critique: phenomenological description is not causal explanation. It normalizes
a specific type of experience (Christian mystical-Protestant) as universal.
The "wholly other" claim may itself be a culturally specific construction.

## Classification Problem

Every taxonomy of religion has problems. The question is which problems you can live with.

```
+----------------------------------------------------------------------+
|                   CLASSIFICATION SCHEMES                             |
|                                                                      |
|  BY DEITY CONCEPT           BY STRUCTURE        BY SCOPE             |
|  ----------------           ------------        --------             |
|  Animism (spirits           Revealed            Universal/missionary |
|    everywhere)              (scripture-based)   (Islam, Christianity,|
|  Polytheism (many gods)     Philosophical       Buddhism)            |
|  Henotheism (one god        (no scripture,      Ethnic/national      |
|    but others exist)        rational)           (Hinduism, Judaism,  |
|  Monotheism (one God,       Folk/popular        Shinto)              |
|    others don't exist)      (no institution)    New/emerging         |
|  Nontheism (Buddhism,       Institutional/      Revitalization       |
|    Jainism — no creator     organized           movements            |
|    God concept)                                                      |
|  Pantheism (God = nature)   BY PRACTICE         BY SOCIAL FORM       |
|  Panentheism (God           Orthoprax            Church              |
|    contains world)          (correct action:     Sect                |
|  Deism (no intervention)    Judaism, Hinduism)   Denomination        |
|                             Orthodox             Cult / NRM          |
|                             (correct belief:                         |
|                             Catholicism)                             |
+----------------------------------------------------------------------+
```

The polytheism → monotheism evolutionary sequence (Tylor's scheme) is wrong as history.
Monotheism emerged independently multiple times: Akhenaten's Egyptian experiment (14th c BCE),
Zoroastrian monotheism, and Second Temple Judaism's development of strict monotheism under
Persian influence are not a linear progression. Polytheistic traditions can have sophisticated
unity concepts: Hindu Advaita reduces all gods to Brahman (one ultimate reality).

"Henotheism" (Müller's coinage): worship of one god without denying others exist.
This describes early Israelite religion more accurately than "monotheism" — "You shall
have no other gods before me" presupposes other gods exist.

The orthoprax/orthodox axis matters: Western (especially Protestant) Christianity is
unusually belief-focused. Many traditions — Judaism, Confucianism, Hinduism — are
primarily about correct practice (halakha, li, dharma). Asking a practitioner
"what do you believe?" may miss the entire point of their tradition.

## The Current Frontier: Cognitive Science of Religion

Since the 1990s, the live research frontier applies cognitive science to explain
why religion is universal and why it takes the specific forms it does.

```
+----------------------------------------------------------------------+
|          COGNITIVE SCIENCE OF RELIGION (CSR)                         |
|                                                                      |
|  Question: Why does religion exist at all?                           |
|  Answer: Byproduct of cognitive adaptations evolved for other        |
|          purposes. Not adaptive in itself — a spandrel.              |
|                                                                      |
|  Mechanism 1: HADD — Hypersensitive Agency Detection Device          |
|  (Justin Barrett, Pascal Boyer)                                      |
|  - We evolved to detect agents (predators, allies) in the            |
|    environment.                                                      |
|  - Calibrated for false positives: rustling = tiger,                 |
|    better wrong than dead.                                           |
|  - Overextends: sees agency in wind, rivers, disease, fortune.       |
|  - Natural cognitive source of spirit/god/demon attribution.         |
|                                                                      |
|  Mechanism 2: Minimally Counterintuitive Concepts (Boyer)            |
|  - Concepts that violate ONE intuitive expectation but               |
|    conform to all others are maximally memorable/transmissible.      |
|  - Ghost: violates physical solidity, normal psychology otherwise.   |
|  - God: violates human limitation, but has human-type intentions.    |
|  - Zombie: violates psychology, conforms to physics.                 |
|  - MCIs spread; maximally counterintuitive concepts don't.           |
|                                                                      |
|  Mechanism 3: Intuitive Dualism (Paul Bloom)                         |
|  - Children naturally treat mind and body as distinct entities.      |
|  - Soul beliefs are cognitively natural, not culturally taught.      |
|  - Essentialist thinking about persons persists after death.         |
|                                                                      |
|  Mechanism 4: Teleological Thinking (Deborah Kelemen)                |
|  - Children (and adults) over-attribute purpose/design to nature.    |
|  - "Rocks are pointy so animals won't sit on them."                  |
|  - Primes creationist intuitions independent of teaching.            |
|                                                                      |
|  Key researchers: Pascal Boyer (Religion Explained, 2001)            |
|                   Scott Atran (In Gods We Trust, 2002)               |
|                   Harvey Whitehouse (modes theory)                   |
|                   Justin Barrett (Why Would Anyone Believe in God?)  |
|                   Robert McCauley (Why Religion is Natural)          |
+----------------------------------------------------------------------+
```

Whitehouse's modes of religiosity theory: two transmission modes with distinct
psychological and social signatures.

| Mode | Arousal | Frequency | Memory type | Group size | Social form |
|------|---------|-----------|-------------|------------|-------------|
| Imagistic | High | Rare, infrequent | Episodic, vivid | Small, intense | Cell, warband, gang |
| Doctrinal | Low | Frequent, routine | Semantic, catechistic | Large, anonymous | Church, institution |

Imagistic mode: initiation ordeals, vision quests, traumatic one-off rituals. Creates
intense cohesion in small groups through shared extraordinary experience. Hard to scale.
Doctrinal mode: weekly sermons, catechism, daily prayer. Creates large anonymous
communities through shared explicit doctrine. Scales to millions.
Many traditions contain both: Catholic mystical orders (imagistic) within the Roman
Church (doctrinal). The Sufi tariqas (imagistic) within Islam (doctrinal).

Geographic distribution:
- Christianity: Americas, sub-Saharan Africa, Europe, Oceania
- Islam: Middle East, North/West Africa, Central/South/Southeast Asia
- Hinduism: concentrated in South Asia (India ~80% Hindu)

- Unaffiliated: Europe, East Asia (China), North America

The "Global South shift": Christianity's center of gravity has migrated to Africa
and Latin America. The modal Christian in 2026 is Nigerian or Brazilian, not European.
This reshapes theology (prosperity gospel, liberation theology), politics, and
demographics. Pentecostalism is the fastest-growing form worldwide.

## The Secularization Thesis — Rise and Fall

```
+--------------------------------------------------------------+
|              SECULARIZATION THESIS                           |
|                                                              |
|  Original thesis (Wilson, early Berger, Bruce, 1960s-80s):   |
|  Modernization -> differentiation of social spheres          |
|  -> religion loses public role -> privatized -> withers      |
|                                                              |
|  Evidence FOR: European church attendance collapse.          |
|  Elite secularity (scientists, academics, mostly secular).   |
|  Institutional religion declining across Western Europe.     |
|  State churches losing membership in Scandinavia.            |
|                                                              |
|  Counter-evidence:                                           |
|  - Global South explosion: Pentecostalism, Islam growing     |
|  - US exceptionalism: modern + highly religious              |
|  - Islamic world resurgence despite modernization            |
|  - "Believing without belonging" (Grace Davie):              |
|    institutional decline != belief decline                   |
|  - "Spiritual but not religious" category expanding          |
|  - New religious movements growing alongside decline         |
|    of mainline denominations                                 |
|                                                              |
|  Peter Berger (original theorist) recanted ~2000:            |
|  "The world is as furiously religious as ever."              |
|  Revised view: Europe is the exception, not the rule.        |
|  Even then: "believing without belonging" still counts.      |
+--------------------------------------------------------------+
```

The more defensible claim: institutional religion is declining in Western Europe
and among educated elites globally. But religion as a phenomenon — belief in the
supernatural, spiritual practice, identity-forming ritual — shows no signs of
disappearing at scale.

## Methodological Approaches Compared

| Approach | Core Question | Key Methods | Key Figures | Limitations |
|----------|--------------|-------------|-------------|-------------|
| Phenomenological | What does religious experience feel like from inside? | Epoché (bracketing), thick description | Otto, Eliade, van der Leeuw | Risk of crypto-theology; treats subject's account as evidence |
| Historical-critical | How did this text/practice originate and change? | Source criticism, redaction, archaeology | Wellhausen, Bultmann, Crossan | Can dissolve the living tradition into fragments |
| Sociological | What social functions does religion serve? | Statistical surveys, ethnography, ideal types | Durkheim, Weber, Berger | Functionalism doesn't explain content or truth claims |
| Anthropological | How does religion work within a specific community? | Participant observation, fieldwork | Geertz, Turner, Evans-Pritchard | Particularism; hard to generalize |
| Cognitive | Why is religion universal? What cognitive mechanisms? | Lab experiments, cross-cultural studies | Boyer, Barrett, Atran | May miss meaning and context while explaining mechanism |
| Psychological | How does religion affect individuals? | Surveys, clinical studies, neuroimaging | James, Allport, Pargament | Early work conflated religion with pathology (Freud) |

## Common Confusion Points

**"Theology" vs. "Religious Studies"**: Theology is confessional — done from within a
tradition, for that tradition. Religious studies is descriptive/analytical — treats all
traditions as objects of study. A theologian argues what Christianity should believe;
a scholar of religion studies what Christians have believed and why. The distinction
is methodological, not hostile — good theology can be good scholarship too.

**The category "religion" itself**: Scholars (Wilfred Cantwell Smith, Talal Asad)
argue "religion" is a Western, Protestant-inflected category exported globally by
colonialism. Pre-colonial Japanese had no term separating religion from everything else.
The category carves up human practice in ways that may distort non-Western traditions.

**"Mysticism" as unified category**: Katz's constructivist thesis: there is no
unmediated mystical experience — the Christian mystic's experience of God, the Sufi's
fana, and the Zen master's satori are shaped by distinct conceptual frameworks and
are genuinely different experiences. Forman's perennialist response: "pure consciousness
events" transcend doctrinal variation. Not resolved.

**Animism as "primitive"**: Tylor's evolutionary ladder (animism → monotheism) encodes
colonial assumptions. Animist traditions are not earlier stages of anything — they are
different ontological frameworks, sometimes philosophically sophisticated (Amazonian
perspectivism is serious metaphysics).

**"Eastern religion" as unified**: Buddhism, Hinduism, Daoism, Confucianism, Shinto
have radically different metaphysics, anthropologies, and soteriology. "Eastern" is
a geographic label that generates analytical confusion.
