## Vietnamese

### Profile

```
+--------------------------------------------------------------------+
| VIETNAMESE (Tiếng Việt)                                            |
+--------------------------------------------------------------------+
| Family:      Austroasiatic → Mon-Khmer → Viet-Muong               |
| Native spkrs: ~76M                                                 |
| Countries:   Vietnam (official); diaspora in USA, France, Cambodia |
| Word order:  SVO                                                   |
| Morphology:  Isolating — no inflection, no affixes                |
| Tonal:       YES — 6 tones (most in the world's major languages)  |
| Gender:      None (grammatical)                                    |
| Script:      Chữ Quốc Ngữ — Latin alphabet + extensive diacritics |
| FSI level:   Category III-IV — ~1100 hours                        |
+--------------------------------------------------------------------+
```

### Vietnamese Tones — 6 Tones

```
6 TONES OF VIETNAMESE
======================
Marked by diacritics on vowels:

+--------+--------+--------+--------------------+------------------+
| Tone   | Mark   | Name   | Description        | Example          |
+--------+--------+--------+--------------------+------------------+
| FLAT   | a      | ngang  | Level, mid pitch   | ma (ghost)       |
| FALLING| à      | huyền  | Low falling        | mà (but)         |
| RISING | á      | sắc    | High rising        | má (mother)      |
| BROKEN | ã      | ngã    | High rising then   | mã (horse code)  |
|        |        |        | creaky drop        |                  |
| HEAVY  | ạ      | nặng   | Low, constricted   | mạ (rice seedling)|
| CURVE  | ả      | hỏi    | Mid dipping then   | mả (tomb)        |
|        |        |        | rising             |                  |
+--------+--------+--------+--------------------+------------------+

Vietnamese has more tonal distinctions than Mandarin (4) or Thai (5).
Northern dialect (Hanoi): all 6 tones clearly distinct.
Southern dialect (Ho Chi Minh City): ngã and hỏi merge = 5 effective tones.

Each vowel can take 6 tone marks:
  a  à  á  ã  ạ  ả
  ă  ằ  ắ  ẵ  ặ  ẳ  (a-breve: short a)
  â  ầ  ấ  ẫ  ậ  ẩ  (a-circumflex: different vowel quality)
  e  è  é  ẽ  ẹ  ẻ
  ê  ề  ế  ễ  ệ  ể  (e-circumflex)
  i  ì  í  ĩ  ị  ỉ
  o  ò  ó  õ  ọ  ỏ
  ô  ồ  ố  ỗ  ộ  ổ  (o-circumflex)
  ơ  ờ  ớ  ỡ  ợ  ở  (o-horn: different vowel)
  u  ù  ú  ũ  ụ  ủ
  ư  ừ  ứ  ữ  ự  ử  (u-horn)
  y  ỳ  ý  ỹ  ỵ  ỷ

Total: 12 vowel base forms × 6 tones = up to 72 vowel representations.
Vietnamese typography is the most diacritic-heavy of any Latin-script language.
```

### Vietnamese Grammar

```
GRAMMAR — PURE ISOLATING
==========================
No conjugation. No declension. No agreement.
Grammar = word order + particles + context.

SENTENCE STRUCTURE:
  Subject + Verb + Object: Tôi ăn cơm. (I eat rice.)
  Negation: Subject + không + Verb: Tôi không ăn cơm. (I don't eat rice.)
  Question: Add không at end (yes/no): Bạn ăn cơm không?
            Or question words: Bạn ăn gì? (What do you eat?)
  Past: Add đã before verb: Tôi đã ăn. (I already ate.)
  Future: Add sẽ before verb: Tôi sẽ ăn. (I will eat.)

PRONOUNS — reflect social relationship:
  Unlike European languages, pronouns encode age/status:
  tôi — neutral "I" (writing/formal)
  mình — reflexive/casual I
  anh — I/you for older male (polite to address him or self = younger brother)
  chị — I/you for older female
  em — I/you for younger person (self-humbling)
  bạn — "friend" as 2nd person (neutral)
  ông/bà — grandfather/grandmother = very formal you

CLASSIFIERS (similar to Chinese/Japanese measure words):
  con (animals, some objects): con chó (dog), con dao (knife)
  cái (inanimate objects): cái bàn (table), cái áo (shirt)
  người (people): người đàn ông (the man)
  chiếc (one of a pair/item): chiếc xe (vehicle)
```

## Thai

### Profile

```
+--------------------------------------------------------------------+
| THAI (ภาษาไทย Phāsā Thai)                                          |
+--------------------------------------------------------------------+
| Family:      Tai-Kadai → Tai → Southwestern Tai                   |
| Native spkrs: ~36M (total speakers ~60M including L2)             |
| Countries:   Thailand (official)                                   |
| Word order:  SVO                                                   |
| Morphology:  Isolating                                             |
| Tonal:       YES — 5 tones                                        |
| Gender:      None (grammatical)                                    |
| Script:      Thai abugida — 44 consonant classes, vowel diacritics |
|              No spaces between words                               |
| FSI level:   Category III-IV — ~1100 hours                        |
+--------------------------------------------------------------------+
```

### Thai Tones — 5 Tones

```
5 TONES OF THAI
================
+--------+--------+-----------+------------------+
| Tone   | Mark   | Pitch     | Example (khai family)|
+--------+--------+-----------+------------------+
| MID    | none   | Level mid | ไข่ (khai) egg   |
| LOW    | ่ (mai ek)| Low fall| ไข้ (khai) fever |
| FALLING| ้ (mai tho)| High to low| ข้าว (khao) rice|
| HIGH   | ๊      | High      | ค่า (khā) value  |
| RISING | ๋      | Low then  | หมา (maa) dog    |
|        |        | high      |                  |
+--------+--------+-----------+------------------+

Thai tones are DETERMINED by 3 factors:
  1. Class of the initial consonant (high/mid/low class)
  2. Tone mark (or absence of one)
  3. Vowel length (long vs short)
  4. Final consonant type (sonorant vs stop)

This is more complex than Mandarin where tone is directly marked.
Thai tone determination = consonant class + mark + syllable type.
Must learn consonant classes to read tones correctly.
```

### Thai Script Overview

```
THAI SCRIPT (อักษรไทย)
========================
Type: Abugida (consonant-inherent vowel with diacritic vowels)
Direction: Left to right
Spaces: No spaces between words (spaces mark phrases/sentences)
        Word segmentation must be inferred from context

CONSONANT CLASSES (critical for tones):
  High class (11):   ข ฉ ถ ผ ฝ พ ฟ ภ ศ ษ ส ห
  Mid class (9):     ก จ ด ต ฎ ฏ บ ป อ
  Low class (24):    ง ญ ณ น ม ย ร ล ว ฃ ค ฅ ฆ ง ช ซ ฌ ญ ฑ ฒ ณ ท ธ น

44 base consonant symbols (but only ~21 distinct consonant sounds — multiple
consonants share the same sound, distinguished only for historical/tone reasons).

VOWEL DIACRITICS — appear above, below, before, or after consonant:
  -า  (aa, long)       -ิ (i, short)        -ี (ii, long)
  -ุ  (u, short)       -ู (uu, long)        -เ- (ee)
  -แ- (ae)             -โ- (o)              -ใ-/-ไ- (ai)
  เ-า (ao)             -ัว (ua)             etc.

No spaces between words:
  อยากกินข้าวผัดกุ้งไหม = Do you want to eat fried rice with shrimp?
  A Thai reader segments this mentally; a learner struggles.
```

## Indonesian / Malay

### Profile — The Easiest Major Asian Language for English Speakers

```
+--------------------------------------------------------------------+
| INDONESIAN (Bahasa Indonesia) / MALAY (Bahasa Melayu)             |
+--------------------------------------------------------------------+
| Family:      Austronesian → Malayo-Polynesian → Malayic           |
| Native spkrs: ~199M Indonesian + ~20M Malay (distinct standards)  |
| Total spkrs:  ~270M+                                               |
| Countries:   Indonesia (official), Malaysia (official), Brunei,    |
|              Singapore (one of 4 official), East Timor             |
| Word order:  SVO                                                   |
| Morphology:  Agglutinative — but regular and learnable             |
| Tonal:       NO                                                    |
| Gender:      NONE                                                  |
| Script:      Latin                                                 |
| Cases:       NONE                                                  |
| FSI level:   Category II — ~900 hours                              |
+--------------------------------------------------------------------+

WHY IT'S EASY:
  No tones. No gender. No case. Latin script.
  Consistent spelling (nearly phonemic).
  No verb conjugation for person/tense (context + time words).
  Large Portuguese/Dutch/English/Arabic loanword base.
  Basic conversation achievable in 3-4 months of serious study.
```

### Grammar Highlights

```
INDONESIAN GRAMMAR
===================
VERBS — no conjugation:
  Saya makan. (I eat.)
  Kamu makan. (You eat.) — same verb form
  Dia makan. (He/She eats.) — same

TENSE — indicated by time words:
  Past: sudah (already), tadi (just), kemarin (yesterday)
  Future: akan (will), besok (tomorrow), nanti (later)
  Saya sudah makan. (I have already eaten.)
  Saya akan makan besok. (I will eat tomorrow.)

AFFIXATION SYSTEM — the main grammatical complexity:
  Base words take prefixes/suffixes that change meaning/part-of-speech:
  makan (to eat, base)
  memakan (to eat, active transitive)
  dimakan (to be eaten, passive)
  makanan (food, noun)
  memakani (to feed something/someone)
  termakan (accidentally eaten)
  pemakan (eater, agent noun)

  me- (active transitive verb)
  di- (passive verb)
  ber- (intransitive/reflexive)
  -an (noun, result/thing)
  -kan (causative, benefactive)
  -i (directional/locative)
  pe- / pe-an (agent/place nouns)

REDUPLICATION — plurals and other meanings:
  buku (book) → buku-buku (books) — full reduplication = plural
  berlari-lari (to keep running, intensification)
  perlahan-lahan (very slowly, intensification)
  orang-orang (people, plural of orang = person)

NUMBERS — no classifiers (unlike Chinese/Japanese):
  dua buku (two books) — no measure word needed
  lima orang (five people) — orang = "people" but also acts as classifier

PRONOUNS:
  saya (I, formal)  aku (I, informal)
  kamu/Anda (you, informal/formal)
  dia/ia (he/she — no gender distinction)
  kami (we, exclusive = I + others, not you)
  kita (we, inclusive = I + you + others)
  mereka (they)
```

## Burmese, Khmer, Lao — Script Overviews

### Burmese

```
BURMESE (မြန်မာဘာသာ Mranma Bhasa)
=====================================
Family:      Sino-Tibetan → Tibeto-Burman → Lolo-Burmese
Speakers:    ~33M native
Script:      Burmese abugida — circular script (curves dominant)
             Adapted from Mon script, derived from Brahmi
Tones:       4 (creaky, low, high, stopped/entering)
Order:       SOV
Morph:       Isolating-Agglutinative hybrid

Script characteristics:
  Circles and curves — historically written on palm leaves (straight lines
  would tear the leaf). Unique rounded appearance.
  44 consonants, 8 vowel diacritics + vowel indicators
  Direction: left-to-right

Sample consonants (partial): က ခ ဂ ဃ င (velar row)
                              စ ဆ ဇ ဈ ည (palatal row)
  Syllable structure: consonant + vowel mark + optional final
  Stacked consonants for clusters: တ + ျ + ာ = ကြာ

Key phrases:
  မင်္ဂလာပါ (Mingalaba) — Hello
  ကျေးဇူးတင်ပါတယ် (Kyay zu tin ba deh) — Thank you
  မသိဘူး (Ma thi buu) — I don't know/understand
```

### Khmer (Cambodian)

```
KHMER (ភាសាខ្មែរ Phiasa Khmaer)
==================================
Family:      Austroasiatic → Mon-Khmer
Speakers:    ~16M native (Cambodia + diaspora)
Script:      Khmer abugida — derived from Brahmi via old Khmer
             Largest alphabet in the world (~74 characters)
Tones:       2 registers (not strictly tonal like Thai)
Order:       SVO
Morph:       Isolating

Script characteristics:
  74 symbols (33 consonants in two series + 24 independent vowels + diacritics)
  Two consonant series (a-series and o-series) affect vowel quality
  Like Thai: no spaces between words
  Subscript consonants (foot consonants): smaller form below main consonant

Sample: ស្រុកខ្មែរ (Srok Khmer = Cambodia/Khmer country)
  ស = s, ្រ = r (subscript), ុ = u, ក = k → sruk

Key phrases:
  ជំរាបសួរ (Chomreab suor) — Hello (formal)
  អរគុណ (Aw-kun) — Thank you
  ខ្ញុំមិនយល់ (Kh'nyom min yul) — I don't understand
```

### Lao

```
LAO (ພາສາລາວ Phasa Lao)
========================
Family:      Tai-Kadai → Tai (closely related to Thai)
Speakers:    ~30M (Laos + Isan region of Thailand + diaspora)
Script:      Lao abugida — closely related to Thai script
Tones:       6 (similar structure to Thai 5, regional variation)
Order:       SVO
Morph:       Isolating

Lao vs Thai: ~85% mutually intelligible in written form.
  Lao script has fewer letters (33 consonants vs Thai 44).
  Some Thai sounds absent in Lao (simplified phonology).
  Isan dialect (NE Thailand): essentially Lao spoken in Thailand.
  Script comparison: Lao ກ vs Thai ก (both = /k/) — similar curvilinear forms.

Thai speakers can read Lao with moderate effort and vice versa.
For practical purposes: if you learn Thai, Lao is accessible.

Key phrases:
  ສະບາຍດີ (Sabai dii) — Hello / How are you
  ຂອບໃຈ (Khob jai) — Thank you
  ບໍ່ເຂົ້າໃຈ (Bo khao jai) — I don't understand
```

## Common Confusion Points

**"Vietnamese Latin script — easy right?"**
The script is Latin but the diacritics encode BOTH tone AND vowel quality.
The same vowel letter with different marks = different sound AND different tone.
Reading speed comes easily; pronunciation without hearing native speakers
is nearly impossible from text alone.

**"Thai has no spaces — how do you read it?"**
Experienced readers segment by grammar intuition. For learners:
use a dictionary with word-break assistance, or apps like Longdo.
Reading Thai without knowing vocabulary is genuinely hard to start.

**"Indonesian is the easiest, so why 900 hours?"**
The 900 hours is for professional proficiency (FSI level 3). Conversational
survival: 3-4 months. The hours go toward the affix system (me-/di-/ber-/per-...),
compound vocabulary, and formal register differences.

**"Khmer and Vietnamese look similar?"**
No — Khmer uses its own script (curvilinear, like Burmese/Thai family).
Vietnamese uses Latin. The Austroasiatic family connection (both Mon-Khmer)
doesn't make the languages mutually intelligible — Khmer and Vietnamese
diverged thousands of years ago and have very different phonologies.
