## Typological Dimensions

Every language can be characterized along these axes. These are the dimensions
that determine how "different" a language feels to an English speaker.

### 1. Word Order

```
+---------------+------------------------------------------+------------------------+
| Order         | Languages                                | Notes                  |
+---------------+------------------------------------------+------------------------+
| SVO           | English, French, Spanish, Portuguese,    | Subject-Verb-Object    |
| (most common) | Mandarin, Vietnamese, Indonesian, Swahili| "I eat rice"           |
+---------------+------------------------------------------+------------------------+
| SOV           | Japanese, Korean, Hindi/Urdu, Turkish,   | Subject-Object-Verb    |
| (2nd most)    | Persian, German (subordinate), Latin,    | "I rice eat"           |
|               | Pashto, Telugu, Tamil, Kannada           | Verb always at end     |
+---------------+------------------------------------------+------------------------+
| VSO           | Arabic (Modern Standard), Welsh, Arabic  | Verb-Subject-Object    |
|               | dialects (Classical)                     | "Eat I rice"           |
+---------------+------------------------------------------+------------------------+
| Free order    | Russian, Latin, Polish, Finnish          | Case marking carries   |
|               | (pragmatic constraints apply)            | meaning, not position  |
+---------------+------------------------------------------+------------------------+
```

### 2. Morphological Type

```
+-------------------+-------------------+----------------------------------------+
| Type              | Examples          | What it means                          |
+-------------------+-------------------+----------------------------------------+
| ISOLATING         | Mandarin,         | Words don't change form. Grammar via   |
| (analytic)        | Vietnamese,       | word order + particles.                |
|                   | Thai, Indonesian  | "我 吃 饭" — no conjugation, no case   |
+-------------------+-------------------+----------------------------------------+
| AGGLUTINATIVE     | Turkish, Japanese,| Suffixes stack up, each with one       |
|                   | Korean, Swahili,  | meaning. Predictable. Long words.      |
|                   | Finnish, Uzbek    | Turkish: "git-me-yebil-ecek-misin?"    |
+-------------------+-------------------+----------------------------------------+
| FUSIONAL          | Russian, Latin,   | Endings bundle multiple meanings.      |
| (inflectional)    | German, Spanish,  | Latin "amo" = I + love + present +    |
|                   | Greek, Arabic     | indicative + active. Unpredictable.   |
+-------------------+-------------------+----------------------------------------+
| POLYSYNTHETIC     | Inuktitut,        | Whole sentences in one word.           |
|                   | Mohawk, Navajo    | Rare in this library's top-35.         |
+-------------------+-------------------+----------------------------------------+
```

### 3. Tonality

```
+---------------------+---------------------------+----------------------------------+
| Category            | Languages                 | Notes                            |
+---------------------+---------------------------+----------------------------------+
| TONAL (many)        | Mandarin (4+neutral),     | Pitch is phonemic — "ma" with    |
|                     | Cantonese (6-9),          | 4 different tones = 4 different  |
|                     | Vietnamese (6),           | words. Must learn tones or you   |
|                     | Thai (5),                 | will not be understood.          |
|                     | Burmese (4)               |                                  |
+---------------------+---------------------------+----------------------------------+
| PITCH-ACCENT        | Japanese,                 | Pitch contour matters for some   |
|                     | (some dialects only)      | words but not as strictly tonal. |
+---------------------+---------------------------+----------------------------------+
| NON-TONAL (most)    | All European languages,   | Pitch used for intonation/        |
|                     | Arabic, Swahili, Turkish, | emotion but not word meaning.    |
|                     | Korean, Indonesian        |                                  |
+---------------------+---------------------------+----------------------------------+
```

### 4. Grammatical Gender

```
+-------------------+---------------------------+----------------------------------+
| System            | Languages                 | Notes                            |
+-------------------+---------------------------+----------------------------------+
| BINARY M/F        | Spanish, French,          | Articles and adjectives agree    |
|                   | Portuguese, Italian,      | with noun gender. Gender of many |
|                   | Arabic                    | nouns is arbitrary.              |
+-------------------+---------------------------+----------------------------------+
| THREE-WAY M/F/N   | German, Russian,          | Der/die/das (German).            |
|                   | Polish, Latin             | Three-way adjective agreement.   |
+-------------------+---------------------------+----------------------------------+
| NOUN CLASSES      | Swahili (15 classes),     | Not "gender" exactly — classes   |

|                   | Yoruba                    | (animate, size, abstract, etc.)  |
+-------------------+---------------------------+----------------------------------+
| NO GENDER         | English, Mandarin,        | No agreement. Easier for some.   |
|                   | Japanese, Korean,         |                                  |
|                   | Turkish, Finnish,         |                                  |
|                   | Hungarian, Indonesian     |                                  |
+-------------------+---------------------------+----------------------------------+
```

### 5. Case System

```
+-------------------+---------------------------+----------------------------------+
| Cases             | Languages                 | Notes                            |
+-------------------+---------------------------+----------------------------------+
| NONE              | French, Spanish, English  | Word order carries grammatical   |
|                   | (mostly), Mandarin,       | relationships.                   |
|                   | Turkish (postpositions)   |                                  |
+-------------------+---------------------------+----------------------------------+
| 4-case            | German, Latin (actually 5)| NOM/ACC/DAT/GEN                  |
+-------------------+---------------------------+----------------------------------+
| 6-case            | Russian, Polish           | NOM/ACC/DAT/GEN/INS/PREP         |
+-------------------+---------------------------+----------------------------------+
| 7-case            | Latin (full), some Slavic | + LOCATIVE, VOCATIVE             |
```

## Scripts of the World

```
SCRIPT LANDSCAPE
================

ALPHABETS (one symbol per phoneme, vowels included)
  Latin      — English, Spanish, French, German, Portuguese, Italian,
               Polish, Romanian, Turkish, Indonesian, Vietnamese, Swahili
  Cyrillic   — Russian, Ukrainian, Bulgarian, Serbian, Kazakh, Mongolian
  Greek      — Greek
  Armenian   — Armenian
  Georgian   — Georgian

ABJADS (consonants only; vowels optional/diacritical)
  Arabic     — Arabic, Persian, Urdu, Pashto, Uyghur
  Hebrew     — Hebrew, Yiddish

ABUGIDAS (consonant+inherent vowel; other vowels as diacritics)
  Devanagari — Hindi, Marathi, Sanskrit, Nepali
  Bengali    — Bengali, Assamese
  Gurmukhi   — Punjabi
  Tamil      — Tamil
  Telugu     — Telugu
  Kannada    — Kannada
  Sinhala, Khmer, Thai, Lao, Burmese — each its own script

SYLLABARIES (one symbol per syllable)
  Hiragana   — Japanese (native Japanese words + grammar)
  Katakana   — Japanese (loanwords + emphasis)
  Hangul     — Korean (technically an "alphasyllabary" — blocks of phonemes)

LOGOGRAPHIC / MORPHO-SYLLABIC
  Chinese (Hanzi) — Mandarin, Cantonese, Classical Chinese
  Japanese (Kanji) — borrowed/adapted Chinese characters
                     Japanese uses Kanji + Hiragana + Katakana simultaneously

DIRECTIONALITY
  Left-to-right: Latin, Cyrillic, Greek, Devanagari, most Asian scripts
  Right-to-left: Arabic, Hebrew, Urdu (Nastaliq)
  Top-to-bottom: Traditional Chinese/Japanese (vertical columns), Mongolian
```

## Difficulty for English Speakers (FSI Rankings)

The U.S. Foreign Service Institute classifies languages by how many hours
of study a native English speaker needs to reach professional proficiency (ILR 3):

```
CATEGORY I (~600-750 hrs) — "Easier"
+-------------------------------+-------------------------------+
| Spanish        Italian        | Portuguese     French         |
| Romanian       Afrikaans      | Dutch          Norwegian      |
| Swedish        Danish         |                               |
+-------------------------------+-------------------------------+
Similar enough to English (or each other) that structure feels familiar.

CATEGORY II (~900 hrs) — "Moderate"
+-------------------------------+
| German         Indonesian     |
| Malay          Swahili        |
+-------------------------------+
Notable structural features but not radically alien.

CATEGORY III (~1100 hrs) — "Hard"
+-------------------------------+-------------------------------+
| Russian        Polish         | Hindi          Urdu           |
| Persian        Turkish        | Vietnamese     Thai           |
| Hungarian      Finnish        |                               |
+-------------------------------+-------------------------------+
New script or complex grammar, but Latin-alphabet or familiar structure.

CATEGORY IV / "Super-Hard" (~2200 hrs) — "Hardest"
+-------------------------------+-------------------------------+
| Mandarin       Cantonese      | Japanese       Korean         |
| Arabic         (all 4)                                        |
+-------------------------------+-------------------------------+
New script + tones OR new script + radically different grammar.
2200 hrs = ~3x longer than Spanish.
```

## Linguistic Concepts You Need

These terms appear throughout every language profile:

```
PHONOLOGY (sound system)
  phoneme    — minimal sound unit that distinguishes meaning (b vs p in English)
  allophone  — phoneme variant; [t] in "top" vs "stop" — same phoneme, different sound
  vowel      — sounds made with open vocal tract (a, e, i, o, u + more)
  consonant  — sounds made with constriction (b, d, g, s, etc.)
  tone       — pitch used to distinguish word meaning (Mandarin, Vietnamese, Thai)

MORPHOLOGY (word structure)
  morpheme   — minimal unit of meaning ("unfriendly" = un + friend + ly)
  inflection — changing a word's form for grammatical purpose (run → ran)
  derivation — making a new word (friend → friendly)
  affix      — anything attached to a root:
               prefix (un-friend), suffix (friend-ly), infix (rare in English)
  stem       — base form after stripping inflections

SYNTAX (sentence structure)
  SVO/SOV/VSO — word order shorthand
  head-final  — modifiers before head (Japanese, Korean: "red apple" → "red apple")
  head-initial — head before modifiers (VSO languages, English mostly)
  pro-drop    — languages where subject pronouns are optional (Spanish, Italian, Japanese)
  topic       — what the sentence is ABOUT (Japanese は wa, Korean 은/는)
  focus       — the new information being highlighted
  V2          — verb-second rule: verb always in second position (German, Dutch)

PRAGMATICS
  register   — formality level (honorifics in Japanese, T-V distinction in European)
  T-V        — tu/vous (or equivalent): informal vs formal 2nd-person pronoun
  speech     — Japanese has 7 politeness levels encoded grammatically
  levels
```

## Language Families: What Gets Inherited

When languages are related, learners get free vocabulary:

```
ROMANCE FAMILY DIVIDEND (if you know Spanish):
  Spanish — French: ~75% vocabulary similarity
  Spanish — Italian: ~82%
  Spanish — Portuguese: ~89%

GERMANIC FAMILY DIVIDEND (if you know English):
  English → Dutch: enormous cognate overlap (water/water, huis/house)
  English → German: ~60% cognates, but grammar diverged heavily
  English → Norwegian/Swedish: surprisingly large vocabulary

SLAVIC MUTUAL INTELLIGIBILITY:
  Russian — Ukrainian: ~85% (politically charged statement but linguistically true)
  Russian — Polish: ~60% (same script helps not at all — different alphabet)
  Czech — Slovak: ~90%

WHAT DOESN'T TRANSFER:
  Sound changes are not random — they follow Grimm's Law and others,
  so "father/Vater/pater/père" are cognates. But grammar diverged independently.
  Knowing Spanish grammar helps you with French grammar;
  knowing Spanish GRAMMAR does NOT help you with Mandarin grammar.
```

## Common Confusion Points

**"Mandarin = Chinese?"**
Chinese is a family (like Romance), not a language. Mandarin, Cantonese, Wu,
Min are mutually unintelligible dialects — they share a writing system (Hanzi)
but not spoken intelligibility. "Chinese" in this library = Standard Mandarin
(Putonghua).

**"Hindi and Urdu are the same language?"**
Spoken Hindustani is mutually intelligible (Hindi and Urdu speakers can converse).
Written Hindi uses Devanagari; written Urdu uses Nastaliq (Arabic script).
At formal registers they diverge: Hindi draws from Sanskrit, Urdu from Persian/Arabic.

**"Spanish in Spain vs Latin America?"**
Mutually intelligible. Like American vs British English, but less divergent.
Key differences: vosotros (Spain plural-informal you, absent in LatAm),
pronunciation (Spain θ sound for "c/z", LatAm seseo), some vocabulary.

**"Which Arabic do I learn?"**
MSA (Modern Standard Arabic) — used in writing, news, formal speech — no one
speaks it natively. Egyptian Arabic — most widely understood dialect (media).
Gulf Arabic — business in UAE/Saudi. Levantine — Syria/Lebanon/Jordan.
Pick MSA for reading; pick a dialect for speaking.

**"Japanese and Chinese are related?"**
Not genetically. Japanese borrowed Chinese characters (kanji) ~1500 years ago
as a writing system, then adapted them. The grammars are completely different.
Korean also borrowed Chinese characters (hanja). All three share vocabulary
derived from Classical Chinese, like European languages share Latin roots.

**"Language vs dialect?"**
The linguist's joke: "A language is a dialect with an army and a navy."
Norwegian/Danish/Swedish are mutually intelligible but counted as separate languages.
Cantonese/Mandarin are not mutually intelligible but often called "Chinese dialects."
This library uses the politically recognized standard counts.
