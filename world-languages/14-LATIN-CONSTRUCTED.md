## The Latin Case System — Cross-Language Comparison

The learner already has German (4 cases) and Russian (6 cases) from earlier files. Latin adds 6 (+ locative relic). Mapping across systems shows which cases are universal and which are language-specific innovations:

```
CASE COMPARISON: LATIN (6+1) vs GERMAN (4) vs RUSSIAN (6)

+---------------+---------+--------+--------+------------------------------------+
| Function      | Latin   | German | Russian| Notes                              |
+---------------+---------+--------+--------+------------------------------------+
| Subject       | NOM     | NOM    | NOM    | Universal — all three have this    |
| Direct object | ACC     | ACC    | ACC    | Universal — all three have this    |
| Possession/of | GEN     | GEN    | GEN    | Universal — all three have this    |
| Indirect obj  | DAT     | DAT    | DAT    | Universal — all three have this    |
| With/by/using | ABL(Lat)| (DAT)  | INS    | German merges into DAT;            |
|               |         |        |        | Latin ABL ≈ Russian INS            |
| In/at/about   | ABL     | (DAT)  | PREP   | Russian PREP = "prepositional"     |
|               | (some)  |        |        | (only after prepositions);         |
|               |         |        |        | Latin ABL carries both meanings    |
| Direct address| VOC     | (NOM)  | (NOM)  | Latin has a dedicated vocative;    |
|               |         |        |        | German/Russian use NOM for address |
| At a place    | LOC     | —      | PREP   | Latin LOC = relic for cities;      |
| (location)    | (relic) |        | (some) | Russian PREP covers this           |
+---------------+---------+--------+--------+------------------------------------+

KEY DIFFERENCES:
  Latin ABL is doing the work that BOTH German DAT and Russian INS do.
  The ablative is the "catch-all" oblique case in Latin — most prepositions
  take it. It encodes: motion-from, means/manner, accompaniment, time-when,
  price, comparison, agent (in passive constructions).

  Russian has a separate INS case for instrument/means/accompaniment
  (я пишу ручкой = I write with a pen) that Latin would put in ABL.
  German uses DAT for those functions instead.

  Latin VOC is a dedicated case for direct address (O Caesar!).
  German and Russian use the nominative for this (sie/Herr Müller!).
  The vocative survives in Polish (7-case system that adds VOC back).

PRACTICAL ANCHOR:
  If you know German: Latin adds VOC and splits DAT into DAT+ABL.
  If you know Russian: Latin merges INS+PREP into ABL; adds VOC.
  The 4-case core (NOM/ACC/GEN/DAT) is shared across all three.
```

### Five Declensions

```
LATIN NOUN DECLENSIONS (5 paradigms)
=======================================
Declension identified by genitive singular ending.

FIRST DECLENSION (-ae) — mostly feminine:
  porta (gate): porta / portam / portae / portae / porta / porta (NOM/ACC/GEN/DAT/ABL/VOC)
  Plural:       portae / portas / portarum / portis / portis / portae
  Exception masc: agricola (farmer), nauta (sailor)

SECOND DECLENSION (-i) — mostly masculine + neuter:
  MASC: servus (slave): servus / servum / servi / servo / servo / serve
  Plural: servi / servos / servorum / servis / servis / servi
  NEUT: bellum (war): bellum / bellum / belli / bello / bello / bellum
  (neuter: NOM=ACC=VOC, plural NOM/ACC/VOC all -a)

THIRD DECLENSION (varied) — all genders:
  NOM singular unpredictable: rex (king), miles (soldier), corpus (body)
  Stem revealed in genitive: rex → reg-is, miles → milit-is, corpus → corpor-is
  MASC: rex / regem / regis / regi / rege / rex
  Plural: reges / reges / regum / regibus / regibus / reges
  NEUT: corpus / corpus / corporis / corpori / corpore / corpus
  Plural: corpora / corpora / corporum / corporibus / corporibus / corpora

FOURTH DECLENSION (-us, mostly masc):
  manus (hand): manus / manum / manus / manui / manu / manus
  Plural: manus / manus / manuum / manibus / manibus / manus

FIFTH DECLENSION (-ei, mostly fem):
  res (thing/matter): res / rem / rei / rei / re / res
  Plural: res / res / rerum / rebus / rebus / res
  dies (day, masc/fem): dies / diem / diei / diei / die / dies
```

### Four Conjugations

```
LATIN VERB CONJUGATIONS (4 paradigms)
=======================================
Identified by infinitive ending: -āre / -ēre / -ere / -īre

PRESENT ACTIVE INDICATIVE:
+---------+-----------+----------+-----------+----------+
| Person  | I: amāre  | II: monēre| III: dūcere| IV: audīre|
+---------+-----------+----------+-----------+----------+
| 1sg     | amō       | moneō    | dūcō      | audiō    |
| 2sg     | amās      | monēs    | dūcis     | audīs    |
| 3sg     | amat      | monet    | dūcit     | audit    |
| 1pl     | amāmus    | monēmus  | dūcimus   | audīmus  |
| 2pl     | amātis    | monētis  | dūcitis   | audītis  |
| 3pl     | amant     | monent   | dūcunt    | audiunt  |
+---------+-----------+----------+-----------+----------+

KEY PRINCIPLE: Latin encodes WHO in the ending → subject pronoun usually omitted.
  amō = "I love" (no separate "ego" needed)
  amant = "they love" (no separate "eī" needed)

SUM (to be) — the essential irregular:
  sum / es / est / sumus / estis / sunt
  eram / erās / erat / erāmus / erātis / erant (imperfect)
  erō / eris / erit / erimus / eritis / erunt (future)

PRINCIPAL PARTS — must memorize for every verb:
  amō / amāre / amāvī / amātum
  (1sg pres / infinitive / 1sg perfect / perfect passive participle)
  Principal parts give the stems for all tenses.
```

### The Ablative Absolute

```
ABLATIVE ABSOLUTE — Latin's signature construction
====================================================
A noun + participle in the ablative case, grammatically independent
from the main clause, expressing time, cause, or circumstance.

  Caesare interfecto, Brutus fugit.
  Caesar-ABL killed-ABL, Brutus fled.
  "With Caesar having been killed, Brutus fled."
  OR: "After Caesar was killed, Brutus fled."
  OR: "Caesar having been killed, Brutus fled."

  Sole oriente, milites progredi coeperunt.
  Sun-ABL rising-ABL, soldiers to march began.
  "As the sun was rising, the soldiers began to march."

  No equivalent in English — requires translation as subordinate clause.
  Extremely common in classical Latin prose (Tacitus, Caesar, Livy).
```

### The 16 Rules (Complete Grammar of Esperanto)

```
ZAMENHOF'S 16 RULES — the entire Esperanto grammar
====================================================
1.  No articles (only "la" = the; "a/an" = none)
2.  Nouns end in -o: libro (book), homo (person)
3.  Adjectives end in -a: bela (beautiful), granda (big)
4.  Numbers don't decline; cardinal numerals invariable
5.  Personal pronouns: mi, vi, li, ŝi, ĝi, ni, vi, ili (and ri for non-binary)
6.  Verbs don't conjugate for person/number:
    mi estas = I am, vi estas = you are, ili estas = they are
7.  Adverbs end in -e: bele (beautifully), rapide (quickly)
8.  All prepositions take nominative (no case after prepositions)
9.  Words spelled as pronounced; stress on penultimate syllable
10. Accusative: -n suffix marks direct object (allows free word order)
11. Prefixes/suffixes with consistent meaning
12. Compound words: just add roots together
13. If "not A" then "mal-A": bona (good) → malbona (bad)
14. All nouns have the same plural: -j (librojlibro+j)
15. (Accusative details — redundant with 10)
16. Foreign letters ĉ ĝ ĥ ĵ ŝ ŭ for sounds not in basic Latin

VERB TENSES — 3 only:
  Present: -as   Mi manĝas. (I eat.)
  Past:    -is   Mi manĝis. (I ate.)
  Future:  -os   Mi manĝos. (I will eat.)
  Conditional: -us   Mi manĝus. (I would eat.)
  Infinitive: -i   manĝi (to eat)
  Participles: active -ant- / -int- / -ont-; passive -at- / -it- / -ot-
```

### Esperanto's Word Formation System

```
PRODUCTIVE MORPHOLOGY — build huge vocabulary from roots
=========================================================
Key prefixes:
  mal-  opposite: bona (good) → malbona (bad), granda → malgranda
  re-   again: veni (come) → reveni (return)
  ge-   both genders: patro (father) / patrino (mother) → gepatroj (parents)
  eks-  former: prezidento → eksprezidento
  mis-  wrongly: kompreni (understand) → miskompreni (misunderstand)
  dis-  scatter: doni (give) → disdoni (distribute)
  bo-   in-law: frato (brother) → bofrato (brother-in-law)

Key suffixes:
  -in   female: kato (cat) → katino (female cat), sinjoro → sinjorino (Ms.)
  -et   small/diminutive: domo (house) → dometo (little house)
  -eg   large/augmentative: vento (wind) → ventego (storm)
  -ej   place of: lerni (learn) → lernejo (school, place of learning)
  -ist  practitioner: meti (work craft) → metiisto (craftsman)
  -aĵ   thing characterized by: manĝi (eat) → manĝaĵo (food, thing to eat)
  -ul   person characterized by: riĉa (rich) → riĉulo (rich person)
  -ar   collection: arbo (tree) → arbaro (forest, collection of trees)
  -an   member of: Ameriko → Amerikano (American)
  -ism/-ist: socialismo, socialisto (socialism, socialist)

EXAMPLE — from root "vid" (see):
  vidi (to see), vido (sight), vidaĵo (sight/scenery), videjo (lookout point)
  vidinda (worth seeing), vidisto (one who sees), revidi (to see again)
  supervidi (to oversee), antaŭvidi (to foresee), previdi (to foresee)
```

## Language Endangerment

**Why this section is in the Latin/Esperanto file:** Zamenhof created Esperanto (1887) in direct response to the language diversity and associated ethnic conflict in 19th-century eastern Europe. One of Esperanto's foundational arguments is that a politically neutral constructed language removes the cultural domination inherent in using a natural national language as a lingua franca — no nation's language wins. The endangerment crisis is the same problem at the other end: linguistic diversity is collapsing under economic and media pressure from dominant languages, the same pressure that drove Zamenhof's project. Constructed languages and endangered natural languages are two sides of the same question: what is the value of linguistic diversity, and who bears the cost of maintaining it?

**Latin as a case study:** Latin did not go extinct — it transformed. Vulgar Latin differentiated into the Romance languages through a process that took ~600 years (roughly 400–1000 CE). No speaker experienced "the death of Latin." What died was the prestige written form, not the spoken vernacular. This is the most common mode of language "death": the spoken form continues, now called something else. Modern endangered languages may follow the same path — the linguistic heritage survives in the descendant language even after the name disappears.

```
THE GLOBAL LANGUAGE CRISIS
============================

  ~7,000 languages exist today
  ~3,000 are endangered (UNESCO atlas)
  1 language dies approximately every 2 weeks

ENDANGERMENT CLASSIFICATIONS (UNESCO):
  VULNERABLE:     Most children speak it but restricted domains
  DEFINITELY:     Children no longer learn at home
  SEVERELY:       Only grandparents + oldest generation; not daily use
  CRITICALLY:     Youngest speakers are grandparents; not used in daily life
  EXTINCT:        No speakers remaining

CURRENT STATUS:
  ~40%  of world's languages have <1000 speakers
  ~23   languages account for half the world's population
  Top 10 languages: >85% of internet content
  Languages with <100 speakers: Ubykh (extinct 1992), Eyak (extinct 2008),
  Chamicuro (Peru, ~8 speakers), Njerep (Nigeria, ~4 speakers)

WHY LANGUAGES DIE:
  Economic pressure: dominant language = better employment
  Education in national language only
  Urbanization: speakers move to cities
  Political suppression (historical: residential schools, colonial policies)
  Media/internet in dominant languages
  Parents choose to raise children in dominant language

NOTABLE CASES:
  Irish Gaelic: ~170,000 native speakers in Ireland despite 5M+ total Irish
                Gaeltacht areas protected by law; all schoolchildren learn Irish
                but English dominates economic life
  Welsh:        ~700,000 speakers; active revival; Welsh-medium schools growing
                S4C (Welsh TV), signage bilingual in Wales
  Hawaiian:     Down to ~2,000 native speakers in 1970s; revival programs;
                Hawaiian immersion schools (Pūnana Leo); now ~24,000 speakers
  Māori:        Similar trajectory; now ~150,000 speakers in New Zealand;
                te reo Māori taught in schools; revival ongoing
  Hebrew:       The ONLY successful case of a dead language becoming a
                full living language with native speakers (20th century revival).
                ~9M native speakers today.

DOCUMENTATION EFFORTS:
  ELDP (Endangered Languages Documentation Programme)
  Rosetta Project: multilingual archive
  ELAR: Endangered Languages Archive
  SIL International: linguistic field work
  Ethnologue: catalog of all known languages
  AILLA, PARADISEC: regional audio archives

WHAT'S BEING LOST:
  Not just words — cognitive frameworks.
  Many endangered languages encode:
    - Plant/animal knowledge unavailable elsewhere (indigenous pharmacology)
    - Spatial reasoning systems (absolute vs relative directions)
    - Time perception systems (cyclic vs linear)
    - Social structures and kinship systems
    - Historical oral records

PIRAHÃ — the case that challenged Chomsky's universal grammar:
  ~200-400 speakers (Amazon, Brazil)
  No embedded clauses (no "he said that she thought that...")
  No numbers / no concept of exact number
  No color terms (only light/dark)
  No past/future tense
  No recursion in the Chomskyan sense (debated)
  Daniel Everett's fieldwork sparked major linguistics debates.
```

## Common Confusion Points

**"Latin is dead — why learn it?"**
Latin isn't used for conversation, but it's very much alive as a reading
language. All of medieval science, philosophy, theology, law, and diplomacy
is in Latin. Botanical/zoological naming is Latin. Medical anatomy is Latin.
Legal maxims are Latin. If you want to read Aquinas, Calvin, Newton's Principia,
or Linnaeus in original — it's Latin.

**"Esperanto failed — isn't it dead?"**
Esperanto is the most successful constructed language by far, with ~2M active
speakers, annual congresses, literature, and a genuine community. The question
is whether it achieved Zamenhof's goal of becoming a universal second language —
it hasn't. But it's a living language with native speakers and real cultural
production.

**"Why do languages die if speakers want to preserve them?"**
Individual preferences and community outcomes diverge. Even if a parent
"wants" to preserve a language, if children can't use it with peers,
can't get a job with it, can't access media in it — they won't use it.
Language transmission requires an intergenerational community, not just
individual will. Welsh is succeeding partly because of institutional support
(schools, TV channel, government services) that creates a viable domain.

**"Shouldn't all languages merge into English?"**
Cognitive diversity argument: different languages encode different conceptual
structures. Guugu Yimithirr (Australia) uses absolute cardinal directions
(north/south/east/west) where English uses relative (left/right/behind).
Native speakers have measurably better spatial orientation. Pirahã's lack of
recursion and number words may reflect a genuinely different cognitive mode.
Losing languages may be losing cognitive frameworks with no equivalent translation.
