# Hindi–Urdu — Language Reference

## Profile at a Glance

```
+--------------------------------------------------------------------+
| HINDI / URDU (हिन्दी / اردو) — Hindustani                           |
+--------------------------------------------------------------------+
| Family:      Indo-European → Indo-Iranian → Indic                 |
| Native spkrs: Hindi ~344M + Urdu ~92M = ~436M Hindustani          |
| Total spkrs:  ~600M (including L2)                                 |
| Countries:   India (Hindi official), Pakistan (Urdu official)      |
|              Widely spoken: Bangladesh, Nepal, diaspora worldwide  |
| Word order:  SOV (Subject-Object-Verb) — strict                   |
| Morphology:  Fusional                                              |
| Tonal:       No                                                    |
| Gender:      Binary (M/F)                                          |
| Script:      Hindi → Devanagari; Urdu → Nastaliq (Arabic-derived) |
| Cases:       2 direct/oblique + postpositions carry case meaning   |
| FSI level:   Category III — ~1100 hours (harder for Urdu script)   |
+--------------------------------------------------------------------+

HINDI vs URDU:
  Spoken Hindustani (everyday speech) is essentially the same language.
  A native Hindi and Urdu speaker can fully understand each other in conversation.
  DIVERGENCE at formal registers:
    Hindi formal → Sanskrit-derived vocabulary
    Urdu formal → Persian/Arabic-derived vocabulary
  SCRIPT:
    Hindi → Devanagari (left-to-right)
    Urdu → Nastaliq calligraphic script (right-to-left, Arabic-derived)
  Political: Hindi is national language of India; Urdu is of Pakistan.
```

## Devanagari Script

```
DEVANAGARI — The script of Hindi (and Sanskrit, Marathi, Nepali)
=================================================================
Type: Abugida — each symbol is a consonant with inherent 'a' vowel.
      Vowels other than inherent 'a' are written as diacritical marks.
Direction: Left to right, top bar (शिरोरेखा shirorēkhā) connects letters.

INDEPENDENT VOWELS (used at start of word or after another vowel):
  अ [a]  आ [ā]  इ [i]  ई [ī]  उ [u]  ऊ [ū]
  ए [e]  ऐ [ai] ओ [o]  औ [au]  ऋ [ṛ]

CONSONANTS with inherent 'a' vowel:
Row 1 (velar):   क [ka]  ख [kha]  ग [ga]  घ [gha]  ङ [ṅa]
Row 2 (palatal): च [ca]  छ [cha]  ज [ja]  झ [jha]  ञ [ña]
Row 3 (retroflex):ट [ṭa] ठ [ṭha] ड [ḍa]  ढ [ḍha]  ण [ṇa]
Row 4 (dental):  त [ta]  थ [tha]  द [da]  ध [dha]  न [na]
Row 5 (labial):  प [pa]  फ [pha]  ब [ba]  भ [bha]  म [ma]
Semi-vowels:     य [ya]  र [ra]   ल [la]  व [va/wa]
Sibilants:       श [śa]  ष [ṣa]   स [sa]  ह [ha]

VOWEL DIACRITICS (shown on क ka):
  का [kā]  कि [ki]  की [kī]  कु [ku]  कू [kū]
  के [ke]  कै [kai] को [ko]  कौ [kau]
  क् [k]  — halant removes the inherent 'a' (consonant-only form)

CONJUNCT CONSONANTS — when two consonants touch (no vowel between):
  क + ष = क्ष  [kṣa] — common in Sanskrit loanwords
  त + र = त्र  [tra]
  ज + ञ = ज्ञ  [jña/gna]
  These are new characters, not obvious from parts.

ASPIRATED vs UNASPIRATED — Hindi distinguishes these phonemically:
  क ka vs ख kha   ग ga vs घ gha   प pa vs फ pha   ब ba vs भ bha
  English speakers hear "ka" and "kha" as the same — they are not.
  "Kha" has a puff of air; "ka" doesn't.
```

## Sound System

```
HINDI PHONOLOGY — CHALLENGES FOR ENGLISH SPEAKERS
===================================================

RETROFLEX CONSONANTS — tongue curled back to palate:
  ट ड ण ठ ढ — vs dental counterparts त द न
  Hindi "ṭ" (retroflex) vs "t" (dental) — phonemically distinct.
  English has only one "t". Hindi has two.
  "पत" vs "पट" — different words.

ASPIRATED vs UNASPIRATED — phonemic in Hindi, allophonic in English:
  English "p" in "pin" is aspirated [pʰ] — same as Hindi फ.
  English "p" in "spin" is unaspirated [p] — same as Hindi प.
  In English these are allophones (same phoneme in different environments).
  In Hindi they are DIFFERENT PHONEMES (change word meaning).

NASALIZED VOWELS — marked with chandrabindu (ँ):
  हाँ [hã] — Yes (nasal ā)

SCHWA DELETION:
  Hindi written schwa (inherent a) is often silent in speech.
  "भारत" is written "bhārata" but pronounced "bhārat".
  Patterns are mostly systematic but must be learned.
```

## Grammar Core

### Direct and Oblique Cases

```
HINDI CASE SYSTEM — simplified but with postpositions
======================================================
Two main forms: DIRECT (subject, direct object) and OBLIQUE (before postposition)

Postpositions (after nouns, unlike English prepositions):
  को  ko   — to, for (marks indirect object or animate DO)
  का  kā   — of (possession, also kī/ke for agreement)
  में  mẽ   — in
  पर  par  — on
  से  se   — from, by, with, than
  तक  tak  — until, up to
  के लिए ke liye — for the purpose of

OBLIQUE case:
  Direct:   लड़का (laṛkā, boy)    → Oblique: लड़के (laṛke)
  Direct:   लड़की (laṛkī, girl)   → Oblique: लड़की (unchanged)
  Direct:   घर (ghar, house)     → Oblique: घर (unchanged)
  Direct:   मेज़ (mez, table)     → Oblique: मेज़ (unchanged)

"The boy" = लड़का laṛkā (direct)
"to the boy" = लड़के को laṛke ko (oblique + ko postposition)
"of the boy" = लड़के का laṛke kā (oblique + kā)
```

### Gender and Agreement

```
GRAMMATICAL GENDER — and its cascade of agreement
===================================================
Masculine: typically ends in -ā in direct singular
  लड़का laṛkā (boy), घर ghar (house, invariable), आदमी ādmī (man)

Feminine: typically ends in -ī or -ī/i
  लड़की laṛkī (girl), किताब kitāb (book, invariable)

AGREEMENT cascades to:
  1. Adjectives: बड़ा (m) / बड़ी (f) / बड़े (m.plur) = big
  2. Verbs (in perfective): agree with subject gender/number
  3. Postposition का: kā (m) / kī (f) / ke (m.plur)

Example:
  बड़ा लड़का गया। (baṛā laṛkā gayā.) — The big boy went. (m.sing)
  बड़ी लड़की गई। (baṛī laṛkī gaī.) — The big girl went. (f.sing)
  बड़े लड़के गए। (baṛe laṛke gae.) — The big boys went. (m.plur)
```

### The Agreement Dispatch Bridge

Hindi-Urdu uses **split ergativity** — two different agreement dispatch rules depending on aspect × transitivity. The split is not an irregularity; it is a systematic two-case type system:

```
AGREEMENT DISPATCH TABLE:

                    INTRANSITIVE verb          TRANSITIVE verb
                    (no object)                (has an object)

IMPERFECTIVE/     NOMINATIVE-ACCUSATIVE:     NOMINATIVE-ACCUSATIVE:
HABITUAL          Subject takes direct case.  Subject takes direct case.
(non-perfective)  Verb agrees with SUBJECT.   Verb agrees with SUBJECT.

                  राम जाता है।               राम किताब पढ़ता है।
                  Ram.DIR go.M.SG be.SG       Ram.DIR book.DIR read.M.SG be.SG
                  Verb agrees with Ram (M.SG) Verb agrees with Ram (M.SG)

PERFECTIVE        NOMINATIVE-ACCUSATIVE:     ERGATIVE CONSTRUCTION:
(completed)       Subject takes direct case.  Subject takes ने (ne) postposition
                  Verb agrees with SUBJECT.   → treated as oblique/ergative.
                                             Verb agrees with OBJECT gender/number.

                  राम गया।                   राम ने किताब पढ़ी।
                  Ram.DIR go.M.SG             Ram.ERG book.F.SG read.F.SG
                  Verb agrees with Ram        Verb agrees with किताब (F.SG),
                                             NOT with Ram

TYPE-DISPATCH ANALOGY:
  dispatch(agreement) {
    if (aspect == PERFECTIVE && verb.transitive) {
      // ergative mode: object drives agreement
      verb.agreeWith(object.gender, object.number);
      subject.case = ERGATIVE; // marked with ने
    } else {
      // default nominative-accusative mode: subject drives
      verb.agreeWith(subject.gender, subject.number);
      subject.case = NOMINATIVE; // direct case
    }
  }

  The dispatch condition is determined COMPOSITIONALLY at runtime:
  same verb, same subject, different aspect → different agreement rule.
```

**Why "split ergativity":** The language is nominative-accusative in most contexts (the typologically common pattern: subject always gets nominative, verb agrees with subject) but switches to ergative-absolutive in perfective transitive contexts (the typologically rarer pattern: the agent of a transitive verb gets marked, and the verb agrees with the patient instead). The "split" is the trigger condition: aspect × transitivity.

**The structural motivation:** In perfective constructions, the verb form is historically a passive participle that agrees with its grammatical subject — which is the patient/object. The ergative marker ने originated as a postposition marking the causative agent of this passive-like construction. The synchronic agreement rule follows from the diachronic origin, but you don't need to know the history to apply the rule — you just need the dispatch table above.

```
VERB STRUCTURE — ergative split is the hardest part
=====================================================
INFINITIVE ends in -nā: जाना (jānā, to go), खाना (khānā, to eat)

IMPERFECTIVE PRESENT (habitual/ongoing — auxiliary from होना):
  मैं जाता हूँ।  (main jātā hūm) — I go / I am going. (m.speaker)
  मैं जाती हूँ।  (main jātī hūm) — I go / I am going. (f.speaker)
  Verb agrees with SUBJECT gender

SOV + AUXILIARY:
  Stem + aspect suffix + auxiliary verb
  जा + ता + हूँ = जाता हूँ (go + habitual + am = I habitually go)
  जा + रहा + हूँ = जा रहा हूँ (go + progressive + am = I am going)

ERGATIVE CONSTRUCTION — perfective transitive:
  In perfective (past completed) with transitive verb:
  → Subject takes ने (ne) postposition (oblique)
  → Verb agrees with the OBJECT, not the subject

  राम ने किताब पढ़ी। (Rām ne kitāb paṛhī.) — Ram read the book.
  [Ram-ERG book-FEM read-FEM]
  Verb पढ़ी agrees with किताब (kitāb, fem), NOT with राम.

  राम ने किताबें पढ़ीं। — Ram read books.
  [Ram-ERG books-FEM.PLUR read-FEM.PLUR]

  This is "split ergativity" — only perfective transitive triggers ergative.
  Intransitive past: subject takes direct case.
  राम गया। (Rām gayā.) — Ram went. (direct case, verb agrees with Ram)
```

### Pronouns and Honorifics

```
HINDI HONORIFIC PRONOUNS — three levels
=========================================
  तू  tū   — intimate/rude (very close OR insulting)
  तुम tum  — informal (friends, younger people, peers)
  आप  āp   — formal/polite (elders, strangers, respectful)

  Each uses different verb forms.
  Using तू with someone incorrectly is an insult.
  Default: आप for strangers, तुम for friends.

  आप verb forms are plural (respectful plural) even for one person.
  आप जाते हैं। (āp jāte haim.) — You go. (formal)
  तुम जाते हो। (tum jāte ho.) — You go. (informal)
  तू जाता है। (tū jātā hai.) — You go. (intimate)
```

## Urdu-Specific Notes

```
URDU DIFFERENCES
================

SCRIPT: Nastaliq — cursive Arabic-derived, right-to-left
  More letters than Arabic (added for Indic sounds):
  ٹ (ṭ — retroflex T)    ڈ (ḍ — retroflex D)    ڑ (ṛ — retroflex R)
  پ (p)   چ (ch)   ژ (zh)   ک (k)   گ (g)

VOCABULARY AT FORMAL REGISTERS:
  Hindi: matr̥bhūmi (motherland) — Sanskrit
  Urdu: watan (motherland) — Arabic/Persian

  Hindi: dhanyavād (thank you) — Sanskrit
  Urdu: shukriyā (thank you) — Arabic shukr

  Hindi: vidyālaya (school) — Sanskrit
  Urdu: maktab / skūl — Arabic / English loanword

AT CONVERSATIONAL LEVEL: completely interchangeable.
  Script is the main barrier for literacy, not speech.
```

## Decision Cheat Sheet

| Situation | Hindi/Urdu approach |
|-----------|-------------------|
| Learning script | Devanagari for Hindi: systematic, ~3-4 weeks to read. Nastaliq for Urdu: harder, cursive. |
| Hardest grammar | Ergative split in perfective tense — verb agrees with object, not subject |
| Second hardest | Honorific pronoun system (3 levels, each with different verb forms) |
| SOV adjustment | Always end sentences with the verb: "I coffee want" not "I want coffee" |

| Pakistan | Urdu is official; regional languages (Punjabi, Sindhi, Pashto) dominate locally |

## Common Confusion Points

**"Ergative construction — verb agrees with object?"**
Yes. In perfective transitive: मैंने खाना खाया (I ate food) — खाया (khāyā)
agrees with खाना (khānā, food, masculine), not with मैं (I).
Switch to a feminine object: मैंने रोटी खाई (I ate bread) — खाई (khāī, fem).

**"Two words for 'tomorrow' and 'yesterday'?"**
कल (kal) = both yesterday AND tomorrow. Context determines meaning.
"कल मैं गया" (yesterday I went — past verb)
"कल मैं जाऊँगा" (tomorrow I will go — future verb)

**"Yes is हाँ hāṃ not हा hā"**
The nasal anusvara (ṃ) is part of the word. Without it: हा (hā) sounds wrong.
Also yes = जी हाँ (jī hāṃ) more formal; जी (jī) alone = respectful yes.

**"Postpositions not prepositions"**
Everything is flipped from English:
  English: IN the house → Hindi: घर में ghar mẽ (house IN)
  English: TO Ram → Hindi: राम को Rām ko (Ram TO)
  The noun comes first, then the "preposition" (= postposition).

**"Aspirated/retroflex contrasts"**
काम kām (work) vs खाम khām (envelope) vs घाम ghām (sweat) — all different.
These are phonemic — changing aspiration changes the word.
