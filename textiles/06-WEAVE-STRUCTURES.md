# 06 — Weave Structures

## Plain, Twill, Satin, Pile, Jacquard — The Grammar of Woven Fabric

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Three fundamental weave structures (all other weaves are variations or combinations):
  1. **Plain weave**: every warp thread alternates over/under successive weft threads; simplest interlacing; maximum number of intersections per square inch → firm, stable, non-stretchy; both faces identical
  2. **Twill weave**: each warp thread floats over 2+ weft threads before going under, with each row offset by one → diagonal rib visible on face; stronger than plain for equivalent yarn weight (longer floats allow closer sett → more threads per inch); can have face-up warp (warp-faced twill) or face-up weft (weft-faced twill)
  3. **Satin weave**: very long floats (warp threads float over 4-7 weft threads); minimum interlacing → smooth, lustrous surface; the floats catch and reflect light; weak (floats snag easily); satin = warp-faced (silk/polyester), sateen = weft-faced (cotton)
- **Plain weave fabrics**:
  - Muslin: basic plain-weave cotton, unbleached; used for toile/mockup garments in pattern development
  - Calico: plain-weave cotton, usually printed; historically specific reference to unbleached Indian cotton
  - Poplin: plain weave, fine warp + coarser weft → slight rib; shirting fabric
  - Taffeta: plain weave, fine, stiff; silk or synthetic; used in evening wear, linings
  - Chiffon: plain weave, sheer, loosely twisted yarn → soft drape
  - Organdy: crisp, stiff, plain weave cotton (starch or resin finish)
  - Voile: similar to chiffon but cotton; sheer curtains
- **Twill weave fabrics**:
  - Denim: 3×1 warp-faced twill; cotton; indigo-dyed warp (blue) + undyed weft (white) → worn denim shows white; rigid denim vs stretch denim (2-4% spandex); selvedge (selvage) denim = woven on narrower shuttle loom, finished edges, premium
  - Gabardine: 2×2 twill, tightly woven, smooth face; wool gabardine = Burberry's trademark waterproof coat fabric (Thomas Burberry, 1879); very durable, water-resistant
  - Herringbone: twill with direction reversal at regular intervals → V-shaped zigzag pattern resembling fish skeleton; traditional suiting and outerwear
  - Tweed: rough, loosely woven twill, multicolored yarns; Harris Tweed PDO; Scottish Highlands
  - Serge: smooth-faced twill (2×2); classic suit fabric; "surge" in French (surface)
  - Flannel: loosely woven twill then napped (brushed) on one or both sides → soft, fuzzy; wool or cotton flannel
- **Satin weave fabrics**:
  - Silk satin: classic luxury; 8-harness satin typical for fashion; slides, doesn't grip → linings, lingerie, evening wear
  - Polyester satin: ubiquitous affordable alternative; more common in fashion than silk satin today
  - Sateen: cotton weft-faced satin; softer than cotton poplin; bedsheets marketed as "sateen"; 4/1 or 5/1 weft float
  - Charmeuse: lightweight silk or poly satin; very soft drape; one face satin, one crêpe
- **Pile weaves** (additional yarn loops or cuts above ground fabric surface):
  - Velvet: cut pile; warp pile loops cut to create dense, plush surface; silk velvet = most luxurious; Genoa velvet historical center; cotton velvet cheaper; "velour" = knitted equivalent; direction of pile reflects light differently → "nap" direction important for cutting
  - Corduroy: cut pile in ridges (cords); wales = ridges per inch (fine = 18-21 wale, standard = 11-14, wide = 6-8, elephant/ultra-wide = 1-3); cotton; casual wear
  - Terry cloth: uncut pile loops; cotton; towels, bath robes; loops absorb water; double-sided loops (both faces) vs single-sided
  - Velveteen: weft-pile (vs velvet's warp-pile); cotton; less expensive than velvet; less depth of pile
- **Jacquard weave** (Joseph-Marie Jacquard, 1804): punch card-controlled harness allows each warp thread to be individually controlled → complex patterns; Jacquard loom → Babbage Analytical Engine → modern computer (Babbage explicitly noted the connection; Jacquard punch cards directly inspired Hollerith's census cards → IBM); fabrics: brocade (raised pattern, supplementary wefts), damask (single-color, self-pattern from satin/sateen contrast), tapestry, matelassé
- **Dobby weave**: simpler than Jacquard — geometric patterns but not individual thread control; small repeating geometric textures; oxford cloth (2×1 basket variation), piqué (raised texture)
- **Thread count**: threads per inch in warp + weft; higher thread count = tighter weave if same yarn weight; but misused in bedding marketing — 1000+ thread count usually achieved by counting ply as separate threads; 200-400 thread count in good quality single-ply yarn = excellent; "ultra-high thread count" often inferior to moderate thread count single-ply
