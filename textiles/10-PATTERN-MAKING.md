# Pattern Making

## Flat Pattern, Draping, Grading — From Body to Blueprint

## The Big Picture

```
GARMENT DESIGN → WEARABLE OBJECT: The Transformation Chain

  Body measurements
  + Ease allowances
  + Design intent
          ↓
  PATTERN (flat paper/card templates)
          ↓
  FABRIC CUTTING (pattern pieces → fabric panels)
          ↓
  ASSEMBLY (panels sewn in sequence)
          ↓
  3D GARMENT that fits a body

  THE CORE PROBLEM:
    Human body = 3D compound-curved surface
    Woven fabric = 2D flat sheet (no inherent stretch)
    SOLUTION: Darts + seams + ease encode 3D shape into flat pieces

  THREE ROUTES TO A PATTERN:
  ┌──────────────────────────────────────────────────────────────┐
  │  FLAT PATTERN DRAFTING                                       │
  │  Mathematical construction from measurements                 │
  │  Tools: rulers, French curves, set squares                   │
  │  Starting point: sloper (basic block, minimum ease)          │
  │  Reproducible, scalable, industrial standard                 │
  ├──────────────────────────────────────────────────────────────┤
  │  DRAPING (MOULAGE)                                           │
  │  Fabric draped and pinned on dress form                      │
  │  Marks transferred to paper                                  │
  │  Intuitive for complex 3D shapes (cowls, bias)               │
  │  Couture standard; reveals how fabric "wants" to behave      │
  ├──────────────────────────────────────────────────────────────┤
  │  COMMERCIAL PATTERN ADJUSTMENT                               │
  │  Start from existing commercial pattern                      │
  │  Baste toile → fit → adjust → re-cut                        │
  │  Common for home sewers                                      │
  └──────────────────────────────────────────────────────────────┘
```

## Ease: The Critical Concept

```
EASE = garment measurement MINUS body measurement

  Without ease: fabric on body like a second skin
  With wearing ease: minimum needed to move normally
  With design ease: additional for style/silhouette

  WEARING EASE GUIDELINES (approximate):
  ┌─────────────────────────────────────────────┐
  │  Garment area    │  Wearing ease            │
  ├─────────────────────────────────────────────┤
  │  Chest/Bust      │  2-4" (minimum for movement) │
  │  Waist           │  1" (or waistband/stretch) │
  │  Hip             │  2-3"                    │
  │  Sleeve/armhole  │  1-2"                    │
  └─────────────────────────────────────────────┘

  DESIGN EASE (added for silhouette):
    Fitted jacket:     +3-4" at chest
    Semi-fitted shirt: +4-5" at chest
    Boxy/oversized:    +10-20" at chest
    Traditional:       negative ease (compression)
      → stretch fabric required (knits, elastane blends)

  NEGATIVE EASE:
    Garment SMALLER than body
    Requires stretch fabric (jersey, lycra blends)
    Swimwear, activewear, shapewear: -1" to -4" at chest
```

## The Sloper (Block)

```
  SLOPER = basic fitted pattern for a given size
    → No design details
    → No seam allowances
    → Minimum wearing ease ONLY
    → Starting point for ALL designs

  STANDARD SLOPER SET:
  ┌────────────────────────────────────────────────────┐
  │  Bodice front  │  Bodice back                      │
  │  Sleeve        │  (one piece — rotated for design) │
  │  Skirt front   │  Skirt back                       │
  │  Trouser front │  Trouser back                     │
  └────────────────────────────────────────────────────┘

  SLOPERS ARE NOT INTERCHANGEABLE BETWEEN PEOPLE:
    A sloper is fitted to ONE body (or one standard size)
    Commercial patterns: drafted to a standard size body
    (which may not match your actual body proportions)

  DRAFTING A SLOPER:
    1. Take body measurements (bust/waist/hip/back length/
       armhole depth/shoulder width/sleeve length...)
    2. Apply drafting formulas (e.g., bust circumference/2 +
       1.5" = half-chest measurement on pattern)
    3. Add wearing ease per garment type
    4. Draw block using rulers and curved rulers
    5. Verify by making toile (muslin mockup) → adjust

  COMMERCIAL BLOCK SOURCES:
    Home patterns (Vogue, Butterick): include ease,
      seam allowances; must back-calculate to true sloper
    Patternmaking textbooks (Armstrong, Aldrich):
      provide sloper drafting formulas
    Custom sloper services: measured to individual body
```

## Pattern Pieces and Markings

```
  WHAT A PATTERN PIECE ENCODES:

  Grain line ────────────────────
    Must align with warp (lengthwise grain) of woven fabric
    Bias cut: 45° to grain line → drape, stretch, diagonal
    Cross grain: 90° to grain = weft direction
    Wrong grain = garment twists, hangs incorrectly

  Notches ─── ▲ ▲ ───
    Triangular cuts at seam edges
    Matching marks for aligning pieces when sewing
    Single notch: front of seam/armhole
    Double notch: back
    Critical for orientation (sleeve front vs back, etc.)

  Darts ──────◇──────
    Wedge-shaped tucks
    Encode 3D shaping (see Dart Manipulation below)

  Seam allowance (SA)
    The extra fabric beyond the stitching line
    US home sewing standard: 5/8" (15 mm)
    Industrial/European standard: 1 cm (10 mm)
    Couture: varies, sometimes none on sloper, added for toile
    Pattern markings often show both stitching line and cut line

  Other marks:
    "Place on fold" → fold of fabric (pattern = half piece)
    Buttonhole positions
    Pocket placement
    Ease indicators (sleeve cap ease marks)
    Matching stripes/plaids: registration marks
```

## Dart Manipulation

```
  DART = wedge-shaped tuck that shapes flat fabric to 3D curve

  FUNDAMENTAL PRINCIPLE: Any dart can be moved to ANY position
  around its pivot point without changing the fit.
  The dart intake (the wedge amount) stays the same.

  PIVOT POINT for bust dart = apex (bust point)

  ┌────────────────────────────────────────────────────────┐
  │  DART ROTATION PRINCIPLE                               │
  │                                                        │
  │  Original:  shoulder dart ──→ apex ←── side dart      │
  │                                                        │
  │  Can become:                                           │
  │    Waist dart (at waistline)                           │
  │    Underarm dart (at underarm seam)                    │
  │    French dart (diagonal from hip to apex)             │
  │    Neck dart (at neckline)                             │
  │    Armhole dart (at armhole)                           │
  │    Gathers at any seam (distribute dart as fullness)   │
  │    Princess seam (dart becomes curved seam)            │
  │                                                        │
  │  ALL EQUIVALENT: same fit, different style lines       │
  └────────────────────────────────────────────────────────┘

  DART DISTRIBUTION:
    Full dart intake → one dart: prominent point
    Dart split in two → two smaller darts: softer shaping
    Dart converted to gathers: soft, romantic look
    Dart converted to pintucks: structured pleats
    Princess seam: entire dart length into seamline → smooth
```

## Flat Pattern Drafting: Process

```
  STEP 1: Start from sloper (correct size or closest + adjust)

  STEP 2: Establish design lines
    Trace sloper onto fresh paper
    Draw new seam lines, style lines
    Mark where design elements change the shape

  STEP 3: Apply ease
    Add design ease beyond wearing ease already in sloper
    Mark ease shaping (e.g., sleeve cap ease = extra length
    that creates smooth, rounded cap when eased in)

  STEP 4: Add seam allowances
    After shaping is correct, add SA to all cut edges
    Corners require "squaring off" SA so they join correctly

  STEP 5: Add all markings
    Grain lines, notches, dart legs, fold lines, match points

  STEP 6: Make toile (muslin mockup)
    Cut in calico/muslin
    Baste seams (removable)
    Fit on body or dress form → pin adjust → mark corrections
    Transfer corrections back to paper pattern

  STEP 7: Cut final pattern pieces
    Clean up all adjustments
    Verify seam lengths match (sleeve head = armhole + ease)
    Test once more if significant changes

  KEY TOOLS:
    Ruler: straight seams, grainlines
    Curved ruler / French curve: armhole, neck curves
    Hip curve: large gentle curves (hip seams, princess lines)
    Awl: pierce pattern at pivot points for dart rotation
    Notcher: cuts 1/8" V notches cleanly
```

## Draping

```
  DRAPING = building pattern directly on dress form in fabric

  THE PROCESS:
  1. Prepare fabric (pin grain lines: pull thread to find true
     grain; press; mark center front/back grain lines)
  2. Begin with front bodice — pin to form center front
  3. Work fabric over form's curves, pinning and trimming
  4. Use hands to smooth, shape; let fabric tell you where
     seams want to go
  5. Mark all seams, darts, style lines with marking pencil
     directly on fabric
  6. Unpin from form; true up (straighten seam lines with ruler)
  7. Transfer to paper; add seam allowances; check symmetry

  WHY USE DRAPING:
    Complex 3D shapes that are hard to draft mathematically:
      Cowl necklines (fabric drapes in specific folds)
      Asymmetric designs
      Draped/wrapped garments
      Bias cut (Madeleine Vionnet's method)
    "Finds" designs that emerge from the fabric's behavior
    Direct feedback on fabric drape, weight, hand
    Couture standard: Balenciaga, Madame Grès worked by draping

  DRESS FORM QUALITY MATTERS:
    Standard size form vs custom adjusted (pad/reduce) to body
    Different forms for different garment types (strapless, etc.)
    Adjustable forms: move measurements within range
```

## Grading: Size Scaling

```
  GRADING = scaling a pattern up and down through a size range

  THIS IS NOT SIMPLE ENLARGEMENT:
    Human body proportions are not linear across sizes
    A size 16 is NOT a size 8 scaled up by a constant factor
    Width:length:height ratios all change at different rates
    Shoulder width, armhole depth, torso length all grade
      differently from chest/waist/hip

  GRADE RULES:
    Tables of precise measurement changes per size per pattern
    piece at specific "grade points" (corners, curve changes)
    Each piece has specific x/y movement at each grade point
    Standard grade rules vary by style category:
      Fitted dress vs casual T-shirt vs tailored jacket
    Grade rules are proprietary and experience-based

  SIZE SYSTEMS:
    Misses (US): 0-16 typically; 1-inch grade between sizes
      (most measurements except shoulder/back length)
    Plus sizes: separate grade rules, different size range
    Petite/Tall: length adjustments + full resize
    Men's: chest/neck/waist/inseam sizing; more direct
    Children's: age-based but body proportion variance huge

  CAD GRADING:
    Computer grading (Gerber AccuMark, Lectra) applies
    grade rules automatically
    Nest of sizes displayed on screen; verify shape progression
    Key check: sleeve head ease must remain consistent at each size
```

## Marker Making

```
  MARKER = arrangement of all pattern pieces for one garment
           (all sizes if cutting multiple) on fabric width

  PURPOSE: minimize fabric waste
  Fabric is expensive → marker efficiency directly = profit/loss

  ┌──────────────────────────────────────────────────────────┐
  │  FABRIC WIDTH (typically 44-45" or 58-60")              │
  │  ┌──────────────────────────────────────────────────┐   │
  │  │  [bodice front] [bodice back]  [sleeve] [collar]│   │
  │  │  [skirt F] [skirt B]   [pkt] [pocket] [band]  │   │
  │  └──────────────────────────────────────────────────┘   │
  │  ←─────────── marker length ──────────────────────→     │
  └──────────────────────────────────────────────────────────┘

  CONSTRAINTS:
    Grain lines: all pieces must sit on correct grain
    Nap: one-way fabric (velvet, corduroy) → all pieces same direction
    Pattern matching: stripes/plaids → pieces must register across seams
    Symmetry: paired pieces (sleeve, trouser leg) — both directions or
              fold-on-fold

  EFFICIENCY METRICS:
    Fabric utilization: (fabric in garment) / (total fabric)
    Industry target: 85-87% or better
    1% improvement in marker efficiency on 100,000 garments = significant
    "Fallout" = fabric wasted between pieces

  COMPUTERIZED MARKER MAKING:
    Gerber AccuMark, Lectra: drag-and-drop with automatic nesting
    Auto-nesting algorithms: test millions of arrangements
    Digital markers sent directly to computerized cutting tables
    Manual marker making: skilled marker maker with paper and scissors,
      hand-laying pieces — still done in small operations
```

```
  ├──────────────────────────────────────────────────────────┤

  │              │ European companies    │ integration       │

  │              │ growing rapidly       │ avatar fitting    │
  ├──────────────────────────────────────────────────────────┤
  │  Optitex     │ Mid-market            │ 2D/3D             │
  └──────────────────────────────────────────────────────────┘

  3D VIRTUAL FITTING (CLO3D, Browzwear, Optitex):
    Digital avatar (body scan data or standard sizes)
    Pattern imported → stitched virtually → simulated on body
    Physics simulation: fabric weight, drape, bending rigidity
    See problems before cutting: collar rolling wrong, seat
    pulling, shoulder seam shifting
    Reduces physical samples needed (was: 4-8 per season)
    Sending virtual samples for buyer approval: adopted by many brands
    Still not fully replacing physical samples (color, texture, hand)
```

## Toile and Fitting

```
  TOILE (US: "muslin") = prototype garment in cheap fabric
    Calico, cotton muslin, gingham (for pattern matching)
    Made BEFORE cutting expensive fashion fabric

  WHY TOILE:
    A $3 worth of calico catches a $200 mistake
    Reveals what no flat pattern can: how fabric behaves on
    a moving body vs a static dress form

  TOILE FITTING PROCESS:
  1. Baste (long, removable stitches) all seams — no machine sewing
  2. Put on body (dress form or fit model)
  3. Identify issues:
     - Pulling lines (too tight → let out seams)
     - Excess fabric bunching (too loose → take in)
     - Grain off (piece shifting → redraft)
     - Neckline gaping (too wide → reduce)
     - Armhole restricting (too small → enlarge)
  4. Pin corrections with tissue pins on the body
  5. Remove, mark corrections clearly
  6. Transfer back to flat paper pattern
  7. Make second toile if changes substantial

  TOILE CONVENTIONS:
    Horizontal balance lines: mark with fabric marker to check
      level/tilt while on body
    Grain marking: mark true grain on toile pieces to verify
    Never cut fashion fabric without a successful toile
    (exception: very simple or previously proven pattern)
```

## Made-to-Measure vs Bespoke

```
  MADE-TO-MEASURE (MTM):
    Standard graded pattern + adjustments for individual measurements
    One to two fittings
    Factory-made but fit-adjusted
    Price: $300-$1,500+ for suit
    Brands: modern luxury shirts/suits (Proper Cloth, Indochino,
             Samuelsohn)

  BESPOKE:
    Pattern drafted specifically for the individual
    Multiple hand-basted fittings:
      1st fitting: basted canvas (structural) + canvas shell
      2nd fitting: forward progress — sleeves in, lapels shaped
      3rd fitting: near-finished garment check
      Final delivery
    Entire pattern is YOUR pattern, stored for future orders
    Savile Row bespoke: 50-80 hours of work on one suit
    Price: $3,000-$15,000+ (Savile Row)

  TAILORING TRADITIONS:
  ┌─────────────────────────────────────────────────────────┐
  │  SAVILE ROW (English):                                  │
  │  Structured, padded chest, roped sleeve head            │
  │  Formal, upright silhouette                             │
  │  Hard shoulders, firm construction                      │
  │  Anderson & Sheppard, Huntsman, Henry Poole             │
  ├─────────────────────────────────────────────────────────┤
  │  NEAPOLITAN (Italian):                                  │
  │  Soft, unpadded construction                            │
  │  Floating canvas, minimal interlining                   │
  │  Suit drapes with body movement                         │
  │  Kiton, Isaia, Cesare Attolini                          │
  ├─────────────────────────────────────────────────────────┤
  │  FLORENTINE/MILANESE (Italian):                         │
  │  Middle ground                                          │
  │  Slightly structured but lighter than English           │
  │  Brioni, Ermenegildo Zegna                              │
  └─────────────────────────────────────────────────────────┘
```

## Standard Sizing Problems

```
  US SIZES NOT STANDARDIZED:
    "Size 8" at Zara ≠ "Size 8" at Levi's ≠ "Size 8" at J.Crew
    No legal requirement to follow any measurement standard

  VANITY SIZING DRIFT (over decades):
    1950 "size 12" body measurements → today's "size 6"
    Size labels got smaller as garment dimensions grew
    Marketing finding: customers prefer smaller label

  AVAILABLE STANDARDS (largely ignored):
    ASTM D5585: Women's standard body measurements
    ASTM D6240: Men's standard body measurements
    ISO 8559: International body measurements
    EN 13402: European size labeling (mandatory labeling
              but not mandatory measurement standard)

  SOLUTION: MEASURE, NOT SIZE LABELS
    Buy by body measurements, not arbitrary size number
    Pattern-making convention: specify body measurements and
    ease preferences separately
    Brands that publish actual garment measurements:
      slowly becoming more common (e.g., Proper Cloth,
      made-to-measure online)

  JAPANESE SIZING:
    JIS L 4004/4005: standardized and actually used
    Height × chest/bust → size designation
    More reliable than US but Japanese body proportions
    assumed
```

## Decision Cheat Sheet

| Task | Method |
|---|---|
| Industrial production, consistent sizes | Flat pattern + CAD grading + digital marker |
| Couture, complex draped/asymmetric design | Draping on form |
| Home sewing from commercial pattern | Commercial pattern + toile fitting |
| Fix commercial pattern for your body | Toile + standard adjustment techniques |
| Reduce sampling cycles | CLO3D virtual fitting before physical sample |
| New design from scratch, fitted | Sloper → dart manipulation → add ease → toile |
| Plaid/stripe matching | Marker making with registration marks |
| Below-the-waist fit only | Skirt or trouser sloper from waist/hip measurements |

## Common Confusion Points

**Ease is not sloppiness — it's engineering**: A well-drafted pattern has precise ease. Wearing ease is the minimum physics of motion; design ease is intent. When a vintage pattern "runs small," it's often because ease standards have shifted (1950s patterns had minimal ease).

**Slopers have no seam allowances — commercial patterns do**: When adapting commercial patterns back to slopers, you must remove seam allowances first. Pattern adjustment is done on the stitching line, not the cut line.

**Dart rotation doesn't change fit**: Moving a dart to a different position is purely aesthetic. The bust dart at the shoulder and the French dart fit identically — the dart intake is the same wedge of fabric. Only the visible location changes.

**Grading is not scaling**: A size 18 pattern is not a size 10 scaled up by 140%. Chest grades at a different rate than shoulder width, which grades differently from armhole depth. Proportional scale distorts the entire fit structure.

**Seam lengths must match**: When constructing a pattern, every seam that will be joined must match in length (plus any intended ease, e.g., sleeve cap ease). Checking that seam lengths "marry up" is a fundamental pattern verification step before cutting.
