# Fabric Construction

## Weaving, Knitting, Felting, Nonwovens — How Yarn Becomes Cloth

## The Big Picture: Four Fundamentally Different Methods

```
+------------------------------------------------------------------+
|              FABRIC CONSTRUCTION METHODS                         |
+------------------------------------------------------------------+
|                                                                  |
|  WEAVING               KNITTING            FELTING              |
|  =======               ========            =======              |
|  Two yarn systems      One yarn system      No yarn system       |
|  interlaced at 90°     looped into chains  Fibers entangled     |
|                                            mechanically         |
|  Warp + weft           Weft knitting:      Wet felting          |
|                          horizontal loops   Needle felting       |
|  Dimensionally stable  Warp knitting:                           |
|  (no inherent stretch)   vertical chains   Cut without fraying  |
|                                            Limited elasticity   |
|  Frays when cut        Ladders when cut    Excellent insulation |
|  Must be hemmed        (weft knitting)                          |
|                                                                  |
|  NONWOVENS             (a 4th method — no yarn, no interlacing) |
|  =========             Fibers bonded: thermally, chemically,    |
|                        mechanically, or hydroentangled          |
|                        Disposables: diapers, masks, wipes       |
+------------------------------------------------------------------+
```

The choice of construction method determines fabric *properties* (stretch, stability, drape) as fundamentally as the fiber choice. A polyester woven fabric and a polyester knit fabric behave entirely differently.

## Weaving

### The Loom Mechanics

```
WARP: yarns running lengthwise (parallel to selvage)
WEFT: yarns interlacing crosswise (perpendicular to warp)
      Also called: "filling" in industrial terminology

LOOM ANATOMY:
  ┌──────────────────────────────────────────────────────────┐
  │  BEAM (warp beam)                                        │
  │  Wound with warp threads at back of loom                 │
  │  Feeds warp under tension toward cloth beam              │
  ├──────────────────────────────────────────────────────────┤
  │  HEDDLES + HARNESSES                                      │
  │  Each warp thread passes through eye of a heddle         │
  │  Heddles grouped in harnesses (frames)                   │
  │  Harnesses rise/fall to create SHED                      │
  ├──────────────────────────────────────────────────────────┤
  │  SHED: the opening between raised and lowered warp       │
  │  Shuttle (or rapier/air jet) passes weft through shed    │
  ├──────────────────────────────────────────────────────────┤
  │  REED (beater): comb that beats weft into previous picks │
  │  Also spaces warp threads evenly                         │
  ├──────────────────────────────────────────────────────────┤
  │  CLOTH BEAM: takes up woven fabric as it is produced     │
  └──────────────────────────────────────────────────────────┘

  SEQUENCE (one weft pick):
  1. Harnesses raise/lower → SHED opens
  2. Shuttle/rapier passes weft through shed
  3. REED beats weft into fell line (previous pick)
  4. Harnesses change position → shed closes + opens new
  5. Repeat
```

### Loom Types: Historical to Modern

```
BACKSTRAP LOOM (~10,000 BCE, still used):
  Warp: one end tied to fixed post, other end to weaver's waist
  Body tension controls warp tension
  Portable — used for fine ceremonial weaving in Andean/Southeast Asia
  No machine — entire weave structure controlled by hand

WARP-WEIGHTED LOOM (ancient Greece/Scandinavia):
  Warp threads hang vertically
  Weights at bottom maintain tension
  Woven from top down; shed stick and heddle rod for shed
  Evidence from Greek vase paintings; archaeological weight finds

FLOOR LOOM (treadle/shaft loom):
  Standard hand-weaving loom today
  4-16+ harnesses controlled by foot treadles
  Plain weave: 2 harnesses
  Complex twill/pattern: 4-16 harnesses (one per pick sequence step)
  Jacquard: individual harness control for each warp thread

POWER LOOM (Cartwright, 1785):
  Steam/water-powered version of shaft loom
  Displaced handweavers 1810-1840 (Luddite movement)
  Mechanized weaving; same principles, automated

MODERN INDUSTRIAL WEAVING:
  Shuttle-less looms (rapier, air-jet, water-jet, projectile)
  Air-jet: compressed air carries weft → 1,500 picks/min
  Water-jet: water jet (only for hydrophobic yarns like polyester)
  Rapier: flexible tape carries weft — versatile for fancy yarns
```

### Fabric Structure

```
SELVEDGE: self-finished edge where weft turns back
          This is why woven fabric frays when cut interior —
          no selvedge locking the yarns

GRAIN LINE: the direction of warp threads
            Cutting on-grain = parallel to warp or weft
            Cutting on-bias = 45° to grain = maximum stretch direction

THREAD COUNT: warp threads + weft threads per square inch
              Higher thread count → tighter, finer weave
              Marketing abuse: 800TC uses 2-ply yarns counted as 2
              Practical limit: ~400TC for single-ply fine weave

GSM (grams per square meter):
  Weight measurement of finished fabric
  Lightweight: 60–150 GSM (voile, lawn, chiffon)
  Midweight: 150–300 GSM (poplin, denim, jersey)
  Heavyweight: 300+ GSM (canvas, upholstery, technical fabrics)
```

## Knitting

### Weft Knitting: Horizontal Loops

```
MECHANISM:
  Single yarn fed into row of needles
  Each needle forms a loop (stitch) by pulling new yarn through previous loop
  All stitches in one row (course) interlink
  Fabric grows row by row

  WHY IT STRETCHES:
    Loops can slide along the yarn → extends without yarn stretching
    Woven fabric cannot do this (warp/weft are fixed)

  STITCH TYPES:
    KNIT stitch (V on face, bump on back):
      Needle pulls new loop through old loop FROM FRONT
    PURL stitch (bump on face, V on back):
      Needle pulls new loop through old loop FROM BACK
    Combinations produce all textured patterns

  COMMON STRUCTURES:
    Jersey (single knit): all knit one face, all purl other
                          T-shirt fabric; curls at edges
    Rib: alternating knit/purl columns
         Stretches in both directions; doesn't curl
    Interlock: two rib structures mirror-knitted together
               Both faces smooth; heavier; less stretch than rib
    Piqué: tuck stitches create raised texture
           Polo shirt fabric
```

### Weft Knitting Machines

```
  CIRCULAR KNITTING MACHINE:
    Needles arranged in a cylinder
    Feeds in all directions simultaneously → seamless tube
    T-shirt blanks, sock blanks, jersey fabric
    Speeds: up to 100 rpm, 1,200+ needles/cylinder

  FLAT BED MACHINE:
    Needles in two flat beds, V-shaped
    Can knit shaped pieces (decrease/increase)
    Full-fashioned garments (shaped to body)
    More expensive per piece; less waste vs cut-and-sew

  HAND KNITTING:
    Two needles; all structures achievable
    Slow: ~15 stitches/min (expert)
    Machine: ~1000 stitches/min
```

### Warp Knitting: Vertical Chains

```
  KEY DIFFERENCE FROM WEFT KNITTING:
    Many yarns run simultaneously in vertical direction
    Each yarn forms its own chain while interlinking with adjacent yarns

  MACHINES:
    Tricot: finer fabrics (lingerie, dress fabrics, gloves)
    Raschel: wider range of constructions (lace, nets, power mesh)

  PROPERTIES:
    More dimensionally stable than weft knitting (doesn't ladder)
    Less stretch than weft knitting
    Can produce complex patterns without Jacquard mechanism
    High production speed

  USES:
    Lace (Rachel lace = majority of commercial lace)
    Lingerie/swimwear fabrics
    Velcro loop component
    Net curtains, mosquito nets
    Geotextile grid fabrics
    Sports mesh fabrics
```

## Felting

### The Scale Mechanism

```
  WOOL FIBER SURFACE:
    ┌─────────────────────────────┐
    │  ← Root end   Tip end →    │
    │  \\\\\\\\\\\\\\\\\\\\\\\\  │
    │   \\\\\\\\\\\\\\\\\\\\\\\  │
    │    \\\\\\\\\\\\\\\\\\\\\\  │
    │  Overlapping scales like   │
    │  roof shingles, pointing   │
    │  toward tip                │
    └─────────────────────────────┘

  FELTING CONDITIONS:
    1. Heat: swells fiber, opens scales
    2. Water: lubricates scales
    3. Mechanical agitation: causes relative fiber motion

  RATCHET EFFECT:
    Scales catch on adjacent scales
    When fiber moves: can only move one direction (toward root)
    Fibers migrate, interlocking with neighbors
    → Irreversible entanglement
    → Fabric shrinks and thickens

  RESULT: Dense, non-fraying fabric
    No yarn — raw fibers bonded mechanically
    Insulating (dense air-trap structure)
    Cut any direction without fraying (no interlocked yarns to unravel)
    Does NOT stretch (tangled = locked)
```

### Felting Methods

```
  WET FELTING:
    Traditional:  hot soapy water + hand agitation (rubbing, rolling)
    Industrial:   fulling mill (water-powered or motor hammers)
                  milling machine (washing-machine-like agitation)

  NEEDLE FELTING:
    Barbed needles punch repeatedly through fiber batt
    Barbs catch fibers and drag them through neighboring fibers
    → Mechanical entanglement without water
    Industrial: automotive interiors, carpet underlays, filters
    Craft: sculptural felting, felt pictures, fiber art

  CONTROLLED FULLING OF WOVEN WOOL:
    Woven wool cloth put through controlled milling/fulling
    Slight intentional felting → denser, softer cloth
    "Boiled wool" = heavily fulled → very dense, insulating
    Melton cloth, walkcoat (overcloat) wool — all heavily fulled
    Historical: Walker, Tucker, Fuller = surname from fulling trade
```

## Nonwoven Fabrics

```
+------------------------------------------------------------------+
|  NONWOVEN = FIBER → WEB → BOND (no yarn formation, no weaving)   |
+------------------------------------------------------------------+

STEP 1: WEB FORMATION
  Dry-laid: carded fiber web (like start of spinning, but not spun)
  Wet-laid: fibers dispersed in water, deposited on screen
            (similar to papermaking — nonwovens and paper are cousins)
  Spunmelt: polymer extruded directly into fibers → laid as web

STEP 2: WEB BONDING
  ┌────────────────────────────────────────────────────────────────┐
  │ Thermal bonding:                                               │
  │   Heat melts binder fibers (often BiCo — bicomponent)        │
  │   BiCo: PE/PP sheath/core; sheath melts at lower temp         │
  │   Core provides structure; sheath bonds fibers together        │
  │                                                                │
  │ Chemical bonding:                                              │
  │   Latex or acrylic binder sprayed onto web then cured          │
  │   "Resin bonded" → stiffer, less drapeable                    │
  │                                                                │
  │ Hydroentanglement (spunlace):                                  │
  │   High-pressure water jets (100+ bar) punch through web        │
  │   Physically entangle fibers around each other                 │
  │   No binder — soft, cloth-like feel                           │
  │   Used: wet wipes, medical drapes, industrial wipes            │
  │                                                                │
  │ Needle punch:                                                  │
  │   Same principle as needle felting                             │
  │   Industrial: carpet backing, geotextiles, automotive          │
  └────────────────────────────────────────────────────────────────┘
```

### SMS: The N95 Mask Structure

```
  S — SPUNBOND (structural layer)
  M — MELTBLOWN (filtration layer)
  S — SPUNBOND (structural layer)

  SPUNBOND:
    Polymer (PP) → melt extrusion → filaments → mechanical bonding
    Filaments: 15–35 μm diameter
    Strong, durable, tear-resistant
    Not good filtration (too coarse)

  MELTBLOWN:
    Molten polymer extruded into high-velocity hot air stream
    Air draws out extremely fine fibers: 0.5–10 μm diameter
    Randomly laid on collector → web
    Excellent filtration — fibers so fine they block submicron particles
    MECHANICALLY WEAK — would tear in use alone

  SMS COMPOSITE:
    Spunbond gives mechanical strength
    Meltblown provides filtration
    Outer spunbond protects the delicate meltblown layer
    N95 mask = SMS + electrostatic charging of meltblown layer
    COVID-19 pandemic: global SMS meltblown shortage in 2020
    (capacity limited; only a few manufacturers globally)
```

## Bias Cutting

```
  WOVEN FABRIC HAS MAXIMUM STRETCH AT 45° (bias):

  ON-GRAIN: warp or weft direction
    Warp: minimal stretch (warp taut on loom)
    Weft: slightly more stretch than warp
    Both: yarns packed perpendicular → no give

  BIAS (45° to grain):
    Square lattice of intersecting warp/weft can
    shear → squares become rhombuses → STRETCH
    Without any fiber elongation, the geometry changes

  MADELEINE VIONNET (1910s-1920s):
    Realized bias cutting allowed woven fabric to drape
    and cling like knit → revolutionary fashion technique
    Bias-cut gowns followed body curves without elastic

  PRACTICAL IMPLICATIONS:
    More fabric required (diagonal waste at edges)
    Harder to sew (fabric stretches during stitching)
    Crepe de Chine, charmeuse, silk jersey often bias-cut
    Stripes/patterns create diagonal when cut on bias (V-shape)
```

## Decision Cheat Sheet

| I want a fabric that... | Construction |
|---|---|
| Has stretch (2-way) | Weft knit |
| Is dimensionally stable | Woven |
| Can be cut in any direction | Nonwoven or felted |
| Drapes well from body bias | Woven, bias cut |
| Is disposable and cheap | Nonwoven (spunbond) |
| Filters air/liquid | Nonwoven meltblown |
| Provides maximum insulation | Felted wool or needle-punched fleece |
| Makes lace | Warp knit (Raschel) |
| Won't ladder if snagged | Warp knit or woven (not weft knit) |

## Common Confusion Points

**Weft knit runs/ladders; warp knit doesn't**: In weft knitting, all loops in a row are connected sequentially from one yarn — if one loop breaks, it can drop through the row. In warp knitting, loops interlink from multiple yarns — no single failure can propagate.

**"French terry" is a knit, not a woven**: French terry = looped pile on inside, smooth jersey outside. Sweatshirt fabric. Not a weaving technique — it's a weft-knit structure. "Terry cloth" (bath towels) is a woven pile structure. Same name family, different constructions.

**Nonwoven is not felt**: Felt uses wool's scale structure; nonwovens use chemical/thermal/mechanical bonding. Nonwovens can be made from any fiber — polyester spunbond is nonwoven but not felt. Felt specifically means fiber-entanglement by scale-ratchet mechanism.

**Bias cut doesn't add stretch**: It redistributes existing geometric give. A woven fabric has zero elasticity in warp/weft directions (yarns don't stretch). At 45°, the rectangular grid can shear (change angle) → apparent stretch. Adding spandex gives true elastic recovery; bias cut gives structural drape.
