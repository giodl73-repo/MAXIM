# Yarn & Spinning

## Drop Spindle to Ring Frame — Twist, Count, and the Spinning Revolution

## The Big Picture: What Spinning Does

```
SPINNING = DRAFTING + TWISTING

  Fiber roving (loose bundle)
       |
       ↓ DRAFTING (attenuation)
  Thinner, longer bundle of aligned fibers
       |
       ↓ TWISTING
  Fibers locked together by inter-fiber friction
       |
       ↓
  YARN — a continuous strand with tensile strength

  WITHOUT TWIST: fibers slide past each other → no strength
  WITH TWIST:    fibers held under compression by their own
                 spiral path → friction creates strength

  Think of twist as: transforming a pile of straws (weak,
  slides apart) into a rope (strong, compressive clamping)
```

## Twist Direction: S vs Z

```
  S-TWIST:          Z-TWIST:
      \                   /
     \ \                 / /
    \ \ \               / / /
   \ \ \ \             / / / /
    \ \ \ \           / / / /
     \ \ \           / / /
      \ \           / /

  When held vertical, the spiral matches:
    - Middle diagonal of "S" → S-twist
    - Middle diagonal of "Z" → Z-twist

  CONVENTION:
    Singles typically: Z-twist (right-hand twist)
    Plied yarn typically: S-twist (opposite to singles)
    Plying in opposite direction → balanced yarn (no residual torque)

  CONSEQUENCES:
    S-plied Z-spun yarn → dimensionally stable, flat
    Z-plied Z-spun yarn → over-twisted, kinky, bias in fabric
    Mixing twist directions in warp/weft → diagonal tension in fabric
    Some design effects use this deliberately (crepe fabrics)
```

## Spinning Technology Timeline

### Drop Spindle (~20,000 BCE – Present)

```
  ANATOMY:
    ┌───────────────────────────────┐
    │  Distaff (holds unspun fiber) │ ←← optional
    │        |                      │
    │  Spindle shaft (stick)        │
    │        |                      │
    │  Whorl (weight disk) at       │
    │  top, middle, or bottom       │
    └───────────────────────────────┘

  OPERATION:
    1. Attach leader thread to shaft
    2. Draft fiber from distaff or hand
    3. Set spindle spinning (fingers or thigh)
    4. Spindle falls under gravity while spinning
    5. Twist travels up drafted fiber as spindle drops
    6. Wind spun yarn onto spindle shaft
    7. Repeat

  LIMITATIONS:
    Spinner must stop to wind yarn
    Speed limited by spindle weight and arm length
    ~1 lb/day of medium-weight yarn for skilled spinner

  ALL PRE-INDUSTRIAL TEXTILES were made from spindle-spun yarn
  Including: Egyptian linen (pyramids), Roman wool, Chinese silk
  The drop spindle was sufficient for civilization for 20,000 years
```

### Spinning Wheel (~1000 CE → Industrial Revolution)

```
  GREAT WHEEL (Walking Wheel, ~1000 CE):
    Large driving wheel turned by hand
    Drive band rotates horizontal spindle
    Spinner drafts fiber and walks backward while spinning
    Then walks forward while winding
    Speed: ~3× drop spindle
    Still: intermittent process (draft then wind)

  SAXONY WHEEL (Treadle Wheel, ~1480 CE):
    Treadle (foot pedal) → crank → drive wheel
    Flyer and bobbin mechanism: CONTINUOUS spinning
    Flyer adds twist; bobbin takes up yarn simultaneously
    Both hands free for drafting
    Speed: ~5-10× drop spindle
    This is the archetypal "spinning wheel" in fairy tales

  FLYER-BOBBIN MECHANISM:
    ┌────────────────────────────────────┐
    │  U-shaped flyer                    │
    │    rotates at slightly different   │
    │    speed from bobbin               │
    │                                    │
    │  Yarn path: through orifice →      │
    │  around flyer arm → onto bobbin    │
    │                                    │
    │  Twist added in orifice-to-flyer   │
    │  section; wind-on at speed        │
    │  differential between flyer/bobbin │
    └────────────────────────────────────┘
```

### The Industrial Revolution: Spinning Machinery

```
CHRONOLOGY:
  1764  Spinning jenny (Hargreaves)
  1769  Water frame (Arkwright)
  1779  Spinning mule (Crompton) = jenny + water frame
  1828  Ring spinning frame (Thorp/Jenks)
  1960s Open-end/rotor spinning

──────────────────────────────────────────────────────────────

SPINNING JENNY (Hargreaves, ~1764):
  Multiple spindles (8 → 80 → 120) operated by one person
  Hand-operated (no water power — cottage industry)
  Spinning only, no automatic drafting
  Enabled massive increase in spindle count per worker
  Hargreaves' cottage attacked by textile workers fearing job loss

WATER FRAME (Arkwright, 1769):
  Roller drafting (not hand drafting)
  Rollers moving at different speeds stretch the roving
  Water-powered (required factory installation)
  Produced STRONGER, harder-twisted yarn than jenny
  (suitable for warp thread — jenny yarn too soft for warp)
  This is why Arkwright built factories: scale + water power
  Patent fights: Arkwright's monopoly bitterly contested

SPINNING MULE (Crompton, 1779):
  Combined: jenny's multiple spindles + Arkwright's roller drafting
  Intermittent action: carriage moves out (drafting) → returns (winding)
  Produced finer, stronger yarn than either machine alone
  Fine Muslim cloth became possible (Dacca muslin recreation attempts)
  Crompton made almost nothing from it — didn't patent properly

RING SPINNING (1828 → dominant through 20th century):
  Continuous process (vs mule's reciprocating carriage)
  Ring: circular rail with C-shaped traveler
  Traveler rides around ring, adding twist as yarn winds onto bobbin
  Traveler speed slightly less than bobbin → differential = twist
  Dominant spinning technology globally from 1890s–present
  Still dominant for fine cotton, wool worsted, synthetic

──────────────────────────────────────────────────────────────

OPEN-END / ROTOR SPINNING (1963, development from ~1960s):
  DIFFERENT PRINCIPLE: no continuous fiber strand
  ┌────────────────────────────────────────────────────────────┐
  │  1. Fiber bundle combed to individual fibers (opening)     │
  │  2. Individual fibers fed to rotating rotor (50,000+ RPM)  │
  │  3. Centrifugal force collects fibers on rotor groove wall │
  │  4. Yarn peeled off rotor through doffing tube             │
  │  5. Twist inserted by rotor rotation                       │
  └────────────────────────────────────────────────────────────┘

  ADVANTAGES vs ring:
    5–10× faster (rotor at 100,000 RPM possible)
    No spindles to doff (continuous output)
    Shorter fiber lengths OK (lower quality raw material)

  DISADVANTAGES vs ring:
    Coarser yarn (can't spin below ~Ne 40s easily)
    Slightly weaker yarn (fibers not as well aligned)
    Less smooth (more hairy)

  USE:
    Denim yarns (rotor-spun cotton)
    Knitting yarns, toweling, fleece fabrics
    ~30% of world's spun yarn is rotor-spun
```

## Yarn Count Systems: The Confusion Explained

```
  TWO INCOMPATIBLE CONVENTIONS:
  ┌──────────────────────────────────────────────────────────────┐
  │  INDIRECT (higher number = FINER yarn)                       │
  │  "How many [length units] per [weight unit]?"               │
  │                                                              │
  │  Ne (English Cotton Count):                                  │
  │    # of 840-yd hanks per pound                              │
  │    Ne 30 = 30 × 840 yd = 25,200 yd per pound               │
  │    Ne 100 = much finer (100 hanks per pound)                │
  │    Used for: cotton, cotton blends, spun synthetic           │
  │                                                              │
  │  Nm (Metric Count):                                          │
  │    # of 1000-m lengths per kg                               │
  │    Nm 30 = 30,000 m per kg                                  │
  │    Used for: worsted wool, synthetics (metric countries)    │
  │                                                              │
  │  Worsted Count (Nw):                                         │
  │    # of 560-yd hanks per pound                              │
  │    Used for: combed wool (worsted system)                   │
  └──────────────────────────────────────────────────────────────┘

  ┌──────────────────────────────────────────────────────────────┐
  │  DIRECT (higher number = THICKER yarn)                       │
  │  "How many [weight units] per [length unit]?"               │
  │                                                              │
  │  Tex:                                                        │
  │    grams per 1000 meters (SI system)                        │
  │    Tex 20 = lighter yarn; Tex 50 = heavier                  │
  │    Used in: technical textiles, ISO standards               │
  │                                                              │
  │  Denier (D or den):                                          │
  │    grams per 9000 meters                                    │
  │    30D = fine (stockings); 70D = heavier; 210D = nylon pack │
  │    Used for: filament yarns (nylon, polyester, silk)        │
  │    Note: 9000m chosen as standard because a 1D silk         │
  │    filament weighs 1 gram per 9000 m — historical           │
  └──────────────────────────────────────────────────────────────┘

  CONVERSIONS:
  tex = 590.5 / Ne
  tex = 1000 / Nm
  denier = tex × 9
  Ne × denier ≈ 5315

  PRACTICAL REFERENCE:
  Ne 10s = coarse (denim fabric weight cotton)
  Ne 20s = medium (standard t-shirt weight)
  Ne 40s = fine (dress shirt fabric)
  Ne 80s = very fine (fine voile/lawn)
  30 denier = sheer hosiery (stockings)
  70 denier = light nylon (thin outdoor)
  420 denier = heavy duty nylon (backpacks)
```

## Plying and Yarn Structure

```
  SINGLES: spun from drafted fiber; has residual torque
  PLY: multiple singles twisted together in opposite direction
       → residual torques cancel → balanced, stable yarn

  PLY NOTATION (knitting/weaving):
    2-ply: 2 singles twisted together
    4-ply: 4 singles (fingering weight in hand knitting)
    "2-ply cashmere" ≠ thin yarn — it means 2 strands; could be bulky

  HAND-KNITTING WEIGHT SYSTEM (US/UK):
  ┌───────────────────────────────────────────┐
  │  Weight     WPI*   Needle   Examples       │
  │  Lace       30+    0-1 mm   Cobweb wool   │
  │  Fingering  14-30  1.5-2.5  Sock yarn     │
  │  Sport      12-14  2.5-3.5  Baby weight   │
  │  DK         11-14  3.5-4.5  Double-knit   │
  │  Worsted    8-11   4.5-5.5  Standard      │
  │  Bulky      5-8    5.5-8 mm Heavy knit    │
  │  Super bulky < 5   8+ mm    Arm knitting  │
  │  *WPI = wraps per inch around a ruler     │
  └───────────────────────────────────────────┘

  NOTE: "Worsted" in knitting = a yarn weight (medium)
  "Worsted" in spinning = a spinning system (combed, parallel fibers)
  These are DIFFERENT uses of the same word (confusing by design,
  apparently)
```

## Specialty Yarn Constructions

```
  CORE-SPUN:
    Spandex/elastane core + cotton or polyester outer fiber
    Result: stretch yarn that looks and feels like cotton
    Used in: stretch denim (1-2% spandex), athletic socks

  FALSE-TWIST TEXTURED POLYESTER:
    POY (partially oriented yarn) → false-twist texturing machine
    Temporary over-twisting → heat-set → untwisting
    Result: helical crimp locked in by heat setting
    Transforms flat, smooth polyester into:
      Bulk polyester (fleece fabric)
      Textured nylon (activewear, tights)
    Without texturing: flat filament polyester feels plastic

  BOUCLÉ:
    Effect yarn with looped texture
    Two yarns fed at different rates → excess forms loops
    "Chanel bouclé" fabric is iconic in fashion

  SLUB YARN:
    Intentional thick-and-thin variation during spinning
    Creates irregular texture → "linen-look" in cotton
    Originally a defect; now a designed effect

  CHENILLE:
    Short fibers cut and twisted into a holding core yarn
    Result: velvet-like pile yarn
    Used in: upholstery, sweaters, decorative textiles
```

## Common Confusion Points

**Ring spinning is not obsolete**: Despite open-end being 5-10× faster, ring spinning produces finer, stronger, smoother yarn. Fine cotton (Ne 40s+), merino wool, and quality knit fabrics still use ring-spun. "Ring-spun cotton" on a shirt label is a quality claim.

**"Denier" for stockings vs "denier" for fabric**: Same unit. 15D = very sheer hosiery (visible legs through). 70D = sturdy outdoor fabric. 1500D = heavy duty industrial. The number just means grams per 9,000 meters — higher = heavier fiber.

**Worsted is a spinning system AND a yarn weight**: Worsted spinning = combed, parallel-aligned fibers → smooth yarn. "Worsted weight" knitting yarn = medium-weight (NOT necessarily worsted-spun). Context matters.

**Twist angle affects everything**: Low twist = soft, fuzzy, absorbent (good for towels), weak. High twist = hard, smooth, strong, but less absorbent. Crepe fabrics use very high-twist yarn → characteristic pebbled texture + stretch. This is why towels and dress shirts feel completely different even if made from similar cotton.
