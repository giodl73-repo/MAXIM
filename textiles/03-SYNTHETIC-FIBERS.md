# Synthetic Fibers

## The Big Picture: Untethering Textiles from Agriculture

```
NATURAL → MANUFACTURED FIBER TIMELINE
+------------------------------------------------------------------+
|                                                                  |
|  All textiles from       1892: Rayon       1938: Nylon           |
|  plant/animal until      First man-made    First fully           |
|  late 19th century       fiber (cellulose  synthetic fiber       |
|                          source, chemical  (petrochem)           |
|                          process)                                |
|                                            1941: Polyester       |
|                                            ICI Terylene          |
|                                                                  |
|                                            1958: Spandex/Lycra   |
|                                            Du Pont — elastomeric |
|                                                                  |
|  TODAY: ~65% of global fiber consumption is synthetic            |
|  Polyester alone: ~54% of all fiber produced                    |
|  Cotton: ~24% (declining)                                        |
+------------------------------------------------------------------+
```

The synthetic fiber revolution allowed properties to be *engineered*: exact fiber diameter, precise tenacity, zero moisture absorption, heat-set crimp, elastic recovery of 500%. It also created the microplastics crisis, dependency on petroleum, and non-biodegradable clothing.

## Regenerated Cellulosic Fibers (Not Synthetic, but Not Natural)

### Rayon/Viscose (1892)

```
SOURCE: Wood pulp or cotton linters (natural cellulose)
CHEMISTRY: Viscose process

  Cellulose (wood pulp)
       ↓ + NaOH → mercerized cellulose
  Alkali cellulose
       ↓ + CS₂ (carbon disulfide) → xanthation
  Cellulose xanthate ("orange crumbs")
       ↓ + dilute NaOH → dissolve
  Viscose solution
       ↓ forced through spinneret into H₂SO₄ bath
  Cellulose fiber (regenerated)

  PROBLEM: CS₂ is acutely toxic (neurotoxin, flammable)
           Industrial rayon plants had serious worker health issues
           Chemical effluent disposal — environmental burden
```

**Properties**: Soft, absorbent, good drape, similar to silk/cotton feel. Cheap. Weak when wet (~50% strength loss). Wrinkles easily. Not durable. Shrinks.

**Use**: Fashion fabrics, linings, blouses, dresses, disposable items.

### Modal (1951, Lenzing Austria)

```
  Improved viscose process with longer cellulose chains (higher DP)
  Source: beechwood cellulose
  Properties: 50% stronger than standard rayon when wet
              Resists pilling (longer chains hold together)
              Softer than cotton (used in underwear)
  Same basic chemistry as viscose but optimized
```

### Lyocell/Tencel (NMMO Process)

```
SOURCE: Wood pulp cellulose (FSC-certified eucalyptus for Tencel)
CHEMISTRY: NMMO (N-methylmorpholine N-oxide) direct dissolution

  Cellulose
       ↓ + NMMO (solvent) + water → direct dissolution
  Cellulose solution (no derivatization — no toxic intermediates)
       ↓ forced through spinneret into dilute NMMO bath
  Lyocell fiber
       ↓ NMMO recovered (>99% closed loop)

  ADVANTAGE: No CS₂, no sulfide effluent
             NMMO is recyclable (>99% recovery)
             More sustainable by lifecycle analysis

PROPERTIES vs. RAYON:
  Stronger (especially when wet — 80% vs 50% of dry strength)
  Fibrillates under mechanical agitation → soft, peach-skin surface
  Biodegradable in ~2-3 months (soil burial)
  Wrinkles like cotton but often softened with finishing

TENCEL brand (Lenzing):
  High-end sustainable fashion positioning
  Now dominant in premium basics and outdoor
```

## Polyamide: Nylon (1938)

### The Du Pont Story

```
  Wallace Hume Carothers:
    Harvard Ph.D, joined Du Pont basic research 1928
    Goal: prove macromolecules exist (theoretical debate at time)
    Method: condensation polymerization of difunctional monomers
    1935: Nylon 6,6 synthesis (hexamethylenediamine + adipic acid)
          Carothers: "an infinite number of fibers"
    1937: Carothers dies by suicide (age 41, clinical depression)
    1938: Du Pont announces Nylon publicly
    1939 World's Fair: "stronger than steel, finer than silk"
    1940: Nylon stockings go on sale — sell out immediately
    1942–1945: All nylon production diverted to WWII (parachutes,
               glider tow ropes, flak jackets, rope, tents)
    1945: "Nylon riot" — post-war release causes store stampedes
```

### Nylon Chemistry

```
NYLON 6,6 (condensation polymerization):
  H₂N-(CH₂)₆-NH₂    +    HOOC-(CH₂)₄-COOH
  hexamethylenediamine     adipic acid
         ↓ condensation (lose H₂O at each bond)
  [-NH-(CH₂)₆-NH-CO-(CH₂)₄-CO-]n

  "6,6" = 6 carbons in each monomer

NYLON 6 (ring-opening polymerization):
  Caprolactam (6-carbon ring) → ring opens → chain
  Same repeat unit as Nylon 6,6 but different production
  Slightly different dyeing behavior; both widely used
```

### Nylon Properties

| Property | Value |
|----------|-------|
| Tenacity | 3.5–6.0 cN/dtex |
| Elongation at break | 25–40% |
| Elastic recovery | Excellent |
| Moisture regain | 4.0–4.5% |
| UV resistance | Poor (yellows) — needs stabilizer |
| Abrasion resistance | Best of common fibers |
| Melting point | Nylon 6,6: 265°C; Nylon 6: 220°C |
| Dyeability | Acid dyes (good) and disperse |

**Primary uses**: Stockings, tights, activewear, swimwear, outerwear, rope, toothbrush bristles, netting, parachute fabric.

## Polyester (1941)

### Development

```
  1941: J.R. Whinfield and J.T. Dickson at ICI/Calico Printers
        Synthesize PET from ethylene glycol + terephthalic acid
        "Terylene" (ICI UK trade name)
  1951: Du Pont licenses technology → "Dacron"
  1960s: Mass adoption; "permanent press" fabrics popular
  1970s: Polyester becomes #1 fiber globally (cheap, wrinkle-free)
  1980s: Polyester becomes unfashionable (associated with cheap)
  1993: Patagonia introduces recycled polyester (rPET) fleece
         made from plastic bottles — first major rPET apparel
  Today: ~54% of all fiber produced globally is polyester
```

### Polyester Chemistry

```
PET SYNTHESIS (condensation polymerization):
  HO-CH₂-CH₂-OH   +   HOOC-C₆H₄-COOH
  ethylene glycol       terephthalic acid
       ↓
  [-O-CH₂-CH₂-O-CO-C₆H₄-CO-]n

  Both monomers from petroleum refining
  PTA (purified terephthalic acid) from p-xylene
  Ethylene glycol from ethylene (ethane cracking)
```

### Polyester Properties

| Property | Value |
|----------|-------|
| Tenacity | 3.5–5.5 cN/dtex |
| Elongation | 20–30% |
| Moisture regain | 0.4% — essentially none |
| UV resistance | Good |
| Melting point | ~260°C (PET) |
| Chemical resistance | Excellent |
| Dyeability | Disperse dyes only (no ionic groups) |

**The moisture problem**: 0.4% moisture regain means sweat stays on fiber surface → clammy feel. Engineered textured microfiber structures (capillary channels) improve *transport* without absorption — moves moisture away from skin.

### The Microplastics Problem

```
  Polyester washing: ~700,000 microfibers per wash cycle
  Global: ~500,000 tons of microplastics/year from synthetic textiles
  Pathway: washing machine → wastewater → WWTP (catches ~70%)
           → remaining 30% to waterways
  Size: 10–5,000 μm — eaten by zooplankton, fish, filter feeders
  Found in: deep ocean sediment, Arctic ice, human blood, placentas

  PROBLEM: PET does not biodegrade in environment
           UV photodegrades to smaller fragments (not mineralized)
           Carries persistent organic pollutants on surface

  MITIGATION: Guppy Friend washing bag (catches fibers),
              Cora Ball, cold water wash (less fiber shedding),
              rPET (same problem, just recycled petroleum-based)
```

## Elastane/Spandex/Lycra (1958)

```
INVENTOR: Joseph Shivers, Du Pont (1958)
TRADE NAMES: Lycra (Du Pont → INVISTA → Lycra Company)
             Elastane (European/BISFA name)
             Dorlastan (Bayer)
             Spandex (generic, US FTC fiber category)

ETYMOLOGY: "Spandex" = anagram of "expands"

CHEMISTRY: Segmented polyurethane-polyurea copolymer
  Hard segments (aromatic diisocyanate + chain extender)
    → crystalline, stiff, anchors fiber structure
  Soft segments (polyol macroglycol)
    → amorphous, high mobility, extensible
  Hard/soft segment phase separation → elastic properties

MECHANICAL PROPERTIES:
  Extension: 500–600% before break
  Elastic recovery: >95% after repeated extension
  Tensile stress at 100% elongation: ~0.03 cN/dtex (very soft)
  Melting range: 230–290°C (depends on composition)

USAGE PATTERN:
  Never used 100% (uncomfortable, tacky feel)
  Typically 2–15% spandex blended with polyester/nylon/cotton
  Swimwear: 15–20% (repeated stretch + chlorine exposure)
  Activewear: 5–15%
  Denim: 1–2% ("stretch denim")
  Socks: 2–3%

CHLORINE DEGRADATION: Pool chlorine attacks urethane bonds
  → Lycra swimwear degrades faster in pools vs sea water
  Chlorine-resistant variants available (Lycra Xtra Life)
```

## Other Synthetic Fibers

### Acrylic (Polyacrylonitrile)

```
  Wool-like appearance (fluffy, warm)
  Orlon (Du Pont), Acrilan brands
  Cheap — used in knitwear, fake fur, industrial applications
  Weakest common synthetic (low tenacity)
  Pills readily (fiber breaks, balls on surface)
  Not biodegradable; microplastic shedder
  Declining as polyester/better synthetics improve
```

### Polypropylene (Olefin)

```
  Lowest density of all fibers (0.91 g/cm³ — floats in water)
  Zero moisture absorption (hydrophobic) — best moisture wicking
  Solution-dyed only (cannot post-dye — no reactive sites)
  Low melting point (165°C) — cannot launder hot
  Excellent chemical resistance
  Used: thermal underwear base layers, geotextiles, medical disposables,
        rope, carpeting
```

### Microfibers and Technical Variants

```
  MICROFIBER: denier < 1.0 (ultra-fine filament)
  Achieved by: meltspun directly (POY → DT process) or
               "islands-in-the-sea" bicomponent splitting

  Properties:
    Ultra-soft hand (surface area increases with fineness)
    High surface area → superior moisture management
    Split microfiber (polyester/nylon): open capillary channels
    → excellent cleaning cloth (mechanical picking action)

  ALCANTARA/ULTRASUEDE:
    Brand-name microfiber composites
    Split polyester/nylon microfibers in polyurethane matrix
    Suede-like appearance without animal leather
    Used: automotive interiors, luxury fashion
```

## Environmental Scorecard

```
FIBER          PRODUCTION                      END OF LIFE
------         ----------                      -----------
Polyester      Petroleum-based; 340M barrels   Non-biodegradable;
               oil/year for textiles           100-500 yr breakdown;
               High energy                     microplastic shedder

Nylon          Petroleum; adipic acid          Non-biodegradable;
               synthesis produces N₂O          microplastic shedder
               (GHG 265× CO₂)

Acrylic        Petroleum; acrylonitrile        Non-biodegradable;
               (carcinogen in production)      microplastic shedder

Rayon/Viscose  CS₂ toxic in production        Biodegrades (cellulose
               Chemical effluent              base) — months

Lyocell        Low-impact NMMO process         Biodegrades in ~3 mo.
               Closed-loop solvent recovery    (soil conditions)

Cotton         25% global insecticides;        Biodegradable;
               7,000–10,000 L water/pair       BUT: dyes/finishes
               of jeans                        may not be

Wool           Methane from sheep (GHG);       Biodegradable;
               land use/overgrazing            ~ months to years
               risk (cashmere)

Silk           Mulberry cultivation;           Biodegradable;
               boiling silkworms (PETA)        ~ months
```

**Bottom line**: No fiber is fully sustainable at the scale of global fashion consumption. Lyocell and organic hemp/linen are best by most lifecycle metrics, but the dominant factor is how long a garment is used — a well-worn polyester piece beats a single-use organic cotton one.

## Decision Cheat Sheet

| I need... | Use |
|---|---|
| Cheapest fiber, wrinkle-resistant | Polyester |
| Best abrasion resistance | Nylon |
| Elastic recovery (stretch) | Spandex (blended) |
| Warmth at low weight | Acrylic (budget) or wool |
| Best moisture wicking (no absorption) | Polypropylene or split microfiber |
| Soft, drape, similar to silk | Rayon/viscose |
| Sustainable synthetic alternative | Lyocell/Tencel |
| Biodegradable synthetic | Lyocell (not truly "synthetic") |

## Common Confusion Points

**Rayon/viscose is not synthetic**: It comes from wood cellulose. It's a "regenerated" or "manufactured" fiber from a natural source. The FTC calls it "rayon" in US labeling.

**Lyocell is not the same as rayon**: Both are cellulose-based, but Lyocell uses NMMO (a recyclable solvent) vs viscose's toxic CS₂. Properties differ too — Lyocell is stronger and fibrillates differently.

**Polyester recycling (rPET) doesn't solve microplastics**: rPET fleece made from bottles still sheds microplastics into waterways on every wash. It reduces virgin petroleum use but doesn't fix the end-of-life problem.

**Nylon and polyester look similar but behave differently**: Nylon absorbs more moisture (4.5% vs 0.4%), has better abrasion resistance, and degrades with UV. Polyester is cheaper, more UV-stable, and dominates volume. The "is this nylon or polyester?" burn test: both melt; polyester drips and smells sweet; nylon drips and smells like celery.
