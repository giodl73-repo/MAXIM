# Construction Materials — Medieval Timber and Masonry

## The Big Picture: Medieval Structural Innovations

```
MEDIEVAL STRUCTURAL EVOLUTION (~500–1700 CE)
┌──────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│  PROBLEM: How to build tall structures with large windows in stone           │
│  CONSTRAINT: Stone has near-zero tensile strength                            │
│                                                                              │
│  Romanesque (~800–1150):                                                     │
│  Round arch + thick walls + small windows                                    │
│  Horizontal thrust → must be absorbed by thick masonry                       │
│  Height limit: ~30–40 m before foundation/wall area impractical              │
│                                                                              │
│  Gothic Revolution (~1150–1500):                                             │
│  Pointed arch → reduces horizontal thrust for given span                     │
│  Ribbed vault → concentrates loads to discrete piers                         │
│  Flying buttress → carries thrust OUTSIDE the building                       │
│  Result: walls become thin screens filled with glass                         │
│  Height achieved: Beauvais choir 48 m (1272); ultimate collapse 1284         │
│                                                                              │
│  Timber Frame (~500–1700, parallel track):                                   │
│  Cruck frames → box frames → half-timber                                     │
│  Post/beam/rafter system → large spans; agricultural; domestic               │
│                                                                              │
│  Lime mortar throughout: flexible, breathable, self-healing                  │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
```

## Timber Framing Evolution

### Cruck Frame

The cruck is the oldest British timber framing system (~1000–1600 CE).
A "blade" — a naturally curved timber (usually a single tree split along its
natural curve) — sweeps from ground level to ridge.

```
CRUCK FRAME STRUCTURE
┌──────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│  Section view:                                                               │
│                    ridge                                                     │
│                    ─┬─                                                       │
│                   / │ \                                                      │
│                  /  │  \       ← blade: single curved timber                 │
│                 /   │   \                                                    │
│  ─ collar ────╱─────┼─────╲──── collar                                       │
│              /      │      \    ← tie collar prevents spreading              │
│             /       │       \                                                │
│  ─ tie ───╱─────────┼─────────╲── tie beam at wall head height               │
│           │  ←space→│←space→  │                                              │
│           └─────────┴─────────┘  ← groundsill or stone pad                   │
│             ← bay spacing →                                                  │
│                                                                              │
│  KEY FEATURE: blades carry BOTH roof and wall loads in one curved element    │
│  → no separate wall frame needed; bay = the fundamental unit                 │
│  → typical bay: 3.5–5 m span                                                 │
│  → buildings measured in "bays": "a 4-bay barn"                              │
│                                                                              │
│  LIMITATION: curved tree scarcity; agricultural buildings mainly             │
│  → English domestic cruck: found in Midlands and North; rare in SE           │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
```

Box frame separates the vertical wall frame from the roof structure.
This is the mainstream English medieval system.

### Half-Timbered Construction: Infill Options

Half-timbering is the wall panel between structural timber members. The infill is
NOT structural — the timber frame carries all loads.

## Timber Joints: How They Work

### Mortise-and-Tenon (see also 01-PREHISTORIC)

The principle carpentry joint of the medieval world. The critical dimensions:

```
MORTISE-AND-TENON PROPORTIONS
──────────────────────────────────────────────────────────────────────────────
  Tenon thickness:  T/3 of member thickness (one third rule)
  Tenon width:      ≤ 5T (wider → risk of splitting mortise)
  Tenon length:     ≥ T (must engage enough material)

  Example: 150 × 100 mm beam → tenon: 50 mm thick × 100 mm wide × 75 mm deep

  DRAWBORING: the peg hole in the tenon is drilled slightly offset (5–10 mm)
    from the hole in the mortise
  → when peg is driven: draws the joint tight (like a cam)
  → works ONLY with green oak → shrinkage further tightens
```

### Scarf Joint: Lengthening a Timber

Timber is limited by available lengths. Scarf joints allow extension.

```
SCARF JOINT TYPES
──────────────────────────────────────────────────────────────────────────────
  Bladed scarf:   ╱╲  Simple overlapping cut; weak in bending
  Bridled scarf:  ╱╲╱ With pegged bridge; better for tension
  Splayed scarf:  ╱═╲ With abutting faces for compression + wedge
  Tabled scarf:   Complex interlocking; used at ridge or tie beam ends
    → best scarf joints: 1.5× member depth in overlap length minimum

  USAGE:
  Sills (ground level): susceptible to rot → scarfed section replacement
  Plates: horizontal wall top members often scarfed at post positions
  Ridges: long ridge boards often multiple sections scarfed
```

### Dovetail Joint: Resisting Withdrawal

Dovetail connections prevent tenon from pulling out — the trapezoid shape locks in.

```
DOVETAIL MORTISE AND TENON
    ╔════════════╗         ┌─────────────────┐
    ║  wider end ║────────→│  dovetail mortise│
    ║  (dovetail)║         │  in receiving    │
    ╚════════════╝         │  member          │
      ↑ tapering section   └─────────────────┘

  → Cannot be withdrawn along the axis once fully inserted
  → Must be slid in from the side
  → Used for: joist to beam connections; brace to post
```

## Lime Mortar: Chemistry and Practice

### Quicklime Production

```
LIME CYCLE
──────────────────────────────────────────────────────────────────────────────
  MINING:     CaCO₃  (limestone, chalk, marble)

  CALCINATION: kiln at 900–1000°C
               CaCO₃ → CaO + CO₂
               Product: quicklime (unslaked lime) — very reactive

  SLAKING (hot lime method):
               CaO + H₂O → Ca(OH)₂ + heat (65 kJ/mol — violent)
               "Runs" of lime: quicklime + water → violent bubbling reaction
               → slaked lime (lime putty, calcium hydroxide paste)

  MATURATION: stored in lime pits or barrels submerged in water
               1 month minimum; 1–5 years preferred
               → colloidal Ca(OH)₂ particles become finer → better mortar

  SETTING (carbonation):
               Ca(OH)₂ + CO₂ → CaCO₃ + H₂O
               → mortar absorbs atmospheric CO₂ and slowly reverts to stone
               → rate: ~5–10 mm depth per year for joint carbonation
               → must never be painted/sealed — needs air access

  CO₂ ACCOUNTING:
    Calcination releases CO₂; carbonation reabsorbs same CO₂
    Net over full cycle: zero (if using renewable fuel for kiln)
    → lime mortar is potentially carbon-neutral
    → Portland cement: C₃S hydration; no re-carbonation → irreversible release
```

### Hot Lime vs Hydraulic Lime

| Property | Non-hydraulic (air lime) | Hydraulic lime (NHL) | Portland cement |
|---|---|---|---|
| Setting mechanism | Carbonation (CO₂) | Hydration + carbonation | Hydration only |
| Sets underwater | No | Yes | Yes |
| Strength at 28 days | 1–2 MPa | 3.5–12 MPa | 25–60 MPa |
| Flexibility | High (elasticity ~2–4 GPa) | Medium | Low (E~20–30 GPa) |
| Vapour permeability | High (μ ~5–15) | Medium | Low (μ ~80–120) |
| Self-healing | Yes | Partial | No |
| Compatibility with historic masonry | Ideal | Good | Damaging |
| Workability period | Hours to days | 1–2 hours | 30–90 min |
| Re-workability | Yes (wet lime) | Limited | No |

**Why Portland cement damages historic masonry:** OPC mortar is stiffer and stronger
than historic brick/stone. When thermal movement or settlement occurs, the masonry
cracks rather than the joint — this is the opposite of intended behaviour. Lime mortar
is the weaker/softer element and cracks first, can be re-pointed; the brick face survives.

## Gothic Structural Revolution

### The Force Diagram: Pointed Arch vs Round Arch

```
HORIZONTAL THRUST COMPARISON: SAME SPAN (L = 10 m), SAME VERTICAL LOAD
──────────────────────────────────────────────────────────────────────────────
  SEMICIRCULAR ARCH:                     POINTED ARCH:
  Rise f = L/2 = 5 m                     Rise f = L × 0.8 = 8 m (for 60° point)

  H = wL²/(8f) = w(10)²/(8×5)           H = wL²/(8f) = w(10)²/(8×8)
  H = 100w/40 = 2.5w                     H = 100w/64 = 1.56w

  → Pointed arch generates 37% LESS horizontal thrust for same span and load

  IMPLICATION:
  Semicircular: wall or buttress must resist H = 2.5w → THICK WALLS
  Pointed arch: wall or buttress must resist H = 1.56w → 37% thinner possible

  BUT Gothic builders didn't just use less material in walls —
  they REMOVED the walls entirely and put glass there instead.
  The thrust was transferred OUTSIDE via flying buttresses.
```

### Notre-Dame de Paris: Structural Logic

GOTHIC CATHEDRAL STRUCTURAL SYSTEM

│                                                                              │

### Ribbed Vault: Load Concentration

```
RIBBED VAULT: HOW IT CONCENTRATES LOAD
──────────────────────────────────────────────────────────────────────────────
  Groin vault (Romanesque):  load distributes across surface → thick walls
  Ribbed vault (Gothic):    load follows stone ribs → concentrated at piers
                             web between ribs: thin stone fill (infill only)

  Structural action:
  ┌─────────────────────────────────────────────┐
  │   diagonal rib                              │
  │     ╲         ╱                             │
  │      ╲       ╱                              │
  │       ╲     ╱   ← two diagonal ribs per bay │
  │        ╲   ╱                                │
  │         ╲ ╱                                 │
  │          ●  ← boss (ornamental keystone)    │
  │         ╱ ╲                                 │
  │        ╱   ╲                                │
  │       ╱     ╲                               │
  │      ╱       ╲                              │
  │ pier ●       ● pier  ← load concentrated    │
  └─────────────────────────────────────────────┘

  RESULT:
  All vault load → four corner piers → walls between piers = zero structural load
  → walls filled with tracery windows
  → Sainte-Chapelle (Paris, 1248): 75% of wall area is glass

  Fan vault (Perpendicular Gothic, England, 1400s):
    Ribs fan out from single springer point
    Conoid shapes in 3D geometry
    King's College Chapel Cambridge (1515): fan vault + large windows
    Structurally equivalent to ribbed vault; more complex geometry; all in compression
```

### Flying Buttress: Mechanics

```
FLYING BUTTRESS FORCE TRANSFER
──────────────────────────────────────────────────────────────────────────────

  Side elevation:

    vault crown
        │
        ↓ vertical load (from vault weight)
    ────●────  nave wall at aisle roof level
        │  ↘
        │   → horizontal thrust from vault
        │    ╲
  clerestory  ╲  ← flying buttress arc (stone arch)
        │      ╲
  aisle roof    ●  ← buttress springer on nave wall
                │
                ↓  resultant force (thrust + self-weight of buttress)
                │  → directed to follow line of buttress arch
                │  → must land on top of outer abutment pier
              ──●──  outer abutment pier top
                │
              outer pier → foundation

  KEY: the buttress arc is a TRUE ARCH → only compression
       Size the arc so its axis follows the thrust line
       → add stone pinnacles atop outer pier: extra weight → steeper resultant → safer
       → pinnacles are structural, not just decorative
```

## Timber Roof Trusses: Structural Types

### King Post Truss

Simplest timber roof truss — a tie beam suspended by a central king post.

```
KING POST TRUSS  (spans 5–8 m)

         ┌─── ridge board ───┐
        ╱│                   │╲
       ╱ │   king post       │ ╲  ← common rafters
      ╱  │    │   │          │  ╲
     ╱   │ strut│  │strut    │   ╲
    ╱    │ ╱   │  │   ╲     │    ╲
───┼─────┼─────┼──┼─────┼───┼─────┼───
       tie beam (also ceiling joist)

  STRUCTURAL MECHANISM:
  - Rafters in compression → push out at feet
  - Tie beam in TENSION → resists rafter spread
  - King post in TENSION → hangs tie beam at mid-span (prevents sag)
  - Struts transfer rafter load to king post panel point

  COMMON ERROR: treating king post as a column in compression
  → it is always in TENSION (hanging member)
  → this is why traditional trusses use pegged mortise-tenon at king post head
    but a bird's mouth bearing at foot — designed for tension
```

### Queen Post Truss

Two vertical queen posts instead of central king post — allows wider span and
space between the posts (useful for rooms-in-roof).

```
QUEEN POST TRUSS  (spans 7–12 m)

      ─────────────────────────────
     ╱  ╲                   ╱    ╲
    ╱    │queen             │queen│
   ╱     │post              │post ╲
  ╱      │                  │      ╲
─┼───────┼──────────────────┼───────┼─
         ←  collar beam  →

  ADVANTAGE over king post:
  Space between queen posts → dormers, habitable rooms in roof zone
  Collar beam higher up → more headroom
  LIMITATION: queen posts still in tension; collar in compression
```

### Hammer Beam Truss

The most spectacular medieval timber invention — achieves spans of 12–20 m with
no tie beam across the full span.

```
HAMMER BEAM TRUSS (spans 10–20 m)  — Westminster Hall 20.7 m, 1399 CE

         ┌─ ridge ─┐
        ╱│         │╲
       ╱ │         │ ╲  arch braces up to ridge
      ╱  collar    │  ╲
     ╱   │         │   ╲
 strut  ham  brace ham  strut
   │    beam│    │beam│    │
───┴────────┴────┴────┴────┴──────
   wall    wall corbel wall    wall

  WHY NO TIE BEAM:
  → hammer beams: short horizontal cantilevers from wall corbel/post
  → arch braces: curved timbers connect hammer beam tip to principal rafter
  → struts from wall posts to hammer beam underside
  → the hammer beam extends the "moment arm" → arch brace carries thrust partially
  → NET EFFECT: significant horizontal thrust still exists at wall level
    → thick masonry walls still required at springing
  → Westminster Hall outer walls: ~1.2 m thick at roof level

  VISUAL EFFECT:
  → series of layered arched frames → visually spectacular (like flying arches)
  → alternating vertical struts and curved arch braces create intricate 3D pattern
```

### Span Capability Comparison

| Truss type | Practical span | Key advantage | Key limitation |
|---|---|---|---|
| King post | 5–8 m | Simple; few members | Central king post blocks room |
| Queen post | 7–12 m | Space between posts | Still some tie constraint |
| Hammer beam | 10–20 m | No full tie beam | Wall thrust; complex joinery |
| Scissor beam | 5–9 m | Decorative; raises ceiling | Complex geometry; limited span |
| Raised tie | 6–10 m | More headroom | Tie must be sized for increased tension |

## Decision Cheat Sheet

| Question | Answer |
|---|---|
| Which lime for hydraulic setting (underwater)? | Natural hydraulic lime (NHL 3.5, NHL 5) |
| Repointing historic brick — which mortar? | Weaker than the brick: usually NHL 2 or NHL 3.5 with sharp sand |
| Gothic wall thin between piers — why? | Flying buttress carries vault thrust outside; wall between piers is non-structural screen |
| King post carries compression or tension? | TENSION always — it hangs the tie beam; do not confuse with a strut |
| Hammer beam truss — does it eliminate wall thrust? | No — reduces it but walls still need significant thickness or buttresses |
| Fan vault vs ribbed vault — structurally? | Equivalent — both channel loads to pier springers; fan is more complex geometry |
| Green oak vs seasoned — which for new build? | Green oak for traditional joinery (seasons in place); seasoned for glulam/engineered |
| Wattle infill cracking around timber — why? | Differential movement; lime mortar needed, not rigid OPC render |
| What joints for timber frame rafter to wall plate? | Bird's mouth: bearing notch cut in rafter resting on plate; simple, reliable in compression |

## Common Confusion Points

**Flying buttresses are structural, not ornamental.** Remove the flying buttresses from
Chartres cathedral and the clerestory walls collapse outward within days. They carry
the vault thrust to the outer abutment piers. The pinnacles atop the abutment piers
are also structural — they add weight to steer the resultant force (vault thrust +
self-weight) into the pier cross-section.

**Lime mortar is the "designed weak element" — intentionally.** Historic masons
designed joints to be weaker than the stone/brick face. This is not a failure of skill;
it is deliberate — when movement occurs, the joint cracks rather than the face, and
joints can be re-pointed. Portland cement mortars reverse this and damage masonry faces.

**Half-timber buildings do NOT rely on the infill panels for structural integrity.**
The oak frame carries all loads. The wattle-and-daub or brick nogging infill is a
weather and fire barrier only. This is why historic timber frames often have missing
or replaced infill panels without structural consequence — and why modern retrofit
insulation in the infill panels is usually structurally valid (but creates vapour
risk if not detailed carefully).

**King post tension is counterintuitive.** People see a vertical member in a truss and
assume column in compression. In a king post truss under load: the rafters push
outward; the tie beam resists the outward push (tension in tie); the king post
HANGS the centre of the tie beam (tension in king post). If you add mid-span floor
loads (attic), the king post tension increases. If you add an in-plane load to the
ridge (pushing down), the king post can go to compression — but this is not the
normal load case for a roof truss.

**Medieval carpentry is not pre-engineering — it IS engineering.** The proportional
rules (tenon = 1/3 of member thickness; mortise depth = 1/3 of member width; peg
diameter = 1/8 of member thickness) are empirical safety factors derived from
centuries of observation, equivalent in function to a code safety factor. The
buildings that failed were demolished; the ones with correct proportions survived.
Selection pressure over centuries IS engineering optimization.
