# Greek Origins of Theater

## The Big Picture

Greek theater is the origin point of Western theatrical tradition and of formal dramatic theory. The physical structure (orchestra, skene, theatron), the formal structural principles (the five-part structure of tragedy), and the philosophical framework (Aristotle's Poetics) all emerged from the Athenian dramatic festivals of the 5th century BCE and have shaped theater ever since.

```
+------------------------------------------------------------------+
|           GREEK THEATER — SYSTEM OVERVIEW                        |
+------------------------------------------------------------------+
|                                                                  |
|  FESTIVAL CONTEXT                                                |
|  City Dionysia (Athens, ~486 BCE): major spring festival         |
|  Lenaia (Athens, ~440 BCE): winter festival, comedy emphasis    |
|  State-sponsored; 3 tragedians compete; each submits a trilogy  |
|  + satyr play; 5 comic poets compete                             |
|                                                                  |
|  PHYSICAL STRUCTURE                                              |
|  THEATRON (seating area): ~14,000 seats, hillside              |
|  ORCHESTRA (dancing floor): circular, central                   |
|  THYMELE (altar of Dionysus): center of orchestra              |
|  SKENE (stage building): backdrop, costume storage, 3 doors    |
|  PROSKENION (raised stage): in front of skene                  |
|  EISODOI/PARODOI: two entrance ramps to orchestra               |
|                                                                  |
|  FIVE-PART TRAGEDY STRUCTURE                                     |
|  PROLOGOS → PARODOS → EPEISODION/STASIMON (×3-5) → EXODOS      |
|                                                                  |
|  ARISTOTLE'S POETICS (c. 335 BCE)                               |
|  Six elements: mythos > ethos > dianoia > lexis > melopoeia >  |
|  opsis. Key concepts: mimesis, catharsis, hamartia, anagnorisis,|
|  peripeteia, the unities                                        |
+------------------------------------------------------------------+
```

## Layer 1: The Festival Context

### The City Dionysia

Theater in Athens was not entertainment — it was a civic religious institution:

```
CITY DIONYSIA — STRUCTURE
--------------------------
Date: late March/early April (spring sailing season opens;
       ambassadors and tribute-bearers from allied cities present)
Duration: 5-6 days
Sponsor: the state (archon eponymous assigned the festival)
Funding: the choregos system:
  Wealthy citizens were assigned to fund and equip
  a chorus for one playwright. The choregos was a
  form of liturgy (compulsory public service for the rich).
  He paid for: costumes, masks, chorus training,
  musicians, a banquet for the chorus.

DAY 1: Grand procession; tribute from allies displayed;
  war orphans (sons of fallen soldiers) paraded in armor;
  generals pour libations
DAYS 2-3: Five comedies (one per day)
DAYS 3-5: Three tragedies + one satyr play per tragedian
  (three tragedians, each with a tetralogy)
FINAL DAY: prizes awarded; financial accounts reviewed publicly

THE POLITICAL DIMENSION:
  The festival opened with a display of imperial tribute.
  Orphans of the war dead were honored.
  Plays explored the limits of justice, power, and fate.
  This was civic deliberation in dramatic form, not
  a separate "cultural" activity.
```

### The Three-Play System (Tetralogy)

```
THE ATHENIAN TETRALOGY
-----------------------
Each tragedian submitted:
  Tragedy 1 +
  Tragedy 2 +
  Tragedy 3 +
  Satyr play (comic, using satyrs)

The three tragedies could be:
  1. A connected trilogy on one subject (Oresteia)
     Aeschylus's Oresteia: Agamemnon + Libation Bearers
     + Eumenides (+ Proteus, satyr play, lost)
  2. Three unrelated tragedies (Sophocles, Euripides
     usually did this)

THE SATYR PLAY:
  A short, boisterous comedy featuring satyrs (half-men,
  half-horses; Dionysus's attendants).
  After three heavy tragedies, the satyr play provided
  release. Cyclops (Euripides) is the only complete
  surviving satyr play.
  Function: the "digestive" complement to tragedy.
```

## Layer 2: The Physical Theater

### The Theater of Dionysus, Athens

```
THEATER OF DIONYSUS — PHYSICAL LAYOUT
----------------------------------------

                [THEATRON — SEATING]
               14,000 seats on hillside
            Wedge-shaped sections (kerkides)
           Priests and officials in front row
          (marble thrones for honored seats)
                        |
              +---------+---------+
              |  EISODOI/PARODOI  |
              |  (entrance ramps) |
              |                   |
              |    ORCHESTRA      |
              |   (dance floor)   |
              |      [   ]        |
              |   THYMELE         |
              |   (altar)         |
              |                   |
              +-------------------+
              |     PROSKENION   |
              |   (raised stage)  |
              +-------------------+
              |       SKENE       |
              |  (stage building) |
              |   3 doors; used   |
              | for exits/entrances|
              +-------------------+

WHAT HAPPENED WHERE:
  Chorus: primarily in orchestra (the dancing/singing space)
  Actors: on/in front of skene, raised slightly
  Audience: in theatron, looking DOWN at the performance
  The god Dionysus: his altar (thymele) in the center
  of the orchestra — performances literally in his presence
```

### Stage Machinery

These are real-time mechanical engineering systems: the mechane is a load-bearing crane (rope, pulley, counterweight) operating during live performance with timing constraints and sight-line specifications. The ekkyklema is a wheeled platform on tracks. Both required rehearsed coordination between operators and performers — the earliest examples of stagecraft as engineering discipline.

```
THEATRICAL MACHINERY
---------------------
EKKYKLEMA (wheeling-out device):
  A low wheeled platform rolled out through the central
  skene door to reveal interior scenes.
  Greek stage conventions: violence happens OFFSTAGE.
  Reported by messengers or revealed via ekkyklema.
  (Clytemnestra standing over the bodies of Agamemnon
  and Cassandra: ekkyklema.)

MECHANE (the "machine"):
  A crane over the skene allowing gods to descend.
  The "deus ex machina" — god from the machine.
  Used by Euripides frequently (hence Aristotle's
  criticism: the machine-god is a poor plot resolution,
  an arbitrary external fix rather than a necessary
  consequence of the plot).

PINAKES (painted panels):
  Inserted into skene doorway slots to indicate location.
  Limited scenic indication — most "scenery" was verbal.

THREE-DOOR CONVENTION:
  Center door: palace/house interior
  Left door: different location or guest room
  Right door: way to the harbor/town
  (conventions varied, but three doors = three locations)
```

### Masks and Costumes

```
MASKS (PROSOPA)
---------------
All actors wore masks. This was not merely practical
(allow one actor to play multiple roles) — it was
the theatrical convention:

  Formal types: each character type had a standard mask
    (old man, young man, woman, soldier, messenger)
  Large mouth opening: amplified voice; theater was huge
  Fixed expression: no facial mobility — emotion was
    conveyed through voice, movement, and words, not face
  Wig attached: hairstyle = age/status signal

Masks erased individuality → characters were TYPES, not
individuals in the psychological sense. The audience
read the mask as a social role.

COSTUMES:
  Thick-soled boots (cothurni): added height, formality
  Long robes (chiton/himation): dignified, formal
  Padding: heroes were larger than life
  Color coding: mourning = black/grey; royalty = purple
```

## Layer 3: The Five-Part Structural Anatomy

Full analysis in literature/03-DRAMA.md. Here, the physical structure of performance:

```
1. PROLOGOS (prologue)
   Before the chorus enters.
   Sets the situation. Delivered by 1 or 2 actors.
   Duration: varies. In Oedipus Rex: opening exchange
   between Oedipus and the Priest + Creon's return.
   Not all tragedies have a formal prologue in this sense.

2. PARODOS (entry ode)
   The chorus's first song as it enters the orchestra
   via the two parodoi (entrance ramps).
   The parodos establishes: who the chorus is, their
   emotional orientation, the world they inhabit.
   Usually includes a formal statement of the problem
   or context.

3. EPEISODION + STASIMON (alternating, 3-5 cycles)
   EPEISODION: "that which comes in" — the actors'
     scene. Action advances; dialogue occurs.
     Characters enter and exit via parodoi or skene doors.

   STASIMON: "stationary ode" — the chorus stands still
     and sings and dances an ode.
     Structure of stasima: strophe + antistrophe (+ epode)
       Strophe: chorus moves in one direction, singing
       Antistrophe: chorus returns, same meter, different words
       Epode (sometimes): different meter, standing still
     Content: reflection on the action, mythological comparison,
       prayer, warning, moral commentary.

   THE ALTERNATING RHYTHM:
     Episode (action) + Ode (reflection)
     Episode (action) + Ode (reflection)
     Episode (action) + Ode (reflection)
     This is the heartbeat of tragedy:
     immediate dramatic action + mediated civic response.

4. EXODOS (exit song)
   The final scene.
   May include: final confrontation, lamentation, proclamation.
   Chorus exits via the parodoi singing.
   In some plays: the exodus is preceded by a KOMMOS
   (lamentation in which actors and chorus alternate).
```

### Mathematical Symmetry of the Choral Odes

```
THE FORMAL SYMMETRY OF STROPHE/ANTISTROPHE
--------------------------------------------
The strophe and antistrophe are metrically IDENTICAL.
Same meter, same structure, same syllable count.
The words differ; the music/meter is the same.

This is not just aesthetic — it is physical:
  The chorus literally moves in one direction (strophe),
  then RETURNS the same path (antistrophe).
  The choreography mirrors the metrical structure.

Multiple strophe/antistrophe pairs in one ode:
  A1 + B1 = strophe A and strophe B
  A2 + B2 = antistrophe A and antistrophe B
  (The ode can have multiple "strophic systems")

The epode (if present) breaks the symmetry:
  It is metrically different and sung standing still.
  It closes the ode and "resolves" the movement.

This is a FORMAL CONSTRAINT that simultaneously:
  - Organizes communal choral movement (dance)
  - Creates a musical call-and-response structure
  - Imposes symmetry on the lyric content
  Compare: the sonnet's volta, the rondeau's refrain,
  the fugue's answer — formal constraints that shape
  both content and performance.
```

## Layer 4: The Three Tragedians in Contrast

### Aeschylus (c. 525–456 BCE)

```
AESCHYLUS — KEY FEATURES
--------------------------
Only surviving CONNECTED TRILOGY: the Oresteia (458 BCE)
  Agamemnon: Clytemnestra kills Agamemnon
  Libation Bearers: Orestes kills Clytemnestra
  Eumenides: Orestes tried at Athens; Furies transformed
    into Eumenides (Kindly Ones); Athena establishes
    the Areopagus (legal court)

THEMES:
  Justice as historical process (not divine punishment
  but human legal institution)
  The transition from vendetta justice to civic justice
  The resolution of the blood feud through law

FORMAL FEATURES:
  Large choruses (50 in older practice; reduced later)
  Heavy, archaic language ("Aeschylean darkness")
  The chorus carries more of the dramatic weight
  Characters speak PAST each other more than in Sophocles
  Introduced the SECOND ACTOR (adding dialogue to choral performance)
```

### Sophocles (c. 497–405 BCE)

```
SOPHOCLES — KEY FEATURES
--------------------------
Introduced the THIRD ACTOR (allows more complex scenes)
Reduced chorus from 50 to 15
Greatest master of dramatic irony: the audience knows
  what the character does not

OEDIPUS REX (c. 429 BCE) — Aristotle's model:
  Structure: investigation of the past reveals the present
  Dramatic irony: audience knows from myth; Oedipus does not
  The anagnorisis and peripeteia are simultaneous:
    The Corinthian messenger means to comfort Oedipus
    (Polybus is dead; fear no patricide now!)
    Instead he reveals Oedipus was adopted
    → this knowledge DESTROYS Oedipus
  Single action, single day, single location (roughly)
  Aristotle: this is the best tragic plot because
    the recognition and reversal are maximally integrated

ANTIGONE (441 BCE):
  The conflict between divine law (bury the dead)
  and civil law (Creon's edict against burying traitors)
  Neither Creon nor Antigone is simply wrong — both
  are absolutely right within their own framework
  The tragedy is the conflict between two valid absolutes
```

### Euripides (c. 484–406 BCE)

```
EURIPIDES — KEY FEATURES
--------------------------
18 surviving plays (most of any tragedian)
"The most tragic of the poets" (Aristotle) — produced
  the most suffering, but controversially
The modernist of ancient tragedy:
  Women protagonists who reason for themselves
    (Medea, Hecuba, Phaedra)
  Slaves who have moral insight
  Gods who are unjust (and depicted as such)
  Rationalist critique of mythology

MEDEA (431 BCE):
  Medea kills her own children to revenge Jason's betrayal
  Euripides apparently INVENTED the child-killing
  (earlier versions had the Corinthians kill the children)
  The protagonist is simultaneously a murderer and
  the most wronged character in the play
  No easy moral resolution

PROBLEMS WITH EURIPIDES (Aristotle):
  Uses the deus ex machina too often
  Characters argue in set speeches (debate structure)
  The Chorus is often disconnected from the action
  Endings are messy, not tidy
  → All this is also why he feels most "modern" to us
```

## Layer 5: Greek Comedy

### Old Comedy: Aristophanes

```
ARISTOPHANES (c. 446–386 BCE)
-------------------------------
11 surviving comedies (of ~40)
"Old Comedy" — fantastical plots, personal satire,
  political commentary, obscenity, parabasis

PARABASIS: unique to Old Comedy
  In the middle of the play, the chorus steps forward,
  removes masks, addresses the audience directly
  as the POET — commenting on current events, defending
  the poet's work, making political arguments.
  A formal break in dramatic illusion; the playwright
  speaks.

PLOTS: fantastical premises with political content
  The Clouds (423): Socrates runs a "Thinkery";
    Strepsiades wants his son to learn clever arguments
    to cheat creditors. Satire of sophistry.
  The Wasps (422): Old man addicted to jury service
    (the Athenian jury system as comic target)
  Lysistrata (411): Women go on sex strike to end the war
  The Birds (414): Two men found a city of birds between
    heaven and earth, cutting the gods off from sacrifice

KEY FEATURE:
  Old Comedy is NAMED — it attacks real living people
  (Socrates, Cleon the demagogue, Euripides).
  It is political satire in dramatic form.
```

## Common Confusion Points

**Greek actors were all male, playing all roles**
Women did not perform in Greek drama. Male actors played women's roles (as boys did in Elizabethan theater). This is not incidental — it shapes what female characters in Greek tragedy CAN do and say (they are constructed by men for a male-citizen audience).

**The "deus ex machina" was a specific device, not just any convenient plot resolution**
The mechane was a literal crane. A "deus ex machina" was an actor playing a god literally lowered by this crane to intervene in the plot. Aristotle criticized Euripides for using it too often as an escape from plots the playwright could not resolve by dramatic logic. The modern meaning (any arbitrary plot resolution) is correct as metaphor.

**Three actors played ALL roles**
Each tragedian was allocated three actors (plus chorus of 15). These three actors played ALL the speaking parts by changing masks and costumes. This required careful management — no two characters who need to speak simultaneously can be in different places. The three-actor convention is a formal constraint with structural consequences for plot design.

**The "unities" are not from Aristotle**
As explained in literature/03-DRAMA.md: Aristotle required only unity of action. The famous three unities are a Renaissance misreading. Greek tragedy itself doesn't follow strict unity of time or place — the Oresteia spans years and moves from Argos to Athens.
