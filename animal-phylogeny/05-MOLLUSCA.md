## Layer 1 — The Molluscan Body Plan

### Four Defining Structures

```
1. MANTLE
   ┌────────────────────────────────────────────┐
   │ Dorsal fold of body wall                   │
   │ Secretes shell (periostracum + calcite     │
   │   layers: prismatic + nacreous)            │
   │ Encloses mantle cavity                     │
   │ Gas exchange, waste removal, gametes       │
   └────────────────────────────────────────────┘

2. FOOT
   ┌────────────────────────────────────────────┐
   │ Muscular, ventral, highly modified         │
   │ Gastropod: flat creeping sole              │
   │ Bivalve: wedge for burrowing               │
   │ Cephalopod: highly modified → funnel + arms│
   └────────────────────────────────────────────┘

3. RADULA (absent in Bivalvia)
   ┌────────────────────────────────────────────┐
   │ Ribbon of chitin teeth on muscular base    │
   │ Moves back and forth like rasp/file        │
   │ Tooth row patterns vary by feeding mode:   │
   │   Herbivore: many small teeth (docoglossat)│
   │   Carnivore: fewer, larger teeth           │
   │   Boring (whelks): drill through shell     │
   │   Cone snail: harpoon-like (venomous)      │
   └────────────────────────────────────────────┘

4. CTENIDIA (gills in mantle cavity)
   ┌────────────────────────────────────────────┐
   │ Ancestral gill form: bipectinate (feather) │
   │ Highly modified in bivalves → filter plate │
   │ Lost in land snails (replaced by lung)     │
   └────────────────────────────────────────────┘
```

### Shell Layers

```
MOLLUSCAN SHELL CROSS-SECTION
+------------------------------------------+
|  Periostracum   Organic protein (quinone)|
|                 Outermost, protective    |
+------------------------------------------+
|  Prismatic     Calcite or aragonite       |
|  layer         Columnar crystals          |
+------------------------------------------+
|  Nacreous      Aragonite platelets +     |
|  layer         protein (conchiolin)      |
|  (mother of    Iridescent interference   |
|   pearl)       NACRE                     |
+------------------------------------------+
|  Interior      Contact with mantle tissue |
+------------------------------------------+

NACRE FORMATION:
  Mantle epithelium secretes calcium carbonate platelets
  Each platelet: ~0.5 μm thick, 5–20 μm wide
  Stacked in offset columns → brick-wall structure
  Light interferes between platelets → iridescence
  Pearl: nacre deposited around irritant (parasite or bead)
  Cultured pearl: bead inserted → mantle wrap → nacre layers
```

## Layer 2 — Gastropoda (Snails and Slugs)

~70,000 described species. Most diverse class. Marine, freshwater, and terrestrial.

### Torsion — The Gastropod Defining Event

```
TORSION: ~180° rotation of visceral mass during development
         (happens in hours to days as larva)

BEFORE TORSION:        AFTER TORSION:
Head   ──► Anus        Head   ──► Anus (above head!)
                       Gills (now above head, facing forward)
                       Mantle cavity (now anterior)

CONSEQUENCE:
  The mouth and anus end up near each other (above head)
  Gills face forward → better water flow
  Mantle cavity (with gills + osphradia) → over head
  Problem: fouling of gills by own waste → "sanitation problem"
  Solutions evolved: slit shells, holes (limpets), operculum

NOTE: Torsion is NOT the same as shell coiling.
Coiling is a separate phenomenon and exists in
both torted and non-torted groups.
```

### Shell Coiling

```
SHELL COILING TERMINOLOGY
+-----------------------------------------------+
|  Dextral (right-handed)  Most common (R→R→R)  |
|  Sinistral (left-handed) Rare (L→L→L)         |
|  Planispiral             Flat coil (Nautilus) |
|  Conical                 High spire           |
|  Limpet                  Cap-shaped, uncoiled |
+-----------------------------------------------+

Coiling handed-ness controlled by single gene (nodal pathway)
Mirror-image snails (sinistral) can't mate with normal dextral
→ reproductive isolation → speciation driver
```

### Gastropod Diversity

| Clade | Shell | Gills | Habitat | Examples |
|-------|-------|-------|---------|---------|
| Prosobranchia (Caenogastropoda) | Yes | Ctenidia | Marine | Cone snails, whelks, periwinkles, limpets |
| Opisthobranchia | Reduced/absent | Modified | Marine | Nudibranchs (sea slugs), sea hares |
| Pulmonata | Yes or no | Lung (not gills) | Land/freshwater | Garden snail, slug, land snail |

### Nudibranchs (Sea Slugs)

```
Opisthobranch gastropods (suborder Nudibranchia)
~3,000 species. Among most colorful animals.

CERATA: dorsal protrusions (secondary gills + defenses)
APOSEMATISM: bright colors = warning (toxic, or borrowed toxins)
  Some eat sponges → store sponge toxins
  Some eat Cnidaria → sequester NEMATOCYSTS from cnidocytes
    (kleptocnidy: stolen cnidocytes remain functional!)

Aeolid nudibranchs: cnidocytes from eaten hydra/jellyfish
migrate through digestive system to cerata tips
→ can sting predators with the prey's own weapons
```

### Cone Snails

```
CONUS (cone snails) — ~800 species

RADULA: modified into HOLLOW HARPOON
  One dart fired at a time
  Connected to venom gland

CONOTOXINS: mix of 50–100+ peptide toxins
  Each species has unique cocktail
  Target: ion channels (Na⁺, K⁺, Ca²⁺), receptors
  Conus geographus: can kill humans ("cigarette snail")
  Paralysis within minutes

MEDICAL USE:
  Ziconotide (Prialt): from Conus magus
  Non-opioid pain drug, 1000× more potent than morphine
  No addiction potential
  ~100 conotoxin-derived drugs in clinical trials
```

## Layer 3 — Bivalvia (Clams, Oysters, Mussels, Scallops)

~20,000 species. NO radula. Both shells lateral.

```
BIVALVE BODY PLAN

LEFT VALVE    VISCERAL MASS    RIGHT VALVE
    |\                              / |
    | \    MANTLE LOBES           / |
    |  \  (secrete shell)        /  |
    |   +------GILLS------------+   |
    |   | (CTENIDIA — also      |   |
    |   |  filter food!)        |   |
    |   +------FOOT-------------+   |
    |   (muscular burrowing     |   |
    |    wedge)                 |   |
    +-----------LIGAMENT--------+
    (elastic hinge connecting valves)

ADDUCTOR MUSCLES: close shell (2 in most; 1 in scallops)
LIGAMENT: elastic, opens shell when muscles relax

FILTER FEEDING:
  Ctenidia modified as GILLS + FILTER:
  Water current → siphons (or shell gape)
  → trapped on mucus-covered gill filaments
  → cilia move particles to mouth
  Filter up to 50 L/day (oyster)
```

### Bivalve Specializations

| Group | Adaptation | Mechanism |
|-------|-----------|----------|
| Mytilidae (mussels) | Attachment | Byssus threads (protein fibers) stronger than steel relative to size |
| Tridacna (giant clam) | Photosynthesis | Zooxanthellae in mantle → clam uses algal products |
| Teredo (shipworm) | Wood boring | Modified foot drills, ctenidia very reduced; cellulose digested by bacteria |
| Pectinidae (scallops) | Swimming | Snap valves shut repeatedly → jet propulsion; 100+ simple eyes |
| Limidae (file clams) | Tentacle display | 100+ brightly colored tentacles that break off and entangle predators |

### Pearl Formation

```
PEARL FORMATION

Natural:   Parasite/irritant → mantle wraps with epithelium
           → nacre secreted around object
           Takes 2–7 years; very rare

Cultured:  Shell bead + mantle tissue fragment inserted
           → nacre deposited around bead
           Akoya (Pinctada fucata): 1–2 years, 6–8mm
           South Sea (Pinctada maxima): 2–4 years, 10–15mm
           Tahitian (Pinctada margaritifera): black lustre

Freshwater: No bead, just tissue; entire pearl is nacre
           Many at once per mussel (up to 50)
           Hyriopsis (China): world's largest freshwater cultured pearl industry
```

## Layer 4 — Cephalopoda (Octopus, Squid, Cuttlefish, Nautilus)

~800 living species. The molluscan apex of neural complexity.

### Body Plan Revolution

```
CEPHALOPOD = "head-foot" — the foot became the arms!

ANCESTRAL MOLLUSK              CEPHALOPOD
  +-----------+               +------------------+
  | Shell     |               | Shell (internal, |
  |           |    →→→        |  reduced, absent)|
  | Foot      |               | Arms/tentacles   |
  |           |               |  (from foot)     |
  | Mantle    |               | Mantle (jets)    |
  +-----------+               +------------------+

LOCOMOTION:
  Water drawn into mantle cavity
  Expelled through muscular funnel (= modified foot)
  JET PROPULSION (fastest invertebrate locomotion)
  Fins on mantle for stability + precise maneuvering

BLOOD:
  Copper-based hemocyanin (blue blood, like Horseshoe crab)
  THREE hearts: one systemic + two branchial (gill hearts)
  CLOSED circulatory system (unique among invertebrates)
```

### Cephalopod Intelligence

The octopus nervous system is a distributed computing architecture: 8 independent arm processors (~37.5M neurons each) with semi-autonomous local control, loosely coordinated by a central bus (the central brain, ~50M neurons). Each arm can execute motor programs (reaching, grasping, manipulation) without central direction — the arm ganglia run local feedback loops. This maps directly to the actor model: each arm is an actor with local state and behavior, receiving high-level goals from the central coordinator but executing without central polling. The central brain does not supervise each motor step — it sends goals and the arm figures out kinematics locally. This is the architecture you'd design for a system where the communication latency to the controller is too high for tight coupling: distribute the control plane.

```
NERVOUS SYSTEM COMPARISON
+------------------------------------------+
|  Animal    | Total neurons |             |
|------------|--------------|              |
|  Nematode  |     302      |              |
|  Fly       |    250,000   |              |
|  Honeybee  |    960,000   |              |
|  Octopus   |  500,000,000 | ← 2/3 in arms|

+------------------------------------------+

OCTOPUS BRAIN DISTRIBUTION:
  ~50M neurons in central brain
  ~30M neurons in optic lobes
  ~300M neurons distributed in EIGHT arms
  Each arm has its own local control (semi-autonomous)

DEMONSTRATED BEHAVIORS:
  Tool use: coconut shell as portable shelter
  Long-term memory: individual human recognition
  Spatial learning: maze solving, delayed gratification
  Play behavior: observed with floating objects
  Camouflage: match texture + color + pattern in <200ms
  Dreaming: color changes during sleep (REM-like?)
  Problem solving: opening jars, escaping tanks
```

### Chromatophores and Camouflage

```
CEPHALOPOD SKIN — THREE LAYERS

Layer 1: CHROMATOPHORES (pigment cells)
  Each contains saccule of pigment (yellow/orange/red/brown/black)
  Ring of 25 radial muscles attached
  Contract → pigment spreads (large spot)
  Relax → pigment contracts (tiny dot)
  ~150,000 chromatophores on 30cm squid

Layer 2: IRIDOPHORES (structural color)
  Stacks of thin plates → interference = iridescent
  Can change color via thin-film effects
  Controlled by acetylcholine

Layer 3: LEUCOPHORES (white reflectors)
  Broadband reflectors
  Underlying muscle texture control

TEXTURE CONTROL:
  Papillae muscles can raise skin to mimic rock textures
  Octopus vulgaris: 15 texture patterns documented

RESULT: active camouflage in ~200ms
        matching complex backgrounds in real time
        The most sophisticated rapid skin display system known
```

## Layer 5 — Monoplacophora, Polyplacophora, Scaphopoda

### Monoplacophora (Single-Cap Shell)

```
Neopilina galeathae: discovered 1952 in deep Pacific trench
Previously known only as fossils (since Silurian)

SIGNIFICANCE: shows serial repetition (like annelids?)
  5–6 pairs gills, 8 pairs foot retractor muscles
  Once interpreted as "missing link" showing annelid-mollusc connection
  Now seen as internal molluscan segmentation, not annelid homolog
```

### Polyplacophora (Chitons)

```
~1,000 species. Oval, 8 articulated shell plates.

BODY PLAN:
  8 overlapping dorsal plates (valves)
  Girdle (mantle edge) around plates
  Broad flat foot → strong suction adhesion
  Radula with MAGNETITE teeth (hardest biological mineral)
    Iron oxide (magnetite) tips for scraping coralline algae

Most are intertidal algae grazers
Some deep sea species: predatory on foraminifera

CAN CURL UP: plates articulate → sphere when detached
```

### Scaphopoda (Tusk Shells)

```
~500 species. Elephant tusk-shaped shell, open both ends.

Burrow in sediment, head-down
Captacula: thin sticky tentacles for catching foraminifera
No gills: gas exchange in mantle cavity

HISTORICAL: Native Americans used as currency (wampum)
```

## Decision Cheat Sheet

```
IDENTIFYING MOLLUSKS

No shell (or just 8 plates), broad adhesive foot, scrapes algae?
  → POLYPLACOPHORA (chiton)

Single coiled or cap shell, one-footed?
  → GASTROPODA
  Land/freshwater, pulmonary sac (no gills)? → Pulmonate (land snail, slug)
  Marine, bright colors, no shell?            → Nudibranch (sea slug)
  Cone-shaped, feeds via harpoon dart?        → Conus (cone snail!)

Two lateral shells, filter feeder, no radula?
  → BIVALVIA
  Attached by byssus threads? → mussel
  Cemented to substrate?      → oyster
  Swims by snapping?          → scallop
  Bores into wood?            → Teredo (shipworm)

Arms/tentacles from head, no external shell (or internal only)?
  → CEPHALOPODA
  8 arms, solitary, benthic? → octopus
  10 arms (8+2), streamlined, schooling? → squid
  External chambered shell?  → nautilus

Tusk-shaped, both ends open?
  → SCAPHOPODA
```

## Common Confusion Points

**Squids and octopuses are mollusks.**
Most people don't make this connection. The foot became the arms. The mantle became the jet-propulsion body. The shell became internal (gladius in squid) or absent entirely (octopus).

**Closed circulation in cephalopods is unusual.**
Almost all mollusks have open circulatory systems (blood bathes organs in sinuses). Cephalopods evolved closed circulation — probably for the higher metabolic demands of active predation. Convergent with vertebrates.

**"Blue blood" is real.**
Cephalopod and horseshoe crab blood uses copper-containing hemocyanin instead of iron-containing hemoglobin. Hemocyanin is blue when oxygenated, colorless when deoxygenated. Less efficient O₂ transport than hemoglobin per unit weight — but produced from seawater copper rather than requiring iron.

**Octopuses don't have suction cups throughout — some have teeth.**
Male octopuses have a modified arm (hectocotylus) for sperm transfer. Some deep-sea octopus arms have teeth-like projections instead of suckers.

**Giant squid die after spawning.**
Cephalopods are semelparous — one reproductive event, then death. The octopus female guards eggs for months without eating, then dies. This is why intelligence doesn't build culturally: no old individuals to learn from.

**Oysters change sex.**
Most bivalves are dioecious. Oysters (Crassostrea) are protandric hermaphrodites: start male, switch to female as they get larger. A single oyster may switch multiple times over its life.
