# Lophotrochozoa — Worms & Lophophore Feeders

## The Big Picture

```
LOPHOTROCHOZOA — defined by two synapomorphies:
  Trochophore larva (shared by many members)
  OR Lophophore (ciliated feeding organ)
  (Not all members have both — it's a molecular clade)

                 BILATERIA
                     |
          +----------+----------+
          |                     |
   PROTOSTOMIA            DEUTEROSTOMIA
          |
   +------+------+
   |             |
LOPHOTROCHOZOA  ECDYSOZOA (molting; next file)

LOPHOTROCHOZOA TREE:
  Platyhelminthes (flatworms)
  Nemertea (ribbon worms)
  Annelida (segmented worms)
  Mollusca (separate file)
  Rotifera
  Bryozoa
  Brachiopoda
  Phoronida
  + several small phyla

KEY INNOVATION GRADIENT:
  Acoelomate (no body cavity)  → Platyhelminthes
  Pseudocoelomate (partial)    → Rotifera
  Eucoelomate (true coelom)    → Annelida, Mollusca
```

## Layer 1 — Platyhelminthes (Flatworms)

~29,000 species. "Flat worm" = acoelomate, dorsoventrally flattened.

### Body Plan

```
PLATYHELMINTH CROSS-SECTION
+----------------------------------------+
|  Epidermis (no cuticle in free-living) |
|  +---------------------------------+   |
|  |  Parenchyma (mesenchyme fill)   |   |
|  |    NO coelom — solid body!      |   |
|  |    Muscle layers                |   |
|  |    Gastrovascular cavity        |   |
|  |    (branched, no anus)          |   |
|  |    Protonephridia (excretion)   |   |
|  |    Ladder-like nervous system   |   |
|  +---------------------------------+   |
+----------------------------------------+
Gas exchange: entirely by diffusion through skin
→ explains flatness (maximizes surface:volume ratio)

THREE GERM LAYERS but NO body cavity
Bilateral symmetry ✓
True brain (cephalization) ✓ — first in animal evolution
```

### Classes

| Class | Lifestyle | Examples | Medical Relevance |
|-------|----------|---------|-----------------|
| Turbellaria | Free-living | Planaria, Dugesia | Model organism for regeneration |
| Monogenea | Ectoparasitic | Gyrodactylus | Fish gill parasites |
| Trematoda (flukes) | Endoparasitic | Schistosoma, Fasciola | Schistosomiasis: 240M people infected |
| Cestoda (tapeworms) | Endoparasitic | Taenia, Echinococcus | Intestinal; cysts in organs |

### Planaria: Model Organism for Regeneration

```
PLANARIAN REGENERATION

Cut into pieces → each regenerates into a complete worm
Head piece → regenerates tail
Tail piece → regenerates head

Mechanism: NEOBLASTS
  ~30% of cells are pluripotent stem cells (neoblasts)
  After injury, neoblasts proliferate + differentiate
  Even a 1/279th fragment can regenerate a whole animal

Used to study:
  Stem cell biology
  Polarity establishment
  Organ regeneration (hopes for human medicine)
```

### Schistosoma: The Worm That Doesn't Hide

```
SCHISTOSOMA LIFE CYCLE

Human host
   |  Adult worms in blood vessels (mesenteric/portal)
   |  Eggs pass through gut → feces → water
   ↓
Freshwater snail (Biomphalaria)
   |  Larval stages: sporocyst → cercariae
   ↓
Free-swimming cercariae
   |  Penetrate human skin in contaminated water
   ↓
Human bloodstream → mature to adults → mate

Pathology: Egg lodging in liver, spleen, bladder
           → granulomas → fibrosis → portal hypertension
           → liver failure in chronic infection

Why hard to treat: worms coat themselves with
host antigens → immune system doesn't see them
```

## Layer 2 — Nemertea (Ribbon Worms)

~1,300 species. Marine, mostly benthic.

```
NEMERTEAN INNOVATIONS

COMPARED TO FLATWORMS:
  +--------------------------------------------+
  | Nemertea adds:                              |
  |   Complete gut (mouth AND anus) ← First!   |
  |   Circulatory system ← First!              |
  |   Proboscis (eversible, for prey capture)  |
  +--------------------------------------------+

PROBOSCIS APPARATUS:
  Stored in rhynchocoel (fluid-filled cavity)
  Everted by hydraulic pressure
  In armed nemerteans: stylet (stabbing weapon)
  Captures prey, pulls it back to mouth

SIZE RANGE:
  Most: 1–20 cm
  Lineus longissimus: up to 55 METERS — longest animal?
  (Not confirmed; may be measurement error)
  Typical longest confirmed: ~8 m

BODY PLAN:
  Acoelomate OR has rhynchocoel (debated as pseudocoelom)
  Simple, closed circulatory system (no heart, cilia-driven)
  Protonephridia
  Brain (bilobed)
```

## Layer 3 — Annelida (Segmented Worms)

~22,000 species. The most complex "worm" grade.

### Defining Innovation: Metamerism

```
ANNELID BODY PLAN
+----------------------------------------------+
|  PROSTOMIUM (head segment, no appendages)    |
|  SEGMENT 1  [Coelom compartment]             |
|  SEGMENT 2  [Coelom compartment]             |
|  ...                                         |
|  SEGMENT n  [Coelom compartment]             |
|  PYGIDIUM (tail, no appendages)              |
+----------------------------------------------+

Each segment contains:
  Coelom compartment (with septum isolating it)
  Nephridia pair (excretion)
  Nerve ganglion pair
  Setae/parapodia (locomotion/gas exchange)
  Circular + longitudinal muscles

METAMERISM = serial repetition of body units
(Also found in Arthropods, and in Vertebrate somites)

TRUE COELOM:
  Fluid-filled cavity lined by mesoderm (peritoneum)
  Functions as hydrostatic skeleton
  Hydraulic locomotion in earthworms
```

### Annelid Classes

| Class | Characteristics | Representatives |
|-------|----------------|-----------------|
| Polychaeta | Parapodia (paddle-like appendages), marine, many setae | Nereis (ragworm), Arenicola (lugworm), tube worms, Christmas tree worms |
| Oligochaeta | Few setae, no parapodia, terrestrial/freshwater | Lumbricus (earthworm), Tubifex |
| Hirudinea | No setae, sucker front and back, fluid coelom | Hirudo medicinalis (medicinal leech), Haemopis |

*(Modern molecular systematics places Oligochaeta + Hirudinea in Clitellata)*

### Earthworm Biology

```
LUMBRICUS TERRESTRIS (common earthworm)

EXTERNAL:
  Clitellum — mucus-secreting band for reproduction
  Setae — 8 per segment, for grip
  Dorsal pore — excretes coelomic fluid for moisture

INTERNAL:
  Brain (fused ganglia) + ventral nerve cord
  Closed circulatory system:
    5 "hearts" (contractile aortic arches)
    Dorsal vessel (blood anterior)
    Ventral vessel (blood posterior)
  Complete digestive system:
    Mouth → pharynx → esophagus → crop → gizzard → intestine → anus

ECOLOGICAL ROLE:
  Soil aeration, mixing, drainage
  Organic matter decomposition
  Darwin studied them for 40 years
  UK has ~2M earthworms per hectare in good soil
```

### Leeches

```
HIRUDO MEDICINALIS — Medicinal Leech

305 segments but only 33 annuli visible externally
Sucker front (mouth) + sucker rear (adhesion)
Hermaphrodite (like earthworms)
Feeds 1–2× per year; meal = 10× body weight

HIRUDIN: anticoagulant in saliva
  Most potent natural thrombin inhibitor known
  → Modern leech therapy: post-surgical reattachment
    to restore venous drainage
  → Lepirudin (drug) derived from hirudin

ANESTHETIC: also in saliva → painless bite
```

## Layer 4 — Rotifera

~2,200 species. Microscopic filter feeders. "Wheel animals."

```
ROTIFER BODY PLAN
+------------------------------------+
|  Coronal cilia (head)              |
|  (spinning = "wheel" illusion)     |
|  ↓ drives water + food to mouth    |
|  Mastax (jaw apparatus)            |
|  Pseudocoelom                      |
|  Protonephridia (flame cells)      |
|  Foot with adhesive toes           |
+------------------------------------+

SIZE: 50–2000 μm (most ~100–500 μm)

REMARKABLE:
  Bdelloid rotifers (one class):
    Reproduce only by PARTHENOGENESIS — no males known
    For ~85 million years!
    Resistant to desiccation (cryptobiosis)
    Survive total dehydration, radiation
    When rehydrated: steal genes from other organisms
    (horizontal gene transfer in metazoans — rare!)
```

## Layer 5 — Lophophore Feeders (Bryozoa + Brachiopoda + Phoronida)

### The Lophophore

```
LOPHOPHORE — the defining structure

Ring of ciliated tentacles around the mouth
Drives water current → traps suspended food particles
No equivalent in ecdysozoans or vertebrates

Unique to:
  Bryozoa (Ectoprocta)
  Brachiopoda
  Phoronida
(+ possibly Kamptozoa/Entoprocta — debated)
```

### Bryozoa (Moss Animals)

```
~6,000 species. Colonial. Almost all marine (one class freshwater).

ZOOECIUM (individual) structure:
  Cystid (body wall + coelom)
  Polypide (lophophore + gut + muscles)

COLONY types:
  Encrusting (sheets on rocks, shells)
  Erect branching (like seaweed)
  Ribbon-like

Each individual (zooid) = 0.5–2 mm
Colony = cm to meters

IMPORTANCE:
  Reef framework contributors (Paleozoic)
  "Lace corals" — common in fossil record
  Bugula produces bryostatin (anticancer research)
```

### Brachiopoda (Lamp Shells)

```
BRACHIOPODS — the dominant Paleozoic "clam" (not a clam)

BIVALVED but:
  Clam valves = LEFT and RIGHT
  Brachiopod valves = DORSAL and VENTRAL

~400 living species (down from ~30,000 fossil species)
Peak diversity: Cambrian–Devonian

Two classes:
  Articulata  Valves locked by tooth-socket hinge
              Pedicle for attachment
  Inarticulata Valves held only by muscles
               Lingula — oldest animal genus (480 Ma unchanged)

Lingula example of STASIS: basically identical to Ordovician fossils
```

## Trochophore Larva — The Unifying Character

```
TROCHOPHORE LARVA
+--------------------------------+
|     Apical tuft (cilia)        |
|                                |
|  Prototroch (equatorial        |
|    ciliary band) ← locomotion  |
|                                |
|  Mouth (ventral)               |
|  Stomach                       |
|                                |
|  Metatroch (second band)       |
|                                |
|  Telotroch (anal band)         |
|  Anus                          |
+--------------------------------+

Found in:
  Polychaeta (annelids)
  Mollusca (before veliger stage)
  Some brachiopods
  Nemertea (pilidium larva = modified trochophore)

The shared trochophore is STRONG evidence for lophotrochozoan clade.
Ecdysozoa never have trochophores — they have molting larvae.
```

## Decision Cheat Sheet

```
Flat body, no anus, no coelom?
  → Platyhelminthes
  Free-living, large brain? → Turbellaria (planarian)
  Has suckers, parasitic?   → Trematoda (flukes) or Cestoda (tapeworms)
  Tapeworm: proglottids (repeating units)? → Cestoda

Elongated, has proboscis, has anus?
  → Nemertea (ribbon worm)

Has segments, setae, true coelom?
  → Annelida
  Parapodia (paddle feet), marine? → Polychaeta
  No parapodia, few setae, terrestrial? → Oligochaeta (earthworm)
  Suckers front + rear, no setae? → Hirudinea (leech)

Microscopic, ciliary crown that looks like spinning wheel?
  → Rotifera

Colonial, tiny zooecia, lophophore in U-shaped gut?
  → Bryozoa

Two shells (dorsal/ventral), attached by pedicle?
  → Brachiopoda
```

## Common Confusion Points

**Tapeworms have no gut.**
They live bathed in the host's digested food. The entire intestinal surface is absorbative tegument. No need for a gut when you're inside one.

**Leeches ≠ bloodsuckers as default.**
Most leech species are predatory (eat invertebrates, swallow whole). ~25% of species are blood-feeding ectoparasites. Medicinal leech therapy is making a comeback in microsurgery.

**Brachiopods ≠ bivalves.**
Both have two shells but they're from different phyla. Brachiopod valves are top/bottom; bivalve (clam) valves are left/right. Brachiopods had 30,000 Paleozoic species; only 400 survive today — exactly the opposite of clams.

**Annelid "segments" ≠ body rings.**
External rings on earthworms don't correspond 1:1 with segments. Each true segment (annulus) is defined internally by septa and coelom compartments.

**The "rhynchocoel" in nemerteans.**
Long debated as either a modified coelom or a unique innovation. Not a true coelom by strict definition (not fully lined with peritoneum). Nemerteans are sometimes called "acoelomate" but the rhynchocoel complicates this.

**Metamerism in annelids vs arthropods.**
Both are segmented protostomes — they share metamerism as a body plan. But the segments are homologous by convergence, not shared ancestry. Annelid segments have independent coeloms; arthropod segments have lost the coelomic compartmentalization.
