# Early Animals — Pre-Bilaterian Grade

## The Big Picture

```
ANIMAL PHYLOGENY — Non-bilaterian grade

                    ANIMALIA
                       |
         +-------------+-------------+
         |             |             |
      Porifera    Ctenophora      Cnidaria
      (sponges)  (comb jellies)  (jellyfish, coral)
         |             |             |
         +------+-------+------+-----+
                |              |
           Placozoa        BILATERIA
           (Trichoplax)    (everything else)

CONTROVERSY: Ctenophora position is debated.
Some molecular trees place them as the FIRST branch
(sister to all other animals), which would imply
neurons evolved TWICE independently.
Others place them sister to Cnidaria.
This is the biggest unresolved debate in animal phylogenetics.

TIMELINE:
~800 Ma   First animal-grade organisms
~635–541  Ediacaran Period — first macroscopic forms
541 Ma    Cambrian explosion — hard parts + predation
```

## Layer 2 — Porifera (Sponges)

### Overview

~9,000 species. Most ancient animal phylum. The basal branch.

```
SPONGE BODY PLAN
+----------------------------------------------+
|                                              |
|    osculum (large exit pore)                 |
|         |                                    |
|    +----+-----+                              |
|    |   spongocoel (central cavity)           |
|    |                                         |
|    |  ←← water current                       |
|    |                                         |
|   incurrent pores (ostia)                    |
|         ↑                                    |
|   choanocytes (collar cells)                 |
|         ↑                                    |
|   water flows in through pores               |
+----------------------------------------------+

KEY: No true tissues. No organs. No nervous system.
     No muscles. No mouth. No gut.
     Water flows in → choanocytes filter bacteria/particles
     → water exits through osculum.

CELL TYPES:
  Choanocytes    Flagellated collar cells; drive water flow + capture food
  Pinacocytes    Flat epithelium-like cells lining the outer surface
  Archaeocytes   Totipotent amoeba-like cells; can become any other cell type
  Sclerocytes    Secrete spicules (the skeleton)
```

### Sponge Classes by Skeleton

| Class | Spicule Material | Representative | Depth/Habitat |
|-------|----------------|----------------|--------------|
| Calcarea | Calcium carbonate | Sycon, Leucosolenia | Shallow marine |
| Demospongiae | Silica + spongin (protein) | Barrel sponge, bath sponge | All depths; ~80% of species |
| Hexactinellida | 6-rayed silica | Venus's flower basket | Deep sea |
| Homoscleromorpha | Silica, no spongin | Oscarella | Shallow, cryptic |

### Sponge Architecture Grades

```
ASCONOID          SYCONOID          LEUCONOID
+--------+        +--------+        +--------+
|        |        |        |        | Many   |
| Simple |        | Folded |        | chambers|
| tube   |        | wall   |        | High   |
|        |        |        |        | pumping |
| Low    |        | More   |        | capacity|
| efficiency|     | cells  |        |        |
+--------+        +--------+        +--------+
Sycon sp.      Scypha sp.         Most demosponges
```

### Ecological Importance

- Filter up to 24,000 L water/day per kg sponge (reef cleaning)
- Reef framework builders (siliceous sponges in Jurassic)
- Chemical defense: produce bioactive compounds (many anticancer drugs)
- Sponge–microbe symbiosis: microbes = up to 50% of sponge biomass
- Deep sea glass sponge reefs: largest biogenic structures after coral reefs

## Layer 3 — Placozoa

One described species: *Trichoplax adhaerens*

```
TRICHOPLAX BODY PLAN — SIMPLEST ANIMAL

+-------------------------------------------+
|  Flat, ~1 mm, no fixed shape              |
|  Crawls on surfaces in flat pseudopod     |
|                                           |
|  FOUR cell types only:                    |
|    Upper epithelium (monociliated)        |
|    Lower epithelium (monociliated)        |
|    Gland cells (digestive enzymes)        |
|    Fiber cells (between layers)           |
|                                           |
|  NO mouth, gut, neurons, muscles          |
|  Feeds by secreting enzymes externally    |
|  Reproduces by fission or budding         |
+-------------------------------------------+

Despite simplicity, has ~11,514 genes.
More gene-rich than expected — secondary reduction,
not primitive state.
```

## Layer 4 — Cnidaria

~11,000 species. Jellyfish, corals, sea anemones, hydra, siphonophores.

### Defining Features

```
CNIDARIAN BODY PLAN

Two body forms (alternating in many species):
+------------------+     +------------------+
|    POLYP         |     |    MEDUSA        |
|                  |     |                  |
|   oral side UP   |     |  oral side DOWN  |
|   attached       |     |  free-swimming   |
|   tentacles up   |     |  tentacles down  |
|                  |     |                  |
|   CORAL          |     |  JELLYFISH       |
|   SEA ANEMONE    |     |  MOON JELLY      |
|   HYDRA          |     |  BOX JELLY       |
+------------------+     +------------------+

ANATOMY:
  Epidermis (ectoderm)
  Mesoglea — jelly layer (acellular, mostly water)
  Gastrodermis (endoderm)
  Gastrovascular cavity — single opening (mouth = anus)
  Cnidocytes — stinging cells (unique to Cnidaria)
  Nerve net — diffuse, no brain, no head
  Radial symmetry (4-fold or 6-fold)

TWO GERM LAYERS ONLY: ectoderm + endoderm (no mesoderm)
```

### Cnidocytes — The Defining Weapon

```
CNIDOCYTE (stinging cell)
+--------------------------------+
|  Cnidocil (trigger hair)       |
|  Operculum (cap)               |
|  Nematocyst (capsule inside)   |
|    Stylet (barb)               |
|    Thread (coiled)             |
+--------------------------------+
Discharge is one of fastest events in biology:
~3 μs, 40,000 g acceleration
Thread penetrates prey, injects venom or entangles

Types:
  Penetrant (stenotele)  — penetrates + injects venom
  Volvent (desmoneme)    — coils around prey
  Glutinant (ptychocyst) — sticky, attaches to substrate
```

### Cnidarian Classes

| Class | Life Cycle | Representatives | Notes |
|-------|-----------|-----------------|-------|
| Hydrozoa | Polyp dominant; medusa small/absent | Hydra, Obelia, Physalia (Portuguese man-o'-war), Turritopsis (immortal jellyfish) | Colonial in many; siphonophores are superorganisms |
| Scyphozoa | Medusa dominant; polyp small | Moon jelly (Aurelia), Lion's mane jelly | "True" jellyfish; polyp → strobilation → ephyra → medusa |
| Cubozoa | Medusa only | Box jellyfish (Chironex) | True eyes with lens + retina; fastest + most venomous cnidarian |
| Anthozoa | Polyp only (no medusa) | Corals, sea anemones, sea fans | Most speciose; reef builders |

### Coral Reef Architecture

```
HERMATYPIC CORAL (reef-building)

Coral polyp
    |
    ↕  mutual symbiosis
Zooxanthellae (dinoflagellates in polyp tissue)
    |
Polyp photosynthesizes via algae + captures plankton
    |
Secretes CaCO₃ skeleton underneath
    |
Colony grows → REEF

CORAL BLEACHING:
Thermal stress → polyp expels zooxanthellae → white skeleton visible
Without algae: polyp starves → death if stress persists
Trigger: SST > normal by ~1°C for 4+ weeks
Great Barrier Reef: 50% bleaching events 2016–2022
```

### Siphonophora: Colonial Superorganism

```
PHYSALIA (Portuguese Man-O'-War)

         Float (pneumatophore)
              |
    +---------+---------+
    |         |         |
  Dactylozooids  Gonozooids  Gastrozooids
  (defense/    (reproduction) (feeding)
   capture)

Each "individual" is a zooid — modified polyp or medusa.
No individual can survive alone.
The "individual" is the entire colony.
Tentacles up to 50m long.
```

## Layer 5 — Ctenophora (Comb Jellies)

~100–200 species. Superficially jellyfish-like but distinct.

```
CTENOPHORE BODY PLAN

Eight rows of CTENES (comb plates)
  = bands of fused cilia for locomotion
  Iridescent when light hits cilia (no bioluminescence)

Bioluminescence: blue-green light from PHOTOPHORES
  (actual bioluminescence, chemically produced)

Body plan:
  Biradial symmetry (not radial like Cnidaria)
  Two tentacles with COLLOBLASTS (sticky cells, not nematocysts)
  Gastrovascular canal system
  Rudimentary apical organ (balance organ = statocyst)
  No Hox genes (if basal position is correct)

LARGEST: Cestum veneris (Venus's girdle) — ribbon up to 1.5m

FEEDING: All are predatory. Catch zooplankton with colloblasts.
  Mnemiopsis leidyi: invasive comb jelly devastated
  Black Sea fisheries after ballast water introduction.
```

### Ctenophora Position Controversy

```
HYPOTHESIS 1 (traditional)
  Porifera
     |
  Ctenophora + Cnidaria
     |
  Bilateria

HYPOTHESIS 2 (molecular, some recent phylogenies)
  Ctenophora ←── FIRST BRANCH
     |
  Porifera
     |
  Cnidaria
     |
  Bilateria

If Hypothesis 2 is correct:
  Nervous systems evolved TWICE independently
  (once in Ctenophora, once in Cnidaria + Bilateria)
  OR Porifera LOST a nervous system
```

## Decision Cheat Sheet

```
IDENTIFYING NON-BILATERIAN ANIMALS

Has pores (ostia), no mouth, filter feeds?
  → PORIFERA (sponge)
  What skeleton? Calcareous → Calcarea
                 Silica + spongin → Demosponge
                 6-rayed silica → Hexactinellida

Flat, ~1mm, no fixed shape, no tissues?
  → PLACOZOA (Trichoplax)

Has stinging tentacles (nematocysts)?
  → CNIDARIA
  Attached to substrate? → POLYP form
  Bell-shaped, swimming? → MEDUSA form
  Polyp only, reef-builder? → ANTHOZOA (coral/anemone)
  "True" free-swimming jelly? → SCYPHOZOA
  Box-shaped bell, true eyes? → CUBOZOA (box jellyfish — dangerous!)
  Long tentacles, colonial float? → Physalia (Hydrozoa)

Has 8 rows of comb plates, sticky tentacles (not stinging)?
  → CTENOPHORA (comb jelly)
  Bioluminescent? → yes (photophores)
```

## Common Confusion Points

**Sponges are animals.**
No brain, no muscles, no mouth — but still animals. Multicellular, heterotrophic, non-photosynthetic, embryonic development. The animal kingdom doesn't require locomotion or nervous systems.

**Portuguese man-o'-war ≠ jellyfish.**
It's a siphonophore (Hydrozoa). "Body" = colony of specialized zooids. Each sting comes from a dactylozooid, each meal digested by a gastrozooid. Phylogenetically distant from true jellyfish (Scyphozoa).

**Coral bleaching vs coral death.**
Bleaching is not immediate death — it's a stress signal. If temperature normalizes within weeks, zooxanthellae return and coral survives. Prolonged bleaching → starvation → death.

**Ctenophores use cilia, not muscles, for swimming.**
The comb rows beat their cilia in coordinated waves. At small scale, cilia-driven locomotion. Not jet propulsion like squid.

**"Immortal" jellyfish (Turritopsis dohrnii).**
Can revert from medusa back to polyp stage under stress — biological immortality in a theoretical sense. But they die in practice from disease and predation. The mechanism (transdifferentiation) is real and studied for aging research.

**Cnidocytes can only fire once.**
Each nematocyst is single-use. After firing, the cell is discarded and a new one grown. Replacement takes ~48 hours.
