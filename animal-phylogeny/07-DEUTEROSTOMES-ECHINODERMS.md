# Deuterostomes — Echinoderms & the Bridge to Chordates

## The Big Picture

```
DEUTEROSTOMIA — the superphylum containing YOU

Defined by embryonic development:
  Blastopore → ANUS (not mouth)
  Radial, indeterminate cleavage
  Coelom from enterocoely (gut pocketing)

DEUTEROSTOME TREE:
  DEUTEROSTOMIA
   |
   +-- Ambulacraria
   |    |
   |    +-- Echinodermata (~7,500 sp)
   |    +-- Hemichordata  (~130 sp)
   |
   +-- CHORDATA (next file)

ECHINODERMATA SUBGROUPS:
  Asteroidea    (sea stars)         ~2,000 sp
  Ophiuroidea   (brittle stars)     ~2,100 sp
  Echinoidea    (sea urchins, sand dollars) ~1,000 sp
  Holothuroidea (sea cucumbers)     ~1,700 sp
  Crinoidea     (feather stars, sea lilies) ~700 sp

HEMICHORDATA SUBGROUPS:
  Enteropneusta (acorn worms)       ~120 sp
  Pterobranchia (colonial tube dwellers) ~30 sp
```

## Layer 1 — Echinodermata: Key Innovations

### Water Vascular System

```
THE WATER VASCULAR SYSTEM (WVS) — uniquely echinoderm

+--------------------------------------------------+
|                                                  |
|  MADREPORITE (pore to outside)                   |
|       |                                          |
|  Stone canal                                     |
|       |                                          |
|  Ring canal (circumoral)                         |
|       |                                          |
|  Five radial canals                              |
|       |                                          |
|  Lateral canals → Ampullae → TUBE FEET           |
|                              (podia)             |
+--------------------------------------------------+

HOW TUBE FEET WORK:
  Ampulla (muscular sac) contracts
  → pushes water into tube foot → foot extends
  Tube foot tip: suckerdisc with adhesive secretion
  Attaches to substrate
  Retractor muscles shorten foot → pull body

  One sea star: ~1,000 tube feet
  Suction force: up to 6 kg per individual
  Used: locomotion, prey capture, gas exchange, sensation

WVS = seawater-based hydraulic skeleton for movement
     Analogous to hydrostatic skeleton of worms but using
     a dedicated plumbing system
```

### Five-Fold Radial Symmetry

```
PENTAMERAL SYMMETRY — 5-fold radial
(unique to adult echinoderms)

     I
    / \
   V   II
   |   |
   IV  III

PARADOX: larvae are BILATERAL
         adults are RADIAL

This is SECONDARY RADIAL SYMMETRY:
  Echinoderm ancestor was bilaterally symmetric
  Radial symmetry evolved IN THIS LINEAGE
  (All other radially symmetric animals → non-bilaterian)

EXPLANATION: sessile/slow ancestor benefited from
  sensing + responding in all directions equally
  5-fold not 4-fold or 6-fold: 5 preserves bilateral-like
  body axes while giving multi-directional capabilities

THE BILATERAL LARVA proves deuterostome origin:
  Bipinnaria (sea star) → radially symmetric adult
  Sea urchin pluteus larva → radially symmetric adult
```

## Layer 2 — Asteroidea (Sea Stars)

~2,000 species. The iconic echinoderm.

```
SEA STAR BODY PLAN

       [Arm]
        |
    [Central disc]──[Arm]
        |       \
       [Arm]   [Arm]
        |
       [Arm]

ORAL SURFACE (bottom): mouth, tube feet
ABORAL SURFACE (top): madreporite, anus, skin gills

INTERNAL:
  Cardiac stomach: everted OUTSIDE to digest prey
  Pyloric stomach + pyloric caeca (2 per arm, digestion)
  Gonads (1–2 per arm)
  Radial nerve cord (per arm)
  Radial canal of WVS (per arm)

FEEDING:
  Prey: bivalves, snails, urchins, coral polyps
  Arms embrace bivalve → tube feet pull valves
  0.1mm gap sufficient → evert cardiac stomach into gap
  Digest prey externally → stomach absorbs nutrients
  Sea star can open a mussel in 5–6 hours
```

### Regeneration

```
ASTERIAS REGENERATION

Full arm regeneration: 6–12 months
Arm + bit of disc can regenerate entire animal
Some species (Linckia) can regenerate from arm alone
("comet" = tiny body on one arm while regenerating)

AUTOTOMY: deliberate arm self-amputation
  Escape predators
  Arm distracts predator while body escapes

CROWN-OF-THORNS (Acanthaster planci):
  12–19 arms, venomous spines
  Feeds on coral polyps
  Population explosions devastate reefs
  Great Barrier Reef: periodic outbreaks kill 40–50% of coral
```

## Layer 3 — Ophiuroidea (Brittle Stars)

~2,100 species. Most speciose echinoderm class.

```
OPHIUROID vs ASTEROID

BRITTLE STAR:                   SEA STAR:
  Disc sharply defined           Disc grades into arms
  Arms clearly distinct          Arms can be broad
  Arms: ossicles + muscles       Arms: no ossicles (soft)
    → snake-like locomotion        → tube foot locomotion
  Tube feet: sensory only        Tube feet: locomotion + feeding
    (no suckers)
  FAST: 2 cm/s possible          SLOW: 0.1 cm/s typical
  Drop arms readily (autotomy)

COMMON BRITTLE STAR (Ophiothrix fragilis):
  Up to 2,000/m² in aggregations
  Feeds by filter feeding (raise arms, trap particles)
  Suspended from coral/sponge communities

BIOLUMINESCENT BRITTLE STARS (Amphipholis):
  Flash when disturbed
  May warn predators (aposematism)
```

## Layer 4 — Echinoidea (Sea Urchins, Sand Dollars)

~1,000 species.

```
SEA URCHIN BODY PLAN

Globular (urchins) or flattened (sand dollars)
Test: rigid endoskeleton of interlocking calcite ossicles
  20 plates in 5 ambulacral + 5 interambulacral columns
  Tube feet protrude through ambulacral pores
  Spines articulate on tubercles (ball-socket joint)
  Aristotle's lantern: 5-part jaw apparatus (inside)

ARISTOTLE'S LANTERN:
  Complex jaw with 5 teeth
  Rasps algae, kelp, encrusting organisms from rock
  Teeth grow continuously to compensate for wear
  Can excavate rock (boring urchins)

PEDICELLARIAE:
  Pincer-like structures on surface
  Grasp + inject venom (in some species)
  Keep surface clean of settling larvae/debris

STRONGYLOCENTROTUS PURPURATUS (purple urchin):
  Model organism: embryology, gene regulation
  Genome sequenced (2006); 23,300 genes (~human count)
  Immune system unexpectedly complex: 222 Toll-like receptor genes
    (humans: 10 TLR genes)
  Gametes easily obtained, fertilization external → developmental studies

SAND DOLLARS (Clypeasteroida):
  Secondary bilateral symmetry (ecological convergence)
  Burrow in sand
  Tube feet for food collection, not locomotion
```

## Layer 5 — Holothuroidea (Sea Cucumbers)

~1,700 species. The "recyclers" of the deep.

```
SEA CUCUMBER BODY PLAN

+----------------------------------------------+
|  Modified echinoderm (elongated body)        |
|  Oral tentacles (modified tube feet)         |
|  for feeding                                 |
|                                              |
|  5 rows of tube feet along body              |
|  Calcite ossicles in skin (reduced)          |
|  Skin texture: "mutable collagenous tissue"  |
|    Can become liquid (autotomy)              |
|                                              |
|  RESPIRATORY TREES: internal                 |
|    (unique gas exchange organ)               |
|    Water pumped in/out through cloaca        |
+----------------------------------------------+

DEFENSE MECHANISMS:
  1. Evisceration: expel internal organs through mouth or anus
     Organs regenerate in weeks
     Cuvierian tubules: sticky threads from respiratory tree
     Entangle predators in seconds
  2. Liquefaction: entire body becomes soft (mutable collagen)
  3. Holothurin: toxic saponins in body wall → poison predators

ECOLOGICAL ROLE:
  Bioturbators: process sediment, recycle nutrients
  Deep sea: some species = 90% of megafaunal biomass
  Commercial: bêche-de-mer (dried sea cucumber) = ~$3,000/kg
  (~30,000 tonnes/year harvest globally)

MUTABLE COLLAGENOUS TISSUE (MCT):
  Unique to echinoderms
  Collagen fibrils with proteoglycan matrix
  Nervous system controls cross-linking state
  → firm (normal) → soft (autotomy/burial) → stiff (defense)
  Studied for potential drug-controlled sutures/implants
```

## Layer 6 — Crinoidea (Feather Stars, Sea Lilies)

~700 species. Living fossils, most ancient echinoderm class.

```
CRINOID BODY PLAN

Sea lily (stalked):           Feather star (free):
  Holdfast                      No stalk (as adult)
  Stalk (may be 1m)             Cirri for temporary grip
  Calyx (body)                  Can swim (arm-waving)
  Pinnules (feathery arms)
  Tube feet (food)

FEEDING: suspension feeding
  Tube feet with mucus traps food particles
  Cilia move particles to ambulacral groove → mouth
  Oriented into current for optimal filtration

FOSSIL RECORD:
  Abundant from Ordovician
  "Crinoid limestone" = rocks mostly made of crinoid ossicles
  Sea lily forests of Carboniferous seas (like modern kelp forests)
  Mass mortality events → preserved whole: Jurassic Holzmaden (Germany)

AMBULACRAL GROOVE: open food gutter on arms → food to mouth
  Same groove homologous to asteroid tube foot channel
```

## Layer 7 — Hemichordata: The Bridge to Chordates

~130 species.

### Enteropneusta (Acorn Worms)

```
ACORN WORM BODY PLAN
+--------------------------------------------+
|  Proboscis (muscular, digging)             |
|  Collar (with mouth)                       |
|  Trunk (long, burrowing)                   |
|                                            |
|  CHORDATE PRECURSOR FEATURES:              |
|  ├── Pharyngeal gill slits                 |
|  |   (filter feeding + gas exchange)       |
|  ├── Dorsal nerve cord (in some species)   |
|  |   (stomochord ≠ notochord but homolog?) |
|  └── Post-anal tail (some species)         |
+--------------------------------------------+

GILL SLITS: identical function to chordate pharyngeal slits
  (filter feeding; ancestral function was feeding, not breathing)
  Strong molecular homology to chordate pharynx

STOMOCHORD: hollow anterior outgrowth
  Long debated as notochord homolog
  Molecular: shares Brachyury + Fox genes with chordate notochord
  Now considered TRUE homolog — notochord = evolutionary derivative

SIZE: 10 cm to 2.5 m (Balanoglossus gigas)
HABITAT: marine sand/mud burrows
ACORN smell: distinctive iodine/bromide compounds (halogenated)
```

### Pterobranchia

```
~30 species. Colonial, tube-dwelling.
Mostly deep sea, some shallow.

Rhabdopleura, Cephalodiscus

BODY PLAN:
  Resembles bryozoan (convergent)
  Collar with lophophore-like arms + tentacles
  Stomochord present
  Gill slits: only 1–2 pairs (vs many in enteropneusts)

FOSSIL CONNECTION:
  Graptolites: extinct colonial hemichordates
  Carbon films in shale (Ordovician–Silurian)
  Now known = colonial pterobranch-like forms
  Important Paleozoic stratigraphic markers
```

## Echinoderm Fossil Record Highlights

```
TIMELINE:
  Cambrian 535 Ma  First echinoderms (Helicoplacus — spiral)
                   Early forms NOT pentameral (bilateral + 3-fold)
  Ordovician       Crinoid "forests," sea star + echinoid radiation
  Devonian         Mass extinction reduced diversity
  Carboniferous    Crinoid limestone → km-thick deposits
  Permian          Great Dying wiped out many classes
                   Blastoids (5-grooved stalked) went extinct
  Mesozoic         Irregular echinoids (sand dollars) radiate
  Present          5 living classes (down from ~20 in Paleozoic)

HELICOPLACUS (Cambrian):
  Spiral-coiled, tri-radial symmetry
  NOT pentameral — shows pentamery evolved within echinoderms
  Moves by screwing into sediment
```

## Decision Cheat Sheet

```
ECHINODERM OR HEMICHORDATE IDENTIFICATION

Has 5-fold radial symmetry, calcite endoskeleton?
  → ECHINODERMATA

  Star-shaped, flexible arms grading into disc?
    → ASTEROIDEA (sea star)
    Venomous, 12+ arms? → Crown of thorns (Acanthaster)

  Star-shaped, arms sharply distinct from disc?
    → OPHIUROIDEA (brittle star)

  Spherical/oval, rigid test, long spines?
    → ECHINOIDEA (sea urchin)
    Flattened, no spines? → Sand dollar

  Elongated, no spines, tentacles around mouth?
    → HOLOTHUROIDEA (sea cucumber)

  Stalked or free-swimming feathers?
    → CRINOIDEA (feather star / sea lily)

Worm-shaped, buried in sand, with gill slits?
  → HEMICHORDATA (acorn worm)
  Colonial, in tubes, lophophore-like? → Pterobranchia
```

## Common Confusion Points

**Adult echinoderms are radially symmetric but bilaterally symmetric as larvae.**
This proves they belong in Bilateria. The radial symmetry is secondary — evolved within the echinoderm lineage, presumably for a sessile/slow-moving lifestyle where directionality mattered less.

**Sea cucumbers breathe through their anus.**
The respiratory tree (unique organ) draws oxygenated water in and out through the cloaca (posterior opening). This is also why some fish (pearlfish, Carapus) live inside sea cucumber intestines — protected habitat, oxygenated water.

**Echinoderms have no true brain.**
Nerve ring around the mouth + radial nerve cords per arm. Decentralized processing. Removing the nerve ring doesn't stop tube foot movement in each arm — they continue independently.

**Hemichordate gill slits = chordate pharyngeal slits.**
Molecular and morphological evidence now strongly supports homology. The pharyngeal gill slits in your embryo are the same structure as the gill slits in an acorn worm. Their original function was filter feeding; breathing came later in the vertebrate lineage.

**"Sea lily" is not a plant.**
Crinoids are animals that look like flowers growing from the seafloor. They've been mistaken for plants since ancient times. The "stem" is an actual animal body; the "petals" are arms used for filter feeding.

**Brittle stars are the most numerous echinoderms.**
2,100 species, highest class diversity. Also form dense aggregations (2,000/m²) in certain habitats. Their abundance in the deep sea makes them ecologically important as detritivores and prey for fish.
