# Nematoda & Ecdysozoa — The Molting Superphylum

## The Big Picture

```
ECDYSOZOA — defined by one character:
  ECDYSIS (molting of a rigid cuticle)
  Cuticle secreted by hypodermis, shed to allow growth

                    ECDYSOZOA
             +----------+----------+
             |                     |
     CYCLONEURALIA               PANARTHROPODA
    (no legs)                    (legs)
         |                           |
    +----+----+              +-------+-------+
    |         |              |       |       |
Nematoda  Kinorhyncha   Onychophora Tardigrada Arthropoda
Nematomorpha Priapulida    (velvet     (water    (next file)
Loricifera                  worms)     bears)

DIVERSITY SNAPSHOT:
  Nematoda:     1,000,000+ estimated species (25,000 described)
  Arthropoda:   1,100,000+ described species
  Together:     ~80–90% of all described animal species
  Tardigrada:   ~1,300 described
  Onychophora:  ~180 described

KEY INNOVATION:
  Rigid cuticle → protection, but limits growth
  Solution: periodic MOLTING (ecdysis)
  Hormone control: ecdysteroids
  Cuticle material: chitin in arthropods, mostly collagen in nematodes
```

## Layer 1 — Nematoda (Roundworms)

### Body Plan

```
NEMATODE CROSS-SECTION
+-------------------------------------------+
|  Cuticle (collagen-rich, NOT chitin)      |
|  Hypodermis (secretes cuticle)            |
|  Muscle cells (only longitudinal!)        |
|  Pseudocoelom (fluid-filled, under        |
|    pressure — hydrostatic skeleton)       |
|  Gut: mouth → pharynx → intestine → anus |
|  Excretory system (H-shaped canals)       |
|  Nerve ring + dorsal/ventral cords        |
+-------------------------------------------+

KEY FEATURES:
  Pseudocoelomate (coelom NOT lined by peritoneum)
  ONLY LONGITUDINAL muscles → sinusoidal locomotion
  No circular muscles → can't shorten body
  Round cross-section (vs flatworm)
  Cuticle molted 4× during development
  Most are dioecious (separate sexes)

LOCOMOTION: body waves in dorsoventral plane
  (contrast with earthworm: peristalsis in any substrate)
```

### C. elegans — The Most Important Research Animal

```
CAENORHABDITIS ELEGANS — why it matters

GENETICS:
  ~100 million bp genome, ~20,000 genes (similar to humans)
  Transparent body → GFP reporter visualization
  Hermaphroditic (self-fertilize) + occasional males
  Short generation time: 3.5 days at 20°C
  Huge laboratory populations, easy genetics

COMPLETE CELL LINEAGE:
  Adult hermaphrodite: EXACTLY 959 somatic cells
  Every cell division from zygote mapped: Sulston + Horvitz 1983
  Nobel Prize 1998 (apoptosis discovery)
  131 cells die by programmed cell death during development

CONNECTOME COMPLETELY MAPPED:
  302 neurons, 7,000+ synapses — fully wired
  First complete nervous system ever mapped (White et al. 1986)
  "Functional connectome" still being decoded

MODEL FOR:
  Aging (DAF-16/FOXO pathway — found in humans too)
  Apoptosis (CED-3/CED-4 → caspases in humans)
  RNAi (Fire + Mello, Nobel 2006)
  Muscle disease (myosin mutations analogous to humans)
```

```
NEMATODES ARE EVERYWHERE
+----------------------------------------------+
| Environment          | Density               |
|----------------------|-----------------------|
| Agricultural soil    | 4–20 million/m²       |
| Forest soil          | 1–10 million/m²       |
| Ocean sediment       | 100,000–10M/m²        |
| Freshwater           | 10,000–1M/m²          |
| Compost              | Up to 90M/m²          |
| Deep subsurface rock | Up to 12 km depth     |
+----------------------------------------------+

FEEDING GUILDS:
  Bacterivores — eat bacteria (most common)
  Fungivores — eat fungi
  Omnivores — eat various microbes
```

```
NEMATODE INFECTION STRATEGY:

Ascaris life cycle:
  Eggs in feces → soil → ingested → hatch in intestine
```

### Plant-Parasitic Nematodes

```
AGRICULTURAL IMPACT: ~$157 billion/year in crop losses

Root-knot nematodes (Meloidogyne):
  Enter young roots, induce GIANT CELLS
  Root swells into "galls"
  Disrupts water/nutrient transport
  Affects >5,000 plant species, every crop

Cyst nematodes (Heterodera, Globodera):
  Female body becomes protective cyst after death
  Potato cyst nematode: survives 20+ years in soil

Burrowing nematode (Radopholus):
  Destroys banana plantations
```

## Layer 2 — Tardigrada (Water Bears)

~1,300 described species. The extremophile champions.

```
TARDIGRADE BODY PLAN

+---------------------------------------+
|  0.1–1.5 mm (most ~0.5 mm)           |
|  4 pairs of stubby legs               |
|  Each leg: 4–8 claws or suckers       |
|  No circular muscles                  |
|  No true coelom (hemocoel)            |
|  Stylets + sucking pharynx for feeding|
|  No blood vessels, no gills           |
|  Gas exchange: direct diffusion       |
+---------------------------------------+

FEEDING: puncture cells, suck contents
  Herbivores: algae, mosses
  Bacterivores, some predatory

LOCOMOTION: "bear walking" — lobopodial legs
  (same type as onychophorans and arthropod ancestors)
```

### Cryptobiosis — The Extreme Survival State

```
CRYPTOBIOSIS ("hidden life")

Normal state:           Cryptobiosis (tun stage):
  Active, hydrated  →→→   Completely dehydrated
  Metabolic rate        Metabolic rate: ~0.01% normal
  ~100% normal          DNA repair ongoing, no other activity

WHAT TARDIGRADES SURVIVE:
+--------------------------------------------------+
| Condition          | Value          | Human limit |
|--------------------|----------------|-------------|
| Vacuum of space    | ~0 Pa          | ~1 Pa deadly|
| Temperature high   | 150°C (briefly) | ~42°C lethal|
| Temperature low    | -272°C (1°C > 0K!)| ~-1°C death|
| Radiation          | 570,000 rad    | 1,000 rad lethal|
| Pressure           | 6,000 atm      | ~60 atm lethal|
| Desiccation        | 100% dehydrated| ~10% lethal |
| Chemical           | Ethanol, HCl   | Very toxic  |
+--------------------------------------------------+

MECHANISMS:
  Trehalose synthesis: sugar replaces water in cells
    Vitrification: cellular contents become glass
  DNA repair enzymes: active even in tun stage
  Dsup protein: binds chromosomes, shields from radiation
    When Dsup gene expressed in human cells → 40% less DNA damage
  Intrinsically Disordered Proteins (IDPs): form gel under stress

LONGEVITY IN TUN STATE:
  Cysts revived after 30 years of storage
  Fossil record: 16-million-year-old tardigrade in amber
  Revived from 24,000-year-old Siberian permafrost (2021)
    (This was actually a rotifer, but tardigrade records are similar)
```

## Layer 3 — Onychophora (Velvet Worms)

~180 described species. Found in humid tropical forests worldwide.

```
ONYCHOPHORAN BODY PLAN — THE ARTHROPOD SISTER

+-------------------------------------------+
|  Soft, velvety cuticle (thin, flexible,   |
|    NOT rigid — molts but rarely)          |
|  14–43 pairs of lobopods (stubby legs)    |
|    Each lobopod: pair of claws at tip     |
|  Slime papillae (unique weapon)           |
|  Head: three pairs appendages:            |
|    Antennae, jaw-like jaws (mandibles),   |
|    Oral papillae (slime cannons)          |
+-------------------------------------------+

LOCOMOTION: hydraulic (like earthworms)
  Coelom used as hydrostatic skeleton
  BUT legs make them arthropod precursor body plan

SLIME CANNON:
  Shoots sticky protein threads up to 30 cm
  Entangles prey (crickets, woodlice, worms)
  Can shoot binocularly — convergent aim!
  Slime re-eaten and recycled after use

OPEN CIRCULATORY SYSTEM: hemocoel + tubular heart
TRACHEAE for gas exchange (like insects — convergent)
```

### Phylogenetic Significance

```
ONYCHOPHORA — "LIVING FOSSIL" OF BODY PLAN EVOLUTION

Position in tree:
  [Annelida] — polychaetes have parapodia-like appendages
       |
  [Onychophora] ← lobopods (intermediate)
       |
  [Arthropoda] ← jointed, rigid appendages

Evidence for arthropod-sister:
  Both ecdysozoan (molt)
  Both have legs
  Similar head organization
  Molecular data strongly supports sister relationship

Onychophora + Tardigrada = Panarthropoda sister groups
  before true arthropod jointing evolved

Cambrian "lobopodians" (Hallucigenia, Anomalocaris)
  represent stem-group panarthropods
```

## Layer 4 — Minor Ecdysozoan Phyla

### Priapulida (Penis Worms)

```
~20 living species. Marine. Among oldest animal fossils.

Body: eversible proboscis covered in recurved teeth
      trunk with 30–40 scalid rows
Named for resemblance to human anatomy (not subtle)

Sedentary ambush predators in soft marine sediment

FOSSIL IMPORTANCE:
  Cambrian fauna dominated by priapulids
  Ottoia (Burgess Shale) — common soft-bodied predator
  Scalidophoran phylum, primitive ecdysozoan
```

### Kinorhyncha (Mud Dragons)

```
~250 described species. Microscopic marine sediment dwellers.

Body: 13 segments (unique among ecdysozoans in having a fixed
      segment number)
Head: eversible, with rings of scalids
Locomotion: anchor front → pull → anchor rear

No known parasitic species. Bacterivores.
Size: 0.1–1 mm
```

### Loricifera

```
Discovered 1983 — newest invertebrate phylum.
~30 described species.

Live INSIDE the interstitial spaces of marine sediment grains
At least one species lives in ANAEROBIC basin (no mitochondria
  — uses hydrogenosomes instead)

Body enclosed in lorica (armor-like structure)
```

### Nematomorpha (Horsehair Worms)

```
~350 described species.

Larvae: parasitic inside arthropods (crickets, mantids)
Adults: free-living in freshwater

BEHAVIOR MANIPULATION:
  Parasite manipulates host to seek water (drowning)
  Then erupts from host to reproduce in water
  Parasites use host neurotransmitter mimicry
  → "Zombie cricket" effect

Adults: do NOT eat (no functional gut)
        live only to reproduce
        up to 1 meter long in some species
```

## Decision Cheat Sheet

```
ECDYSOZOAN IDENTIFICATION (non-arthropod)

Round in cross-section, no legs, sinusoidal movement?
  → NEMATODA
  Free-living, <1 mm, soil/water? → probably free-living nematode
  In intestinal tissue → likely parasitic (Ascaris, hookworm, etc.)
  On plant roots, causing galls? → root-knot nematode (Meloidogyne)

Tiny (<1.5 mm), 4 pairs stubby legs, claws?
  → TARDIGRADA
  In a dehydrated barrel-shape (tun)? → cryptobiosis state

Soft, velvety body, many pairs stubby legs (>8)?
  Shoots sticky slime? → ONYCHOPHORA (velvet worm)

Very long, thread-like, in freshwater or dead arthropod?
  → NEMATOMORPHA (horsehair worm)

Marine sediment, eversible toothed proboscis?
  → PRIAPULIDA

Microscopic, exactly 13 segments, marine sediment?
  → KINORHYNCHA
```

## Common Confusion Points

**Roundworms (nematodes) ≠ earthworms (annelids).**
Both are called "worms" but from different superphyla. Earthworm = Lophotrochozoa, segmented, true coelom. Roundworm = Ecdysozoa, unsegmented, pseudocoelom, molting cuticle.

**C. elegans has only 302 neurons — but the connectome isn't solved.**
The wiring diagram is complete (every synaptic connection mapped). But which behaviors arise from which circuit patterns is still an active research field. "Knowing the wiring diagram" ≠ "understanding the system" — a lesson for neuroscience.

**Tardigrades in space.**
In 2007, Tardigrades survived 10 days of direct space vacuum + UV radiation exposure on the FOTON-M3 spacecraft. Most died from UV; ~70% survived vacuum alone. They don't fly through space; they'd need a vehicle. But the experiment confirmed cryptobiosis extends to near-vacuum.

**The "water bear genome" controversy.**
A 2015 paper claimed ~17% of the tardigrade genome came from horizontal gene transfer (HGT) from bacteria. Re-analysis showed ~1.2% — still high for animals but not exceptional. The initial contamination artifacts inflated the number dramatically.

**Nematodes have no circular muscles.**
This is why they can't crawl like earthworms. An earthworm extends by contracting circular muscles; a nematode undulates in S-curves using only dorsoventral longitudinal muscles pressing against the stiff cuticle and pseudocoelomic fluid.

**Onychophora are "living fossils" of body plan, not genetics.**
Their genome evolves at normal rates. The morphology is conserved because their ecological niche (humid-forest generalist predator) hasn't demanded change. The velvet-worm body plan works.
