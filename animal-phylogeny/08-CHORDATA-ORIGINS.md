# Chordata Origins — Tunicates, Lancelets, and the Vertebrate Spark

## The Big Picture

```
PHYLUM CHORDATA — ~60,000 species

Four defining features (present at SOME life stage):
+----------------------------------------------------------+
|  1. NOTOCHORD    Stiff but flexible rod of cells         |
|                  Provides support + muscle attachment    |
|                  (replaced by vertebral column in most   |
|                   vertebrates, but persists in discs)    |
|                                                          |
|  2. DORSAL HOLLOW NERVE CORD                             |
|     Hollow tube above notochord                          |
|     (vs solid, ventral nerve cord in annelids/arthropods)|
|                                                          |
|  3. PHARYNGEAL SLITS (gill slits)                        |
|     Openings in pharyngeal wall                          |
|     Original function: filter feeding                    |
|     Secondarily: gas exchange in fish                    |
|     Vestigial: in amniote embryos only                   |
|                                                          |
|  4. POST-ANAL TAIL                                       |
|     Extension of body posterior to anus                  |
|     Contains muscle, notochord                           |
|     Often reduced in adult (vestigial coccyx in humans)  |
+----------------------------------------------------------+

CHORDATE TREE:
           CHORDATA
      +-------+--------+
      |        |        |
 Urochordata Cephalo- Vertebrata
 (tunicates)  chordata  (with backbone)
             (lancelets)
      |
   AMBULACRARIA (echinoderms + hemichordates)
   = sister group of Chordata
```

## Layer 1 — Urochordata (Tunicates / Sea Squirts)

~3,000 species. Basal chordates — but they don't look like it.

### The Adult vs Larva Paradox

```
TUNICATE LIFE CYCLE

LARVA (tadpole stage):          ADULT (sessile stage):
  Free-swimming, ~1 mm          Sessile, filter feeder
  +-----------------------+     +----------------------+
  |  Notochord ✓          |     | Notochord LOST       |
  |  Dorsal nerve cord ✓  |     | Nerve cord → ganglion|
  |  Pharyngeal slits ✓   |     | Pharyngeal slits ✓   |
  |  Post-anal tail ✓     |     | Tail LOST            |
  |  Simple eye (ocellus) |     | Tunic (test)         |
  |  Statocyst (balance)  |     | Siphons (in/out)     |
  +-----------------------+     +----------------------+
  Metamorphosis: seconds to minutes
  Larva finds substrate, attaches head-first
  Most chordate features RESORBED during metamorphosis

INTERPRETATION:
  All four chordate features present in larva
  → Confirms tunicate = chordate
  Adult is secondarily simplified (regression)
```

### Tunicate Body Plan (Adult)

```
CIONA INTESTINALIS (solitary tunicate)

Oral siphon (water in)
      ↓
Pharynx (basket of gill slits)
  = giant filter: millions of cilia move water
  Mucus secreted by endostyle traps particles
  Endostyle = HOMOLOG of vertebrate thyroid gland
      ↓
Stomach + Intestine
      ↓
Atrial siphon (water out)

TUNIC (test): cellulose-like polysaccharide (tunicin)
  Unique in animals to produce cellulose
  Tough outer covering → name "Urochordata"
  Tunicin is a structural polysaccharide like plant cell wall

BLOOD: contains vanadium (in some species) — unique in biology
  Vanadocytes accumulate vanadium 10⁷× above seawater
  Function unknown (possibly antifouling?)
```

### Colonial Tunicates

```
BOTRYLLUS SCHLOSSERI (star tunicate)
  Colonies of zooids arranged in star patterns
  Share common circulatory system
  Can fuse with genetically compatible colonies
  CHIMERISM: fused colonies compete for germline cells
    → One clone's germ cells may "win" and reproduce
    while other's somatic cells serve the colony
  Model for allorecognition + cancer biology

PYROSOMA (colonial pelagic tunicates)
  Colonies form luminescent cylinders up to 20 m
  Each zooid beats cilia → collective jet propulsion
  Bioluminescent: entire colony flashes in blue-green

DOLIOLIDA + SALPIDA (salps)
  Pelagic, barrel-shaped
  Contract muscles → jet propulsion
  Alternate asexual + sexual generations
  Salp blooms: millions/km², rapid population explosion
  Fecal pellets: rapid carbon sinking (biological pump)
```

## Layer 2 — Cephalochordata (Lancelets / Amphioxus)

~32 described species. The textbook chordate. Branchiostoma lanceolatum.

```
BRANCHIOSTOMA (AMPHIOXUS) BODY PLAN:

  Oral hood with cirri (tentacles) at front.
  Body axis features:
    - Notochord (full length, persists as ADULT)
    - Dorsal hollow nerve cord (above notochord)
    - Pharyngeal slits (>100 pairs!)
    - Myomeres (V-shaped muscle blocks, segmented)
    - Hepatic cecum (liver homolog?)
  Post-anal tail ✓ at rear.

ALL FOUR CHORDATE FEATURES PRESENT IN ADULT — diagnostic.

SIZE: 3–6 cm
HABITAT: burrows into marine sand, filter feeds
REPRODUCTION: external fertilization; no larva (direct dev.)

MYOMERES: V-shaped muscle blocks
  Same plan as fish trunk muscles
  Notochord for muscle attachment (no vertebrae yet)
  → Undulatory swimming by myomere contraction

GILL BARS: ~100+ pairs of pharyngeal slits
  Water in → trapped on mucus/endostyle
  Cilia move food to gut
  Endostyle: iodine-binding, thyroid-homolog — biochemically
```

### Why Amphioxus Matters

```
AMPHIOXUS AS MODEL FOR VERTEBRATE ANCESTOR

Genome reveals deep homology:
  Has Hox genes in a single cluster (vertebrates have 4 clusters
    from two rounds of whole-genome duplication)
  Has NK2 (cardiac transcription factors), PAX, FOXQ2
  All in single copies — pre-duplication state

VERTEBRATE ORIGIN HYPOTHESIS:
  Amphioxus-like ancestor
    + Two rounds whole-genome duplication (2R WGD)
    + Evolution of NEURAL CREST CELLS
    → ancestral vertebrate

The "2R WGD" theory (Ohno 1970):
  Vertebrates have 2–4× as many gene family members as amphioxus
  Evidence: 4 Hox clusters in jawed vertebrates
            (amphioxus: 1; lamprey: 3; gnathostomes: 4)
```

## Layer 3 — Vertebrata: Origin and Defining Features

### The Vertebrate Body Plan

```
VERTEBRATA — key innovations over non-vertebrate chordates:

+------------------------------------------------------+
| 1. VERTEBRAL COLUMN                                  |
|    Bone or cartilage replacing notochord             |
|    (notochord persists between vertebrae in adults   |
|     as nucleus pulposus of intervertebral discs)     |
|                                                      |
| 2. CRANIUM (skull)                                   |
|    Braincase of bone or cartilage                    |
|    Protects enlarged brain                           |
|                                                      |
| 3. NEURAL CREST CELLS                                |
|    Unique migratory cell population                  |
|    See below — the "fourth germ layer"               |
|                                                      |
| 4. ENLARGED BRAIN                                    |
|    Forebrain + midbrain + hindbrain differentiation  |
|    (not just a nerve cord tube)                      |
|                                                      |
| 5. PAIRED SENSORY ORGANS                             |
|    Eyes (camera eyes), ears (labyrinth), lateral line|
|    Nose                                              |
|                                                      |
| 6. TWO ROUNDS WHOLE-GENOME DUPLICATION               |
|    Massively expanded gene toolkit                   |
+------------------------------------------------------+
```

### Neural Crest Cells — The Fourth Germ Layer

```
NEURAL CREST: vertebrate-specific migratory cell population

Origin: junction of neural plate + epidermis during neurulation
Migration: delaminate, migrate throughout embryo
Differentiation: most diverse fate of any cell type

NEURAL CREST DERIVATIVES:
+-----------------------------------------------------------+
| Craniofacial skeleton  Cartilage + bone of skull + face   |
| Teeth (dentine)        Odontoblasts from neural crest     |
| Peripheral nervous     Dorsal root ganglia, ANS ganglia   |
|   system               Schwann cells (myelination)        |
| Melanocytes            Skin pigment cells (all of them)   |
| Adrenal medulla        Chromaffin cells (adrenaline)      |
| Smooth muscle of       Cardiac outflow tract              |
|   great vessels                                           |
| Cornea                 Keratocytes in eye                 |
+-----------------------------------------------------------+

WHY IT MATTERS:
  Neural crest enabled the vertebrate head
  Craniofacial cartilage from neural crest → jaws, ear bones
  Jaws were originally gill arch cartilage redirected
  Neural crest disruption → neurocristopathies:
    DiGeorge syndrome, Waardenburg syndrome, Hirschsprung's disease

NEURAL CREST IN PHYLOGENETIC CONTEXT:
  Absent in amphioxus (the key difference)
  Rudimentary in lamprey (all gill arches, some pigment)
  Full neural crest in gnathostomes (jawed vertebrates)
```

Whole-genome duplication is a fork() of the entire codebase at a single event. Every gene now has a paralog — an identical copy that immediately starts diverging because selective pressure on each copy is relaxed (the other copy maintains the original function). Paralogs diverge like feature branches that never merge back: one copy retains the original function, the other accumulates mutations freely and may eventually acquire a new function (neofunctionalization), a subset of the original function (subfunctionalization), or become a pseudogene. The vertebrate 2R WGD explains why vertebrate genomes have four HOX clusters (vs. one in amphioxus) and four copies of most ancient regulatory genes — each copy a branch of the original that has diverged since ~550 Ma.

### Two Rounds of Whole-Genome Duplication (2R WGD)

```
VERTEBRATE GENOME EXPANSION

Amphioxus (pre-WGD):
  1 copy of HOX cluster
  1 copy of most regulatory genes

Lamprey (1–2 rounds, debated):
  ~3 HOX clusters

Jawed vertebrates (2 rounds confirmed):
  4 HOX clusters (HoxA, HoxB, HoxC, HoxD)
  4 copies of many transcription factor families
  (Pax, Fox, Dlx, Nkx, etc.)

EVIDENCE FOR 2R WGD:
  ~4-fold redundancy of gene families
  Block synteny: regions on different chromosomes
    share similar gene arrangements (paralogs)
  HOX cluster expansion: 1 → 4 paralogy groups

CONSEQUENCE:
  Duplicate genes can diverge in function (neofunctionalization)
  Or one copy becomes redundant → mutation tolerance
  Or subfunctionalize (each does part of ancestor's job)
  → Massively expanded developmental toolkit
  → New cell types, new structures, new complexity possible
```

## Layer 4 — Early Vertebrate Evolution

```
VERTEBRATE PHYLOGENY:

  VERTEBRATA
   |
   +-- AGNATHA (jawless): Lampreys, Hagfish
   |
   +-- GNATHOSTOMATA (jawed)
        |
        +-- CHONDRICHTHYES (cartilaginous): Sharks, rays
        |
        +-- OSTEICHTHYES (bony fishes)
             |
             +-- Actinopterygii (ray-finned)
             +-- Sarcopterygii (lobe-finned)
                  |
                  +-- TETRAPODA (Amphibia + Amniota)

KEY TRANSITION POINTS:
  Agnatha → Gnathostomata: evolution of JAWS
  Osteichthyes → Sarcopterygii: lobe fins with bones
  Sarcopterygii → Tetrapoda: limbs, land transition
  Tetrapoda → Amniota: amniotic egg, land independence
```

### Origin of Jaws

```
JAW EVOLUTION — from gill arch

Primitive pharynx:
  Gill arch 1 (mandibular arch): supports mouth
  Gill arch 2 (hyoid arch): supports gill arch 1
  Gill arches 3–7: bear gills

In gnathostome ancestor:
  Arch 1 became UPPER JAW (palatoquadrate) +
              LOWER JAW (Meckel's cartilage)
  Arch 2 (hyomandibula): bracing jaw to skull
  Later: hyomandibula → STAPES (ear bone!) in tetrapods

EVIDENCE: Hox gene expression
  Gill arch cartilage uses same Hox codes in fish
  as jaw cartilage in mammals
  → Jaws ARE modified gill arches
  Dental enamel proteins: teeth = modified scales (placoid)
```

## Decision Cheat Sheet

```
IS IT A CHORDATE? (at any life stage)
  Has notochord + dorsal nerve cord + pharyngeal slits?
  → CHORDATA

  No vertebrae, sessile adult, tunic covering?
    → UROCHORDATA (tunicate)
    Solitary, two siphons? → Ascidiacea (sea squirt)
    Colonial, star-shaped? → Botryllus
    Pelagic, barrel-shaped? → Salp or Doliolid

  No vertebrae, buries in sand, fish-like but translucent?
    → CEPHALOCHORDATA (lancelet / amphioxus)

  Has vertebral column?
    → VERTEBRATA (next files)
    No jaws, eel-like? → AGNATHA (lamprey or hagfish)
    Has jaws?          → GNATHOSTOMATA (all jawed vertebrates)
```

## Common Confusion Points

**Tunicates are NOT the vertebrate ancestor.**
They're the sister group to vertebrates + cephalochordates. Their sessile adult lifestyle is probably a derived state — the mobile, headward-complex larva is the ancestral body plan. Evolution went from motile larvae to complex, not the other direction.

**The notochord doesn't disappear in vertebrates.**
It persists as the nucleus pulposus, the soft gel core of each intervertebral disc. A herniated disc = nucleus pulposus pushed through the annulus fibrosus — your notochord causing sciatica.

**Pharyngeal "gill slits" in human embryos.**
Human embryos at ~4 weeks have 4–5 pairs of pharyngeal arches with pouches between them. They never form actual slits (openings). The pouches give rise to: middle ear cavity (1st), palatine tonsil (2nd), thymus + inferior parathyroid (3rd), superior parathyroid (4th), and (sometimes) thyroid (5th). The gill-slit ancestry is real; the expression is developmental.

**Amphioxus ≠ fish.**
Despite looking like a small fish, amphioxus has no vertebrae, no skull, no paired fins, no eyes (photoreceptors only), no lateral line, no neural crest, no jaw. It's more informative about the pre-vertebrate body plan than about fish specifically.

**The endostyle → thyroid gland homology is biochemical.**
Amphioxus endostyle binds iodine and processes tyrosine — exactly what the vertebrate thyroid does. The genes (NKX2.1, PAX8, FOXE1) are expressed in both. The endostyle is unambiguously the evolutionary precursor to the thyroid. Invertebrate thyroid hormones (T3/T4) also exist.

**Whole-genome duplication events are detectable in synteny.**
When you compare mouse chr 2, 7, 12, 17 — they share clusters of paralogous genes in the same relative order. This "block synteny" is the genomic fingerprint of the ancient 2R WGD that happened ~500 Ma, long before the mammal-fish divergence.
