# Saussure: Sign, Signifier, Signified

## Who Was Saussure

Ferdinand de Saussure (1857–1913) is the founding figure of modern linguistics and, through his influence on structuralism, of semiotics broadly. His Cours de linguistique générale (Course in General Linguistics, 1916) was published posthumously from student notes — he never published it himself. Despite this, it became one of the most influential texts of the 20th century, launching structuralism in linguistics, anthropology, literary theory, and philosophy.

His central move: shift linguistics from the historical/comparative study of language change (19th-century philology) to the synchronic study of language as a system of differences.

```
+-----------------------------------------------------------------------+
|                    SAUSSURE'S FRAMEWORK                               |
|                                                                       |
|  LANGUE vs PAROLE              SYNCHRONY vs DIACHRONY                 |
|  ----------------------        ----------------------------           |

|  social institution.           in time. A cross-section.            |

|                                Diachrony: how a language            |

|                                Saussure's key methodological        |
|  Linguistics studies           claim: synchronic linguistics        |
|  LANGUE, not parole.           is the primary science.              |
|  (The system, not the          Diachrony is secondary.              |
|  instances)                                                           |
+-----------------------------------------------------------------------+
```

## The Sign

The linguistic sign is the central unit of analysis.

```
  THE SIGN is a two-sided psychological entity:

  +------------------------------------------+
  |                 SIGN                     |
  |  +-----------+      +-----------+        |
  |  | SIGNIFIER |  ←→  | SIGNIFIED |         |
  |  | (Signifiant)     | (Signifié)|         |
  |  |           |      |           |         |
  |  | Sound-    |      | Concept   |         |
  |  | image.    |      | Mental    |         |
  |  | The        |      | content. |         |
  |  | phonological     | Not the  |         |
  |  | / written |      | thing     |         |
  |  | form.     |      | itself.   |         |
  |  +-----------+      +-----------+         |
  +------------------------------------------+

  IMPORTANT:
  The signified is NOT the external object/thing.
  "Tree" as a sign:
    Signifier: the acoustic image /triː/
    Signified: the concept of a tree (mental)

  The real-world tree is NOT part of the sign.
  This is why Saussure's linguistics is mentalist/psychological:
  both halves of the sign are mental entities.
```

## The Three Principles

### 1. Arbitrariness of the Sign

```
  PRINCIPLE: No natural or motivated link between signifier and signified.

  "tree" (English) ──→ concept of tree
  "arbre" (French) ──→ concept of tree      same concept,
  "Baum" (German)  ──→ concept of tree      different forms
  "木" mù (Chinese) ──→ concept of tree

  There is nothing tree-like about the sounds /triː/, /arbr/, /baʊm/.
  The link is purely CONVENTIONAL.

  APPARENT EXCEPTIONS:
  Onomatopoeia: "splash", "buzz", "cuckoo" — but these vary:
    English: "cock-a-doodle-doo"
    French: "cocorico"
    Japanese: "ko-ke-kok-ko"
  Even motivated signs are partly conventional.

  Interjections: "ouch", "ow", "aïe" (French) — different across languages.

  CONSEQUENCE: No one sound-sequence is "naturally right" for any concept.
  Language is a social institution, not a mapping from sounds to nature.
```

### 2. The Linear Nature of the Signifier

```
  Spoken language unfolds in time — one dimension, linearly.
  Signifiers cannot be perceived simultaneously; they sequence.

  a - r - b - r - e   ← one after another in time

  This linearity is what makes GRAMMAR necessary:
  "dog bites man" ≠ "man bites dog"
  Order matters because time is one-dimensional.

  Contrast with visual signs:
  A painting presents many elements simultaneously.
  A traffic light = simultaneous spatial arrangement.
  Language is fundamentally sequential.
```

### 3. The Value System: Meaning as Difference

```
  SAUSSURE'S DEEPEST CLAIM:
  Signs do not have positive content — they have value through
  DIFFERENCES from other signs in the system.

  EXAMPLE: color terms

  French: "mouton" covers:
    the animal (sheep)
    the meat (mutton = the word English uses)

  English splits this into two signs:
    "sheep" (the animal)
    "mutton" (the meat on the plate)

  The French sign "mouton" does not have a richer meaning.
  The English and French systems CARVE UP the same conceptual
  space differently. Neither is more correct.

  FORMAL STATEMENT:
  The value of a sign = what it is NOT.
  "cat" has value partly because it is ≠ "bat", "cut",
  "cap", "can", "cot", "cab", "car"...

  This is the fundamental structuralist move:
  MEANING IS RELATIONAL, NOT INTRINSIC.
```

## The Two Axes of Language

Every linguistic act operates on two structural dimensions simultaneously:

```
  SYNTAGMATIC AXIS (horizontal — combination)
  ← — — — — — — — — — — — — — — — — — — — →
  "The  | big  | black | cat  | sat  | quickly"
   ↑       ↑      ↑      ↑      ↑       ↑
   in presentia: these elements are PRESENT in the chain,
   related by contiguity, governed by grammar rules

  +---------+
  | "big"   |
  +---------+  ←— PARADIGMATIC AXIS (vertical — substitution)
  | "small" |      elements NOT present but POTENTIALLY there
  +---------+      related by substitutability in the same slot
  | "ugly"  |      = synonym groups, antonyms, paradigm class
  +---------+
  | "fat"   |
  +---------+

  SYNTAGMATIC relations: in praesentia (both terms present)
  PARADIGMATIC relations: in absentia (one term present, alternatives absent)

  CS PARALLEL:
  Syntagmatic = program sequence (each token in a specific position)
  Paradigmatic = the type of each token (what alternatives could fill that slot)
```

## Synchrony vs Diachrony

```
  DIACHRONIC (historical linguistics):           SYNCHRONIC (Saussure's focus):
  "foot" → "feet" (plural) traces back           The present-day system:
  to Germanic umlaut.                            "foot/feet" as a pair in
  "knight" had a /k/ in Middle English.          modern English — its value
  Latin "equus" → French "cheval"                relative to "toe/toes",
                                                 "hand/hands", etc.

  Analogy: CHESS
  Diachronic: studying how a chess game develops move by move.
  Synchronic: the state of the board at a given moment.
  You can analyze the POSITION without knowing its history.
  The same position has the same value regardless of how it was reached.
```

## Semiology: Saussure's Broader Project

```
  Saussure's Course ends with a vision:

  "A science that studies the life of signs within society
   is conceivable; it would be a part of social psychology
   and consequently of general psychology; I shall call it
   semiology (from Greek semeîon 'sign')."

  He imagined linguistics as ONE branch of a broader science of signs.
  His actual contribution was the linguistic sign theory.
  The broader semiological program was developed by others:
  Barthes (cultural signs), Eco (general semiotics), etc.

  SAUSSURE'S PLACE IN THE FIELD:
  - He established LINGUISTICS as a rigorous systematic science
  - He gave structuralism its theoretical vocabulary
  - He did NOT develop visual semiotics, narrative semiotics, etc.
  - His binary sign model was extended and criticized by Peirce's followers
```

## Langue vs Parole in Practice

```
  LANGUE (the system):            PAROLE (the instances):
  The rules of chess.             The actual moves in a game.
  A grammar of English.           This sentence I wrote.
  The phonological system of      The sound you just heard.
  French.

  STUDYING LANGUE:                STUDYING PAROLE:
  Abstract, systematic.           Corpus linguistics.
  Reveals the code.               Reveals actual usage patterns.
  Saussure's goal.                Chomsky: competence vs performance
                                  (a parallel distinction in generative grammar)

  TENSION: Langue is a fiction (no one speaker has all of it)
  but a useful theoretical construct. It is the shared resource
  that makes communication possible.
```

## Decision Cheat Sheet

| I need to... | Use this concept |
|-------------|-----------------|
| Separate the form of a sign from its content | Signifier / signified split |
| Ask why the same concept has different words across languages | Arbitrariness of the sign |
| Analyze which choices were made from a set of alternatives | Paradigmatic axis (substitution; what COULD be here) |
| Analyze the structure of what is actually present in sequence | Syntagmatic axis (combination; what IS here in order) |
| Study the system that makes a particular sign meaningful | Langue (the code/system) vs. parole (the instance) |
| Understand why a term means what it means in a system | Difference: value comes from contrast, not from intrinsic content |
| Analyze a system at a single moment (vs. its history) | Synchronic analysis |
| Analyze how a system has changed over time | Diachronic analysis |
| Understand the mental/psychological vs. physical dimension | Signifier = acoustic image (mental); not the physical sound wave |

## Formal Systems Bridge

```
SAUSSURE ↔ TYPE SYSTEMS AND RELATIONAL ALGEBRA

PARADIGMATIC AXIS ↔ TYPE / DOMAIN:
  The paradigmatic axis is the set of items that could
  occupy a given position in a structure — items related
  by substitutability.
  In a type system: the type of a variable is the set of values
  that can occupy that position. String | Int | Bool are
  paradigmatic alternatives for a parameter slot.
  In relational algebra: a column domain is the paradigmatic
  set — all valid values for that attribute position.
  Saussure: "Je mange du pain" — paradigmatic set at the
  verb position: mange, vois, donne, etc.

SYNTAGMATIC AXIS ↔ EXPRESSION SEQUENCE / ROW TUPLE:
  The syntagmatic axis is the linear combination of items
  actually present — items related by co-presence and order.
  In a type system: an expression sequence, a function call
  with its ordered arguments, a tuple type (Int × String × Bool).
  In relational algebra: a row is the syntagmatic realization —
  the specific values occupying each column position in order.
  Saussure: "Je mange du pain" — the four-element syntagm
  is the actual sequence instantiated from paradigmatic choices.

LANGUE / PAROLE ↔ SCHEMA / INSTANCE:
  Langue (the system/code) = the relational schema;
    the type signatures; the grammar.
  Parole (individual utterances) = instances / rows / programs.
  No individual speaker has access to all of langue;
  they have a model of it that enables production and recognition.
  Exactly as no program has access to the type system;
  it operates within and according to it.

VALUE THROUGH DIFFERENCE ↔ ENUMERATION SEMANTICS:
  Saussure: a sign's meaning is constituted by its differences
  from other signs in the system, not by intrinsic content.
  In enumeration types: NORTH, SOUTH, EAST, WEST have no
  intrinsic meaning — only their distinctness from each other
  within the type. Change the type and the meanings shift.
  This is why Saussure says "in language there are only
  differences without positive terms."
```

## Common Confusion Points

**The signified is NOT the external object**: This is the most common mistake. The sign relation is between a sound-image and a concept, both mental. The external tree is not the signified — the CONCEPT of a tree is. Saussure is explicit about this.

**"Arbitrary" does not mean "random" or "unstable"**: Arbitrary means the link is conventional, not natural. Once established, convention is binding and stable. Language does not change because individuals decide to change it — convention is social and has inertia.

**Saussure's linguistics does not handle meaning in context**: Saussure brackets pragmatics. He studies the system, not utterances. Grice, Austin, and later pragmatics theories fill this gap. Saussure's framework is explicitly about langue, not parole.

**Structuralism ≠ Saussure**: Saussure gave structuralism its vocabulary. The actual structuralist movement (Lévi-Strauss, Barthes applying structure to myth and culture) goes well beyond what Saussure said. Lévi-Strauss applied the paradigm/syntagm distinction to kinship systems and myth; Saussure never intended this extrapolation.

**Signifier ≠ the physical sound wave**: The signifier is the acoustic image — a mental impression, a psychological representation of the sound pattern. Saussure carefully distinguishes this from the physical phonetics. It is a mental form, not a material substance.
