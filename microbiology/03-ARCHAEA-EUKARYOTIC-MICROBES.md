# Archaea and Eukaryotic Microbes

## The Big Picture

```
THE FORGOTTEN DOMAINS: ARCHAEA AND MICROBIAL EUKARYOTES
=========================================================

  PHYLOGENETIC REALITY:
  Archaea are more closely related to Eukaryotes than to Bacteria.
  The three-domain tree was established by Carl Woese (1977) using 16S rRNA.
  Before 1977: Two kingdoms (prokaryote / eukaryote). This was wrong.

  ┌─────────────────────────────────────────────────────────────────┐
  │                        LUCA                                     │
  │              (Last Universal Common Ancestor)                   │
  │                 /                    \                          │
  │         BACTERIA              ARCHAEA/EUKARYA                   │
  │                                    /       \                    │
  │                              ARCHAEA      EUKARYA               │
  │                                                                 │
  │  Key evidence: Archaea share:                                   │
  │  - Similar histone-like proteins (H3/H4 relatives)              │
  │  - Introns and similar RNA processing                           │
  │  - Eukaryotic-like initiation factors for translation           │
  │  - Similar DNA replication enzymes                              │
  │  BUT: Prokaryotic-like cell organization (no nucleus)           │
  └─────────────────────────────────────────────────────────────────┘

  ARCHAEA: KEY DIFFERENCES FROM BACTERIA
  ┌────────────────────────────────────────────────────────────────┐
  │ FEATURE          BACTERIA            ARCHAEA                   │
  │ Cell wall        Peptidoglycan       Pseudopeptidoglycan or    │
  │                                      S-layer (no murein)       │
  │ Membrane lipids  Ester-linked        Ether-linked + isoprenoid │
  │                  fatty acids         side chains               │
  │ Transcription    Bacterial σ factor  TATA-box, TBP, TFB        │
  │                  (σ70 etc.)          (eukaryote-like!)         │
  │ Ribosome         70S (5S,16S,23S)    70S size but eukaryotic   │
  │                                      sequence                  │
  │ Histone          None                YES (HMfA/B in Mth)       │
  │ Antibiotics      Many targets        Resistant to most         │
  │                                      (different targets)       │
  └────────────────────────────────────────────────────────────────┘
```

## Archaea: The Extremophile Champions

```
  ARCHAEA AND EXTREME ENVIRONMENTS
  ==================================

  WHY EXTREMOPHILES MATTER:
  ─ Industrial enzymes: heat-stable polymerases, proteases
  ─ Taq polymerase: From Thermus aquaticus (hot springs)
    → Enables PCR (without it, no molecular biology revolution)
  ─ Understanding early life: Earth was hot, anoxic → archaea-like first life
  ─ Astrobiology: Extremophile survival suggests life might exist elsewhere

  MAJOR ARCHAEAL GROUPS:
  ┌──────────────────────────────────────────────────────────────────┐
  │ THERMOPHILES / HYPERTHERMOPHILES                                 │
  │   Grow at 55–115°C (!)                                           │
  │   Pyrococcus furiosus: Optimal 100°C; ferments sugars to H₂    │
  │   Sulfolobus solfataricus: Grows in volcanic sulfur springs      │
  │   Thermus aquaticus: 70°C, hot springs (Yellowstone)           │
  │   Enzyme adaptation: Proteins with more disulfide bonds,         │
  │   ion pairs, hydrophobic cores for thermal stability             │
  │                                                                  │
  │ HALOPHILES (salt lovers)                                         │
  │   Haloarcula marismortui: 5 M NaCl (Dead Sea isolate)          │
  │   Halobacterium salinarum: Pink coloration in salt ponds         │
  │   Adaptation: Acidic proteome (high Asp/Glu) → stable in salt  │
  │   Bacteriorhodopsin: Light-driven proton pump → solar energy   │
  │                                                                  │
  │ METHANOGENS (methane producers)                                  │
  │   STRICT ANAEROBES — O₂ is lethal                                │
  │   Methanobacterium, Methanosarcina, Methanobrevibacter         │
  │   Metabolism: CO₂ + 4H₂ → CH₄ + 2H₂O (or acetate → CH₄)      │
  │   Location: Gut of ruminants, termites, marshes, landfills     │
  │   Significance: 15% of global methane from methanogen activity   │
  │   Ruminant methane: ~2% of global greenhouse gas emissions     │
  │   Human gut: Methanobrevibacter smithii, ~30% of adults carry  │
  │                                                                  │
  │ ACIDOPHILES                                                      │
  │   Ferroplasma acidarmanus: pH 0 (mine drainage)                │
  │   Picrophilus torridus: pH –0.06, 60°C (most acid-tolerant known)│
  │                                                                  │
  │ PSYCHROPHILES (cold lovers)                                      │
  │   Active below 0°C (permafrost, sea ice, Antarctic lakes)      │
  │   Methanogenium frigidum: methanogens in cold sediments          │
  └──────────────────────────────────────────────────────────────────┘
```

## Archaea in Normal Environments

```
  ARCHAEA BEYOND EXTREMES
  =========================

  Marine archaea constitute ~20% of marine picoplankton
  Thaumarchaeota: Ammonia oxidizers — major role in nitrogen cycle
    nitrification: NH₄⁺ → NO₂⁻ (first step; previously thought only bacteria did this)

  ASGARD ARCHAEA (discovered 2015–2020):
  ─ Deep-sea metagenomics revealed: Asgard superphylum
  ─ Lokiarchaeota, Thorarchaeota, Odinarchaeota, Heimdallarchaeota
  ─ Closest known relatives to eukaryotes
  ─ Have eukaryote-like proteins: actin, ESCRTs, Ras GTPases
  ─ Supports "two-domain tree": Eukaryotes ARE a derived archaeal lineage
  ─ Endosymbiosis model: Archaeal ancestor + bacterial endosymbiont
    → eukaryote (mitochondria from alpha-proteobacterial ancestor)
```

## Protists: Unicellular Eukaryotes

```
  PROTISTS: A POLYPHYLETIC GRAB-BAG
  ===================================

  "Protist" is not a phylogenetic group — it means "eukaryotic microbe
  that isn't a plant, animal, or fungus." Extremely diverse.

  MEDICALLY IMPORTANT PROTISTS:
  ┌──────────────────────────────────────────────────────────────────┐
  │ PLASMODIUM spp. (Apicomplexan)                                   │
  │   P. falciparum: Most dangerous malaria species                  │
  │   P. vivax: Most common; relapsing malaria (liver dormancy)      │
  │   Life cycle: Mosquito → sporozoite → liver → merozoite          │
  │   → red blood cell → gametocytes → mosquito                    │
  │   RBC destruction → anemia, fever, splenomegaly                │
  │   Treatment: Artemisinin-based combination therapy (ACT)         │
  │   Resistance: ACT partial resistance (P. falciparum K13 mutation)│
  │                                                                  │
  │ TOXOPLASMA gondii (Apicomplexan)                                 │
  │   Definitive host: Cats (oocysts in feces)                       │
  │   Intermediate hosts: Warm-blooded animals including humans      │
  │   Bradyzoites: Cysts in brain + muscle → lifelong persistence  │
  │   Risk: Immunocompromised (CD4<100) → toxoplasma encephalitis  │
  │   Congenital: If primary infection in pregnancy → fetal damage   │
  │                                                                  │
  │ GIARDIA lamblia (excavate)                                       │
  │   Diplomonad: Two nuclei; no mitochondria (reduced to mitosome)  │
  │   Cyst: Environmental resistance; infectious dose: ~10 cysts   │
  │   Trophozoite: Attaches to intestinal epithelium → malabsorption│
  │   Transmission: Fecal-oral, contaminated water (hikers, "beaver fever")│
  │                                                                  │
  │ TRYPANOSOMA brucei (kinetoplastid)                               │
  │   African sleeping sickness; transmitted by tsetse fly           │
  │   VSG coat: Hypervariable surface glycoprotein → antigenic variation│
  │   Switches VSG gene every few generations → escapes immunity     │
  │   Large archive of silent VSG genes → millions of variants       │
  │                                                                  │
  │ ENTAMOEBA histolytica (amoeba)                                   │
  │   Amoebic dysentery; cysts in contaminated water                 │
  │   Trophozoites invade colon → bloody diarrhea; liver abscess     │
  │   ~50 million invasive infections/year globally                  │
  └──────────────────────────────────────────────────────────────────┘

  ECOLOGICALLY IMPORTANT PROTISTS:
  Diatoms: Silica cell walls; ~25% of global primary production
  Dinoflagellates: Coral reef symbionts (zooxanthellae); also toxic blooms
  Foraminifera: Calcareous shells; geologic timekeeping (index fossils)
  Radiolarians: Silica shells; deep sea sediments
```

## Microscopic Fungi

```
  FUNGI: THE EUKARYOTIC DECOMPOSERS AND PATHOGENS
  ==================================================

  FUNGAL CELL BIOLOGY:
  ─ Cell wall: Chitin + β-glucan + mannoproteins (NOT peptidoglycan)
  ─ Ergosterol: Fungal membrane sterol (vs. cholesterol in humans)
    → Target for antifungal drugs (polyenes, azoles)
  ─ True nucleus, mitochondria, ER: Fully eukaryotic

  FUNGAL GROWTH FORMS:
  YEAST (unicellular): Saccharomyces cerevisiae, Candida
    Budding: Mitosis → asymmetric bud formation
  MOLD (multicellular): Aspergillus, Mucor, Rhizopus
    Filamentous: Hyphae → network = mycelium
  DIMORPHIC FUNGI: Two forms (temperature-dependent):
    At 25°C (soil): Mold form
    At 37°C (host): Yeast form
    Examples: Histoplasma capsulatum, Coccidioides, Blastomyces

  MEDICALLY IMPORTANT FUNGI:
  ┌────────────────────────────────────────────────────────────────┐
  │ Candida albicans:                                              │
  │   Commensal in gut/vaginal flora in 50% of people              │
  │   Yeast ↔ hyphal switching (virulence factor)                  │
  │   Invasive candidiasis: Major cause of ICU infections          │
  │   Azole resistance (FKS1 mutations, ERG11 mutations): rising   │
  │   Candida auris: Multidrug-resistant; emerging global threat   │
  │                                                                  │
  │ Aspergillus fumigatus:                                         │
  │   Ubiquitous mold; conidia inhaled constantly                  │
  │   Healthy: Cleaned by alveolar macrophages + neutrophils       │
  │   Immunocompromised: Invasive aspergillosis → angioinvasion    │
  │   → Pulmonary infarction; 40–90% mortality untreated           │
  │   Treatment: Voriconazole (azole); isavuconazole               │
  │                                                                  │
  │ Cryptococcus neoformans/gattii:                                │
  │   Encapsulated yeast; polysaccharide capsule: anti-phagocytic  │
  │   Cryptococcal meningitis: AIDS-defining illness (CD4<100)     │
  │   Treatment: Amphotericin B + flucytosine induction            │
  │                                                                  │
  │ Pneumocystis jirovecii:                                        │
  │   Atypical fungus; doesn't grow on routine culture             │
  │   PCP (Pneumocystis pneumonia): AIDS-defining (CD4<200)        │
  │   Diagnosis: BAL + GMS stain or PCR                            │
  │   Treatment: TMP-SMX (trimethoprim-sulfamethoxazole)           │
  │   Prophylaxis: TMP-SMX when CD4<200                            │
  └────────────────────────────────────────────────────────────────┘

  ANTIFUNGAL DRUG TARGETS:
  ┌────────────────────────────────────────────────────────────────┐
  │ Azoles (fluconazole, voriconazole, itraconazole):              │
  │   Inhibit CYP51 (lanosterol 14α-demethylase) → ergosterol ↓  │
  │   Resistance: ERG11 mutations, overexpression                  │
  │                                                                  │
  │ Polyenes (amphotericin B, nystatin):                           │
  │   Bind ergosterol → membrane pores → cell death                │
  │   Broad spectrum; significant nephrotoxicity (AmB)             │
  │   Liposomal formulation: Reduced nephrotoxicity                │
  │                                                                  │
  │ Echinocandins (caspofungin, micafungin, anidulafungin):        │
  │   Inhibit β-1,3-glucan synthase (FKS1)                         │
  │   Fungal cell wall specific → low human toxicity               │
  │   Active vs. Candida + Aspergillus                             │
  │   NOT active vs. Cryptococcus (lacks target)                   │
  └────────────────────────────────────────────────────────────────┘
```

## Algae and Marine Microbes

```
  ALGAE: PHOTOSYNTHETIC PROTISTS
  ================================

  PHYLOGENETICALLY DIVERSE (multiple independent groups):
  Green algae (Chlorophyta): Related to land plants; Chlamydomonas model organism
  Red algae (Rhodophyta): Source of agar (microbiology medium) and carrageenan
  Brown algae (Phaeophyta): Kelp forests — technically macroalgae
  Diatoms (Bacillariophyta): ~25% of Earth's primary production
  Dinoflagellates: Toxin producers; coral symbiosis; bioluminescence

  DIATOMS:
  ─ Silica (SiO₂) frustule (shell) — intricate geometric designs
  ─ When they die → sink → accumulate as diatomaceous earth
  ─ Industrial uses: Filtration, abrasives, fire retardants
  ─ Biogeochemical: Important in silica cycling + carbon export to deep ocean

  HARMFUL ALGAL BLOOMS (HABs):
  Dinoflagellates (Karenia brevis, Alexandrium, Pseudo-nitzschia):
  ─ Produce toxins: saxitoxin (paralytic shellfish toxin), brevetoxin
  ─ Bioaccumulate in shellfish → human poisoning (PSP, NSP)
  ─ Mass fish kills (oxygen depletion + direct toxicity)
  ─ Climate change expanding HAB frequency and range
```

## Common Confusion Points

**Archaea are not "primitive bacteria"**: This is a common misconception. Archaea diverged from bacteria early but are not ancestral to bacteria — they're a separate domain that evolved in parallel. They are actually more closely related to eukaryotes than to bacteria, which is exactly the opposite of the intuitive assumption based on appearance.

**Eukaryote origin as endosymbiosis**: Mitochondria are derived from alpha-proteobacteria taken up by an archaea-like ancestor (endosymbiosis). Chloroplasts are derived from cyanobacteria. Evidence: mitochondria and chloroplasts have their own genomes, 70S ribosomes, replicate by binary fission, and have outer membrane from proteobacterial origin. This means YOU are partially a bacterium.

**Dimorphic fungi: temperature switches form**: Histoplasma grows as a mold at room temperature (in soil) and as a yeast at body temperature. This is why dimorphic fungal infections occur when people inhale spores from contaminated soil — the organism converts to the pathogenic form once inside the host.

**Protozoa vs. helminth vs. ectoparasite**: These are all "parasites" but very different organisms. Protozoa = single-cell eukaryote (Plasmodium, Giardia). Helminths = multicellular worms (tapeworm, roundworm, flukes). Ectoparasites = live on outside (lice, mites, ticks). Only protozoa are covered here; helminths and ectoparasites are macroscopic and covered in disease/ modules.
