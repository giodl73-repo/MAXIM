# The Human Microbiome

## The Big Picture

```
THE HUMAN MICROBIOME: SCALE AND COMPOSITION
=============================================

  YOU ARE NOT ALONE IN YOUR OWN BODY

  ┌─────────────────────────────────────────────────────────────────┐
  │  NUMBERS (2016 revision of "10:1 ratio" — now ~1:1)             │
  │                                                                 │
  │  Human cells:     ~37 trillion (Bianconi et al. 2013)           │
  │  Bacterial cells: ~38 trillion (Sender et al. 2016)             │
  │  Ratio: ~1:1  (old "10:1" was based on flawed estimates)        │
  │                                                                 │
  │  But by GENE COUNT:                                             │
  │  Human genome:      ~20,000 protein-coding genes                │
  │  Gut microbiome: ~3,000,000 unique microbial genes (150x more)  │
  │  → You are ~0.6% human by gene count                            │
  │                                                                 │
  │  ANATOMICAL DISTRIBUTION:                                       │
  │  ┌──────────────────────────────────────────────────────────┐   │
  │  │ Site         Dominant taxa          Density              │   │
  │  │ Stomach      Helicobacter pylori    ~10²–10³ cells/mL    │   │
  │  │ Small int.   Lactobacillus, Strep   ~10³–10⁷ cells/mL   │   │
  │  │ Large int.   Firmicutes, Bacteroidetes ~10¹¹–10¹² /mL   │   │
  │  │ Skin         Staphylococcus, Cutibacterium ~10⁶/cm²     │   │
  │  │ Oral         Streptococcus, Veillonella  ~10⁹/mL        │   │
  │  │ Vaginal      Lactobacillus-dominant  ~10⁸–10⁹/mL       │   │
  │  └──────────────────────────────────────────────────────────┘   │
  └─────────────────────────────────────────────────────────────────┘

  WHERE THE MASS LIVES: GUT
  ─────────────────────────
  Colon is the densest microbial ecosystem on Earth
  ~2 kg of bacteria in the human colon
  ~500–1,000 species in a healthy adult (varies by definition)
  Firmicutes + Bacteroidetes account for ~80–90% of gut bacteria
```

## Gut Microbiome Composition

```
  GUT MICROBIOME TAXONOMY
  ========================

  MAJOR PHYLA (healthy Western adult approximation):
  ┌───────────────────────────────────────────────────────────────┐
  │ FIRMICUTES (50–65% of sequences)                              │
  │   Class Clostridia: Clostridium, Ruminococcus, Faecalibacterium│
  │   Class Bacilli: Lactobacillus, Enterococcus, Streptococcus   │
  │   Key species: Faecalibacterium prausnitzii (most abundant)   │
  │   → Major butyrate producer → colonocyte fuel                 │
  │   → Anti-inflammatory; depleted in IBD                        │
  │                                                               │
  │ BACTEROIDETES (20–35%)                                        │
  │   Bacteroides: B. thetaiotaomicron (polysaccharide degrader)  │
  │   Prevotella: Diet-associated; fiber diet → more Prevotella   │
  │   Key function: Polysaccharide utilization loci (PULs)        │
  │   → Can digest >100 types of dietary polysaccharide           │
  │                                                               │
  │ ACTINOBACTERIA (~5%)                                          │
  │   Bifidobacterium: Major infant microbiome constituent        │
  │   Collinsella: SCFA producer                                  │
  │                                                               │
  │ PROTEOBACTERIA (~5%)                                          │
  │   E. coli, Klebsiella: Low-abundance commensals normally      │
  │   Bloom in disease/dysbiosis → inflammophiles                 │
  │                                                               │
  │ VERRUCOMICROBIA (<1%)                                         │
  │   Akkermansia muciniphila: Mucus layer inhabitant             │
  │   → Correlates with metabolic health, diabetes protection     │
  │   → Pasteurized (heat-killed) A. muciniphila: clinical trials│
  └───────────────────────────────────────────────────────────────┘

  FIRMICUTES/BACTEROIDETES (F/B) RATIO:
  ─ Often cited as "obesity biomarker" but CONTROVERSIAL
  ─ Original mouse study (Turnbaugh 2006): obese mice had higher F/B
  ─ Human studies: inconsistent, diet confounds everything
  ─ Modern view: Specific species > phylum-level ratios
    Faecalibacterium prausnitzii abundance more informative than F/B
```

## Metabolic Functions of the Gut Microbiome

```
  WHAT YOUR MICROBIOME DOES FOR YOU
  ===================================

  SHORT-CHAIN FATTY ACIDS (SCFAs): The key metabolic output
  ┌────────────────────────────────────────────────────────────────┐
  │ SUBSTRATE: Dietary fiber → fermentation by anaerobes           │
  │                                                                │
  │ PRODUCTS:                                                      │
  │   Butyrate (butyric acid): C4:0                                │
  │   → PRIMARY fuel for colonocytes (epithelial cells)            │
  │   → Histone deacetylase inhibitor → epigenetic regulation      │
  │   → Anti-inflammatory; stimulates mucus production             │
  │   → Producers: Roseburia, Faecalibacterium, Clostridium IV     │
  │                                                                │
  │   Propionate: C3:0                                             │
  │   → Liver metabolism; gluconeogenic substrate                  │
  │   → Appetite regulation (GPR41/GPR43 on L-cells)               │
  │   → Producers: Bacteroides, Ruminococcus                       │
  │                                                                │
  │   Acetate: C2:0  (most abundant SCFA)                          │
  │   → Peripheral tissue fuel; muscle, heart                      │
  │   → Appetite signaling via vagal nerve                         │
  │   → Producers: Bifidobacterium, Bacteroides                    │
  │                                                                │
  │ TOTAL SCFA PRODUCTION:                                         │
  │   ~50–100 mmol/day — meaningful energetic contribution         │
  │   Provides ~10% of human daily caloric intake                  │
  └────────────────────────────────────────────────────────────────┘

  OTHER METABOLIC FUNCTIONS:
  ─ Vitamin synthesis: K₂ (menaquinone), B12, folate, biotin, riboflavin
  ─ Bile acid transformation: Primary (liver) → secondary bile acids
    Deoxycholic acid, lithocholic acid: TGR5 agonists, FXR agonists
    → Regulate glucose metabolism, energy expenditure, lipid homeostasis
  ─ Xenobiotic metabolism: Some drugs activated or deactivated by microbiome
    Example: Digoxin inactivated by Eggerthella lenta (cytochrome cgr2)
    Example: NSAID gut toxicity mediated by Bacteroides beta-glucuronidase
  ─ Tryptophan metabolism: Dietary Trp → indoles (bacterial) → AhR ligands
    → Gut barrier integrity, IL-22 production, serotonin precursors
```

## The Gut-Brain Axis

```
  GUT-BRAIN COMMUNICATION
  ========================

  NOT METAPHOR — Bidirectional anatomical and biochemical connections:

  ┌────────────────────────────────────────────────────────────────┐
  │                                                                │
  │   BRAIN                                                        │
  │     ↑↓                                                         │
  │   VAGUS NERVE (80% afferent — gut → brain)                     │
  │     ↑↓                                                         │
  │   ENTERIC NERVOUS SYSTEM (gut's own nervous system)            │
  │   ~500 million neurons — more than spinal cord                 │
  │   Serotonin: ~95% of body's serotonin made in gut (EC cells)   │
  │     ↑↓                                                         │
  │   GUT EPITHELIUM + IMMUNE CELLS                                │
  │     ↑↓                                                         │
  │   GUT MICROBIOME                                               │
  │                                                                │
  └────────────────────────────────────────────────────────────────┘

  MECHANISMS:
  1. Serotonin production:
     Spore-forming bacteria (Clostridia) stimulate EC cells
     → Produces 5-HT → regulates gut motility + signals via vagus
     Germ-free mice: reduced colonic serotonin → restored by colonization

  2. GABA production:
     Lactobacillus rhamnosus JB-1: Produces GABA
     → Reduced anxiety/depression in mice (Bravo 2011)
     → Vagotomy abolished the effect → vagal requirement confirmed

  3. Microbiome → HPA axis (stress response):
     Germ-free mice: exaggerated cortisol response to stress
     → Colonization with specific bacteria normalizes HPA
     → Suggests early microbiome development calibrates stress response

  4. Tryptophan → Serotonin + Kynurenine pathway:
     Microbiome competes for dietary Trp
     → Less Trp available → less brain serotonin synthesis
     → Kynurenine pathway: associated with depression, neuroinflammation

  5. Short-chain fatty acids (SCFAs) → brain:
     Butyrate: Crosses blood-brain barrier; activates microglia
     → Germ-free mice show impaired microglial maturation
     → Restored by SCFA supplementation (Erny et al. 2015)

  CLINICAL EVIDENCE (cautious):
  ─ IBS ↔ anxiety/depression: Strong bidirectional correlation
  ─ Autism: Distinct microbiome composition (but causation unclear)
  ─ Parkinson's: Gut-first hypothesis; Lewy bodies in enteric neurons
    before substantia nigra (Braak staging)
  ─ Depression: Fecal transplant from depressed patients → depressive
    behaviors in germ-free rats (Kelly et al. 2016) — rodent caveat
  ─ Human probiotic trials for depression: small, mixed results
    Psychobiotics: promising but evidence quality still low
```

## Development and Disruption

```
  MICROBIOME DEVELOPMENT: FROM BIRTH TO DEATH
  =============================================

  BEFORE BIRTH: Womb not sterile? (Controversial)
  Classic dogma: Uterus is sterile; microbiome begins at birth
  Recent claims of "placental microbiome" — mostly contamination artifact
  Current consensus: Sterile womb is likely correct

  BIRTH: The founding event
  ┌────────────────────────────────────────────────────────────────┐
  │ VAGINAL BIRTH                  CESAREAN BIRTH                  │
  │ Inoculated with:               Inoculated with:                │
  │ Lactobacillus, Prevotella,    Skin bacteria, hospital env.     │
  │ Sneathia (vaginal flora)       Staphylococcus, Cutibacterium   │
  │                                                                │
  │ C-section microbiome:                                          │
  │ → Higher rates of asthma, obesity, T1D in population studies   │
  │ → Confounded by indication bias (C-sections for high-risk)     │
  │ → Vaginal seeding trials: painting C-section babies with       │
  │   maternal vaginal fluid → partially transfers microbiome      │
  └────────────────────────────────────────────────────────────────┘

  INFANT PERIOD:
  Breast milk: Contains oligosaccharides (HMOs — cannot be digested by infant)
  → HMOs are FOOD FOR Bifidobacterium infantis
  → Bifido blooms → produces indole-3-lactate → type 1 regulatory T cells
  → Colonizes gut with HMO-utilizing specialists
  Formula-fed infants: More diverse but less Bifidobacterium-dominated

  MATURATION (~2–3 years):
  Adult-like microbiome composition established by ~3 years
  Critical window: Antibiotic exposure in first 3 years → lasting effects
  Evidence: Antibiotic use <2 years → increased risk of obesity (Danish cohort)

  STABILITY IN ADULTS:
  Core microbiome stable for years if lifestyle stable
  But: Disrupted by antibiotics, travel, illness, diet change
  Antibiotics: Diversity recovers in weeks but original community
               may not fully restore (some studies show 6-month perturbations)

  AGING:
  Decrease in Firmicutes diversity (especially Clostridiales)
  Increase in Proteobacteria (inflammophiles)
  Reduced SCFA production
  Inflammaging: Chronic low-grade inflammation — microbiome contribution?
  Centenarian microbiomes: Often retain high Faecalibacterium, Akkermansia

  DYSBIOSIS: LOSS OF BENEFICIAL COMPOSITION
  Triggers: Antibiotics, infection, diet change, immunosuppression
  Features: Reduced diversity, blooms of pathobionts
  Pathobiont: Normal low-abundance commensal that causes disease
              when allowed to expand (Proteobacteria, C. difficile)
```

## Clostridioides difficile: Dysbiosis in Action

```
  C. DIFFICILE — THE DYSBIOSIS DISEASE
  =====================================

  C. difficile (CDI): Most common hospital-acquired infection in US
  ~500,000 cases/year in US; ~30,000 deaths

  PATHOPHYSIOLOGY:
  ─ Normal gut microbiome: "colonization resistance" → C. diff can't bloom
  ─ Antibiotics → kill colonization resistance organisms
  ─ C. diff spores (environmental) → germinate in disrupted microbiome
  ─ Toxin A (TcdA): Enterotoxin → disrupts tight junctions
  ─ Toxin B (TcdB): Cytotoxin → inactivates Rho GTPases → cell death
  ─ → Pseudomembranous colitis, watery/bloody diarrhea, fulminant colitis

  RISK FACTORS:
  ─ Recent antibiotics (especially fluoroquinolones, clindamycin, cephalosporins)
  ─ Age >65 (less diverse microbiome)
  ─ Hospitalization/healthcare exposure
  ─ Proton pump inhibitors (alter gastric pH → more spore survival)

  FECAL MICROBIOTA TRANSPLANT (FMT):
  ┌────────────────────────────────────────────────────────────────┐
  │ RATIONALE: Restore colonization resistance by restoring        │
  │            the diverse microbiome from a healthy donor         │
  │                                                                │
  │ EFFICACY: ~90% cure rate for recurrent CDI                     │
  │ (vs. ~50% for antibiotics in recurrent disease)                │
  │                                                                │
  │ MECHANISM: Donor microbiome outcompetes C. diff                │
  │ Route: Colonoscopy, nasojejunal tube, or oral capsules         │
  │                                                                │
  │ FDA STATUS: Vowst (oral, 2023) and Rebyota (enema, 2022)       │
  │             — first FDA-approved microbiome-based drugs        │
  │                                                                │
  │ RISKS: Serious adverse events from unscreened donors (cases    │
  │        of drug-resistant organism transfer, COVID-19 in 2020)  │
  │ Screening: Stool donors screened for pathogens, parasites,     │
  │            genetic markers of disease                          │
  └────────────────────────────────────────────────────────────────┘
```

## Skin, Oral, and Vaginal Microbiomes

```
  OTHER MICROBIOME NICHES
  ========================

  SKIN MICROBIOME:
  ─ 1.8 m² surface; ~10⁶ bacteria/cm² (varies by site)
  ─ Sebaceous sites (back, face): Cutibacterium acnes dominates
    → Converts sebum → short-chain fatty acids → low pH
    → Protective, but C. acnes virulent strains in acne vulgaris
  ─ Moist sites (axilla, groin): Staphylococcus, Corynebacterium
  ─ Dry sites (forearm): Proteobacteria, Firmicutes
  ─ Atopic dermatitis: Staphylococcus aureus bloom
    → S. aureus protease disrupts skin barrier → more allergen entry
    → Anti-S. aureus interventions: bleach baths, topical microbiome restoration
  ─ Malassezia (fungus): Universal scalp resident; drives seborrheic dermatitis

  ORAL MICROBIOME:
  ─ ~700 species; second most diverse after gut
  ─ Highly site-specific: teeth vs. gingiva vs. tongue vs. cheek
  ─ Human Oral Microbiome Database (HOMD): Reference database
  ─ Dental caries: Streptococcus mutans metabolizes sugars → lactic acid
    → pH drops → enamel demineralization
  ─ Periodontal disease:
    Subgingival dysbiosis: Porphyromonas gingivalis, Treponema denticola
    P. gingivalis: Peptidylarginine deiminase (citrullination) →
    → Connection to RA (ACPA generation by citrullinated antigens)
  ─ Aspiration pneumonia: Oral bacteria aspirated → lung → Strep, anaerobes

  VAGINAL MICROBIOME:
  ─ Unique: LOWER diversity = HEALTHY (opposite of gut rule)
  ─ Lactobacillus-dominated community types (Community State Types, CSTs):
  ┌────────────────────────────────────────────────────────────────┐
  │ CST I:   L. crispatus dominant  → most protective              │
  │ CST II:  L. gasseri dominant    → protective                   │
  │ CST III: L. iners dominant      → transitional, less stable    │
  │ CST IV:  Low Lactobacillus, diverse → Bacterial Vaginosis      │
  │ CST V:   L. jensenii dominant   → protective                   │
  └────────────────────────────────────────────────────────────────┘
  ─ Lactobacillus produces lactic acid → pH ~3.5–4.5 → protection
  ─ Bacterial Vaginosis (BV): CST IV, Gardnerella vaginalis bloom
    → pH rises → increased susceptibility to STIs (HIV, HSV, HPV)
    → Preterm labor risk (BV in pregnancy)
  ─ Ethnic variation: Black/Hispanic women more likely to have CST IV
    → Health disparity implications
```

## Microbiome and Disease

```
  MICROBIOME-DISEASE ASSOCIATIONS
  =================================

  CHALLENGE: Correlation ≠ causation. Microbiome is deeply confounded
  by diet, genetics, lifestyle. Many associations; few proven causes.

  OBESITY AND METABOLIC DISEASE:
  ─ Germ-free mice: Resistant to diet-induced obesity
  ─ FMT from obese → lean germ-free mice: increases fat mass
  ─ Proposed mechanisms:
    SCFA signaling → energy harvest from fiber
    Bile acid modification → FXR/TGR5 signaling → metabolism
    Endotoxemia: LPS from Gram- bacteria → low-grade inflammation → insulin resistance
  ─ Human FMT trials (obese → lean): Transient insulin sensitivity improvement
  ─ GLP-1 drugs (Ozempic/semaglutide): Alter microbiome composition —
    cause or effect of weight loss? (active research question 2024-2026)

  INFLAMMATORY BOWEL DISEASE (IBD):
  ─ Crohn's disease + Ulcerative colitis: ~3 million affected in US
  ─ Microbiome changes: Reduced Faecalibacterium, increased Proteobacteria
  ─ Genetics: NOD2 (bacterial sensor mutation) → most reproducible Crohn's gene
  ─ Dysbiosis-inflammation loop: Dysbiosis promotes inflammation → inflammation
    selects for more dysbiosis (nitrate from neutrophils fuels Enterobacteriaceae)
  ─ FMT for Ulcerative colitis: Phase 3 trials showing ~30% remission
    (Paramsothy 2017, 2019 Lancet)

  COLORECTAL CANCER:
  ─ Fusobacterium nucleatum: Enriched in CRC; promotes tumor growth via FadA adhesin
  ─ Enterotoxigenic B. fragilis (ETBF): BFT toxin cleaves E-cadherin → oncogenic
  ─ F. nucleatum found inside CRC metastases to liver → not just bystander
  ─ Microbiome modulates checkpoint immunotherapy response (see immunotherapy/)

  CARDIOVASCULAR DISEASE:
  ─ TMAO pathway: Gut bacteria metabolize choline/carnitine → TMA
    → Liver FMO3 converts TMA → TMAO (trimethylamine-N-oxide)
    → TMAO promotes atherosclerosis (Wang et al. 2011, Nature Medicine)
    → Mechanistic in mice; in humans: confounded by diet (choline = eggs)

  MENTAL HEALTH:
  ─ See Gut-Brain Axis section above
  ─ Most human trials small; effect sizes modest but real
  ─ Largest RCT (SMILES trial, 2017): Dietary intervention → depression symptoms
    → Diet → microbiome → mental health? Plausible but unproven causal chain

  TYPE 1 DIABETES:
  ─ TEDDY study: prospective birth cohort; microbiome sampled from birth
  ─ Reduced microbial diversity + reduced Clostridiales before T1D onset
  ─ Waning of protective milk-adapted microbiome when weaning occurs
```

## Metagenomics in Microbiome Research

```
  HOW WE STUDY THE MICROBIOME
  ============================

  See also: genomics/08-BIOINFORMATICS-PIPELINE.md for file formats

  TWO MAIN APPROACHES:
  ┌────────────────────────────────────────────────────────────────┐
  │ 16S rRNA AMPLICON SEQUENCING                                   │
  │                                                                │
  │ Target: The 16S ribosomal RNA gene (present in ALL bacteria)   │
  │ Method: PCR amplify variable regions (V1-V2, V3-V4, V4)        │
  │ Cost: ~$10–30/sample (cheap, scalable)                         │
  │                                                                │
  │ Analysis:                                                      │
  │   QIIME2 / DADA2: ASV (amplicon sequence variants) calling     │
  │   ASVs > OTUs: 100% identical sequences, not 97% clustering    │
  │   Alpha diversity: Within-sample diversity (Shannon, Faith PD) │
  │   Beta diversity: Between-sample (Bray-Curtis, UniFrac)        │
  │   PCoA: Principal coordinates analysis for visualization       │
  │                                                                │
  │ LIMITATION: Can't identify most species below genus level      │
  │             No functional information                          │
  │             Primer bias: Archaea/fungi not amplified           │
  └────────────────────────────────────────────────────────────────┘

  ┌────────────────────────────────────────────────────────────────┐
  │ SHOTGUN METAGENOMICS                                           │
  │                                                                │
  │ Sequence ALL DNA in sample → reconstruct genomes + function    │
  │ Cost: ~$100–500/sample                                         │
  │                                                                │
  │ Analysis:                                                      │
  │   MetaPhlAn4: Species-level profiling from marker genes        │
  │   HUMAnN3: Functional annotation (MetaCyc pathways)            │
  │   MAG assembly: MetaSPAdes + binning (MaxBin2, MetaBAT2)       │
  │   CheckM: MAG completeness + contamination assessment          │
  │                                                                │
  │ ADVANTAGES:                                                    │
  │   Species-level resolution, strain-level with care             │
  │   Functional pathways, resistance genes, virulence genes       │
  │   Viruses, fungi, eukaryotes in same run                       │
  │                                                                │
  │ LIMITATION: Host DNA contamination (human reads dominate)      │
  │             Requires host read removal (BMTagger, Kraken2)     │
  └────────────────────────────────────────────────────────────────┘

  MULTI-OMICS LAYERS (integrating with genomics/):
  ─ Metatranscriptomics: What is actively expressed (RNA)
  ─ Metaproteomics: Proteins present (mass spectrometry)
  ─ Metabolomics: Small molecules — metabolite output (LC-MS)
  ─ Integration: microbiome → metabolites → host phenotype
    Example: Metabolon platform for untargeted metabolomics
    Statistical challenge: High-dimensional, compositional data
    (Aitchison geometry, CLR transformation required for correlation)
```

## Common Confusion Points

**"Diversity is always good"**: True in the gut (more species = more robust), FALSE in the vagina (Lactobacillus dominance = protective; diversity = BV risk). Context matters — the "rule" of diversity is niche-specific.

**Firmicutes/Bacteroidetes ratio**: Massively hyped in 2006 based on mouse data. Human studies show it is highly variable, diet-dependent, and population-specific. Do not cite this ratio as a reliable biomarker.

**Germ-free mouse data**: Used extensively for causal evidence (you can colonize a sterile mouse with specific bacteria). Critical caveat: germ-free mice have profoundly abnormal immune systems, enlarged cecums, and developmental differences from birth. Findings do not always translate to humans.

**Microbiome → mental health causality**: The gut-brain axis is real and anatomically documented. But the claim that taking probiotics will cure depression is not well-supported. Most human probiotic trials for psychiatric conditions have been small, short, and methodologically weak. The FMT-to-rat transfer data is compelling mechanistically but rodent→human translation is uncertain.

**Microbiome-based drugs vs. probiotics**: Traditional probiotics (yogurt Lactobacillus) rarely engraft in an adult gut — they pass through. New microbiome drugs (Vowst, Rebyota, SER-109) use defined or complex consortia from screened donors and are fundamentally different in mechanism and evidence quality from commercial probiotics.
