# 02 — Viral Disease

## Viral Replication, Key Virus Families, Tropism, Immune Evasion

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Viral classification: Baltimore classification (I–VII by genome type/replication strategy), +sense/−sense RNA, ssDNA/dsDNA
- Replication cycle: attachment (receptor binding) → entry → uncoating → replication → assembly → release
- Key RNA viruses:
  - Influenza: segmented genome, antigenic drift vs shift, hemagglutinin/neuraminidase
  - SARS-CoV-2: +ssRNA, spike protein, ACE2 receptor, variant evolution
  - HIV: retrovirus, reverse transcriptase, CD4+ T cell tropism, latency, AIDS progression
  - Hepatitis C: +ssRNA, NS3/4A protease (drug target), cirrhosis/HCC progression
  - Dengue/Zika/WNV: flaviviruses, ADE (antibody-dependent enhancement) in dengue
- Key DNA viruses:
  - HSV/VZV: latency in ganglia, reactivation
  - CMV/EBV: herpesvirus latency, oncogenesis (EBV + Burkitt/NPC)
  - HPV: E6/E7 → p53/Rb degradation → cervical/oropharyngeal cancer
  - Hepatitis B: partially dsDNA, HBsAg/HBeAg/HBcAg, integration
- Viral immune evasion: MHC I downregulation, interferon antagonism, complement evasion
- Oncogenic viruses: EBV, HPV, HBV, HCV, HTLV-1, KSHV
