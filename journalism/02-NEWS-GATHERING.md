# News Gathering: Beat Reporting, Source Cultivation, FOIA, Verification

## The Gathering Framework

News gathering is intelligence work — systematic collection of information from people, documents, and databases, followed by rigorous verification. The process is structured rather than ad hoc: beat systems organize knowledge accumulation; source networks provide flow of information; FOIA and document analysis access withheld records.

```
NEWS GATHERING WORKFLOW
========================

  SYSTEMATIC KNOWLEDGE BASE          REACTIVE COVERAGE
  ┌────────────────────────────┐     ┌────────────────────────────┐
  │  Beat:                     │     │  Breaking news:            │
  │  • City hall reporter      │     │  • Witness interviews      │
  │  • Courts reporter         │     │  • Official statements     │
  │  • Science reporter        │     │  • Wire feeds              │
  │                            │ ←→  │  • Social media monitoring │
  │  Proactive investigation:  │     │  • Scanner monitoring      │
  │  • FOIA requests           │     │                            │
  │  • Document analysis       │     │  Real-time verification:   │
  │  • Source development      │     │  • Two-source standard     │
  │  • Database analysis       │     │  • Official confirmation   │
  └────────────────────────────┘     └────────────────────────────┘
                   ↓                               ↓
           VERIFICATION LAYER (see below)
                   ↓
           PUBLICATION / BROADCAST
```

## The Beat System

A beat is a geographic area or topic area assigned to a reporter as their primary coverage responsibility. The beat creates systematic knowledge but also systematic risks.

```
BEAT SYSTEM ANALYSIS
=====================

  ADVANTAGES:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Source relationships: Sources call when news happens            │
  │  Institutional knowledge: Understand context for new events      │
  │  Document familiarity: Know which records exist, where, access   │
  │  Pattern recognition: Anomalies visible against baseline         │
  └──────────────────────────────────────────────────────────────────┘

  RISKS:
  ┌──────────────────────────────────────────────────────────────────┐
  │  SOURCE CAPTURE:                                                 │
  │  Long-term beat relationships → sources become friends           │
  │  Friends get favorable coverage → independence compromised       │
  │  "Beat sweetening": Withholding critical stories to preserve     │
  │   access for future stories                                      │
  │                                                                  │
  │  COGNITIVE CAPTURE:                                              │
  │  Reporter internalizes beat community's worldview                │
  │  City hall reporter starts thinking like city hall               │
  │  Mitigation: Rotation, outside editor perspective, fresh eyes    │
  │                                                                  │
  │  INSTITUTIONAL BIAS:                                             │
  │  Beats defined around institutions (police, courts, city hall)   │
  │  → Institutionally powerful sources dominate coverage            │
  │  → Non-institutional perspectives systematically underrepresented│
  └──────────────────────────────────────────────────────────────────┘

  STANDARD BEATS AT A METRO DAILY:
  City hall / municipal government     Police / criminal justice
  Courts / legal system                 Education / schools
  Business / economic development       Health / medicine
  State capitol (state government)      Federal courts
  Sports                                Arts/entertainment
```

## Source Cultivation and Confidential Sources

Sources are the lifeblood of beats. Source management is a craft with ethical constraints.

```
SOURCE TAXONOMY
===============

  BY RELIABILITY:
  ┌──────────────────────────────────────────────────────────────────┐
  │  PRIMARY SOURCES: Witnesses, participants, direct knowledge      │
  │  SECONDARY SOURCES: Secondhand knowledge, hearsay                │
  │  DOCUMENTARY SOURCES: Records, filings, databases, reports       │
  │  OFFICIAL SOURCES: Authorized spokespeople, official statements  │
  └──────────────────────────────────────────────────────────────────┘

  BY ATTRIBUTION:
  ┌──────────────────────────────────────────────────────────────────┐
  │  ON RECORD: Named and quoted; strongest attribution              │
  │  ON BACKGROUND: Information usable, source identity protected    │
  │   ("according to people familiar with the matter")               │
  │  ON DEEP BACKGROUND: Information usable, no attribution at all   │
  │   ("The White House is considering...")                          │
  │  OFF THE RECORD: Cannot be published in any form                 │
  │   (Information used only to guide further reporting)             │
  │                                                                  │
  │  CRITICAL: Attribution rules must be established BEFORE the      │
  │   source begins speaking, not retroactively negotiated           │
  └──────────────────────────────────────────────────────────────────┘
```

### Confidential Sources: Legal Protections and Limits

```
CONFIDENTIAL SOURCE FRAMEWORK
===============================

  SHIELD LAWS (US state level):
  ┌──────────────────────────────────────────────────────────────────┐
  │  49 states + DC have some form of shield protection              │
  │  Shield laws: Reporters cannot be compelled to reveal sources    │
  │  Variation: Absolute vs. qualified privilege; civil vs. criminal │
  │                                                                  │
  │  Federal level: No federal shield law (as of 2025)               │
  │  Federal courts: Qualified privilege from Branzburg v. Hayes     │
  │   (1972) — reporters can be compelled in federal proceedings     │
  │   if need is compelling, no other means, direct to issue         │
  └──────────────────────────────────────────────────────────────────┘

  WHEN CONFIDENTIAL SOURCES ARE APPROPRIATE:
  ┌──────────────────────────────────────────────────────────────────┐
  │  SPJ Code: Use only when:                                        │
  │  ① Information is vital and cannot be obtained otherwise         │
  │  ② Source has demonstrated knowledge and reliability             │
  │  ③ Source faces real risk if identified                          │
  │  ④ Story importance outweighs risk to source                     │
  │                                                                  │
  │  EVALUATE SOURCE MOTIVE:                                         │
  │  Sources leak information for reasons:                           │
  │   Whistleblower (genuine public interest)                        │
  │   Faction within organization (serving in-group)                 │
  │   Grievance (personal axe to grind)                              │
  │   Trial balloon (floating idea, seeing reaction)                 │
  │   Manipulation (feeding false information)                       │
  │  Ask: Why is this person telling me this now?                    │
  └──────────────────────────────────────────────────────────────────┘
```

## FOIA: Freedom of Information Act

The Freedom of Information Act (5 U.S.C. § 552, enacted 1966) gives any person the right to request federal agency records. State equivalents vary widely.

```
FOIA PROCESS (FEDERAL)
=======================

  STEP 1: IDENTIFY THE AGENCY
  ┌──────────────────────────────────────────────────────────────────┐
  │  Which agency holds the records you need?                        │
  │  FBI, EPA, SEC, FDA, DoD, DHS — each has separate FOIA office    │
  │  Agency locator: FOIA.gov                                        │
  └──────────────────────────────────────────────────────────────────┘

  STEP 2: CRAFT THE REQUEST
  ┌──────────────────────────────────────────────────────────────────┐
  │  Be specific but not over-narrow:                                │
  │  Specific: Names, dates, document types (emails, memos)          │
  │  Not too specific: "All emails from X about Y" → 0 records       │
  │                    if Y never named in emails                    │
  │                                                                  │
  │  Include fee waiver request (news media qualify for reduced fee)│
  │  Include expedited processing request if time-sensitive          │
  │  Send by certified mail (creates evidence of submission)         │
  └──────────────────────────────────────────────────────────────────┘

  STEP 3: AGENCY PROCESSING
  ┌──────────────────────────────────────────────────────────────────┐
  │  Statutory response time: 20 business days (rarely met)          │
  │  Actual wait: Weeks to years depending on backlog                │
  │  Track with request number                                       │
  └──────────────────────────────────────────────────────────────────┘

  STEP 4: EXEMPTIONS (why agencies deny)
  ┌──────────────────────────────────────────────────────────────────┐
  │  Nine FOIA exemptions:                                           │
  │  Exemption 1: Classified national security information           │
  │  Exemption 2: Internal agency personnel rules (limited)          │
  │  Exemption 3: Statutes prohibiting disclosure (CIA Act, etc.)    │
  │  Exemption 4: Trade secrets and commercial confidential info     │
  │  Exemption 5: Deliberative process privilege (internal memos)    │
  │  Exemption 6: Personal privacy (individuals in records)          │
  │  Exemption 7: Law enforcement records (active investigations)    │
  │  Exemption 8: Financial institution examination reports          │
  │  Exemption 9: Geological/geophysical information                 │
  │                                                                  │
  │  Most frequent invocations: Exemptions 5 and 6                   │
  │  Common problem: Agencies over-classify or over-apply Exmp 5     │
  └──────────────────────────────────────────────────────────────────┘

  STEP 5: APPEALS
  ┌──────────────────────────────────────────────────────────────────┐
  │  Administrative appeal: 20 days to file within same agency       │
  │   (different office reviews decision)                            │
  │  Federal district court: If administrative appeal denied;        │
  │   de novo review; some news organizations litigate routinely     │
  │  OGIS (Office of Government Information Services): Mediator      │
  └──────────────────────────────────────────────────────────────────┘
```

### State Open Records

Most US states have open records/sunshine laws, typically stronger than federal FOIA:
- **Police incident reports**: Usually public after active investigation closed
- **Criminal court records**: Public unless sealed by court order
- **Government contracts**: Usually public (spending disclosure)
- **Property records**: Public at county level
- **Corporate filings (SEC)**: Public through EDGAR for public companies

```
NON-SOURCE GATHERING METHODS
==============================

  OBSERVATION:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Attending public meetings, trials, press conferences            │
  │  Walking a beat, seeing conditions firsthand                     │
  │  Immersive reporting (embedding, extended presence)              │
  │  Advantage: No source filter; direct primary observation         │
  │  Best practice: Notes contemporaneous to observation             │

  ┌──────────────────────────────────────────────────────────────────┐
  │  WHAT TO LOOK FOR:                                               │

  │  • Dates and timelines (who knew what, when)                     │
  │  • Signatures and approvals (accountability trail)               │

  │                                                                  │
  │  KEY DOCUMENT TYPES:                                             │
  │  Court filings: Publicly filed, usually available                │
  │  Corporate SEC filings: 10-K, 10-Q, proxy statements             │
```

## Verification Standards

```
VERIFICATION FRAMEWORK
=======================

  THE TWO-SOURCE RULE (Watergate standard):
  ┌──────────────────────────────────────────────────────────────────┐
  │  For damaging claims about named individuals:                    │
  │  Require minimum two independent sources, OR                     │
  │  One source + documentary corroboration                          │
  │                                                                  │
  │  "Independent" means sources who don't know each other's info    │
  │  Two sources repeating the same rumor ≠ two independent sources│
  └──────────────────────────────────────────────────────────────────┘

  FIVE-QUESTION VERIFICATION CHECKLIST:
  ┌──────────────────────────────────────────────────────────────────┐
  │  ① Is the source in a position to know what they claim to know?│
  │  ② Is the source reliable (track record of accuracy)?            │
  │  ③ Does the source have a motive to distort or mislead?          │
  │  ④ Can the claim be corroborated by document or other source?    │
  │  ⑤ Has the subject of the claim been given opportunity to        │
  │     respond?                                                     │
  └──────────────────────────────────────────────────────────────────┘

  PRE-PUBLICATION FACT-CHECKING:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Call-back checks: Contact every named person in story           │
  │  Numerical checks: Verify all figures, calculations, statistics│
  │  Quote checks: Read back direct quotes to sources for accuracy   │
  │   (NOT for approval — for accuracy)                              │
  │  Date/title checks: Official titles, exact dates verified        │
  │  Photo captions: Each caption independently verified             │
  └──────────────────────────────────────────────────────────────────┘

  WIRE FEED VERIFICATION:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Wire service stories must be verified before republication      │
  │  Wire errors propagate: Once published by AP, repeated           │
  │   everywhere → corrections harder to spread than errors          │
  │  Mark as "developing" if verification incomplete                 │
  └──────────────────────────────────────────────────────────────────┘
```

## Decision Cheat Sheet

| Situation | Gathering Approach |
|-----------|-------------------|
| Breaking news, unknown facts | Official sources first, verify with witnesses |
| Investigative tip | Document it before pursuing; assess source motive |
| Government records needed | FOIA first; identify correct agency |
| Corporate malfeasance | SEC filings + court records + FOIA of relevant agencies |
| Election accountability | FEC filings + state campaign finance databases |
| Source wants deep background | Establish rules before source begins speaking |
| Key source refuses to go on record | Seek documentary corroboration; pursue other sources |
| Viral social media claim | Reverse image search + source metadata + original context |

## Common Confusion Points

**"No comment" is not confirmation.** A subject declining to comment is not evidence of guilt or truth of a claim. It is a non-answer that should be reported as such: "X did not respond to requests for comment" or "X declined to comment."

**Two sources is a floor, not a ceiling.** The two-source rule (derived from Watergate practice) is the minimum for damaging claims about named individuals. Complex investigative stories require more — pattern of evidence, documents, multiple corroborating sources, subject interview.

**FOIA is slow but powerful.** Average federal FOIA response time is months to over a year. Investigative reporters plan FOIA requests far in advance. The backlog is real; expedited processing is rarely granted. File FOIA requests early and follow up persistently.

**Attribution rules must be established in advance.** A source cannot retroactively declare an already-given interview "off the record." Attribution status is negotiated before the conversation, not after. If a source tries to retroactively impose confidentiality, the journalist is not obligated to accept it — though professional relationships depend on judgment.
