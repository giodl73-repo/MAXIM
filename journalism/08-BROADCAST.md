# Broadcast Journalism: Television News, Radio, Live Reporting, Chyrons

## The Broadcast Journalism Landscape

Broadcast journalism is news delivered through audio and video media — structurally constrained by time (30-second packages vs. 3,000-word print articles), distribution technology (spectrum licensing, satellite, cable, streaming), and the performance demands of on-camera or on-microphone presentation. The format shapes the journalism: broadcast favors the visual and emotional over the analytical; immediacy over depth; conflict and drama over process.

  THE NIGHTLY NEWS RUNDOWN:
  ┌──────────────────────────────────────────────────────────────────┐
  │  TEASE (10–15s):                                                 │
  │  Top story preview before commercial; creates urgency/retention  │
  │                                                                  │
  │  OPEN / ANCHOR INTRO:                                            │
  │  Anchor delivers 10–20s setup from desk; introduces package      │
  │                                                                  │
  │  PACKAGE (1:30–2:30):                                            │
  │  Main news story form — see below                                │
  │                                                                  │

  │                                                                  │
  │  VO/SOT (Voice Over / Sound On Tape, 30–60s):                    │
  │  Anchor VO + embedded sound bite from interview; no stand-up     │

  ┌──────────────────────────────────────────────────────────────────┐

  │             ▼                                                    │

  │             │                                                    │

  │  SOT #2 (10–20s)                                                 │

## Chyrons and On-Screen Text

```
CHYRON SYSTEMS AND LOWER THIRDS
=================================

  TERMINOLOGY:
  ┌──────────────────────────────────────────────────────────────────┐
  │  CHYRON: Brand name → genericized to mean any on-screen text     │
  │  LOWER THIRD: Graphic element in lower third of screen           │
  │   (name + title identification)                                  │
  │  SUPER: Superimposed text over video                             │
  │  CRAWL / TICKER: Horizontal scrolling text across bottom         │
  │  BANNER HEADLINE: Full-width text typically at bottom or top     │
  └──────────────────────────────────────────────────────────────────┘

  LOWER THIRD STANDARDS:
  ┌──────────────────────────────────────────────────────────────────┐
  │  NAME LINE: Person's name as they identify themselves            │
  │  TITLE LINE: Current role or context (not historical)            │
  │                                                                  │
  │  ACCURACY REQUIREMENTS:                                          │
  │  • Title must be current at time of interview                    │
  │  • Former officials: "Former [Title]" explicitly                 │
  │  • "Republican strategist" vs. "Republican consultant":          │
  │    active political alignment vs. paid advisory role             │
  │  • Misidentification = defamation risk                           │
  │                                                                  │
  │  NOTABLE FAILURES:                                               │
  │  CNN (2022): Lower third labeling issues on cable news panels    │
  │  Networks often mislabel party affiliation (R as D or vice versa │
  │   — corrections run but damage already done via screenshot)     │
  └──────────────────────────────────────────────────────────────────┘

  BREAKING NEWS CHYRONS:
  ┌──────────────────────────────────────────────────────────────────┐
  │  PROBLEM: Breaking news chyrons often overstate certainty        │
  │   before information is confirmed                                │
  │   "SHOOTER DEAD" → shooter apprehended (not dead)               │
  │   "MULTIPLE FATALITIES" → number unknown initially               │
  │                                                                  │
  │  CABLE NEWS FAILURE MODE: Chyron becomes leading fact for        │
  │   viewers who catch partial broadcast; correction added later    │
  │   but chyron screenshot spreads on social media                 │
  │                                                                  │
  │  STANDARD: Chyron should match actual confirmed information;     │
  │   prefer "Reports of..." to definitive statement when unconfirmed│
  └──────────────────────────────────────────────────────────────────┘
```

## Live Reporting

```
LIVE SHOT MECHANICS AND ETHICS
================================

  TECHNICAL COMPONENTS:
  ┌──────────────────────────────────────────────────────────────────┐
  │  IFB (Interruptible Foldback): Earpiece — producer/anchor       │
  │   can interrupt correspondent live; correspondent hears          │
  │   anchor intro and own delayed audio feed                       │
  │  MICROWAVE/SATELLITE UPLINK: Signal path from field to air     │
  │  STAND-UP POSITION: Location with visual context behind         │
  │   correspondent — courthouse, scene, landmark                  │
  └──────────────────────────────────────────────────────────────────┘

  LIVE SHOT PROBLEM: PRESENCE OVER SUBSTANCE:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Classic scenario:                                               │
  │  Event occurred at 9 AM; live shot at 6 PM news                  │
  │  Scene is: empty parking lot, closed building                    │
  │  Correspondent provides no new information                       │
  │  Rationale: "Shows we have someone there"                        │
  │                                                                  │
  │  Critique: Live shot for presence rather than news value         │
  │   • Consumes slot that could hold verified new information       │
  │   • Creates illusion of journalism where information is static   │
  │   • Correspondent speaking confidently about things not known    │
  └──────────────────────────────────────────────────────────────────┘

  TWO-WAY ANCHOR/CORRESPONDENT:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Function: Extends package; anchor asks 1–2 questions            │
  │  Risk: Anchor may ask question correspondent cannot answer live  │
  │  Correspondents must pre-brief anchor on what they know and      │
  │   don't know — avoid speculating live to fill air time           │
  └──────────────────────────────────────────────────────────────────┘
```

## Network vs. Cable News

```
NETWORK vs. CABLE NEWS MODEL
==============================

  NETWORK NEWS (ABC/CBS/NBC):
  ┌──────────────────────────────────────────────────────────────────┐
  │  Half-hour evening news → 22 minutes of content                  │
  │  (8 minutes advertising at 30 minutes)                           │
  │  Appointment TV model: 6:30/7pm EST daily slot                   │
  │  Audience: Aging but large (6–9 million viewers per network)     │
  │  Economics: Still profitable via advertising; declining          │
  │  Format: Traditional editorial gatekeeping; tight story counts   │
  │  Average story length: 1:30–2:00 package; 20–40s VO              │
  └──────────────────────────────────────────────────────────────────┘

  CABLE NEWS (CNN/Fox News/MSNBC):
  ┌──────────────────────────────────────────────────────────────────┐
  │  24-hour cycle: Must fill every minute                           │
  │  Result: Repetition, panel discussion, prediction/analysis       │
  │  Breaking news mode: Continuous coverage with incomplete info    │
  │                                                                  │
  │  CABLE ECONOMICS:                                                │
  │  Revenue: Subscriber fees (cable bundle) + advertising           │
  │  Fox News revenue: ~$3B+/year (subscription fees dominant)       │
  │  CNN: Restructuring post-cord-cutting (bundle collapse)          │
  │                                                                  │
  │  PARTISAN DIFFERENTIATION:                                       │
  │  Fox News: Conservative audience; opinion-news blending          │
  │  MSNBC: Liberal audience; same blending                          │
  │  CNN: Attempted neutrality; declining ratings pre-2024           │
  │   restructuring; moving to streaming                             │
  │                                                                  │
  │  24-HOUR PROBLEM:                                                │
  │  Real news doesn't fill 24 hours daily                           │
  │  Solution: Analysts, pundits, speculation, repetition            │
  │  Result: Opinion content crowding out reported journalism        │
  └──────────────────────────────────────────────────────────────────┘

  RATINGS MEASUREMENT (NIELSEN):
  ┌──────────────────────────────────────────────────────────────────┐
  │  RATINGS: % of all TV households watching                        │
  │  SHARE: % of TVs in use watching (more useful for time slots)    │
  │  DEMO: 25–54 age demo commands advertising premium               │
  │  Cable sells on "Total Day" viewership averages                  │
  │  Nielsen PPM (Portable People Meter): passive metering           │
  │   replaced paper diaries; panel-based; streaming now measured    │
  └──────────────────────────────────────────────────────────────────┘
```

## Broadcast Ethics

```
BROADCAST-SPECIFIC ETHICAL STANDARDS
======================================

  SOUND BITE SELECTION:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Full interview may run 20–30 minutes                            │
  │  Broadcast: 20–25 seconds selected                               │
  │  RISK: Out-of-context selection changes meaning                  │
  │                                                                  │
  │  STANDARD: Sound bite must accurately represent speaker's view   │
  │  NOT acceptable: Editing within a single sentence (creates       │
  │   statement never made)                                          │
  │  NOT acceptable: Selecting bite that contradicts the main        │
  │   thrust of what the subject actually said                       │
  └──────────────────────────────────────────────────────────────────┘

  LIVE COVERAGE ERRORS:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Breaking news is the highest-risk period for inaccuracy         │
  │  First reports almost always wrong in details                    │
  │                                                                  │
  │  BOSTON MARATHON BOMBING (2013):                                 │
  │  CNN, AP, Post incorrectly reported arrests                      │
  │  Root cause: Competitive pressure; unverified scanner traffic    │
  │  Post-mortem: Verification process bypassed under competition    │
  │                                                                  │
  │  BROADCAST LIVE ERROR PROBLEM:                                   │
  │  Live air is not correctable in real time                        │
  │  Anchor must interrupt own script to correct mid-broadcast       │
  │  Social media amplifies initial error before correction airs    │
  └──────────────────────────────────────────────────────────────────┘

  STAGING FOR BROADCAST:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Re-enactment: Prohibited without explicit labeling              │
  │  Walking shot: Reporter "walking" to camera setup — accepted     │
  │   as production convention                                       │
  │  Re-asking interview questions for camera coverage ("noddies")   │
  │   — accepted convention when disclosed to production team       │
  │  B-roll deception: Pairing archival footage with current story   │
  │   without disclosure — not acceptable                            │
  └──────────────────────────────────────────────────────────────────┘
```

## Decision Cheat Sheet

| Broadcast Situation | Standard |
|--------------------|----------|
| Breaking news: first reports unconfirmed | Report confirmed facts only; hedge with "reports of" or "unconfirmed"; do not chase competitor air |
| Live shot at empty scene hours after event | Question news value; consider whether presence advances story or just performs presence |
| Lower third title outdated mid-broadcast | Correct immediately; note "former" if applicable |
| Chyron overstates confirmed facts | Use hedging language until confirmed; update chyron in real time |
| Sound bite selected out of context | Ensure bite represents speaker's overall position; don't use bite that inverts meaning |
| Guest/analyst speculates live about pending event | Anchor must push back; label speculation as speculation, not fact |
| Re-enactment needed for documentary | Label explicitly ("re-enactment") in frame; never present as documentary footage |
| Podcast: host discusses reporting vs. opinions | Label clearly; ensure listeners understand difference between original reporting and analysis |
| Nielsen ratings vs. editorial judgment | Ratings inform scheduling; should not dictate story selection; that is advertiser capture |

## Common Confusion Points

**The package length constraint drives the journalism failure modes.** A 2-minute package cannot carry the complexity a 3,000-word investigative article can. This is not laziness — it is structural. Broadcast's editorial compromises (oversimplification, false conflict, emotional framing) flow directly from time compression. Understanding this is understanding broadcast journalism.

**Live shots are performance, not journalism.** The presence of a reporter at a location is a production value, not a journalistic value. It signals the network has resources and investment. It rarely produces new information hours after the event. Treating the live shot as journalism confuses the form with the function.

**Cable news opinion shows are not news broadcasts.** Tucker Carlson, Rachel Maddow, and equivalents are opinion programs occupying the same channel as news broadcasts. The FCC does not regulate content of cable (only over-the-air broadcast). Viewers who do not distinguish news from opinion receive a distorted information product.

**Sound bite length did not always compress.** The average presidential candidate sound bite on US network news shrank from 43 seconds (1968) to 8 seconds (1988) and continued shrinking. This is a structural change in broadcast journalism — it did not begin with social media. Social media accelerated an existing trend.

**Podcast "downloads" and "listeners" are different metrics.** A download occurs when a device retrieves the file. A listener is a person who plays it. Download numbers overstate audience significantly. Industry moved to "starts" (pressed play) and "completions" (finished episode) as more meaningful metrics.
