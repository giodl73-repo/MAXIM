# Information Geometry

```
THE GEOMETRIC STRUCTURE OF STATISTICS

Probability distributions form a manifold.
The natural metric is the Fisher information matrix.
The natural "straight lines" are exponential families.
The natural notion of distance is KL divergence.

┌───────────────────────────────────────────────────────────────────┐
│  Statistical manifold M = {p(x;θ) : θ ∈ Θ ⊆ ℝᵈ}                │
│                                                                   │
│  Tangent vector at θ = score function ∂_i log p(x;θ)             │
│                                                                   │
│  Riemannian metric = Fisher information: g_{ij}(θ) = I(θ)_{ij}  │
│                                                                   │
│  Two natural connections (not the Levi-Civita connection):        │
│    e-connection (exponential):  α = +1                           │
│    m-connection (mixture):      α = −1                           │
│                                                                   │
│  Exponential families are FLAT in both e and m coordinates       │
│  → "dually flat" structure is the key geometric fact             │
└───────────────────────────────────────────────────────────────────┘
```

## 1. Fisher Information Metric

The Fisher information matrix measures how fast the likelihood changes as θ changes:

```
g_{ij}(θ) = E_{x~p(·;θ)}[∂_i log p(x;θ) · ∂_j log p(x;θ)]
           = -E_{x~p(·;θ)}[∂_i ∂_j log p(x;θ)]   [second form, by regularity]

For 1D: I(θ) = E[(∂_θ log p(x;θ))²] = Var[score]

Cramér-Rao bound:
  Var[T(X)] ≥ 1/I(θ)    for any unbiased estimator T(X) of θ

  Geometric interpretation: estimator variance ≥ inverse Fisher metric
  Equality: T(X) is the MLE (maximum likelihood estimator)
```

**Invariance under reparametrization**: If φ = f(θ), then the Fisher metric transforms as
a (0,2) tensor — it's a genuine Riemannian metric independent of coordinate choice.

**Geodesic distance**: The Rao distance between distributions p(·;θ₀) and p(·;θ₁) is:
```
d(θ₀,θ₁) = min_{γ(0)=θ₀, γ(1)=θ₁} ∫₀¹ √(γ'(t)ᵀ I(γ(t)) γ'(t)) dt
```
This is the information-theoretic "true distance" between distributions, invariant under
sufficient statistics (data compression that preserves all information).

## 2. Exponential Families: Dually Flat Geometry

The canonical exponential family has the form:

```
p(x;θ) = h(x) exp(θᵀ T(x) - A(θ))

  θ:    natural parameters  (e-coordinates)
  T(x): sufficient statistics
  A(θ): log-partition function (cumulant generating function)
  h(x): base measure

Key functions:
  A(θ) = log ∫ h(x) exp(θᵀ T(x)) dx     [convex in θ]
  η = ∇A(θ) = E[T(X)]                    [mean parameters / m-coordinates]
  A*(η) = sup_θ [θᵀη - A(θ)]             [Legendre conjugate, convex in η]

Examples:
  Gaussian N(μ,σ²):   θ = (μ/σ², -1/2σ²),  T = (x, x²),   A = -θ₁²/4θ₂ - ½log(-2θ₂)
  Bernoulli(p):        θ = log(p/1-p),       T = x,          A = log(1+eᶿ)
  Poisson(λ):          θ = log λ,            T = x,          A = eᶿ
  Multinomial:         θ = log(pₖ/p₀),      T = (1_{x=k}),  A = log(Σ eᶿᵢ)
```

**Dually flat structure** (Amari-Nagaoka):

```
e-flat:  The manifold is flat under the e-connection (exponential connection)
         e-geodesics = lines in θ-coordinates

m-flat:  The manifold is flat under the m-connection (mixture connection)
         m-geodesics = lines in η-coordinates

Duality: g(∇^e_X Y, Z) + g(Y, ∇^m_X Z) = X·g(Y,Z)
  The two connections are dual w.r.t. the Fisher metric
```

**Bregman divergence and KL**: For exponential families:
```
KL(p_θ‖p_θ') = D_A(η‖η') = A*(η) + A(θ') - η·θ'
             = Bregman divergence generated by A*

Reverse KL:   KL(p_θ'‖p_θ) = D_{A*}(θ‖θ') = Bregman(A*, θ, θ')
```

## 3. α-Divergences and α-Connections

The α-divergence family unifies KL and other divergences:

```
D_α(P‖Q) = (4/(1-α²)) [1 - ∫ p^{(1+α)/2} q^{(1-α)/2} dμ]    for α ≠ ±1

Special cases:
  α = +1:   D₁(P‖Q)  = KL(P‖Q)            (e-connection / forward KL)
  α = −1:   D₋₁(P‖Q) = KL(Q‖P)            (m-connection / reverse KL)
  α = 0:    D₀(P‖Q)  = 4/3 · Hellinger²   (Hellinger distance, squared × constant)

  Hellinger²: H²(P,Q) = ½ ∫ (√p - √q)² dμ
              = 1 - ∫ √(pq) dμ    [Bhattacharyya coefficient subtracted from 1]

α-connection ∇^{(α)}: defined from α-divergence via Hessian of log
  α = +1: e-connection
  α = −1: m-connection
  α = 0:  Levi-Civita connection (Riemannian, metric-compatible)
```

**Statistical interpretation**:
- Forward KL (α=+1): zero-avoiding. Fitting p to q with forward KL → q covers all modes of p.
- Reverse KL (α=-1): zero-forcing. Fitting q to p with reverse KL → q concentrates on modes of p.
- Hellinger (α=0): symmetric, bounded in [0,1], satisfies triangle inequality.

## 4. Pythagorean Theorem for KL Divergence

**The key geometric fact enabling iterative algorithms**:

```
Let M be an e-flat submanifold (closed under exponential mixture).
Let q* = argmin_{q ∈ M} KL(p‖q)  [e-projection of p onto M]

Pythagorean theorem:
  KL(p‖r) = KL(p‖q*) + KL(q*‖r)    ∀r ∈ M

  p
  │╲
  │  ╲  KL(p‖r)
  │    ╲
  q* ──── r      (q* is the "right angle" on M)
  ∈ M

Analogously for m-flat M and m-projection:
  KL(r‖q*) + KL(q*‖p) = KL(r‖p)    where q* = argmin_{q∈M} KL(q‖p)
```

**Bregman projections** (generalization): For any Bregman divergence D_f:
D_f(p,r) = D_f(p,q*) + D_f(q*,r) when q* is the Bregman projection.
Used in: iterative projection algorithms, ADMM, Sinkhorn iterations.

## 5. Natural Gradient (Amari)

Standard gradient descent ignores the geometry of parameter space:

```
Standard SGD: θ_{t+1} = θ_t - α · ∇_θ L(θ)
  Problem: gradient ∇_θ L is in Euclidean coordinate space
  Different parameterizations give different update directions
  Not invariant under reparametrization

Natural gradient: θ_{t+1} = θ_t - α · F(θ)^{-1} ∇_θ L(θ)
  F(θ) = Fisher information matrix = E[∇log p · ∇log p^T]
  Fisher-natural gradient moves in the steepest descent direction
  w.r.t. the Fisher-Rao metric on the statistical manifold
  Invariant under reparametrization
```

**Why natural gradient is faster**: For exponential families, natural gradient in η-coordinates
(m-flat) corresponds to Newton's method on the log-likelihood. Convergence is quadratic near
optimum vs linear for gradient descent.

**K-FAC** (Kronecker-Factored Approximate Curvature): Approximates F^{-1} for neural networks
by factoring across layers. Layer l's Fisher block ≈ A_{l-1} ⊗ Γ_l where A is activation
covariance and Γ is backprop gradient covariance. Makes natural gradient tractable for deep
networks.

```
Fisher matrix structure for neural network weight W (layer l):

  F_W = E[(a_{l-1}a_{l-1}^T) ⊗ (g_l g_l^T)]  ≈  A_{l-1} ⊗ G_l   [K-FAC]

  Exact: O(p⁴) to store and invert
  K-FAC: O(d_in² + d_out²) per layer — tractable
```

## 6. EM Algorithm as Alternating Projections

The EM algorithm has an elegant geometric interpretation:

```
Complete data: (X, Z) where Z is latent
Observed data: X
Parameter: θ ∈ Θ

Geometric picture:
  S = {p(x,z;θ) : θ ∈ Θ}     [model manifold, e-flat for exp families]
  M = {r(x,z) : marginal_z r = p_observed(x)}   [data constraint, m-flat]

E-step: m-projection of current model onto M
  → q(x,z) = p(x,z;θ_old) updated to match observed marginal
  Equivalently: q(z|x) = p(z|x;θ_old)  [posterior]

M-step: e-projection of q back onto S
  → θ_new = argmax_θ E_q[log p(x,z;θ)]   [maximize expected complete log-likelihood]

Pythagorean theorem ensures KL(target‖p_θ) decreases monotonically:
  KL(p*‖p_{θ_new}) ≤ KL(p*‖p_{θ_old})

┌─────────────────────────────────────────────────────────────────┐
│  E-step = m-projection:  q ← best model matching observed data  │
│  M-step = e-projection:  θ ← best exponential family at q      │
│  Alternate projections converge to intersection point           │
│  (if S ∩ M ≠ ∅) or to closest point on S to M                  │
└─────────────────────────────────────────────────────────────────┘
```

**For exponential families**, M-step is exact (closed form): the natural gradient is zero
at the M-step solution, and the exponential family structure makes the projection analytic.

## 7. Optimal Transport and Wasserstein Geometry

Complementary geometry to Fisher-Rao: treats distributions as masses to be transported.

```
Monge-Kantorovich problem (optimal transport):
  Given source p, target q, cost c(x,y):

  W(p,q) = min_{γ ∈ Π(p,q)} E_{(X,Y)~γ}[c(X,Y)]

  Π(p,q) = {joint distributions with marginals p, q}
  γ = transport plan  (how to move mass from p to q)

Wasserstein-2 distance (c = ‖x-y‖²):
  W₂(p,q)² = min_{γ ∈ Π(p,q)} E[‖X-Y‖²]

For 1D: W₂²(p,q) = ∫₀¹ (F_p^{-1}(u) - F_q^{-1}(u))² du
  [L₂ distance between quantile functions]
```

**Monge-Kantorovich duality**:
```
W₁(p,q) = max_{‖f‖_Lip≤1} E_p[f] - E_q[f]   [Kantorovich-Rubinstein]
W₂²(p,q) = max_{φ,ψ c-conjugate} E_p[φ] + E_q[ψ]
```

**Geodesics = displacement interpolation** (McCann 1997):
```
Wasserstein geodesic from p to q:
  μ_t = (T_t)_# p    where T_t = (1-t)·id + t·T_Monge
  T_Monge: optimal transport map from p to q

  μ_0 = p, μ_1 = q
  Particles move in straight lines in x-space
```

**Benamou-Brenier formula**: W₂ = min ∫₀¹ E_{μ_t}[‖v_t‖²] dt over all velocity fields v_t
satisfying continuity equation ∂_t μ_t + ∇·(μ_t v_t) = 0. Wasserstein distance = minimum
kinetic energy of fluid flow.

**Sinkhorn algorithm** (entropic regularization):
```
W_ε(p,q) = min_{γ ∈ Π(p,q)} E_γ[c] + ε·KL(γ‖p⊗q)

Sinkhorn iterations (Bregman projections):
  γ ← argmin_{γ' with row marginal p} KL(γ'‖γ)   [project onto row constraint]
  γ ← argmin_{γ' with col marginal q} KL(γ'‖γ)   [project onto col constraint]

Converges geometrically; ε→0 recovers Wasserstein distance
O(n²/ε²) vs O(n³ log n) for exact OT — preferred in practice
```

## 8. Applications in Machine Learning

**Information bottleneck** (Tishby-Pereira-Bialek 2000):

```
Lagrangian on information plane (H_X vs H_Y):
  min I(X;T) - β·I(T;Y)

  T = compressed representation of X
  I(X;T): compression (minimize)
  I(T;Y): relevance to Y (maximize)
  β: Lagrange multiplier (tradeoff)

Optimal T satisfies:
  p(t|x) ∝ p(t) exp(-β·D_KL(p(y|x) ‖ p(y|t)))

IB-Lagrangian = free energy in exponential family interpretation
  β = inverse temperature; IB curve = phase diagram of optimal compression
```

**VAE ELBO as free energy minimization** (connects 04-ML-CRYPTOGRAPHY-BRIDGE.md):

```
ELBO = E_q[log p(x|z)] - KL(q(z|x)‖p(z))
     = -KL(q(z|x)‖p(z|x)) - log p(x)   [rewriting]

Geometric: ELBO maximization = E-projection of q onto model manifold
  Encoder defines q(z|x) [recognition model]
  Decoder defines p(x|z) [generative model]
  ELBO maximization iterates projections, converging to local optimum
```

**Natural gradient in practice**:

| Method | Fisher approximation | Cost | When to use |
|--------|---------------------|------|-------------|
| SGD | Identity (ignore curvature) | O(p) | Large models |
| Adam | Diagonal, adaptive | O(p) | Default in DL |
| K-FAC | Kronecker-factored per layer | O(d²) | Medium networks |
| Shampoo | Block-diagonal Kronecker | O(d²) | Large language models |
| Exact NGD | Full Fisher F | O(p²) | Tiny models only |

## Common Confusion Points

**Fisher-Rao vs Wasserstein geometry**: Both are metrics on probability distributions.
Fisher-Rao is local (infinitesimal deformations of the density function). Wasserstein is
global (mass transport). Fisher-Rao is appropriate for statistical inference (perturbing
parameters). Wasserstein is appropriate for interpolating distributions (moving mass).

**KL divergence is not a metric**: KL(p‖q) ≠ KL(q‖p) and triangle inequality fails.
It's a Bregman divergence in the exponential family framework. Symmetric versions (JS
divergence, √JS is a metric) fix these issues but lack the clean information-theoretic interpretation.

**Forward vs reverse KL for fitting**: Minimizing KL(p‖q) over q (m-projection) = mean-seeking
(q captures the mean/mode of p but spreads out). Minimizing KL(q‖p) over q (e-projection) =
mode-seeking (q concentrates on one mode of p). This explains why variational inference
(ELBO = −KL(q‖p)) tends to underestimate variance relative to MCMC.

**Natural gradient invariance**: Natural gradient = ordinary gradient in intrinsic coordinates
of the Fisher manifold. Invariance under reparametrization means: if you fit N(μ,σ²) by θ = (μ,σ²)
or by φ = (μ/σ, log σ), natural gradient takes the same step in distribution-space. Ordinary
gradient does not.

**Sinkhorn ≠ exact OT**: Sinkhorn with regularization ε computes W_ε, which approximates W₂ as
ε→0 but blurs the optimal transport plan. The blurring = entropy regularization. Convergence
to exact OT is uniform on compact sets but slows badly as ε→0. In practice, choose ε large
enough for stable convergence; the resulting cost is slightly larger than W₂.
