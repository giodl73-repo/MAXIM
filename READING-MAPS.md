# Reading Maps

*13 guided paths through the 52-volume MAXIM reference library. Each path starts in one section and radiates outward — following the ideas, not the shelves.*

**How to read these maps**: Each stop is a volume (card). The line shows the recommended order. Branches show optional detours. One sentence per stop says why it's next.

## Cross-Path Connections

Some volumes are crossroads — they appear on multiple paths. These are the library's natural hubs:

```
MOST CONNECTED VOLUMES (appear on 3+ paths):
──────────────────────────────────────────────
J♣  Mathematics/Physics/Electronics     Paths: 2, 6, 7, 10
5♣  Natural Sciences/Biology            Paths: 1, 2, 4
6♦  Philosophy/Mythology/Anthropology   Paths: 4, 5, 9
K♣  Computing/AI/Data Science           Paths: 7, 10, 12
7♣  Mechanical/Structural/Aeronautics   Paths: 3, 6, 0
9♣  Economics/Finance/Game Theory       Paths: 5, 8
```

These hubs are where paths cross. If you're following the Naturalist path and want to understand the physics underneath, J♣ is the bridge to the Mathematician's path. If you're on the Engineer's path and hit formal methods, 8♦ bridges to the Technologist's path.

**The map is not the territory — but it tells you where the trails cross.**
