# History of Science — Overview: Science as Social Process

## The Big Picture

Science is not a method — it is an institution, a practice, and a self-correcting social process with structure, pathologies, and failure modes. The 20th century produced more sophisticated theories of how science actually works than any prior era. None of them fully agree.

## Layer 1: Popper — Falsificationism

### The Core Claim

Science is demarcated from non-science by **falsifiability**: a theory is scientific if and only if it makes predictions that could in principle be shown false by observation.

```
+---------------------------+        +---------------------------+
|   SCIENTIFIC THEORY       |        |   NON-SCIENTIFIC THEORY   |
+---------------------------+        +---------------------------+
| Makes specific predictions|        | Explains everything       |
| Could be wrong            |        | Unfalsifiable by design   |
| Survives attempts to      |        | Any data is consistent    |
| refute it — corroborated  |        | "The stars were aligned   |
+---------------------------+        | poorly that day"          |
| Example: General          |        +---------------------------+
| Relativity predicts light | vs     | Example: Astrology,       |
| bending at 1.75 arcsec    |        | Freudian theory (covers   |
| around the Sun            |        | all outcomes)             |
+---------------------------+        +---------------------------+
```

### The Problem of Induction

Popper's move: **don't solve the problem of induction — bypass it**. We never justify theories inductively. We only provisionally accept theories that have survived serious attempts to falsify them. A single black swan falsifies "all swans are white" — but a million white swans cannot prove it.

```
Hume's problem:                  Popper's response:
"No finite number of              "Modus tollens works asymmetrically"
observations can prove a
universal statement"              (H → P) ∧ ¬P → ¬H

                                  If theory predicts P and P is false,
                                  theory is false. One counterexample
                                  is logically decisive.
```

### Ad Hoc Modifications and the Duhem-Quine Problem

**Duhem (1906) / Quine (1951)**: any hypothesis is tested in conjunction with auxiliary hypotheses. When a prediction fails, you can always protect the central hypothesis by adjusting auxiliaries.

```
Observation: Mercury perihelion precession anomaly (43 arcsec/century)

Newtonian response options:
  Option A: Newton's gravity is wrong                (hypothesis revision)
  Option B: Vulcan (unknown inner planet) exists     (auxiliary revision)
  Option C: Newton's model of the Sun is wrong       (auxiliary revision)

All three are logically available. Popper: B and C are ad hoc
if they make no NEW testable predictions. GR made new predictions
(light bending, gravitational redshift) — not ad hoc.
```

**Lakatos sharpens this**: an ad hoc modification is one that adds no "excess empirical content." A non-ad-hoc modification predicts something new that turns out to be correct.

### Falsificationism's Problems

| Problem | Description |
|---------|-------------|
| Duhem-Quine | Falsification is never of single hypotheses |
| Corroboration | Why does surviving tests give us confidence? |
| Probability | Probabilistic theories are never strictly falsifiable |
| History | Scientists don't actually abandon theories on single disconfirmations |

## Layer 2: Kuhn — Paradigms and Scientific Revolutions

### Normal Science and Paradigms

Kuhn's **paradigm** = the constellation of shared beliefs, values, exemplary problems, and techniques that define a scientific community's practice. Not just a "theory" — a world-view plus toolkit.

```
+--------------------------------------------------+
|              NORMAL SCIENCE                       |
|  Paradigm defines:                               |
|  - What counts as a legitimate problem           |
|  - What counts as a legitimate solution          |
|  - What instruments/methods are appropriate      |
|  - What the world is made of (ontology)          |
|                                                  |
|  Activity: "puzzle-solving" — fitting nature     |
|  into the paradigm's boxes                       |
|                                                  |
|  Anomalies arise and are initially SET ASIDE     |
|  ("the instrument was faulty", "we'll fix it     |
|  later") — paradigm is NOT abandoned             |
+--------------------------------------------------+
          |
          | Anomalies accumulate until they cannot be
          | explained away — CRISIS
          v
+--------------------------------------------------+
|              CRISIS                               |
|  Proliferation of competing theories             |
|  Foundational assumptions questioned             |
|  Older scientists cling to paradigm              |
|  ("science advances one funeral at a time"       |
|   -- Planck's observation)                       |
+--------------------------------------------------+
          |
          | A new paradigm emerges — REVOLUTION
          v
+--------------------------------------------------+
|              PARADIGM SHIFT                       |
|  New paradigm incommensurable with old           |
|  Scientists convert (Gestalt-switch)             |
|  History rewritten in light of new paradigm     |
|  Back to normal science under new paradigm      |
+--------------------------------------------------+
```

### Historical Examples of Kuhnian Paradigm Shifts

| Revolution | Old Paradigm | New Paradigm | Anomaly That Forced Crisis |
|------------|-------------|--------------|---------------------------|
| Copernican | Ptolemaic geocentrism | Heliocentric system | Complexity of epicycles, qualitative accuracy limits |
| Chemical | Phlogiston theory | Lavoisier oxygen | Weight gain on calcination |
| Geological | Catastrophism | Uniformitarianism + plate tectonics | Continental shapes, fossil distribution |
| Physical | Classical mechanics | Quantum + relativity | Ultraviolet catastrophe, Michelson-Morley null result |
| Biological | Species fixism | Darwinian evolution | Fossil succession, biogeography |

### The Incommensurability Problem

Kuhn's most controversial claim: paradigms are **incommensurable** — no neutral language exists in which both can be evaluated. The Newtonian "mass" and the relativistic "mass" are different concepts with the same word.

```
Ptolemaic astronomer says: "The Sun moves around the Earth"
Copernican astronomer says: "The Earth moves around the Sun"

They appear to disagree. But do they share a reference frame?
Ptolemaic "Earth" = privileged center of the cosmos
Copernican "Earth" = one planet among several

Same word, different conceptual framework — incommensurable.
```

**Consequences**: If incommensurability is real, theory choice involves non-rational elements (values, aesthetics, sociology). Kuhn never said science was irrational — just that paradigm changes are not purely algorithmic.

## Layer 3: Lakatos — Research Programmes

### More Fine-Grained Than Kuhn

Lakatos distinguishes what Kuhn collapses. A **scientific research programme** has structure:

```
+--------------------------------------------------+
|          RESEARCH PROGRAMME STRUCTURE             |
+--------------------------------------------------+
|                                                  |
|  HARD CORE                                       |
|  +-----------------------------------------+    |
|  | Central theoretical assumptions          |    |
|  | Protected by convention — NOT abandoned  |    |
|  | Example (Newtonian): F=ma, inverse       |    |
|  | square gravity, absolute space/time      |    |
|  +-----------------------------------------+    |
|                                                  |
|  PROTECTIVE BELT                                 |
|  +-----------------------------------------+    |
|  | Auxiliary hypotheses                     |    |
|  | Modified to absorb anomalies             |    |
|  | Where actual science happens             |    |
|  | Example: initial conditions, instrument  |    |
|  | error theories, existence of new planets |    |
|  +-----------------------------------------+    |
|                                                  |
|  HEURISTICS                                      |
|  Positive: what to do next (research agenda)    |
|  Negative: what to avoid (don't touch hard core)|
+--------------------------------------------------+
```

### Progressive vs Degenerating Research Programmes

| Type | Description | Example |
|------|-------------|---------|
| **Progressive** | Auxiliary modifications predict new facts that are then confirmed | Einstein adding GR corrections to Newtonian mechanics |
| **Degenerating** | Auxiliary modifications are ad hoc — only explain known anomalies, predict nothing new | Phlogiston theory adding "negative weight" to explain calcination |

A research programme should be **abandoned** when it becomes chronically degenerating — but Lakatos (unlike Popper) allows considerable latitude. Newton's programme was "degenerating" on several fronts for decades before being vindicated.

**Key Lakatos insight**: scientists are right to cling to a hard core for a long time. The history of science shows that premature abandonment of a programme (like Wegener's continental drift, initially dismissed) can be a mistake.

## Layer 4: Feyerabend — Against Method

### The Strongest Critique

Feyerabend's *Against Method* (1975) argued that no single methodological rule has not been violated by successful science at some point.

```
Feyerabend's Galileo argument:

  Orthodox methodology of Galileo's time:
  1. Theory must be consistent with accepted observations
  2. Theory must match current empirical evidence

  Galileo's violations:
  - Copernican theory contradicted tower-drop argument
    (if Earth rotates, a dropped stone should land west)
  - Galileo's telescopic observations were widely doubted
    (instruments distort, why trust glass?)
  - He used rhetoric, propaganda, and aesthetic arguments
  - He violated methodological consistency with known physics

  Result: a methodological "outlaw" was RIGHT.

  Therefore: "anything goes" — not nihilism but pluralism.
  No algorithm for science. Proliferate theories freely.
```

### What Feyerabend Was NOT Saying

- He was not saying all theories are equally valid
- He was not saying science should abandon standards
- He was saying: **rigid methodological rules stifle science**; pluralism and tolerance of "deviant" research is productive

**"Science is an essentially anarchistic enterprise: theoretical anarchism is more humanitarian and more likely to encourage progress than its law-and-order alternatives."** — Feyerabend

## Layer 5: Mertonian Norms — Science as Institution

### CUDOS Norms (Merton 1942)

Robert Merton: science is a social institution maintained by norms. Scientists are socialized to follow these norms; violation is sanctioned by the community.

```
+------------------------------------------------------------------+
|                    MERTONIAN NORMS (CUDOS)                        |
+------------------------------------------------------------------+
|                                                                   |
|  C — COMMUNALISM                                                  |
|  Scientific knowledge belongs to the community, not individuals   |
|  Publication is obligation; secrecy is norm violation             |
|  Patent vs publication tension is norm conflict                   |
|                                                                   |
|  U — UNIVERSALISM                                                 |
|  Truth claims evaluated by pre-established impersonal criteria    |
|  Neither nationality nor religion nor race determines validity    |
|  "Nazi physics" and "Lysenkoist genetics" are norm violations     |
|                                                                   |
|  D — DISINTERESTEDNESS                                            |
|  Scientists act for knowledge, not personal gain                  |
|  Fraud is norm violation (not just epistemically bad)             |
|  Modern: Industry-funded research creates disinterest problem     |
|                                                                   |
|  O — ORGANIZED SKEPTICISM                                         |
|  Systematic critical scrutiny before acceptance                   |
|  Peer review as institutionalized skepticism                      |
|  No ideas immune from critique (contrast dogma)                   |
|                                                                   |
+------------------------------------------------------------------+
```

### Counter-Norms (Mitroff 1974)

Merton's critics noted scientists actually exhibit counter-norms:

| Mertonian Norm | Counter-Norm | Reality |
|----------------|-------------|---------|
| Communalism | Proprietary research | Patents, trade secrets, pre-publication secrecy |
| Universalism | Particularism | Citations favor famous labs, prestigious journals |
| Disinterestedness | Self-interestedness | Priority disputes, career incentives |
| Organized Skepticism | Organized Dogmatism | Resistance to revolutionary ideas |

Merton: norms are **institutional ideals**, not behavioral descriptions. The tension between norms and counter-norms is itself sociologically productive.

## Layer 6: Sociology of Scientific Knowledge (SSK)

### The Strong Programme (Bloor 1976)

David Bloor's Edinburgh School moved sociology of science from studying *institutions* to studying *beliefs*. The Strong Programme has four tenets:

```
+--------------------------------------------------+
|          THE STRONG PROGRAMME                     |
+--------------------------------------------------+
|                                                  |
|  1. CAUSALITY                                    |
|  Social causes explain scientific beliefs        |
|  (not just sociology of error — sociology of    |
|  correct science too)                            |
|                                                  |
|  2. IMPARTIALITY                                 |
|  Same type of explanation for true AND false     |
|  beliefs. Don't just explain error socially      |
|  and truth by "the world itself."                |
|                                                  |
|  3. SYMMETRY                                     |
|  Same causes explain success and failure.        |
|  Don't invoke rationality for success and        |
|  irrationality for failure.                      |
|                                                  |
|  4. REFLEXIVITY                                  |
|  These explanations apply to sociology itself.  |
|                                                  |
+--------------------------------------------------+
```

### Controversy Studies

The Edinburgh school focused on **controversies** as natural experiments: when scientists disagree, social factors become visible. Key case studies:

| Controversy | SSK Analysis |
|-------------|-------------|
| Phlogiston vs oxygen (1770s-1790s) | National styles, Lavoisier's institutional power |
| Spontaneous generation debate (1859-1880) | Political stakes of materialism |
| Cold fusion (1989) | Laboratory culture, career pressure, measurement theory |
| Einstein vs Bohr | Philosophical commitments shaped interpretation |

**Latour and Woolgar** (*Laboratory Life*, 1979): ethnographic study of a Salk Institute lab. Scientists construct facts through negotiation — inscriptions, literary technologies, credibility cycles. "Serotonin" went from hypothesis to fact through social process, not revelation.

### The Science Wars (1990s)

SSK's strong claims provoked backlash:

```
SSK (constructivism):                   Scientists (realism):
"Gravity is socially constructed"  <--> "Gravity works whether
meaning gravity AS A CONCEPT and         you believe in it or
its acceptance is socially shaped        not. Try jumping off
                                         a building."

Sokal Affair (1996):
Alan Sokal submitted deliberate nonsense
"Transgressing the Boundaries: Towards a
Transformative Hermeneutics of Quantum Gravity"
to Social Text — accepted.
Demonstrated: humanist science studies had
lost quality control.
```

## Layer 7: The Demarcation Problem

### No Clean Answer

What distinguishes science from pseudo-science? Every proposed criterion has been defeated.

```
+------------------------------------------------------------------+
|                  DEMARCATION CRITERIA                             |
+------------------------------------------------------------------+
|                                                                   |
|  VERIFICATIONISM (Logical Positivists)                            |
|  Statement is scientific if verifiable in principle               |
|  Problem: universal statements not verifiable; theory-laden       |
|  observation; mathematical truth not empirically verifiable       |
|                                                                   |
|  FALSIFICATIONISM (Popper)                                        |
|  Theory is scientific if falsifiable in principle                 |
|  Problem: Duhem-Quine; string theory; evolutionary psychology;    |
|  even astrology makes some falsifiable predictions                |
|                                                                   |
|  RESEARCH PROGRAMME PROGRESS (Lakatos)                            |
|  Science has progressive research programmes                      |
|  Problem: who decides when a programme is "degenerating"?         |
|  Creationism can always add auxiliary hypotheses                  |
|                                                                   |
|  CONSENSUS (sociological)                                         |
|  Science = what scientists agree on                               |
|  Problem: scientific consensus has been wrong;                    |
|  circular (defines science by who is a scientist)                 |
|                                                                   |
+------------------------------------------------------------------+
            |
            v
  Conclusion: demarcation is a family-resemblance concept.
  No single criterion works. Science has characteristic
  features (predictive success, mathematical modeling,
  reproducibility, peer review, anomaly tracking) none
  of which are individually necessary.
```

### Practical Demarcation: The Courts

U.S. courts have had to decide what is science (Daubert standard, Kitzmiller v. Dover 2005). Judge Jones in Kitzmiller: Intelligent Design fails as science because:
1. Not falsifiable
2. Invokes supernatural causation
3. Relies on "God of the gaps" argumentation
4. Rejected by scientific community

This practical demarcation is imperfect but workable.

## Decision Cheat Sheet

| You want to understand... | Use this framework | Key insight |
|---------------------------|-------------------|-------------|
| Why scientists reject new theories initially | Kuhn: paradigm inertia | Normal science suppresses anomalies; this is adaptive |
| Whether a theory is "scientific" | Popper: falsifiability as rough guide | Plus Duhem-Quine caveat: falsification is never clean |
| Why a research program was abandoned | Lakatos: progressive vs degenerating | Did later modifications predict new facts? |
| Why science produces different results in different countries | SSK: social factors | Merton's universalism is a norm, not a guarantee |
| Whether "Big Science" changes the epistemic process | Merton + institutionalism | Yes: funding sources affect disinterestedness |
| Why scientific consensus can be wrong | All frameworks agree | Consensus ≠ truth; Bayesian updating required |

## Common Confusion Points

**Kuhn's "incommensurability" does not mean there is no progress.** Kuhn always maintained science makes genuine progress. Incommensurability means paradigms can't be compared in a paradigm-neutral language — not that they're equally good.

**Lakatos's "hard core" is not dogma in a pejorative sense.** The hard core is a methodological decision — treating certain assumptions as non-negotiable during a research phase. This is rational, not irrational.

**Popper's "corroboration" is not a form of inductive confirmation.** A corroborated theory has survived tests but this does NOT mean it is more likely to be true. Popper explicitly rejects probability of truth.

**SSK's "social construction" does not mean reality is invented.** Even Latour later (in *Pandora's Hope*) clarified: "constructed" facts are still real. Social process is the mechanism by which facts get established — the fact itself is not thereby unreal.

**Mertonian norms describe an ideal, not behavior.** Fraud, priority disputes, particularism, and secrecy all exist in science. Norms are violated regularly. The norms' existence is evidenced by the fact that violations are criticized.

**Feyerabend's "anything goes" is prescriptive for methodology, not epistemology.** He means: no rigid rule should be used to exclude research programs in advance. He doesn't mean all beliefs are equally supported by evidence.

## Bridges — Science Studies as Systems Thinking

The frameworks in this module have direct structural analogues in software engineering and organizational dynamics.

**Kuhnian paradigms → architectural paradigms**
Normal science (puzzle-solving within an accepted framework) maps precisely to feature development within an established architecture. The anomalies that accumulate before a paradigm shift are exactly the "tech debt warnings" that precede a platform rewrite. The paradigm shift itself — messy, contested, often generational — mirrors the mainframe-to-client-server, client-server-to-cloud, monolith-to-microservices transitions. Each involved: resistance from practitioners with sunk investment, incommensurability arguments ("you can't compare the old and new directly"), and eventual dominance of the new regime once network effects tipped.

**Lakatos's research programmes → product architecture lifecycles**
A progressive research programme generates novel predictions confirmed by experiment — equivalent to an architecture that keeps unlocking new capabilities with low friction. A degenerating programme piles up post-hoc fixes to protect its core assumptions — equivalent to a legacy architecture accumulating adapters, shims, and workarounds. Lakatos's insight: you can't evaluate a research programme at a single point in time; you need the *trajectory*. Same for architectural decisions.

**Mertonian norms → open-source community norms**
Communalism (results shared freely) is the open-source default. Universalism (judge work on merit, not author identity) is the PR review ideal — imperfectly realized in both science and software. Disinterestedness (no personal stake in outcome) is violated in both domains by career incentives and NIH syndrome. Organized Skepticism is code review, adversarial testing, and reproducibility requirements. Open source is a Mertonian institution embedded in a commercial context, with the resulting tensions.

**Feyerabend's "anything goes" → exploratory research spikes**
Feyerabend's prescriptive claim is not that all approaches are equal — it's that methodological rigidity blocks valid innovation. The equivalent in engineering: premature process standardization (mandating a testing framework before the problem space is understood) can kill the productive exploration that reveals what the right process should be. The caveat in both domains: "anything goes" is for exploration phases, not production gates.

**SSK's social construction → consensus mechanisms in distributed systems**
The Strong Programme's claim — that scientific consensus is partly a social achievement, not pure logic — maps to the insight that distributed consensus is a protocol, not a logical necessity. The fact that two reasonable people examining the same evidence can reach different conclusions (incommensurability) is a feature of knowledge systems that lack a canonical arbiter. How you design the arbitration protocol (peer review, Paxos, BFT) determines what the system converges on.
