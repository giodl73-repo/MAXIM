# Phonetics & Phonology — Sound Systems

## The Big Picture

Two related but distinct disciplines:

```
+----------------------------------------------------------+
|  PHONETICS                    PHONOLOGY                  |
|  (physical facts of sound)    (abstract sound systems)  |
|                                                          |
|  Articulatory:                 Phonemes:                 |
|  where/how vocal tract         the distinctive units     |
|  produces sounds               (not physical sounds)     |
|                                                          |
|  Acoustic:                     Allophony:                |
|  waveforms, spectrograms,      predictable variants      |
|  formants, F0                  of a phoneme              |
|                                                          |
|  Auditory:                     Phonological rules:       |
|  how cochlea/auditory          how phonemes combine,     |
|  cortex processes sound        change in context         |
|                                                          |
|  Studies physical signal       Studies the grammar of    |
|  and its production            sound — the abstract      |
|                                system speakers know      |
+----------------------------------------------------------+
```

## Part I: The IPA — International Phonetic Alphabet

Every human language sound maps to exactly one IPA symbol. The IPA is organized by **place** and **manner** of articulation.

### Consonant Chart: Place × Manner

```
                 BILABIAL  LABIODENT  DENTAL  ALVEOLAR  POST-ALV  RETROFLEX  PALATAL  VELAR  UVULAR  PHARYNGEAL  GLOTTAL
PLOSIVE          p  b                         t  d                  ʈ  ɖ      c  ɟ     k  g   q  ɢ              ʔ
NASAL            m          ɱ                 n                     ɳ         ɲ        ŋ        ɴ
TRILL            B                            r                                                 ʀ
TAP/FLAP                                      ɾ                     ɽ
FRICATIVE        ɸ  β       f  v     θ  ð     s  z      ʃ  ʒ        ʂ  ʐ     ç  ʝ     x  ɣ   χ  ʁ   ħ  ʕ       h  ɦ
LAT. FRICATIVE                                ɬ  ɮ
APPROXIMANT                  ʋ                ɹ                     ɻ         j        ɰ
LAT. APPROXIMANT                              l                     ɭ         ʎ        ʟ
AFFRICATE                               ts dz tʃ dʒ (common pairs — not official IPA row)
```

**Note on the table**: Left symbol = voiceless, right = voiced. Blank cells = articulatorily impossible or not used in any known language.

### Place of Articulation — Where

| Place | Active articulator | Passive | Examples |
|-------|--------------------|---------|----------|
| **Bilabial** | Both lips | — | [p b m] |
| **Labiodental** | Lower lip | Upper teeth | [f v] |
| **Dental** | Tongue tip | Upper teeth | [θ ð] (think, the) |
| **Alveolar** | Tongue tip/blade | Alveolar ridge | [t d n s z l r] |
| **Post-alveolar** | Tongue blade | Behind alveolar ridge | [ʃ ʒ] (ship, measure) |
| **Retroflex** | Tongue tip curled back | — | [ʈ ɖ ɭ] (Hindi, Tamil) |
| **Palatal** | Tongue body | Hard palate | [j c ɟ ɲ] |
| **Velar** | Tongue dorsum | Soft palate (velum) | [k g ŋ] |
| **Uvular** | Tongue dorsum | Uvula | [q ʁ ʀ] (French r, Arabic q) |
| **Pharyngeal** | Tongue root | Pharynx wall | [ħ ʕ] (Arabic ح ع) |
| **Glottal** | Glottis itself | — | [h ʔ] (h, glottal stop) |

### Manner of Articulation — How

| Manner | Definition | Example |
|--------|------------|---------|
| **Plosive / Stop** | Complete closure, then release | [p t k b d g ʔ] |
| **Nasal** | Complete oral closure, velum lowered — air through nose | [m n ŋ] |
| **Fricative** | Narrow constriction, turbulent airflow | [f v s z ʃ ʒ h] |
| **Affricate** | Stop released as fricative at same/adjacent place | [tʃ dʒ ts] |
| **Approximant** | Constriction narrower than vowel, no turbulence | [j w ɹ ʋ] |
| **Lateral** | Partial closure, air flows around sides of tongue | [l ɭ ʎ] |
| **Trill** | Articulator vibrates from airstream | [r ʀ] (Spanish r, French R) |
| **Tap/Flap** | Single rapid tap of articulator | [ɾ] (Spanish r single, American "butter") |

### Voicing

**Voiced**: vocal folds vibrate during production — [b d g z v ʒ]
**Voiceless**: vocal folds open, no vibration — [p t k s f ʃ]

You can feel this: hold fingers on larynx, say "sss" vs "zzz".

## Part II: Vowels — The Quadrilateral

Vowels are classified by tongue height, tongue backness, and lip rounding:

```
VOWEL QUADRILATERAL (IPA)
                FRONT         CENTRAL      BACK
                unrd | rnd    unrd | rnd   unrd | rnd

CLOSE (HIGH)     i · y         ɨ · ʉ        ɯ · u
CLOSE-MID        e · ø         ɘ · ɵ        ɤ · o
MID              e̞ · ø̞        ə            ʌ · o̞
OPEN-MID         ɛ · œ         ɜ · ɞ        ʌ · ɔ
NEAR-OPEN        æ             ɐ
OPEN             a · œ̞        ä            ɑ · ɒ
```

**Key vowels to internalize:**

| Symbol | Description | Example language |
|--------|-------------|------------------|
| [i] | Close front unrounded | "machine", French "si" |
| [u] | Close back rounded | "boot" |
| [e] | Close-mid front unrounded | French "été" |
| [o] | Close-mid back rounded | Spanish "no" |
| [ɛ] | Open-mid front unrounded | "bed", French "être" |
| [ɔ] | Open-mid back rounded | "caught" (non-rhotic) |
| [æ] | Near-open front unrounded | "cat" |
| [a] | Open front unrounded | Spanish "casa" |
| [ɑ] | Open back unrounded | "father" |
| [ə] | Mid central (schwa) | unstressed "the", "about" |
| [ɪ] | Near-close near-front unrounded | "bit" |
| [ʊ] | Near-close near-back rounded | "book" |
| [ʌ] | Open-mid back unrounded | "but" (General American) |
| [y] | Close front rounded | French "tu", German "über" |
| [ø] | Close-mid front rounded | French "feu", German "schön" |

### Diphthongs

Vowels with movement — tongue moves from one position to another:
- [aɪ] — "price"
- [aʊ] — "mouth"
- [ɔɪ] — "choice"
- [eɪ] — "face"
- [oʊ] — "goat" (non-rhotic)

## Part III: Tone and Suprasegmentals

### Tone

Tone languages use pitch phonemically — same consonants + vowels, different pitch = different lexical item.

```
MANDARIN 4-TONE SYSTEM (+ neutral):
Tone 1: ˥˥ high level    — mā (mother)    ¯
Tone 2: ˧˥ rising        — má (hemp)      /
Tone 3: ˨˩˦ dipping      — mǎ (horse)    ∨
Tone 4: ˥˩ falling       — mà (scold)    \
Tone 0: neutral/atonal   — ma (question)  unstressed
```

**Register tone** (high vs. low — Bantu languages, ~many in Africa): fixed pitch level per syllable.

**Contour tone** (Mandarin, Vietnamese, Cantonese): pitch trajectory across syllable.

**Tonal languages:** ~70% of the world's languages have tone. Indo-European languages (except Latvian, Lithuanian, Serbo-Croatian pitch accent) mostly don't. This is typologically unusual — European perspective is the outlier.

### Suprasegmentals

Features that extend over more than one segment (syllable, word, phrase):

| Feature | Description | Symbol | Example |
|---------|-------------|--------|---------|
| **Stress** | Prominence by intensity + duration + pitch | ˈ (primary) ˌ (secondary) | ˈpermit (noun) vs. perˈmit (verb) |
| **Tone** | Phonemic pitch | ˥˧˩ numerals or ¯ / ∨ \ diacritics | Mandarin above |
| **Length** | Vowel or consonant duration | ː | Italian: "nono" vs "nonno" |
| **Intonation** | Pitch contour over phrase/sentence | ↗ ↘ (simplified) | Question vs. statement in English |
| **Rhythm** | Stress-timed vs. syllable-timed vs. mora-timed | — | English stress-timed; Spanish syllable-timed; Japanese mora-timed |

## Part IV: Phoneme vs. Allophone

This is the central theoretical distinction in phonology.

```
PHONEME                              ALLOPHONE
+---------------------------+       +---------------------------+
| Abstract mental unit       |       | Physical realization       |
| /p/ — the phoneme          |       | [pʰ] in "pin" (aspirated)  |
| (underlying form)          |       | [p] in "spin" (unaspirated)|
| Contrastive                |       | Predictable from context   |
| Minimal pairs reveal them  |       | NOT contrastive            |
| Written /slashes/          |       | Written [brackets]         |
+---------------------------+       +---------------------------+

/p/ is realized as [pʰ] in syllable-initial position (before stressed vowel)
/p/ is realized as [p] after /s/
/p/ is realized as [p̚] in syllable-final position (unreleased)

The three are in COMPLEMENTARY DISTRIBUTION — mutually exclusive environments.
```

**How to identify phonemes: Minimal pairs**

Two words differing in exactly one sound, in the same environment, with different meanings:
- /p/ vs. /b/: "pin" vs. "bin" — both phonemes
- [pʰ] vs. [p]: no minimal pair — not distinct phonemes in English (but in Hindi they are: [pʰal] "fruit" ≠ [pal] "moment")

**Contrast is language-specific.** The phoneme inventory varies:

| Language | Phoneme inventory size | Notes |
|----------|----------------------|-------|
| Hawaiian | 13 | Among smallest known |
| English (RP) | ~44 | Depends on dialect |
| Mandarin | ~25 consonants + 5 vowels | But 4 tones multiply contrasts |
| !Xóõ (Khoisan) | ~100+ | With clicks, ejectives, implosives |
| Georgian | 33 consonants | Rich obstruent system |

## Part V: Syllable Structure

Syllables have internal structure:

```
         SYLLABLE σ
           /     \
        ONSET   RHYME
                /    \
           NUCLEUS  CODA

Example: "strong"  /stɹɒŋ/
  ONSET = /stɹ/ (complex onset — 3 consonants)
  NUCLEUS = /ɒ/
  CODA = /ŋ/
```

### Sonority Hierarchy

The nucleus is always the **most sonorous** element. Sonority decreases outward from the nucleus:

```
MOST SONOROUS ←——————————————→ LEAST SONOROUS
   Vowels  Glides  Liquids  Nasals  Fricatives  Stops
   a e i u   j w     l r      m n     f v s z    p t k

"string" /stɹɪŋ/: s=fricative, t=stop (going up), r=liquid (going up), ɪ=vowel (peak), ŋ=nasal (going down) ✓
"*tnɪ" — would violate sonority profile (stop before nasal in onset = going up too fast then down)
```

**Maximal Onset Principle:** When placing a consonant between two vowels, assign it to the onset of the second syllable if possible:
- "happy" = hæ.pi (not hæp.i) — [p] onset of second syllable

### Cross-linguistic syllable typology

| Pattern | Example languages | Predominance |
|---------|------------------|-------------|
| CV (mandatory onset, no coda) | Japanese, Polynesian | Most common |
| (C)V(C) | Spanish, Arabic | Very common |
| CVC | Tibetan, Mandarin | Common |
| (C)(C)(C)V(C)(C)(C)(C) | English, Polish | More complex |

English has one of the more complex syllable structures in the world.

## Part VI: Phonological Rules

Phonological rules operate on underlying representations (URs) to produce surface representations (SRs).

### Rule Notation

Standard SPE (Sound Pattern of English, Chomsky & Halle 1968) notation:

```
A → B / C __ D

"A becomes B in the environment: C precedes, D follows"

Example: Aspiration rule in English
   p, t, k → pʰ, tʰ, kʰ / ___ V   (before vowel, syllable-initial)

   More precisely:
   [-son, -cont] → [+spread glottis] / ___ V (in syllable-initial position)
```

### Key Phonological Processes

**Assimilation** — segment becomes more like neighboring segment:

| Type | Description | Example |
|------|-------------|---------|
| **Voicing assimilation** | Obstruent assimilates in voicing | "have to" → [hæftə] (v→f before voiceless t) |
| **Nasal assimilation** | Nasal assimilates place of following stop | "impossible" — [m] before bilabial p; "incredible" — [ŋ] before velar k |
| **Vowel nasalization** | Vowel nasalized before nasal consonant | "man" — [mæ̃n] |
| **Palatalization** | Consonant palatalizes before front vowel | "did you" → [dɪdʒu] |

**Dissimilation** — segments become less alike:
- Latin: *ferre* (to carry), Greek: *φέρω* — but *referō* (I carry back) not *feferō* (dissimilation of two r's)

**Deletion / Elision:**
- Schwa deletion: "every" /ɛvərɪ/ → [ɛvrɪ] in fast speech
- Syncope: "chocolate" /ˈtʃɒklɪt/ not /ˈtʃɒkəlɪt/

**Epenthesis** — insertion:
- "warmth" → [wɔrmθ] — but in rapid speech, epenthetic [p] between [m] and [θ]: [wɔrmpθ]
- Arabic loanwords: "film" → /filim/ (insert vowel to break consonant cluster)

**Metathesis** — reordering:
- "ask" historically /aks/ → /æsk/ (Modern English reversed the order — then later reversed back in some dialects)

## Part VII: Underlying vs. Surface Representation — Alternations

The key motivation for distinguishing UR and SR: morphologically related forms show alternations that reveal the underlying form.

### English Plural: {-Z}

```
UNDERLYING REPRESENTATION: /-z/ (one morpheme)

Surface forms:
  [s]  after voiceless consonants: cats [kæts], backs [bæks]
  [z]  after voiced sounds: dogs [dɒgz], cars [kɑːrz], bees [biːz]
  [ɪz] after sibilants: buses [bʌsɪz], roses [roʊzɪz], judges [dʒʌdʒɪz]

Rules:
  1. Insert [ɪ] between sibilant and plural morpheme
  2. Assimilate plural morpheme voicing to preceding consonant
  3. Elsewhere: [z]

The underlying form is /z/. The [s] and [ɪz] variants are ALLOMORPHS
derived by phonological rules.
```

### English Past Tense: {-D}

Same pattern:
- /t/ after voiceless: walked [wɔːkt], helped [hɛlpt]
- /d/ after voiced: begged [bɛgd], loved [lʌvd]
- /ɪd/ after /t d/: wanted [wɒntɪd], landed [lænd ɪd]

UR = /d/. Three allomorphs by phonological rule.

## Part VIII: Phonological Features

Phonemes are not primitives — they're **bundles of distinctive features**. Features are binary [+ / -]:

```
FEATURE GEOMETRY FOR CONSONANTS:
         [± voice]
         [± sonorant]
         [± consonantal]
         PLACE (bilabial, coronal, dorsal, laryngeal)
         MANNER (plosive, fricative, nasal, etc.)

/p/: [+cons, -son, -voice, bilabial, plosive]
/b/: [+cons, -son, +voice, bilabial, plosive]
/m/: [+cons, +son, +voice, bilabial, nasal]

Natural classes are defined by shared features:
  {p, b, m} = bilabials = [+cons, bilabial]
  {p, t, k, f, s, ʃ} = voiceless obstruents = [-son, -voice]
  {m, n, ŋ} = nasals = [+son, +nasal, -lateral]
```

Rules that affect natural classes are more "natural" (typologically frequent) than arbitrary sets. Nasal place assimilation affects all nasals together — not /n/ and /k/ but not /m/.

## Decision Cheat Sheet — Phonetics vs. Phonology

| Question | Domain | Key concept |
|----------|--------|-------------|
| How are [t] and [tʰ] physically different? | Phonetics | VOT (voice onset time) |
| Are [t] and [tʰ] different phonemes in English? | Phonology | Complementary distribution |
| Why does the plural sound different in "cats" vs "dogs"? | Phonology | Allomorphy via phonological rule |
| What's the sonority of [ŋ]? | Phonology | Nasal — between fricatives and vowels |
| Is Mandarin tone a prosodic or segmental phenomenon? | Phonology | Tonal — suprasegmental but lexically contrastive |
| What muscles produce [k]? | Phonetics | Dorsal tongue, velar place |
| Why does "spin" have no aspiration on [p]? | Phonology | Aspiration rule: voiceless stop after /s/ → [-spread glottis] |
| What distinguishes "beat" from "bit"? | Phonetics + Phonology | Vowel quality ([iː] vs [ɪ]) + length |

## Common Confusion Points

**[Brackets] vs. /slashes/ vs. ⟨angle brackets⟩:**
- [t] = phonetic/surface form (physical)
- /t/ = phonemic/underlying form (abstract)
- ⟨t⟩ = orthographic (spelling) — don't confuse with phonetics

**"Letter" vs. "sound":**
English orthography is not phonemic. "ph" = /f/, "c" = /s/ or /k/, "ough" = /ʌf/ (tough), /oʊ/ (though), /uː/ (through), /ɒf/ (cough). Always use IPA.

**Aspiration is not voicing:**
[pʰ] and [p] are both voiceless. Aspiration = [+spread glottis]. Hindi contrasts all four: [p] [pʰ] [b] [bʱ] (voiceless/voiced × aspirated/plain).

**Phoneme count is theory-dependent:**
English has anywhere from 35–46 phonemes depending on dialect and analysis. "How many phonemes does English have?" has no single answer.

**Tone is not intonation:**
Tone = phonemic, lexically contrastive pitch. Intonation = phrasal pitch contour conveying discourse meaning (question vs. statement). Different levels of abstraction.

**Sonority is not loudness:**
Sonority is an abstract scale of "openness" of the vocal tract. It correlates with loudness and salience but is a theoretical construct, not a physical measurement.
