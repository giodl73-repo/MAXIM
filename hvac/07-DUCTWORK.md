# 07 — Ductwork

## Manual D, Static Pressure, Sheet Metal vs Flex, Sealing, Zoning

> **STUB** — outline only, content to be authored

**Planned coverage:**
- **Why ductwork matters**: even a perfectly sized, efficient HVAC unit delivers poor results through leaky, undersized, or improperly designed ductwork; duct leakage studies show typical existing US residential ductwork leaks 20-30% of conditioned air into unconditioned attic/crawlspace (energy directly wasted); duct design failure is extremely common
- **Manual D** (ACCA): duct design standard; inputs are Manual J room loads; determines supply air volume (CFM) per room to deliver calculated load; sizes supply and return ducts for proper airflow at acceptable pressure drop; friction rate calculation: available static pressure (fan capacity) minus fittings pressure drop / total duct length = allowable friction rate per 100 feet; determines duct diameter
- **Static pressure**: air pressure in duct system; supply side = positive pressure (fan pushing); return side = negative pressure (fan pulling); total external static pressure (TESP) = what the fan works against; rating on equipment spec (e.g., 0.5" w.c. = inches water column); TESP = sum of pressure drops through filter, coil, supply ductwork, return ductwork; exceeding equipment-rated static pressure = reduced airflow → reduced capacity and efficiency; the "dirty filter" problem writ large
- **Equivalent length**: fittings (elbows, tees, transitions) create turbulent pressure drop converted to equivalent straight-duct length; sharp 90° elbow = 50-100 feet equivalent length; sweep elbow = 30-50 feet; too many fittings = excessive pressure drop; duct routing should minimize turns
- **Duct types**:
  - Sheet metal (galvanized steel): most durable, lowest pressure drop (smooth interior), no compression degradation; rectangular or round; requires sheet metal fabrication; seams sealed with mastic or metal (foil-backed) tape
  - Flex duct (insulated flexible duct): wire helix + plastic liner + insulation (R-6 or R-8) + vapor barrier jacket; easy to install around obstacles; lower cost; CRITICAL: must be fully extended (no compression, no sag, no tight bends) — compressed or kinked flex duct dramatically increases pressure drop and reduces airflow; "2 feet of slack" installation method = code violation; very common source of duct problems; 5% compression doubles pressure drop
  - Duct board (fiberglass duct board): rigid fiberglass panels fabricated into ductwork; good insulation value; pressure concerns at higher velocities; requires sealing; mostly commercial/commercial residential
- **Duct sealing**: leakage pathways: longitudinal seams in sheet metal, duct connection to boot (register box), return plenum leaks, flex duct tee connections; mastic sealant (gray paste — brush-applied, permanent, best for rigid duct seams and connections); metal tape (aluminum foil tape, UL181-rated — NOT "duct tape" which fails in heat/cold); Aeroseal (spray-from-inside polymer sealing — inflates equipment, blows particles that seal leaks from inside → very effective for inaccessible leaks in attics/crawlspaces)
- **Duct leakage testing**: Duct Blaster (similar to blower door but for duct system); measures CFM25 (leakage at 25 Pa pressure); total leakage vs leakage to outside (only outside leakage wastes energy); IECC requires ≤4 CFM25/100 sq ft for new construction; Energy Star requires ≤3 CFM25/100 sq ft
- **Supply register vs return grille placement**: supply air should deliver heating/cooling where needed (perimeter, under windows historically; now with better windows, ceiling or high-wall OK); return grilles must have free path from supply to return; transfer grilles or undercut doors allow air to return from closed rooms; central return only (common in old systems) creates pressure imbalance and duct leakage issues; each room should have return or free path
- **Zoning with multiple equipment or dampers**: multiple pieces of equipment each with zone control (mini-split model — each unit = one zone); motorized dampers in duct system redirect airflow per thermostat zone; bypass damper or variable-speed air handler required with damper zoning to manage static pressure when some zones close; improper zoning = elevated static pressure = reduced life and efficiency
