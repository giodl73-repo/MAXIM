# Measure Theory — Complete Reference

## The Big Picture

Measure theory is the rigorous foundation for integration, probability, and functional analysis. It resolves the pathologies that plagued 19th-century analysis: non-measurable sets, exchanging limits and integrals, and the proper definition of "almost everywhere." If you know Riemann integration from calculus, this guide will show you exactly where it fails and why Lebesgue's construction fixes it.

## 1. σ-Algebras and Measurable Spaces

### Why σ-Algebras?

The Banach-Tarski paradox and Vitali's construction show that not every subset of ℝ can be assigned a consistent "length." We restrict to **measurable** sets — those to which we can assign measure consistently.

```
Problems with naive measure assignment to ALL subsets of ℝ:
  Vitali set (using Axiom of Choice): subsets of [0,1] that cannot be measured.
  Banach-Tarski: decompose sphere into 5 pieces, reassemble into two spheres.
  → We cannot extend Lebesgue measure to all subsets of ℝ.
  → We must specify which sets are measurable.
```

### σ-Algebra Definition

A **σ-algebra** ℱ on a set Ω is a collection of subsets satisfying:
```
(S1) Ω ∈ ℱ
(S2) A ∈ ℱ → Aᶜ ∈ ℱ                           [closed under complement]
(S3) A₁, A₂, ... ∈ ℱ → ∪ₙ Aₙ ∈ ℱ             [closed under countable unions]

Closure under countable intersections follows from (S2)+(S3): ∩Aₙ = (∪Aₙᶜ)ᶜ
```

The pair (Ω, ℱ) is a **measurable space**.

### The Borel σ-Algebra

```
𝒝(ℝ) = σ-algebra generated by all open sets of ℝ

Equivalent generating sets:
  - All open sets
  - All closed sets (by complement)
  - All open intervals (a,b)
  - All half-open intervals (-∞, a]
  - All half-open intervals (a, ∞)

Borel sets: all sets in 𝒝(ℝ). Contains all "reasonable" subsets of ℝ.
Includes: every open, closed, Gδ (countable intersections of opens), Fσ sets.
Does NOT include Vitali sets (not provably Borel from ZFC).

Borel hierarchy:
  Σ⁰₁ = open sets
  Π⁰₁ = closed sets
  Σ⁰₂ = Fσ sets (countable unions of closed)
  Π⁰₂ = Gδ sets (countable intersections of open)
  ... → 𝒝(ℝ) is the union of all levels
```

### Generated σ-Algebras

For any collection 𝒞 of subsets of Ω:
```
σ(𝒞) = smallest σ-algebra containing 𝒞
      = intersection of all σ-algebras containing 𝒞

This is well-defined because ℙ(Ω) is a σ-algebra, and
arbitrary intersection of σ-algebras is a σ-algebra.
```

**Product σ-algebra**: For measurable spaces (Ω₁,ℱ₁) and (Ω₂,ℱ₂):
```
ℱ₁ ⊗ ℱ₂ = σ({A×B : A∈ℱ₁, B∈ℱ₂})
```

## 2. Measures

### Measure Definition

A **measure** μ on (Ω, ℱ) is a function μ: ℱ → [0,∞] satisfying:
```
(M1) μ(∅) = 0
(M2) Countable additivity: for disjoint A₁, A₂, ... ∈ ℱ:
     μ(∪ₙ Aₙ) = Σₙ μ(Aₙ)
```

The triple (Ω, ℱ, μ) is a **measure space**.

**Immediate consequences**:
```
Monotonicity:   A ⊆ B → μ(A) ≤ μ(B)
Subadditivity:  μ(∪ₙ Aₙ) ≤ Σₙ μ(Aₙ)  [even without disjointness]
Continuity from below: Aₙ ↑ A → μ(Aₙ) → μ(A)
Continuity from above: Aₙ ↓ A and μ(A₁)<∞ → μ(Aₙ) → μ(A)
  (Finiteness condition needed: μ((0,1/n)) → 0, but μ((n,∞)) = ∞ doesn't → 0)
```

### Types of Measures

**Probability measure**: μ(Ω) = 1. Then μ = P, a probability.

**σ-finite measure**: Ω = ∪ₙ Ωₙ where μ(Ωₙ) < ∞.
- Lebesgue measure on ℝ is σ-finite: ℝ = ∪ₙ [-n,n].
- Required for Fubini's theorem.

**Lebesgue measure on ℝ**: μ((a,b)) = b-a (length), extended via σ-additivity.
- Exists (Carathéodory extension theorem).
- Translation-invariant.
- Assigns measure 0 to all countable sets (including ℚ).
- Assigns positive measure to many uncountable sets (Cantor set has measure 0; fat Cantor sets can have positive measure).

**Dirac measure**: δₓ(A) = 1 if x ∈ A, else 0. Models a point mass at x.

**Counting measure**: μ(A) = |A| (cardinality). On ℕ, gives natural number-counting.

### Null Sets and Completion

A set N is a **null set** if ∃ measurable M ⊇ N with μ(M) = 0.

**Completed measure space**: Add all null sets to ℱ. For Lebesgue measure, the completed σ-algebra is the **Lebesgue σ-algebra** (strictly larger than Borel).

**Almost everywhere (a.e.)** / **almost surely (a.s.)**: Property holds on Ω \ N where μ(N) = 0.

## 3. Measurable Functions

### Definition

f: (Ω,ℱ) → (Λ,𝒢) is **measurable** if:
```
f⁻¹(B) = {ω ∈ Ω : f(ω) ∈ B} ∈ ℱ  for all B ∈ 𝒢

For f: Ω → ℝ: f⁻¹((-∞, a]) ∈ ℱ for all a ∈ ℝ suffices (generates 𝒝(ℝ)).
```

**Measurability-preserving operations**:
```
If f, g measurable then: f+g, fg, |f|, max(f,g), min(f,g) all measurable.
If fₙ measurable then: sup fₙ, inf fₙ, limsup fₙ, liminf fₙ all measurable.
Composition: g measurable, f continuous → f∘g measurable.
```

**Simple functions**: φ = Σᵢ aᵢ 1_{Aᵢ} where Aᵢ ∈ ℱ, aᵢ ∈ ℝ. Finite linear combination of indicators.

**Approximation theorem**: For any measurable f ≥ 0, there exist simple functions φₙ with 0 ≤ φₙ ↑ f pointwise. If f bounded, convergence is uniform.

### Random Variables

A random variable X on probability space (Ω, ℱ, P) is a measurable function X: (Ω,ℱ) → (ℝ, 𝒝(ℝ)).

**Distribution**: P_X = P∘X⁻¹ is a probability measure on (ℝ, 𝒝(ℝ)).
```
P_X(B) = P(X ∈ B) = P(X⁻¹(B))
```

**Independence**: X, Y independent iff σ(X) and σ(Y) are independent σ-algebras (all events in σ(X) independent of all events in σ(Y)).

## 4. Lebesgue Integration

### Construction (Building up in stages)

**Stage 1: Simple functions**:
```
If φ = Σᵢ aᵢ 1_{Aᵢ} ≥ 0 (aᵢ ≥ 0, Aᵢ disjoint):
∫ φ dμ = Σᵢ aᵢ · μ(Aᵢ)
(with convention 0·∞ = 0)
```

**Stage 2: Non-negative measurable functions**:
```
∫ f dμ = sup { ∫ φ dμ : 0 ≤ φ ≤ f, φ simple }
```

**Stage 3: General measurable functions**:
```
f = f⁺ - f⁻ where f⁺ = max(f,0), f⁻ = max(-f,0) ≥ 0.
∫ f dμ = ∫ f⁺ dμ - ∫ f⁻ dμ
Defined (and finite) when ∫ f⁺ dμ < ∞ or ∫ f⁻ dμ < ∞.
f is integrable (f ∈ L¹(μ)) if ∫ |f| dμ < ∞.
```

### Lebesgue vs Riemann

```
Riemann: partition the domain [a,b] into intervals.
  ∫_Riemann f = lim (Σ f(xᵢ*) Δxᵢ) as partition gets fine.
  Fails for: f = 1_ℚ (Dirichlet function). Upper Riemann sum = 1, lower = 0.

Lebesgue: partition the RANGE into intervals.
  ∫_Lebesgue f = lim Σ yᵢ · μ(f⁻¹([yᵢ, yᵢ₊₁)))
  Works for f = 1_ℚ: ∫ 1_ℚ dμ = μ(ℚ) = 0 (countable set has measure 0).

If f is Riemann integrable on [a,b], then f is Lebesgue integrable and integrals agree.
Lebesgue integrable ⊃ Riemann integrable (strictly).
```

**Key insight**: Lebesgue measures sets where f takes certain values. This is more natural for probability (P(a < X < b)) and works for much more irregular functions.

### The Convergence Theorems

These are the main payoff of Lebesgue theory — conditions under which ∫ lim fₙ = lim ∫ fₙ.

**Monotone Convergence Theorem (MCT)**:
```
If 0 ≤ fₙ ↑ f pointwise (fₙ increasing, converging to f):
  ∫ fₙ dμ → ∫ f dμ   (including if both are ∞)

Proof idea: ∫fₙ monotone increasing, bounded above by ∫f. Show lim ≥ ∫f via simple function approx.
```

**Fatou's Lemma**:
```
If fₙ ≥ 0 measurable:
  ∫ liminf fₙ dμ ≤ liminf ∫ fₙ dμ

Inequality can be strict: fₙ = 1_{[n,n+1]} on ℝ with Lebesgue measure.
  ∫ fₙ = 1 for all n, but liminf fₙ = 0, so ∫ liminf fₙ = 0 < 1.
```

**Dominated Convergence Theorem (DCT)** (the workhorse):
```
If fₙ → f pointwise a.e.
AND |fₙ| ≤ g a.e. for some integrable g (∫ g dμ < ∞):
  ∫ fₙ dμ → ∫ f dμ

Also: ||fₙ - f||₁ = ∫ |fₙ - f| dμ → 0   (L¹ convergence)

The dominating function g is crucial — need uniform integrability.
Without domination, limit and integral need not commute.
```

**Uniform Integrability** (generalized DCT condition):
```
{fₙ} uniformly integrable if:
  sup_n ∫_{|fₙ|>M} |fₙ| dμ → 0  as M → ∞.

Vitali Convergence Theorem: fₙ → f in measure + UI ⟺ fₙ → f in L¹.
```

### Fubini-Tonelli Theorem

For product measure spaces:

```
Tonelli (non-negative functions):
  If f ≥ 0 measurable on (X×Y, ℱ⊗𝒢, μ×ν):
  ∫_{X×Y} f d(μ×ν) = ∫_X (∫_Y f(x,y) dν(y)) dμ(x)
                    = ∫_Y (∫_X f(x,y) dμ(x)) dν(y)

Fubini (integrable functions):
  If ∫_{X×Y} |f| d(μ×ν) < ∞:
  Same iterated integral formula holds.

Warning: Without integrability or non-negativity, iterated integrals may disagree.
Example: f(x,y) = (x²-y²)/(x²+y²)². ∫₀¹∫₀¹f dx dy = 1, ∫₀¹∫₀¹f dy dx = -1.
Always check Tonelli/Fubini conditions before swapping integrals.
```

## 5. Lᵖ Spaces

### Definition and Norms

For (Ω, ℱ, μ) and p ∈ [1,∞):
```
Lᵖ(μ) = {f measurable : ∫ |f|ᵖ dμ < ∞}  (equivalence classes: f=g a.e.)

Norm: ||f||_p = (∫ |f|ᵖ dμ)^{1/p}

L∞(μ) = {f measurable : esssup|f| < ∞}
Norm: ||f||_∞ = inf{M : μ(|f| > M) = 0}  [essential supremum]
```

These are norms on equivalence classes (identify f=g a.e.):
- Positive definiteness: ||f||_p = 0 iff f = 0 a.e.
- Homogeneity: ||αf||_p = |α| ||f||_p
- Triangle inequality: Minkowski's inequality: ||f+g||_p ≤ ||f||_p + ||g||_p

### Key Inequalities

**Hölder's inequality** (p and q are conjugate: 1/p + 1/q = 1):
```
∫ |fg| dμ ≤ ||f||_p · ||g||_q

Special case p=q=2: Cauchy-Schwarz:
  ∫ |fg| dμ ≤ (∫ f²)^{1/2} · (∫ g²)^{1/2} = ||f||₂ · ||g||₂

Proof: Young's inequality: ab ≤ aᵖ/p + bᵍ/q for a,b ≥ 0.
Apply pointwise with a = |f|/||f||_p, b = |g|/||g||_q, integrate.
```

**Jensen's inequality**: For convex φ and random variable X:
```
φ(E[X]) ≤ E[φ(X)]
```
Proof: By convexity, ∃ supporting hyperplane φ(x) ≥ φ(μ) + c(x-μ) at μ=E[X]. Integrate.

**Lᵖ inclusion**: If μ(Ω) < ∞: for p ≤ q, Lᵍ(μ) ⊆ Lᵖ(μ).
For μ = counting measure on ℕ: inclusion reverses (ℓᵖ ⊆ ℓᵍ for p ≤ q).

### Completeness (Riesz-Fischer Theorem)

```
Lᵖ(μ) is complete for p ∈ [1,∞]: every Cauchy sequence has a limit in Lᵖ.

Proof sketch (p=1):
  Cauchy {fₙ}: find subsequence with ||f_{n_{k+1}} - f_{n_k}||₁ < 2⁻ᵏ.
  Define g = Σₖ |f_{n_{k+1}} - f_{n_k}|. By MCT, ||g||₁ ≤ 1.
  So g < ∞ a.e. → telescoping series converges a.e.
  Limit function f ∈ L¹; use DCT to show fₙ → f in L¹.
```

L²(μ) is a **Hilbert space** with inner product ⟨f,g⟩ = ∫ fḡ dμ.

### Dense Subspaces

```
In Lᵖ(ℝ, Lebesgue):
  - Continuous compactly supported functions C_c(ℝ) are dense in Lᵖ for p<∞
  - Step functions (simple functions on intervals) are dense
  - Polynomials are dense on bounded intervals (Weierstrass + DCT)
  - C_c∞ (smooth compactly supported) are dense — used in distribution theory

Practical: can approximate any Lᵖ function by smooth functions.
This underlies: Fourier series convergence in L², spectral theory, PDE solutions.
```

## 6. Radon-Nikodym Theorem

### Absolute Continuity

Measure ν is **absolutely continuous** with respect to μ (ν ≪ μ) if:
```
μ(A) = 0 → ν(A) = 0  for all measurable A.
```

Intuition: ν cannot assign positive measure to sets that μ says have zero measure. ν is "dominated" by μ.

**Mutual singularity** (ν ⊥ μ): ∃ disjoint A, B with Ω = A ∪ B, μ(A) = 0, ν(B) = 0.

**Lebesgue decomposition**: Any σ-finite measure ν decomposes uniquely:
```
ν = νₐ_c + ν_s
where νₐ_c ≪ μ (absolutely continuous part)
and ν_s ⊥ μ (singular part)
```

### Radon-Nikodym Theorem

```
If ν ≪ μ and both σ-finite, there exists a measurable function f ≥ 0 such that:
  ν(A) = ∫_A f dμ  for all measurable A.

f is called the Radon-Nikodym derivative: f = dν/dμ.
Unique up to μ-null sets.

Chain rule: if λ ≪ ν ≪ μ, then dλ/dμ = (dλ/dν)(dν/dμ) μ-a.e.
```

**Why it matters**:
1. **Probability densities**: If X has distribution P_X ≪ Lebesgue measure, then f = dP_X/dλ is the density function (pdf). Existence guaranteed by Radon-Nikodym.
2. **Change of measure**: If P, Q are probabilities with P ≪ Q, then dP/dQ exists and ∫ f dP = ∫ f(dP/dQ) dQ.
3. **Conditional expectation**: Construction of E[X|ℱ] uses Radon-Nikodym.

### Conditional Expectation

```
Let X ∈ L¹(Ω,ℱ,P) and 𝒢 ⊆ ℱ a sub-σ-algebra.
E[X|𝒢] is the unique 𝒢-measurable Y such that:
  ∫_A Y dP = ∫_A X dP  for all A ∈ 𝒢.

Existence: Define measure ν(A) = ∫_A X dP on (Ω,𝒢).
  ν ≪ P|_𝒢.
  By Radon-Nikodym: Y = dν/d(P|_𝒢) exists.

Properties:
  E[E[X|𝒢]] = E[X]                          [tower law / law of total expectation]
  If X is 𝒢-measurable: E[X|𝒢] = X          [nothing new to learn]
  E[X|𝒢] = E[X] if X ⊥ 𝒢                  [independence]
  E[XY|𝒢] = Y·E[X|𝒢] if Y is 𝒢-measurable  [pulling out known factors]
  Jensen's: φ(E[X|𝒢]) ≤ E[φ(X)|𝒢] for convex φ
```

## 7. Convergence Theorems

### Four Types of Convergence

```
a.s. (almost sure) convergence:
  P(fₙ → f) = 1, i.e., fₙ(ω) → f(ω) for a.e. ω.
  Pointwise convergence on full-measure set.

Convergence in measure (in probability):
  P(|fₙ - f| > ε) → 0 for all ε > 0.

Lᵖ convergence:
  ||fₙ - f||_p = (E|fₙ-f|ᵖ)^{1/p} → 0.

Weak convergence (distributions):
  E[φ(Xₙ)] → E[φ(X)] for all bounded continuous φ.
```

### Relationships

```
                    a.s. convergence
                   /                \
                  ↓                  ↓
             a.s. convergence   a.s. convergence
         (on finite measure spaces)    ↓
                  →    convergence in measure
                  ↑
             Lᵖ convergence (p ≥ 1)
                  ↓
             convergence in distribution (weakest)

None of these arrows reverse in general.
Counterexamples:
  • Convergence in measure but not a.s.: "typewriter sequence" on [0,1]
    f₁=1_{[0,1]}, f₂=1_{[0,1/2]}, f₃=1_{[1/2,1]}, f₄=1_{[0,1/4]}, ...
    ||fₙ||₁ → 0, but fₙ(x) ≢ 0 for any x.
  • a.s. convergence but not L¹: fₙ = n·1_{[0,1/n]}. fₙ → 0 a.s., but ∫fₙ = 1.
    (Needs DCT's dominating function, which doesn't exist here.)
```

**Subsequence principle**: fₙ → f in measure iff every subsequence has a further subsequence converging a.s.

### Weak Convergence (in Distribution)

```
Xₙ →_d X if F_{Xₙ}(x) → F_X(x) at all continuity points of F_X.

Equivalently: E[φ(Xₙ)] → E[φ(X)] for all bounded continuous φ.

Portmanteau theorem (equivalent characterizations):
  (1) E[φ(Xₙ)] → E[φ(X)] for all bounded continuous φ
  (2) E[φ(Xₙ)] → E[φ(X)] for all bounded Lipschitz φ
  (3) limsup P(Xₙ ∈ F) ≤ P(X ∈ F) for all closed F
  (4) liminf P(Xₙ ∈ G) ≥ P(X ∈ G) for all open G
  (5) P(Xₙ ∈ B) → P(X ∈ B) for all B with P(X ∈ ∂B) = 0

Characteristic functions (Lévy continuity theorem):
  Xₙ →_d X iff φ_{Xₙ}(t) → φ_X(t) for all t.
  φ_X(t) = E[e^{itX}] is the characteristic function.
  This is how CLT is proved rigorously.
```

### CLT — Measure-Theoretic Proof Sketch

```
Let X₁, X₂, ... iid with mean 0, variance σ².
Sₙ = (X₁+...+Xₙ)/(σ√n).

Characteristic function of Sₙ:
  φ_{Sₙ}(t) = [φ_X(t/(σ√n))]ⁿ

Taylor expand φ_X near 0:
  φ_X(t) = 1 + E[X]·it - E[X²]t²/2 + o(t²) = 1 - σ²t²/2 + o(t²)

Substituting:
  φ_{Sₙ}(t) = [1 - σ²(t/σ√n)²/2 + o(t²/n)]ⁿ
             = [1 - t²/(2n) + o(1/n)]ⁿ
             → e^{-t²/2}   (as n → ∞)

e^{-t²/2} is characteristic function of N(0,1).
By Lévy continuity: Sₙ →_d N(0,1). □
```

## 8. Product Measures and Integration

### Product Measure Construction

For σ-finite (Xᵢ, ℱᵢ, μᵢ):
```
Product measure μ₁×μ₂ on ℱ₁⊗ℱ₂:
  (μ₁×μ₂)(A×B) = μ₁(A)·μ₂(B)  for A∈ℱ₁, B∈ℱ₂.
Extends uniquely to ℱ₁⊗ℱ₂ by Carathéodory.

Lebesgue measure on ℝⁿ = Lebesgue ⊗ ... ⊗ Lebesgue (n times).
```

### Change of Variables (substitution)

For diffeomorphism T: U → V (open subsets of ℝⁿ):
```
∫_V f(y) dy = ∫_U f(T(x)) |det DT(x)| dx

|det DT(x)|: absolute value of Jacobian determinant.

This is Radon-Nikodym in disguise:
  Push-forward measure: (T₊μ)(A) = μ(T⁻¹(A))
  d(T₊μ)/dμ = 1/|det DT| ∘ T
```

**Polar coordinates**: T: (r,θ) → (r cos θ, r sin θ). |det DT| = r.
```
∫∫_{ℝ²} f(x,y) dx dy = ∫₀^∞ ∫₀^{2π} f(r cos θ, r sin θ) r dθ dr
```

## 9. Applications and ML Connections

### Ergodic Theory and MCMC

**Ergodic theory** studies measure-preserving transformations — dynamical systems that preserve volume. The central result connects time averages to space averages.

**Setup**: (Ω, ℱ, μ) is a probability space. T: Ω → Ω is **measure-preserving** if μ(T⁻¹(A)) = μ(A) for all A ∈ ℱ. Think of T as "advance the system by one time step."

T is **ergodic** if the only T-invariant sets are trivial: T⁻¹(A) = A ⟹ μ(A) = 0 or 1. Ergodicity means the system cannot be decomposed into independent subsystems — it explores the whole space.

**Birkhoff Ergodic Theorem** (time average = space average):
```
If T is measure-preserving on (Ω, ℱ, μ) and f ∈ L¹(μ):

  (1/n) Σ_{k=0}^{n-1} f(Tᵏω) → E_μ[f | ℐ]   μ-a.s. and in L¹

where ℐ = σ-algebra of T-invariant sets.

If T is ergodic: ℐ is trivial → E_μ[f | ℐ] = ∫ f dμ.

So for ergodic T:  (1/n) Σ_{k=0}^{n-1} f(Tᵏω) → ∫_Ω f dμ  a.s.

Time average of f along a single trajectory = space average of f over Ω.
```

**MCMC connection**: A Markov chain with stationary distribution π defines a (random) measure-preserving transformation. The chain is ergodic on (Ω, π) iff it is irreducible and aperiodic. The Birkhoff theorem is precisely why MCMC works: time-averages along the chain converge to expectations under π. Convergence rate is governed by the spectral gap of the transition operator — the gap between 1 and the second-largest eigenvalue of the Markov transition kernel acting on L²(π).

```
MCMC algorithm → generates trajectory ω₀, ω₁, ω₂, ...
Estimator: (1/n) Σ f(ωₖ) → ∫ f dπ  by ergodic theorem.
Speed of convergence ← spectral gap of transition kernel P on L²(π).
Mixing time ∝ 1 / spectral gap.
```

**Statistical mechanics**: The ergodic hypothesis (Boltzmann) is the physical claim that a gas system is ergodic — time averages of macroscopic observables equal ensemble averages over the microcanonical measure. This justifies thermodynamics from Hamiltonian mechanics. The hypothesis is false for most Hamiltonian systems (KAM theory), but true for sufficiently chaotic ones.

### KL Divergence and Radon-Nikodym

**KL divergence** (Kullback-Leibler) between probability measures P and Q:
```
KL(P || Q) = ∫ log(dP/dQ) dP    when P ≪ Q  (P absolutely continuous w.r.t. Q)
           = +∞                  when P ⊄ Q

The integrand is exactly the Radon-Nikodym derivative dP/dQ (log-likelihood ratio).
```

This makes precise several properties that are otherwise opaque:

**Asymmetry**: KL(P||Q) ≠ KL(Q||P) because dP/dQ and dQ/dP are different functions. Swapping them changes the measure you integrate against (P vs Q), not just the integrand.

**Why P ≪ Q is required**: If P(A) > 0 but Q(A) = 0 for some set A, then dP/dQ is undefined (or +∞) on A, and log(dP/dQ) = +∞ on a P-positive set → KL = +∞. KL divergence detects whether Q has full support relative to P.

**Connection to log-likelihood**: For parametric family {P_θ}, the score function is ∇_θ log(dP_θ/dμ) where μ is a reference measure. Fisher information = variance of the score = second derivative of KL at zero. Maximum likelihood estimation minimizes KL(P_data || P_θ) over θ.

**Variational formula** (Donsker-Varadhan):
```
KL(P || Q) = sup_{f bounded measurable} { E_P[f] - log E_Q[e^f] }

This is the measure-theoretic foundation for variational autoencoders (ELBO),
contrastive learning (InfoNCE), and density ratio estimation.
```

### Measure Theory in Probability and Statistics

```
Laws of large numbers (formal version):
  Strong LLN: (X₁+...+Xₙ)/n → E[X₁] a.s. if E[|X₁|] < ∞.
  Proof: Kolmogorov's criterion + Borel-Cantelli lemma.

Borel-Cantelli:
  If Σₙ P(Aₙ) < ∞: P(Aₙ i.o.) = 0   [events happen finitely often a.s.]
  If Σₙ P(Aₙ) = ∞ and independent: P(Aₙ i.o.) = 1   [infinitely often a.s.]
  Used to prove a.s. convergence results.

Kolmogorov extension theorem:
  Consistent family of finite-dimensional distributions → exists stochastic process
  with those distributions. Justifies defining Brownian motion, Gaussian processes.
```

### Measure Theory in Machine Learning

**Distributional shift and importance weighting**: Training data comes from source distribution P_S on X×Y; at deployment the distribution is P_T (target). Distributional shift means P_S ≠ P_T. The key quantity is the density ratio (Radon-Nikodym derivative):

```
Importance weight: w(x,y) = dP_T/dP_S (x,y)

Covariate shift assumption: P_S(Y|X) = P_T(Y|X), but P_S(X) ≠ P_T(X).
Then: w(x) = dP_T^X/dP_S^X (x)   [marginal density ratio]

Importance-weighted ERM:
  minimize  E_{(x,y)~P_S} [ w(x) · L(h(x), y) ]
  = E_{(x,y)~P_T} [ L(h(x), y) ]   [corrects for shift]

This is valid iff P_T ≪ P_S (target absolutely continuous w.r.t. source).
If P_T has support outside P_S, no reweighting can correct — must collect new data.
```

Density ratio estimation (estimating w(x) = dP_T/dP_S from samples) is a direct application of Radon-Nikodym theory. Methods include kernel mean matching, logistic regression on pooled samples (which estimates the log-ratio), and the variational formula above.

```
PAC learning (measure-theoretic):
  "Probably Approximately Correct" learning is defined over distributions.
  Loss: L(h) = P_{(X,Y)~D}(h(X) ≠ Y).
  Empirical loss: L̂(h) = (1/n) Σ 1[h(Xᵢ) ≠ Yᵢ].
  Uniform convergence: sup_{h∈H} |L̂(h) - L(h)| → 0 — needs measurability of sup.

Gaussian processes (GP):
  GP is a distribution over functions f: X → ℝ.
  All finite marginals (f(x₁),...,f(xₙ)) are jointly Gaussian.
  Kolmogorov extension: defines GP on infinite-dimensional function space.
  GP posterior: Bayes' theorem for functional Bayesian inference.

Neural networks and function spaces:
  Universal approximation: neural networks dense in C([0,1]ⁿ) (continuous functions).
  Density in Lᵖ follows from density of C_c.
  Why convergence theorems matter: loss function ∫ L(f(x), y) dP(x,y)
  — interchanging limit and integral uses DCT.
```

### Stochastic Processes and the Itô Integral

```
Filtration: 𝒻 = (ℱₜ)_{t≥0} increasing sub-σ-algebras of ℱ.
  ℱₜ = information available at time t.

Martingale: E[Xₜ|ℱₛ] = Xₛ for s ≤ t.  [fair game]
Submartingale: E[Xₜ|ℱₛ] ≥ Xₛ.
Optional stopping theorem: if τ is stopping time, E[X_τ] = E[X₀] under conditions.

Brownian motion:
  B₀ = 0, B has independent stationary increments, Bₜ-Bₛ ~ N(0,t-s).
  Sample paths: continuous but nowhere differentiable a.s.
  Quadratic variation: Σ |Bₜ_{i+1} - Bₜᵢ|² → T in probability (not 0).
  This unbounded variation is why Riemann-Stieltjes integration fails for Brownian paths.
```

**The Itô integral via L² isometry**: The central measure-theoretic fact that makes stochastic calculus work:

```
For adapted process H = (Hₜ)_{t∈[0,T]} with E[∫₀ᵀ Hₜ² dt] < ∞:

  ∫₀ᵀ Hₜ dBₜ  is defined as an L²(Ω,P) limit of Riemann-Itô sums.

Itô isometry:  E[ (∫₀ᵀ Hₜ dBₜ)² ] = E[ ∫₀ᵀ Hₜ² dt ]

This is an L² isometry: the map H ↦ ∫H dB is an isometry from
  L²([0,T]×Ω, Lebesgue⊗P)  →  L²(Ω, P).

Consequence: the Itô integral is defined by extending the isometry from
  simple (step-function) processes — where the sum is finite — to all of L²
  using completeness. Exactly the same construction as the Lebesgue integral
  (extend from simple functions using MCT/DCT), now in L².

Itô's formula:  df(Bₜ) = f'(Bₜ)dBₜ + (1/2)f''(Bₜ)dt
  The extra (1/2)f'' dt term (vs. classical chain rule) comes from the quadratic
  variation [B,B]_t = t — the L² norm of Brownian increments is non-negligible.
```

## Common Confusion Points

**σ-algebra vs topology**: A topology is closed under arbitrary unions; a σ-algebra under countable unions. Borel σ-algebra is generated BY the topology, not equal to it. Every Borel set is measurable (for Lebesgue measure); not every measurable set is Borel.

**Null set vs empty set**: A null set has measure zero but may be non-empty. The rationals ℚ are a null set for Lebesgue measure but are countably infinite.

**"Almost everywhere" is measure-specific**: A.e. for Lebesgue measure says something different from a.e. for counting measure. Under counting measure, every set has positive measure unless empty.

**Fubini requires checking**: The condition ∫∫|f| < ∞ is NOT automatic. Without it, iterated integrals can disagree. Always verify integrability first (use Tonelli with |f| ≥ 0 if unsure).

**DCT: dominating function in L¹, not just bounded**: The dominating function g must satisfy ∫g < ∞, not just g < ∞ everywhere. On an infinite measure space, a bounded function need not be in L¹.

**L² ≠ square-summable functions**: L²(μ) consists of equivalence classes modulo null sets. Two functions that differ on a null set are the same element of L². This distinction matters when talking about representatives.

**Conditional expectation is an equivalence class**: E[X|𝒢] is defined uniquely only up to 𝒢-null sets (a.s. unique). There are infinitely many valid representatives; we usually work with any one of them.

**Weak convergence does not imply convergence of densities**: Even if Xₙ →_d X, the densities fₙ need not converge pointwise to f_X. Weak convergence is about integration against test functions, not pointwise density convergence.

**KL divergence is not a metric**: KL(P||Q) ≠ KL(Q||P) and the triangle inequality fails. The symmetric version KL(P||Q)+KL(Q||P) and the Jensen-Shannon divergence (average of two KLs) are used when symmetry is needed.

**Ergodicity is a property of the transformation, not the space**: A probability space (Ω,ℱ,μ) can carry many different measure-preserving transformations, some ergodic and some not. Ergodicity is about the dynamics, not the static measure structure.
