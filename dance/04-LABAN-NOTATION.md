# Labanotation and Movement Analysis

## The Big Picture

Rudolf Laban (1879-1958) created two distinct but related systems: Labanotation (a written score system for specific movement sequences) and Laban Movement Analysis (LMA, a qualitative framework for analyzing movement qualities). These are as different from each other as musical notation is from music theory -- one records specific pieces, the other provides a vocabulary for talking about music.

```
+-------------------------------------------------------------------+
|                LABAN'S TWO SYSTEMS                                |
|                                                                   |
|  LABANOTATION              LABAN MOVEMENT ANALYSIS (LMA)          |
|  (Kinetography Laban)                                             |
|                                                                   |
|  PURPOSE: Write down a           PURPOSE: Analyze movement quality |
|  specific dance sequence         regardless of specific sequence  |
|                                                                   |
|  WHAT IT RECORDS:                WHAT IT DESCRIBES:               |
|  -- Which body part              -- Effort (dynamic quality)      |
|  -- Which direction              -- Space (spatial intent)        |
|  -- At what level                -- Shape (body form change)      |
|  -- For how long                 -- Body (body organization)      |
|  -- With what path                                                |
|                                                                   |
|  OUTPUT: A score you can         OUTPUT: A description of HOW     |
|  reproduce exactly               movement feels and functions     |
|                                                                   |
|  ANALOG: Musical notation        ANALOG: Music theory terminology |
|  (write what notes/when)         (describe timbre, tension, arc)  |
+-------------------------------------------------------------------+
```

## Labanotation: The Score System

Labanotation is a staff-based notation system. Unlike musical notation (horizontal time axis), Labanotation uses a **vertical time axis** -- you read from bottom to top, as the body moves upward through time.

### The Staff Structure

```
LABANOTATION STAFF (vertical reading, bottom to top = forward in time)

  |    |    |    |    |    |    |    |
  | LS | LA | LT | RL | RT | RA | RS |

  LS = Left support (weight-bearing)      RS = Right support
  LA = Left arm                           RA = Right arm
  LT = Left torso                         RT = Right torso
  RL = Right leg / Left leg (center)

  COLUMN POSITIONS from center out:
  CENTER:  body center, whole body actions
  INNER:   leg gesture
  MIDDLE:  torso actions
  OUTER:   arm actions
  OUTER+:  head actions (sometimes)

TIME reads bottom to top: what's at the bottom happens first.
```

### Direction Symbols

Labanotation encodes direction as the primary spatial descriptor. There are 27 possible directions (8 horizontal + high/middle/low for each = combinations):

```
DIRECTION SYMBOLS (simplified)

  Place    Forward   Back     Right    Left     Right Forward  etc.
  (still)  diagonal diagonal

  Each direction symbol is a geometric shape.
  The SHADING of the symbol encodes LEVEL (height):
    White/empty = high level (above normal standing)
    Grey/dotted = middle level (at normal height)
    Black/filled = low level (below normal height)

EXAMPLE: A symbol that looks like a rectangle pointing forward,
filled black = moving forward at low level (crouching forward motion)
```

### Duration Encoding

Time is shown by the LENGTH of the symbol on the staff:

```
SHORT SYMBOL = SHORT DURATION

  |----| = quarter note equivalent
  |--------| = half note equivalent
  |---------|
  |         | = whole note equivalent (4x as long)
  |---------|

The staff has a TIME SIGNATURE equivalent: measure bars drawn
across the full width of the staff.
```

### Body Part Specification

```
BODY PART INDICATIONS

Head/face direction: symbols in outer columns or special indicators
Torso: indicated in torso columns; can specify spinal curve direction
Arms: outer columns; full arm direction + specific joint modifiers
Hands: separate symbols for wrist, hand orientation, fingers
Legs: inner columns; distinction between gesture leg and support leg
Feet: support symbols show which part of foot carries weight
```

### Pins and Modifiers

Additional symbols modify the primary direction symbol:

```
MODIFIER TYPES

  Timing modifiers:   hold, pause, accelerate, decelerate
  Dynamic modifiers:  sharp/sudden, sustained, strong, light
  Relationship signs: touch, grasp, support
  Contact signs:      body contact indications
  Path signs:         curved vs straight spatial path
  Turn signs:         rotation around vertical axis
  Elevation signs:    jumps, aerial work
```

### Reading a Score: Practical Example

```
READING A PLIE IN LABANOTATION (simplified)

TIME: Measure 1
                 |    |    |    |    |    |    |    |
  (2 counts)    |    |    |    | SS |    |    |    |
                |    |    |    |    |    |    |    |
  (1 count)     |    |    |    |    |    |    |    |
                 ================================
                (START POSITION shown below bar)

SS in center = support symbol with both feet
Direction: Place (not moving horizontally)
Level: Low (bend indicator over support symbol)

Combined with a time signature and key at bottom of score,
a notation reader can reconstruct the exact movement.
```

### Labanotation Organizations

```
INSTITUTIONAL STRUCTURES

  ICKL (International Council of Kinetography Laban)
    -- Standards body for the notation system
    -- Annual congresses
    -- Certification programs

  Dance Notation Bureau (New York, 1940-)
    -- Primary archive of notated scores (5,000+ scores)
    -- Reconstructs works from notation when choreographer unavailable
    -- Laban/Bartenieff Institute (LIMS) for LMA certification

  BENESH INSTITUTE (London)
    -- Parallel notation system (5-line staff)
    -- Standard in Royal Ballet and many UK companies
    -- Different system, not interchangeable
```

## Laban Movement Analysis (LMA): The Qualitative Framework

LMA provides a vocabulary for describing HOW movement happens -- its qualities, spatial patterns, and organizational principles. This is widely used in dance education, dance/movement therapy, actor training, and motion capture analysis.

### The Four Components: BESS

```
+----------------------------------------------------------+
|                    LMA FRAMEWORK                         |
|                                                          |
|  BODY              EFFORT              SPACE              SHAPE  |
|  (What moves)      (How it feels)      (Where it goes)    (Form) |
|                                                          |
|  Body organization  Motion factors:    Spatial intent:    Shape  |
|  Body part          Weight            Near reach space    modes: |
|  relationships      Space             Kinesphere         Modes of |
|  Connectivity       Time              Directions         shape    |
|  patterns           Flow              Diagonals          change   |
+----------------------------------------------------------+
```

### BODY: Organization and Connectivity

```
BODY COMPONENT

BREATH SUPPORT:
  All movement supported by (or opposed to) the breath cycle
  Inhale / Exhale as the primary movement rhythm

BODY CONNECTIVITY:
  Breath     -- most basic; connects all parts
  Core-Distal -- center to periphery (radial)
  Head-Tail   -- vertical spine as organizing axis
  Upper-Lower -- above/below horizontal midline
  Body-Half   -- left/right differentiation
  Cross-Lateral -- diagonal connections (right arm/left leg)

BODY PART EMPHASIS:
  Which body part leads the movement? (head leading vs foot leading)
  Active body parts vs passive follow-along

INITIATION:
  Where does the movement start?
  Distal initiation (fingertip starts): reaching quality
  Proximal initiation (spine starts): whole-body quality
```

### EFFORT: The Dynamic Quality

Effort is the most psychologically rich component -- it describes the inner attitude of movement, which shapes how it looks and feels from the outside.

```
THE FOUR MOTION FACTORS AND THEIR POLARITIES

WEIGHT:         Strong/Heavy <-----------> Light/Delicate
  How does the mover relate to gravity?
  Strong: commits to weight, earthward, assertive, condensing
  Light: resists/ignores weight, upward, attentive, floating

SPACE:          Direct/Focused <-----------> Indirect/Flexible
  How does the mover relate to space?
  Direct: one path, channeled attention, single-pointed
  Indirect: wandering, multi-focused, flexible, encompassing

TIME:           Sudden/Quick <-----------> Sustained/Lingering
  How does the mover relate to time?
  Sudden: urgent, decisive, instantaneous
  Sustained: indulgent, prolonged, taking all the time

FLOW:           Bound/Controlled <-----------> Free/Released
  How easy is it to stop the movement?
  Bound: controlled, held back, stoppable at any moment
  Free: abandoned, ongoing, hard to interrupt

EFFORT COMBINATIONS (Basic Effort Actions):

  Strong + Direct + Sudden   = PUNCH    (decisive attack)
  Strong + Direct + Sustained = PRESS   (steady firm push)
  Strong + Indirect + Sudden = SLASH   (broad powerful cut)
  Strong + Indirect + Sustained = WRING (twisting struggle)
  Light + Direct + Sudden    = DAB     (delicate touch)
  Light + Direct + Sustained = GLIDE   (smooth, controlled skim)
  Light + Indirect + Sudden  = FLICK   (airy dismissal)
  Light + Indirect + Sustained = FLOAT  (dreamy drift)
```

**Bridge to computing:** Effort qualities are analogous to attributes of computational processes. A "punch" action (Strong/Direct/Sudden) maps to an interrupt -- sudden, targeted, strong. A "float" (Light/Indirect/Sustained) maps to a long-running background process with no fixed path. LMA effort vocabulary is used in animation and robotics to make movement feel "natural."

### SPACE: Kinesphere and Pathways

```
SPACE COMPONENT

KINESPHERE:
  The bubble of personal space around the body --
  all directions reachable without changing weight support.

  Near reach:    immediately adjacent to body
  Mid reach:     arms' length away
  Far reach:     full extension, on toes, stretching

SPATIAL TENSION:
  Movement that aims toward a specific point (arriving)
  vs movement that explores space without destination

DIMENSIONS AND DIAGONALS:
  Horizontal plane:  Width (door plane)
  Vertical plane:    Height (wheel plane)
  Sagittal plane:    Depth (table plane)

  The three dimensions intersect at the center of the kinesphere.
  SCALES: movement patterns that trace through multiple dimensions

  LABAN'S CHOREUTICS:
    The geometry of the kinesphere mapped as a crystalline structure
    -- Icosahedron as the geometric model of full body reach space
    -- Spatial scales trace paths through the vertices
    -- Connects to Buckminster Fuller's geometry
```

### SHAPE: The Body's Form

```
SHAPE COMPONENT

SHAPE QUALITIES (Modes of Shape Change):
  Advancing/Retreating    -- moving toward / away from something
  Rising/Sinking          -- vertical shape change
  Spreading/Enclosing     -- widening / narrowing

SHAPE FORMS:
  Pin      -- elongated, linear, narrow
  Wall     -- flat, spread, planar
  Ball     -- rounded, enclosing, spherical
  Screw    -- twisted, spiral

SHAPE FLOW:
  The self-referential change of body shape (breath-related)
  Not in relation to external space, but internal bodily change

DIRECTIONAL:
  Arc-like movement (spoke-like, drawing arcs in space)

SHAPING:
  Three-dimensional molding (adapting to and molding space)
  -- Most complex; requires clear spatial intent
```

## LMA in Motion Capture and Digital Applications

The bridge from LMA to digital movement data is increasingly formalized:

```
MOCAP + LMA PIPELINE

  Performer with markers -->  3D position data per marker
  captured at 60-120fps         (x, y, z over time)
             |
             v
  RAW DATA: joint positions, velocities, accelerations

  LMA MAPPING:
    Effort/Weight:   Acceleration magnitude + path quality
    Effort/Time:     Velocity change rate (sudden vs sustained)
    Effort/Space:    Path directness (DTW metric)
    Effort/Flow:     Jerk (rate of acceleration change) -- high jerk = bound
    Shape:           Body outline convex hull changes
    Space/Kinesphere: Reach space envelope over time

  OUTPUT: Movement annotated with LMA qualities
  USE CASES:
    -- Choreographic analysis (which Effort qualities dominate?)
    -- Animation parameter control (make robot move with "punch" quality)
    -- Dance therapy assessment (how does patient's movement change?)
    -- Performance comparison (how different are two Giselle performances?)

SPECIFIC TOOLS:
  -- Choreographic Language Agent (CLA) -- Forsythe/MIT Media Lab
  -- Movementlab tools (CIRMMT Montreal)
  -- OpenLaban (open-source score editor)
  -- MotionBank project (Forsythe Company) -- digital scores
```

## Benesh Movement Notation: The Alternative

Benesh (1955) takes a different approach than Laban -- it uses a 5-line horizontal staff (like music) and represents the dancer as a stick figure viewed from behind:

```
BENESH NOTATION COMPARISON

  LABANOTATION:            BENESH:
  -- Vertical staff        -- Horizontal staff (musical)
  -- Symbols encode        -- Stick figure on 5-line staff
     direction/level          (like musical notes but body position)
  -- Geometric notation    -- Pictographic notation
  -- More widely used      -- UK/Royal Ballet standard
     internationally       -- Easier for musicians to learn
  -- Preferred for LMA     -- Preferred in commercial UK companies
     integration

Neither system is "better" -- they encode the same information differently.
Like XML vs JSON: different syntax, equivalent semantic content.
```

## Common Confusion Points

**Labanotation and LMA are not the same thing.**
Both created by Laban; both deal with movement. Labanotation: write down a specific dance. LMA: analyze/describe movement qualities. A person can be expert at one and ignorant of the other.

**Effort is not about effort level (exertion).**
"Effort" in LMA is a technical term meaning inner motivation or dynamic quality of movement. "Light Effort" does not mean low energy or laziness -- it means attentive, delicate, resisting gravity. A very energetic dancer can be using Light Weight.

**Labanotation scores are not widely used for recording new work.**
Video has largely replaced notation for documentation purposes. Labanotation remains most valuable for reconstruction of works where choreographers are unavailable (dead or memory-dependent) and for research analysis.

**The kinesphere is not the same as personal space in proxemics.**
The kinesphere is the geometrically defined space of reachable movement (all directions from a fixed base of support). Edward Hall's proxemics (intimate/personal/social/public distance zones) measures social space for interpersonal interaction. Different concepts that happen to both involve spatial bubbles.

**Bound flow is not "tense" in the emotional sense.**
Bound flow means movement that can be stopped at any moment -- controlled, held back. It is a technical description of movement quality, not an assessment of the mover's psychological state. A skilled ballet dancer performs with highly bound flow not because they are tense but because they are controlled.
