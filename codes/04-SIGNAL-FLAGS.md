# International Code of Signals — Maritime Signal Flags

## The Big Picture

Signal flags are the maritime lingua franca: a standardized visual communication system that works across language barriers. The **International Code of Signals (ICS)**, published by the International Maritime Organization (IMO) and updated periodically, is the global standard.

```
┌──────────────────────────────────────────────────────────────────┐
│              SIGNAL FLAG SYSTEM MAP                              │
│                                                                  │
│  40 distinct flags:                                              │
│  ┌──────────────┬───────────────────────────────────────────┐    │
│  │ 26 Letter     │ One per letter A–Z, each with:            │   │
│  │ Flags         │ (1) Single-letter meaning (procedural)    │   │
│  │               │ (2) Part of multi-flag combination (ICS   │   │
│  │               │     codebook, thousands of messages)      │   │
│  ├──────────────┼───────────────────────────────────────────┤   │
│  │ 10 Numeral   │ Pennants 0–9, hoisted with letter flags   │   │
│  │ Pennants     │ for bearings, speeds, chart references    │   │
│  ├──────────────┼───────────────────────────────────────────┤   │
│  │ 3 Substitute  │ Allows same flag repeated in one hoist    │   │
│  │ Pennants      │ (can't carry two identical flags)         │   │
│  ├──────────────┼───────────────────────────────────────────┤   │
│  │ Answering    │ "I understand/am copying" pennant         │   │
│  │ Pennant      │ Dipped = "I have not yet understood"      │   │
│  └──────────────┴───────────────────────────────────────────┘   │
└──────────────────────────────────────────────────────────────────┘
```

## Single-Letter Flag Meanings (ICS A–Z)

Each letter flag has a **single-letter procedural signal** — a meaning when flown alone. These are the most important signals; memorized by all mariners.

## Numeral Pennants

Numeral pennants 0–9 have a distinctive pennant (triangular) shape, easily distinguished from the square/rectangular letter flags:

```
Numeral pennants are used for:
— Bearing numbers (e.g., "enemy bearing 225°")
— Speed signals
— Time references
— Latitude/longitude in coded messages
— Course alterations

Hoisting: pennant 2 over pennant 2 over pennant 5 = bearing 225°
          Code flag over pennant 5 = "your speed is 5 knots"
```

Color scheme:
```
0 — Red/yellow alternating stripes
1 — Solid red
2 — Solid blue
3 — Red/white/red/white/red vertical stripes
4 — Red/white checkerboard (small)
5 — Yellow/blue alternating stripes
6 — Black/white horizontal stripes
7 — Yellow/red diagonal
8 — Red/white/red/white/red/white vertical
9 — Gold/black vertical
```

## Substitute (Repeater) Pennants

When the same flag needs to appear twice in one hoist, substitute pennants fill in:

```
┌─────────────────────────────────────────────────────────────┐
│  SUBSTITUTE PENNANTS                                        │
│                                                             │
│  1st Substitute: repeats the top flag in the hoist          │
│  2nd Substitute: repeats the second flag from top           │
│  3rd Substitute: repeats the third flag from top            │
│                                                             │
│  Example: Hoist "ALLY" (4 flags)                            │
│  A = Alpha (flag)                                           │
│  L = Lima (flag)                                            │
│  L = Lima — PROBLEM: can't hoist Lima twice                 │
│           → Use 2nd Substitute (repeats 2nd flag = Lima)    │
│  Y = Yankee (flag)                                          │
│                                                             │
│  Hoist reads top-to-bottom: Alpha / Lima / 2nd Sub / Yankee │
└─────────────────────────────────────────────────────────────┘
```

## Answering Pennant

```
┌─────────────────────────────────────────────────────────────┐
│  ANSWERING PENNANT (Code and Answer Pennant)                │
│                                                             │
│  Shape: Long pennant (swallowtail narrow)                   │
│  Colors: Red/white horizontal stripes                       │
│                                                             │
│  Usage:                                                     │
│  FULL HOIST (at mast):  "I understand your signal /         │
│                          message is received and understood" │
│  HALF HOIST (dipped):   "I am receiving / do not yet        │
│                          understand — repeat if needed"     │
│                                                             │
│  Also used to prefix a reply hoist in coded messages:       │
│  Sending station hoists message flags                       │
│  Receiving station hoists answering pennant at dip          │
│  When understood: hauls answering pennant to full mast      │
│  Sending station hauls down their message flags             │
│  → Protocol complete                                        │
└─────────────────────────────────────────────────────────────┘
```

### Structure of ICS Codebook

```
ICS Publication (IMO, current edition):
├── Single-letter signals (procedural, memorized)
├── Two-letter signals (urgency/common situations)
│   ├── A__ — Ship in distress (AM, AN, AC...)
│   ├── B__ — Ship movements
│   ├── C__ — Navigational/meteorological
│   ├── D__ — Distress and salvage
│   ├── E__ — Casualty
│   ├── F__ — Casualties (medical)
│   ├── G__ — Salvage/towing
│   ├── H–Z — Pilotage, weather, routing...
├── Three-letter medical signals (three flags)
│   (M__ codes cover detailed medical situations)
└── Supplementary (vessel IDs, port signals)
```

## Ground-to-Air Emergency Signals

When survivors on the ground need to signal aircraft, these symbols are laid out in open terrain using any available material (rocks, logs, cloth, footprints in snow):

```
┌─────────────────────────────────────────────────────────────┐
│  GROUND-TO-AIR EMERGENCY SIGNALS (ICAO/survival standard)   │
│                                                             │
│  Symbol  Meaning                                            │
│  ──────  ─────────────────────────────────────────────────  │
│    V     Require assistance                                 │
│    X     Require medical assistance                         │
│    →     Going this direction (arrow pointing route)        │
│    N     No / Negative (response)                           │
│    Y     Yes / Affirmative (response)                       │
│    LL    All is well                                        │
│    F     Need food and water                                │
│    △     Safe to land here                                  │
│    K     Tell us the direction to proceed                   │
│                                                             │
│  Minimum size: 3m long for air visibility at 1,000 ft       │
│  Better: 6–10m for high-altitude aircraft                   │
│  Contrast: dark on light ground, or light on dark           │
│  Night: fires in the symbol pattern                         │
│                                                             │
│  ICAO standard (same symbols used by militaries worldwide)  │
└─────────────────────────────────────────────────────────────┘
```

## Practical Flag Handling

### Hoisting Protocol

```
STANDARD FLAG COMMUNICATION SEQUENCE:

Sender                              Receiver
──────                              ────────
Prep message flags on halyard
Hoist flags to top of mast ──────→
                              Haul answering pennant to half-mast
                              (= "I see the signal, decoding")
                              ↓
                              Decode flag combination
                              ↓
                              Haul answering pennant full ──────→
Haul down message flags
(= acknowledge completion)
```

### Reading Hoist Order

Flags are **read top-to-bottom** on a single halyard. Multiple halyards: port halyard has priority over starboard if both flown simultaneously.

### International vs. NATO Usage

```
┌────────────────────────────────────────────────────────┐
│  ICS (civilian maritime): universal, IMO published     │
│  NATO Signal Book (ACP 175): military, classified      │
│                                                        │
│  NATO uses additional flags and signals not in ICS.    │
│  The 26 letter flags and their shapes/colors are       │
│  identical between ICS and NATO — only the codebook    │
│  content differs.                                      │
└────────────────────────────────────────────────────────┘
```

## Common Confusion Points

**The diving flag (A flag) vs. the US recreational diving flag (red/white diagonal)**: In US coastal waters, recreational divers commonly use a red flag with a white diagonal stripe (the "diver-down flag"). This is NOT the international ICS Alpha flag. When diving in international or offshore waters, the ICS Alpha flag (blue/white swallowtail) is the correct international signal. Both are often flown together in US waters.

**Q flag is not quarantine detained**: Q (Quebec) means "I am healthy — I request free pratique (permission to enter port)." Quarantine detention (vessel held) has different signals. "Yellow flag = quarantine" is the historical shorthand; in the ICS the Q flag means "I'm clean, please let me in."

**C (Charlie) = Yes, N (November) = No**: The most useful pair for answering yes/no questions in flag communication. Charlie = affirmative response, November = negative. Easy to confuse alphabetically.

**The ICS codebook has thousands of messages**: Single-letter signals are just the entry level. A full set of 40 signal flags can encode any message from the codebook. Most commercial vessels carry the full codebook; yachts carry the abbreviated table of most-used signals.

**"Blue Peter" = P flag**: The tradition of hoisting the P (Papa) flag — a white square on blue border — before departure dates to 18th-century naval practice. It means "vessel about to proceed to sea; all persons should come aboard." Still used in yacht clubs today to signal imminent departure.
