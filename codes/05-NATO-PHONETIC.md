# NATO/ICAO Phonetic Alphabet & Radio Procedure

## The Big Picture

The NATO/ICAO phonetic alphabet is a **noise-robust spelling alphabet**: each letter has a distinct spoken word, chosen so that individual letters sound unmistakably different even over degraded radio channels. It eliminates the ambiguity of spelling letters aloud (B/D, M/N, S/F all sound similar under noise).

```
┌──────────────────────────────────────────────────────────────────┐
│            NATO/ICAO PHONETIC ALPHABET — SYSTEM MAP              │
│                                                                  │
│  Purpose: Anti-ambiguity communication over radio/telephone      │
│  Standard: ICAO Annex 10 (aviation), STANAG 3910 (NATO military) │
│  Adopted: 1956 (finalized, after ICAO 1951 trial version)        │
│                                                                  │
│  Three parallel usage domains:                                   │
│  ┌───────────────────────────────────────────────────────────┐   │
│  │ Aviation    │ ATC uses ICAO alphabet for all callsigns,   │  │
│  │             │ runways, altimeter settings, frequencies    │  │
│  ├─────────────┼───────────────────────────────────────────  │  │
│  │ Military    │ NATO standard for all inter-allied comms;   │  │
│  │             │ also tactical radio procedure (STANAG)      │  │
│  ├─────────────┼───────────────────────────────────────────  │  │
│  │ Maritime    │ GMDSS (Global Maritime Distress Safety      │  │
│  │             │ System) and VHF Channel 16 calls            │  │
│  └─────────────┴───────────────────────────────────────────  │  │
│                                                                  │
│  Also ubiquitous: customer service, police dispatch, IT         │
│  helpdesk (spelling account names, serial numbers, passwords)   │
└──────────────────────────────────────────────────────────────────┘
```

## NATO Phonetic Numbers

Numbers are spoken to eliminate cross-language confusion too:

```
┌──────────────────────────────────────────────────────────┐
│           PHONETIC NUMBERS (NATO/ICAO)                   │
├───────┬──────────────┬────────────────────────────────── ┤
│  0    │  Nadazero    │  nah-dah-ZAY-roh                  │
│  1    │  Unaone      │  oo-nah-WUN                       │
│  2    │  Bissotwo    │  BEES-soh-TOO                     │
│  3    │  Terrathree  │  TAY-rah-TREE                     │
│  4    │  Kartefour   │  KAR-tay-FOWER                   │
│  5    │  Pantafive   │  PAN-tah-FIVE                     │
│  6    │  Soxisix     │  SOK-see-SIX                     │
│  7    │  Setteseven  │  SAY-tay-SEV-en                  │
│  8    │  Oktoeight   │  OK-toh-AYT                      │
│  9    │  Novenine    │  NOV-ay-NINER                    │
└───────┴──────────────┴────────────────────────────────── ┘
```

**Important**: Aviation specifically uses "NINER" for 9 to avoid confusion with the German word "Nein" (No) in multilingual airspace. The full phonetic number names are less commonly used in practice — most operators say "one," "two," etc. but always use "niner" for 9.

## History: From ABLE to ALFA

The current alphabet is the third major English spelling alphabet:

```
┌──────────────────────────────────────────────────────────────┐
│           EVOLUTION OF ENGLISH PHONETIC ALPHABETS            │
│                                                              │
│  WWI — British/US military (various, non-standardized)       │
│  Example: Ack Beer Can Don Edward Frederick George...        │
│                                                              │
│  WWII — ABLE BAKER alphabet (US military, 1941)              │
│  A=Able  B=Baker  C=Charlie  D=Dog  E=Easy  F=Fox            │
│  G=George  H=How  I=Item  J=Jig  K=King  L=Love              │
│  M=Mike  N=Nan  O=Oboe  P=Peter  Q=Queen  R=Roger            │
│  S=Sugar  T=Tare  U=Uncle  V=Victor  W=William               │
│  X=Xray  Y=Yoke  Z=Zebra                                     │
│                                                              │
│  Problem: "Able", "Baker", "George", "Dog" are English-      │
│  centric. Non-native English speakers couldn't use them.     │
│                                                              │
│  ICAO 1951 — First international attempt                     │
│  Changed most words; tested against 31 languages             │
│                                                              │
│  ICAO/NATO 1956 — Current alphabet                           │
│  Tested for:                                                 │
│  — Intelligibility in noise (signal-to-noise tests)          │
│  — Pronounceability across language families                 │
│  — Acoustic distinctiveness (no two words confusable)        │
│  — Each word recognizable when heard in isolation            │
│                                                              │
│  Key changes from ABLE BAKER to current:                     │
│  Able → Alfa (ph not ph; "Alpha" spelled differently)        │
│  Baker → Bravo                                               │
│  Dog → Delta                                                 │
│  Easy → Echo                                                 │
│  Fox → Foxtrot                                               │
│  George → Golf                                               │
│  How → Hotel                                                 │
│  Jig → Juliett                                               │
│  King → Kilo                                                 │
│  Love → Lima                                                 │
│  Nan → November                                              │
│  Oboe → Oscar                                                │
│  Peter → Papa                                                │
│  Queen → Quebec                                              │
│  Roger → Romeo                                               │

│  Uncle → Uniform                                             │
│  William → Whiskey                                           │
```

## Radio Procedure Words (Prowords)

Standard procedure words are the protocol layer on top of the phonetic alphabet. They compress multi-word concepts into single recognizable terms:

### Core Aviation Prowords (ICAO)

| Word | Meaning | Notes |
|------|---------|-------|
| **AFFIRM** | Yes | Aviation prefers "Affirm" over "Roger" for yes/no |
| **NEGATIVE** | No | Clear negative — don't just say "no" |
| **ROGER** | I have received your last transmission | Does NOT mean "yes." Only means "received." |
| **WILCO** | I have received, understand, and will comply | Combines "received" + "will comply" |
| **OVER** | My transmission is ended; response expected | Implies two-way exchange |
| **OUT** | This exchange is complete; no reply expected | Final closure of communication |
| **SAY AGAIN** | Repeat your last transmission | NOT "repeat" — "repeat" has specific military meaning (repeat artillery fire) |
| **STANDBY** | Wait; I will respond shortly | Pause, don't transmit |
| **BREAK** | Separating ATC instruction from other content | "Speedbird 123, cleared to land runway 27L, BREAK, surface wind 270 at 8" |
| **CORRECTION** | An error was made; the correct version follows | "N5...correction, N7..." |
| **VERIFY** | Check and confirm | |
| **CONFIRM** | I confirm (or: please confirm) | Context-dependent |
| **DISREGARD** | Ignore my previous transmission | |
| **I SAY AGAIN** | I am repeating for clarity | Operator-initiated repeat |
| **WORDS TWICE** | Request each phrase repeated twice (poor channel) | |

### Military-Specific Prowords

| Word | Meaning |
|------|---------|
| **ACKNOWLEDGE** | Inform me that you have received and understood |
| **ACTUAL** | The person in command (not the radio operator) — "Sunray Actual" = commander himself |
| **FIGURES** | Numerals follow (signal to interpret following as numbers) |
| **NINER** | 9 (always used; prevents confusion with "nein") |
| **RADIO CHECK** | What is my signal quality? (response: ROGER/WEAK/UNREADABLE etc.) |
| **SEND YOUR TRAFFIC** | Go ahead, I am ready to copy |
| **THIS IS** | My callsign follows |
| **WAIT OUT** | I must pause; I will return |
| **NOTHING FOLLOWS** | No further traffic to send (end of message indicator) |

### GMDSS / Maritime Prowords

Maritime radio (GMDSS) on VHF Channel 16 adds:

| Word | Meaning |
|------|---------|
| **MAYDAY** | Distress signal (life in danger); repeat three times |
| **PAN-PAN** | Urgency signal (urgent, not immediate danger); repeat three times |
| **SÉCURITÉ** | Safety signal (navigation hazard warning); repeat three times |
| **ALL STATIONS** | Call to all vessels/stations (broadcast) |
| **SEELONCE MAYDAY** | Silence requested — distress working in progress |
| **SEELONCE FEENEE** | Silence lifted — distress working complete |
| **PRUDONCE** | Limited radio communication may resume |

## Full Radio Call Structure

### Aviation Initial Call

```
PILOT TO ATC:
"[ATC unit], [aircraft callsign], [position/altitude], [request]"

"Boston Center, Speedbird Two Three, flight level three five zero,
 request direct BOSOX"

ATC REPLY:
"[Callsign], [ATC unit], [instruction/information]"

"Speedbird Two Three, Boston Center, roger, cleared direct BOSOX"
```

### Spelling Out with Phonetic Alphabet

```
"Your callsign — November Four Seven Alpha Sierra, N-4-7-A-S —
 spelled: November-Four-Seven-Alfa-Sierra"

"Registration: Golf-Golf-Alpha-Bravo-Charlie"

"Transponder code: Seven-Seven-Zero-Zero"
  (Note: "SQUAWK SEVEN SEVEN ZERO ZERO")
```

### Military Net Call (STANAG)

```
Station calling a net:
"TANGO TANGO TANGO, THIS IS NOVEMBER FOUR SIX, OVER"

Net control answering:
"NOVEMBER FOUR SIX, THIS IS TANGO, SEND YOUR TRAFFIC, OVER"

Sending message:
"TANGO, THIS IS NOVEMBER FOUR SIX,
 MESSAGE FOLLOWS:
 [PRECEDENCE] [ADDRESSEE] [FROM] [DATE-TIME] [TEXT]
 OVER"

Acknowledgment:
"NOVEMBER FOUR SIX, THIS IS TANGO, ROGER, OUT"
```

## Read-Back Requirements (Aviation)

In aviation, ATC clearances require **read-back** by the pilot — repeated verbatim to confirm:

```
ATC: "Speedbird 456, turn left heading 270, descend and maintain
     flight level 280, reduce speed to 250 knots"

PILOT read-back: "Left heading 270, descend flight level 280,
     speed 250, Speedbird 456"

ATC: "Speedbird 456, readback correct"
```

If read-back is wrong, ATC issues a correction: "Speedbird 456, CORRECTION, turn LEFT heading 270..." The read-back protocol is safety-critical — CFIT (Controlled Flight Into Terrain) accidents have been traced to ambiguous ATC instructions with inadequate read-back verification.

## Signal Quality Scale (Radio Check)

```
Aviation (ICAO readability):
1 = Unreadable
2 = Readable now and then
3 = Readable but with difficulty
4 = Readable
5 = Perfectly readable

Military RST equivalent used:
READABILITY: 1–5 (as above)
SIGNAL STRENGTH: 1–5 (weak to strong)
→ "Radio check, over" → "Five by five" = perfect readability and strength

Common responses:
"Loud and clear" / "Five by five" = perfect
"Weak but readable" = audible but low power
"Broken" = intermittent, words being lost
"Unreadable" = cannot decode
"Scratchy / in the mud" = excessive noise
```

## Emergency Frequencies

```
Aviation distress frequencies:
— 121.5 MHz VHF — Guard (civil air emergency)
— 243.0 MHz UHF — Guard (military)
— 406 MHz — ELT (Emergency Locator Transmitter) digital
  (COSPAS-SARSAT satellite detection)

Maritime distress:
— VHF Channel 16 (156.8 MHz) — international calling/distress
— MF 2182 kHz — medium frequency maritime distress
— DSC (Digital Selective Calling) — automated distress alerts

Amateur radio emergency:
— 146.52 MHz VHF simplex calling (US)
— 14.300 MHz HF (Maritime Mobile Service Net)
— Net control contacts USCG on request
```

## Decision Cheat Sheet

| Situation | What to say |
|-----------|-------------|
| Spell out a letter | "Alfa", "Bravo", etc. |
| Say the number 9 | "Niner" |
| Acknowledge receipt | "Roger" |
| Acknowledge + will comply | "Wilco" |
| Ask to repeat | "Say again" (NOT "repeat") |
| End of over, expect reply | "Over" |
| End of all comms | "Out" |
| Ask for signal quality | "Radio check, how do you read, over?" |
| Report all good | "Loud and clear" or "Five by five" |
| Report distress at sea | "Mayday, Mayday, Mayday — [vessel name] — [position] — [nature of distress] — [persons aboard] — [other info] — Over" |

## Common Confusion Points

**"Roger" ≠ "Yes"**. Roger means "received your transmission." It says nothing about compliance or agreement. "Wilco" (will comply) is needed to confirm you'll act on an instruction. This is one of the most common misconceptions in media depictions of radio communication.

**"Over" and "Out" are never said together**. "Over" = I await your reply. "Out" = conversation ended. "Over and out" (a Hollywood staple) is contradictory — you can't simultaneously expect a reply and end the conversation.

**"Alfa" not "Alpha"**: The official ICAO spelling is "Alfa" (and "Juliett" with two t's). The spelling was deliberately modified from the English dictionary spelling so that the words wouldn't be "owned" by any single language's conventions. In practice, most people say "Alpha" and "Juliet" and it doesn't matter.

**"Lima" rhymes with "seema"**: The ICAO-correct pronunciation is "LEE-mah." The city in Peru is pronounced "LYE-mah." Aviation ATC and military use "LEE-mah" universally.

**"Repeat" in military = repeat the artillery/fire mission**: This is why "say again" replaced "repeat" for asking someone to retransmit. On a combined-arms net, "repeat" to a fire support coordinator means "fire the same mission again." Fatal misunderstanding possible.

**"Sécurité" and "Pan-Pan" are French**: Both originate from French maritime/aviation terminology. Sécurité (safety) warns of navigational hazards. Pan-Pan comes from French "panne" (breakdown). Mayday comes from "m'aidez" (help me). Three French origin words form the GMDSS urgency hierarchy.
