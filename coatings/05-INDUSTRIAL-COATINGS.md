# Industrial Coatings

## Powder Coating, Anodizing, Electroplating, Galvanizing, PVD

## The Big Picture

```
INDUSTRIAL SURFACE TREATMENT HIERARCHY:

  PURPOSE: Protect substrate OR achieve functional properties
           OR enhance appearance OR ALL THREE

  ┌────────────────────────────────────────────────────────────────┐
  │  ORGANIC COATINGS         │ Powder coat; epoxy; polyurethane  │
  │  (applied as film)        │ field-applied or factory-applied  │
  ├────────────────────────────────────────────────────────────────┤
  │  CONVERSION COATINGS      │ Anodizing; phosphate; chromate    │
  │  (substrate becomes coat) │ substrate surface CONVERTS to     │
  │                           │ protective oxide/compound         │
  ├────────────────────────────────────────────────────────────────┤
  │  ELECTROCHEMICAL          │ Electroplating; electroless Ni;   │
  │  DEPOSITION               │ electrogalvanizing                │
  ├────────────────────────────────────────────────────────────────┤
  │  HOT DIP / THERMAL        │ Hot-dip galvanizing; thermal spray│
  │  PROCESSES                │ zinc arc spray; plasma spray      │
  ├────────────────────────────────────────────────────────────────┤
  │  VAPOR DEPOSITION         │ PVD (sputtering/evaporation);     │
  │  (vacuum processes)       │ CVD; DLC                         │
  └────────────────────────────────────────────────────────────────┘

  KEY DIFFERENTIATOR: "Is it applied TO the surface or
  does the surface BECOME the coating?"
  Anodizing/phosphating = conversion (substrate IS the coating)
  Plating/powder = applied (separate material deposited)
```

## Powder Coating

```
  PROCESS:
  ┌──────────────────────────────────────────────────────────────┐
  │  STEP 1: SURFACE PREPARATION                                │
  │    Iron phosphate pretreatment (mild steel)                 │
  │    OR zinc phosphate (better corrosion resistance)          │
  │    Blast or chemical etch for aluminum                      │
  │    Critical: contamination removal (oil, oxide)             │
  │                                                              │
  │  STEP 2: POWDER APPLICATION                                 │
  │    Part grounded (earth/chassis ground)                     │
  │    Powder spray gun charges particles (corona or            │
  │    tribo-charging): 40,000-100,000 V corona charge          │
  │    Charged particles attracted to grounded part             │
  │    Uniform powder deposit; even inside corners and complex  │
  │    geometry (electrostatic lines wrap around)               │
  │                                                              │
  │  STEP 3: CURE IN OVEN                                       │
  │    Typically 350-400°F (177-204°C)                          │
  │    Duration: 15-30 minutes at temperature                   │
  │    Thermoset powder: crosslinks chemically → permanent      │
  │    Thermoplastic: melts → fuses → cools → no crosslink      │
  └──────────────────────────────────────────────────────────────┘

  POWDER CHEMISTRY:
  ┌─────────────────────────────────────────────────────────────────┐
  │  TYPE           │ CURE CHEMISTRY    │ PROPERTIES              │
  ├─────────────────────────────────────────────────────────────────┤
  │  Epoxy          │ Amine crosslink   │ Interior; excellent      │
  │                 │                   │ adhesion/hardness;       │
  │                 │                   │ chalks outdoors          │
  │  Polyester      │ TGIC or HAA       │ Exterior; UV stable     │
  │  (TGIC/HAA)     │ crosslinker       │ most common exterior    │
  │  Epoxy-polyester│ Hybrid            │ Better than pure epoxy  │
  │  (hybrid)       │                   │ outdoors; cost effective │
  │  Polyurethane   │ Isocyanate cure   │ Excellent flexibility;  │
  │                 │                   │ premium; low cure temp  │
  │  PVDF           │ Thermoplastic     │ Very high UV/weathering │
  │                 │ (Kynar)           │ High-end architectural  │
  └─────────────────────────────────────────────────────────────────┘

  ADVANTAGES:
    Essentially zero VOC during application (no solvent)
    Single coat = full film build (no multi-coat needed usually)
    Very uniform thickness (25-75 microns typical)
    Good edge coverage (electrostatic wrapping effect)
    Reclaim overspray (recycle unused powder)

  LIMITATIONS:
    Requires conductive substrate (grounding essential)
    Requires oven (min ~350°F): no field application possible
    Large structures: too large for most ovens
    Color change: cleaning spray gun between colors required
    Thin/heat-sensitive substrates: may warp or damage in oven
    Minimum practical thickness: ~1.5-2 mils (can't do ultra-thin)
```

## Anodizing

```
  SUBSTRATE: Aluminum only (titanium also anodizes; different process)

  WHAT ANODIZING IS:
    NOT a coating applied to aluminum
    The aluminum BECOMES aluminum oxide (Al₂O₃)

  ELECTROCHEMICAL PROCESS:
  ┌──────────────────────────────────────────────────────────────┐
  │  Electrolyte: sulfuric acid (Type II) or chromic acid (I)  │
  │  DC current: part = ANODE (+); tank = cathode (-)          │
  │                                                              │
  │  At anode surface: Al → Al³⁺ + 3e⁻                        │
  │  With water: Al³⁺ + 3OH⁻ → Al(OH)₃ → Al₂O₃               │
  │                                                              │
  │  Oxide grows BOTH outward from surface AND inward          │
  │  (roughly 50% penetrates into metal, 50% builds outward)  │
  │                                                              │
  │  Porous hexagonal structure with channels perpendicular    │
  │  to surface → open for dyeing before sealing               │
  └──────────────────────────────────────────────────────────────┘

  THREE TYPES:
  ┌────────────────────────────────────────────────────────────┐
  │  TYPE I: Chromic Acid Anodize                             │
  │    Thinnest: 0.0002"-0.0003" (0.5-0.8 μm)                │
  │    Minimal dimensional change                              │
  │    AEROSPACE: fatigue-sensitive parts (thinnest → minimal │
  │    stress concentration); MIL-A-8625                       │
  ├────────────────────────────────────────────────────────────┤
  │  TYPE II: Sulfuric Acid Anodize (standard)                │
  │    Thickness: 0.0002"-0.001" (0.5-25 μm)                 │
  │    Most common decorative and protective                   │
  │    Accepts color dyes; sealing after dyeing                │
  │    Architectural aluminum (AAMA 611 standards)             │
  ├────────────────────────────────────────────────────────────┤
  │  TYPE III: Hard Anodize (hardcoat)                        │
  │    Thickness: 0.001"-0.002" (25-50 μm, up to 100 μm)     │
  │    Very hard: 60-70 Rockwell C equivalent                 │
  │    Very abrasion resistant                                 │
  │    Requires cold electrolyte (near freezing) + higher current│
  │    Uses: piston skirts, firearms, cookware, sliding parts  │
  └────────────────────────────────────────────────────────────┘

  COLORING:
    Organic dyes absorbed into pores before sealing
    → Wide color range (black, red, blue, gold, green...)
    Electrolytic coloring: metal salts (tin, cobalt) → metallic
    shades; more lightfast than organic dyes
    Sealing: hot water or nickel acetate solution closes pores
    → prevents color migration; improves corrosion resistance
    Natural anodize (no dye): clear-to-gray appearance

  DIMENSIONING:
    Type II builds 50% outward / 50% inward
    Design critical parts: anodize adds 0.00025"-0.0005" to each
    surface; tolerance must account for this
    This is why precision machined parts have anodize noted on drawings
```

## Electroplating

```
  MECHANISM: ELECTROLYTIC METAL DEPOSITION

  ┌──────────────────────────────────────────────────────────────┐
  │  Metal ion solution (electrolyte)                           │
  │  DC current: work part = CATHODE (-); anode = plating metal │
  │                                                              │
  │  At cathode: M^n+ + ne⁻ → M (metal deposits on part)       │
  │  At anode: M → M^n+ + ne⁻ (or inert anode: O₂ evolution)  │
  │                                                              │
  │  Plating thickness proportional to current × time          │
  │  (Faraday's laws of electrolysis)                          │
  └──────────────────────────────────────────────────────────────┘

  COMMON PLATING METALS:
  ┌─────────────────────────────────────────────────────────────────┐
  │  METAL  │ PURPOSE                                │ KEY NOTES   │
  ├─────────────────────────────────────────────────────────────────┤
  │  Zinc   │ Corrosion protection (galvanic)        │ Sacrificial │
  │  Chrome │ Decorative + corrosion (hex chrome)   │ Cr⁶⁺ toxic! │
  │         │ Hard chrome: wear resistance          │ OSHA regulated│
  │  Nickel │ Intermediate layer under chrome       │ Hardness     │
  │         │ Corrosion resistance alone            │ Barrier coat │
  │  Gold   │ Electronics contacts (non-tarnishing) │ Expensive    │
  │         │ Jewelry (thin layer)                  │              │
  │  Silver │ Electronics (conductivity)            │ Tarnishes    │
  │         │ Flatware, decorative                  │              │
  │  Copper │ Electronics (PCB through-holes)       │ Adhesion base│
  │         │ Pre-plate layer                        │              │
  │  Tin    │ Food cans (no toxicity)               │ Solderable   │
  │         │ Electronics (solder adhesion)         │              │
  └─────────────────────────────────────────────────────────────────┘

  CHROME PLATING ENVIRONMENTAL ISSUES:
    Decorative chrome: typically Ni + thin Cr
    Hard chrome: thick Cr for wear resistance
    Traditional bath: hexavalent chromium (Cr⁶⁺) — highly toxic,
    carcinogenic, regulated under EPA, OSHA
    Industry transition: trivalent chrome (Cr³⁺) baths for decorative
    (lower toxicity); hard chrome alternatives harder to replace
    California Prop 65; OSHA 1910.1026 (Cr⁶⁺ standard)
    EU REACH restriction on Cr⁶⁺ use

  THICKNESS DISTRIBUTION ISSUE:
    Electrolytic plating deposits more at high-current-density areas
    (corners, edges, protrusions) than recessed areas
    → Uneven thickness on complex geometry
    → Solution: electroless plating (see below)
```

### Electroless Plating

```
  AUTOCATALYTIC PROCESS: no electricity required
  Chemical reducing agent in bath drives metal deposition

  ELECTROLESS NICKEL:
    Most common electroless process
    Hypophosphite reducing agent → deposits Ni-P alloy
    CRITICAL ADVANTAGE: follows geometry exactly
    Identical thickness in recessed areas as on flat surfaces
    → Essential for: threaded fasteners, holes, complex internal shapes

  PROPERTIES:
    Ni-P alloy: ~10% phosphorus typical
    Hardness: ~50 Rockwell C (increases to ~70 RC after heat treat)
    Corrosion resistance: excellent (uniform protective layer)
    Hardness uniformity: same everywhere on part
    High phosphorus (>10%): better corrosion resistance
    Low phosphorus (<5%): more magnetic; harder
    Mid phosphorus: general purpose

  COMPOSITE ELECTROLESS:
    Co-deposited particles in electroless Ni bath
    PTFE-Ni: "teflon" release/anti-friction coating (cookware,
             mechanisms requiring low friction)
    Diamond-Ni: extreme hardness
    SiC-Ni: wear resistance

  APPLICATIONS:
    Precision hydraulic components (complex internal passages)
    Molds and tooling (uniform release surface on complex forms)
    Electronics (uniform plating on PCB features)
    Cookware: PTFE composite "Teflon-nickel" hard coating
```

## Hot-Dip Galvanizing

```
  PROCESS:
  1. CLEANING: pickle in hydrochloric acid → remove scale/rust
  2. FLUX: ammonium zinc chloride → prevents re-oxidation
  3. GALVANIZING: immerse in molten zinc at ~840°F (449°C)
  4. QUENCH or air cool

  WHY IT WORKS (METALLURGICAL BOND):
  ┌──────────────────────────────────────────────────────────────┐
  │  At steel/zinc interface: intermetallic layers form          │
  │                                                              │
  │  From steel outward:                                         │
  │  Steel → Gamma (FeZn₃) → Delta (FeZn₇) → Zeta (FeZn₁₃)    │
  │         → Eta (pure zinc outer layer)                       │
  │                                                              │
  │  These are actual iron-zinc alloy phases, NOT just zinc     │
  │  stuck to steel surface.                                     │
  │  Result: coating that cannot peel; metallurgically bonded   │
  └──────────────────────────────────────────────────────────────┘

  GALVANIC PROTECTION:
    Zinc is anodic to steel (higher in electromotive series)
    When both present in electrolyte: zinc sacrifices first
    → Steel protected as long as zinc is present
    → Even at cut edge or small exposed steel area:
      zinc continues to protect surrounding steel
      (unlike paint: scratch → steel immediately exposed)

  COATING THICKNESS:
    ASTM A123 (structural steel): minimum 3.9 mils (100 μm)
    ASTM A153 (fasteners): varies by thickness of part
    Corrosion life estimate: 1 mil zinc ≈ 50 years in rural,
    15 years in industrial, 10 years in marine environment

  HOW IT'S APPLIED:
    Entire part must fit in galvanizing kettle (tank of molten zinc)
    Large structural steel: up to 60 ft long kettles exist
    Very large components: double-dipping (one end, then other)
    Complex assemblies: drainage and vent holes critical (zinc
    must drain from cavities; trapped zinc or air = problems)

  AFTER-FABRICATION:
    Galvanize AFTER all cutting, drilling, welding
    Welding on galvanized: generates zinc fumes (toxic → metal
    fume fever); must grind galvanizing away before welding,
    then re-coat with cold galvanizing compound
```

## PVD (Physical Vapor Deposition)

```
  VACUUM PROCESS:
    Chamber pumped to 10⁻⁶ torr (high vacuum)
    Target material vaporized by:
      Electron beam (e-beam evaporation)
      Magnetron sputtering (Ar plasma bombardment of target)
      Thermal evaporation (resistive heating)
    Vapor travels to substrate → deposits as thin film
    Film thickness: nanometers to a few microns

  COMMON PVD MATERIALS:
  ┌──────────────────────────────────────────────────────────────┐
  │  MATERIAL  │ COLOR     │ HARDNESS │ APPLICATION             │
  ├──────────────────────────────────────────────────────────────┤
  │  TiN       │ Gold      │ ~2000 HV │ Drill bits, cutting     │
  │  (Titanium │           │          │ tools, architectural     │
  │  nitride)  │           │          │ hardware finishes        │
  │  TiAlN     │ Gray-black│ ~2800 HV │ High-speed dry machining│
  │  CrN       │ Silver    │ ~1600 HV │ Automotive, forming     │
  │  DLC       │ Black     │ ~2500 HV │ Medical devices,        │
  │  (diamond- │           │          │ bearings, luxury watches │
  │  like C)   │           │          │                          │
  │  ZrN       │ Gold      │ ~2200 HV │ Watchmaking, decorative │
  └──────────────────────────────────────────────────────────────┘

  WHY PVD FINISHES DON'T TARNISH:
    TiN, CrN, ZrN are ceramic compounds — chemically very stable
    Will not oxidize, tarnish, or react with normal environments
    Surface hardness much greater than underlying metal
    → Scratch resistant
    "PVD brass" faucet: actually steel or zinc with TiN or ZrN film
    Dramatically more durable than electroplated finishes

  CUTTING TOOL COATINGS:
    TiN: first commercial cutting tool PVD coating (1970s)
    TiAlN: higher hot hardness → dry machining at higher speeds
    Multi-layer coatings: alternating nanolayers → combined properties
    Modern carbide insert: 50% of value is in the PVD coating
    Coated drill bit vs uncoated: 3-5× life increase minimum

  CVD (Chemical Vapor Deposition):
    Gaseous precursors react at substrate surface to deposit film
    Higher temperature than PVD (may distort precision parts)
    Thicker coatings achievable; better for complex shapes
    Al₂O₃ (alumina) for high-temp stability: CVD only
    Diamond CVD coatings on tungsten carbide tools
    Primarily semiconductor and high-performance tooling applications
```

## Thermal Spray

```
  PRINCIPLE:
    Feedstock (powder, wire, or rod)
    → Heated to molten/semi-molten state
    → Propelled at high velocity toward surface
    → Impacts → flattens (splats) → bonds mechanically
    → Successive splats build coating

  THERMAL SPRAY PROCESSES:
  ┌──────────────────────────────────────────────────────────────┐
  │  FLAME SPRAY: Combustion of fuel gas + oxygen              │
  │    Moderate velocity; moderate cost; metals + ceramics     │
  │                                                              │
  │  ARC SPRAY: Electric arc between two wire electrodes       │
  │    High productivity; zinc, aluminum, steel wire           │
  │    Corrosion protection on large structures                 │
  │    (bridges, offshore, storage tanks)                       │
  │                                                              │
  │  PLASMA SPRAY: Plasma torch (Ar/He/H₂ at ~15,000°C)       │
  │    Highest temperatures → ceramics, refractory materials   │
  │    Thermal barrier coatings (TBC) on turbine blades:       │
  │    YSZ (yttria-stabilized zirconia) — low thermal          │
  │    conductivity at high temperature                         │
  │                                                              │
  │  HVOF (High Velocity Oxy-Fuel):                            │
  │    Supersonic particle velocity; dense, hard deposit       │
  │    WC-Co (tungsten carbide-cobalt): extreme wear resistance │
  │    Replacing hard chrome plating for wear applications     │
  └──────────────────────────────────────────────────────────────┘

  TURBINE BLADE TBC (Thermal Barrier Coating):
    Substrate: Ni superalloy blade
    Bond coat: MCrAlY (Ni/Co base + Cr + Al + Y) → oxidation
               resistance + bond for ceramic top coat
    Top coat: 7-8% YSZ (yttria-stabilized zirconia)
              Low thermal conductivity (~2.0 W/m·K vs Ni: ~12 W/m·K)
              Allows gas turbine inlet temp 100-200°C above alloy
              melting point → higher efficiency
    Failure mode: thermally grown oxide (TGO) grows at bond coat
    interface → eventually spalls the ceramic top coat
```

## Passivation

```
  PASSIVATION = forming protective oxide layer by controlled
                chemical treatment

  STAINLESS STEEL PASSIVATION:
    Stainless steel naturally passivates (Cr₂O₃ forms spontaneously)
    Problem: fabrication (machining, welding) embeds free iron
    particles in surface → can corrode → brown rust stains despite
    "stainless" base metal

    PASSIVATION TREATMENT:
      Nitric acid or citric acid bath
      → Dissolves free iron from surface
      → Does NOT remove material from bulk stainless
      → Chromium oxide layer reforms more completely
      → Surface now truly stainless

    STANDARDS:
      ASTM A967: Passivation specifications and test methods
      QQ-P-35C (MIL spec, superseded): basis of ASTM
      Acceptance test: copper sulfate test; salt spray (ASTM B117)

  MEDICAL AND FOOD PROCESSING:
    All stainless components in medical devices, food contact,
    pharmaceutical should be passivated
    FDA requirement for food-grade stainless: passivated
    Implantable devices: passivation prevents iron leaching

  TITANIUM:
    Passivates naturally and extremely well (TiO₂)
    Essentially immune to corrosion in biological environments
    → Medical implants (bone screws, joints)
    No additional passivation treatment needed
```

## Decision Cheat Sheet

| Application | Best industrial coating |
|---|---|
| Bicycle frame, fence, aluminum window | Powder coat (polyester TGIC — exterior UV stable) |
| Structural steel outdoors | Hot-dip galvanize (ASTM A123) |
| Faucet finish that won't tarnish | PVD TiN or ZrN (gold or silver appearance) |
| Complex geometry, uniform hardness | Electroless nickel |
| Drill bits, machining tools | PVD TiAlN or TiN coating |
| Turbine blade thermal protection | Plasma spray YSZ TBC |
| Large structure corrosion (bridge) | Zinc arc thermal spray |
| Aluminum precision part | Type III hard anodize |
| Stainless steel medical device | Passivate per ASTM A967 |
| Cookware non-stick hard surface | Electroless Ni-PTFE composite |

## Common Confusion Points

**Anodizing is NOT a coating — the aluminum becomes the oxide**: This is the critical distinction. Anodizing grows the aluminum oxide from the aluminum itself. You can't scrape or peel an anodize coating as you can with paint. The surface IS aluminum oxide. This is why it has fundamentally different properties (hardness, permanence) compared to a deposited coating.

**Hot-dip galvanizing and electrogalvanizing are very different thicknesses**: Hot-dip galvanize (ASTM A123) typically deposits 3-6 mils of zinc. Electrogalvanizing (ASTM A591) deposits 0.1-0.4 mils — 10-30× thinner. Hot-dip provides decades of protection; electrogalvanizing is intended to be painted over for additional protection. Do not specify electrogalvanized where hot-dip thickness is needed.

**Chrome plating and chromate conversion coating are completely different**: Chrome plating (electroplating Cr from a bath) deposits a metal layer for hardness/appearance. Chromate conversion (chromating, Alodine, Iridite) is a chemical conversion coating on aluminum for corrosion resistance and paint adhesion — thin, not metallic-looking. Don't use these terms interchangeably.

**Powder coating requires oven curing**: Field application of powder coat is not possible (requires grounding and curing oven). For structural steel that's been assembled or large in-place structures, liquid zinc-rich paint or thermal spray zinc is the alternative. Powder coat is a factory process.

**Semiconductor thin-film bridge:** PVD (Physical Vapor Deposition) and CVD (Chemical Vapor Deposition) for industrial coatings are the same unit processes as semiconductor thin-film deposition — same vacuum chambers, same plasma physics, same deposition mechanisms, different scale and substrate. In semiconductor fab: TiN and TaN are PVD-deposited as diffusion barriers in interconnect stacks; SiO₂ and Si₃N₄ are CVD-deposited as dielectrics and passivation. In industrial coatings: TiN is PVD-deposited on cutting tools (same golden color, same mechanism); DLC (diamond-like carbon) is CVD on engine components; CrN is PVD on forming tools. The distinction that matters: PVD is line-of-sight (can shadow complex geometries), CVD is conformal (follows surface contours). On semiconductor wafers, CVD is preferred precisely because conformality is required in high-aspect-ratio features. On cutting tools, PVD dominates because the simpler geometry is acceptable and lower deposition temperature protects the steel substrate.

**PVD coatings on faucets are far more durable than lacquered finishes**: "Oil-rubbed bronze" in a plumbing supply store is usually lacquered brass. PVD oil-rubbed bronze is the same color but wears for decades vs years. The price premium is real but warranted for high-use fixtures.
