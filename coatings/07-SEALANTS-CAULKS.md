## Silicone, Polyurethane, Acrylic Latex — Movement, Compatibility, Durability

## The Big Picture: Sealant Taxonomy

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                     SEALANT / CAULK LANDSCAPE                               │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  FIRST SPLIT: Static joints vs. Dynamic joints                      │   │
│  │                                                                     │   │
│  │  CAULK  ──── fills static gaps (trim to wall, no movement)         │   │
│  │  SEALANT ─── accommodates movement (expansion joints, envelope)    │   │
│  │                                                                     │   │
│  │  "Caulk" and "sealant" are used interchangeably in trade           │   │
│  │  but sealants have a rated movement accommodation factor (MAF)     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  SILICONE          POLYURETHANE        ACRYLIC LATEX       BUTYL RUBBER    │
│  Most flexible     Paintable           Paintable           Tacky/roofing   │
│  UV/temp extreme   Masonry/concrete    Interior trim       HVAC tape       │
│  Not paintable     Horizontal joints   Lowest cost         Not paintable   │
│  ±50% MAF          ±25% MAF            ±12.5% MAF          ±12.5% MAF     │
│                                                                             │
│  SMP HYBRIDS ─────── Silicone flexibility + polyurethane paintability      │
│  INTUMESCENT ──────── Fire-rated through-penetrations                      │
│  EPDM TAPE ────────── Pre-formed gaskets, roofing, windows                │
└─────────────────────────────────────────────────────────────────────────────┘

Movement Accommodation Factor (MAF) — ASTM C719:
  ±12.5%  — 1/4" joint moves ±1/32"   — Acrylic, butyl
  ±25%    — 1/4" joint moves ±1/16"   — Polyurethane
  ±50%    — 1/4" joint moves ±1/8"    — Silicone, SMP
```

## 1. Silicone Sealant

### Chemistry

Polydimethylsiloxane (PDMS) backbone — Si-O-Si-O repeating chain. The Si-O bond is
fundamentally different from carbon-based polymers: stronger, more flexible at low
temperature, stable at high temperature, UV-resistant (silicon doesn't absorb UV).

```
    CH₃   CH₃   CH₃
     |     |     |
─ O─Si─O─Si─O─Si─O ─
     |     |     |
    CH₃   CH₃   CH₃

   Polydimethylsiloxane (PDMS) — the silicone backbone
   Si-O bond: ~110 kcal/mol (strong) | Si-C bond: ~76 kcal/mol
   C-C bond: ~83 kcal/mol (for comparison)
```

### Two Cure Systems

```
┌─────────────────────────────────────────────────────────────────────────┐
│ ACETOXY CURE                    │  NEUTRAL CURE                        │
│ (most common, "acidic")         │  (oxime or alkoxy)                   │
├─────────────────────────────────┼──────────────────────────────────────┤
│ Byproduct: acetic acid          │  Byproduct: oxime or alcohol         │
│  → vinegar smell during cure    │   → mild or no odor                  │
│ Fast cure                       │  Similar cure speed                  │
│ Excellent adhesion to glass     │  Use on:                             │
│  tile, ceramic, painted walls   │   - Metals (acid corrodes)           │
│ DO NOT USE ON:                  │   - Concrete/masonry (carbonation)   │
│  - Bare metals (acid corrodes)  │   - Electronics                      │
│  - Natural stone (discolors)    │   - Cementite stone                  │
│  - Concrete (surface damage)    │   - Anodized aluminum                │
│  - Electronics                  │  Higher cost                         │
│ GE Silicone I, DAP All-Purpose  │  GE Silicone II, Dow 795             │
└─────────────────────────────────┴──────────────────────────────────────┘
```

### Properties and Applications

| Property | Value |
|----------|-------|
| Temperature range | −65°F to 400°F (−54°C to 204°C) |
| Movement accommodation | ±50% (ASTM C719 Class 50) |
| UV resistance | Excellent — no degradation |
| Paintability | **Cannot be painted** — topcoat beads off |
| Adhesion | Glass, metal (neutral cure), ceramic, plastic |
| Service life | 20+ years |
| Elongation at break | 300-600% |

**Specialty formulations:**
- **Mold-resistant**: antimicrobial additives; bathroom/kitchen use; GE Mold-Free
- **Structural glazing (SSG)**: Dow 995, GE SSG4000; engineered for curtain walls;
  "Two-sided" vs "four-sided" structural systems; tested to ASTM C1184/C1401
- **Fire-rated**: intumescent silicone; expands and chars to block flame/smoke
- **High-temp**: RTV silicone (Room Temperature Vulcanizing); automotive, oven doors

## 2. Polyurethane Sealant

### Chemistry

Isocyanate (-N=C=O) + atmospheric moisture → carbamic acid → CO₂ + amine → further
reaction with remaining isocyanate → crosslinked polyurea/polyurethane network.

```
   R─N=C=O  +  H₂O  →  R─NH₂  +  CO₂  →  crosslinked urethane network
   isocyanate    moisture   amine

   Crosslink density → modulus (stiffness)
   Higher modulus = stiffer sealant = less flexible but more abrasion-resistant
```

### Properties and Applications

| Property | Polyurethane | vs. Silicone |
|----------|-------------|--------------|
| Movement accommodation | ±25% (some to ±35%) | ±50% |
| Paintability | **Yes** — excellent | No |
| Adhesion to concrete/masonry | Excellent | Moderate |
| Adhesion to wood | Excellent | Poor |
| Abrasion resistance | Excellent | Moderate |
| UV resistance | Moderate (yellows) | Excellent |
| Low-temp flexibility | Good | Better |
| Service life | 10-15 years | 20+ years |
| Skin time | 30-60 min | 10-20 min |

**Best applications:**
- Concrete construction joints (horizontal — parking decks, driveways)
- Window/door perimeter on stucco, EIFS, CMU
- Wood-to-masonry transitions
- Any joint that will be painted
- Marine: above waterline (Sika 291, Boat Life LifeCaulk)

**1-part vs. 2-part polyurethane sealant:**

```
  1-PART:
  - Moisture cure, single cartridge
  - Slower cure (varies with humidity)
  - DIY/trade standard
  - Sikaflex 1a, Tremco Dymeric 240

  2-PART:
  - Base + accelerator, hand mixed
  - Faster cure, predictable schedule
  - Better performance in cold or dry conditions
  - Tremco Spectrem, Sika 2c NS
  - Commercial/industrial applications
```

## 3. Acrylic Latex Caulk

The workhorse residential caulk. Water-based, water cleanup, inexpensive, paintable.

### Chemistry and Limitations

Film formation by water evaporation (not crosslinking). The polymer is already formed —
you're just depositing it as water leaves. Consequences:
- **Shrinks** as water evaporates — expect 10-15% volume reduction
- **Cannot accommodate movement** — the deposited film has no elastomeric properties
- **Does not flex** — cracks if substrate moves

```
  Wet → Dry:  Acrylic emulsion → water evaporates → polymer chain entanglement
  No crosslinks, no chemical cure, no elasticity

  Acceptable for:    ✓ Interior trim (baseboards, crown molding, window trim)
                     ✓ Filling small static gaps before painting
                     ✓ Smoothing transitions before topcoat

  Not acceptable for: ✗ Wet areas (shower, tub — use silicone)
                      ✗ Any joint subject to seasonal movement
                      ✗ Exterior applications with thermal cycling
                      ✗ Gaps >3/8" (shrinkage too great)
```

### Product Variants

| Type | Silicone % | Waterproof | Paintable | Best For |
|------|-----------|------------|-----------|----------|
| Painters caulk | 0% | No | Yes | Interior trim, paint-before-caulk |
| Siliconized acrylic | 2-15% | Slightly | Yes | Interior/limited exterior |
| "All-purpose" | Variable | Moderate | Yes | General use |

The "siliconized acrylic" marketing claim is often oversold — the silicone content is
minimal and doesn't confer true silicone performance. Flexibility improvement is real
but modest.

## 4. Butyl Rubber Caulk

Saturated synthetic rubber (isobutylene-isoprene). Vulcanized or tape form.

**Key properties:**
- Extremely tacky and adhesive to almost any surface
- Doesn't dry/cure — remains permanently soft and conforming
- Excellent adhesion to aluminum, EPDM rubber, asphalt, glass
- Not paintable — remains tacky
- Hard to tool smooth (sticks to everything including tools)

**Applications:**
- Roofing: sealing flashing, vent boots, metal seams
- HVAC: aluminum-backed butyl tape = standard for rigid duct sealing
  (not duct tape — silver fabric duct tape fails within years)
- Glazing: bedding compound for window glass
- Pre-formed gaskets, tape products: EPDM/butyl laps in roofing

## 5. SMP Hybrid Sealants

Silyl-Modified Polymer (SMP) — also called MS polymer. Engineered to combine best
properties of silicone and polyurethane.

```
  POLYURETHANE  ──────────────  SMP HYBRID  ──────────────  SILICONE

  Paintable ✓                  Paintable ✓              Not paintable ✗
  Isocyanate ✗                 No isocyanate ✓          No isocyanate ✓
  ±25% movement                ±35-50% movement          ±50% movement
  Moderate UV                  Good UV                   Excellent UV
  Moderate multi-substrate     Excellent multi-substrate Good multi-substrate
```

**Chemistry**: Polyether or polyurethane backbone terminated with alkoxysilane groups.
Moisture-cure (like both silicone and PU), but releases alcohol (safe byproduct).
No isocyanates — safer for workers in enclosed spaces.

**Products**: OSI Quad Max, Gorilla Clear Silicone 2, GE Advanced, Loctite PL Premium
Advanced. Premium price tier — roughly 2-3× acrylic, competitive with polyurethane.

**Use when**: You need paintability AND high movement AND multi-substrate adhesion.
The "one product for everything" ideal.

## 6. EPDM Tape and Pre-Formed Gaskets

Non-cured rubber — no mixing, no cure time, compression seals.

  - Temperature range: -40°F to 250°F
  - Excellent weathering

  Forms:
  ┌─────────────────────────────────────────────────────────┐
  │ Self-adhering tape    │ Butyl adhesive backing          │
  │ (w/ butyl backing)   │ Exposed EPDM face               │

## 7. Fire-Rated Caulk (Intumescent)

Code requirement, not optional. Every penetration through a fire-rated assembly must
be firestopped. A fire-rated wall with unfirestopped conduits is no longer rated.

### How Intumescent Works

```
  Normal: compressed acrylic or silicone sealant fills gap

  Under fire (≈300°F+):

  Intumescent agent (graphite, expandable compounds) activates
              ↓
  Material expands 3-10× original volume
              ↓
  Expansion compresses pipe/conduit, blocks opening
              ↓
  Char layer forms — insulating, stops heat transfer

  Net: fire and smoke cannot pass through penetration
```

### UL System Listings

Critical concept: firestop products are **not self-certifying**. You must use a
**UL-listed system** — specific product + specific assembly configuration. The UL
listing number covers:
- What product (3M Fire Barrier 2000+, Hilti CP 606, Specified Air FP201)
- What wall/floor type (2-hour rated GWB, concrete, CMU)
- What penetrant (rigid pipe, flexible conduit, cables, cable bundles)
- What pipe material (steel, CPVC, PVC)
- Maximum pipe/conduit size
- Required application thickness and details

**Products**: 3M Fire Barrier series, Hilti CP series, Specified Air, STI products.
For PVC pipe in fire-rated walls: PVC pipe expands/collapses in fire — requires
intumescent collar or sleeve that clamps pipe, collapses it as pipe burns away.

## 8. Joint Design and Application

### Geometry: The 2:1 Width-to-Depth Rule

```
  WRONG — backer rod missing, too deep:

  ══════════════════════════
  |          gap           |
  |                        |
  |      deep fill         |  ← hourglass deformation on movement
  |     poor geometry      |     causes adhesion failure
  |                        |
  ══════════════════════════

  CORRECT — backer rod controls depth:

  ══════════════════════════
  |        sealant         |  ← Width = 2×, Depth = 1×
  |    [BACKER ROD]        |  ← Three-sided adhesion prevented
  ══════════════════════════

  Backer rod (closed-cell polyethylene foam):
  - Controls depth (width:depth ≈ 2:1)
  - Provides backing for tooling
  - Prevents "three-sided adhesion"
    (sealant adhering to back of joint = no freedom to stretch)

  For joints >1/2" wide: backer rod is not optional
```

### Application Protocol

```
  1. SURFACE PREP
     - Dry, clean, oil-free
     - Remove old failed sealant (razor, sealant remover)
     - Wipe with isopropyl alcohol (especially for silicone)
     - Prime per product TDS if required

  2. MASK
     - Blue painter's tape on both sides for clean lines
     - Remove tape while sealant is still wet/uncured

  3. APPLY
     - Cut nozzle at 45°, match bead diameter to joint width
     - Steady consistent pressure, maintain contact
     - Pull gun (don't push) for better joint filling

  4. TOOL
     - Smooth with wet finger, caulk tool, or popsicle stick
     - Concave profile: pressed into the joint (not peaked)
       (concave profile stays in compression during movement)
     - Remove masking tape immediately after tooling

  5. CURE
     - Follow product cure time before water exposure
     - Typical: acrylic 24h, silicone 24-48h (skin 30min),
       polyurethane 48-72h, SMP 24-48h
```

## Decision Cheat Sheet

| Situation | Use | Why |
|-----------|-----|-----|
| Shower/tub surround | Silicone (neutral cure on metal) | Wet, permanent, movement |
| Interior painted trim | Acrylic latex ("painters caulk") | Paintable, static joint |
| Window perimeter on stucco | Polyurethane or SMP | Paintable, ±25% movement |
| Expansion joint, concrete | Polyurethane (ASTM C920 Class 25) | Abrasion, ±25% |
| Multi-substrate, need paint | SMP hybrid | Best of both |
| Aluminum curtain wall glass | Structural silicone (neutral cure) | UV, temp, flexibility |
| Fire-rated wall penetration | UL-listed intumescent system | Code requirement |
| HVAC rigid duct seams | Aluminum-backed butyl tape | Pressure, durability |
| Metal flashing to siding | Polyurethane or SMP | Paintable, adhesion |
| Aquarium | Acetoxy silicone | Fish-safe after cure |
| Bare copper/brass | Neutral cure silicone only | Acetoxy acid corrodes |

## Common Confusion Points

**"Can I paint over caulk?"**
Acrylic latex: yes, paint right away. Polyurethane: yes, after full cure. Silicone:
no — paint won't bond to cured silicone. If you painted over silicone and paint is
peeling, you'll need to remove the silicone.

**"Will silicone adhere to old silicone?"**
Fresh silicone bonds to fully cured silicone with a clean surface. Prep: clean with
isopropyl alcohol, let dry. Some manufacturers sell silicone primer/activator to
improve adhesion to aged silicone. Remove failed sealant completely — don't just
apply fresh sealant over top.

**"Mold on caulk — antimicrobial or bleach?"**
Surface mold: bleach solution. Mold inside/behind: remove sealant, treat substrate,
replace with mold-resistant silicone. Standard caulk mold growth = replace, not clean.

**"Duct tape for HVAC ducts?"**
Silver fabric "duct tape" (cloth-backed PSA) fails within 5-10 years from heat cycling.
HVAC code (IMC) prohibits it in most applications. Use: aluminum-backed butyl tape or
mastic (brush-applied duct sealant + fiberglass mesh for large gaps). The name "duct
tape" is a marketing artifact — it was originally used elsewhere, not on ducts.

**"Backer rod — open cell or closed cell?"**
Closed-cell polyethylene backer rod is standard for sealant joints — it doesn't
absorb the sealant and doesn't off-gas. Open-cell foam is used in specific applications
only (some structural glazing specs). Default to closed-cell.

**MAF vs. elongation:**
MAF (movement accommodation factor) is the joint movement relative to joint width —
a ±25% MAF means a 1" wide joint can move ±1/4". Elongation-at-break is a material
property — they're related but not the same. A sealant might elongate 400% but only
be rated ±25% MAF because the joint geometry and adhesion also factor in.
