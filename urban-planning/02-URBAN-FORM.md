# Urban Form and Morphology

## The Big Picture

Urban morphology is the study of city form — how streets, plots, and buildings combine to create the physical fabric of cities. Form shapes behavior: where people walk, whether retail survives, whether streets feel safe, how heat builds up. Understanding form is understanding the hidden geometry that produces urban life.

## Urban Morphology: Analytical Frameworks

### The Conzen School (British Morphology)

M.R.G. Conzen (1960, study of Alnwick, Northumberland) established the foundational framework for reading city form analytically.

```
CONZEN'S THREE FORM COMPLEXES:

  1. TOWN PLAN (Street + Plot):
     The most durable layer. Streets and plot
     boundaries persist for centuries. Medieval
     street patterns survive in European cities
     that have been rebuilt many times.

  2. BUILDING FABRIC:
     Buildings are built on plots within
     the street network. They are replaced
     over time, but plot boundaries constrain
     what can be built. A narrow medieval
     plot produces a narrow building even
     when rebuilt in the 20th century.

  3. LAND USE PATTERN:
     The most volatile. Shifts rapidly with
     economic conditions. A street that was
     residential becomes commercial in a decade.

FRINGE BELT THEORY:
  At the edge of the built-up area,
  institutions that need space but cannot
  afford central land (hospitals, parks,
  universities, cemeteries, allotments)
  establish in the fringe.
  As the city grows outward, these fringe
  belts become fixed nodes within the
  urban fabric -- they resist redevelopment
  because of their institutional character.
  Examples: MIT in Cambridge MA (fringe belt
  of 19th-century Boston), Hampstead Heath
  in London.
```

### Typomorphology (Italian School)

Gianfranco Caniggia and Saverio Muratori developed a building-type analysis approach.

```
TYPOMORPHOLOGY:

  Building types are not arbitrary -- they
  are responses to the plot size, street type,
  climate, and cultural practice of their era.

  TYPE SERIES:
  Each building type generates the next
  through transformation rules.

  Row house (narrow plot, shared wall)
    --> Apartment building (multiple units,
        same plot logic)
      --> Perimeter block (multiple row houses
          aggregated around a courtyard)
        --> Tower (same area, vertical stack)

  The type embeds the plot in its dimensions.
  A Manhattan brownstone is dimensioned for
  its 25ft-wide lot. Its replacement must
  work within the same constraint unless
  lots are assembled.

  LOT ASSEMBLY:
  The fundamental mechanism for replacing
  fine-grained urban fabric with coarse-grained.
  Assembling many narrow lots into a single
  large parcel enables tower-in-park typology.
  Loss of grain = loss of diversity.
```

## Block Structure: The Fundamental Unit

Block size and configuration drive urban character more than any other single variable.

```
BLOCK SIZE COMPARISON:

  PORTLAND, OREGON: 200ft x 200ft
  +------+------+------+------+
  |      |      |      |      |  Each block:
  | 200ft|      |      |      |  40,000 sq ft
  +------+------+------+------|  Many intersections
  |      |      |      |      |  per mile
  +------+------+------+------+
  High intersection density = many route
  choices = more pedestrian traffic
  distributed across network.

  TYPICAL US BLOCK: 400ft x 200ft
  +------------------------------------------+
  |                                          |  Each block:
  |                 400ft                    |  80,000 sq ft
  +------------------------------------------+
  Standard American subdivision block.

  SUPERBLOCK: 600ft-1200ft
  +--------------------------------------------------------------------------+
  |                                                                          |
  |                           1000ft                                         |
  |                                                                          |
  +--------------------------------------------------------------------------+
  Modernist housing project, suburban
  campus, mall. Few entry points.
  Pedestrian hostile. Internal roads
  are private, not public ROW.

WHAT BLOCK SIZE CONTROLS:

  Short blocks --> many corners
  Corners --> highest pedestrian traffic
  Corners --> highest retail rent
  Corners --> most viable retail locations

  Long blocks --> few corners
  Midblock --> low pedestrian traffic
  Midblock --> retail "dead zones"
  Result: Strip mall pattern; retail
  clusters only at major intersections.
```

### Street Network Typologies

```
GRID (Manhattan, Portland, Salt Lake City):
  +---+---+---+---+
  |   |   |   |   |
  +---+---+---+---+
  |   |   |   |   |
  +---+---+---+---+
  + Equal treatment of all directions
  + Legible, predictable
  + Efficient for pedestrians
  + Redundant (multiple routes)
  - Not optimized for radial travel to center
  - Ignores terrain

RADIAL (Paris, Washington DC, Moscow):
       |
  \    |    /
   \   |   /
    \  |  /
     \ | /
      \|/
       O (city center)
      /|\
     / | \
    /  |  \
  + Fast travel to/from center
  + Grand boulevards (Haussmann)
  - Peripheral travel is circuitous
  - Prone to bottleneck at center

ORGANIC / MEDIEVAL (Bologna, Siena):
  ~~~~~~~~~~~~~~~~~~~~~
  Follows terrain, water,
  ~~~~~  historical paths  ~~~~~
  Evolved from desire paths.
  + Adapted to topography
  + Legible to inhabitants
  - Confusing to visitors
  - Traffic congestion on historic routes

CURVILINEAR SUBURBAN (Radburn, Levittown, most US suburbs):
  Curving cul-de-sac streets. Every route
  converges on a collector, which feeds
  an arterial. No alternatives.
  + Quiet residential streets
  + Low through-traffic
  - Dead ends for pedestrians
  - All trips funnel to arterials
  - No transit viability
  - Emergency access vulnerability

HYBRID (Barcelona Eixample):
  Diagonal (Cerda plan 1859):
  Grid with chamfered corners and
  diagonal boulevards. Short blocks
  (113m), chamfered corners create
  octagonal intersections.
  Maximum pedestrian permeability +
  diagonal speed for cross-city travel.
```

## Space Syntax: Network Analysis of Movement

Space Syntax (Bill Hillier and Julienne Hanson, UCL, 1984) provides quantitative tools for predicting movement potential from street network geometry.

```
SPACE SYNTAX MEASURES:

  INTEGRATION:
  How easily can you reach every other
  space from this one?
  (Global: from everywhere in city)
  (Local: from 3-5 steps away)

  High integration --> movement
  attracts more people.

  +---+---+---+---+---+
  |   | * |   | * |   |  * = high integration
  +---+*--+---+--*+---+  (many routes pass through)
  |   |   |   |   |   |
  +---+---+---+---+---+
  | * |   |   |   | * |
  +---+---+---+---+---+

  Correlation: high integration streets
  attract retail. Retail placement on
  low-integration streets fails.

  CHOICE (Betweenness Centrality):
  On how many shortest paths between
  all pairs of spaces does this street
  appear?
  (This IS graph-theoretic betweenness centrality — Brandes algorithm, O(VE) —
   applied to the street network graph. Standard CS network analysis metric.)
  High choice = "through movement"
  High integration = "to movement"

  ANGULAR ANALYSIS:
  Pedestrians minimize turns (cognitive
  load), not distance. Straight streets
  feel faster even if longer.
  Angular integration predicts pedestrian
  movement better than metric distance.

PRACTICAL APPLICATIONS:
  - Predict retail viability before building
  - Design street networks for new developments
  - Diagnose why a street "died" after a road closure
  - Evaluate transport network changes
  - Identify streets that will attract crime
    (isolation = low surveillance)

  Microsoft campus analogy: Space Syntax can
  analyze internal building circulation the same
  way. Open-plan offices with high "integration"
  (everybody passes through) promote collaboration.
  Cellular offices on isolated corridors do not.
```

## Density: Measurement and Typologies

```
DENSITY SPECTRUM (residential):

  2 DU/acre   --- Large-lot single family (1+ acre lots)
  4 DU/acre   --- Typical US suburban (1/4 acre lots)
  8 DU/acre   --- Small-lot single family / townhouses
  15 DU/acre  --- Attached townhouses / duplexes
  20 DU/acre  --- WALKABILITY THRESHOLD
                  Below this: walking for daily needs
                  is not economically viable. Too few
                  customers per block to support
                  neighborhood retail.
  30 DU/acre  --- Low-rise apartment buildings (3-4 storey)
  50 DU/acre  --- Mid-rise apartments (5-8 storey)
  100 DU/acre --- High-rise apartments (15+ storey)
  200 DU/acre --- Manhattan average
  300 DU/acre --- Manhattan Upper West Side
  400 DU/acre --- Hong Kong Kowloon City (walled city, demolished)

  KEY INSIGHT: Paris (20 storeys max, no towers)
  achieves 80-100 DU/acre through consistent
  mid-rise perimeter blocks, NOT through towers.
  Height and density are not the same thing.

  Density without height: Paris model (perimeter block)
  Height without density: Le Corbusier towers-in-park
  (towers + parking + greenspace = lower density than Paris!)
```

### Missing Middle Housing

The "missing middle" concept (Daniel Parolek, 2010) identifies a typological gap in American housing between single-family houses and large apartment buildings.

```
MISSING MIDDLE SPECTRUM:

  SINGLE         MISSING MIDDLE           LARGE
  FAMILY         (typically illegal in    APARTMENT
                 most US R-1 zones)

  +----+   +--+--+  +--+--+--+  +---------+  ++++++++++
  |    |   |  |  |  |  |  |  |  |         |  +        +
  | SF |   |duplex|  |triplex|  | apt bldg|  + tower  +
  +----+   +--+--+  +--+--+--+  +---------+  ++++++++++

  2-8 unit buildings at single-family scale:
  - Duplex, triplex, fourplex
  - Townhouse / rowhouse
  - Bungalow court (cottages around shared green)
  - Courtyard apartment (2-story walk-up around court)
  - Live-work building

  WHY "MISSING":
  1920s-1940s: these were built everywhere.
  Post-war Euclidean zoning: required minimum
  lot sizes and single-family use only.
  Existing missing middle "grandfathered in."
  New missing middle: illegal in most US cities.

  WHY IT MATTERS:
  - Fits in existing neighborhoods without disruption
  - Lower cost per unit than large apartments
  - Ownership options (townhouse condos)
  - Gentle densification without "towers"
  - The housing type that most cities need most
```

## Urban Heat Island (UHI): Form and Microclimate

Cities are warmer than surrounding rural areas. The Urban Heat Island effect is a direct consequence of urban form choices.

```
UHI MECHANISM:

  RURAL                    URBAN
  -----                    -----
  Vegetation               Impervious surface
  Evapotranspiration       No evapotranspiration
  High albedo (grass,      Low albedo (asphalt = 0.05,
  crops: 0.20-0.25)        dark roofing = 0.05-0.10)
  Air circulation          Canyon effect (streets
                           between buildings trap air)

  Solar radiation --> absorbed by dark surfaces
                  --> reradiated as heat
                  --> canyon effect traps heat
                  --> city surface temp: 2-5C higher
                  --> city air temp: 1-3C higher
                  --> peak differential: up to 12C
                      (summer nights, urban core)

UHI MAGNITUDE FORMULA (simplified):
  UHI intensity ~ f(impervious surface fraction,
                    vegetation cover,
                    anthropogenic heat release,
                    street canyon H/W ratio)

  H/W ratio (canyon height to width):
  Narrow, tall canyons trap heat.
  Wide streets with low buildings
  allow heat to escape upward.

MITIGATION STRATEGIES:

  TREE CANOPY:
  - Shade reduces surface temp 10-45°F
  - Evapotranspiration cools air
  - Nowak (USDA): 1 large tree = $1-5/yr
    net benefit (energy, stormwater,
    air quality, aesthetics combined)
  - Street tree program ROI: 3:1 to 5:1
    (benefits/costs)
  - 40% urban tree canopy cover = significant
    UHI reduction

  GREEN ROOFS:
  - Extensive (2-4 in growing medium):
    lighter, lower cost, sedum/moss,
    good for stormwater and insulation
  - Intensive (6-24 in): heavier, higher cost,
    true landscaping, higher UHI benefit
  - Cost: $10-25/sq ft (extensive),
    $25-60/sq ft (intensive)
  - Stormwater retention: 50-80% of rainfall

  COOL PAVEMENT:
  - High-albedo concrete (0.35-0.40 vs
    asphalt 0.05-0.15)
  - Permeable pavement (allows evaporation)
  - Reflective coating on asphalt
  - Tradeoff: light-colored surfaces
    can increase glare

  URBAN MORPHOLOGY:
  - Street orientation: E-W streets in hot
    climates get more direct sun; N-S streets
    shade better
  - Building H/W ratio: taller buildings with
    wider streets = better sky view factor =
    more heat escape
  - Courtyard buildings: cool microclimate
    from sheltered outdoor space
  - Trees on S and W sides of buildings:
    maximum cooling effect

  DATA CENTER SITING NOTE:
  Microsoft's hyperscale data centers add
  significant anthropogenic heat load to
  local areas. Siting near rivers (cooling water),
  in cooler climates (PUE reduction), and with
  onsite vegetation are urban-form decisions
  with real UHI consequences.
```

## Sprawl: Measurement and Consequences

```
SPRAWL DEFINITION:
  Low-density, auto-dependent development
  at the urban fringe, separated uses,
  poor street connectivity.

REID EWING'S SPRAWL INDEX:
  Four dimensions:
  1. Residential density (DU/sq mile)
  2. Neighborhood mix of homes, jobs,
     services (entropy measure)
  3. Activity centering (clustering near
     downtowns or suburban centers)
  4. Street accessibility (intersection
     density, % of 4-way intersections)

  Combined into Sprawl Index score.
  Most sprawling US metro: Hickory NC (2014)
  Least sprawling: NYC, San Francisco, Atlantic City

SPRAWL CONSEQUENCES:

  TRANSPORTATION:
  - VMT (vehicle miles traveled) per capita
    correlates directly with density
  - Low-density areas: 20-30 VMT/day
  - Dense urban: 10-15 VMT/day
  - Each VMT: ~0.35 kg CO2 (average US car)

  HEALTH:
  - Walkability score correlated with:
    lower BMI, lower rates of Type 2 diabetes,
    lower cardiovascular disease
  - Ewing/Cervero meta-analysis: 10% higher
    sprawl index --> 5.7% more VMT

  FISCAL:
  - Infrastructure cost per unit:
    Compact (urban infill): $10,000-20,000/unit
    Suburban greenfield: $50,000-100,000/unit
    Sprawl: $100,000-200,000/unit
  - Low-density areas generate less tax revenue
    per $ of infrastructure maintained
    (Strong Towns: "the growth Ponzi scheme")

  ECONOMIC:
  - Sprawl reduces labor market thickness
  - Longer commutes reduce productivity
  - Job-housing mismatch: workers cannot
    reach jobs without cars
```

### Smart Growth Principles (as a counter to sprawl)

```
SMART GROWTH FRAMEWORK:

  The 10 Smart Growth Principles
  (EPA Smart Growth Program):

  1. Mix land uses
  2. Compact building design
  3. Variety of housing types and opportunities
  4. Walkable neighborhoods
  5. Distinctive, attractive communities
     with strong sense of place
  6. Preserve open space, farmland,
     natural beauty, critical environments
  7. Strengthen and direct development
     toward existing communities
  8. Variety of transportation choices
  9. Make development decisions predictable,
     fair, and cost-effective
  10. Encourage community collaboration

  POLICY INSTRUMENTS:
  - Urban Growth Boundaries (Portland Metro UGB:
    1979, Oregon Senate Bill 100, most studied
    urban containment policy in US)
  - Infill incentives (fee waivers, fast-tracking)
  - Transfer of Development Rights (TDR):
    send density from rural "sending zones"
    to urban "receiving zones"
  - Regional fair-share housing (RHNA in California)
```

## Common Confusion Points

**"Density and height are the same thing"**
A 6-storey perimeter block city (Paris) achieves higher residential density than a city of towers-in-parks (Le Corbusier's model). Height increases FAR potential but towers take up little lot coverage — the space between them is empty. Density is people or units per area; height is a single building dimension.

**"Organic street networks are irrational"**
Medieval European street networks appear random but are locally optimized: they follow terrain, water, historical paths, and pedestrian desire lines. They often have high Space Syntax integration scores in their cores. The "irrationality" is from the perspective of cars, not pedestrians.

**"A walkable neighborhood just needs sidewalks"**
Walkability requires density (enough destinations), mixed use (destinations worth walking to), short blocks (many routes), building setbacks that respect the street (active frontages), and human-scale building heights. Sidewalks along a 6-lane arterial between parking lots are not walkable regardless of pavement quality.

**"Green space cools cities"**
Maintained lawns with irrigation and mowing machinery may produce little UHI benefit — the energy use and lack of tree shade reduces the benefit. Urban tree canopy (especially large shade trees) is far more effective than ground-level grass. Heavily irrigated grass in arid climates (Las Vegas, Phoenix) can increase urban temperature by replacing natural desert albedo.

**"Cul-de-sac streets are safer"**
Safer from through traffic, yes. But also isolated, harder to serve by transit, and create longer travel paths that push trips to cars. Emergency response times increase. Crime studies show mixed results: cul-de-sacs reduce some crime types (burglary from strangers) while potentially increasing others (domestic violence is less visible). The safety benefit is real but narrow.

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| What is the most durable element of urban form? | Street network (persists for centuries; plot pattern next; buildings most volatile) |
| What block size supports walkable urbanism? | 200-300ft (Portland model) vs. typical US 400-600ft |
| What density is required for walkability? | ~20 DU/acre minimum for neighborhood retail viability |
| How do you measure street connectivity? | Intersection density (intersections/sq mile) and % four-way intersections |
| What does Space Syntax predict? | Pedestrian movement potential from network geometry; correlates with retail viability and crime |
| How does Paris achieve high density without towers? | Consistent perimeter block typology, 5-8 storeys, full lot coverage at street level |
| What is missing middle housing? | 2-8 unit buildings (duplex, triplex, courtyard apt) -- the typological gap between SF houses and large apartment buildings |
| What is the primary cause of urban heat island? | Dark impervious surfaces + loss of evapotranspiration + canyon effect |
| Most effective UHI mitigation? | Urban tree canopy (shade + evapotranspiration + neighborhood scale) |
| What does FAR not tell you? | Building height, density (units per acre), or architectural quality |
