# 13 — Neuroscience

## Neurons, Action Potentials, Synapses, Brain Regions, Cognition

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Neuron anatomy: soma, dendrites, axon, myelin sheath, nodes of Ranvier, axon terminal
- Neuron types: sensory (afferent), motor (efferent), interneuron; unipolar, bipolar, multipolar
- Glia: astrocytes (BBB, K⁺ buffering), oligodendrocytes (myelin), Schwann cells (PNS myelin), microglia (immune)
- Resting membrane potential: −70 mV; Na⁺/K⁺-ATPase; Nernst equation; Goldman equation
- Action potential: threshold (~−55 mV), depolarization (Na⁺ in), repolarization (K⁺ out), hyperpolarization, refractory period
- Hodgkin-Huxley model: voltage-gated Na⁺ and K⁺ channel conductances, differential equations
- Propagation: saltatory conduction in myelinated axons (200 m/s); unmyelinated C fibers (~0.5 m/s)
- Synaptic transmission: vesicle fusion (Ca²⁺-triggered, SNARE proteins), neurotransmitter release into cleft
- Neurotransmitters: glutamate (excitatory), GABA (inhibitory), dopamine, serotonin, acetylcholine, norepinephrine, glycine
- Postsynaptic receptors: ionotropic (fast, channel-linked: AMPA, NMDA, GABA_A) and metabotropic (slow, GPCR: mGluR, GABA_B)
- NMDA receptor: voltage-dependent Mg²⁺ block, Ca²⁺ permeability, role in LTP (Hebbian plasticity)
- Synaptic plasticity: long-term potentiation (LTP), long-term depression (LTD), spike-timing dependent plasticity (STDP)
- Neural coding: rate coding vs temporal coding, population codes, sparse coding, place cells (O'Keefe)
- Brain organization: cerebral cortex, basal ganglia, thalamus, hypothalamus, brainstem, cerebellum
- Cortical hierarchy: primary sensory → association areas → prefrontal; feedforward + feedback
- Visual system: retina (rods/cones/bipolar/ganglion), LGN, V1 (orientation selectivity, Hubel-Wiesel), V2-V5, ventral (what) vs dorsal (where) streams
- Motor system: primary motor cortex (homunculus), corticospinal tract, basal ganglia (action selection), cerebellum (error correction)
- Limbic system: hippocampus (episodic memory), amygdala (fear/emotion), cingulate cortex
- Memory consolidation: short-term (working memory, PFC) → long-term (hippocampal consolidation → neocortex)
- Declarative vs non-declarative memory: explicit (episodic/semantic) vs implicit (procedural, priming, conditioning)
- Neurological disorders: Parkinson's (dopamine loss, basal ganglia), Alzheimer's (amyloid-β, tau, hippocampal atrophy), epilepsy (seizure, E/I imbalance)
- Consciousness: neural correlates of consciousness (NCCs), global workspace, default mode network
- AI/neuroscience connections: artificial neural networks inspired by neurons; deep learning vs cortical hierarchy; predictive coding
