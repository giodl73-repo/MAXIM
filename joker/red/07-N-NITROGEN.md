## The Puzzle

Ten messages follow. Each is encoded in a different system from the `codes/` section of the encyclopedia. For each message:

1. Identify the encoding system.
2. Find the reference material you need in `codes/`.
3. Decode the message to reveal a single English word.
4. Extract the first letter of each decoded word.

Read the decoded letters top to bottom. They spell your answer.

### Message 6

A signer spells a word one letter at a time. You see:

```
Position 1:  Curved hand, like gripping a cylinder — fingers and thumb
             form an open C shape.

Position 2:  Index and middle fingers together, pointing straight up.
             Ring finger and pinky curled. Thumb tucked.

Position 3:  Index and middle fingers crossed — index over middle.
             Other fingers curled.

Position 4:  Index finger points up, thumb extends outward at a right
             angle — forming an L shape.
```

### Message 10

A strip of colored bands, divided into five groups by thin gaps. Each group encodes one or two digits. Convert each color to its digit using the standard electronic component marking system described in `codes/09`. Then convert each resulting number to a letter of the alphabet (A=1, B=2, ..., Z=26).

```
Group 1:  Brown — Yellow
Group 2:  White
Group 3:  Violet
Group 4:  Gray
Group 5:  Red — Black
```
