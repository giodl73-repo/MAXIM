# Measure-Theoretic Probability Foundations

## The Big Picture

Probability theory is measure theory with total mass normalized to 1. The machinery exists to make "random" rigorous.

```
+------------------------------------------------------------------+
|              PROBABILITY SPACE  (Omega, F, P)                    |
+------------------------------------------------------------------+
|                                                                  |
|  Omega  = sample space      all possible outcomes               |
|  F      = sigma-algebra     the events we can assign probs to   |
|  P      = probability meas  assigns numbers in [0,1] to events  |
|                                                                  |
|  +--------------------+                                         |
|  |  Omega             |  <- the "universe"                      |
|  |  +------+          |                                         |
|  |  |  A   |  <- event: a subset of Omega in F                  |
|  |  +------+          |                                         |
|  |                    |                                         |
|  +--------------------+                                         |
|           |                                                      |
|           v                                                      |
|  P(A) in [0,1]    probability measure applied to event A        |
|                                                                  |
|  Random variable X: Omega -> R                                   |
|  (a measurable function on this space)                          |
|                                                                  |
+------------------------------------------------------------------+
```

## Sample Spaces and Events

**Sample space Omega**: The set of all possible outcomes of an experiment.

```
  Coin flip:  Omega = {H, T}
  Dice roll:  Omega = {1, 2, 3, 4, 5, 6}
  Real line:  Omega = R  (lifetimes, temperatures, ...)
  Sequences:  Omega = {0,1}^N  (infinite binary sequences)
```

**Event**: A subset A of Omega. "The event that an odd number comes up" = {1, 3, 5}.

**The problem with naive "all subsets"**: For finite Omega, P(A) for every subset A is fine. For uncountable Omega (like R), assigning probabilities to *all* subsets leads to contradictions (Vitali sets — non-measurable sets exist if you have the Axiom of Choice). The sigma-algebra specifies *which* subsets are measurable.

---

## Sigma-Algebras

A sigma-algebra (sigma-field) F on Omega is a collection of subsets satisfying:

```
  1.  Omega in F                     (whole space is measurable)
  2.  A in F  =>  A^c in F           (closed under complement)
  3.  A_1, A_2, ... in F  =>         (closed under countable unions)
      Union_{i=1}^{inf} A_i in F

  Consequence: also closed under countable intersections
  (De Morgan: Intersection A_i = (Union A_i^c)^c)
```

**Why countable, not arbitrary?** Because we want to pass limits through probability — P(lim A_n) = lim P(A_n) under appropriate conditions. Uncountable unions break this.

**Key sigma-algebras**:

```
  Trivial:     F = {emptyset, Omega}          -- no information
  Discrete:    F = P(Omega) (all subsets)     -- full info, OK for countable Omega
  Borel:       B(R) = generated by open sets  -- the standard on R
  Cylinder:    on function spaces (Wiener measure lives here)
```

**Generated sigma-algebra**: sigma(C) for a collection C is the smallest sigma-algebra containing C. This is how B(R) is defined: sigma({open sets in R}).

**Filtration**: An increasing family F_0 subset F_1 subset ... subset F — models accumulating information over time. Essential for stochastic processes (see 04).

## The Kolmogorov Axioms

A probability measure P: F -> [0,1] satisfies:

```
  K1.  P(A) >= 0                for all A in F          (non-negativity)
  K2.  P(Omega) = 1                                      (normalization)
  K3.  A_1, A_2, ... disjoint   =>                       (sigma-additivity)
       P(Union A_i) = Sum P(A_i)
```

**Everything else follows**:

```
  P(emptyset) = 0              (from K2 + K3: Omega = Omega union emptyset union ...)
  P(A^c) = 1 - P(A)            (from K2 + K3: Omega = A union A^c)
  A subset B => P(A) <= P(B)   (monotonicity)
  P(A union B) = P(A) + P(B) - P(A intersect B)  (inclusion-exclusion)
  Boole's inequality:  P(Union A_i) <= Sum P(A_i)  (sigma-subadditivity)
```

## Conditional Probability and Independence

**Conditional probability** (definition, not theorem):

```
  P(A | B) = P(A intersect B) / P(B),    P(B) > 0
```

This updates the probability of A given that B has occurred. The sample space effectively shrinks to B.

**Independence**:

```
  A, B independent  <=>  P(A intersect B) = P(A) P(B)
  <=> P(A | B) = P(A)   (when P(B) > 0)
  <=> P(B | A) = P(B)   (when P(A) > 0)

  Events A_1, ..., A_n are mutually independent if:
  For all S subset {1,...,n}:
  P(Intersection_{i in S} A_i) = Product_{i in S} P(A_i)

  Pairwise independence does NOT imply mutual independence.
```

**Bayes' Theorem** (follows directly from conditional probability definition):

```
  P(A | B) = P(B | A) P(A) / P(B)

  Full form with partition {H_i}:
  P(H_j | E) = P(E | H_j) P(H_j) / Sum_i P(E | H_i) P(H_i)

  Posterior    = Likelihood x Prior / Evidence
```

## Random Variables as Measurable Functions

A **random variable** X on (Omega, F, P) is a measurable function X: Omega -> R:

```
  "Measurable" means: for all Borel sets B in B(R),
  X^{-1}(B) = {omega in Omega : X(omega) in B} is in F

  This is the same as "measurable function" from analysis.
  The randomness comes from P, not from X itself.
```

**The type-theory analogy**: A random variable X: Omega -> R has type `Omega -> R`, just like a measurable function. The sigma-algebra F is the "measurability type class" — it constrains which functions are valid. This is analogous to how a type system constrains which terms are well-typed.

**Distribution (pushforward measure)**: Given X: Omega -> R and P on Omega, define the distribution P_X on B(R):

```
  P_X(B) = P(X^{-1}(B)) = P(X in B)

  This is the "law" or "distribution" of X.
  It lives on (R, B(R)), not on the original (Omega, F, P).
```

Once you have P_X, you typically don't care about the underlying Omega anymore — you just work with the distribution.

## Expectation as Lebesgue Integral

```
  E[X] = Integral_{Omega} X(omega) dP(omega)

  For discrete X (takes values x_1, x_2, ...):
  E[X] = Sum_i x_i P(X = x_i)

  For continuous X with density f:
  E[X] = Integral_{-inf}^{inf} x f(x) dx

  Both are special cases of the Lebesgue integral.
```

**Expectation rules** (from linearity of integration):

```
  E[aX + bY] = aE[X] + bE[Y]              (linearity, always)
  E[XY] = E[X]E[Y]                        (if X, Y independent)
  E[g(X)] = Integral g(x) dP_X(x)        (law of the unconscious statistician)
```

**Dominated Convergence Theorem** (the workhorse for limit exchange):

```
  If X_n -> X pointwise a.s.,
  and |X_n| <= Y with E[Y] < inf,
  then E[X_n] -> E[X]

  This justifies: lim E[X_n] = E[lim X_n]
  under an integrability condition.
```

## Key Inequalities

These appear constantly in proofs:

```
  MARKOV'S INEQUALITY
  P(X >= a) <= E[X] / a        for X >= 0, a > 0
  Crude but universal. Tight for degenerate distributions.

  CHEBYSHEV'S INEQUALITY
  P(|X - mu| >= k*sigma) <= 1/k^2
  Follows from Markov applied to (X - mu)^2.
  Gives LLN (weak): sample mean converges in probability.

  JENSEN'S INEQUALITY
  If phi is convex: E[phi(X)] >= phi(E[X])
  If phi is concave: E[phi(X)] <= phi(E[X])
  Key examples: E[X^2] >= E[X]^2 (variance >= 0)
                E[log X] <= log E[X] (important for KL divergence)

  CAUCHY-SCHWARZ
  |E[XY]|^2 <= E[X^2] E[Y^2]
  Equivalently: |Cov(X,Y)| <= sqrt(Var(X) Var(Y))
  Correlation in [-1, 1] follows directly.

  HOEFFDING'S INEQUALITY
  For bounded independent r.v.s X_i in [a_i, b_i]:
  P(|mean - mu| >= t) <= 2 exp(-2n^2 t^2 / Sum(b_i - a_i)^2)
  Exponential tail bound. Foundation of PAC learning.
```

**Beyond Hoeffding — the concentration inequality family:**

```
  CONCENTRATION INEQUALITY HIERARCHY:
  ──────────────────────────────────────────────────────────────────────
  BERNSTEIN'S INEQUALITY (variance-adaptive, tighter than Hoeffding):
  For independent X_i in [a_i, b_i] with Var(X_i) = sigma_i^2:
  P(|S_n - E[S_n]| >= t) <= 2 exp(-t^2 / (2(Sum sigma_i^2 + ct/3)))
  Tighter than Hoeffding when variance << range (common in practice).

  AZUMA-HOEFFDING (for martingales):
  If {M_k} is a martingale with |M_k - M_{k-1}| <= c_k:
  P(|M_n - M_0| >= t) <= 2 exp(-t^2 / (2 Sum c_k^2))
  Needed for adaptive algorithms where choices depend on past data.

  McDIARMID'S INEQUALITY (bounded differences):
  If f(X_1,...,X_n) changes by at most c_i when X_i varies:
  P(|f - E[f]| >= t) <= 2 exp(-2t^2 / Sum c_i^2)
  Ubiquitous in learning theory: apply to empirical risk, VC bounds.

  SUB-GAUSSIAN / SUB-EXPONENTIAL (modern unifying framework):
  X is sub-Gaussian(sigma) if E[exp(t(X-mu))] <= exp(sigma^2 t^2/2)
  All bounded r.v.s are sub-Gaussian. Gaussian is sub-Gaussian.
  Sub-Gaussian tails: P(|X - mu| >= t) <= 2 exp(-t^2/(2 sigma^2))

  X is sub-exponential if E[exp(t|X-mu|)] < inf for small t.
  Chi-squared, exponential r.v.s are sub-exponential (not sub-Gaussian).
  Sub-exponential tails: Gaussian for small t, exponential for large t.

  MATRIX CONCENTRATION (for random matrices):
  Matrix Bernstein: ||Sum X_i - E[Sum X_i]|| <= O(sqrt(sigma^2 log d) + c log d)
  Used in: compressed sensing, random graph theory, spectral methods.
```

## Almost Sure vs. In Probability vs. In L^p

Multiple notions of convergence exist for sequences of random variables:

```
  X_n --a.s.--> X:
  P(lim_{n->inf} X_n = X) = 1
  Convergence on a set of measure 1. Strongest.

  X_n --P--> X  (in probability):
  For all eps > 0: P(|X_n - X| > eps) -> 0
  Weaker: each individual event shrinks, but not uniformly.

  X_n --L^p--> X  (in L^p):
  E[|X_n - X|^p] -> 0
  L^2 convergence is mean-square convergence.

  X_n --d--> X  (in distribution / weak convergence):
  E[f(X_n)] -> E[f(X)] for all bounded continuous f
  Weakest. CLT conclusion is convergence in distribution.

  HIERARCHY:
  a.s.  =>  in probability  =>  in distribution
  L^2   =>  in probability  =>  in distribution
  (a.s. and L^2 do NOT imply each other)
```

## Independence, Sigma-Algebras, and the Factorization Lemma

**Independence via sigma-algebras**: Sub-sigma-algebras G_1, G_2 subset F are independent if:

```
  For all A in G_1, B in G_2: P(A intersect B) = P(A) P(B)
```

**Factorization lemma (Doob-Dynkin)**: Y is measurable with respect to sigma(X) (the sigma-algebra generated by X) if and only if Y = f(X) for some measurable f. This characterizes which random variables "depend on" X.

**Conditional expectation**: E[X | G] for a sub-sigma-algebra G is the unique G-measurable function Z satisfying:

```
  For all A in G:  Integral_A Z dP = Integral_A X dP

  This exists and is unique (Radon-Nikodym theorem).
  When G = sigma(Y): E[X | G] = E[X | Y] in the usual sense.
```

The Radon-Nikodym theorem is the measure-theoretic engine behind conditional expectation — it guarantees the existence of the conditional expectation as a measurable function.

---

## The Radon-Nikodym Theorem and Densities

If nu is absolutely continuous with respect to mu (nu << mu — nu assigns zero measure to any mu-null set), then there exists a measurable function f (the Radon-Nikodym derivative) such that:

```
  nu(A) = Integral_A f dmu    for all measurable A

  Notation: f = dnu/dmu  (the "density" of nu w.r.t. mu)
```

**In probability**: If P_X has a probability density function f, this means P_X << Lebesgue and f = dP_X/dLeb. The density is the Radon-Nikodym derivative with respect to Lebesgue measure.

## Decision Cheat Sheet

| Concept | Formal Statement | Practical Use |
|---------|-----------------|---------------|
| Probability space | (Omega, F, P) with P sigma-additive | Foundation for everything |
| Random variable | Measurable function X: Omega -> R | The object we compute with |
| Distribution | Pushforward measure P_X on (R, B(R)) | What we usually work with |
| Independence | P(A intersect B) = P(A)P(B) | Factoring joint distributions |
| Conditional expectation | E[X|G]: L^1 -> L^1(G) | Bayesian updates, martingales |
| a.s. convergence | P(lim X_n = X) = 1 | LLN (strong) conclusion |
| Convergence in distribution | E[f(X_n)] -> E[f(X)] | CLT conclusion |
| Radon-Nikodym | dnu/dmu = f iff nu = int f dmu | Defines densities rigorously |

## Common Confusion Points

**"The sigma-algebra is the sigma-algebra of events — why does it matter which one?"**
It matters because measurability is relative to F. A function X might be measurable w.r.t. a finer sigma-algebra but not a coarser one. Filtrations in stochastic processes encode "what information is available at time t" — a process adapted to the filtration means its value at time t is determined by information available at t.

**"Why is Omega usually not mentioned after setup?"**
Once you have the distribution P_X on (R, B(R)), you can forget Omega. The pushforward contains all the probability information. This is why distributions are the central objects in practice.

**"Almost surely vs. surely — can anything fail on a null set?"**
Yes, and it matters. The strong LLN says the sample mean converges to the true mean a.s. — meaning on a set of probability 1, not on *every* sequence. There exist sequences where the sample mean diverges; they just have probability 0.

**"Is conditional expectation E[X|Y] a number or a function?"**
E[X|Y=y] is a number (for fixed y). E[X|Y] is a random variable — a function of Y. This distinction is crucial for the tower property: E[E[X|Y]] = E[X].
