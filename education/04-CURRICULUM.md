# Curriculum Design and Learning Objectives

## The Big Picture

Curriculum design is the systematic specification of what to teach, in what sequence, through what methods, and assessed how. The central insight of modern curriculum theory: start with the end (what should learners be able to do?) and design backward toward the beginning (what instruction enables that?). This inverts the traditional content-coverage approach.

```
+------------------------------------------------------------------+
|                  CURRICULUM DESIGN LANDSCAPE                     |
+------------------------------------------------------------------+
```

## Tyler Rationale (1949)

Ralph Tyler's *Basic Principles of Curriculum and Instruction* defined the modern curriculum design framework through four fundamental questions:

```
  TYLER'S FOUR QUESTIONS
  =======================

  1. WHAT EDUCATIONAL PURPOSES should the school seek to attain?
     (Objectives)

  2. WHAT EDUCATIONAL EXPERIENCES can be provided
     to attain these purposes?
     (Learning experiences / instruction)

  3. HOW CAN THESE EXPERIENCES be effectively organized?
     (Sequence and structure)

  4. HOW CAN WE DETERMINE whether these purposes
     are being attained?
     (Assessment / evaluation)

  +------------+          +--------------+
  | OBJECTIVES |--------> | EXPERIENCES  |
  +------------+          | (Instruction)|
        ^                 +--------------+
        |                       |
        |                       v
  +------------+          +--------------+
  | EVALUATION |<-------- | ORGANIZATION |
  +------------+          | (Sequence)   |
                          +--------------+

  This is Tyler's rationale: objectives drive everything.
  Assessment measures whether objectives were met.
  Linear, objectives-driven model.
```

## Bloom's Taxonomy (1956, Revised 2001)

The most widely used framework for specifying learning objectives across educational levels:

### Original Taxonomy (Bloom et al., 1956)

```
  BLOOM'S TAXONOMY (Original Cognitive Domain)
  =============================================

  Level 1: KNOWLEDGE
    Recall of facts, terms, procedures.
    "Name the parts of a cell."

  Level 2: COMPREHENSION
    Understanding; paraphrase, explain.
    "Explain the function of the mitochondria."

  Level 3: APPLICATION
    Use knowledge in new situations.
    "Calculate the area of this triangle."

  Level 4: ANALYSIS
    Break into components; relationships.
    "Compare the narrative structures of these two novels."

  Level 5: SYNTHESIS
    Create new whole from parts.
    "Design an experiment to test this hypothesis."

  Level 6: EVALUATION
    Judgments using criteria.
    "Evaluate the validity of this argument."

  Higher Order ← ← ← ← ← ← ← Lower Order
  EVALUATION / SYNTHESIS / ANALYSIS
  ←  ←  ←  ←  ←  ←  ←  ←  ←  ←  ←  ←
  APPLICATION / COMPREHENSION / KNOWLEDGE
```

### Anderson/Krathwohl Revision (2001)

```
  REVISED BLOOM'S TAXONOMY
  =========================

  KEY CHANGES:
  1. Nouns → Verbs (knowledge types → cognitive processes)
  2. Added KNOWLEDGE DIMENSION (separate axis)
  3. Moved Synthesis to top (now "Creating")

  COGNITIVE PROCESS DIMENSION:
  1. REMEMBER: Recognize, recall
  2. UNDERSTAND: Interpret, classify, summarize, explain
  3. APPLY: Execute, implement
  4. ANALYZE: Differentiate, organize, attribute
  5. EVALUATE: Check, critique
  6. CREATE: Generate, plan, produce

  KNOWLEDGE DIMENSION (orthogonal axis):
  A. Factual knowledge (facts, terminology)
  B. Conceptual knowledge (categories, principles)
  C. Procedural knowledge (how to do something)
  D. Metacognitive knowledge (self-knowledge, strategies)

  TAXONOMY TABLE:
         |Remember|Understand|Apply|Analyze|Evaluate|Create|
  Factual|        |          |     |       |        |      |
  Concept|        |          |     |       |        |      |
  Proced.|        |          |     |       |        |      |
  Metacog|        |          |     |       |        |      |

  Each cell = a type of learning objective.
  Good curriculum covers multiple cells, not just
  Factual-Remember in the top-left corner.
```

## Backward Design / Understanding by Design (Wiggins & McTighe)

*Understanding by Design* (1998) is the dominant curriculum design framework in US K-12 education:

```
  BACKWARD DESIGN: THREE STAGES
  ================================

  STAGE 1: IDENTIFY DESIRED RESULTS
  "What should students know, understand, and be able to do?"

    Established goals (standards, enduring understandings)
    Enduring Understandings: "Students will understand that..."
    Essential Questions: "What makes a good argument?"
    Knowledge and skills to acquire.

  STAGE 2: DETERMINE ACCEPTABLE EVIDENCE
  "How will we know if students have achieved the results?"

    Performance tasks (authentic assessment)
    Other evidence (quizzes, tests, homework)
    Design assessment BEFORE designing instruction.
    If you can't assess it, was it really your goal?

  STAGE 3: PLAN LEARNING EXPERIENCES AND INSTRUCTION
  "What instruction will enable students to perform?"

    Instructional strategies
    Learning activities
    Resources
    Sequence

  KEY INSIGHT:
    Traditional design: select content first, teach it,
    assess at the end.
    Backward design: decide what learning looks like
    FIRST, then design instruction to produce it.

  EXAMPLE (High School English):
    Stage 1: Students will understand that effective
             arguments require evidence, reasoning, and
             addressing counterarguments.
    Stage 2: Write a persuasive essay on a policy issue;
             peer critique using a rubric.
    Stage 3: Study argument structure, model essays,
             guided practice on evidence evaluation.
```

### Enduring Understandings vs. Knowledge

```
  WIGGINS/McGTIGHE DISTINCTION
  ================================

  KNOWLEDGE:
    Discrete facts and skills.
    "Students will know the dates of WWI."
    "Students will be able to multiply fractions."

  ENDURING UNDERSTANDINGS:
    Big ideas that transfer across contexts.
    "Students will understand that historical events
    have complex multiple causes."
    "Students will understand that mathematical
    structure can be used to analyze phenomena."

    Enduring understandings are expressed as full
    sentences with "Students will understand that..."
    They require inference and insight, not just recall.

  Why both? Knowledge is the foundation.
  Understandings are what knowledge is in service of.
  Most teaching only produces knowledge (bottom of Bloom).
  Understanding requires sustained engagement with
  essential questions.
```

## Writing Learning Objectives (Mager)

Robert Mager (1962) operationalized Tyler's objectives into a practical format:

```
  MAGER'S ABCD FORMAT
  =====================

  A = AUDIENCE: Who will be doing the performing?
      "The student will..."

  B = BEHAVIOR: Observable, measurable action.
      (MUST be a verb from Bloom's taxonomy or equivalent)
      "...identify, calculate, write, distinguish,
       design, evaluate..."
      NOT: "understand, know, appreciate, be aware"
      (unobservable)

  C = CONDITION: Under what circumstances?
      "Given a passage of 500 words..."
      "Without the use of a calculator..."
      "In a classroom assessment..."

  D = DEGREE: How well? What criterion?
      "...with 80% accuracy."
      "...correctly identifying all 5 structural features."
      "...within 30 minutes."

  EXAMPLE:
  POOR: "Students will understand photosynthesis."
  BETTER: "Given a diagram of a leaf cell, students will
           label the organelles involved in photosynthesis
           and describe the function of each in
           one paragraph without reference materials."
```

## SOLO Taxonomy (Biggs and Collis, 1982)

Structure of Observed Learning Outcomes -- an alternative to Bloom's with different theoretical grounding:

```
  SOLO LEVELS
  ===========

  PRESTRUCTURAL:
    Incompetent; task not attacked properly.
    "I don't know."

  UNISTRUCTURAL:
    One relevant aspect.
    Simple, obvious connection.
    "The mitochondria makes energy."

  MULTISTRUCTURAL:
    Several relevant aspects, but not integrated.
    List without connection.
    "Mitochondria makes ATP. Ribosomes make proteins.
    Nucleus contains DNA."

  RELATIONAL:
    Integration of all aspects.
    Understanding how parts relate to each other.
    "These organelles form a system where DNA in
    the nucleus codes for proteins made by ribosomes,
    which include enzymes used by mitochondria..."

  EXTENDED ABSTRACT:
    Transfer to new domain; generalize.
    "This cellular division of labor is an instance
    of the general principle that complex systems
    achieve efficiency through functional specialization
    and integration, as seen also in..."

  SOLO vs. BLOOM:
    SOLO: Describes quality of understanding of a topic.
    Bloom: Describes type of cognitive process across topics.
    SOLO is more useful for assessing depth within a topic.
    Bloom is more useful for planning across content types.
```

## Curriculum Organization

### Scope and Sequence

```
  SCOPE: What is included?
    Breadth: How many topics are covered?
    Depth: How deeply is each topic covered?
    Breadth vs. depth trade-off: US curricula criticized
    as "mile wide, inch deep" (Schmidt, 1997 TIMSS comparison)
    Singapore math: fewer topics, much deeper mastery.

  SEQUENCE: In what order?
    Simple to complex: prerequisite skills first.
    Concrete to abstract: hands-on before symbolic.
    Chronological: useful in history, literature.
    Spiral: revisit concepts at increasing depth.
    Problem-centered: around real questions.
```

### Spiral Curriculum (Bruner)

```
  SPIRAL CURRICULUM
  ==================

  Core concepts revisited at increasing levels of
  sophistication throughout schooling.

  Grade 2:   Fractions as parts of a whole (pizza)
  Grade 4:   Fractions on a number line; equivalence
  Grade 6:   Operations with fractions; ratios
  Grade 8:   Proportional reasoning; algebraic fractions
  Grade 10:  Rational functions; limits

  vs. MASTERY SEQUENCE (more common in US):
    Teach topic once, move on.
    Risk: students who miss it, miss it permanently.
    Spiral: allows re-entry points.
```

### Curriculum Alignment

```
  THREE CURRICULUM TYPES (Goodlad)
  ====================================

  INTENDED CURRICULUM:
    What the standards/policy specify.
    Written documents, syllabi.

  ENACTED CURRICULUM:
    What teachers actually teach.
    May diverge from intended.

  ACHIEVED CURRICULUM:
    What students actually learn.
    May diverge from enacted.

  HIDDEN CURRICULUM:
    Values, norms, behaviors learned implicitly.
    Punctuality, authority relations, competition,
    gender roles -- not in any syllabus.
    Transmitted through school culture and organization.

  ALIGNMENT PROBLEM:
    Intended <-> Enacted: teacher doesn't cover all standards.
    Enacted <-> Assessed: test doesn't reflect what was taught.
    Intended <-> Assessed: standards don't align to tests.
    All three must align for curriculum to function.
```

## Universal Design for Learning (UDL)

```
  UNIVERSAL DESIGN FOR LEARNING
  ================================

  CAST (Center for Applied Special Technology) framework.
  Based on three principles:

  1. MULTIPLE MEANS OF REPRESENTATION
     (the "what" of learning)
     Present information in multiple formats.
     Example: text + audio + visual + tactile.
     Reduce barriers for students with
     reading disabilities, visual impairments.

  2. MULTIPLE MEANS OF ACTION AND EXPRESSION
     (the "how" of learning)
     Allow students to express knowledge in multiple ways.
     Written, oral, visual, dramatic.
     Reduce barriers for students with
     writing or speaking difficulties.

  3. MULTIPLE MEANS OF ENGAGEMENT
     (the "why" of learning)
     Provide options for interest, persistence,
     self-regulation.
     Reduce barriers for students with
     attention or motivational challenges.

  DESIGN METAPHOR:
    Like wheelchair ramps: designed for wheelchair users,
    useful for everyone (strollers, bikes, luggage).
    UDL designed for diverse learners;
    benefits all learners.
```

## Decision Cheat Sheet

| Design Question | Framework | Tool |
|----------------|-----------|------|
| What should students be able to do? | Bloom's/SOLO taxonomy | Write specific objectives |
| What does understanding look like? | Wiggins/McTighe UbD | Enduring understandings + essential questions |
| How should I order the curriculum? | Scope/sequence + spiral | Prerequisite map; revisiting spiral |
| How do I write a good objective? | Mager ABCD format | Observable verb + condition + criterion |
| How do I plan a unit? | Backward design | Stage 1 goals --> Stage 2 assessment --> Stage 3 instruction |
| How do I accommodate diverse learners? | UDL | Multiple representations, expressions, engagement |

## Common Confusion Points

**Coverage Is Not Learning**
The most common curriculum failure: covering everything but students retain little. Coverage (presenting content) is not the same as learning (students can recall, apply, transfer). Deep engagement with fewer topics produces more learning than shallow coverage of many topics. The "mile wide, inch deep" problem is well-documented.

**Objectives Should Use Observable Verbs**
"Students will understand X" is not a measurable objective -- you cannot observe "understanding." You can observe writing, explaining, applying, demonstrating, analyzing. Every learning objective should have an observable verb. This is not pedantry -- vague objectives prevent assessment design.

**Backward Design Doesn't Mean Assessment-Obsessed**
Backward design starts with desired outcomes to avoid teaching toward unclear goals. It does not mean teaching to the test. Authentic performance tasks (portfolio, exhibition, project) are the preferred assessments in UbD -- richer than standardized tests, not substitutable for them.

**Hidden Curriculum Is Not Optional**
Every school transmits values through its structure and culture -- the hidden curriculum. Students learn that: punctuality matters, authority should be deferred to, competition is natural, some bodies of knowledge are more valued than others. Ignoring the hidden curriculum doesn't make it go away; it makes it unexamined.
