# Assessment: Formative, Summative, and Authentic

## The Big Picture

Assessment is the process of gathering evidence about student learning. The type, timing, and use of assessment determines whether it supports learning or merely measures it. The central distinction: formative assessment (assessment FOR learning, used to adjust instruction) vs. summative assessment (assessment OF learning, used to measure outcomes). These are different functions that require different designs.

```
+------------------------------------------------------------------+
|                    ASSESSMENT LANDSCAPE                          |
+------------------------------------------------------------------+
|                                                                  |
|  FORMATIVE                        SUMMATIVE                      |
|  (assessment for learning)        (assessment of learning)       |
|  -----------------------          -----------------------        |
|  During instruction               After instruction              |
|  Adjust teaching + learning       Record achievement             |
|  Ungraded or low-stakes           Graded, high-stakes            |
|  Specific feedback                Score or grade                 |
|  Examples: exit tickets,          Examples: final exams,         |
|  think-pair-share, quizzes        standardized tests             |
|                                                                  |
|  DIAGNOSTIC                       AUTHENTIC                      |
|  (pre-assessment)                 (performance tasks)            |
|  ----------------------           ----------------------         |
|  Before instruction               Realistic tasks                |
|  Reveal prior knowledge           Demonstrate competence         |
|  Inform differentiation           Portfolio, project, exhibition |
|                                                                  |
|  STANDARDIZED TESTING             PSYCHOMETRIC CONCEPTS          |
|  -------------------              ---------------------          |
|  NAEP, SAT, ACT, AP               Validity                       |
|  State tests (PARCC, SBAC)        Reliability                    |
|  International: PISA, TIMSS       Fairness / bias                |
+------------------------------------------------------------------+
```

## The Formative/Summative Distinction

### Formative Assessment

```
  FORMATIVE ASSESSMENT: DEFINITION AND MECHANISMS
  ==================================================

  Black and Wiliam (1998) meta-analysis:
    "Formative assessment" -- one of the highest
    effect sizes in education research: d = 0.4-0.7
    (Large effect)

  MECHANISM:
    Formative assessment works when it produces:
    1. Evidence about the gap between current
       and desired performance.
    2. Action by TEACHER (adjust instruction).
    3. Action by STUDENT (adjust study behavior).

  KEY: Formative assessment is not about TESTS.
       It is about using evidence to change what
       teachers and students DO NEXT.

  EXAMPLES:
    Exit tickets: 3x2 minutes at end of class.
    Think-Pair-Share: student responses reveal thinking.
    Cold calling + wait time: reveals individual understanding.
    Hinge questions: designed to reveal misconceptions.
    Mini whiteboards: every student responds simultaneously.
    Traffic lights: green/yellow/red self-assessment.
    Muddiest point: "What is still unclear?"
    Live polling (Poll Everywhere, Mentimeter).
```

### Hinge Questions

A hinge question is a diagnostic multiple-choice question where each wrong answer maps to a specific misconception:

```
  HINGE QUESTION DESIGN
  ======================

  EXAMPLE (Science):
    "What happens to the weight of a candle as it burns?"
    A. It stays the same (Misconception: "nothing leaves")
    B. It increases (Misconception: "oxygen absorbed from air weighs more")
    C. It decreases (CORRECT: wax + oxygen --> CO2 + H2O, which leave)
    D. It first increases, then decreases (Misconception: combination of above)

  If 60% answer C: proceed.
  If 30% answer A: teach conservation of mass more explicitly.
  If 25% answer B: teach combustion chemistry more carefully.

  POWER: Each wrong answer diagnoses a specific gap.
  Teacher response is DIFFERENT for each misconception.
  This is diagnostic teaching, not just item-scoring.
```

## Summative Assessment

```
  SUMMATIVE ASSESSMENT PURPOSES
  ================================

  1. ACCOUNTABILITY: Did students achieve standards?
     Individual: grades, promotion, graduation.
     Institutional: school report cards, accreditation.
     System: state/national performance reporting.

  2. SELECTION: Who gets into advanced programs?
     AP exams, college admissions (SAT/ACT).
     University entrance (GCSE, A-levels, IB).

  3. CERTIFICATION: Is this person competent?
     Professional licensure (bar exam, medical boards).
     Trade certifications.
     Credential award.

  4. RESEARCH: What is happening nationally?
     NAEP (National Assessment of Educational Progress)
     PISA, TIMSS, PIRLS (international comparisons)

  DESIGN CHALLENGE:
    Summative tests measure a SAMPLE of learning.
    Sample may or may not represent full learning.
    High-stakes summative assessment creates teaching-to-test.
    (Goodhart's Law: when a measure becomes a target,
    it ceases to be a good measure)
```

## Standardized Testing

### Major US Assessments

```
  US ASSESSMENT LANDSCAPE
  ========================

  HIGH STAKES / ACCOUNTABILITY:
    State tests (required by ESSA/Every Student Succeeds Act)
    Annual math + reading, Grades 3-8 + once in high school
    Proficiency percentages used for school accountability
    SBAC (Smarter Balanced) -- Common Core aligned
    PARCC (Partnership for Assessment) -- Common Core
    State-specific: STAAR (TX), MCAS (MA), etc.

  RESEARCH / MONITORING:
    NAEP ("Nation's Report Card")
    Sample-based; no individual scores
    Benchmark for state comparisons
    Subject areas: reading, math, science, history, civics
    Trend data since 1970s

  COLLEGE ADMISSIONS:
    SAT (College Board): redesigned 2016, 2024
    ACT (ACT Inc.)
    Both: math + reading/writing
    Optional at many colleges post-COVID
    AP (Advanced Placement): college-level courses + exams
    IB (International Baccalaureate): global diploma program

  INTERNATIONAL:
    PISA (Programme for International Student Assessment)
    15-year-olds; reading, math, science; 80+ countries
    3-year cycle; OECD administered
    TIMSS (Trends in International Mathematics and Science)
    Grades 4 and 8; IEA administered
    PIRLS: reading in Grade 4
```

### Critique of High-Stakes Standardized Testing

```
  STANDARDIZED TESTING CRITIQUES
  =================================

  1. CURRICULUM NARROWING:
     Schools focus on tested subjects (math + reading).
     Science, social studies, arts, PE: reduced.
     Evidence: Pre-NCLB elementary schools had ~3 hours
     weekly science; post-NCLB: ~1.5 hours.

  2. TEACHING TO THE TEST:
     Item formats practiced repeatedly.
     Transfer to non-test contexts: poor.
     "Score inflation" -- scores rise; real learning doesn't.

  3. RACIAL/SES BIAS:
     Test scores correlate strongly with income.
     Content may use culturally loaded language.
     "Test-taking skills" advantage wealthy students.
     Stereotype threat (Steele, 1997) affects performance.

  4. RELIABILITY vs. VALIDITY TENSION:
     Standardized tests are highly reliable (consistent).
     May have poor validity for what we care about.
     "What gets measured isn't what matters most."
     Collaboration, creativity, persistence: not tested.

  5. METRIC FIXATION (Muller, "The Tyranny of Metrics"):
     When quantified measures become primary goals:
     Distorts motivation.
     Gaming of measures.
     Obsession with scores crowds out non-measured goods.
```

## Authentic Assessment

Assessment tasks that mirror real-world use of knowledge and skills:

```
  AUTHENTIC ASSESSMENT CHARACTERISTICS
  ========================================

  1. REALISTIC: Tasks resemble real-world problems,
     not artificial test items.

  2. REQUIRES JUDGMENT and innovation:
     No single right answer;
     requires application, not just recall.

  3. STUDENT CONSTRUCTS, not selects:
     Write, design, build, perform, explain.
     Not: choose A, B, C, or D.

  4. REPLICATION OF HOW KNOWLEDGE IS USED:
     Scientists design experiments (not answer MCQs about science).
     Writers write for real audiences (not analyze isolated paragraphs).

  5. CRITERIA KNOWN IN ADVANCE:
     Rubric shared before the task.
     Students can use criteria to guide their work.

  EXAMPLES:
    Portfolio: collection of student work over time.
    Capstone project: integrative, complex task.
    Exhibition: public presentation of work.
    Performance: drama, music, debate.
    Community project: solve a real local problem.
    Design challenge: engineer a solution.
    Literary journal: write and publish student work.
```

### Rubric Design

```
  RUBRIC TYPES
  =============

  ANALYTIC RUBRIC:
    Separate criteria, each scored independently.
    Example: Writing rubric.
    +----------------+----+----+----+----+
    | Criterion      | 4  | 3  | 2  | 1  |
    +----------------+----+----+----+----+
    | Ideas/content  |    |    |    |    |
    | Organization   |    |    |    |    |
    | Voice          |    |    |    |    |
    | Conventions    |    |    |    |    |
    +----------------+----+----+----+----+
    Advantage: Detailed feedback; diagnostic.
    Disadvantage: Time-intensive; may fragment holistic judgment.

  HOLISTIC RUBRIC:
    Single overall score.
    Describes the whole performance at each level.
    Faster to apply; less diagnostic feedback.

  SINGLE-POINT RUBRIC:
    One column describing proficient performance.
    Space for comments: "Not yet" and "Beyond."
    Growth-oriented; focuses attention on the standard.
```

## Psychometric Concepts

### Validity

```
  VALIDITY: "Does this test measure what it claims to measure?"
  =============================================================

  CONTENT VALIDITY:
    Does the test cover the full domain?
    A math test with only geometry doesn't validly
    assess "math" -- it's not representative.

  CONSTRUCT VALIDITY:
    Does the test accurately measure the theoretical
    construct of interest?
    If I'm measuring "critical thinking," does the
    test actually tap critical thinking, or just
    reading comprehension?

  CRITERION VALIDITY:
    Does the test predict an external criterion?
    PREDICTIVE: Does SAT predict college GPA? (Moderate)
    CONCURRENT: Does this new test correlate with the
    established gold standard?

  FACE VALIDITY:
    Does the test appear to measure what it claims?
    Not a technical criterion -- but affects stakeholder
    buy-in. A professional licensing exam that looks
    like it's testing the wrong things will be rejected.

  KEY: A reliable test can be invalid.
       An unreliable test cannot be valid.
```

### Reliability

```
  RELIABILITY: "Does this test produce consistent scores?"
  =========================================================

  TEST-RETEST RELIABILITY:
    Same students; same test; two time points.
    Scores should correlate highly.
    r > 0.80: good; r > 0.90: excellent.

  INTERNAL CONSISTENCY:
    Do items that should measure the same thing
    actually correlate?
    Cronbach's alpha: commonly reported.
    α > 0.70: acceptable for low stakes.

  INTER-RATER RELIABILITY:
    Do different scorers give the same score?
    Critical for essay, performance, portfolio assessment.
    Cohen's kappa; intraclass correlation coefficient.
    Training and calibration sessions required.

  RELIABILITY-VALIDITY TENSION:
    Standardized tests: very high reliability.
    (Same score for same student regardless of scorer)
    Authentic tasks: lower reliability.
    (Different scorers may score differently)
    But: authentic tasks may have higher validity.
    (Actually measure what we care about)
    Trade-off is inherent; cannot be fully resolved.
```

## Standards-Based Grading (SBG)

```
  STANDARDS-BASED GRADING
  ========================

  TRADITIONAL GRADING:
    Points accumulate across all tasks.
    Homework, participation, extra credit included.
    Final grade = average of all points.
    What does a B mean? Unclear.

  STANDARDS-BASED:
    Grade = proficiency on specific standards.
    Each standard scored separately.
    Multiple opportunities to demonstrate proficiency.
    Early low scores overwritten by later evidence.
    Final grade = latest/best evidence of learning.

  ADVANTAGES:
    Clarity: grade means specific competencies.
    Separates behavior (effort, compliance) from learning.
    Growth mindset compatible: not locked into early grade.
    Actionable feedback: "Proficient on X; not yet on Y."

  CHALLENGES:
    Parent/community resistance (unfamiliar).
    Transcript translation for college admissions.
    Record-keeping complexity.
    Teacher buy-in requires significant professional development.
```

## Decision Cheat Sheet

| Need | Assessment Type | Design Features |
|------|----------------|-----------------|
| Adjust tomorrow's instruction | Formative | Hinge questions, exit tickets, cold calls |
| Diagnose prior knowledge | Diagnostic | Pre-test, concept mapping |
| Measure final learning | Summative | Valid coverage, reliable scoring |
| Demonstrate real-world competence | Authentic | Performance task, rubric, portfolio |
| Compare across schools/nations | Standardized | Norm/criterion referenced, sampling frame |
| Provide detailed feedback | Analytic rubric | Separate criteria, levels defined |
| Demonstrate overall proficiency | Holistic rubric | Comprehensive level descriptions |

## Common Confusion Points

**Formative Assessment Is Not More Tests**
The word "assessment" suggests tests, but formative assessment includes everything a teacher does to gather evidence during instruction: observing, questioning, checking, listening. The distinguishing feature is that evidence is used immediately to adjust instruction -- not graded and recorded.

**Reliability Is Not the Same as Validity**
A scale that always shows 10 lbs too heavy is perfectly reliable (consistent) but completely invalid (doesn't measure what it claims). A highly standardized test can have poor validity if it doesn't actually measure the learning it claims to measure.

**High Stakes Create Incentives That Corrupt the Measure**
This is Goodhart's Law applied to education. When test scores become the goal (not the proxy for learning), behaviors emerge that improve scores without improving learning: test prep, score gaming, inappropriate exclusion of low-scoring students from test populations, teaching test-specific vocabulary. Every high-stakes system has to manage these incentive effects.

**Authentic Assessment Is Hard to Scale**
Portfolio and performance assessment have high validity (they show real competence) but require substantial teacher time and training, have lower reliability (harder to score consistently), and don't generalize easily across schools. That is why standardized testing dominates despite its limitations -- it's cheap, fast, and reliable.
