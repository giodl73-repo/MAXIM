# Learning Theory: Behaviorism to Constructivism

## The Big Picture

Learning theory is the attempt to explain what learning is, how it occurs, and what conditions facilitate it. Three major paradigms have dominated: behaviorism (learning as behavior change), cognitive theory (learning as information processing), and constructivism (learning as active knowledge construction). Each generated genuine insights and practical applications; no single paradigm is sufficient.

```
+------------------------------------------------------------------+
|                  LEARNING THEORY LANDSCAPE                       |
+------------------------------------------------------------------+
|                                                                  |
|  BEHAVIORISM (1900s-1960s)                                       |
|  Pavlov, Thorndike, Watson, Skinner                              |
|  Learning = behavior change through conditioning                 |
|  Black box: mental states unobservable, irrelevant               |
|                                                                  |
|  COGNITIVE REVOLUTION (1950s-1990s)                              |
|  Miller, Neisser, Atkinson-Shiffrin                              |
|  Learning = information processing and storage                   |
|  Mental representations are real and observable (via behavior)   |
|                                                                  |
|  CONSTRUCTIVISM (1960s-present)                                  |
|  Piaget (cognitive), Vygotsky (social)                           |
|  Learning = active construction of knowledge                     |
|  Learner builds schemas through interaction with environment     |
|                                                                  |
|  EMBODIED / SITUATED (1990s-present)                             |
|  Lave & Wenger, Gibson                                           |
|  Learning = participation in practice, not just in head          |
|  Cognition is distributed across people, tools, environment      |
+------------------------------------------------------------------+
```

## Behaviorism

### Classical Conditioning (Pavlov, 1890s)

```
  CLASSICAL CONDITIONING STRUCTURE
  ==================================

  BEFORE TRAINING:
    Unconditioned Stimulus (food) --> Unconditioned Response (salivation)
    Neutral Stimulus (bell) --> No response

  DURING TRAINING:
    Neutral Stimulus (bell) + Unconditioned Stimulus (food)
    --> Unconditioned Response (salivation)

  AFTER TRAINING:
    Conditioned Stimulus (bell alone) --> Conditioned Response (salivation)

  KEY CONCEPTS:
    Acquisition: learning the association
    Extinction: CS presented alone repeatedly --> CR diminishes
    Spontaneous recovery: extinguished CR returns after rest
    Generalization: CS-like stimuli also elicit CR
    Discrimination: learning which stimuli do/don't predict UCS

  EDUCATIONAL RELEVANCE:
    Emotional responses to learning environments.
    Test anxiety: neutral stimulus (test room) paired with
    aversive experience --> conditioned fear response.
    Attitude formation: classroom + boring/aversive experience
    --> negative attitude toward learning.
```

### Operant Conditioning (Thorndike, Skinner)

```
  OPERANT CONDITIONING
  =====================

  Thorndike's Law of Effect (1898):
    Responses followed by satisfying consequences:
    more likely to recur.
    Responses followed by annoying consequences:
    less likely to recur.

  Skinner's Extension:
    Behavior is SELECTED by its consequences.
    No mentalism required.
    Contingencies of reinforcement fully explain behavior.

  REINFORCEMENT TYPES:
  +----------------------+------------------+------------------+
  |                      | ADD SOMETHING    | REMOVE SOMETHING |
  +----------------------+------------------+------------------+
  | INCREASES BEHAVIOR   | Positive         | Negative         |
  |                      | Reinforcement    | Reinforcement    |
  |                      | (add reward)     | (remove aversive)|
  +----------------------+------------------+------------------+
  | DECREASES BEHAVIOR   | Positive         | Response Cost /  |
  |                      | Punishment       | Negative Pun.    |
  |                      | (add aversive)   | (remove reward)  |
  +----------------------+------------------+------------------+

  SCHEDULES OF REINFORCEMENT:
    Fixed Ratio (FR): Reinforce after N responses.
    Variable Ratio (VR): Reinforce after average N responses.
    Fixed Interval (FI): Reinforce after time T.
    Variable Interval (VI): Reinforce after average time T.

    VR schedule = strongest, most persistent behavior.
    (Slot machines; social media likes)
    FI schedule = scalloped pattern; burst before interval ends.
    (Studying before exam)
```

### Behavioral Applications in Education

```
  APPLIED BEHAVIOR ANALYSIS (ABA) IN EDUCATION
  ===============================================

  Direct Instruction (Engelmann, Becker):
    Highly structured, scripted instruction.
    Rapid teacher-student exchanges.
    Immediate corrective feedback.
    Mastery criterion before advancing.
    Effective for basic skills (reading, math) especially
    for students with learning difficulties.
    Project Follow Through: largest US education experiment;
    Direct Instruction outperformed other models.

  Token Economy:
    Tokens earned for desired behavior.
    Tokens exchangeable for backup reinforcers.
    Used in special education, behavior management.
    Problem: External motivation can reduce intrinsic motivation
    (overjustification effect: Lepper et al., 1973).

  Programmed Instruction (Skinner):
    Linear or branching sequences.
    Small steps; immediate feedback; self-paced.
    Precursor to computer-based learning.
    Khan Academy is essentially programmed instruction.
```

## Cognitive Learning Theory

The cognitive revolution (1950s) rejected behaviorism's black box. Mental representations, attention, memory, and cognitive processes became scientifically legitimate objects of study.

### Information Processing Model

```
  ATKINSON-SHIFFRIN MULTI-STORE MODEL (1968)
  ===========================================

  ENVIRONMENT
       |
       | Sensory input
       v
  +-----------+
  | SENSORY   |  Capacity: vast
  | REGISTER  |  Duration: <1 second (iconic/echoic)
  +-----------+
       |
       | Attention selects
       v
  +-----------+
  | SHORT-TERM|  Capacity: ~7 +/- 2 items (Miller, 1956)
  | MEMORY    |  Duration: 15-30 seconds without rehearsal
  | (Working  |  Rehearsal: maintenance vs. elaborative
  |  Memory)  |  Entry gate to long-term memory
  +-----------+
       |                    ^
       | Encoding            | Retrieval
       v                    |
  +-----------+
  | LONG-TERM |  Capacity: essentially unlimited
  | MEMORY    |  Duration: lifetime (with consolidation)
  +-----------+  Organization: semantic networks, schemas

  Modern update (Baddeley):
  Working memory is not unitary.
  Components:
  - Phonological loop (verbal information)
  - Visuospatial sketchpad (visual/spatial)
  - Central executive (control/attention)
  - Episodic buffer (integration)
```

### Schema Theory (Bartlett, Rumelhart)

```
  SCHEMAS AS KNOWLEDGE STRUCTURES
  ==================================

  Schema = mental template for a concept,
           event, or procedure.
  Organized knowledge that enables:
  - Rapid recognition
  - Inference beyond given information
  - Efficient storage (chunk, don't store everything)

  Bartlett (1932) "Remembering":
    Subjects recalled "War of the Ghosts" (Native American story)
    Errors were systematic: conformed to English cultural schema.
    Memory is RECONSTRUCTIVE, not reproductive.
    We fill gaps with schema-consistent information.

  EDUCATIONAL IMPLICATIONS:
    Prior knowledge determines what new knowledge means.
    Expert-novice difference is largely schema difference.
    Novices see physics problems by surface features.
    Experts see by deep structure (conservation laws).
    Instruction should build, activate, and correct schemas.
```

## Constructivism

### Cognitive Constructivism (Piaget)

Covered in depth in 02-PIAGET-VYGOTSKY. Core claim:

```
  PIAGET'S CONSTRUCTIVIST PRINCIPLES
  =====================================

  1. ACTIVE CONSTRUCTION:
     Knowledge is not passively received.
     Learner actively builds mental structures.

  2. ASSIMILATION vs. ACCOMMODATION:
     Assimilation: fit new experience into existing schema.
     Accommodation: change schema to fit new experience.
     Equilibration: balance-seeking between the two.

  3. INTRINSIC MOTIVATION:
     Cognitive curiosity drives exploration.
     Optimal challenge: slightly beyond current level.
     (Later: Csikszentmihalyi's "flow" state)

  EDUCATIONAL IMPLICATION:
    Discovery learning: let students explore.
    Teacher as facilitator, not transmitter.
    CRITIQUE: Minimal guidance discovery often fails
    for novices. Explicit instruction + scaffolding needed.
```

### Social Constructivism (Vygotsky)

Covered in depth in 02-PIAGET-VYGOTSKY. Core additions:

```
  VYGOTSKY'S SOCIAL DIMENSION
  =============================

  Learning happens FIRST socially (inter-psychological),
  then becomes internalized individually (intra-psychological).

  Language and thought are deeply connected.
  Social speech --> Private speech (self-talk) --> Inner speech.
  Thought develops through language.

  ZPD: What I can do with support today,
       I can do independently tomorrow.
  Scaffolding: Expert support that is gradually withdrawn
               as competence develops.

  EDUCATIONAL IMPLICATION:
    Peer collaboration.
    Scaffolded instruction.
    Teacher as knowledgeable other.
    Social context matters for learning.
```

## Humanistic Learning Theory (Maslow, Rogers)

```
  HUMANISTIC APPROACH
  ====================

  Maslow's Hierarchy of Needs:
    Physiological --> Safety --> Love/Belonging
    --> Esteem --> Self-Actualization

    Implication: If lower needs are unmet, learning
    is impaired. A hungry, unsafe, or socially
    isolated child cannot learn optimally.
    Schools must attend to whole person.

  Carl Rogers: Student-Centered Learning
    "Significant learning takes place when
    the subject matter is relevant to the
    personal interests of the student."

    Conditions: psychological safety, empathy,
    genuine regard, authenticity.

    Teacher role: facilitator of learning, not
    transmitter of knowledge.

    Critique: Warm conditions are necessary but
    not sufficient. Evidence shows structured
    explicit instruction also needed, especially
    for basic skills and novice learners.
```

## Situated and Distributed Cognition

```
  SITUATED LEARNING (Lave & Wenger, 1991)
  ==========================================

  "Legitimate Peripheral Participation":
    Learning is participation in communities of practice.
    Not transmission of abstract knowledge.
    Apprenticeship model: newcomer participates at
    periphery, moves toward full participation.

  Examples:
    Tailors' apprentices: observe, then assist,
    then produce with decreasing supervision.
    Butchers' apprentices: package first, then cut.
    Medical interns: observe, then assist, then lead.

  CRITIQUE OF SCHOOL:
    School decouples knowledge from practice.
    "Inert knowledge": students know the formula
    but can't apply it to real problems.
    (Whitehead, 1929: "inert ideas")
    Transfer is poor from school to non-school contexts.

  DISTRIBUTED COGNITION (Hutchins):
    Cognition is not just in the head.
    It is distributed across: people + tools + environment.
    Ships' navigation: distributed across instruments,
    charts, crew, procedures.
    Educational implication: tools matter;
    working in authentic environments matters.
```

## Connectivism (Siemens, 2005)

Proposed specifically for the digital age:

```
  CONNECTIVISM (Siemens, 2005)
  ==============================

  Core claims:
  1. Learning occurs across networks of connections.
  2. Current knowledge is more important than past knowledge.
  3. Maintaining connections to information sources is key.
  4. Decision-making is itself a learning process.

  CRITIQUE:
    Considered by many as a pedagogical approach
    not a learning theory (no novel explanatory mechanism).
    Similarity to earlier network/constructivist ideas.
    Doesn't address how individuals actually learn
    from networks -- just that they do.

  LEGACY:
    Siemens + Downes designed the first MOOC (2008).
    The cMOOC model (connectivist MOOC) remains the
    original, more participatory form vs. xMOOC (video lectures).
```

## Decision Cheat Sheet

| Theory | What It Explains Well | What It Misses | Best Applied To |
|--------|----------------------|----------------|-----------------|
| Classical conditioning | Emotional responses to learning | Intentional learning | Anxiety, motivation, environment |
| Operant conditioning | Skill acquisition, behavior management | Complex cognition | Basic skills, behavior support |
| Information processing | Memory limitations, cognitive load | Social/contextual factors | Instructional design |
| Schema theory | Prior knowledge effects | Affective factors | Building on existing knowledge |
| Piaget constructivism | Active knowledge construction | Social dimension | Developmental tasks, science ed |
| Vygotsky social | Social learning, scaffolding | Individual differences | Collaborative learning, language |
| Humanistic | Motivation, whole person | Mechanisms of learning | Emotional safety, engagement |
| Situated learning | Real-world application, transfer | Scalability | Apprenticeship, vocational |

## Common Confusion Points

**Constructivism ≠ Discovery Learning (Necessarily)**
This is the most important misconception in contemporary education. Constructivism is a theory of how learning works (learners build knowledge). Minimal guidance discovery learning is one pedagogical approach sometimes derived from it. The evidence shows discovery learning with little guidance fails, especially for novices. Explicit instruction can be constructivist when it helps learners build accurate schemas.

**Behaviorism Is Not Dead**
Applied Behavior Analysis is the only treatment with strong evidence for autism spectrum disorder. Direct Instruction has strong evidence for basic literacy and numeracy. Token economies work in behavior management. Behaviorism was wrong that mentalism is unscientific -- not wrong that consequences shape behavior.

**Working Memory Is the Bottleneck**
The 7 +/- 2 items limit (Miller) is probably too generous for meaningful learning. For complex material, effective working memory capacity is ~4 chunks (Cowan) or even less. Cognitive load theory is the primary instructional design framework for managing this constraint. Every instructional design decision should consider WM load.

**Transfer Is the Hard Problem**
Learning something is not the same as being able to use it in new contexts. Near transfer (same domain, slightly different surface) is moderate. Far transfer (different domain) is rare and hard to predict. "Teaching for transfer" is one of the hardest problems in instructional design.
