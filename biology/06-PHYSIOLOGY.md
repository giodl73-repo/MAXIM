## Big Picture: The Body as an Integrated Control System

```
PHYSIOLOGY: How the body's organs work individually and together
            to maintain a stable internal environment (homeostasis)

HOMEOSTASIS PRINCIPLE (Claude Bernard, Walter Cannon):
  Maintain constant internal environment despite external variation
  Mechanism: negative feedback loops
  Variable: set point → sensor → comparator → effector → correction

  Example: Core temperature regulation
  Set point: 37°C
  Sensor: thermoreceptors (skin + hypothalamus)
  Comparator: hypothalamus (thermostat)
  Effectors: sweating (cool), shivering (warm), vasodilation/constriction

ORGAN SYSTEMS AND THEIR FUNCTIONS:
  ┌──────────────────────────────────────────────────────────────┐
  │ Nervous:       rapid signal integration; sensory + motor     │
  │ Endocrine:     slow chemical signals; long-range coordination│
  │ Cardiovascular:pump + distribute O₂, nutrients, hormones     │
  │ Respiratory:   O₂/CO₂ exchange; acid-base balance           │
  │ Digestive:     absorb nutrients; eliminate waste             │
  │ Renal (kidney):filter blood; regulate fluid/electrolyte/pH  │
  │ Immune:        defense against pathogens; self/non-self      │
  │ Musculoskeletal:movement, posture, calcium reservoir         │
  │ Reproductive:  gamete production; development                │
  └──────────────────────────────────────────────────────────────┘

INTEGRATION:
  Nervous + Endocrine work together (neuroendocrine axis)
  Cardiovascular + Respiratory tightly coupled (O₂ delivery)
  Kidney + Cardiovascular: blood pressure regulation
  Immune + all others: recognizes and responds to damage
```

## Nervous System

```
CENTRAL vs PERIPHERAL:
  CNS: brain + spinal cord
  PNS: sensory + motor neurons; Autonomic + Somatic divisions

AUTONOMIC NERVOUS SYSTEM:
  Sympathetic ("fight or flight"):
    Preganglionic: ACh (nicotinic receptor)
    Postganglionic: norepinephrine (adrenergic receptors α₁, α₂, β₁, β₂)
    Effects: ↑ heart rate, ↑ BP, bronchodilation, ↑ glucose (glycogenolysis)

  Parasympathetic ("rest and digest"):
    Preganglionic + postganglionic: ACh (muscarinic receptor)
    Effects: ↓ heart rate, ↑ GI motility, bronchoconstriction

  Enteric: gut's own neural network (~500M neurons); works semi-autonomously

ACTION POTENTIAL:
  Resting: −70 mV (inside negative); maintained by Na⁺/K⁺ ATPase
  Threshold: ~−55 mV
  Rising phase: voltage-gated Na⁺ channels open → Na⁺ influx → depolarization
  Peak: ~+40 mV; Na⁺ channels inactivate
  Falling: K⁺ channels open → K⁺ efflux → repolarization
  Undershoot: K⁺ channels slow to close → brief hyperpolarization
  Absolute refractory period: no AP possible (Na⁺ channels inactivated)
  Relative refractory: AP possible with stronger stimulus (K⁺ channels still open)

  Conduction velocity:
    Unmyelinated (C fibers): 0.5-2 m/s (pain, temperature)
    Myelinated (Aβ): 30-70 m/s (touch, pressure)
    Myelinated (Aα): 70-120 m/s (proprioception, motor)
    Saltatory conduction: AP jumps node-to-node in myelinated axons

SYNAPTIC TRANSMISSION:
  AP arrives → voltage-gated Ca²⁺ channels open → Ca²⁺ influx
  → synaptic vesicles fuse (SNARE proteins) → neurotransmitter release
  → binds postsynaptic receptor → EPSP (excitatory) or IPSP (inhibitory)

  Key neurotransmitters:
    Glutamate: primary excitatory (AMPA, NMDA, kainate receptors)
    GABA: primary inhibitory (GABA-A ionotropic, GABA-B metabotropic)
    Acetylcholine (ACh): neuromuscular junction, autonomic, CNS
    Dopamine: reward, motor control (nigrostriatal), PFC (cognitive)
    Serotonin (5-HT): mood, sleep, appetite (gut-brain)
    Norepinephrine: arousal, attention, sympathetic

BRAIN REGIONS (functional map):
  Cerebral cortex: perception, cognition, voluntary movement
    Frontal: executive function, motor planning, working memory, Broca's area
    Parietal: somatosensory, spatial processing
    Temporal: auditory, language comprehension (Wernicke's), memory
    Occipital: visual processing
  Hippocampus: spatial memory, episodic memory consolidation (H.M. case)
  Amygdala: fear, emotion, threat detection
  Basal ganglia: motor learning, habit formation, reward circuits
  Cerebellum: motor coordination, balance, motor learning
  Thalamus: sensory relay (all sensory except olfaction)
  Hypothalamus: homeostasis (temperature, hunger, thirst, sleep), HPA axis
  Brainstem: vital functions (respiration, heart rate, consciousness, cranial nerves)

SYNAPTIC PLASTICITY:
  LTP (Long-Term Potentiation): repeated stimulation → stronger synapse
    Mechanism: NMDA receptor Mg²⁺ block relieved → Ca²⁺ influx
    → AMPA receptor insertion → CaMKII activation → synapse strengthening
    LTP = cellular correlate of learning and memory
  LTD (Long-Term Depression): weak/uncorrelated stimulation → weaker synapse
  Hebbian plasticity: "neurons that fire together wire together"
```

## Cardiovascular System

```
HEART ANATOMY AND FUNCTION:
  4 chambers: RA/LA (atria, receive blood) + RV/LV (ventricles, pump blood)
  Right: receives deoxygenated blood from body → pumps to lungs (pulmonary)
  Left: receives oxygenated blood from lungs → pumps to body (systemic)
  LV wall 3× thicker than RV (must generate higher pressure for systemic)

  Cardiac cycle:
    Systole: ventricular contraction → ejection
    Diastole: ventricular relaxation → filling
    Heart sounds: S1 (AV valves close, beginning systole), S2 (semilunar valves close)
    Normal cardiac output: ~5 L/min at rest; 20-25 L/min during exercise
    CO = HR × SV (heart rate × stroke volume)

ELECTRICAL CONDUCTION:
  SA node (pacemaker) → AV node → Bundle of His → Purkinje fibers
  SA node: intrinsic rate ~60-100 bpm; modified by ANS
  AV node: delays conduction → ventricles fill before contracting
  ECG:
    P wave: atrial depolarization
    QRS complex: ventricular depolarization (atrial repolarization buried)
    T wave: ventricular repolarization
    PR interval: AV node delay (~0.12-0.20 s)

BLOOD PRESSURE:
  Systolic/Diastolic: ~120/80 mmHg normal
  Mean arterial pressure MAP = Diastolic + (Pulse pressure)/3 ≈ 93 mmHg
  MAP = CO × Total Peripheral Resistance (TPR)
  Regulation:
    Baroreflex (fast, seconds): carotid sinus/aortic arch baroreceptors → brainstem
    RAAS (renin-angiotensin-aldosterone, minutes-hours): angiotensin II → vasoconstriction
    Vasopressin (ADH): antidiuretic hormone → water retention
    Atrial natriuretic peptide (ANP): ↓ BP (released by atria with distension)

MICROCIRCULATION:
  Arteries → arterioles (resistance vessels, control distribution) → capillaries
  → venules → veins (capacitance vessels, ~70% of blood volume at rest)
  Starling forces across capillary wall:
    Filtration pressure: hydrostatic (pushes out) − oncotic (pulls in)
    Net filtration at arterial end; net reabsorption at venous end
    Excess → lymphatics → lymph nodes → thoracic duct → venous return
```

## Respiratory System

```
LUNG MECHANICS:
  Tidal volume (TV): ~500 mL per breath (at rest)
  Vital capacity (VC): ~4.8 L (TV + IRV + ERV)
  Residual volume (RV): ~1.2 L (cannot be expelled)
  Total lung capacity (TLC): ~6 L

  Breathing mechanics:
    Inspiration: diaphragm contracts → thorax expands → intrapulmonary pressure ↓
    → air flows in (Boyle's law: P₁V₁ = P₂V₂)
    Expiration: diaphragm relaxes → elastic recoil → passive expiration
    Accessory muscles for forced expiration (internal intercostals, abs)

  Surfactant (Type II pneumocytes): reduces surface tension in alveoli
    LaPlace's law: P = 2T/r (smaller alveoli → higher pressure → collapse without surfactant)
    Newborn: insufficient surfactant → respiratory distress syndrome (RDS)

GAS EXCHANGE:
  Alveolar-capillary membrane: ~0.5 μm thick; ~70 m² surface area
  Diffusion: O₂ and CO₂ across membrane (Fick's law: J = D·A·ΔP/thickness)
  Ventilation-perfusion (V/Q) matching: blood flow must match air flow
    V/Q = 1: ideal
    V/Q mismatch → shunt (V/Q → 0) or dead space (V/Q → ∞) → hypoxemia

O₂ TRANSPORT:
  Hemoglobin (Hb): 4 subunits (2α + 2β), each with heme group (Fe²⁺)
  O₂ saturation curve: sigmoidal (cooperative binding — all-or-none)
  p50: PO₂ at which Hb is 50% saturated (normally ~27 mmHg)
  Bohr effect: ↑ CO₂/↑ H⁺/↑ temperature → right shift (↓ affinity) → O₂ release in tissues
  2,3-DPG: allosteric effector → right shift (favors O₂ delivery in chronic hypoxia)

CO₂ TRANSPORT:
  ~70% as bicarbonate (CO₂ + H₂O ⇌ H₂CO₃ ⇌ H⁺ + HCO₃⁻, catalyzed by carbonic anhydrase)
  ~23% as carbaminohemoglobin (CO₂ binds Hb N-terminal)
  ~7% dissolved in plasma

ACID-BASE BALANCE:
  Normal pH: 7.35-7.45
  Respiratory acidosis: ↑ CO₂ (hypoventilation) → ↑ H⁺
  Respiratory alkalosis: ↓ CO₂ (hyperventilation)
  Metabolic acidosis: ↓ HCO₃⁻ (diabetic ketoacidosis, lactic acidosis)
  Metabolic alkalosis: ↑ HCO₃⁻ (vomiting, diuretics)
  Compensation: respiratory adjusts pH in minutes; renal in hours-days
```

## Renal System

```
KIDNEY FUNCTION:
  Filter ~180 L/day; excrete ~1-2 L urine
  Filter small molecules (glucose, amino acids, ions, waste) at glomerulus
  Reclaim valuable molecules in tubules; excrete waste

NEPHRON (functional unit, ~1 million per kidney):
  Glomerulus → Bowman's capsule → Proximal convoluted tubule
  → Loop of Henle → Distal convoluted tubule → Collecting duct

  GFR (glomerular filtration rate): ~125 mL/min (~180 L/day filtered)
  Starling forces at glomerulus: filtration (high hydrostatic) vs retention (oncotic)

TUBULAR HANDLING:
  Proximal tubule: reabsorb ~65% Na⁺, all glucose, amino acids, HCO₃⁻
    Cotransport: Na⁺ gradient drives glucose/AA reabsorption (SGLT transporters)
    Isosmotic reabsorption: water follows Na⁺
  Loop of Henle (countercurrent multiplier):
    Descending: permeable to H₂O (concentrates tubular fluid)
    Ascending: impermeable to H₂O but actively pumps Na⁺/K⁺/Cl⁻ out
    → creates medullary osmotic gradient (up to 1200 mOsm in papilla vs 300 in cortex)
  Collecting duct: ADH → insert aquaporin-2 → concentrate urine
    Aldosterone → insert ENaC Na⁺ channels → Na⁺ reabsorption + K⁺ excretion

FLUID AND ELECTROLYTE REGULATION:
  Na⁺ balance: ADH (controls water) + RAAS (controls Na⁺) together control ECF volume
  K⁺ balance: aldosterone controls K⁺ excretion; intracellular insulin/epinephrine shift
  Ca²⁺: PTH ↑, calcitonin ↓; vitamin D ↑ gut absorption
  Phosphate: PTH ↓ reabsorption; vitamin D ↑ absorption
  Mg²⁺: mostly freely regulated; hypomagnesemia → impaired PTH secretion

CLINICAL KEY NUMBERS:
  BUN (blood urea nitrogen): waste marker (normal 7-20 mg/dL)
  Creatinine: muscle waste; inversely proportional to GFR (normal 0.6-1.2 mg/dL)
  GFR by CKD stage:
    CKD1: ≥90 mL/min (normal/high with other damage marker)
    CKD3b: 30-44 mL/min (moderate-severe)
    CKD5: <15 mL/min (kidney failure, dialysis threshold)
```

## Immune System

```
INNATE vs ADAPTIVE:
  Innate:   fast (minutes-hours); germline-encoded; non-specific; no memory
  Adaptive: slow (days-weeks first exposure); generated by recombination; specific; memory

INNATE IMMUNITY:
  Physical barriers: skin, mucous membranes, cilia, stomach acid
  Pattern recognition receptors (PRRs):
    Toll-like receptors (TLRs): detect PAMPs (pathogen-associated molecular patterns)
      TLR4: LPS (Gram-negative bacteria)
      TLR3: dsRNA (viruses)
    NOD-like receptors (NLRs): cytosolic; inflammasome assembly → IL-1β, IL-18, pyroptosis
    RIG-I: cytosolic RNA sensing → interferon response
  DAMP (damage-associated molecular patterns): from damaged cells → sterile inflammation

  Complement system: cascade of ~30 proteins
    Classical: antibody binds → C1q activation
    Alternative: spontaneous C3 hydrolysis → amplification loop
    Lectin: MBL binds mannose on pathogens → MASP1/2 activation
    MAC (membrane attack complex): lyse bacteria
    Opsonization: C3b-coated bacteria → phagocytosis

  Phagocytes: neutrophils (acute, short-lived), macrophages (long-lived, APC)
  NK cells: kill virus-infected or tumor cells (missing self — no MHC I → kill)

ADAPTIVE IMMUNITY:
  B cells: antibody production; mature in bone marrow
    Antibody isotypes:
      IgM: first produced; good at complement activation; pentamer
      IgG: most abundant in serum; long-lived; transplacental; 4 subclasses
      IgA: mucosal immunity; secreted in milk, saliva, tears; dimer
      IgE: allergy; mast cell/basophil degranulation
      IgD: B cell antigen receptor; function unclear

  T cells: cell-mediated; mature in thymus
    CD4⁺ T helpers (Th): coordinate immune response; recognize antigen via MHC II
      Th1: activate macrophages (IFN-γ) → intracellular pathogens
      Th2: activate eosinophils/mast cells (IL-4/IL-5/IL-13) → helminths/allergy
      Th17: IL-17 → neutrophils → extracellular bacteria/fungi
      Treg: suppress immune response (IL-10, TGF-β) → self-tolerance
    CD8⁺ Cytotoxic T cells (CTL): kill infected cells; recognize antigen via MHC I
      Mechanism: perforin + granzyme → target cell apoptosis; Fas-FasL

MHC (MAJOR HISTOCOMPATIBILITY COMPLEX):
  MHC I: on all nucleated cells; presents intracellular peptides → CD8+ T cells
  MHC II: on APCs (macrophages, DCs, B cells); presents extracellular peptides → CD4+ T cells
  HLA (human MHC): most polymorphic locus in human genome
  Transplant rejection: T cells recognize foreign HLA → immune attack
  Autoimmune disease: T cells recognize self-peptides (HLA association with risk)

B CELL AFFINITY MATURATION AND MEMORY:
  Germinal center: B cell clones → somatic hypermutation → selection by antigen
  → affinity maturation (antibodies improve over course of infection)
  Memory cells: long-lived; rapid high-affinity response upon re-exposure
  Vaccination: prime memory without disease
```

## Endocrine System

```
HORMONE TYPES AND SIGNALING:
  Peptide hormones (water-soluble): insulin, glucagon, GH, FSH, LH, PTH
    Cannot cross membrane → bind surface receptor → intracellular signaling cascade
    cAMP: GPCR → adenylyl cyclase → PKA pathway (glucagon, epinephrine)
    IP₃/DAG: GPCR (Gq) → PLC → IP₃ (Ca²⁺ release) + DAG (PKC)
    JAK-STAT: cytokine receptors, growth hormone
    Tyrosine kinase: insulin receptor → PI3K-AKT-mTOR + MAPK

  Steroid hormones (lipid-soluble): cortisol, aldosterone, estrogen, testosterone, vitamin D
    Cross membrane → bind intracellular receptor → nuclear translocation
    → transcription factor → changes gene expression (hours timescale)

HYPOTHALAMIC-PITUITARY-ADRENAL (HPA) AXIS:
  Stress → hypothalamus → CRH → anterior pituitary → ACTH → adrenal cortex
  Cortisol (glucocorticoid) released:
    Effects: ↑ gluconeogenesis, ↑ blood glucose, ↑ lipolysis, immunosuppression
    Negative feedback: cortisol → hypothalamus + pituitary → reduce CRH/ACTH
    Circadian rhythm: cortisol peaks in morning (awakening response)
  Chronic stress: cortisol chronically elevated → insulin resistance, immunosuppression,
    hippocampal volume reduction

HYPOTHALAMIC-PITUITARY-THYROID (HPT) AXIS:
  Hypothalamus → TRH → anterior pituitary → TSH → thyroid gland
  T₃ (triiodothyronine) + T₄ (thyroxine) released
  T₄ mostly inactive (prohormone); peripheral deiodination → T₃ (active)
  Effects: ↑ basal metabolic rate, heart rate, temperature, GI motility
  Hypothyroidism: weight gain, fatigue, cold intolerance, bradycardia
  Hyperthyroidism (Graves): weight loss, tremor, tachycardia, exophthalmos

REPRODUCTIVE AXIS (HPG):
  Hypothalamus → GnRH (pulsatile) → LH + FSH → gonads → sex steroids
  Female:
    Follicular phase: FSH → estradiol → LH surge → ovulation
    Luteal phase: progesterone → prepare endometrium
    Menopause: follicles exhausted → estrogen ↓ → FSH rises (negative feedback lost)
  Male: LH → Leydig cells → testosterone; FSH → Sertoli cells → spermatogenesis

PANCREATIC HORMONES — GLUCOSE HOMEOSTASIS:
  Insulin (β cells): released by ↑ blood glucose
    Promotes: glucose uptake (GLUT4 in muscle/fat), glycogenesis, lipogenesis
    Suppresses: gluconeogenesis, lipolysis, ketogenesis
    Mechanism: IR → IRS → PI3K → PIP3 → PDK1 → AKT → GLUT4 vesicle fusion
  Glucagon (α cells): released by ↓ blood glucose
    Promotes: glycogenolysis, gluconeogenesis, lipolysis, ketogenesis
    Mechanism: GPCR → cAMP → PKA → phosphorylase kinase → glycogen phosphorylase

  Type 1 diabetes: autoimmune destruction of β cells → absolute insulin deficiency
  Type 2 diabetes: insulin resistance → β cell exhaustion → relative insulin deficiency
```

## Decision Cheat Sheet

| Physiological Goal | System | Key Mechanism |
|--------------------|--------|---------------|
| Maintain blood pressure | Cardiovascular + Renal | Baroreflex (fast) + RAAS (slow) |
| Regulate blood glucose | Endocrine | Insulin vs Glucagon balance |
| Regulate body temperature | Hypothalamus + ANS | Sweating, shivering, vasodilation |
| Respond to infection | Immune | Innate (fast/broad) → Adaptive (slow/specific) |
| Control acid-base | Respiratory + Renal | Respiratory (minutes) + Bicarbonate (hours-days) |
| Coordinate fight-or-flight | ANS (sympathetic) + Adrenal | Epinephrine/cortisol release |
| Long-term reproduction/metabolism | HPG/HPT/HPA axes | Feedback loops with brain |
| Filter blood; water balance | Kidney | ADH for water; Aldosterone for Na⁺ |
| Deliver O₂ to tissues | Cardiovascular + Hb | Bohr effect, O₂ saturation curve |
| Remember/fight pathogen again | Adaptive immune memory | Long-lived B + T memory cells |

## Common Confusion Points

**ADH (vasopressin) controls water, not Na⁺**: ADH inserts aquaporins → concentrates urine → retains water. Aldosterone retains Na⁺ (and loses K⁺). High ADH → dilutional hyponatremia if water retained without Na⁺. These are distinct pathways.

**MHC I vs MHC II**: I = intracellular (viral peptides, tumor antigens) → CD8+ T cells kill infected cells. II = extracellular (phagocytosed bacteria) → CD4+ T cells coordinate response. Confusing them leads to wrong immune prediction.

**Innate and adaptive work together**: Innate inflammation creates the context for adaptive response. Dendritic cells (innate) present antigen to T cells (adaptive). Adaptive antibodies recruit innate effectors (complement, ADCC). They're not separate — they're a sequential + cooperative system.

**Cortisol is anti-inflammatory acutely but causes inflammation chronically**: Short-term cortisol suppresses acute inflammation (used for asthma, autoimmune disease). Chronically elevated cortisol → immune dysregulation, metabolic disease, hippocampal damage.

**Bohr effect**: Right shift of O₂-Hb dissociation curve means Hb has lower O₂ affinity (releases O₂). CO₂ and H⁺ produced by exercising muscle cause right shift → deliver more O₂ exactly where it's needed. Not the same as competitive inhibition.

**Affinity maturation happens AFTER primary response**: First encounter → B cells clone expand → germinal center → somatic hypermutation → affinity maturation → memory cells. Second encounter → memory B cells give faster, higher-affinity (already matured) response. This is why second vaccination often more effective.

**T₄ is a prohormone**: Most thyroid hormone circulates as T₄ (inactive). Deiodinases in target tissues convert T₄ → T₃ (active). TSH measurements reflect pituitary response to circulating T₄ levels. Low T₃ activity can occur with normal T₄ if deiodinase activity is low (sick euthyroid syndrome).
