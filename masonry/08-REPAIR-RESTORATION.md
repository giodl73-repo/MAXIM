# Masonry Repair and Restoration

## The Big Picture

Masonry repair is a diagnostic-first discipline. The wrong repair — particularly incompatible mortar — causes more damage than doing nothing. Before touching a historic masonry building, determine what's failing and why, then match the repair to the cause and the original material.

```
REPAIR DECISION FRAMEWORK
==========================

  OBSERVE (symptoms)
  ↓
  DIAGNOSE (causes — do not skip this)
  ↓
  TEST (material compatibility)
  ↓
  SPECIFY (appropriate repair)
  ↓
  EXECUTE (minimum intervention)
  ↓
  MONITOR (ensure repair is working)

  MOST COMMON ERRORS:
  ┌──────────────────────────────────────────────────────────────────┐
  │  ① Repointing with Portland-rich mortar on soft historic brick │
  │    → mortar harder than unit → brick face spalls               │
  │    (correct: match original mortar or use NHL/lime putty)        │
  │                                                                  │
  │  ② Applying sealants to masonry                                │
  │    → traps moisture inside → freeze-thaw destroys from within  │
  │    (masonry must breathe — vapor-permeable coatings only)        │
  │                                                                  │
  │  ③ Pressure-washing historic brick                             │
  │    → high pressure damages fired face, drives water into joints  │
  │    (correct: low-pressure water + gentle chemical cleaning)      │
  │                                                                  │
  │  ④ Sandblasting historic stone/brick                           │
  │    → removes hardened surface layer; accelerates weathering      │
  │    (prohibited by Secretary of Interior Standards)               │
  └──────────────────────────────────────────────────────────────────┘
```

## Repointing Procedure

Repointing is the most common masonry repair — removing deteriorated mortar from joints and replacing with new mortar. Done wrong, it's one of the most damaging interventions possible.

```
REPOINTING PROCEDURE
====================

  STEP 1: Joint preparation
  ┌──────────────────────────────────────────────────────────────────┐
  │  RAKE DEPTH REQUIREMENT:                                         │
  │  Minimum: 3/4" depth below unit face                           │
  │  ┌────────────────────────────────────────────────────────────┐  │
  │  │  [BRICK]                          [BRICK]                  │ │
  │  │          ← 3/4" min →                                      │ │
  │  │  ════════════════════════════════════════  ← original mortar│ │
  │  │          ← remove to here →                                │ │
  │  └────────────────────────────────────────────────────────────┘ │
  │                                                                   │
  │  WHY 3/4" minimum:                                              │
  │  • Provides enough depth for new mortar to bond mechanically    │
  │  • Allows new mortar to cure properly (surface-to-depth ratio)  │
  │  • Shallower repointing falls out within 5–10 years             │
  │                                                                   │
  │  TOOLS: Cold chisel + hammer (preferred for historic)           │
  │         Oscillating tool with carbide blade (for CMU)           │
  │         NEVER: angle grinder with diamond blade                 │
  │         (cannot avoid cutting into brick face)                  │
  └──────────────────────────────────────────────────────────────────┘

  STEP 2: Joint cleaning
  Remove all loose mortar, dust, debris
  Blow out joints with compressed air
  Do NOT wet-blast — can drive contamination deeper

  STEP 3: Pre-wetting
  Dampen joints with water spray (fogger preferred)
  Substrate must be damp — not saturated, not dry
  Dry substrate draws water from new mortar → poor bond + cracking

  STEP 4: Mortar application
  Apply in layers if joint depth > 3/4":
    First pass: fill to within 1/4" of face (allowed to firm up)
    Second pass: fill to within 1/8" of face
    Final pass: fill flush and strike
  Press mortar firmly into joint (consolidate)
  Use pointing iron or margin trowel sized to fit joint

  STEP 5: Tooling (striking)
  Tool after mortar begins to set but before initial set
  Test: leave a thumbprint — if it springs back, mortar too soft;
        if crumbles, too hard; slight indentation = correct timing
  Strike to match original joint profile

  STEP 6: Cure protection
  Lime mortars: mist with water 2× daily for 3–5 days (slow cure)
  Portland mortars: 24 hr moisture protection
  Hot weather: shade joint from direct sun (prevents rapid drying)
  Cold weather: protect from freezing for 24–48 hrs minimum
```

## Compatibility Testing

Before specifying repointing mortar for historic masonry, test compatibility.

```
COMPATIBILITY TEST PROTOCOL
============================

  TEST 1: HARDNESS (Moh's scratch) — quick field test
  ┌──────────────────────────────────────────────────────────────────┐
  │  Scratch existing mortar with fingernail (Mohs ~2.5)           │
  │  Scratch existing mortar with knife blade (Mohs ~5.5)          │
  │                                                                  │
  │  Result interpretation:                                          │
  │  Fingernail scratches → very soft (Type O / lime putty)        │
  │  Knife scratches, fingernail doesn't → moderate (Type N)       │
  │  Neither scratches → hard (Type S or M Portland mortar)        │
  │                                                                  │
  │  Replacement mortar: same hardness or slightly softer          │
  └──────────────────────────────────────────────────────────────────┘

  TEST 2: PETROGRAPHIC ANALYSIS (ASTM C856) — definitive
  ┌──────────────────────────────────────────────────────────────────┐
  │  Thin-section microscopy of extracted mortar sample            │
  │  Identifies: binder type, aggregate type and size,             │
  │    approximate proportions, signs of deterioration             │
  │  Cost: $300–600 per sample; usually worthwhile for             │
  │    significant historic structures                               │
  └──────────────────────────────────────────────────────────────────┘

  TEST 3: MOCK-UP PANEL
  ┌──────────────────────────────────────────────────────────────────┐
  │  Test 3–5 candidate mortar formulations on inconspicuous area  │
  │  Evaluate after 28 days (Portland) or 90 days (lime):          │
  │  • Color and texture match                                       │
  │  • Bond quality                                                  │
  │  • Workability                                                   │
  └──────────────────────────────────────────────────────────────────┘
```

## Crack Diagnosis

Crack patterns in masonry are diagnostic. The pattern reveals the cause. Fix the cause, not just the crack.

```
CRACK PATTERN DIAGNOSIS
========================

  VERTICAL CRACKS:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Pattern:  ┃  (vertical, through mortar joints or units)       │
  │                                                                  │
  │  DIFFERENTIAL SETTLEMENT: vertical crack where two sections    │
  │    of building move independently (e.g., at addition/original) │
  │  THERMAL EXPANSION: long wall with no expansion joint; mortar  │
  │    joints in compression → failure in tension at ends          │
  │  LINTEL DEFLECTION: vertical crack at corners of openings      │
  │    → lintel rotating at support                                  │
  │                                                                  │
  │  FIX: Address cause first. Expansion joints if thermal.        │
  │       Monitor settlement before repair (may still be moving).  │
  └──────────────────────────────────────────────────────────────────┘

  DIAGONAL (STAIR-STEP) CRACKS:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Pattern: steps along mortar joints diagonally                 │
  │  ┌─┬─┬─┐                                                         │
  │  │ │╲│ │                                                         │
  │  ├─┴─┼─┤                                                         │
  │  │  ╲│ │                                                         │
  │  └───┴─┘                                                         │
  │                                                                   │
  │  DIFFERENTIAL SETTLEMENT: one end dropping faster              │
  │  FOUNDATION MOVEMENT: corner settlement most common             │
  │  THERMAL RACKING: long, low wall racking under thermal         │
  │                                                                   │
  │  FIX: Investigate and correct foundation. Do not repoint until │
  │       movement has stopped (typically 2–5 years monitoring).   │
  └──────────────────────────────────────────────────────────────────┘

  HORIZONTAL CRACKS:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Pattern: ─────────────── (horizontal, at bed joints)            │
  │                                                                  │
  │  BED JOINT SLIDING: lateral seismic force exceeding mortar     │
  │    shear strength → wall slides on bed joint                   │
  │  CORRODED LINTEL/FLASHING: steel corrosion expands, pushes     │
  │    brickwork apart horizontally above lintel                     │
  │  FROST HEAVE: saturated soil expands upward into foundation    │
  │  OVERLOADED WALL: axial + bending exceeds mortar tensile bond  │
  │                                                                  │
  │  FIX: Seismic → reinforce and anchor. Steel → replace corroded │
  │       metal (flashing, lintels). Frost → drainage correction.  │
  └──────────────────────────────────────────────────────────────────┘

  MAP CRACKING (crazing):
  ┌──────────────────────────────────────────────────────────────────┐
  │  Pattern: random network of fine cracks over surface           │
  │  CAUSE: usually plaster or stucco over masonry — shrinkage     │
  │    during curing, or differential thermal movement             │
  │  Less common: surface scaling of brick from freeze-thaw        │
  │  FIX: If in stucco, reapply with proper mesh reinforcement.   │
  │       If in brick face, evaluate unit absorption and freeze-thaw│
  └──────────────────────────────────────────────────────────────────┘
```

## Efflorescence: Causes and Treatment

Efflorescence is the white crystalline deposit that forms on masonry surfaces. It is a symptom of moisture movement through the wall, not a structural problem, but it indicates a water management failure.

```
EFFLORESCENCE MECHANISM
========================

  ┌──────────────────────────────────────────────────────────────────┐
  │  1. Water enters masonry (rain, ground moisture, condensation)   │
  │  2. Water dissolves soluble salts in mortar, brick, or concrete  │
  │     (calcium sulfate, sodium sulfate, potassium carbonate)       │
  │  3. Water migrates through masonry to external face              │
  │  4. Water evaporates at face → salts crystallize on surface    │
  │  5. Crystals grow → white powdery or fluffy deposit            │
  └──────────────────────────────────────────────────────────────────┘

  PRIMARY EFFLORESCENCE: first flush from new masonry
  (soluble salts in cement + mortar) — usually disappears in 1–2 years

  SECONDARY EFFLORESCENCE: ongoing moisture infiltration
  — requires source identification and correction

  TREATMENT:
  ┌──────────────────────────────────────────────────────────────────┐
  │  FIRST: Identify and fix moisture source (flashing, grade,     │
  │         cracks, missing caulk, failed gutters/downspouts)      │
  │                                                                  │
  │  THEN: Clean surface                                             │
  │  • Light efflorescence: dry brush, then mild acid wash         │
  │  • Stubborn: 10% muriatic acid solution + brush + rinse        │
  │    (wear PPE; test on small area; do not use on limestone or     │
  │     polished stone — acid dissolves calcium carbonate)         │
  │  • Organic (biogenic) deposits: sodium hypochlorite solution   │
  │                                                                  │
  │  NEVER: Apply sealant to suppress efflorescence — only traps   │
  │         salts inside, which then crystallize WITHIN masonry    │
  │         (cryptoflorescence) → spalling from inside out         │
  └──────────────────────────────────────────────────────────────────┘
```

## Masonry Cleaning

```
CLEANING METHOD SELECTION
==========================

  ACCEPTABLE (in order of preference):
  ┌──────────────────────────────────────────────────────────────────┐
  │  LOW-PRESSURE WATER WASHING                                      │
  │  < 400 psi; gentle rinse; pre-soak                             │
  │  Safe for most masonry; does not abrade surface                │
  │  Best for general dirt and biological growth                     │
  │  ──────────────────────────────────────────────────────────────  │
  │  CHEMICAL CLEANING (acidic or alkaline)                        │
  │  Acidic: dilute muriatic or proprietary masonry acid           │
  │    Good for efflorescence, mortar smears, iron stains          │
  │    NEVER on limestone, marble, calcareous sandstone (dissolves)  │
  │    Always: pre-wet surface + rinse thoroughly                  │
  │  Alkaline: trisodium phosphate (TSP) or proprietary            │
  │    Good for organic stains, atmospheric soot, oil stains       │
  │    Apply, dwell 5–10 min, scrub, rinse                        │
  │  ──────────────────────────────────────────────────────────────  │
  │  STEAM CLEANING                                                  │
  │  Hot water vapor; no abrasion; penetrates biological growth    │
  │  Good for historic limestone, terra cotta, decorative stone    │
  │  Slow; expensive; but gentle                                     │
  │  ──────────────────────────────────────────────────────────────  │
  │  MICRO-ABRASIVE (controlled abrasive blast at very low pressure)│
  │  TORC / Doff system / Jos system                               │
  │  Acceptable for some historic stone when other methods fail    │
  │  Must be performed by specialist; test panel required          │
  └──────────────────────────────────────────────────────────────────┘

  PROHIBITED ON HISTORIC MASONRY:
  ┌──────────────────────────────────────────────────────────────────┐
  │  HIGH-PRESSURE WASHING (>400 psi)                              │
  │  SANDBLASTING / GLASS-BEAD BLASTING                            │
  │  WIRE BRUSHING (on soft historic brick or stone)               │
  │  GRINDING / WHEEL TOOLS                                          │
  │  All prohibited by Secretary of Interior Standards (36 CFR 68)   │
  └──────────────────────────────────────────────────────────────────┘
```

## Historic Preservation Standards

The Secretary of the Interior's Standards for the Treatment of Historic Properties (36 CFR Part 68) defines four treatment levels:

```
SECRETARY OF INTERIOR TREATMENT HIERARCHY
==========================================

  PRESERVATION (most conservative)
  ┌──────────────────────────────────────────────────────────────────┐
  │  Retain and repair existing materials to the maximum extent      │
  │  No removal of historic fabric                                   │
  │  Stabilize, consolidate, protect existing material               │
  └──────────────────────────────────────────────────────────────────┘

  REHABILITATION (most common)
  ┌──────────────────────────────────────────────────────────────────┐
  │  Allow alterations for new use while protecting significant      │
  │  historic character                                              │
  │  Repair rather than replace; if replacement necessary, match     │
  │  original in design, color, texture, and where possible material │
  │  No sandblasting, incompatible coatings, damaging cleaning       │
  └──────────────────────────────────────────────────────────────────┘

  RESTORATION (less common)
  ┌──────────────────────────────────────────────────────────────────┐
  │  Recover the form and details of a specific period             │
  │  Remove features of other periods; reconstruct missing features  │
  └──────────────────────────────────────────────────────────────────┘

  RECONSTRUCTION (most intrusive)
  ┌──────────────────────────────────────────────────────────────────┐
  │  Rebuild non-surviving portions using physical and               │
  │  documentary evidence                                            │
  └──────────────────────────────────────────────────────────────────┘
```

## Decision Cheat Sheet

| Situation | Repair Approach |
|-----------|----------------|
| Failed mortar joints, pre-1900 brick | Petrographic analysis → NHL or lime putty repoint |
| Failed mortar joints, post-1950 CMU | Match original — typically Type S |
| Stair-step diagonal cracks | Monitor for movement first; address foundation then repoint |
| Horizontal crack above window | Investigate corroded lintel or flashing; replace metal first |
| Efflorescence, new building | Watch 1–2 years; likely primary; fix moisture entry points |
| Efflorescence, recurring | Investigate source; do not seal surface |
| Atmospheric dirt cleaning | Low-pressure wash first; acidic/alkaline chemical if needed |
| Post-earthquake crack assessment | Document pattern; consult structural engineer before repair |

## Common Confusion Points

**Repointing does not repair structural cracks.** Structural cracks (from settlement, overload, seismic) require addressing the cause. Repointing a crack that is still moving will fail immediately. Monitor with tell-tales (adhesive glass strips across crack) for 12–24 months before attempting repair.

**Sealants make masonry worse.** Water vapor moves through masonry constantly (vapor drive). Sealing the face prevents vapor from escaping; it condenses internally, saturates the wall, and accelerates freeze-thaw damage. Use only breathable consolidants (e.g., silane-siloxane water repellents with high vapor transmission) on historic masonry, and only when clearly needed.

**Chemical cleaners require neutralization.** Acidic cleaners leave acid in the masonry if not thoroughly rinsed. Residual acid continues to attack carbonate binders (lime, cement). Every chemical cleaning requires a long, thorough rinse sequence — typically 3× the application volume of water.

**Cleaning does not fix the problem.** Biological growth (moss, lichen, algae) returns unless the moisture environment changes. Cleaning removes the symptom; improved drainage, grade, gutters, and waterproofing address the cause.
