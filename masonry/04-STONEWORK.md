# Stonework: Ashlar, Rubble, Dry-Stone, and Quarrying

## The Stone Masonry Landscape

Stone masonry spans an enormous range: from highly engineered, precision-cut ashlar in classical temples to unmodified field stone piled dry in a pasture wall. The structural principles remain the same (compression, interlocking, gravity), but the skill and tooling required vary by an order of magnitude.

```
STONE MASONRY CLASSIFICATION
=============================

  BY UNIT DRESSING:
  ┌─────────────────────────────────────────────────────────────────┐
  │  ASHLAR (dressed)                                               │
  │  ├── Coursed ashlar: cut to equal height courses                │
  │  ├── Random coursed: cut, but courses vary in height            │
  │  └── Random rubble (semi-dressed): irregular faces, dressed     │
  │                                     edges only                  │
  │                                                                   │
  │  RUBBLE (undressed or minimally dressed)                        │
  │  ├── Coursed rubble: uncut stones laid in approximate courses   │
  │  └── Random rubble: no courses, stones as found                 │
  │                                                                   │
  │  DRY-STONE (no mortar)                                          │
  │  ├── Dry-stone wall (agricultural boundary)                     │
  │  └── Cyclopean masonry (prehistoric large blocks)               │
  └─────────────────────────────────────────────────────────────────┘

  PRECISION SPECTRUM:
  Cyclopean  →  Random Rubble  →  Coursed Rubble  →  Random Coursed  →  Coursed Ashlar
  (fit by    →  (unmodified)   →  (approximate    →  (dressed but    →  (cut to exact

```

## Ashlar Masonry

Ashlar means stone cut and dressed to specified dimensions. The face is smooth (sawn or tooled), the bed and joints are flat (true plane contact), allowing tight joints and predictable structural behavior.

```
ASHLAR CLASSIFICATION
======================

  COURSED ASHLAR
  ┌──────────────────────────────────────────────────────────────────┐
  │  ┌──────────────────┬──────────────────┬──────────────────────┐  │
  │  │                  │                  │                      │ │  Course 3 (same height)
  │  ├────────┬─────────┴────────┬─────────┴──────┬──────────────┤ │
  │  │        │                  │                 │              │ │  Course 2
  │  ├────────┴─────┬────────────┴──────┬──────────┴─────────────┤ │
  │  │              │                   │                         │ │  Course 1
  │  └──────────────┴───────────────────┴─────────────────────────┘ │
  │  All stones in a course are the same height; courses may vary   │
  └──────────────────────────────────────────────────────────────────┘

  RANDOM COURSED ASHLAR
  ┌──────────────────────────────────────────────────────────────────┐
  │  ┌────────────┬──────┬────────────────┬──────┐                 │
  │  │  8" course │      │   8" course     │      │ ← vary heights  │
  │  ├────────────┤  6"  ├────────────────┤  6"  │                 │
  │  │  6" course │      │   6" course    │      │                 │
  │  └────────────┴──────┴────────────────┴──────┘                 │
  │  Stones cut and dressed; courses vary in height but each        │
  │  course is consistent in its own height range                   │
  └──────────────────────────────────────────────────────────────────┘

  RANDOM ASHLAR (non-coursed)
  ┌──────────────────────────────────────────────────────────────────┐
  │  ┌────────────┬──────────────┬──────┐                          │
  │  │            │  ┌───────┬───┴──┐   │  ← no horizontal courses  │
  │  │            │  │       │      │   │                           │
  │  ├──────┬─────┴──┤       ├──────┤   │  ← dressed faces, flat   │
  │  │      │        │       │      │   │    beds, random layout   │
  │  └──────┴────────┴───────┴──────┴───┘                          │
  └──────────────────────────────────────────────────────────────────┘
```

### Ashlar Joint Widths

| Type | Joint Width | Mortar |
|------|------------|--------|
| Fine ashlar | 1/16" – 1/8" | Thin-set or near-dry lime paste |
| Normal ashlar | 3/8" – 1/2" | Standard Type N or S |
| Rubble pointing | 1/2" – 1" | Type N or NHL |
| Dry-stone | None | None |

Greek classical ashlar (Parthenon, c. 432 BCE) used zero-mortar construction with bronze cramps. Joint tolerances in the 1/100" range have been measured archaeologically.

## Rubble Masonry

Rubble uses stone that has not been precision-cut. The mason selects and fits stones from available material, shaping minimally with a hammer.

## Dry-Stone Wall Construction

Dry-stone walls use no mortar. Stability comes entirely from gravity, friction, and careful stone selection and placement. A properly built dry-stone wall can last centuries; a poorly built one fails in one winter.

```
DRY-STONE WALL ANATOMY
=======================

  CROSS-SECTION:
  ┌─────────────────────────────────────────────────────────────────┐
  │                    COPING STONES (cap)                          │
  │  ///////////////////////////////////////////////////            │
  │  ┌──────────────────────────────────────────────┐               │
  │  │          THROUGH-STONE (tie stone)           │              │
  │  │  ┌──────┬───────────────────────┬───────┐   │              │
  │  │  │FACE  │   HEARTING (fill)     │ FACE  │   │              │
  │  │  │STONES│  (smaller stones,     │STONES │   │              │
  │  │  │      │   packed tight)       │       │   │              │
  │  │  └──────┴───────────────────────┴───────┘   │              │
  │  │          THROUGH-STONE                       │              │
  │  │  ┌──────┬───────────────────────┬───────┐   │              │
  │  │  │FACE  │   HEARTING            │ FACE  │   │              │
  │  │  └──────┴───────────────────────┴───────┘   │              │
  │  └──────────────────────────────────────────────┘              │
  │                    FOUNDATION COURSE                            │
  └─────────────────────────────────────────────────────────────────┘

  KEY ELEMENTS:
  ┌──────────────────────────────────────────────────────────────────┐
  │  BATTER: Wall leans slightly inward on both faces                │
  │    Typical: 1" in 2 feet (1:24) per face                       │
  │    Purpose: Lowers center of gravity, increases stability        │
  │                                                                  │
  │  THROUGH-STONE (bond stone, tie stone):                          │
  │    Spans full wall width at intervals                            │
  │    Frequency: Every 3–4 sq ft of face area                     │
  │    Critical: Without through-stones, two separate stacks form    │
  │                                                                  │
  │  HEARTING:                                                       │
  │    Small stones filling the core between face stones             │
  │    Must be packed tight — loose hearting compresses/settles      │
  │                                                                  │
  │  COPING:                                                         │
  │    Top course: flat or rounded stones protecting joint from rain │
  │    Often set on edge (pitching stones) or large flat slabs       │
  └──────────────────────────────────────────────────────────────────┘
```

### Dry-Stone Principles

1. **Two-over-one, one-over-two**: Each stone must be supported by two stones below it, covering the joint between them. Same principle as running bond in mortared masonry.

2. **Long stone runs deep**: The deeper a stone extends into the wall, the more it contributes to stability. Shallow stones slide out; deep stones lock in.

3. **Rising joints kill walls**: A continuous vertical joint running up the wall face is a fracture plane waiting to happen. Stagger all joints.

4. **Fill voids**: Air pockets in the hearting allow stone migration under load. Pack all voids with small chips.

5. **Best stone at base**: The largest, flattest stones go in the foundation course. They distribute load and resist heaving.

## Cyclopean Masonry

Cyclopean masonry uses massive stone blocks (typically 5–50+ tons) fitted together without mortar and without scaffolding as commonly understood.

```
CYCLOPEAN EXAMPLES
==================

  MYCENAE (Treasury of Atreus, 1250 BCE):
   ├── Lintel stone: 120 tons
   ├── Corbelled vault: each course projects slightly inward
   └── No mortar: precision fitting + massive weight

  INCA SACSAYHUAMAN (Cusco, Peru, 1440s CE):
   ├── Some blocks: 100–300+ tons
   ├── Fitting tolerance: 0.0004" (1/100 mm) in some joints
   ├── No mortar — polyhedral (multi-face) fitting resists seismic
   └── SEISMIC INSIGHT: Polyhedral joints allow micro-movement then
       re-seat — vastly superior to mortared work in earthquakes

  STONEHENGE (3000–1500 BCE):
   ├── Sarsen stones: up to 25 tons, transported 25 miles
   ├── Mortise-and-tenon joints in horizontal lintels
   └── Astronomical alignment — not purely structural

  WHY CYCLOPEAN SURVIVES EARTHQUAKES:
  ┌──────────────────────────────────────────────────────────────────┐
  │  Mortared masonry: rigid assembly → seismic force = shear →      │
  │    joints crack at mortar → catastrophic failure                 │
  │                                                                  │
  │  Cyclopean masonry: blocks rock and reseat on polyhedral fits →  │
  │    absorbs seismic energy in friction → reseat by gravity →      │
  │    survives 8.0+ earthquakes that destroy modern buildings       │
  └──────────────────────────────────────────────────────────────────┘
```

## Quarrying and Stone Dressing

```
QUARRYING PROCESS
==================

  CHANNEL DRILLING (diamond wire or drill-and-blast):
  Rock face → Drill holes → Insert wire saw or wedge → Extract block

  PLUG-AND-FEATHER (traditional):
  ┌──────────────────────────────────────────────────────────────────┐
  │  1. Drill line of holes along desired break plane                │
  │  2. Insert steel "feathers" (half-round channels) in each hole   │
  │  3. Drive steel "plug" between feathers with hammer              │
  │  4. Progressive tightening opens controlled fracture             │
  │  5. Stone splits cleanly along natural grain or bedding plane    │
  └──────────────────────────────────────────────────────────────────┘

STONE DRESSING FINISHES
========================

  QUARRY-SAWN: Smooth flat face from diamond saw
   Use: Interior flooring, wall cladding

  HONED: Ground to uniform matte finish
   Use: Interior floors, moderate-polished walls

  POLISHED: High-gloss buffed surface
   Use: Interior only — polished limestone slippery when wet

  RUBBED: Wet-rubbed with carborundum block
   Use: Smooth but non-reflective exterior finish

  BUSH-HAMMERED: Mechanical impact texturing
   Use: Exterior — non-slip, good weathering character

  FLAMED (thermal): High-heat torch followed by water quench
   Use: Granite and hard stone — crystalline surface rupture
   Appearance: Rough, textured — high coefficient of friction

  SPLIT-FACE: Rock-faced, no dressing — quarry split only
   Use: Rustic aesthetic, CMU split-face blocks

  DRAFTED MARGIN: Smooth borders + rock-face center
   Use: Traditional building stone character
```

## Dimensional Stone Specification

When specifying stone for a project, the key parameters are:

| Parameter | What to Specify | Why |
|-----------|-----------------|-----|
| Stone type | Specific quarry + geological formation | Same "granite" from different quarries varies enormously |
| ASTM classification | C568 (limestone), C615 (granite), C616 (sandstone), C503 (marble) | Minimum strength/absorption |
| Finish | As above table | Determines appearance and slip resistance |
| Thickness | Nominal + tolerance (±1/16" for cut stone) | Structural vs. veneer implications |
| Anchoring | Mechanical (pin + clip) vs. adhesive | Exterior requires mechanical anchors |
| Bedding plane | Natural bed vs. face bed | Critical for sedimentary stone |
| Calibration | Thickness tolerance (±1mm for interior tile) | Flat floor or wall system |

## Decision Cheat Sheet

| Situation | Stone Masonry Type |
|-----------|-------------------|
| Classical architectural facade | Coursed ashlar — limestone or granite |
| Rustic exterior wall | Random coursed ashlar or coursed rubble |
| Agricultural boundary wall | Dry-stone — local field stone |
| Garden retaining wall, no mortar | Dry-stone with batter (1" per 2' per face) |
| Seismic zone, historic stone structure | Cyclopean joints analyzed; do not add mortar to historic dry-stone |
| Interior flooring | Polished granite or marble (verify slip resistance) |
| Exterior plaza paving | Flamed or bush-hammered granite (non-slip) |
| Historic restoration | Match quarry, finish, and bedding orientation of original |

## Common Confusion Points

**Ashlar vs. rubble is about dressing, not stone type.** Limestone can be either ashlar or rubble. Granite can be either ashlar or rubble. The distinction is whether the stone has been precision-cut and dressed to dimension.

**Dry-stone walls fail from the inside.** The hearting (core fill) settles over decades, especially if not tightly packed. The face stones then have no interior support and eventually tilt outward. Rebuild from the top down — replace hearting, reset face stones, repack coping.

**Bedding plane is critical for sedimentary stone.** Limestone and sandstone have distinct horizontal lamination from deposition. If a stone is face-bedded (lamination running vertically), freeze-thaw water penetrates between laminae and pops the face off. Always specify "bed on natural bed" for sedimentary stone in exterior work.

**Marble fails outdoors.** Marble absorbs moisture (anisotropic thermal and moisture expansion) and warps. Thin marble panels (under 1.5" thick) in exterior applications have failed spectacularly (Amoco Building, Chicago — replaced entirely in 1990 after only 16 years). Use for interior only or accept heavy panel thicknesses.

**Through-stones are not decorative.** In a dry-stone wall, through-stones (spanning full width) are the structural elements that tie the two faces together. A wall without them is two separate faces leaning on each other, not one wall. The appearance of through-stone ends on the face is a functional indicator of quality construction.
