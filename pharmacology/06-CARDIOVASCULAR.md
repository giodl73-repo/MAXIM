# Cardiovascular Pharmacology

## The Big Picture

Cardiovascular (CV) pharmacology targets the heart, vasculature, kidneys, and the hormonal systems that regulate them. Understanding CV drugs requires a working model of cardiac physiology: preload, afterload, contractility, and the renin-angiotensin-aldosterone system (RAAS).

```
+──────────────────────────────────────────────────────────────────+
|              CARDIOVASCULAR PHARMACOLOGY LANDSCAPE               |
|                                                                  |
|  TARGET                DRUGS                   GOAL              |
|  ──────────────────    ──────────────────────  ───────────────   |
|  Sympathetic NS        β-blockers, α-blockers  ↓HR, ↓BP        |
|  Renin-Angiotensin     ACE-I, ARB, aliskiren   ↓BP, ↓cardiac   |
|  Aldosterone           Spironolactone,         ↓fluid, heart     |
|                        eplerenone              failure, HTN      |
|  Cardiac channels      CCB, digoxin            ↓HR, ↓BP        |
|  Diuretics             Thiazides, loops, K+    ↓volume, ↓BP      |
|                        sparing                                   |
|  Lipid pathways        Statins, PCSK9i, ezetimibe ↓LDL         |
|  Coagulation           Anticoagulants,         ↓thrombosis       |
|                        antiplatelets                             |
|  Channels (arrhythmia) Antiarrhythmics         Restore rhythm    |
+──────────────────────────────────────────────────────────────────+
```

## Blood Pressure Drugs: The RAAS and Adrenergic Targets

```
RENIN-ANGIOTENSIN-ALDOSTERONE SYSTEM (RAAS)
────────────────────────────────────────────
  Kidney (JGA) → [renin]
  ↓
  Angiotensinogen → Angiotensin I (inactive)
  ↓ [ACE in lungs]
  Angiotensin II → vasoconstriction, aldosterone release, sodium retention
  ↓
  Aldosterone → kidney → Na+ retention, K+ excretion → ↑blood volume

DRUG INTERVENTION POINTS
  ACE inhibitors (ACE-I):   Block ACE → less Ang II + less bradykinin breakdown
    Examples: lisinopril, enalapril, ramipril (-pril suffix)
    First-line for: HTN, heart failure, post-MI, diabetic nephropathy
    Side effect: DRY COUGH (bradykinin accumulation); angioedema (rare)

  ARBs (Angiotensin Receptor Blockers): Block Ang II receptor AT1
    Examples: losartan, valsartan, candesartan (-sartan suffix)
    Use when: ACE-I intolerance (cough); comparable efficacy
    No cough (bradykinin not affected)

  ARNI (ARB + Neprilysin Inhibitor): sacubitril/valsartan (Entresto)
    Sacubitril blocks neprilysin → ↑natriuretic peptides → ↓preload/afterload
    Heart failure with reduced ejection fraction (HFrEF): mortality benefit

  Aldosterone antagonists:
    Spironolactone: K+-sparing diuretic; heart failure, resistant HTN, primary hyperaldosteronism
    Eplerenone: more selective; fewer sex hormone effects; post-MI heart failure
    Finerenone (SGLT2 drug era): used in CKD + diabetes

  Direct renin inhibitor:
    Aliskiren: Blocks renin directly; limited clinical use (combinations often inferior)
```

## Beta-Blockers

```
BETA-ADRENERGIC RECEPTORS
──────────────────────────
  β1: Heart (↑HR, ↑contractility), kidney (↑renin release)
  β2: Lungs (bronchodilation), vasculature (vasodilation), metabolism
  β3: Adipose tissue, bladder detrusor

BETA-BLOCKER CLASSIFICATION
  Nonselective (β1 + β2):
    Propranolol: prototype; crosses BBB → CNS effects (anxiety, migraine)
    Carvedilol:  also α1 blocker → vasodilation; heart failure
    Labetalol:   also α1 blocker; IV for hypertensive emergency, pre-eclampsia

  Cardioselective (β1 >> β2):
    Metoprolol (succinate for once-daily): HTN, heart failure, post-MI
    Atenolol:  Renal excretion; less CNS effect; once daily
    Bisoprolol: Heart failure (with carvedilol and metoprolol, the "big 3")

  Use cardioselective in COPD/asthma patients (β2 blockade = bronchospasm)
  Even cardioselective blockers carry small risk in severe asthma.

CLINICAL USES
  • Hypertension: ↓HR + ↓renin release + ↓cardiac output
  • Heart failure (EF ≤ 40%): Metoprolol succinate, carvedilol, bisoprolol
    PARADOX: Negative inotropes IMPROVE heart failure long-term
    Mechanism: Block sympathetic over-activation → reverse remodeling
  • Post-MI: Reduce mortality; prevent arrhythmias
  • Angina: ↓O₂ demand (↓HR, ↓contractility)
  • Arrhythmias: ↓AV node conduction; rate control in AF
  • Tremor, anxiety, performance anxiety: CNS-penetrating (propranolol)
  • Migraine prophylaxis: propranolol, metoprolol

SIDE EFFECTS
  Bradycardia, heart block (do NOT give if HR < 50, PR > 0.24)
  Bronchospasm in asthmatics (β2 blockade)
  Masking hypoglycemia symptoms in diabetics (β2 blockade of tachycardia)
  Fatigue, cold extremities
  ABRUPT WITHDRAWAL: Rebound tachycardia, angina (upregulation)
```

## Calcium Channel Blockers (CCBs)

```
CCB MECHANISM AND CLASSES
───────────────────────────
Voltage-gated L-type calcium channels:
  Heart: Ca²+ entry → contraction + pacemaker activity
  Vasculature: Ca²+ entry → smooth muscle contraction → vasoconstriction

  TWO MAIN CLASSES
  ─────────────────
  DIHYDROPYRIDINES (DHP)        NON-DIHYDROPYRIDINES
  ──────────────────────────    ────────────────────────────
  Amlodipine, nifedipine,       Diltiazem (benzothiazepine)
  felodipine, lercanidipine     Verapamil (phenylalkylamine)
  Act primarily on vasculature  Act on heart AND vasculature
  → vasodilation → ↓BP          → ↓HR + ↓AV node conduction
  Little cardiac rate effect     + ↓BP
  Safe to combine with          CONTRAINDICATED with β-blockers
  beta-blockers                  (both block AV node → heart block)

DIHYDROPYRIDINE USES
  HTN: Amlodipine — long-acting, well-tolerated, evidence-based
  Angina (vasospastic / Prinzmetal): CCBs are drug of choice
  Raynaud's phenomenon: vasodilation

NON-DHP USES
  Rate control in AF/flutter: Verapamil, diltiazem (oral or IV)
  SVT: Verapamil IV (AVNRT, AVRT) — terminates via AV node blockade
  Angina: Both; diltiazem more popular due to verapamil's constipation/drug interactions

SIDE EFFECTS
  Peripheral edema (DHPs — vasodilation effect)
  Constipation (verapamil — GI smooth muscle)
  Gingival hyperplasia (rare, nifedipine)
  Reflex tachycardia (short-acting DHPs — not amlodipine)
```

## Diuretics

```
DIURETIC CLASSES AND MECHANISMS
──────────────────────────────────

  THIAZIDE DIURETICS
  Site: Distal convoluted tubule (DCT)
  Mechanism: Block Na+-Cl- cotransporter (NCC)
  Effect: ↓Na/water → ↓blood volume → ↓BP (also vasodilatory effect)
  Examples: Hydrochlorothiazide (HCTZ), chlorthalidone, indapamide
  Use: First-line HTN, edema
  Key points:
    • Cause HYPOKALEMIA, hyponatremia, hyperuricemia (gout risk)
    • Chlorthalidone has longer t½ and stronger evidence than HCTZ
    • INEFFECTIVE in renal failure (GFR < 30) — loop diuretics instead

  LOOP DIURETICS
  Site: Thick ascending limb of loop of Henle
  Mechanism: Block Na+-K+-2Cl- cotransporter (NKCC2)
  Effect: Most powerful diuresis; blocks 25% of Na reabsorption
  Examples: Furosemide (Lasix), bumetanide, torsemide
  Use: Acute heart failure, pulmonary edema, renal failure (CKD), edema
  Key points:
    • Hypokalemia, hypomagnesemia (electrolyte wasting)
    • Ototoxicity at high IV doses (especially with aminoglycosides)
    • Furosemide is CYP-independent; mainly tubular secretion

  POTASSIUM-SPARING DIURETICS
  Three types:
    ENaC blockers: Amiloride, triamterene (cortical collecting duct)
    Aldosterone antagonists: Spironolactone, eplerenone (MR receptor)
    All → ↑K+ retention; used for hypokalemia prevention

  CARBONIC ANHYDRASE INHIBITORS
  Acetazolamide: proximal tubule; alkaline diuresis; altitude sickness, glaucoma

  OSMOTIC DIURETICS
  Mannitol: freely filtered; draws water; raised intracranial pressure
```

## Engineering Bridge: Cardiovascular Pharmacology as Control Systems

The cardiovascular system is a closed-loop feedback control system with sensors, actuators, and multiple regulatory cascades. Each drug class targets a specific component of the loop.

```
  CARDIOVASCULAR PHARMACOLOGY   CONTROL SYSTEMS / FEEDBACK PARALLEL
  ──────────────────────────────────────────────────────────────────────
  RAAS cascade                  Nested feedback control loop:
    Renin (JGA sensor)          → Sensor: detects low renal perfusion/Na+
    Angiotensin I → ACE → Ang II → Signal amplification cascade
    Aldosterone (effector)      → Actuator: Na+/H₂O retention
    Blood pressure              → Controlled variable (setpoint = normotension)

  ACE inhibitor / ARB           Loop-breaking intervention:
                                Cut the signal path at different points:
                                ACE-I: blocks Ang I → Ang II (signal transduction)
                                ARB: blocks Ang II at AT1 receptor (actuator input)
                                Either breaks the runaway positive feedback
                                in heart failure (↑AngII → vasoconstriction →
                                ↑afterload → ↓CO → ↑renin → more AngII).

  Vaughan Williams classes       State machine taxonomy — each class targets
                                a different component of the cardiac AP
                                state machine:
    Class I: block the AP upstroke driver (Na+ channel = fast depolarizer)
    Class II: block sympathetic override (β-adrenergic modulator)
    Class III: prolong the refractory period (K+ channel = repolarizer)
    Class IV: block the pacemaker/AV node (Ca2+ channel = slow depolarizer)
    Adenosine: hyperpolarize the AV node (K+ current activator)

  β-blockers in heart failure   Counter-intuitive feedback inversion:
                                Acute HF → sympathetic hyperactivation →
                                tachycardia → ↓diastolic filling time →
                                β-receptor downregulation → further ↓function.
                                Chronic β-blocker: interrupt the positive
                                feedback → receptors upregulate → improved
                                function at lower baseline activation.
                                Starting during acute decompensation = removing
                                the compensatory drive too early.

  Statins → PCSK9 inhibition    Two-level intervention on the same pathway:
                                HMG-CoA → statin blocks endogenous synthesis.
                                PCSK9 degrades LDL receptors → PCSK9-I blocks
                                degradation → more recycling of receptors.
                                Additive mechanism targeting different
                                rate-limiting steps of the same control
                                variable (plasma LDL).
  ──────────────────────────────────────────────────────────────────────
```

## Antiarrhythmics: The Vaughan Williams Classification

```
CARDIAC CONDUCTION TARGET
───────────────────────────
  SA node → AV node → Bundle of His → Bundle branches → Purkinje → Ventricles

  ION CHANNELS CONTROLLING EACH PHASE OF CARDIAC ACTION POTENTIAL
  Phase 0: Rapid Na+ influx (atrial/ventricular muscle) or Ca²+ influx (SA/AV)
  Phase 1: K+ efflux (Ito)
  Phase 2: Ca²+ plateau (L-type channels)
  Phase 3: K+ efflux (IKr, IKs) → repolarization
  Phase 4: Na+ leak → spontaneous depolarization (pacemaker cells)

VAUGHAN WILLIAMS CLASSES
  Class I: Na+ channel blockers (slow conduction)
    Ia: Quinidine, procainamide — prolong repolarization; QT↑; torsades risk
    Ib: Lidocaine, mexiletine — shorten repolarization; ventricular arrhythmias
    Ic: Flecainide, propafenone — no repolarization effect; most potent Na+ block
        DANGEROUS post-MI (CAST trial: ↑mortality in post-MI patients)

  Class II: β-blockers — rate control, post-MI
    Metoprolol, atenolol, esmolol (IV, ultra-short t½ for emergencies)

  Class III: K+ channel blockers (prolong repolarization, lengthen QT)
    Amiodarone: also Class I/II/IV effects; most effective antiarrhythmic;
                multiple toxicities (thyroid, pulmonary, hepatic, corneal)
    Sotalol: β-blocker + K+ channel block; atrial fibrillation; QT monitoring
    Dofetilide, ibutilide: pure Class III

  Class IV: Non-DHP CCBs — AV nodal blockade
    Verapamil, diltiazem (IV for acute rate control)

  Adenosine: Not classified; terminates SVT via AV node suppression
             Ultra-short t½ (seconds); safe diagnostic + therapeutic tool
```

## Statins and Lipid-Lowering Therapy

```
STATINS (HMG-CoA Reductase Inhibitors)
────────────────────────────────────────
  Mechanism: Block HMG-CoA reductase → ↓cholesterol synthesis in liver
             → Liver upregulates LDL receptors → ↑LDL clearance from blood

  LDL reduction: High-intensity (atorvastatin 40-80 mg, rosuvastatin 20-40 mg): ↓LDL ~50%
                 Moderate-intensity (atorvastatin 10-20 mg, simvastatin 40 mg): ↓LDL ~30-50%

  Benefits beyond LDL: "pleiotropic effects" — anti-inflammatory, plaque stabilization

  SIDE EFFECTS
    Myopathy: muscle pain + ↑CK. Risk: simvastatin > atorvastatin > rosuvastatin.
    Rhabdomyolysis: rare but serious (CK > 10× ULN + myoglobinuria)
    Risk factors: CYP3A4 inhibitors (simvastatin/lovastatin), hypothyroidism, renal failure
    Simvastatin 80 mg: HIGH rhabdomyolysis risk; FDA warning; avoid in most patients
    Hepatotoxicity: rare; routine LFT monitoring no longer recommended

PCSK9 INHIBITORS (evolocumab, alirocumab)
  PCSK9 normally degrades LDL receptors after LDL uptake.
  Blocking PCSK9 → LDL receptors recycle → more LDL cleared.
  ↓LDL by 60% on top of statins. Injectable (q2-4 weeks).
  Inclisiran: siRNA that silences PCSK9 gene; twice-yearly injection.
  Used in: familial hypercholesterolemia, statin intolerance, very high CV risk.

EZETIMIBE
  Blocks NPC1L1 transporter in intestine → ↓cholesterol absorption
  ↓LDL ~18%. Combined with statin when statin alone insufficient.
  IMPROVE-IT trial: adding ezetimibe to statin → further CV benefit.

BILE ACID SEQUESTRANTS
  Cholestyramine, colesevelam: bind bile acids in gut → ↓recirculation
  ↓LDL 15-30%; major side effects: GI (constipation, bloating); drug interactions.
```

## SGLT2 Inhibitors

```
SGLT2 INHIBITORS (Gliflozins)
───────────────────────────────
  Empagliflozin (Jardiance), dapagliflozin (Farxiga), canagliflozin (Invokana)

  PRIMARY MECHANISM:
    Block SGLT2 (sodium-glucose cotransporter 2) in proximal tubule.
    → Prevents glucose reabsorption → glucosuria even at normal glucose.
    → Osmotic diuresis + natriuresis → ↓preload, ↓blood pressure.

  CARDIOVASCULAR BENEFIT (beyond glucose control):
    EMPA-REG OUTCOME (empagliflozin, 2015): ↓CV death 38%, ↓hospitalization for HF 35%
    DAPA-HF (dapagliflozin, 2019): benefit in HFrEF regardless of diabetes status
    EMPEROR-Preserved (empagliflozin, 2021): first drug to show benefit in HFpEF
    DAPA-CKD: renal protection in CKD with or without T2D

  WHY THE HEART BENEFIT?
    Hemodynamic: ↓preload/afterload via diuresis + natriuresis
    Metabolic: shift from glucose to ketone/fat oxidation (more efficient cardiac fuel)
    Direct cardiac: anti-fibrotic, anti-inflammatory, ↓inflammation
    Renal: reduce glomerular hyperfiltration → preserve kidney → preserve CV

  NOW STANDARD OF CARE FOR:
    HFrEF (EF < 40%): add to ACE-I/ARB + β-blocker + MRA regardless of diabetes
    HFpEF (EF ≥ 50%): first proven pharmacotherapy
    CKD (eGFR 20-75 + albuminuria): renal + CV protection
    T2D with high CV risk: empagliflozin or dapagliflozin

  SAFETY:
    Genital mycotic infections (glucosuria → fungal niche)
    DKA risk: rare in T2D, higher in T1D (not approved for T1D in US)
    Fournier's gangrene: very rare, severe necrotizing fasciitis
    Hold before surgery/prolonged fasting (DKA risk)
```

## Anticoagulants and Antiplatelets

```
DIRECT ORAL ANTICOAGULANTS (DOACs)
────────────────────────────────────
  Mechanism: direct inhibition of coagulation factors (no warfarin cofactor)

  DIRECT THROMBIN INHIBITORS:
    Dabigatran (Pradaxa): IIa (thrombin) inhibitor. Renal clearance 80%.
    Reversal: idarucizumab (Praxbind) — specific antibody fragment.

  DIRECT FACTOR Xa INHIBITORS:
    Rivaroxaban (Xarelto): once daily, hepatic CYP3A4/P-gp
    Apixaban (Eliquis): twice daily, lower bleeding risk in trials
    Edoxaban (Savaysa): once daily
    Reversal: andexanet alfa (Andexxa) for Xa inhibitors; 4F-PCC as alternative.

  vs. WARFARIN:
    DOACs: fixed dosing, no INR monitoring, fewer drug interactions, faster onset
    Warfarin: CYP2C9/VKORC1 genetic variation → highly variable dosing; INR 2-3;
              reversible with vitamin K; preferred in mechanical heart valves,
              antiphospholipid syndrome, severe renal failure (CrCl < 15)

ANTIPLATELETS
──────────────
  ASPIRIN:
    Irreversibly acetylates COX-1 → ↓TXA2 → ↓platelet aggregation.
    Low dose (81 mg): minimal GI effect; antiplatelet without anti-inflammatory.
    Effect lasts platelet lifetime (7-10 days) — platelets have no nucleus.

  P2Y12 RECEPTOR ANTAGONISTS (ADP receptor blockade):
    Clopidogrel (Plavix): prodrug → CYP2C19 → active metabolite
      CYP2C19 loss-of-function alleles → poor activation → ↑MACE risk
      (*2 allele: 30% prevalence in East Asians, 15% in Europeans)
    Ticagrelor (Brilinta): reversible, non-prodrug; consistent effect
      Better outcomes in ACS vs clopidogrel (PLATO trial)
    Prasugrel (Effient): prodrug, faster/stronger; not in prior stroke/TIA

  DUAL ANTIPLATELET THERAPY (DAPT):
    Aspirin + P2Y12 inhibitor:
    Mandatory post-PCI (stent): ≥1 year (bare metal), ≥12 months (drug-eluting)
    Acute coronary syndrome (ACS): 12 months regardless of stent

  GP IIb/IIIa INHIBITORS (IV):
    Abciximab, eptifibatide, tirofiban: block final common path of aggregation.
    Used only in high-risk PCI; IV only; high bleeding risk.
```

## Decision Cheat Sheet

| Clinical Scenario | Preferred Drug Class | Key Drug |
|------------------|---------------------|----------|
| HTN + heart failure (reduced EF) | ACE-I or ARB + β-blocker | Lisinopril + metoprolol |
| HTN + CKD + diabetes | RAAS inhibitor | ACE-I or ARB |
| HTN + angina | β-blocker or CCB | Metoprolol or amlodipine |
| HTN + pregnancy | Methyldopa, labetalol, nifedipine | NOT ACE-I/ARB (teratogenic) |
| AF rate control | β-blocker or non-DHP CCB | Metoprolol or diltiazem |
| AF rhythm control | Amiodarone, flecainide (no struct disease) | Amiodarone if structural disease |
| Acute SVT termination | Adenosine → if fails: verapamil IV | Adenosine first |
| Post-MI LDL lowering | High-intensity statin | Atorvastatin 40-80 mg |
| Acute heart failure edema | Loop diuretic | Furosemide IV |
| Resistant HTN | Add spironolactone | Loop diuretic if fluid overloaded |

## Common Confusion Points

**"ACE inhibitors cause cough — can I use ARBs instead?"**
Yes — ARBs do not accumulate bradykinin (ACE inhibitors prevent bradykinin breakdown). Cough is a class effect of ACE inhibitors (~10-15% incidence), not ARBs. Both classes have similar cardiovascular benefit; ACE inhibitors have more evidence base for heart failure and nephroprotection. ARBs are appropriate substitutes when cough is intolerable.

**"Why are beta-blockers used in heart failure if they're negative inotropes?"**
Short-term: β-blockers reduce contractility — harmful in acute decompensated failure. Long-term: chronic heart failure involves excessive catecholamine stimulation → receptor downregulation → further dysfunction. β-blockers interrupt this cycle, allow receptor re-upregulation, and reverse cardiac remodeling. Three specific agents (carvedilol, metoprolol succinate, bisoprolol) have proven mortality benefit. Start only when patient is stable (not during acute decompensation).

**"Amiodarone is the most effective antiarrhythmic — why isn't it first-line for everything?"**
Toxicity. Amiodarone is highly lipophilic, has an enormous Vd (66 L/kg), and an extremely long half-life (40-55 days). It accumulates in every tissue. Side effects: thyroid disease (hypo or hyper; 10-20%), pulmonary toxicity (2-5%, can be fatal), hepatotoxicity, photosensitivity, corneal microdeposits, peripheral neuropathy, and bluish skin discoloration. Essentially every long-term patient gets some toxicity. Used when benefit exceeds these risks (life-threatening arrhythmias, AF when other drugs fail).
