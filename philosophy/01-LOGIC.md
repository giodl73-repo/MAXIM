# 01 — Logic

## Propositional, Predicate, Modal Logic, Proof Theory, Incompleteness

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Propositional logic: syntax (atoms, connectives ∧/∨/¬/→/↔), semantics (truth tables), tautologies
- Arguments: validity vs soundness; modus ponens, modus tollens, disjunctive syllogism
- Natural deduction: introduction and elimination rules for each connective, proof trees
- Propositional resolution: CNF transformation, Robinson resolution, refutation proofs
- First-order (predicate) logic: quantifiers ∀/∃, variables, predicates, functions, terms
- FOL semantics: interpretations, models, satisfaction, validity, logical consequence
- Limitations of propositional logic: what FOL adds (quantification, relations)
- Proof theory in FOL: sequent calculus, Gentzen's LK system, cut elimination
- Undecidability of FOL: Church's theorem — no algorithm decides validity of arbitrary FOL formulas
- Modal logic: □ (necessarily), ◇ (possibly); Kripke semantics (possible worlds)
- Normal modal systems: K, T, S4, S5 — axioms and their philosophical interpretations
- Temporal logic: CTL, LTL — use in formal verification (model checking) — CS bridge
- Gödel's first incompleteness theorem: any consistent formal system containing arithmetic has true-but-unprovable statements
- Gödel's second incompleteness theorem: no such system can prove its own consistency
- Proof of incompleteness: Gödel numbering, diagonalization, self-reference construction
- Tarski's undefinability: truth in a language cannot be defined in that language
- Löb's theorem: □(□P → P) → □P; relation to provability logic
- Computability bridge: Turing's proof via halting problem = independence proof via diagonalization
- Curry-Howard correspondence: propositions as types, proofs as programs (→ 23-PL-THEORY.md)
- Non-classical logics: intuitionistic logic (no excluded middle), relevant logic, paraconsistent logic
- Inference engines: forward/backward chaining, Prolog as a logic programming language
