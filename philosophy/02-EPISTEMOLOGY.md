# 02 — Epistemology

## Knowledge, Justification, Skepticism, Scientific Method

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Classical definition of knowledge: Justified True Belief (JTB) — Plato's Meno, Theaetetus
- Gettier problem (1963): JTB is insufficient; counterexamples showing justified true belief ≠ knowledge
- Post-Gettier responses: reliabilism, externalism, virtue epistemology, knowledge-first (Williamson)
- Epistemic justification: foundationalism vs coherentism vs infinitism
- Foundationalism: basic vs derived beliefs, regress problem, Descartes' Cogito as foundation
- Coherentism: web of belief, no foundational beliefs, Quine-Duhem thesis
- Reliabilism: Goldman's process reliabilism — justification as reliably produced true belief
- Skepticism: Cartesian demon, brain in a vat (Putnam), closure principle
- Responses to skepticism: contextualism, relevant alternatives, semantic externalism
- Induction: Hume's problem — no deductive justification for inductive inference
- Goodman's new riddle of induction: "grue" predicate — which predicates are projectible?
- Bayesian epistemology: degrees of belief as probabilities; Dutch book argument; conditionalization
- Bayesian updating: P(H|E) = P(E|H)·P(H)/P(E) — belief change as probabilistic update
- Bayesianism and ML: connection between Bayesian inference and machine learning priors
- Reliabilism and neural networks: reliability without internalist justification
- Testimony and social epistemology: when to trust others; epistemic injustice; peer disagreement
- A priori vs a posteriori knowledge: Kant's contribution, analytic/synthetic distinction
- Analytic/synthetic: Quine's critique — "Two Dogmas of Empiricism" (1951)
- Scientific confirmation: Hempel's H-D model, Bayesian confirmation theory, Carnap's inductive logic
- Underdetermination of theory by evidence: multiple theories fit the same data
- Naturalised epistemology: Quine's proposal — replace epistemology with cognitive science
