# Planetary Geochemistry and Cosmochemistry

## The Big Picture

Cosmochemistry studies the chemical composition and evolution of the solar system from the first condensates 4.567 billion years ago to present-day planetary bodies. Planetary geochemistry extends these methods to understand how planets differentiated and what their interiors are made of.

```
+------------------------------------------------------------------+
|              COSMOCHEMISTRY / PLANETARY GEOCHEMISTRY             |
+------------------------------------------------------------------+
|                                                                  |
|  COSMOCHEMISTRY          PLANETARY GEOCHEMISTRY                  |
|  --------------          ----------------------                  |
|  Meteorite analysis      Mantle composition                      |
|  Solar photosphere comp. Core composition                        |
|  Presolar grains         Crust formation                         |
|  Short-lived nuclides    Impact mixing                           |
|  Nebular condensation    Volatile delivery                       |
|  Chondrite classification Mantle convection geochemistry         |
|                                                                  |
|  KEY REFERENCE: CI chondrites = solar system composition         |
|  KEY TOOLS: Mass spectrometry (TIMS, ICP-MS, SIMS, NanoSIMS)    |
|             Electron microprobe; Synchrotron XRF                 |
+------------------------------------------------------------------+
```

## Meteorite Classification

```
METEORITE TAXONOMY
==================

  METEORITES
  |
  +-- CHONDRITES (~86% of all falls) — undifferentiated; primitive
  |   |
  |   +-- CARBONACEOUS CHONDRITES (CC):
  |   |     CI, CM, CR, CO, CV, CK, CH, CB types
  |   |     CI: most primitive; highest volatile element content
  |   |     CM: hydrated; organics; matrix-rich
  |   |     CV: refractory-rich; CAI-rich (Allende, Vigarano)
  |   |
  |   +-- ORDINARY CHONDRITES (OC): most common falls
  |   |     H (high Fe), L (low Fe), LL (very low Fe)
  |   |     ~80% of total falls; S-type asteroid fragments
  |   |
  |   +-- ENSTATITE CHONDRITES (EC):
  |   |     Reduced; MgSiO₃ (enstatite) dominant
  |   |     Formed closest to Sun; low volatile content
  |   |     EH (high Fe), EL (low Fe)
  |   |
  |   +-- RUMURUTITES (R): high oxidation state; rare
  |   +-- KAKANGARI (K): ungrouped; primitive
  |
  +-- ACHONDRITES (~8%) — differentiated; from igneous bodies
  |   |
  |   +-- HED (Howardite-Eucrite-Diogenite): from Vesta
  |   |     Eucrite = basalt; Diogenite = dunite/harzburgite; Howardite = regolith
  |   +-- SNC (Shergottite-Nakhlite-Chassignite): from Mars
  |   |     Youngest ages (180-600 Ma + old 4.1 Ga); confirm Mars origin
  |   +-- Lunar meteorites: ejected by impacts from Moon
  |   +-- Aubrites: enstatite achondrites (Mercury?)
  |   +-- Pallasites: olivine + metal (core-mantle boundary)
  |   +-- Ureilites: carbon-rich; primitive achondrite; carbon parent body?
  |
  +-- IRONS (~5%) — core material
  |     Chemical groups (IAB, IIIAB, IVA, etc.) from different parent bodies
  |     Widmanstätten pattern: intergrowth of kamacite + taenite (diagnostic)
  |
  +-- STONY-IRONS (~1%) — mantle-core boundary material
        Pallasites: olivine crystals in metal matrix
        Mesosiderites: silicates + metal (chaotic mixing = impact event)
```

## Chondrites: Structure and Significance

### Chondrules

```
CHONDRULES
==========

  WHAT THEY ARE:
    Spherical silicate objects; 0.1-2 mm diameter
    Found only in chondrites (hence the name)
    Composed of olivine, pyroxene, glass, metal, troilite
    Textures: porphyritic, barred, radial, granular

  ORIGIN: STILL DEBATED (major unsolved problem)
    They were once molten (glass, rapid cooling textures)
    They formed in the solar nebula (not inside a planet)
    Peak temperature: ~1700-2000 K (above olivine liquidus)
    Cooling rate: 10-1000°C/hr (fast, not planetesimal interior)

  PROPOSED MECHANISMS:
    Shock waves: passing shock waves in solar nebula → melting of dust
    Lightning: nebular lightning discharges (less favored now)
    X-wind model: launch and irradiation near proto-Sun
    Most favored: multiple melting events by shock waves in the nebula

  AGE: 4.567-4.563 Ga
    (same age as CAIs or slightly younger → formed in first few Myr)

  SIGNIFICANCE:
    Chondrules represent the "building blocks" of planetesimals
    Streaming instability: aggregates of chondrules + matrix
    They record the thermal history of the early solar nebula
```

### CAIs: The Oldest Solids

```
CALCIUM-ALUMINUM-RICH INCLUSIONS (CAIs)
=========================================

  DEFINITION: Refractory mineral aggregates in CV, CO, CM chondrites
  SIZE: 0.1 mm to 2 cm (some "fluffy" Type A can be 10+ cm in Allende)
  COMPOSITION: Al₂O₃, CaO, TiO₂ (highest-T condensates from solar nebula)
  MINERALS: Corundum (Al₂O₃), Hibonite (CaAl₁₂O₁₉), Melilite, Spinel, Pyroxene

  AGE: 4567.3 ± 0.1 Ma (TIMS U-Pb on Allende CAIs; Connelly et al. 2012)
    → OLDEST OBJECTS IN THE SOLAR SYSTEM
    → Define t=0 for solar system chronology

  ²⁶Al ENRICHMENT:
    CAIs have ²⁶Al/²⁷Al = ~5 × 10⁻⁵ (initial)
    → ²⁶Al was alive when they formed → within first ~3 Myr of solar system
    Decay products: excess ²⁶Mg (measured by SIMS)
    Bodies that accreted while ²⁶Al was hot: melted → differentiated
    Bodies that accreted later: stayed cold → chondritic

  WHY CAIs FORMED FIRST:
    They are the most refractory (highest condensation T) materials
    As solar nebula cooled from ~1500 K:
    First: corundum → hibonite → perovskite → melilite → spinel → CAIs
    Then: Mg-silicates (olivine, pyroxene) → chondrules
    Then: sulfides → Fe-Ni metal alloys → water ice (beyond frost line)
```

## Bulk Silicate Earth and Core Composition

```
BSE COMPOSITION AND CONSTRAINTS
=================================

  BSE = BULK SILICATE EARTH = Mantle + Crust

  APPROACH:
    1. Model BSE as CI chondrite - core fraction, corrected for volatiles
    2. Constrain core composition from seismic + density + metal/silicate experiments
    3. Check: does remaining silicate match mantle xenolith + basalt geochemistry?

  MAJOR ELEMENTS OF BSE (McDonough & Sun 1995):
    SiO₂:  45.0%   Al₂O₃: 4.45%   FeO:   8.05%
    MgO:   37.8%   CaO:   3.55%   Na₂O:  0.36%
    TiO₂:  0.20%   Cr₂O₃: 0.38%

  Mg/Si ratio: key chondritic comparison
    BSE Mg/Si ≈ 1.07 (molar)
    CI chondrite Mg/Si ≈ 1.07 (match! → Si not preferentially in core)
    But: some argue for Si in core (lower Mg/Si) + debate continues

  CORE COMPOSITION (INDIRECT):
    Density: measured by seismology → 10-12 g/cm³ outer core
    Pure Fe-Ni alloy: ~12.5 g/cm³ at core P and T
    Actual: ~6-8% lower density → light element(s) required
    Candidates: S, O, Si, H, C (single or mixture)
    CURRENT MODELS:
      Most consistent: S + O + Si mixture
      Si (~4-8 wt%): explains density deficit
      O (~2-4 wt%): from high-P experiments showing O solubility
      S (<2 wt%): chalcophile; high-T atmosphere in magma ocean
```

## Siderophile Elements and Core Formation

```
HIGHLY SIDEROPHILE ELEMENTS (HSE)
===================================

  HSE: Re, Os, Ir, Ru, Rh, Pt, Pd, Au
  Theory: D(metal/silicate) > 10,000 at low P/T
  → should be almost entirely in the core
  → mantle should be essentially empty of HSE

  OBSERVATION: Mantle has ~1-3% of chondritic HSE abundance (not zero!)
  PROBLEM: Why are HSEs present in the mantle at all?

  SOLUTION: LATE VENEER HYPOTHESIS
    After core formation was complete (~30-40 Myr):
    Additional chondrite accretion (~0.5-0.8% of Earth's mass)
    → Added HSEs to the MANTLE (core already closed)
    → Re-established chondritic relative HSE ratios in mantle
    Evidence:
    HSE ratios in mantle are near-chondritic (not fractionated by core formation)
    → They came in as a batch AFTER core formation

  IMPLICATION:
    Earth received ~0.5-0.8% of its mass as a late veneer
    This veneer may have delivered some of Earth's water and CHNS
    (Enstatite chondrites also proposed as early water source)

  METAL-SILICATE PARTITION EXPERIMENTS:
    High-P/T experiments show D values decrease with increasing P and T
    At magma ocean P (50-100 GPa): HSE D values lower than expected
    → Maybe core formation at high P explains some of the HSE budget
    (But still need late veneer to explain chondritic ratios)
```

## Isotopic Fingerprinting of Solar System Bodies

The mass-independent oxygen isotope composition of a planetary body acts as a provenance hash: a value computed from the body's isotopic state that reflects its formation location in the solar nebula and cannot be altered by subsequent geological processing. Unlike mass-dependent fractionation (which shifts isotope ratios by amounts proportional to the mass difference — any chemical or physical process can do this), mass-independent anomalies (Δ¹⁷O ≠ 0) record nuclear heterogeneity in the original nebular material inherited from different stellar nucleosynthesis sources. Chemical reactions cannot change the nucleosynthetic mixing ratio — it is "unforgeable" in the sense that no planetary process can shift a body's position on the three-isotope diagram. If two fragments share the same Δ¹⁷O value, they almost certainly formed from the same nebular reservoir — a provenance match analogous to a hash collision that is effectively impossible by chance.

```
MASS-INDEPENDENT ISOTOPE ANOMALIES
=====================================

  DISCOVERY:
    Clayton et al. 1973: CAIs have anomalous oxygen isotopes
    Δ¹⁷O = δ¹⁷O - 0.52 × δ¹⁸O ≠ 0 in many meteorites
    This violates mass-dependent fractionation
    → RECORDS NUCLEAR (not chemical) differences between solar system objects

  THREE-ISOTOPE OXYGEN DIAGRAM:
    All Earth + Moon samples: on single mass-dependent line (Δ¹⁷O ≈ -0.08)
    Mars: Δ¹⁷O ≈ +0.32
    Vesta (HED): Δ¹⁷O ≈ -0.24
    CAIs: Δ¹⁷O ≈ -0.50 (highly anomalous)
    → EACH BODY HAS A UNIQUE OXYGEN ISOTOPE SIGNATURE (its provenance hash)

  IMPLICATIONS:
    Earth + Moon have IDENTICAL oxygen isotopes → Theia (Moon impactor)
    must have come from the same nebular reservoir as Earth
    (This was NOT predicted; challenges simple giant impact models)

    Mars origin for SNC meteorites confirmed by Δ¹⁷O matching orbiters

  OTHER ISOTOPE SYSTEMS SHOWING ANOMALIES:
    ⁵⁴Cr, ⁹⁶Mo, ⁵⁰Ti: also show nucleosynthetic anomalies
    CC vs NC dichotomy:
    CC (carbonaceous) meteorites: enriched in s-process anomalies
    NC (non-carbonaceous) meteorites: Earth, Moon, Mars, OC, EC
    Suggests two distinct reservoirs in solar nebula → possibly Jupiter
    formed a barrier between them early
```

## Lunar Geochemistry

```
LUNAR GEOCHEMISTRY
==================

  ORIGIN: Giant Impact (Theia + Proto-Earth → Earth + Moon)
  AGE: ~4.51 ± 0.01 Ga (Hf-W and U-Pb chronometry)

  COMPOSITION COMPARED TO EARTH:
    SIMILAR: O, Ti, Cr, Si isotopes (same Δ¹⁷O = -0.08‰)
    DIFFERENT: Depleted in volatile elements (K, Na, Zn, Cl)
               Depleted in moderately siderophile (Fe: ~0.3× Earth's core fraction)
               Enriched in refractory lithophile (Ca, Al, Ti) in highlands

  GEOCHEMISTRY CONSTRAINTS ON GIANT IMPACT:
    Identical O isotopes: impactor Theia from same nebular region as Earth
    Volatile depletion: impact-driven devolatilization
    Small Fe core (~1.4% of Moon's mass): core mostly stayed in Earth or
    Theia had already depleted its iron (controversial)

  UREILITE PUZZLE:
    Moon's mass is ~99.5% from proto-Earth's mantle (not Theia)
    → High angular momentum impact; mixed silicates
    OR: vigorous mixing of Theia + Earth in a "synestia" (rapidly rotating
    disc-like body) before Moon condensed

  KREEP TERRANE:
    K (Potassium), REE, P — enriched in last-to-crystallize melt (LMO residue)
    Lunar Magma Ocean residue; concentrated on nearside
    PKT (Procellarum KREEP Terrane): youngest volcanic region; Th-enriched
    GRAIL gravity: crustal thickness 34-43 km; thinner nearside

  MARE vs HIGHLANDS:
    Highlands: anorthosite (plagioclase cumulate from Lunar Magma Ocean)
               Ferroan anorthosite (FAN): 4.46-4.51 Ga; primary crust
    Mare basalts: Mg-Fe basalt; 3.1-3.9 Ga; secondary volcanism
                  Low Ti/High Ti varieties; erupted from 200-400 km depth
```

## Mars Geochemistry

```
MARS GEOCHEMISTRY FROM METEORITES AND SPACECRAFT
==================================================

  BULK COMPOSITION:
    More FeO in mantle (18-19 wt%) vs Earth (8 wt%)
    More sulfur: higher S content in mantle → sulfide-rich
    Less Si in core: different partitioning during core formation
    Mars/Earth Mn:Fe ratio higher → more FeO in Martian system (more oxidized?)

  SNC METEORITES (Mars):
    Shergottites (youngest: 165-600 Ma; basalts)
    Nakhlites (1.3 Ga; clinopyroxenite; aqueous alteration = carbonates, salts)
    Chassignites (1.3 Ga; dunite)
    ALH84001 (4.09 Ga; orthopyroxenite; oldest; biosignature controversy 1996)

    MARTIAN ORIGIN CONFIRMED BY:
    Oxygen isotopes (Δ¹⁷O = +0.32; unique to Mars)
    Trapped gas: matches Mars atmosphere composition (measured by Viking)
    Cosmic ray exposure age: ~10-20 Myr in space (ejected by impact)

  SURFACE CHEMISTRY:
    High SO₄ content (sulfate minerals): acidic water; Hesperian volcanism
    Perchlorate (ClO₄⁻): ~0.5-1 wt% at some sites; Phoenix + CRISM
    Iron oxides (goethite, hematite): Fe³⁺ → red color; past oxidation
    Phyllosilicates (clay): Noachian; pH-neutral water required
    Carbonates: trace; if abundant → would expect in SNC; may be in subsurface

  PERSEVERANCE FINDINGS (Jezero, 2021-):
    Mafic igneous rocks at landing site (basaltic floor + delta)
    Organic compounds detected (not yet confirmed biological)
    Sulfate-bearing strata in delta sequences
    Sample tubes cached: 18+ cores → MSR program
```

## Common Confusion Points

**Meteorites are not all the same age as the solar system**: Iron meteorites and chondrites formed at 4.56 Ga. But SNCs from Mars are 165-1300 Ma — much younger. Lunar meteorites are 3.0-4.5 Ga. The age of formation depends on when the parent body last crystallized, not when the solar system formed.

**Carbonaceous chondrites are not all similar**: "Carbonaceous" spans CI, CM, CR, CO, CV, CK types with very different compositions. CI is most primitive (highest volatile content); CV (Allende) has abundant CAIs and chondrules but is more processed; CM (Murchison) has organic molecules. The classification reflects different nebular regions and alteration histories.

**The CI chondrite "reference" is rarely CI chondrite composition in practice**: Most comparative work is actually CI-normalized — you divide measured values by CI abundances to compare relative enrichments. Very few rocks in Earth's crust match CI composition in any element; the normalization shows patterns of depletion/enrichment.

**The Moon is not made from Theia**: Contrary to initial expectations from giant impact models, Lunar oxygen isotopes are essentially identical to Earth's. This means the Moon is mostly made of proto-Earth material (>99%), not from the impactor Theia. This is explained by either near-Earth origin for Theia or vigorous mixing in a post-impact synestia.
