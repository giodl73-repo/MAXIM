# Carbon Cycle Geochemistry

## The Big Picture

The carbon cycle operates on multiple timescales simultaneously. The short cycle (biology, ocean, atmosphere) exchanges carbon in years to millennia. The long cycle (tectonics, weathering, burial) regulates CO₂ over millions of years and is the ultimate controller of Earth's climate on geological timescales.

```
+------------------------------------------------------------------+
|                    CARBON CYCLE ARCHITECTURE                      |
+------------------------------------------------------------------+
|                                                                  |
|  SHORT CYCLE (yr-kyr)          LONG CYCLE (Myr-Gyr)              |
|  --------------------          ---------------------------        |
|  Atmosphere CO₂ (~800 Gt C)   Carbonate weathering              |
|  Land biosphere (~600 Gt C)   Silicate weathering               |
|  Soil (~1600 Gt C)            Volcanic outgassing               |
|  Ocean surface (~1000 Gt C)   Organic C burial                  |
|  Deep ocean (~38,000 Gt C)    Carbonate burial                  |
|  Marine sediments              Subduction recycling              |
|                                                                  |
|  KEY FLUXES (Gt C/yr):                                           |
|  GPP:         ~120 (photosynthesis, land)                       |
|  Respiration: ~119 (land ecosystem)                             |
|  Ocean-atm:   ~90 ± 2 (net: -2.5 Gt C/yr to ocean)             |
|  Weathering:  ~0.3 (silicate; long-cycle)                       |
|  Volcanism:   ~0.3 (outgassing; long-cycle)                     |
|  Human:       ~10 Gt C/yr (fossil fuels; short-cycle timescale) |
+------------------------------------------------------------------+
```

## Carbon Reservoirs and δ¹³C

```
CARBON RESERVOIR δ¹³C VALUES
==============================

  RESERVOIR              δ¹³C (‰ vs V-PDB)    EXPLANATION
  ---------              -----------------    -----------
  Mantle / volcanic CO₂  -5 to -7‰           Magmatic carbon
  Marine carbonates      0 to +5‰             Inorganic precipitation; near mantle
  Marine organic carbon  -20 to -25‰          Marine photosynthesis (C3 pathway)
  C3 land plants         -24 to -30‰          Calvin cycle; partial ¹³C discrim.
  C4 land plants         -10 to -14‰          Hatch-Slack pathway; less discrim.
  Marine DIC (dissolved) +0.5 to +2.5‰        Modern ocean surface
  Deep ocean DIC         -0.5 to +1.5‰        Remineralization removes ¹²C
  Freshwater DIC         -12 to +2‰           Soil CO₂ (respiration) influence
  Soil CO₂               -20 to -25‰          Plant decomposition
  Methane (biogenic)     -50 to -80‰          Methanogenesis (extreme depletion)
  Methane (thermogenic)  -25 to -50‰          Thermal breakdown of organic matter
  Coal                   -23 to -26‰          Land plant kerogen
  Marine shale TOC       -25 to -32‰          Marine organic matter burial
  Petroleum              -25 to -32‰          Marine origin

  KEY DISCRIMINANT:
  ¹²CO₂ is preferentially fixed by photosynthesis (RuBisCO enzyme)
  → Organic carbon is ¹³C-depleted by ~20-25‰ relative to source CO₂
  → This ¹³C fractionation is the fingerprint of biological activity
```

## The Long-Term Carbon Cycle

### Silicate Weathering: The Thermostat

```
SILICATE WEATHERING FEEDBACK
=============================

  REACTION (simplified):
    CaSiO₃ + CO₂ → CaCO₃ + SiO₂
    Silicate rock + CO₂ → Carbonate + silica
    (Urey reaction: reaction consuming atmospheric CO₂)

  FULL CYCLE:
    WEATHERING: CO₂ + H₂O + CaSiO₃ → Ca²⁺ + 2HCO₃⁻ + SiO₂
    (on land; ~100,000 yr timescale; consumes CO₂)
         |
         v  rivers transport Ca²⁺ + HCO₃⁻ to ocean
    CARBONATE BURIAL: Ca²⁺ + 2HCO₃⁻ → CaCO₃ + CO₂ + H₂O
    (in ocean; net CO₂ removed from atmosphere half returned)
         |
         v  carbonate sediment subducted
    VOLCANIC OUTGASSING: CaCO₃ + SiO₂ → CaSiO₃ + CO₂
    (metamorphism + volcanism returns CO₂ to atmosphere)
         |
         v CYCLE REPEATS

  NET RESULT:
    Silicate weathering + carbonate burial = CO₂ drawdown
    Volcanism = CO₂ release
    FEEDBACK: Warm → more rainfall → more weathering → CO₂ drawdown → cool
              Cold → less weathering → CO₂ builds up → warm
    → THERMOSTAT ON GEOLOGICAL TIMESCALES (~100,000 to 1 million yr)
    → Prevented Venus-like runaway greenhouse on early Earth
```

### Organic Carbon Burial

```
ORGANIC CARBON BURIAL FEEDBACK
================================

  PHOTOSYNTHESIS: CO₂ + H₂O → CH₂O + O₂
    Removes CO₂; produces O₂
    On short timescale: reverses by respiration

  BURIAL: If organic matter buried before oxidizing:
    Net: atmospheric CO₂ consumed; O₂ released
    Kerogen (in sedimentary rock) is the buried organic record
    Black shale = high TOC (Total Organic Carbon) = burial event

  FEEDBACKS:
    More O₂ in atm. → more oxidation of organic C at surface → less burial
    Less O₂ → less oxidation → more burial
    Self-limiting feedback; prevents runaway O₂ or anoxia

  HISTORICAL SIGNIFICANCE:
    Carboniferous (~350-300 Ma): massive organic burial (swamp forests)
    → Atmospheric O₂ peaked at ~30-35% (vs 21% today)
    → CO₂ fell to <300 ppm → Carboniferous glaciation
    → Coal deposits = that buried organic carbon

  δ¹³C COUPLING:
    More organic burial → ¹²C removed from ocean/atmosphere
    → Residual DIC becomes ¹³C-enriched
    → Positive carbonate δ¹³C excursion = burial event
    → NEGATIVE excursion = release of organic carbon
    δ¹³C stratigraphy is a primary tool for identifying perturbations
```

## Oceanic Anoxic Events (OAEs)

```
OCEANIC ANOXIC EVENTS
======================

  Definition: Episodes of widespread ocean anoxia; black shale deposition;
  marked by positive δ¹³C excursions (massive organic carbon burial)

  CAUSE: Enhanced marine productivity + stratified ocean + reduced O₂

  MAJOR OAEs:
  OAE2 (Cenomanian-Turonian, 93.5 Ma):
    Largest δ¹³C excursion: +3-4‰ in marine carbonates
    Duration: ~500,000 yr
    Global: black shales on all continents
    Trigger: Large Igneous Province (Caribbean LIP) volcanism
    → CO₂ → ocean acidification → nutrient injection → productivity spike
    → Organic matter overwhelms O₂ supply → anoxia

  OAE1a (Selli Event, ~120 Ma):
    +3-4‰ δ¹³C excursion
    Ontong Java Plateau LIP volcanism

  Toarcian OAE (~183 Ma):
    +3-5‰; Karoo-Ferrar LIP
    Also: negative CIE (carbon isotope excursion) preceding the positive one
    Negative CIE = initial carbon release; positive CIE = subsequent burial

  DIAGNOSTIC SIGNALS:
    Black shale: high TOC (1-20%); pyrite; laminated (no bioturbation)
    Positive δ¹³C in carbonates: records enhanced organic burial
    Negative Δ¹⁷O in sulfate: change in photosynthesis rate
    Os isotope spike: LIP volcanism brings mantle Os to surface
```

## Kerogen and Petroleum Systems

```
ORGANIC MATTER TRANSFORMATION (MATURATION)
==========================================

  FRESH ORGANIC MATTER
  → Diagenesis (burial, <50°C, bacteria active)
    → Biochemical degradation
    → Type I kerogen (algal): waxy, oil-prone
    → Type II kerogen (mixed): marine; most common oil source
    → Type III kerogen (woody): coaly; gas-prone
    → Type IV kerogen (inertinite): oxidized; no generation

  → Catagenesis (50-150°C, 2-5 km burial)
    → Early: waxy crude oil (high wax)
    → Peak: light oil (API 30-45°)
    → Late: wet gas + condensate

  → Metagenesis (>150°C)
    → Dry gas (CH₄ only)
    → Graphitization: all H driven off

  VITRINITE REFLECTANCE (Ro):
    Vitrinite (woody organic matter) reflectance increases with T
    Ro < 0.5%: immature (no oil generation)
    Ro 0.5-0.9%: oil window
    Ro 0.9-1.3%: wet gas/condensate
    Ro > 1.3%: dry gas; or barren (overmature)
    USED: measure Ro → determine thermal maturity of source rock

  δ¹³C OF KEROGEN AND OIL:
    Marine type II source: δ¹³C ≈ -25 to -30‰
    Terrestrial type III source: δ¹³C ≈ -23 to -27‰
    Oil δ¹³C closely tracks source rock δ¹³C
    → Used in oil-source correlation studies
    → Also: n-alkane δ¹³C patterns (compound-specific isotope analysis)
```

## The Carbonate System in the Ocean

```
OCEAN CARBONATE CHEMISTRY
===========================

  CO₂ DISSOLUTION:
    CO₂(g) ⇌ CO₂(aq)    [KH = Henry's law]
    CO₂(aq) + H₂O ⇌ H₂CO₃ ⇌ H⁺ + HCO₃⁻ ⇌ 2H⁺ + CO₃²⁻

  EQUILIBRIUM (seawater pH ~8.1):
    CO₂ + H₂O: ~0.5% of DIC
    HCO₃⁻:     ~88% of DIC     (dominant species)
    CO₃²⁻:     ~11% of DIC

  DIC = dissolved inorganic carbon = [CO₂] + [HCO₃⁻] + [CO₃²⁻] ≈ 2100 μmol/kg

  ALKALINITY (TA):
    TA ≈ [HCO₃⁻] + 2[CO₃²⁻] + [B(OH)₄⁻] + [OH⁻] - [H⁺]
    TA ≈ 2350 μmol/kg (modern ocean)
    TA set by riverine input of cations (weathering products)
    TA removed by carbonate burial

  SATURATION STATE (Ω):
    Ω = [Ca²⁺][CO₃²⁻] / K_sp
    Ω > 1: oversaturated → calcite precipitates
    Ω < 1: undersaturated → calcite dissolves
    Modern surface ocean: Ω_calcite ≈ 4-6 (oversaturated)
    Modern deep ocean: Ω < 1 (undersaturated → carbonate dissolves)

  CARBONATE COMPENSATION DEPTH (CCD):
    Depth where dissolution rate = rain rate of carbonate
    Below CCD: no carbonate preserved in sediment
    Modern CCD: ~4-5 km (varies with ocean basin)
    Lysocline: depth of onset of significant dissolution (shallower than CCD)
    During PETM/OAEs: CCD shoaled dramatically → carbonate dissolution widespread
```

## GEOCARB and Long-Term CO₂ Reconstruction

```
GEOCARB MODEL AND CO₂ HISTORY
================================

  GEOCARB (Berner & Kothavala 2001; COPSE model):
    Models long-term CO₂ by balancing:
    - Silicate weathering (CO₂ sink)
    - Organic carbon burial/oxidation (C sink/source)
    - Volcanic/metamorphic outgassing (C source)
    - Carbonate cycling

  PALEO CO₂ PROXY DATA:
    Stomatal density: plants reduce stomata in high CO₂ → counts fossils
    δ¹¹B in forams: paleo-pH → paleo-CO₂ (known DIC)
    Nahcolite minerals: NaHCO₃ forms only when CO₂ > ~1000 ppm?
    Paleosol carbonates: δ¹³C of soil carbonate + organic matter → pCO₂

  APPROXIMATE CO₂ HISTORY:
    ~600 Ma (pre-Cambrian): very uncertain; possibly 1000-7000 ppm
    Ordovician (450 Ma): ~4000-8000 ppm; yet glaciation (high obliquity + weak Sun)
    Carboniferous (300 Ma): ~300-500 ppm (forests; organic burial)
    Early Jurassic (180 Ma): ~1000-2000 ppm (CAMP volcanism)
    Cretaceous (100 Ma): ~1000-2000 ppm (warm; no polar ice)
    Eocene (50 Ma): ~500-1000 ppm
    Pliocene (3 Ma): ~350-400 ppm (warmer; sea level 10-25 m higher)
    Pre-industrial: 280 ppm
    2024: ~425 ppm (fastest increase in geological record)
```

## Decision Cheat Sheet

| Question | Answer |
|---|---|
| What is the Urey reaction? | CaSiO₃ + CO₂ → CaCO₃ + SiO₂; the net silicate weathering reaction that consumes CO₂; the key long-term climate thermostat |
| What does a positive δ¹³C excursion in marine carbonates mean? | Enhanced organic carbon burial; ¹²C removed from ocean/atmosphere → residual DIC becomes ¹³C enriched; often associated with OAEs |
| What does a negative δ¹³C excursion mean? | Release of ¹³C-depleted carbon from organic reservoir (kerogen, methane hydrates, permafrost); e.g., PETM |
| What is the CCD? | Carbonate Compensation Depth — below this, carbonate dissolves faster than it rains from surface; no carbonate in deep sediments below CCD |
| How does organic carbon burial oxygenate the atmosphere? | Photosynthesis produces O₂ + organic C; if organic C is buried instead of oxidized, the O₂ is not consumed → net O₂ accumulation |
| What is vitrinite reflectance? | Optical reflectance of vitrinite particles in sediment; increases with thermal maturity; Ro 0.5-0.9% = oil generation window |
| Why do Large Igneous Provinces trigger OAEs? | LIP volcanism injects CO₂ → ocean acidification → collapse of carbonate buffer → plus nutrient runoff → productivity spike → organic burial → anoxia |
| What is kerogen type III? | Woody/terrestrial organic matter; high C/H; gas-prone; associated with coals and deltaic/terrigenous sequences |

## Common Confusion Points

**The short carbon cycle and long carbon cycle operate on very different timescales**: Human emissions (~10 Gt C/yr) overwhelm the long-cycle fluxes (volcanism + weathering ~0.3 Gt C/yr) by ~30×. The silicate weathering thermostat cannot respond on human timescales (~100 kyr). The ocean absorbs ~25-30% of current emissions, but over centuries the only "fix" is geological.

**DIC is not the same as carbonate**: DIC = dissolved inorganic carbon = [CO₂] + [HCO₃⁻] + [CO₃²⁻]. Most DIC in seawater is bicarbonate (HCO₃⁻), not carbonate (CO₃²⁻). Carbonate ion is only ~11% of DIC despite being what organisms use to build shells.

**Ocean acidification doesn't make the ocean acidic**: Modern surface ocean pH is ~8.1 (alkaline). "Acidification" means pH is decreasing (currently ~0.1 units since pre-industrial). The ocean becoming "acid" would require orders of magnitude more CO₂ than is plausible. The concern is carbonate mineral dissolution at decreased saturation state.

**Positive CIEs (carbon isotope excursions) are not warming events**: A positive δ¹³C excursion records organic carbon burial, which removes CO₂ and COOLS climate. A negative CIE (like the PETM) records carbon injection and WARMS. The sign is the opposite of what intuition might suggest if you conflate δ¹³C excursion sign with temperature direction.
