# Element Distribution and Geochemical Reservoirs

## The Big Picture

The distribution of elements in Earth and the solar system is the result of nucleosynthesis (what the universe made), nebular condensation (what condensed where), and planetary differentiation (what went where inside planets).

```
ELEMENT DISTRIBUTION: HIERARCHY + FEEDBACKS
+------------------------------------------------------------------+
|                                                                  |
|  NUCLEOSYNTHESIS → SOLAR SYSTEM COMPOSITION → PLANET FORMATION  |
|  (Big Bang + stars   (CI chondrites ≈ solar    (differentiation: |
|   set cosmic abund.)  photosphere composition)   core/mantle/crust)|
|                                                        |         |
|       ←————————————————————————————————————————————————         |
|       RECYCLING LOOPS (cyclic, not one-way):                     |
|                                                                  |
|  CRUST ——[subduction]——→ MANTLE                                  |
|    ↑ [volcanism / arc magmatism]  ↓                              |
|    ←————[partial melting]—————————                               |
|                                                                  |
|  ATMOSPHERE ←→ OCEAN ←→ SEDIMENTS ←→ CRUST                      |
|  (weathering; diagenesis; carbonate burial; uplift erosion)      |
|                                                                  |
|  The system has been cycling for 4+ Gyr: the continental crust  |
|  is the enriched residue of billions of years of this loop,     |
|  not a static reservoir from initial differentiation.           |
+------------------------------------------------------------------+
```

## Cosmic Abundances

### The Solar System Pattern

```
COSMIC ABUNDANCE PATTERN
=========================

  (relative to Si = 10⁶ atoms)

  H:  2.79 × 10¹⁰    DOMINANT — Big Bang nucleosynthesis
  He: 2.72 × 10⁹     Big Bang + stellar He burning
  O:  1.69 × 10⁷     Stellar burning; most abundant heavy element
  C:  9.09 × 10⁶     Stellar burning (CNO cycle)
  Ne: 3.44 × 10⁶     Stellar burning
  N:  2.27 × 10⁶     Stellar burning (CNO cycle)
  Mg: 1.07 × 10⁶     Silicon burning + s-process
  Si: 1.00 × 10⁶     (reference)
  Fe: 9.00 × 10⁵     Iron peak — most stable nucleus

  KEY PATTERN:
  Even atomic number elements > odd (Oddo-Harkins rule)
  Iron is the most stable nucleus → "iron peak" in abundance
  Elements heavier than Fe made by neutron capture (s/r process)
  → exponential decrease in abundance with increasing mass number

  COMPARISON TO EARTH:
  Earth is depleted in: H, C, N, noble gases (volatile elements)
  Earth is enriched in: Fe, Mg, Si (refractory, non-volatile)
  This reflects CONDENSATION SEQUENCE: Earth formed from refractory condensates
```

## CI Chondrite Reference

```
CI CHONDRITE REFERENCE COMPOSITION
=====================================

  CI chondrites: most primitive meteorites; formed from low-T condensates
  Composition closely matches solar photosphere (except H, He, noble gases)
  → Used as the reference "bulk solar system" composition

  ELEMENT     CI CHONDRITE    CONTINENTAL CRUST
  -------     ------------    -----------------
  Si          106,000 ppm     265,000 ppm (enriched in CC)
  Fe          182,000 ppm     39,000 ppm (depleted in CC vs CI)
  Mg          95,700 ppm      15,900 ppm (depleted in CC)
  Al          8,680 ppm       77,400 ppm (enriched in CC)
  Ca          9,200 ppm       25,700 ppm (enriched in CC)
  K           550 ppm         23,000 ppm (STRONGLY enriched in CC)
  U           0.007 ppm       1.68 ppm   (STRONGLY enriched in CC)
  Th          0.03 ppm        6.03 ppm   (STRONGLY enriched in CC)

  WHY CC IS ENRICHED IN K, U, Th:
    All three are incompatible; K is LILE; U and Th are lithophile
    Repeated partial melting over 4 Ga extracts them into crust
    → Continental crust is the "scum of the Earth" chemically
      (concentrated incompatible elements from billions of years of magmatism)

  BSE (Bulk Silicate Earth) reference:
    BSE = mantle + crust (excludes core)
    Approximately: CI composition × (1 - core fraction) with volatiles removed
```

## Incompatibility and Fractionation

### Partition Coefficients

```
PARTITION COEFFICIENTS IN DETAIL
==================================

  D(mineral/melt) = C_mineral / C_melt

  GENERAL CONTROLS ON D:
  1. IONIC RADIUS: must fit into the crystal lattice site
  2. IONIC CHARGE: must match the host site charge
  3. TEMPERATURE: higher T → more disordered → elements easier to accommodate
  4. PRESSURE: changes which phase is stable; olivine vs garnet vs spinel

  KEY PARTITION DATA (olivine):
    Ni:   D ~ 4-10     (fits Mg site; compatible)
    Co:   D ~ 1-3      (fits Mg site; slightly compatible)
    Cr:   D ~ 0.1-10   (variable; spinel > olivine)
    Ca:   D ~ 0.001    (too large for Mg site; incompatible)
    REE:  D << 0.01    (too large; highly incompatible)
    U:    D ~ 0.0001   (highly incompatible)

  GARNET vs SPINEL:
    Garnet: HREE compatible (HREEs fit in garnet dodecahedral site)
    Spinel: all REE incompatible
    → Melt from garnet-bearing mantle: HREE depleted, (La/Yb)_N > 1
    → Melt from spinel lherzolite: flat REE pattern
    → This is a DEPTH OF MELTING indicator
```

### REE Patterns

```
RARE EARTH ELEMENT (REE) PATTERNS
====================================

  REEs: La, Ce, Pr, Nd, Sm, Eu, Gd, Tb, Dy, Ho, Er, Tm, Yb, Lu
  Normalized to CI chondrite to show enrichments/depletions

  MORB (Mid-Ocean Ridge Basalt):
    Slightly depleted in LREE; flat to MREE-HREE
    Source: depleted upper mantle
    (La/Sm)_N < 1; (Dy/Yb)_N ≈ 1

  OIB (Ocean Island Basalt — e.g., Hawaii):
    Enriched in LREE (La/Sm)_N > 1
    Source: less depleted "plume" mantle or enriched mantle
    Steep enrichment pattern

  CONTINENTAL CRUST:
    Strongly enriched in LREE; depleted in HREE
    Classic "steep" REE pattern
    Eu ANOMALY (negative): plagioclase fractionation removes Eu²⁺
      → melts/crustal rocks derived from fractionation show Eu depletion

  EU ANOMALY:
    Eu²⁺ (unlike other REEs which are 3+) can substitute for Ca²⁺
    → Plagioclase has D_Eu >> D_other REE
    → Fractionation of plagioclase leaves Eu-depleted melt
    → Plagioclase accumulate are Eu-enriched
    → Used to detect plagioclase fractionation in crustal evolution
```

## Geochemical Reservoirs in Detail

```
CONTINENTAL CRUST COMPOSITION
==============================

  UPPER CRUST:       Average measured from surface rocks
    SiO₂:  66.0%     Al₂O₃: 15.2%   FeOt:  4.5%
    MgO:    2.2%     CaO:    4.2%    Na₂O:  3.9%
    K₂O:    3.4%     TiO₂:   0.6%
    "Granodioritic" composition on average

  LOWER CRUST:       Inferred from xenoliths + seismic + xenoliths
    More mafic: higher Mg, Fe, Ca; lower Si, K
    Represents cumulate + residue from early magmatism

  BULK CC:           ~60% SiO₂ by mass
    Continental crust is ~0.6% of Earth's total mass
    Yet contains ~30% of Earth's K, U, Th, Rb
    → It is the chemical "differentiated scum"

DEPLETED MANTLE (DM):
    MORB source mantle
    Strongly depleted in LILE, LREE
    143Nd/144Nd relatively high (positive εNd) — time-integrated Nd enrichment
    87Sr/86Sr relatively low — time-integrated Sr depletion

PRIMITIVE MANTLE (PM):
    Hypothetical; CI-chondrite normalized to refractory/involatile composition
    Approximately = mantle before any crust extraction (model)
    εNd = 0 by definition; 87Sr/86Sr = 0.7045 (approximate)

ENRICHED MANTLE (EM1, EM2):
    Mantle domains with recycled crustal or sedimentary material
    EM1: high 206Pb/204Pb; low 143Nd/144Nd
    EM2: very high 87Sr/86Sr (sediment recycling)
```

## Trace Element Spider Diagrams

```
SPIDER DIAGRAM INTERPRETATION
==============================

  Normalized to primitive mantle (or CI chondrite)
  Elements ordered by increasing incompatibility (left = most incompatible)

  MORB PATTERN:
    Ba Rb Th Nb La Ce Sr Nd Sm Zr Hf Ti Y Yb
    ↑ depleted relative to PM (flat-slightly enriched in MREE-HREE)
    Shows: extraction of most incompatible elements into crust

  ARC MAGMA PATTERN (subduction zone):
    Strong Nb-Ta TROUGH
    High Ba, Rb (fluid-mobile LILEs: released from slab)
    Low Nb, Ta, Ti (immobile in fluids; retained in subducted slab)
    → THE Nb-Ta depletion is the diagnostic signature of arc/subduction

  OCEAN ISLAND BASALT (OIB):
    Enriched overall relative to MORB
    No Nb-Ta depletion (not contaminated by slab fluids)
    Steep pattern: LREEs >> HREEs

  WHY Nb-Ta IS DIAGNOSTIC:
    Nb and Ta are immobile in aqueous fluids (high field strength)
    When the subducted oceanic crust dehydrates → fluid fluxes the overlying wedge
    → LILEs (Ba, Rb, K) go up with fluid into arc crust
    → Nb, Ta stay behind in the slab
    → Arc magma has LILE-enriched, Nb-Ta-depleted pattern
    → This signature is imprinted on the continental crust
       (which forms partly via arc magmatism)
```

---

**Spider diagrams as source deconvolution**: A spider diagram (trace element concentrations normalized to a reference composition, elements ordered by incompatibility) is a source fingerprint. The shape encodes the history of which phases were present during melting and metasomatism. Reading a spider diagram is analogous to matching a signal's frequency-domain signature against known source templates: the Nb-Ta trough is as diagnostic of subduction involvement as a specific spectral notch is of a known filter. Different mantle reservoirs (DMM, HIMU, EM1, EM2) have distinct spider diagram shapes; mixing between them produces intermediate patterns. This is geochemical signal processing — decompose the observed pattern into contributions from end-member sources.

## Mantle Heterogeneity

```
MANTLE DOMAINS (Zindler-Hart Classification 1986)
===================================================

  DMM: Depleted MORB Mantle
    The average depleted upper mantle
    Source of most MORB
    Low ⁸⁷Sr/⁸⁶Sr; high εNd

  HIMU: High μ = high ²³⁸U/²⁰⁴Pb
    Ancient recycled oceanic crust (≥1-2 Ga in mantle)
    High ²⁰⁶Pb/²⁰⁴Pb; ²⁰⁷Pb/²⁰⁴Pb; moderate ⁸⁷Sr/⁸⁶Sr
    St. Helena, Mangaia, Cook Islands

  EM1: Enriched Mantle 1
    Recycled lower continental crust or ancient subcontinental lithosphere
    High ⁸⁷Sr/⁸⁶Sr; low εNd; low ²⁰⁶Pb/²⁰⁴Pb
    Tristan da Cunha, Pitcairn

  EM2: Enriched Mantle 2
    Recycled sediment (subducted oceanic sediments)
    Very high ⁸⁷Sr/⁸⁶Sr; low εNd; moderate Pb
    Samoa, Society Islands

  FOZO/C: Common component
    Focus zone; primitive-ish; seen in many OIBs
    May be deep primitive mantle reservoir (lower mantle?)

  KEY POINT: Mantle is heterogeneous due to 4+ billion years of
  subduction cycling and incomplete mixing
  Blobs of ancient subducted material persist in the mantle
```

## Decision Cheat Sheet

| Question | Answer |
|---|---|
| Why does continental crust have so much K and U? | Highly incompatible elements; billions of years of partial melting extract them preferentially into the melt → accumulate in crust |
| What is the Eu anomaly and what causes it? | Deficit or excess in Eu relative to neighboring REE; caused by plagioclase fractionation (Eu²⁺ substitutes in plagioclase); negative Eu = plagioclase fractionated out |
| What is the Nb-Ta depletion signature? | Deficit of Nb and Ta relative to neighboring elements in spider diagrams; diagnostic of arc/subduction magmas; HFSE retained in subducted slab while LILEs fluid-mobilized |
| Why is depleted mantle called "depleted"? | It has had partial melts extracted (which went to form crust), removing incompatible elements; residue is depleted relative to primitive mantle |
| What is MORB vs OIB? | MORB = Mid-Ocean Ridge Basalt (depleted mantle source, flat REE); OIB = Ocean Island Basalt (plume, enriched source, steep REE) |
| How are mantle domains identified? | Sr-Nd-Pb radiogenic isotope ratios record time-integrated parent/daughter element fractionation; different domains = different histories |
| What is the iron peak in cosmic abundances? | ⁵⁶Fe has the highest binding energy per nucleon → most stable nucleus → stellar nuclear burning converges toward Fe → peak in cosmic abundances at Z=26 |

## Common Confusion Points

**Incompatible elements are not rare**: U, Th, K are highly incompatible (they hate staying in minerals) but are not necessarily rare overall. "Incompatible" describes behavior in crystals, not cosmic or planetary abundance.

**REE are not rare**: Despite the name, rare earth elements are not rare in Earth's crust. Cerium (Z=58) is as common as copper. "Rare" referred to their unusual minerals and difficulty of separation — not abundance. China dominates production because of favorable ore deposit geology, not rarity.

**Depleted mantle does not mean iron-depleted**: "Depleted" in geochemical context means depleted in incompatible trace elements (K, Rb, U, Th, REEs, Ba). The depleted mantle still has normal major element composition (Fe, Mg, Si, O). Don't confuse trace-element depletion with major-element composition.

**EM1 and EM2 enriched mantles are enriched by recycling, not primordial**: The "enrichment" in EM1/EM2 came from subduction of crustal or sedimentary material back into the mantle — it's been there for 1-2 billion years and developed high 87Sr/86Sr via Rb decay. It was not enriched from the start.
