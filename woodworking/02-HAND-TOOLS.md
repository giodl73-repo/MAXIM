# Hand Tools: Planes, Chisels, Saws, Marking Tools — Setup and Use

## The Big Picture

Hand tools are not slower versions of power tools — they are precision instruments for specific operations. The learning curve is steep because the tools require setup and tuning, and because technique is inseparable from the tool. A properly tuned bench plane in skilled hands produces surfaces that no power sander can match and does so quietly, with precise control.

```
HAND TOOL CATEGORIES AND ROLES
================================

BENCH PLANES (stock preparation + final surfaces)
  No.4 Smoother:    Final surface; narrow mouth; light cut
  No.5 Jack:        Rough removal; cambered iron; fast stock reduction
  No.7 Jointer:     Flattening long boards; jointing edges for glue-up

SPECIALIZED PLANES
  Router plane:     Clean grooves, dadoes, tenon cheeks to precise depth
  Shoulder plane:   Trim tenon cheeks (metal-bodied; full-width iron)
  Rabbet plane:     Cut rabbets (ledges); iron extends to edge
  Plow plane:       Cut grooves/dadoes; adjustable fence + depth stop

CHISELS
  Bench chisel:     Paring and chopping mortises (thick back, strong)
  Mortise chisel:   Deep mortises (thicker, stiffer than bench)
  Paring chisel:    Light trimming (longer, thinner; push only; no mallet)
  Skew chisel:      Angled edge; into corners

SAWS
  Western crosscut: Cut perpendicular to grain (teeth cut on push stroke)
  Western rip:      Cut with grain (fewer, larger teeth)
  Japanese dozuki:  Fine crosscut; cuts on pull; thin kerf; precise
  Japanese ryoba:   Double-edged (rip + crosscut combined)
  Coping saw:       Curves (thin blade, frame)
  Bow saw:          Curves (thicker; traditional frame saw)

LAYOUT AND MARKING
  Marking gauge:    Scribe a line parallel to face or edge
  Mortise gauge:    Two scribers; marks both sides of mortise simultaneously
  Sliding bevel:    Sets/transfers angles
  Combination square: 90° and 45° + marking depth and height
  Marking knife:    Scribe precise layout lines (knife, not pencil)
```

## Bench Planes — The Core System

The Stanley/Bailey-pattern bench plane system (still the dominant design):

```
BENCH PLANE ANATOMY
====================

    chip breaker (back iron)
      |
      |    blade (iron)
      |      |
    +------------------+
    |   ===frog===     |
    |   adjustable     |
    |   angle and      |
    |   Y-adjustment   |
    +------------------+
    |   body (sole)    |
    |   mouth          |
    +------------------+
    (mouth = the gap between sole and blade)

ADJUSTMENTS:
  Frog position: controls mouth opening (narrower = finer cut; tighter = harder to clear shavings)
  Blade depth (Y-adjuster): how much iron projects below sole
  Lateral adjustment lever: left-right alignment of cutting edge

THE THREE KEY PLANES:
+----+----------+--------+-----+------------------------------------------------+
| No | Name     | Length | Iron| Primary role                                   |
+----+----------+--------+-----+------------------------------------------------+
| 4  | Smoother | 9–10"  | 2"  | FINAL SURFACE; fine cut; narrow mouth          |
| 5  | Jack     | 14–15" | 2"  | FAST STOCK REMOVAL; cambered iron preferred    |
| 7  | Jointer  | 22–24" | 2⅜" | FLATTENING long boards; jointing edges         |
+----+----------+--------+-----+------------------------------------------------+

OTHER USEFUL NUMBERS:
  No.3: shorter smoother; better for small panels
  No.6: between jack and jointer; some prefer as all-purpose
  Low-angle block plane: end grain; chamfers; one-handed use
  Low-angle jack: end grain and difficult grain (20° bed angle vs. 45°)
```

## Setting Up a Bench Plane

Most new planes (and all used planes) require setup before they work well:

```
PLANE SETUP SEQUENCE
======================

STEP 1: FLATTEN THE SOLE
  The sole must be flat, especially at the toe, heel, and mouth
  Use: coarse sandpaper (80–120 grit) on a flat reference surface
  (granite plate or float glass)
  Check with straightedge: should rock on nothing
  Time: 15–60 min per plane; do it once

STEP 2: SHARPEN THE BLADE
  Back (flat side): must be FLAT — this is the reference surface
  Grind flat on coarse stone (120–400 grit); polish to mirror (1000–8000 grit)
  Bevel: 25° primary, 30° secondary (micro-bevel on A2/O1 steel)
  Method: honing guide OR freehand (both work with practice)
  Test: the "hair-shaving" test (blade should shave arm hair)
  The back matters as much as the bevel

STEP 3: CHIP BREAKER SETUP
  The chip breaker (back iron) breaks shavings before they cause tearout
  Distance from cutting edge: 0.5–1.5mm for typical work
  Closer: fewer tearouts in difficult grain; more resistance/drag
  Farther: faster cutting; more tearout risk
  The seating: chip breaker must mate perfectly with blade back (no gap)
  A gap allows shavings to jam between them → clog the mouth

STEP 4: MOUTH ADJUSTMENT
  Narrow mouth: reduces tearout; required for smoothing
  Wide mouth: faster; more clearance for shavings
  Set narrow for final passes; widen if choking on shavings
```

## Chisel Selection and Use

```
CHISEL TYPES BY APPLICATION
=============================

BENCH CHISELS:
  Geometry: thicker back; can be used with mallet
  Width set needed: 1/4", 3/8", 1/2", 3/4", 1"
  (Cover most mortise and joint work)

MORTISE CHISELS:
  Geometry: very thick, nearly square cross-section
  Purpose: prying out waste from deep mortises without flexing
  Sizes: match mortise width (common: 3/8", 1/2")
  Only use with mallet; too stiff for paring

PARING CHISELS:
  Geometry: long, thin, delicate
  Purpose: light trimming of joints to fit; final fitting
  NEVER hit with mallet; push with hand pressure only
  The feel: remove paper-thin amounts; control the cut

BEVEL ANGLE REFERENCE:
  Tool steel      Primary bevel    Secondary bevel (micro-bevel)
  O1 (oil-harden) 25°              30°
  A2 (air-harden) 25°              30°
  PMV-11          25°              no secondary needed
  Japanese chisels use hollow-ground back (ura) — makes back flattening easier
```

## Saw Geometry and Use

The fundamental distinction: western saws cut on the push stroke, Japanese saws cut on the pull stroke.

```
SAW GEOMETRY COMPARISON
=========================

WESTERN (push-stroke):
  Blade: thick (stiffer; requires less blade tension)
  Teeth: set alternately left/right to provide clearance (kerf wider than blade)
  Cut control: push into wood; body weight helps on crosscut

JAPANESE (pull-stroke):
  Blade: thin (strong in tension on pull; would buckle on push)
  Kerf: narrower; less waste; cleaner cut
  Tooth geometry: impulse-hardened; cannot resharpen (replace blade)
  Beginner advantage: pull stroke tends to track straighter on start

TOOTH GEOMETRY:
  CROSSCUT: angled teeth; sever fibers perpendicular to grain
            (knife-like cutting action)
  RIP:       chisel-like teeth; scrape fibers parallel to grain
             (filing/chiseling action)
  Use wrong type: twice the effort, rough cut, possible damage

JAPANESE SAW TYPES:
  Dozuki:    back saw equivalent; thin blade; fine teeth; joinery
  Ryoba:     double-edged; one side rip, one crosscut; versatile
  Kataba:    single-edged; deeper cuts (no spine); general use
  Azebiki:   flush-cutting; teeth on both faces of curved blade
             for cutting dowels/plugs flush to surface
```

## Marking and Layout Tools

In hand-tool work, precision starts with layout. Pencil lines are for approximation; knife lines are for cutting.

```
MARKING TOOL PURPOSES
======================

MARKING GAUGE:
  One fence + one pin/wheel
  Scribes a line parallel to a reference face/edge
  Use: marking tenon thickness, dovetail baseline, dado depth
  Wheel-type (veritas) vs. pin-type: wheel leaves cleaner line on end grain

MORTISE GAUGE:
  Two adjustable pins + fence
  Sets both sides of a mortise simultaneously
  Lock pin spacing = chisel width; fence = distance from face
  Essential for consistent mortise-and-tenon layout

COMBINATION SQUARE:
  12" rule with sliding head; 90° and 45° surfaces
  Check for square (set 90° face on known flat; scribe line; flip; check offset)
  Uses: check workpieces for square; set marking gauge; depth of cut

SLIDING BEVEL:
  Adjustable blade for transferring and setting any angle
  Set to protractor; or copy angle from actual joint
  Uses: dovetail angles, chair leg angles, miter checks

MARKING KNIFE (vs. pencil):
  Knife: severs surface fibers; creates a perfect reference line
         that can be used to register chisel directly in the line
  Pencil: approximate; ~0.5mm wide line; not precise enough for joinery
  Rule: always mark joinery with a knife; mark general layout with pencil
```

## Sharpening — The Non-Negotiable Skill

```
SHARPENING SYSTEM OPTIONS
==========================

WATERSTONES (Japanese):
  Grits: 220/400 (coarse), 1000 (medium), 4000–6000 (fine), 8000+ (polish)
  Fast cutting; softer than oilstones; must be flattened periodically
  Best for: O1, A2 steel; Japanese chisels
  Water-based; quick cleanup

OILSTONES (Arkansas, India):
  India stones: medium-fast; long life; oil-based
  Arkansas (soft/hard/translucent): progresses to extremely fine
  Slower than waterstones; cuts O1 well; lasts essentially forever
  Traditional; now less common than waterstones

DIAMOND PLATES:
  Very flat (no periodic flattening needed)
  Fast cutting; good for A2 and PMV-11 steel
  Also used to flatten waterstones
  Monocrystalline: last longer; electroplated: cheaper but wear faster

HONING GUIDES:
  Hold chisel/plane blade at consistent angle
  Eliminates rocking during honing stroke
  Trade-off: slower tool changes; but produces consistent results
  Best for beginners and for precision work

STROP (final step):
  Leather loaded with chromium oxide or diamond paste
  Aligns wire edge; produces mirror polish
  5–10 strokes per session (more frequent than full sharpening)

SHARPNESS TEST:
  Arm-hair shaving test (blade should slice cleanly)
  Newspaper test (cuts without tearing)
  Thumbnail test (catches on nail without skating)
  The light reflection test (a sharp edge reflects no light)
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| Which plane for final surface? | No.4 smoother (narrow mouth, fine cut) |
| Which plane for fast stock removal? | No.5 jack with cambered (curved) iron |
| Which plane for flattening/jointing? | No.7 jointer (length bridges low spots) |
| Chip breaker distance for difficult grain? | 0.5mm or less from cutting edge |
| What angle to sharpen bench chisels? | 25° primary, 30° micro-bevel (secondary) |
| Western vs. Japanese saw for joinery? | Either works; Japanese (dozuki) has thinner kerf and cuts on pull |
| What is a mortise gauge? | Two adjustable pins set to mortise width simultaneously |
| Why use a marking knife instead of pencil? | Severs fibers for a precise line; registers chisel directly |
| What is a paring chisel used for? | Light, controlled trimming; push only, no mallet |
| How do you test sharpness? | Shaves arm hair cleanly; slices newspaper; no light reflection on edge |

## Common Confusion Points

**A properly set plane should produce whisper-thin shavings.** If you're getting thick, choppy shavings or the plane skips, it's almost always a sharpness issue or the chip breaker seating. Diagnose in this order: (1) Is the blade sharp? (2) Is the chip breaker gapped? (3) Is the mouth too wide?

**Japanese saws are not universally superior.** They cut on the pull stroke, which makes them easy to start and control. But they can't be resharpened (teeth are impulse-hardened), so eventually you replace the blade. Western saws can be resharpened by hand (or a saw filer). For high-volume production work where blades would be replaced anyway, Japanese saws win on cut quality.

**The back of a chisel is as important as the bevel.** The hollow ground back (ura) of Japanese chisels and the perfectly flat back of Western chisels serve the same purpose: a flat reference surface that creates a defined cutting edge when mated with the ground bevel. If the back is not flat, you cannot truly sharpen the edge.

**Freehand sharpening takes time to learn but pays off.** Honing guides produce consistent results immediately and are valid tools. Freehand sharpening is faster once learned and allows you to feel the edge change. Both approaches produce sharp tools; the guide is more reliable for beginners with thick, wide plane irons.

**Handle a No.4 before a No.5 for learning.** The jack plane (No.5) is often recommended first because it's versatile. But the smoother (No.4) gives faster feedback on surface quality and sharpness. Learning to surface-plane on the No.4 builds the feel needed for all plane work.
