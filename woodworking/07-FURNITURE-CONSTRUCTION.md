# Furniture Construction: Casework, Frame-and-Panel, Chairs, Tables

## The Big Picture

Furniture construction is applied structural engineering with wood movement as the governing constraint. Every construction method is fundamentally a strategy for accommodating wood movement: either by making the structure rigid enough to resist it (which eventually fails) or by designing for it explicitly (which lasts).

```
FURNITURE CONSTRUCTION SYSTEMS
================================

CASEWORK (box construction)
  +-----------------+
  | FACE FRAME      |  Solid wood face frame glued to
  |   STYLE         |  engineered core (plywood, MDF)
  |                 |  Movement: face frame moves; core stable
  +-----------------+
  | FRAMELESS       |  European 32mm system; all engineered
  | (European)      |  sheet goods; no solid wood large panels
  |                 |  Movement: minimal (ply/MDF stable)
  +-----------------+

FRAME AND PANEL (solid wood panels)
  +-----------+
  |  stile    |  Rails and stiles (solid wood frame)
  |   +-----+ |  Panel floats in groove (not glued)
  |   |panel| |  Panel moves freely with humidity
  |   |float| |  Frame stays square and stable
  |   +-----+ |
  |  stile    |
  +-----------+

SOLID WOOD PANEL (tabletop, bench)
  Panel glued up edge-to-edge
  Must be free to move cross-grain
  Attach to base with figure-8 clips, slot screws, or breadboard ends

BENT LAMINATION / STEAM BENDING
  Thin strips bent around form → glued
  Or: steamed solid wood bent to form → dried in position
  For curved components (chair backs, rockers)
```

## Casework: Face Frame vs. Frameless

```
FACE FRAME vs. FRAMELESS (32mm) CASEWORK
==========================================

FACE FRAME (American traditional):
  Core: plywood carcass (no face shown on front)
  Face: solid wood stiles and rails glued to carcass front
  +---------+
  |  PLYW.  |  ← plywood sides
  |  CARCS  |
  | [door]  |  ← doors/drawers overlay or inset in face frame
  |  FRAME  |  ← solid wood applied to front
  +---------+

  Advantages: hides plywood edge; traditional look; good for inset doors
  Disadvantages: narrower interior opening (face frame eats width)
  Door mounting: inset (flush) or overlay (on top of face frame)
  Joint: face frame members: pocket screws, biscuits, or mortise-tenon

FRAMELESS (European/32mm):
  Core: plywood or melamine-coated particleboard
  Front: NO face frame; cabinet edge exposed (banded or edge-tape)
  +---------+
  |         |  ← sides are finished face; edge tape on front edge
  |         |
  | [drawer]|  ← full-overlay doors/drawers cover entire front
  +---------+

  32mm system: hardware mounting holes on 32mm centers (standard)
               all European hardware (Blum, Hafele) based on this
  Advantages: wider interior access; fast assembly; European hardware
  Disadvantages: requires precise drilling; no exposed traditional joinery

CABINET JOINERY:
  Plywood carcass corners: dadoes + rabbet; or pocket screws
  Back panel: glued into rabbet (square and stiffen the carcass)
  Shelves: dado (fixed) or shelf pin holes (adjustable)
```

## Frame and Panel Construction

The solution to solid wood panel movement in doors, cabinet sides, and panels:

```
FRAME AND PANEL SYSTEM
========================

  +--stile--+------rail------+--stile--+
  |         |                |         |
  |         +----------------+         |
  |         |                |         |
  |         |    PANEL       |         |
  |         | (floating in   |         |
  |         |  groove; NOT   |         |
  |         |  glued)        |         |
  |         +----------------+         |
  |         |                |         |
  +--stile--+------rail------+--stile--+

KEY PRINCIPLE: The PANEL floats freely in the groove
  Panel can expand and contract with humidity changes
  Frame maintains its dimensions (joints resist movement)
  Panel held in groove but NOT glued (leave 1/8" gap at panel ends)

GROOVE / PANEL DIMENSIONS:
  Groove: 1/4" wide × 3/8" deep (typical); centered on edge
  Panel: thickness = groove width (fits without play)
  Panel width: subtract 2× groove depth from total opening,
               minus 1/4" per side (expansion clearance)

PANEL TYPES:
  Flat panel: plywood (stable; no movement allowance needed)
  Raised panel: beveled field; wider solid wood; elegant
                Router bit sets: stile-and-rail + raised panel bit
  Solid flat panel: thick; requires significant movement allowance

HAUNCHED TENON on frame corners:
  Rail tenon has a step (haunch) to fill the panel groove in stile
  Without haunch: open groove visible at joint
  +---+---+
  |   | H |  ← haunch fills groove
  |   +---+
  |   tenon|  ← tenon enters mortise
  +--------+
```

## Chair Joinery Engineering

Chairs are the most demanding furniture to engineer:

```
CHAIR STRESS ANALYSIS
======================

FORCES ON A CHAIR (150 lb person, dynamic use):
  Back leg at rear: 200+ lbs compression
  Front leg: bending moment from sitting + leaning back
  Back assembly: racking from leaning back
  Seat-to-rail joint: shear and tension from sitting + leaning
  Stretchers: prevent leg spreading

  Dynamic loads (rocking, tipping): 3–5× static load
  Chair joinery must be over-engineered vs. tables/cabinets

COMPOUND ANGLES:
  Chair legs often not vertical → dual taper and cant
  Splay: legs angle outward from vertical in side view
  Rake: legs angle outward in front view
  Combined splay + rake = compound angle at joint
  Setting: use bevel gauge; calculate compound angle
  Tables: compound angle = arctan(√(tan²(splay) + tan²(rake)))

SEAT-TO-RAIL JUNCTION (most stressed joint):
  Standard: mortise and tenon + glue
  Critical: rail meets leg at angle → angled tenon
  Back rail must handle rearward thrust from occupant leaning back
  Long tenons help; pinning helps

GLUE SELECTION:
  PVA (yellow glue): standard; adequate if joints fit well
  Polyurethane glue: fills gaps slightly; some movement tolerance
  Epoxy: best for end-grain joints; gap-filling; use for chair repairs
  Hot hide glue: historical; reversible (steam); self-clamping in warm state
                 Strongest of all for tight-fitting joints
                 Preferred for chair repair (can reactivate existing joint)

WINDSOR CHAIR GEOMETRY:
  Legs mortised into seat blank (from below)
  Spindles mortised into seat (from above) and into arm + top crest
  No rectangular joints — all round mortise and tenon
  Critical: leg splay angle consistent across all legs
```

## Table Construction

```
TABLE APRON SYSTEM
===================

  +---------APRON-----------+
  |                         |   Mortise-tenon or bridle
  LEG                     LEG   joint at corners
  |                         |
  |_________________________|   Apron spans between legs

TENON LAYOUT:
  Multiple aprons meeting at one leg: tenons must NOT overlap in mortises
  Solution: miter the inner ends of tenons in shared leg mortises
  Standard rule: tenon goes 2/3 of the way to leg center from each face
  +-------+
  | LEG   |
  |  +-+  |
  | /   \ |  ← tenons meet at 45° miter inside leg
  |/     \|
  |\     /|
  | \   / |
  |  +-+  |
  |       |
  +-------+

TABLETOP ATTACHMENT (wood movement accommodation):
  Problem: solid wood tabletop moves 1/4"–3/4" seasonally
  Frame is rigid: must let top slide relative to frame

  SOLUTIONS:
    Figure-8 clips: metal figure-8; screws into apron + slot in top
    Slot + screw: elongated slot in apron; screw into top (can slide)
    Z-clips (tabletop fasteners): engage slot in apron; screw into top
    Wooden clips: shop-made from off-cuts; engage dado in apron

  RULE: Never glue a solid wood tabletop to its apron.
        It WILL crack the apron or the top within 1–2 seasonal cycles.

BREADBOARD ENDS (optional):
  Cross-grain end cap on tabletop
  Keeps top flat; ends grain finished; traditional look
  Critical: breadboard must allow top to move
  Correct: center mortise glued; outer mortises elongated;
           pins through elongated slots (not glued)
  Incorrect: gluing breadboard full length = top will crack
```

## Drawer Construction

```
DRAWER CONSTRUCTION SYSTEMS
==============================

TRADITIONAL (half-blind dovetail):
  Front: hardwood (will be seen); half-blind dovetail to sides
  Sides: secondary wood (poplar, pine, cedar); thinner (1/2" typical)
  Back: secondary wood; dadoed into sides; thinner than front
  Bottom: plywood (stable); slides in groove; not glued
          Usually secured with one screw through bottom into back
  Joint: half-blind dovetail (front × side); through dovetail (back × side)

  +----------+--------+----------+
  | [FRONT]  |[SIDE]  |[BACK]    |
  | (half-bld|        |(through  |
  |  dovetail|        | dovetail)|
  +----------+--------+----------+
  |    bottom (floats in groove)  |
  +-------------------------------+

UTILITY (pocket screws or box joint):
  All secondary wood or plywood
  Box joints on all corners
  Bottom in dado; one screw through bottom into back
  Fast; perfectly functional

DRAWER SIZING:
  Drawer height: depends on opening; standard 3/4" clearance top and bottom
  Drawer depth: 1/2" less than opening depth (clearance at back)
  Side thickness: 1/2" typical solid wood; 5/8" plywood
  Bottom thickness: 1/4" plywood

DRAWER SLIDES (hardware):
  Wood-on-wood runners (traditional): bottom or side runners; no hardware
  Epoxy-waxed runners slide well; swell issues in humidity changes
  Side-mount ball-bearing slides (Blum, Accuride): standard modern
    Full extension (100%): pulls drawer completely out
    Soft-close: built-in dampener; premium feature
  Under-mount slides (Blum Tandem): invisible; smooth; premium cabinets
```

## Knock-Down Hardware

```
KNOCK-DOWN (KD) HARDWARE SYSTEMS
===================================

Used when: flat-pack shipping; installation in-place required;
           user-assembled furniture; commercial furniture production

BOLT + BARREL NUT:
  Machine screw passes through one member; threads into barrel nut
  in second member; tightened with Allen key
  Common: IKEA-style furniture; solid and reliable if tight
  Limitation: cannot be overtightened; strip risk

CAM LOCK (confirmat):
  Cam-shaped fitting turns to lock; used with dowel peg
  Fast to assemble; used by commercial flat-pack (IKEA)
  Less strong than bolt; designed for repeated assembly/disassembly

DOMINO LOOSE TENON + KD BOLTS:
  Loose tenons align pieces; steel bolts provide mechanical strength
  Higher quality flat-pack furniture

FURNITURE BOLTS (hanger bolt + T-nut):
  Hanger bolt: wood thread one end; machine thread other
  Install wood-thread end into leg; T-nut into apron; bolt through
  Creates knock-down joint for table leg; also used for chair legs
  Advantage: can tighten periodically as joint loosens with use

DESIGN RULE: KD hardware is a legitimate engineering solution,
  not a compromise. A well-designed KD joint is stronger and more
  serviceable than a glued joint that can't be disassembled.
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| Face frame vs. frameless — which is harder? | Face frame (requires fitting inset/overlay doors precisely) |
| What must never be glued in frame-and-panel? | The panel (must float to allow wood movement) |
| What is a haunched tenon? | Tenon with a step to fill panel groove; prevents open groove at corner |
| How do you attach a solid wood tabletop? | Figure-8 clips, Z-clips, or slotted screws — never glue cross-grain |
| What is the most stressed joint in a chair? | Rear leg to seat rail junction (rearward thrust from leaning) |
| What glue is best for chair repair? | Hot hide glue (reversible; strongest in tight-fitting joints) |
| What is the 32mm system? | European frameless cabinet with hardware holes on 32mm centers |
| What prevents tenons from overlapping in a shared leg? | Miter the inner tenon ends at 45° inside the leg |
| What is a traditional drawer bottom attachment? | Slides in groove (not glued); one screw through bottom into back |
| How do breadboard ends work correctly? | Center mortise glued; outer mortises elongated; pins in slots (not glued) |

## Common Confusion Points

**Plywood does not move like solid wood.** This is the primary reason modern casework uses plywood: it moves less than 0.1% with humidity changes vs. 3–10% for solid wood across the grain. Using solid wood for large case sides (without frame-and-panel) creates wood movement that will eventually crack the joints or the panels.

**Frameless cabinets are not inferior to face-frame.** In European woodworking (and most of the world), frameless construction is the professional standard. The quality difference between face-frame and frameless is in the execution, not the system. Frameless requires precise drilling; face-frame requires precise fitting. Both can produce excellent results.

**All chair joints should be cut slightly undersize and fitted.** Unlike table or cabinet joinery where tight-fitting joints can be glued in place, chair joints are often cut slightly loose to allow adjustment during assembly. You can always add filler or re-cut to tighten; you can't make a joint looser.

**Drawer bottoms must float.** A drawer bottom glued into its grooves will crack the drawer sides or front as the bottom expands/contracts. The standard practice — groove to hold the bottom, single screw through bottom into back — allows movement while keeping the bottom from rattling.

**KD hardware is engineering, not cheap.** Hanger bolts + T-nuts in table legs allow periodic re-tightening as joints loosen with use — something glued joints cannot offer. Many professional furniture lines use KD hardware specifically for serviceability, not because it's cheaper. The distinction between "cheap flat-pack KD" and "professional KD hardware" is enormous.
