# Power Tools: Table Saw, Band Saw, Router, Lathe — Safety and Use

## The Big Picture

Power tools are force multipliers for stock preparation and repetitive operations. The table saw is the center of the modern power shop — every other tool position relative to it. Safety in power tools is not optional and not separable from technique: most serious injuries result from technique errors, not equipment failure.

```
POWER TOOL WORKFLOW IN A TYPICAL PROJECT
==========================================

ROUGH LUMBER
     |
     v
JOINTER ────────────────── Face joint: flatten one face
     |                     Edge joint: flatten one edge at 90° to face
     v
THICKNESS PLANER ────────── Mill to uniform thickness (parallel faces)
     |
     v
TABLE SAW ────────────────── Rip to final width
     |                        Crosscut to length (with sled)
     |
     v

     |                        Cutting veneers; gentle cuts near joinery
     v

                               Template routing (pattern work)
                               Raised panels, mortises
     |

                               Forstner bits for flat-bottomed holes

LATHE (separate workflow) ──── Spindles, bowls, pens, chair legs
```

## Table Saw — The Primary Machine

The table saw is the most useful and most dangerous woodworking machine. Respect the geometry: the blade rotates toward the operator, and kickback throws workpieces back toward the operator at 100+ mph.

```
TABLE SAW ANATOMY
==================

        ←    fence    →
  ___________________________
  |  |           |  |
  |  |    table  |  |
  |  |    surface|  |     ← miter slots (left and right of blade)
  |  |___________|  |
  |    BLADE         |     ← blade protrudes through table top
  |    ↑ rotation    |     ← riving knife / splitter behind blade
  |__________________|
        arbor / motor

CRITICAL COMPONENTS:
  Blade:       Height and angle adjustable (arbor tilt)
  Fence:       Parallel to blade; sets rip width
  Riving knife: Curved plate behind blade; same thickness as blade kerf
               Prevents workpiece from closing on blade → prevents kickback
  Anti-kickback pawls: Fingers that let wood pass forward; grip if reversed
  Blade guard:  Clear plastic; should be used whenever possible
  Miter gauge:  Guides crosscuts (slides in miter slot)

BLADE HEIGHT:
  Rule: blade height = workpiece thickness + ~1/4"
  Minimum gullet clearance: blade tooth gullets must clear the wood at bottom
  Higher = more aggressive cut; lower = safer cut angle but more friction/heat
  For through-cuts: expose approximately 1/4" above workpiece
```

### Table Saw Safety — Non-Negotiable Rules

```
KICKBACK PREVENTION
====================

WHAT CAUSES KICKBACK:
  1. Workpiece pinches blade (wood movement, internal stress releasing)
  2. Workpiece tilts into blade after cut (small offcut caught by back of blade)
  3. Running workpiece into back of rotating blade
  4. Fence not parallel to blade (angled = wedges workpiece)

PREVENTION:
  Always use riving knife / splitter — keeps kerf open
  Always use anti-kickback pawls on rip cuts
  Stand to the side, not directly behind workpiece (kickback direction)
  Never reach over or behind the blade while cutting
  Never let small offcuts remain on table near blade — they get thrown
  Don't rip warped, twisted stock (uncontrolled contact with blade)
  Support large workpieces with roller stands or outfeed table

FENCE ALIGNMENT:
  Fence must be EXACTLY parallel to blade (or 0.005" toe-out max)
  Toe-in (rear of fence closer than front to blade) = kickback trigger
  Check with dial indicator against miter slot; adjust as needed

CROSSCUTTING:
  Use miter gauge or crosscut sled (never fence alone for crosscuts)
  Fence + miter gauge together = workpiece can be caught between fence + blade
  Exception: fence as stop block with gap between fence and blade
```

### Table Saw Operations

```
COMMON TABLE SAW CUTS
======================

RIP CUT (with grain):
  Set fence to desired width
  Keep workpiece against fence throughout
  Push past blade with push stick for final 6" (keep hands away)

CROSSCUT (against grain):
  Use crosscut sled (preferred) or miter gauge
  Sled: accurate, safe, repeatable; build one early
  Never hold with hands; use clamps or sled fence

DADO CUT (flat-bottomed groove):
  Install dado stack: outer blades + chippers + shims = width
  Typical: 1/4" to 13/16" width in 1/16" increments
  Use sacrificial fence face (wood attached to metal fence face)
  Depth: raise blade to desired depth

RABBET CUT (step at edge):
  Can use dado stack or two-pass method (rip fence + regular blade)
  Two-pass: cut face first (horizontal), then edge (vertical)
  Third option: rabbet plane (hand tool; clean results)

TAPER JIG:
  Sled with adjustable angle; for cutting tapered legs
  Or: hand-made jig (angled fence holder)
  Set offset = taper per foot × board length
```

## Band Saw

The band saw is safer than the table saw (cuts toward the table; no kickback) and essential for curves, resawing, and many joinery cuts.

```
BAND SAW CAPABILITIES AND LIMITS
===================================

STRENGTHS:
  Curves of any radius (limited by blade width)
  Resawing (ripping thick stock into thin slabs for book-matching)
  Cutting near joinery (close to glue-up; gentle action)
  Green wood (thin kerf; less waste of expensive material)
  Stack cutting (multiple identical curved parts)

LIMITS:
  Less accurate for long straight rip cuts (blade drift)
  Rougher surface than table saw (requires further preparation)
  Limited table tilt for compound angles (compared to miter saw)

BLADE SELECTION:
  Width:     Wider blade = straighter cuts; narrower blade = tighter curves
    1/4":    Very tight curves; 1" minimum radius
    3/8":    General curves; good balance
    1/2":    Gentle curves + resawing
    3/4":    Heavy resawing (hardwoods >6" thick)
  TPI (teeth per inch):
    6–10 TPI: Fast; rougher; green wood or thick stock
    10–14 TPI: General purpose hardwood
    14–18 TPI: Fine work; veneers; thin stock

DRIFT COMPENSATION:
  Band saw blades don't always cut exactly parallel to fence
  "Drift angle": the angle at which the blade naturally tracks
  Solution: find the drift angle by cutting freehand; mark angle on table;
            set fence (or fence guide block) to that angle
  Modern blades with less set have less drift; still check per blade change
```

## Router and Router Table

The router is essentially a high-speed motor with a collet — it takes interchangeable cutters and produces profiles, grooves, dadoes, and complex shapes.

```
ROUTER CONFIGURATIONS
======================

HANDHELD ROUTER:
  Base: fixed-base (depth set before plunge) or plunge (depth adjustable mid-cut)
  Use: edge profiles; template routing; stopped dadoes; inlay work
  Technique: move against the bit rotation (climb-cutting usually unsafe)
  Direction: router bit rotates clockwise → move router left to right on far edge;
             right to left on near edge (correct feeding direction)

ROUTER TABLE:
  Router mounted inverted below table; bit projects upward through table
  Fence guides workpiece; bit shapes profile
  Advantages: both hands free; more control; safer for narrow stock
  Limitations: router still requires significant depth and rigidity
  Use: raised panels; edge profiles on cabinet doors; cope-and-stick joinery

BITS:
+----------------+------------------+------------------+
| PROFILE BITS   | JOINERY BITS     | SPECIALTY        |
| Roundover      | Straight (dado)  | Spiral up-cut    |
| Chamfer        | Rabbeting        | Spiral down-cut  |
| Roman ogee     | Slot cutter      | Flush trim       |
| Cove           | Box joint        | Template pattern |
| Raised panel   | Cope and stick   | Dovetail         |
+----------------+------------------+------------------+

CARBIDE vs. HSS bits:
  Carbide: lasts longer; required for MDF; preferred for production
  HSS: cheaper; sharper edge; resharpenable; fine hardwood work

COLLET SIZES: 1/4" and 1/2" shank (use 1/2" where possible — more stable)
```

## Lathe

The lathe is a different machine class: work rotates; cutting tool is stationary. Two modes: spindle turning (between centers) and faceplate turning (bowl/plate work).

```
LATHE ANATOMY
==============

    headstock  |  toolrest  |  tailstock
    (drive end)|            | (live center end)
    +-------+  |  +------+  |  +------+
    | SPUR  |  |  | rest |  |  | LIVE |
    | DRIVE |  |  | post |  |  | CTR  |
    +-------+  |  +------+  |  +------+
       ↑motor                   adjustable

SPINDLE TURNING:
  Stock held between spur drive (headstock) + live center (tailstock)
  Used for: legs, spindles, handles, balusters
  Grain orientation: PARALLEL to lathe axis (long grain turning)
  Gouges used: spindle gouges, skew chisel

FACEPLATE TURNING (bowl / platter):
  Stock attached to faceplate with screws OR chuck on headstock
  Grain orientation: PERPENDICULAR to lathe axis (end grain at rim)
  Used for: bowls, platters, hollow forms
  Gouges used: bowl gouge (swept-back flute)

SPEED SELECTION:
  Formula: safe rim speed for most wood = 9,000 ÷ (diameter in inches) RPM
  Example: 12" bowl = 9,000/12 = 750 RPM maximum
  Large blanks: start SLOW (200–400 RPM); out-of-balance dangerous at speed
  Small work (pens): can run 2,000–3,000+ RPM
  Rule: if workpiece vibrates significantly at speed, SLOW DOWN
```

## Dust Collection — The Critical Safety System

```
FINE DUST HAZARD
=================

DUST PARTICLE SIZES:
  > 10 microns:  Visible dust; settles quickly; nose filters most
  2–10 microns:  Inhaled; trapped in airways; coughing removes
  < 2 microns:   FINE DUST — penetrates deep lung tissue
  < 1 micron:    ULTRA-FINE — reaches alveoli; stays permanently
  < 0.3 micron:  NANOPARTICLES — not filtered by most dust collectors

Hardwood dust: IARC Group 1 carcinogen (definitively causes nasal/sinonasal cancer)
Softwood dust: Group 2A (probably carcinogenic)
MDF dust:      Contains formaldehyde + fine silica; extremely hazardous
Finish mist:   Lacquer, varnish spraying — solvent + aerosol particles

DUST COLLECTION SYSTEM REQUIREMENTS:
  Single-stage collector: 650–1200 CFM at 4" hose for table saw / jointer
  Two-stage (cyclone + bag): better; dust settled in bin before reaching filter
  Minimum filter: 1 micron — standard filter bags don't catch fine dust
  Better: 0.5 micron bag filter
  Best: canister filter rated MERV 15+ (catches sub-micron particles)

RESPIRATORY PROTECTION:
  N95 minimum for occasional sanding
  For MDF, turning, production sanding: P100 half-face respirator
  (N95 does NOT filter fine wood dust adequately in sustained exposure)

COLLECTION POINTS:
  Table saw: under table + blade guard (if guard has port)
  Jointer: under table
  Planer: rear port (high volume; planer produces most volume)
  Router table: under table through fence port
  Sanding: random orbital sander → use with dust bag or vacuum hookup
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| What prevents kickback on table saw rip cuts? | Riving knife (keeps kerf open) + anti-kickback pawls + correct stance |
| What is the safest way to crosscut on table saw? | Crosscut sled (most accurate and safest) or miter gauge |
| Why should fence not be used alone for crosscuts? | Workpiece can be trapped between fence and blade |
| What blade width for resawing? | 1/2" to 3/4" with 6 TPI for hardwood |
| What is blade drift on band saw? | Blade's natural tracking angle; compensate by adjusting fence angle |
| Which router base for template work? | Plunge base (allows lowering into stopped grooves) or fixed-base with template |
| What is the lathe speed formula? | 9,000 ÷ diameter (inches) = maximum safe RPM |
| What dust particle size is most dangerous? | <2 microns (penetrates deep lung); <0.3 microns most insidious |
| What filtration is needed for fine hardwood dust? | 0.5 micron or better filter; P100 respirator for sustained work |
| Can you rip warped stock? | No — contact with blade is unpredictable; joint flat first |

## Common Confusion Points

**The riving knife is not the same as a splitter.** A riving knife curves with the blade and rises with blade height, always sitting behind the teeth at the right height. A fixed splitter stays at one height. The riving knife is superior because it maintains correct geometry at all blade heights. Modern saws have riving knives; older saws have splitters.

**"Don't stand directly behind the workpiece" is physics, not superstition.** Kickback launches a workpiece along the blade's plane, directly toward whatever is behind the workpiece. You don't have to touch the blade to be injured — the workpiece itself is a projectile. Operators should stand to the left or right of the cut line.

**Band saw drift compensation is per-blade.** Install a new blade and re-check drift. Even blades from the same package can drift at different angles. This is why the "fence parallel to miter slot" setup is often wrong for band saws — the fence must be set to the blade's actual drift angle, not theoretical parallel.

**Router bit feed direction is critical for safety and quality.** Always feed against the bit rotation (conventional milling). Climb cutting (feeding with the rotation) produces a smoother surface but risks the router pulling itself forward suddenly ("climbing") — potentially extremely dangerous with a handheld router. On a router table with a fence and featherboards, limited climb cutting is sometimes used for final passes, but requires technique and awareness.

**Lathe work requires a face shield, not just safety glasses.** A bowl blank can come off the lathe with lethal force. A chip from a catch can fly at high speed. Safety glasses are insufficient — a full face shield is required for turning. This is one of the few power tool scenarios where safety glasses are genuinely inadequate.
