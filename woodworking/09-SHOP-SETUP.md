# Shop Setup: Workbench, Tool Storage, Dust Collection, Electrical

## The Big Picture

The shop is a system: machine positions determine workflow; electrical infrastructure determines what machines can run; dust collection determines health risk; and the workbench is the primary work surface for everything done by hand. Getting the infrastructure right before buying more machines pays dividends for years.

```

  [LUMBER STORAGE]

       v
  [BAND SAW] ──────────── curves; resawing

  FINISHING AREA ──────── separate from dust; ventilated
  ASSEMBLY TABLE ──────── flat surface; glue-up

SPACE REQUIREMENTS:
  8' × 8' minimum workspace around each major machine
  Table saw: biggest machine; position first; let others fill around it
  Bench: needs light from side (not behind); space to walk around 3 sides
```

## The Workbench — Design Philosophy

The workbench is not furniture — it is a precision vise and assembly table. Two dominant traditions:

```
WORKBENCH TRADITIONS
=====================

ROUBO (French, 18th century — André Roubo, 1769):
  +-------------------------------------------------+
  |   THICK TOP (3–4" solid wood)                   |
  +-------------------------------------------------+
  | LEG    LEG VISE                       TAIL VISE |
  |        (on left face,                           |
  |         front leg)                              |
  |                                                 |
  |        STRETCHER                                |
  +-------------------------------------------------+
  Legs: massive; 4–5" thick; direct to floor
  Leg vise: large wooden vise on left front leg
  Tail vise: sliding vise at right end; extends top face
  Holdfast holes: holes through top; holdfasts (iron hooks) secure work
  Construction: traditional mortise-and-tenon; no metal hardware

SCANDINAVIAN (German/Scandinavian tradition):
  +--------------------------------------------+
  |       TOP (2–3" thick)                     |
  |  +-------+                       +------+  |
  |  |FACE   |                       | TAIL |  |
  |  | VISE  |                       | VISE |  |
  |  |       |                       +------+  |
  |  +-------+                                 |
  |  ===STRETCHER============================= |
  +--------------------------------------------+
  Face vise: quick-release vise on left front
  More open below (storage access)
  Easier to build than Roubo; widely used
  Vise: quick-release metal face vise (Record, Veritas, Lake Erie)

COMPARISON:
  Roubo:       Massive; more clamping options; harder to build
  Scandinavian: Simpler; metal vises; more common in modern shops
  Both work well; both are significant investments (50–100 hours to build)
```

## Workbench Top Specifications

```
TOP CONSTRUCTION REQUIREMENTS
================================

THICKNESS:
  Minimum: 3" for adequate mass and rigidity
  Optimal: 3.5–4" (Roubo-style) — dampens tool vibration; holds holdfasts
  Thin tops flex under mallet use; can't hold holdfasts effectively

MATERIAL:
  Traditional: laminated hardwood (beech, maple, ash, sycamore)
  Beech: hardest, best wear resistance; traditional German/European
  Hard maple: excellent; widely available in North America
  Douglas fir: much softer; cheaper; works but dents easily
  MDF core with hardwood face: acceptable; very flat; no holdfast holes

FLATNESS:
  Top must be flat: check with winding sticks (like any large panel)
  New laminated tops: often slightly cupped or bowed; hand-plane flat
  Periodic maintenance: flatten as needed (major task once; minor after)

DOG HOLES:
  3/4" or 20mm holes through the top (or in the apron/face)
  Pattern: single row along front edge; spacing every 4–8"
  Use: bench dogs (wooden or metal stops); hold work for planing
  Tail vise dogs + top dogs: sandwich work; secure for planing

HEIGHT:
  Optimal: wrist height when arms hang straight down
  Typical: 33–36" for most adults
  Too high: can't apply downward force efficiently
  Too low: back strain from bending over work
  Verify: stand with arms hanging; measure wrist to floor
```

## Vise Types and Use

```
VISE SELECTION
==============

LEG VISE (wooden, traditional):
  Mounted on front face of left front leg
  Chop: large wooden jaw (often figured wood)
  Screw: wooden threaded rod (or modern metal parallel screw)
  Parallel guide: keeps jaw parallel to leg as it opens
  Strength: enormous clamping force; can hold very large, heavy work
  Use: holding boards vertical for edge jointing; ripping with hand saw

QUICK-RELEASE FACE VISE (metal):
  Mounted left front face (below top or at top level)
  Parallel guide: metal; maintains jaw parallel
  Quick release: press lever; jaw opens rapidly; re-engage; clamp
  Brands: Veritas, Record 52E, Lake Erie Toolworks
  Use: general clamping; edge jointing; light face work

TAIL VISE (wagon vise):
  Extends the right end of the top (slides horizontally)
  Has dog hole: works with top dog holes to clamp boards face-side-up
  Use: holding wide panels for face planing; secondary clamping
  Alternative: Moxon vise (surface-mounted; portable; effective)

HOLDFASTS:
  Iron or steel hook; dropped through hole; sharp hammer blow locks
  One holdfast + one hand = work held flat; hands free
  Must-have for Roubo-style bench
  Modern: Gramercy holdfasts (well-reviewed for 3/4" holes)
  Historical: cast iron (heavier; good in thick tops)
  Need: 3.5"+ top thickness for reliable holding
```

## Tool Storage

```
TOOL STORAGE PRINCIPLES
========================

EDGE TOOLS (chisels, plane irons):
  Priority: PROTECT THE CUTTING EDGE
  Never pile chisels in a drawer (edges contact each other → dull)
  Chisel roll: individual leather pockets; each chisel separate
  Wall rack with individual slots: tip exposed but edge-protected
  Drawer with foam inserts: each tool in its own slot; protects edge

BENCH PLANES:
  Store with blade retracted OR blade-down on a soft surface (non-scratching)
  Never store sole-down on abrasive or gritty surfaces (scratches sole)
  Hanging on wall hooks (through front knob): keeps them visible and accessible
  In plane till (large drawer below bench): lid protects soles

RUST PREVENTION:
  Any bare metal surface will rust in humid shop (especially seasonal: shop heats up in spring)
  Wipe plane soles and saw blades with light oil or paste wax
  Camellia oil: traditional Japanese; excellent rust inhibitor; light
  WD-40: short-term only; evaporates; not a good long-term inhibitor
  Silicone: never in a wood shop (contaminates wood; finish adhesion fails)
  Low-humidity storage: dehumidifier or desiccant in tool chest

SAW STORAGE:
  Western saws: hang from handle holes; teeth down OR in canvas roll
  Japanese saws: hang from hole at handle or in sleeve
  Never stack saws (tooth-to-tooth damage)
```

## Dust Collection Design

```
DUST COLLECTION SYSTEM DESIGN
================================

SYSTEM TYPES:
  Shop vacuum (1.5–2 HP): adequate for small machines (router, sander)
  Dust collector (1–3 HP): adequate for medium machines (band saw, jointer)
  Cyclone + collector (3–5 HP): high-efficiency; separates fine dust before filter

REQUIREMENTS BY MACHINE:
  Machine         CFM needed    Duct size    Notes
  ----------      ----------    ---------    -----
  Table saw       350–500 CFM   4" main      Under-table box + blade guard port
  Jointer         300–500 CFM   4"           Under-table; chip direction matters
  Planer          500–700 CFM   4–6"         Highest volume machine; priority
  Band saw        150–250 CFM   4"           Less chips; finer dust
  Router table    200–350 CFM   4"           Under-table port in fence
  Sander (belt)   300–400 CFM   4"           Very fine dust; good filter critical

MDF RULE: MDF dust is the most hazardous material in a wood shop.
  Formaldehyde + very fine silica = lung and carcinogen hazard
  P100 respirator + MERV 16 filter minimum for any MDF work

CYCLONE SEPARATOR:
  Primary separator before filter (removes 99%+ of chips)
  Fine dust still reaches filter but in much smaller quantity
  Extends filter life dramatically
  Thien baffle (homemade): effective budget alternative to Oneida cyclone

FILTER QUALITY:
  Cloth bag: passes fine dust; not adequate for hardwood dust
  Pleated filter cartridge (1 micron): standard minimum
  0.5 micron canister: better
  MERV 16+: required for protection from the most hazardous particles

BLAST GATES:
  Install at each machine connection point
  Manual or automatic (automatic open when machine runs)
  Keep closed except for active machine
  Reason: system CFM distributed equally across all open ports
          → lower velocity in each branch → insufficient pickup
          → blast gates route all CFM to one machine at a time
```

## Wood Storage

```
WOOD STORAGE REQUIREMENTS
===========================

PRINCIPLE: Allow air circulation around all surfaces to equalize MC

STICKERING:
  Stickers = thin uniform spacers between boards (typically 3/4" × 3/4")
  Place every 16–24" across the length of boards
  Keep stickers aligned vertically (prevent localized sag)
  Allow 1" minimum between boards side-by-side (airflow)

  +-------------+
  | BOARD (top) |
  +-------------+
       sticker (3/4" × 3/4")
  +-------------+
  | BOARD (mid) |
  +-------------+
       sticker
  +-------------+
  | BOARD (bot) |
  +-------------+
  | FLOOR       |   (4" minimum above floor — airflow; rodents)

LOCATION:
  Indoors: acclimate to shop humidity (2–4 weeks for most thicknesses)
  Avoid: unheated garage (humidity swings → checking and warping)
  Avoid: basement (high humidity; mold risk)
  Good: attached heated/conditioned shop space

ORIENTATION:
  Flat stack: standard; stickers maintain gaps
  Vertical lean: not recommended for long boards (causes curve to set)

END SEALING:
  Rough-sawn lumber: end coat exposed end grain immediately after sawing
  Anchorseal or latex paint: prevents end checking from rapid moisture loss
  At board ends (most rapid drying point)
```

## Electrical Requirements

```
SHOP ELECTRICAL PLANNING
==========================

MACHINE REQUIREMENTS:
  Machine               Voltage     Circuit     Notes
  -------------------   -------     -------     -----
  Table saw (10")        120/240V    20A 120V    Can run 120V; 240V preferred
                         240V        15A 240V    for large motors
  Cabinet table saw (3HP) 240V       20A 240V    Required; 240V only
  Jointer (6")           120/240V    20A 120V    or 240V
  Jointer (8"+)          240V        20A 240V    Required
  Thickness planer (15") 120/240V    20A 120V    or 240V
  Band saw (14")         120V        15–20A      Standard
  Band saw (17"+)        240V        15–20A      Required
  Lathe (variable speed) 120/240V    20A         Depends on model
  Dust collector (2 HP)  120V        20A         Dedicated circuit
  Dust collector (3 HP)  240V        20A 240V    240V preferred
  Air compressor (5 HP)  240V        30A 240V    Required
  Spray booth exhaust    120V        20A         Separate motor

CIRCUIT PLANNING:
  Each major machine gets its own dedicated circuit
  Total load calculation: sum of nameplate amperage + 25% margin
  220V = 2× the power of 120V on same wire gauge
  #12 AWG wire: 20A circuit; minimum for most shop circuits
  #10 AWG wire: 30A circuit; compressor, 3HP+

LIGHTING:
  Minimum: 50 foot-candles at work surfaces
  Better: 100 foot-candles (task lighting at bench + machines)
  LED shop lights: energy efficient; color temperature 4000–5000K (clear, bright)
  Natural light: ideal; north-facing windows (consistent, no direct sun glare)
  South windows: direct sun glare interferes with fine surface inspection
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| What is the optimal bench height? | Wrist height when arms hang straight down (typically 33–36") |
| Roubo vs. Scandinavian bench — key difference? | Roubo: massive leg vise + holdfast holes; Scandinavian: metal face vise, simpler build |
| What is a holdfast? | Iron hook dropped through bench hole; locked with hammer blow; one-handed work holding |
| Minimum bench top thickness? | 3" (3.5–4" for holdfasts to work reliably) |
| How should chisels be stored? | Individually in roll, rack, or foam-lined drawer — edges protected |
| What must never be used in a wood shop? | Silicone-based products (contaminate wood; finish adhesion failure) |
| What dust filter rating is minimum? | 1 micron pleated canister; 0.5 micron or MERV 16 for fine hardwood/MDF |
| What are blast gates for? | Direct full CFM to one machine at a time instead of diluting across all ports |
| What is stickering? | Uniform spacers between stored boards for airflow; prevents moisture gradient warping |
| What circuit for a 3HP table saw? | 240V, 20A dedicated circuit |

## Common Confusion Points

**Dust collectors with cloth bags do not adequately filter fine dust.** The bags on most standard dust collectors pass particles below 5–10 microns. The most hazardous particles (below 2 microns, the carcinogens) pass straight through. You can have running dust collection and still inhale all the hazardous material. A pleated canister filter or a MERV-rated cartridge is required to capture fine particles.

**Shop layout is a workflow problem before it's a space problem.** Many small shops have enough space but terrible workflow because machines are placed without regard to material movement. Think through the entire sequence from rough lumber to finished part; place machines so each operation's output is positioned at the input of the next. A shop with good layout feels spacious; a shop with poor layout feels cramped regardless of size.

**The workbench height rule is personal, not universal.** The "33–36 inches" range is typical but the correct height depends on your height, the work you do, and whether you use a plane or chisel primarily. Plane work benefits from slightly lower bench (more body weight over the work); chisel and marking work benefits from slightly higher bench (less bending). Build adjustable height or accept that you'll modify it after using it.

**240V does not mean 240V outlets everywhere.** Running a 240V circuit requires a 2-pole breaker in the panel; a 4-wire run (#10 or #12 gauge); and the correct outlet (NEMA 6-20 or 6-30 depending on amperage). This requires an electrician if you're not licensed, or careful personal installation. The payoff: 240V machines run cooler, more efficiently, and are capable of higher sustained loads.

**Acclimation time is species- and thickness-dependent.** A 4/4 (1") board needs 2–4 weeks in the shop to acclimate. An 8/4 (2") board takes 6–8 weeks minimum. For really thick stock (4/4+ slabs), a moisture meter is mandatory — you cannot judge MC by time in the shop alone. The consequences of working wood that isn't at equilibrium MC are usually cracks or warped panels within the first year of use.
