# 04 — Avionics

## Navigation, FBW, Autopilot, Sensor Fusion, Data Buses

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Avionics architecture: federated vs Integrated Modular Avionics (IMA), DO-178C
- Navigation fundamentals: latitude/longitude/altitude, coordinate transforms
- GPS: signal structure, pseudorange, DGPS, SBAS (WAAS/EGNOS), RAIM integrity monitoring
- Inertial Navigation System (INS): IMU (gyros + accelerometers), strapdown integration
- GPS/INS fusion: Kalman filter architecture, error states, complementary filter
- Air data system: pitot-static, ADC, airspeed/altitude/Mach/vertical speed
- Autopilot: control law architecture (inner/outer loops), auto-throttle, VNAV/LNAV
- Flight Management System (FMS): route programming, RNAV/RNP, LNAV/VNAV coupling
- Fly-by-wire control laws: normal law, alternate law, direct law, envelope protections
- Redundancy management: triplex/quadruplex voting, dissimilar hardware
- Aircraft data buses: ARINC 429, ARINC 664 (AFDX), MIL-STD-1553
- Cockpit displays: EFIS, PFD, ND, ECAM/EICAS, heads-up display (HUD)
- Traffic Collision Avoidance System (TCAS II): resolution advisories, ACAS-X
- ILS/GLS: localizer, glideslope, decision height, CAT I/II/III approaches
- ADS-B: surveillance, Out/In, 1090ES encoding, NextGen/SESAR transition
- Cybersecurity in avionics: attack surface (FMS, EFB, cabin networks), DO-326A
