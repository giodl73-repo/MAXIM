# Rings and Ideals

## The Big Picture

```
+====================================================================+
|              THE RING HIERARCHY                                     |
+====================================================================+
|                                                                    |
|  RING (R, +, ·): abelian group under +, monoid under ·, distrib.  |
|    ↓ commutativity of ·                                            |
|  COMMUTATIVE RING                                                  |
|    ↓ no zero divisors (ab=0 → a=0 or b=0)                         |
|  INTEGRAL DOMAIN                                                   |
|    ↓ every nonzero element = unit (invertible)                     |
|  FIELD (Q, R, C, F_p)                                             |
|                                                                    |
|  OR instead:                                                       |
|  INTEGRAL DOMAIN                                                   |
|    ↓ unique factorization into irreducibles                        |
|  UFD (unique factorization domain)                                 |
|    ↓ every ideal is principal (generated by one element)           |
|  PID (principal ideal domain)                                      |
|    ↓ Euclidean algorithm exists                                    |
|  EUCLIDEAN DOMAIN                                                  |
|    ↓ every nonzero element is a unit                               |
|  FIELD                                                             |
|                                                                    |
|  Examples: Field → ED → PID → UFD → domain → ring                 |
|  Z: ED (PID, UFD) but NOT a field.                                |
|  Z[x]: UFD but NOT a PID (ideal (2,x) is not principal).          |
|  Z[√-5]: domain, NOT UFD (6 = 2·3 = (1+√-5)(1-√-5)).             |
+====================================================================+
```

## Ring Axioms

```
A RING (R, +, ·) satisfies:
  (R, +): abelian group (with identity 0)
  (R, ·): monoid (with identity 1; associative; often commutative)
  Distributivity: a(b+c) = ab+ac  and  (a+b)c = ac+bc.

Standard consequences:
  0·a = 0 for all a (proof: 0·a = (0+0)·a = 0·a + 0·a → 0·a = 0)
  (-1)·a = -a (proof: a + (-1)a = (1+(-1))a = 0·a = 0)
  (-a)b = -(ab) = a(-b)

NOTE on "1": Some authors don't require a multiplicative identity.
  A ring without 1 is a "rng" (ring without identity).
  We always include 1 here.

Commutative ring: ab = ba for all a,b.
  Almost all rings in algebraic number theory are commutative.
  Matrix rings are the main non-commutative example.
```

## Examples of Rings

```
COMMUTATIVE RINGS:
  Z — the prototype ring (integers).
  Z/nZ — integers mod n.
  Q, R, C — fields (every nonzero element invertible).
  F_p = Z/pZ — finite field.
  Z[x] — polynomials over integers.
  Q[x], R[x], C[x] — polynomial rings over fields.
  Z[i] = {a+bi : a,b ∈ Z} — Gaussian integers.
  Z[√-5] — does not have unique factorization.
  Z[ζ_n] — cyclotomic integers.

NON-COMMUTATIVE RINGS:
  M_n(F) — n×n matrices over field F.
  H = {a+bi+cj+dk : a,b,c,d ∈ R} — quaternions.
  Group ring R[G] = {Σ rₐ·a : rₐ ∈ R, a ∈ G, finite sum} with convolution multiplication.
  Endomorphism ring End_R(M) of an R-module M.

UNITS (invertible elements):
  Z*: {±1}.
  Z/nZ*: {k : gcd(k,n)=1}, order φ(n).
  F_p*: {1,2,...,p-1}, order p-1.
  Z[i]*: {1,-1,i,-i} = {units of norm 1}.
  M_n(F)*: GL(n,F).

ZERO DIVISORS:
  a ≠ 0 is a zero divisor if ∃b ≠ 0: ab = 0.
  Z/6Z: 2·3 = 6 ≡ 0. Both 2 and 3 are zero divisors.
  Z/pZ: no zero divisors (field).
  M_n(F): if n≥2, zero divisors exist (e.g., rank-deficient matrices).
```

## Ideals

```
DEFINITION: A (two-sided) ideal I ⊆ R satisfies:
  I is an additive subgroup of R.
  For all r ∈ R, a ∈ I: ra ∈ I and ar ∈ I.  (absorbing condition)

Left ideal: only ra ∈ I required.
Right ideal: only ar ∈ I required.
For commutative rings: all three notions coincide.

SMALLEST IDEAL CONTAINING S:
  (S) = ⟨S⟩ = intersection of all ideals containing S
       = {Σ rᵢ aᵢ : rᵢ ∈ R, aᵢ ∈ S, finite sum}

PRINCIPAL IDEAL: (a) = aR = {ra : r ∈ R}.
  In Z: (n) = nZ = {multiples of n}.
  In F[x]: (f) = {multiples of f}.

EXAMPLES:
  In Z: ideals are exactly nZ = (n) for n ≥ 0. Z is a PID.
  In Z[x]: ideals (2) and (x) are principal; (2,x) is NOT principal.
  In a field F: only ideals are (0) and F=(1). Consistent with: fields have no
    nontrivial ideals (which is equivalent to being a field for commutative rings).
```

### Types of Ideals

```
PRIME IDEAL: P ⊆ R such that:
  ab ∈ P → a ∈ P or b ∈ P  (and P ≠ R).
  Equivalent: R/P is an integral domain.

MAXIMAL IDEAL: M ⊆ R such that no ideal properly between M and R.
  Equivalent: R/M is a field.
  Every maximal ideal is prime.
  In Z: prime ideals = (p) for primes p, plus (0).
        maximal ideals = (p) for primes p.

RADICAL IDEAL: I = rad(I) = {a : aⁿ ∈ I for some n}.
  rad((n)) in Z: if n = p₁^{a₁}...pₖ^{aₖ}, rad((n)) = (p₁p₂...pₖ).

NILRADICAL of R: nil(R) = rad((0)) = {a : aⁿ=0 for some n}.
  nil(Z/12Z) = {0,6} since 6² = 36 ≡ 0 mod 12.
  Reduced ring: nil(R) = (0) — no nilpotent elements.
```

## Quotient Rings

```
For ideal I ◁ R:
  R/I = {a+I : a ∈ R} — cosets of I in (R,+).
  Addition: (a+I) + (b+I) = (a+b)+I.
  Multiplication: (a+I)(b+I) = ab+I.
  (Well-defined because I is an ideal: if a+I = a'+I (a-a' ∈ I) and b+I=b'+I,
   then ab-a'b' = ab-ab' + ab'-a'b' = a(b-b') + (a-a')b' ∈ I.)

QUOTIENT RING EXAMPLES:
  Z/nZ: the standard clock arithmetic.
  Z[x]/(x²+1) ≅ C (adjoining a root of x²+1=−1 → complex numbers).
  Z[x]/(2, x) ≅ F_2 (forcing 2=0 and x=0 gives just {0,1}).
  R[x]/(x²+1) ≅ C (over R, adjoin i).
  F_p[x]/(f) = GF(p^n) when f is irreducible of degree n.

THE RING ISOMORPHISM THEOREM:
  If φ: R → S is a ring homomorphism:
  R/ker(φ) ≅ im(φ).  [Same structure as group version]

CHARACTERIZATION:
  R/I is a domain iff I is prime.
  R/I is a field iff I is maximal.
  (For commutative Noetherian rings, every prime ideal is maximal iff every
   prime ideal is also maximal — this holds in PIDs and polynomial rings over fields.)
```

## The Domain Hierarchy in Detail

### Integral Domain

```
No zero divisors: ab = 0 → a = 0 or b = 0.
Equivalent: cancellation law holds: ac = bc, c≠0 → a = b.

EXAMPLES: Z, Z[x], Z[i], Z[√d] (for appropriate d), any field.
NONEXAMPLE: Z/6Z (has zero divisors 2 and 3).

FRACTION FIELD: Every integral domain embeds in its fraction field.
  Frac(Z) = Q, Frac(Z[x]) = Q(x), Frac(Z[i]) = Q(i) = Q(√-1).
```

### UFD (Unique Factorization Domain)

```
Every nonzero non-unit a ∈ R factors uniquely (up to units and order) as:
  a = p₁ · p₂ · ... · pₖ  (each pᵢ is irreducible)

In a UFD: irreducible ⟺ prime.
  (Recall: p prime means p|ab → p|a or p|b. This is stronger than irreducible
   in general domains but equivalent in UFDs.)

EXAMPLES: Z, Z[x], F[x₁,...,xₙ] (polynomial rings over fields).
NONEXAMPLE: Z[√-5] — 6 = 2·3 = (1+√-5)(1-√-5), two distinct factorizations.

GAUSS'S LEMMA: If R is a UFD, then R[x] is a UFD.
  Implies: Z[x], Z[x,y], Z[x,y,z], ... are all UFDs.
```

### PID (Principal Ideal Domain)

```
Every ideal is principal: I = (a) for some a.
PIDs are UFDs (the "easy direction": gcd of two elements exists, use it).
  Proof sketch: in a PID, every ascending chain of ideals stabilizes (Noetherian),
  and every irreducible is prime. These together give unique factorization.

EXAMPLES: Z, Z[i], Z[ω] (ω = e^{2πi/3}), F[x] for field F.
  Class number = 1 ↔ O_K is PID for number fields (see number-theory/).

NONEXAMPLE: Z[x] — ideal (2, x) is not principal.
  Proof: Suppose (2,x) = (f). Then f|2 and f|x in Z[x].
  f|2 → f = ±1 or ±2. f|x → deg(f) ≤ 1.
  f = 2 would give (f) = (2) ≠ (2,x) (since x ∉ (2)). f = 1 gives (f) = Z[x] ≠ (2,x).

  KEY STRUCTURE: PIDs are "better" than UFDs but short of fields.
  PID → UFD, but UFD → PID fails (Z[x] is UFD but not PID).
```

### Euclidean Domain

```
Has a "size function" d: R\{0} → N such that:
  For any a, b with b ≠ 0: ∃q,r with a = qb+r and r=0 or d(r) < d(b).

EXAMPLES:
  Z with d(n) = |n|.
  F[x] with d(f) = deg(f).
  Z[i] with d(a+bi) = a²+b² (the norm — division algorithm exists).
  Z[ω] with ω=e^{2πi/3}, d(a+bω) = a²-ab+b² (the Eisenstein integers).

Euclidean → PID:
  Proof: Let I be any nonzero ideal. Let b ∈ I be a nonzero element with minimal d(b).
  For any a ∈ I: a = qb + r. Since r = a - qb ∈ I, and d(b) minimal: r = 0.
  So a = qb → a ∈ (b). Thus I = (b) is principal.  □

IMPORTANT: Not all PIDs are Euclidean.
  Z[(1+√-19)/2]: a PID but NOT Euclidean (no Euclidean algorithm works).
  But this is rare — most natural PIDs are Euclidean.
```

## Polynomial Rings

```
R[x] = {a_n x^n + ... + a₁x + a₀ : aᵢ ∈ R, n ≥ 0}.

If R is commutative: R[x] is commutative.
If R is a domain: R[x] is a domain (degree(fg) = degree(f)+degree(g)).
If R is a field: R[x] is a Euclidean domain (with d(f) = deg(f)).
  → R[x] is PID → R[x] is UFD.

IRREDUCIBILITY OVER Q:
  Eisenstein's criterion: if p is prime and p|a₀,...,aₙ₋₁, p∤aₙ, p²∤a₀:
    f = aₙxⁿ + ... + a₀ is irreducible over Q.
  Example: x² + 2x + 2 is irreducible over Q? Discriminant = 4-8 = -4 < 0 → no real roots → irreducible over R, hence over Q.
  x⁵ - 4x + 2: Eisenstein with p=2 (but need to check leading coeff). a₅=1, p∤1, so Eisenstein doesn't apply directly. But by rational root theorem, no rational roots, and more analysis needed for degree 5.

DIVISION ALGORITHM in F[x]:
  For f,g ∈ F[x], g ≠ 0: ∃! q,r with f = qg + r and deg(r) < deg(g).
  GCD algorithm in F[x]: Euclidean. gcd(f,g) = last nonzero remainder.
```

## Localization

```
LOCALIZATION at multiplicative set S (0 ∉ S, S closed under ·):
  S^{-1}R = {a/s : a ∈ R, s ∈ S} / equivalence.
  a/s ~ a'/s' iff ∃t ∈ S: t(as' - a's) = 0.

EXAMPLES:
  S = R\{0}: gives the fraction field Frac(R).
  S = Z\{0}: gives Q from Z.
  S = {1, p, p², ...}: inverting one prime.
  S = R\P (P prime ideal): gives local ring R_P at P.
    R_P has unique maximal ideal PR_P — "looking at R locally near P."

WHY LOCALIZATION MATTERS:
  Local rings are simpler — one maximal ideal instead of many.
  Algebraic geometry: localize at a prime ideal = study the variety locally
  at the corresponding point.
  Algebraic number theory: localize at a prime ideal p in O_K to study
  ramification at p.
```

## Decision Cheat Sheet

| Task | Structure needed |
|------|-----------------|
| Guaranteed unique factorization | UFD (e.g., Z, F[x]) |
| Every ideal is principal | PID (e.g., Z, Z[i], F[x]) |
| Division algorithm exists | Euclidean domain (e.g., Z, F[x], Z[i]) |
| Build field from ring | Quotient by maximal ideal |
| Build domain from ring | Quotient by prime ideal |
| Adjoin a root of f(x) | R[x]/(f) |
| Understand R/P (P prime) | Integral domain (no zero divisors) |
| Understand R/M (M maximal) | Field (every nonzero element invertible) |
| Invert elements of R | Localization S^{-1}R |

## Common Confusion Points

**"Z[x] is a PID."**
It is NOT a PID. The ideal (2, x) is not principal. Z[x] IS a UFD.
The theorem is: if R is a UFD then R[x] is a UFD. The PID property is NOT preserved.

**"Every UFD is a PID."**
The reverse of what's true. PID → UFD (proved via noetherian + prime=irreducible).
UFD does not imply PID. Z[x] is UFD, not PID.

**"Prime ideal and prime element are the same."**
In a PID (e.g., Z): the principal ideal generated by a prime element is a prime ideal.
In a general domain: prime elements generate prime ideals, but a prime ideal need not be
principal or generated by a single prime. In Z[√-5], the ideal (2, 1+√-5) is prime
(not principal), and 2 is irreducible but NOT a prime element (Euclid's lemma fails for 2).

**"Maximal ⟹ prime, always."**
In commutative rings: yes, maximal ideals are prime. The quotient by a maximal ideal
is a field, which is a domain, which is the characterization of prime.
In non-commutative rings: the analogy breaks down.
