# Economic Geology and Ore Deposits

## The Big Picture

```
+------------------------------------------------------------------+
|               ORE DEPOSIT CLASSIFICATION SYSTEM                  |
|                                                                  |
|  MAGMATIC          HYDROTHERMAL        SEDIMENTARY              |
|  (igneous process) (hot fluid)         (sedimentary basin)       |
|                                                                  |
|  Chromite           Porphyry Cu-Mo      BIF (iron)              |
|  Ni-Cu-PGE          Epithermal Au-Ag    SEDEX (Pb-Zn)           |
|  Kimberlite (diamond)VMS (Cu-Zn-Pb)    MVT (Pb-Zn)             |

|                                                                  |
|  SUPERGENE          PLACER              RESIDUAL                 |
|  (weathering)       (erosion/transport) (in-situ weathering)    |
|                                                                  |
|  Secondary enrichment Beach/river sands Laterite (Ni, Al)       |
|  (Cu, Ag, Au)       (Ti, Sn, Au, REE)  Bauxite (Al)            |
+------------------------------------------------------------------+
```

An ore deposit is a concentration of useful mineral(s) that can be extracted economically. The economics matter as much as the geology — changing metal prices, mining technology, or processing capability converts waste into ore and vice versa.

## Magmatic Ore Deposits

### Chromite Deposits

```
LAYERED MAFIC INTRUSIONS (LMI)
  Earth's largest chromite, platinum-group element, and Ti-V deposits

Mechanism:
  Large volumes of mafic magma (basaltic) intrude → cool slowly
  → Crystal settling (gravitational) or in-situ crystallization
  → Dense early-crystallizing minerals form layers

  Dunite layer → chromite layer → norite/gabbro → anorthosite

KEY DEPOSITS:
  Bushveld Complex, South Africa (2.06 Ga):
    World's largest LMI; 100,000 km² exposed
    Merensky Reef: PGE (Pt, Pd) + chromite
    Chromitite layers: commercial chromite
    UG2 Chromitite: PGE
    World's dominant source of Pt (>70%), Pd, Cr, V

  Great Dyke, Zimbabwe: chromite, PGE
  Stillwater Complex, Montana: PGE (palladium-rich)
  Duluth Complex, Minnesota: Cu-Ni-PGE
```

### Ni-Cu-PGE Sulfide Deposits

```
MECHANISM: sulfide immiscibility
  Mafic/ultramafic magma becomes S-saturated
  → Sulfide liquid separates (like oil in water)
  → Sulfide droplets scavenge Ni, Cu, Co, PGE from silicate melt
  → Sulfide liquid pools settle → ore body

SUDBURY, ONTARIO (1.85 Ga):
  Formed by meteorite impact: 10–15 km diameter bolide
  Impact melted crust → melt sheet → sulfide immiscibility
  Largest Ni-Cu-PGE deposit in Western Hemisphere
  Falconbridge, Inco (now Vale, Glencore): world-class producers

NORILSK-TALNAKH, RUSSIA (250 Ma):
  Associated with Siberian Traps flood basalts (Permian mass extinction event)
  Largest Ni-Cu-Co-PGE deposit in world (by value)
  Norilsk Nickel produces ~25% of world's Pd, >10% Pt, ~12% Ni

KAMBALDA, WESTERN AUSTRALIA:
  Komatiite-hosted Ni sulfide; archean ultramafic flows
```

### Kimberlites and Diamonds

```
KIMBERLITE: ultramafic volcanic rock; emplaced from >150 km depth
  Transports diamonds from mantle source regions
  Diatremes: carrot-shaped volcanic pipes; flared at top

Diamond stability field: >45 km depth (lithospheric mantle)
Kimberlite brings them to surface rapidly (ascent rate ~30 km/hr)

MAJOR KIMBERLITE PROVINCES:
  South Africa: Kimberley pipes (gave "kimberlite" its name)
  Botswana: Jwaneng, Orapa (De Beers; among world's largest)
  Russia: Mirny (Sakha Republic); Udachnaya
  Canada: Ekati, Diavik (Northwest Territories)
  Australia: Argyle (pink diamond; exhausted 2020; actually lamproite)

NOT all kimberlites are diamondiferous → geologic exploration required
Diamond grade measured in carats/hundred tonnes (cpht) or carats/tonne
```

### Pegmatites

```
PEGMATITE: extremely coarse-grained igneous rock
  Final crystallization fraction of granitic melt (+ water, F, B, Li, etc.)
  → Incompatible elements concentrate here
  → Large crystals (feldspar, mica, tourmaline)
  → Exotic minerals otherwise rare

MINERALIZATION:
  Lithium: spodumene (LiAlSi₂O₆), lepidolite (Li-mica), petalite
    → Greenbushes, Australia (world's largest hard-rock Li deposit)
    → Pilbara, WA; Minas Gerais, Brazil; Manono, DRC
  Tantalum-Niobium: columbite-tantalite ("coltan")
    → Central Africa: DRC, Rwanda (conflict mineral history)
  Tin: cassiterite (SnO₂)
  REE: monazite, xenotime, gadolinite
  Beryllium: beryl, chrysoberyl
  Gems: tourmaline, topaz, aquamarine, morganite, kunzite

Pegmatite lithium vs. brine lithium:
  Both feed the Li battery supply chain
  Brine (Atacama, South America): cheaper to produce; huge volumes
  Spodumene pegmatite (Australia): higher grade, different refining process
```

## Hydrothermal Ore Deposits

Hot fluids (50–400°C) move through crust, leach metals, and deposit them where chemistry changes.

```
HYDROTHERMAL ORE SYSTEM SCHEMATIC
+------------------------------------------------------------------+
|                                                                  |
|  HEAT SOURCE: magma body, geothermal gradient                   |
|  ↓                                                              |
|  FLUID: groundwater or magmatic water, saline brines            |
|  ↓                                                              |
|  TRANSPORT: metals dissolved as chloride or sulfide complexes   |
|  AuCl₂⁻, FeCl₂, ZnCl⁺, PbCl⁺ in high-T, acidic fluids       |
|  ↓                                                              |
|  DEPOSITION TRIGGERS:                                           |
|  - Temperature drop (cooling)                                   |
|  - Pressure drop (boiling or level change)                      |
|  - Chemical change (mixing with different water, reaction       |
|    with limestone/carbonate rocks)                              |
|  - pH change (acid-neutralization)                              |
|  ↓                                                              |
|  ORE: metal sulfides/oxides precipitate in fractures/cavities   |
+------------------------------------------------------------------+
```

### Porphyry Copper Deposits

```
PORPHYRY Cu-Mo (and Au):
  Form above cooling porphyritic granite intrusions
  Magmatic hydrothermal fluid → stockwork veins → disseminated ore
  Low grade (0.3–1.5% Cu) but enormous tonnage → economic
  Open pit mining (most of world's copper)

  ALTERATION ZONES (from center outward):
  Potassic core: K-feldspar + biotite + chalcopyrite + molybdenite
  Phyllic: quartz + sericite (fine muscovite) + pyrite
  Argillic: kaolinite + alunite (acid leaching)
  Propylitic: chlorite + epidote + calcite (peripheral)

GIANT PORPHYRY DEPOSITS:
  Bingham Canyon, Utah: 1.5 billion tonnes at 0.4% Cu; open pit 1.2 km deep
  Escondida, Chile: world's largest copper mine (production)
  Collahuasi, Chuquicamata, El Teniente: all Chilean porphyries
  Grasberg, Indonesia: Cu + Au + Ag; operated by Freeport-McMoRan

SUPERGENE ENRICHMENT ON PORPHYRIES:
  Oxidized zone (top): malachite, azurite, chrysocolla, limonite
  Enriched blanket (below water table): chalcocite, bornite (high Cu grade)
  Primary sulfide zone (deep): chalcopyrite, pyrite (lower grade)
  → The chalcocite enrichment blanket = high-grade zone in many deposits
```

### Epithermal Gold-Silver

```
EPITHERMAL DEPOSITS:
  Low depth (< 1.5 km), low T (50–300°C), near surface
  Associated with active or fossil geothermal systems / volcanics

HIGH SULFIDATION:
  Acid system; enargite, pyrite, covellite, luzonite
  Examples: El Indio (Chile), Rodalquilar (Spain), Yanacocha (Peru)

LOW SULFIDATION:
  Near-neutral pH; adularia, calcite, stibnite, native gold
  Veins, stockworks, disseminations
  Examples: Comstock Lode (Nevada), Waihi (New Zealand), Hishikari (Japan)

Carlin Trend, Nevada:
  Disseminated micro-scale "invisible" gold in decarbonated limestone
  Gold too fine to see; detected by fire assay
  Nevada produces >70% of US gold; USA = 4th globally
```

### Skarn Deposits

```
SKARN: contact metasomatic zones where granite magma intersects carbonate rock

Mechanism:
  Hot magmatic fluids react with limestone/dolomite
  → Replace carbonate with calcium silicate minerals (wollastonite, garnet, pyroxene)
  → Metal sulfides precipitate in the skarn zone

Types: Fe skarn, Cu skarn, W skarn, Zn-Pb skarn, Au skarn, Mo skarn

Iron skarns: magnetite + pyrrhotite (e.g., Iron Springs, Utah; Iscaycruz, Peru)
Tungsten skarns: scheelite (CaWO₄) + wolframite; important W source
Cu-Au skarns: chalcopyrite + magnetite + bornite (e.g., Antamina, Peru)
```

### VMS (Volcanogenic Massive Sulfide) Deposits

```
VMS: seafloor hydrothermal systems
  Modern analog: black smoker chimneys at mid-ocean ridges
  Ancient VMS preserved in orogenic belts

Stratigraphy:
  Cap (upper): massive sulfide lens (chalcopyrite, sphalerite, galena, pyrite)
  Stringer zone (below): veins of sulfides extending down
  Footwall: strongly altered volcanic rocks

Types:
  Kuroko (Cu-Zn-Pb): Japanese orogenic belt
  Cyprus (Cu): ophiolite sequences (ancient ocean floor)
  VHMS (Cu-Zn): Bathurst Camp (Canada), Neves-Corvo (Portugal)
```

## Sedimentary Ore Deposits

### Banded Iron Formations (BIF)

```
BIF: alternating iron-rich + silica-rich layers
  Precambrian origin (2.5–1.8 Ga primarily)
  Formed during Great Oxidation Event: photosynthetic O₂ oxidized
    dissolved Fe²⁺ → insoluble Fe³⁺ precipitated as hematite/magnetite

CHARACTERISTICS:
  Fine lamination (mm-scale): chert (SiO₂) + hematite + magnetite
  Low grade (20–30% Fe) as deposited
  High-grade ore: secondary enrichment (supergene) → +60% Fe

MAJOR DEPOSITS:
  Hamersley Province, Western Australia: world's largest iron ore province
  Carajás, Brazil: highest grade large iron deposit
  Mesabi Range, Minnesota: US iron ore; Steel Belt history
  Krivoy Rog, Ukraine; Kursk Magnetic Anomaly, Russia

Global steel production (~1.8 billion tonnes/year) depends entirely on BIF
```

### Mississippi Valley Type (MVT) Pb-Zn

```
MVT:
  Carbonate-hosted Zn-Pb deposits (no magmatic connection)
  Basinal brines migrated through carbonate aquifers
  Metals precipitated in cavities, along faults

MINERALS: sphalerite + galena + calcite + dolomite + barite
GRADE: moderate (3–10% combined Pb+Zn)

TRI-STATE DISTRICT (Oklahoma-Kansas-Missouri): classic MVT
Southeast Missouri: "Lead Belt" (Doe Run); world's largest Pb deposit
Pine Point (Canada): Zn-Pb in Devonian carbonates
```

### Evaporites and Potash

```
EVAPORITE DEPOSITS: formed by evaporation of saline water bodies
  Potassium minerals: sylvite (KCl), carnallite, langbeinite
    → Fertilizer K₂O; Potash = K₂O equivalent
    → Saskatchewan, Canada (largest reserves); Germany; Belarus; USA

  Halite (NaCl): salt domes; chemical industry feedstock
  Gypsum (CaSO₄·2H₂O): drywall, cement retarder, soil amendment
  Trona (Na₂CO₃·NaHCO₃): Wyoming; soda ash source
```

## Mineral Resource Economics

```
RESOURCE CLASSIFICATION (JORC/NI 43-101 standard)

INFERRED RESOURCE: geological confidence low; order-of-magnitude
INDICATED RESOURCE: sufficient sampling for reasonable confidence
MEASURED RESOURCE: detailed drilling confirms grade/tonnage
      |
      v  (technical/economic studies)
PROBABLE RESERVE: indicated + economic/technical viability shown
PROVEN RESERVE: measured + economic/technical viability shown

KEY ECONOMIC PARAMETERS:
  Grade: concentration of metal (% or g/tonne for precious metals)
  Tonnage: total ore mass
  Strip ratio: tonnes waste / tonne ore (open pit mining)
  Recovery rate: % of contained metal actually extracted
  Operating cost (C1 cash cost): $/lb or $/oz to produce
  All-in sustaining cost (AISC): total cost including capex amortization
  Cutoff grade: minimum grade worth mining given costs
```

## Mining and Environmental Impact

```
MINING METHODS vs. DEPOSIT TYPE
+------------------------------------------------------------------+
|  OPEN PIT: flat-lying or large, low-grade deposits             |
|  Strip ratio determines economics                              |
|  Bingham Canyon: world's largest open pit (1.2 km deep, 4 km wide)|
|                                                                  |
|  UNDERGROUND: narrow veins, high-grade, deep                   |
|  Block caving (mass caving): large low-grade bodies underground |
|  Cut and fill: narrow high-grade veins                         |
|                                                                  |
|  PLACER: alluvial/beach mining; gold, Ti, Sn                  |
|  Dredging, sluicing, hydraulic mining (environmental damage)   |
|                                                                  |
|  IN-SITU LEACHING: inject solution, pump out dissolved metals  |
|  Copper (heap leach), uranium (ISL)                            |
+------------------------------------------------------------------+

ENVIRONMENTAL ISSUES:
  Acid mine drainage (AMD): sulfide oxidation → H₂SO₄ → heavy metal mobilization
  Tailings dam failure: Mount Polley (BC), Brumadinho (Brazil) disasters
  Smelter emissions: SO₂, Pb, Hg, As before modern controls
  Heap leach cyanide: solution ponds for Au extraction; spill risk
  Land disturbance: habitat destruction, watershed effects
  Artisanal/small-scale mining: mercury use in Au, human rights issues
```

## Common Confusion Points

**Reserves vs. resources**: Reserves are the subset of resources that can be mined economically with current technology and prices. If the gold price doubles, some "resource" becomes "reserve." These are financial reporting terms with specific legal definitions under JORC/NI 43-101.

**Grade vs. tonnage tradeoff**: High grade + low tonnage = typical underground mine. Low grade + enormous tonnage = typical open pit. The economics determine which to mine, and the mine plan must account for strip ratio (waste to ore ratio).

**Critical minerals vs. rare earths**: REE is a specific group (La-Lu + Y, Sc). "Critical minerals" is a policy category that varies by country and time — it includes REE but also lithium, cobalt, graphite, gallium, germanium, and others. Not all rare earth elements are equally rare — cerium and lanthanum are relatively abundant.

**China's position in REE**: China does not own all REE deposits — Australia, USA (Mountain Pass), India, and others have significant resources. China dominates because it invested heavily in separation/processing infrastructure over decades. Separation of individual REE is technically challenging (adjacent lanthanides are chemically very similar) and requires long plant optimization. This is the actual bottleneck, not raw mining.

**Byproduct metals**: Indium, germanium, gallium, cadmium, and tellurium are almost entirely produced as byproducts of zinc, aluminum, or copper smelting. Their supply is constrained not by ore grades but by production of the primary metal. You can't just "open an indium mine" — you need a Zn or Cu mine that has the right trace element concentrations.
