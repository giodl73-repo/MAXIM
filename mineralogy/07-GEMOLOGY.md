## The 4Cs — Diamond Grading System

The Gemological Institute of America (GIA) introduced standardized 4C grading in 1953. Before that, diamond quality had no universal standard.

### Cut

```
CUT QUALITY: proportions, symmetry, polish
+------------------------------------------------------------------+
|  IDEAL CUT PHYSICS:                                              |
|  Light enters through table (top flat face)                      |
|  Internal total reflection off pavilion facets                   |
|  Exits through crown → "brilliance"                              |
|                                                                  |
|  Table   Crown facets                                            |
|    ___________                                                   |
|   /     |     \    ← Crown: angles 34-35° from horizontal        |
|  /      |      \                                                 |
|  |   Girdle    |   ← Where stone is set                          |
|  \             /                                                 |
|   \           /    ← Pavilion: angles ~40.75°                    |
|    \    |    /                                                   |
|     \___|___/                                                    |
|         Culet (tiny bottom facet)                                |
|                                                                  |
|  Round Brilliant: 57 or 58 facets (standard)                     |
|  If pavilion too deep: light exits through sides (dark center)   |
|  If pavilion too shallow: light exits through bottom (washed out)|
|  Ideal proportions: total internal reflection back through crown|
+------------------------------------------------------------------+

GIA CUT GRADES: Excellent, Very Good, Good, Fair, Poor
AGS grades: Ideal, Excellent, Very Good, Good, Fair, Poor

Other cuts:
  Princess (square brilliant), Emerald (step cut), Oval, Pear, Marquise,
  Heart, Cushion, Radiant, Asscher (square step), Trillion
Step cuts (emerald, Asscher): parallel facets → less brilliance,
  more "hall of mirrors" effect; show clarity more prominently
```

### Color

```
GIA COLOR SCALE (colorless to yellow/brown, for diamonds):
  D, E, F  ← Colorless (rare, premium)
  G, H, I, J ← Near colorless
  K, L, M ← Faint yellow
  N–R ← Very light yellow
  S–Z ← Light yellow/brown

Why color at all?
  Nitrogen (N) impurities: most common; yellow/brown hues
  Boron (B): blue (rare; Hope Diamond is Type IIb)
  Plastic deformation: pink/purple/red (rarest colors; Argyle mine)
  Irradiation: green (natural + lab-induced)

FANCY COLOR DIAMONDS: Different scale — more color = more value
  Fancy Blue (Hope Diamond): nitrogen-free; B impurity
  Fancy Red: rarest diamond color; plastic deformation related
  Fancy Pink: Argyle mine (Australia; closed 2020) was primary source
  Fancy Yellow (Canary): high N content; some quite saturated
  Fancy Black: graphite inclusions or surface irradiation
```

### Clarity

```
CLARITY GRADES (GIA scale):
  FL  — Flawless: no inclusions or blemishes at 10× magnification
  IF  — Internally Flawless: no internal inclusions at 10×
  VVS1, VVS2 — Very Very Slightly Included: hard to see at 10×
  VS1, VS2 — Very Slightly Included: minor inclusions at 10×
  SI1, SI2 — Slightly Included: noticeable at 10×
  I1, I2, I3 — Included: obvious inclusions, may affect durability

Inclusions in diamonds:
  Crystals: mineral crystals trapped during growth (garnet, olivine, etc.)
  Clouds: hazy areas from microscopic inclusions
  Feathers: fractures/cleavage planes
  Needles: elongate crystals (rutile, etc.)
  Pinpoints: tiny dots
  Naturals: unpolished original crystal surface left on girdle

DIAMOND FINGERPRINTING:
  Every diamond has a unique inclusion map (plotted on a clarity diagram)
  Insurance grading: photographs + clarity plot = identity record
  Forged grading reports: check that inclusion map matches the stone
  GIA report number laser-inscribed on girdle of most certified stones
```

### Carat Weight

```
1 carat = 0.2 grams exactly (metric; standardized 1907)
1 carat = 100 points (dealer notation: 50 points = 0.50 ct = 1/2 ct)

Price premium: non-linear (per-carat price increases with size)
  0.99 ct stone: lower per-carat price than 1.00 ct stone
  Thresholds: 0.50, 0.75, 1.00, 1.50, 2.00, 3.00 ct create price jumps
  → A 0.95 ct stone may sell for significantly less per carat than 1.00 ct

Size vs. weight: face-up appearance depends on cut depth + diameter
  Two 1.00 ct diamonds can have different face-up sizes (different depths)
  "Spread" = diameter for the weight → more spread = larger appearance
```

## Colored Gem Quality — Beyond 4Cs

For colored gemstones (rubies, emeralds, sapphires, etc.), **origin** and **color quality** dominate value.

```
COLORED GEM VALUE HIERARCHY
+------------------------------------------------------------------+
|  Most valuable: unheated natural stone from premium origin       |
|  Ruby:    Burma (Mogok Valley) > Mozambique > Thailand           |
|  Sapphire: Kashmir > Burma > Ceylon (Sri Lanka) > Madagascar     |
|  Emerald: Colombia > Zambia > Brazil > Zimbabwe                  |
|                                                                  |
|  Origin affects: color quality + prestige + provenance           |
|  Kashmir sapphire (no longer mined): velvety blue; 20-30× premium|
|  Mogok ruby ("pigeon's blood"): fluorescent red; 5-10× premium   |
+------------------------------------------------------------------+

THE COLORED STONE 4Cs (modified):
  Color: hue + tone + saturation (HSB color model essentially)
    → "Pigeon's blood" ruby: pure red hue, medium-dark tone, vivid saturation
    → "Royal blue" sapphire: pure blue or violet-blue, medium-dark, vivid
  Clarity: different standards (inclusions more accepted in emeralds)
  Cut: less standardized than diamonds; optimizing color takes priority
  Carat: same concept; size premium exists

COLOR GRADING:
  Hue: primary color + secondary modifier (bluish-green vs. green)
  Tone: lightness/darkness (1 = very light; 10 = very dark; optimal 5–7)
  Saturation: grayness/brownness (1 = dull gray; 6 = vivid)
```

## Gem Species and Varieties

```
MAJOR GEM SPECIES

CORUNDUM (Al₂O₃):
  Ruby: red (Cr³⁺); all other colors = sapphire
  Sapphire: blue (Fe + Ti), pink, yellow, orange (Padparadscha = pink-orange),
    green, purple, colorless (white sapphire)
  Star ruby/sapphire: rutile needles → asterism (6-ray star)
  Hardness 9; excellent durability; second hardest natural mineral

BERYL (Be₃Al₂Si₆O₁₈):
  Emerald: green (Cr³⁺, V³⁺); characteristic jardin (garden) of inclusions
  Aquamarine: blue (Fe²⁺)
  Morganite: pink (Mn²⁺)
  Heliodor: yellow (Fe³⁺)
  Goshenite: colorless
  Red Beryl: scarlet (Mn); extremely rare; Utah

QUARTZ GROUP (SiO₂):
  Amethyst: purple (Fe⁴⁺ in specific orientation); most popular quartz gem
  Citrine: yellow (Fe³⁺); often heat-treated amethyst
  Rose quartz: pink (Ti/Mn or microscopic rutile inclusions)
  Smoky quartz: gray-black (Al + radiation)
  Rock crystal: colorless
  Ametrine: natural amethyst-citrine zone (Bolivia)
  Tiger's eye: golden fibrous (pseudomorph of crocidolite)
  Jasper, agate, chalcedony: microcrystalline quartz varieties

GARNET GROUP (multiple species):
  Tsavorite: green grossular (V, Cr); Tanzania/Kenya
  Demantoid: green andradite (Fe, Cr); Russia; highest dispersion of gems
  Spessartite: orange-red (Mn)
  Rhodolite: purplish-red (Mg-Al garnet blend)
  Malaya: yellow-orange (Mg-Al-Mn)
  Color-change garnet: alexandrite-like; green daylight/red incandescent

CHRYSOBERYL (BeAl₂O₄):
  Alexandrite: color-change green ↔ red (Cr³⁺; alexandrite effect)
    Daylight: green (Cr absorbs red)
    Incandescent: red (Cr allows red through; human eye more sensitive to R)
  Cat's eye chrysoberyl: rutile needles → chatoyancy (cat's eye effect)
  Yellow/brown chrysoberyl: hardness 8.5

TOPAZ: Al₂SiO₄(F,OH)₂ — hardness 8
  Colorless, blue (common; often treated), pink, yellow, orange
  Imperial topaz: yellow-orange to pink-orange; rare
  "London blue," "Swiss blue," "Sky blue": treated blue varieties (irradiation)

SPINEL (MgAl₂O₄):
  Red (Fe²⁺, Cr³⁺): historically confused with ruby (e.g., Black Prince's "Ruby")
  Blue, purple, orange, pink: other colors
  Hardness 8; good durability; undervalued historically but rising

TOURMALINE (complex boron silicate):
  Paraiba: blue-green (Cu²⁺); Brazil, Mozambique; extreme rarity premium
  Chrome tourmaline: vivid green (Cr); Tanzania
  Watermelon: pink core/green rim
  Rubellite: red-pink; Liddicoatite, elbaite
  Indicolite: blue
  Hardness 7–7.5

OPAL (SiO₂·nH₂O — amorphous):
  Play-of-color from diffraction by ~150–300 nm silica sphere arrays
  White, black, crystal, fire (Mexican), boulder opal
  Hardness 5.5–6.5; sensitive to dehydration (crazing)

DIAMOND (C):
  See 4Cs section above
```

## Treatments and Enhancement

**Disclosure requirements**: GIA, AGL, and major labs require disclosure of treatments. Trade standards mandate disclosure; failure is considered fraudulent.

```
COMMON TREATMENTS
+------------------------------------------------------------------+
|  HEAT TREATMENT (thermal)                                        |
|  Ruby/sapphire: 99%+ of commercial stones heated                 |
|  Purpose: improve color and/or clarity (dissolve silk)           |
|  Temperature: 1500–1900°C in controlled atmosphere               |
|  Detection: inclusions altered (melted, tension halos),          |
|    no rutile silk, altered chemistry at parting planes           |
|  Emerald: heat improves clarity (dissolves inclusions) rarely used|
|                                                                  |
|  FRACTURE FILLING                                                |
|  Emerald: 99%+ of commercial emeralds have fractures filled      |
|    Filler: cedar oil, synthetic resins, Opticon                  |
|  Ruby/sapphire: glass or borax filling of heavy fractures        |
|    Lead glass (Pb) filling in rubies: changes density, RI        |
|    Detection: RI measurement, heavy element by XRF               |
|  Diamond: Laser drilling + filling (improving clarity)           |
|    Laser drill channel → acid bleaches inclusion                 |
|    Then resin fills channel                                      |
|                                                                  |
|  IRRADIATION (radiation exposure)                                |
|  Blue topaz: colorless/pale → blue (neutron/electron/gamma)      |
|  Blue diamond: irradiation (then heating for green diamond)      |
|  Fancy colored diamond: gamma or electron bombardment            |
|  Detection: temperature gradient of color (color zoned vs.       |
|    uniform); spectroscopy; distinctive absorption features       |
|                                                                  |
|  COATING                                                         |
|  Pale blue sapphire: thin Be-oxide coating = pink                |
|  Topaz: mylar film on pavilion for "mystic topaz" colors         |
|  Beryllium diffusion: heat + Be saturate lattice → new colors    |
|  Detection: by immersion (color only at surface)                 |
+------------------------------------------------------------------+
```

## Synthetic Gemstones

**Same chemical composition and crystal structure as natural** — different production method.

```
SYNTHESIS METHODS

CVD DIAMOND (Chemical Vapor Deposition):
  H₂ + CH₄ gas + microwave plasma → carbon atoms deposit on substrate
  Conditions: ~800–1200°C, low pressure plasma chamber
  Growth rate: up to ~0.1 mm/day (fast for diamond growth)
  Result: type IIa (nitrogen-free) diamond plates
  Then cut like mined diamonds

  CVD diamonds: optically identical to natural; spectroscopic detection
  HPHT treatment can remove brown color from CVD diamonds

  CVD diamond uses the same principle as CVD in semiconductor manufacturing:
  a reactive gas mixture (H₂/CH₄ here, vs silane/TEOS for silicon oxides)
  decomposes in a plasma and deposits the target material on a substrate.
  Diamond CVD runs hotter (~800–1200°C vs ~400–700°C for many Si CVD steps)
  and uses microwave plasma rather than thermal activation. The atomic hydrogen
  in the plasma etches sp² carbon (graphite) preferentially, leaving only sp³
  carbon (diamond) — a selective chemistry that the semiconductor industry
  would recognize as analogous to selective etch processes.

HPHT DIAMOND (High Pressure High Temperature):
  Metal flux (Fe, Co, Ni) dissolves graphite at high P/T
  Conditions: ~5–6 GPa, ~1300–1500°C
  Natural diamonds form at similar conditions in Earth's mantle
  Result: often yellow (N impurities from the growth process)
  HPHT treatment of natural brown diamonds → colorless or fancy yellow

  Detection: UV fluorescence patterns, strain patterns, inclusions

HYDROTHERMAL SYNTHESIS (emerald, alexandrite, quartz):
  Nutrient material dissolved in water at high T/P in sealed vessel
  (autoclave) → slow crystallization on seed crystal
  Mimics natural hydrothermal ore-forming processes
  Chatham, Gilson emeralds: hydrothermal; near gem quality

FLUX GROWTH (emerald, alexandrite, ruby, spinel):
  Mineral dissolved in molten flux (B₂O₃, Li₂O, MoO₃, etc.)
  Flux solidifies at lower T than the mineral; crystals nucleate and grow
  Carroll Chatham, Knischka, Sumitomo: flux-grown rubies

FLAME FUSION / VERNEUIL (corundum, spinel):
  Powdered feedstock dropped through oxy-hydrogen flame → melts
  Drips onto rotating pedestal → boule crystallizes
  Very fast and cheap → flame fusion rubies/sapphires (very low value)
  Crystal strain patterns visible under microscope; curved growth lines

CZOCHRALSKI METHOD (corundum):
  Seed crystal pulled from melt with precise rotation + pull rate
  High quality; used for sapphire watch crystals and phone screens
  Same process as single-crystal Si growth in semiconductor manufacturing

LASER INSCRIPTION:
  GIA, IGI, HRD inscribe lab-grown/treated stone reports and lot codes
  on girdle → allows traceability
```

### Natural vs. Synthetic Detection

```
DETECTION METHODS AT LABS (GIA, AGL, Gübelin, SSEF)
+--------------------------------------------------+
|  Spectroscopy (UV-VIS, FTIR, Raman):             |
|  → Color origin (natural vs. irradiation)        |
|  → Treatment indicators (glass fill, fracture)   |
|                                                  |
|  Photoluminescence spectroscopy:                 |
|  → CVD diamond signature peaks                   |
|  → HPHT treatment markers                        |
|                                                  |
|  UV fluorescence:                                |
|  → Natural blue diamond: red phosphorescence     |
|  → CVD: typically no UV phosphorescence          |
|                                                  |
|  Inclusions:                                     |
|  → Flux inclusions (tiny solidified flux drops)  |
|  → Curved striae (Verneuil)                      |
|  → Natural inclusions (minerals, fingerprints)   |
|                                                  |
|  X-ray fluorescence (XRF):                       |
|  → Heavy elements (Pb) in glass-filled ruby      |
|  → Be detection (Be diffusion treated sapphire)|
+--------------------------------------------------+
```

## Trade Standards and Certification

| Organization | Focus | Notes |
|-------------|-------|-------|
| GIA (Gemological Institute of America) | Diamond + colored stones | Most internationally recognized lab |
| AGL (American Gemological Laboratories) | Colored stones + origin | Very detailed origin reports |
| Gübelin Gem Lab | Colored stones + origin | Swiss; authoritative |
| SSEF (Swiss Gemological Institute) | Colored stones | Kashmiri sapphire specialists |
| IGI | Diamonds + colored stones | Faster/cheaper; used for lab-grown |
| HRD Antwerp | Diamonds | European market |

**Lab-grown diamond disclosure**: FTC (2018) ruled "lab-grown," "lab-created," or "man-made" must appear before "diamond" in marketing. Cannot use "synthetic" alone as misleading. The underlying material IS diamond — chemically, physically, optically identical.

The market has bifurcated sharply. De Beers launched Lightbox in 2018 — explicitly pricing lab-grown diamonds as fashion accessories at a fixed $800/ct, deliberately separating them from the mined diamond luxury market. Lab-grown 1-ct colorless stones that sold for ~$4,000 in 2016 fell to ~$300–500 by 2023 as CVD capacity scaled. Mined diamonds have held price for the top quality tiers (D/E/F, IF/VVS) but mid-market mined stones face pricing pressure. The regulatory and market trajectory is the same as other manufactured goods that displaced natural equivalents: price compression once production scales, with a residual premium for "natural" positioned as provenance rather than physical superiority.

## Common Confusion Points

**Ruby vs. garnet vs. spinel**: All can be red. Red garnet (almandine/pyrope): hardness 7–7.5, no cleavage, vitreous luster. Ruby: hardness 9, no cleavage, corundum. Red spinel: hardness 8, octahedral cleavage. The "Black Prince's Ruby" in the British Crown Jewels is a red spinel, not a ruby — the distinction wasn't understood until modern gemology.

**Synthetic vs. simulant**: A synthetic ruby IS ruby (Al₂O₃ with Cr). A simulant for ruby (red glass, red garnet, red cubic zirconia) is a different material imitating ruby's appearance. Entirely different disclosure requirements and value.

**Heat treatment ubiquity**: Virtually all commercial rubies and blue sapphires are heated. An unheated sapphire of equivalent quality is worth 2–10× more due to rarity. The label "no heat" requires lab certification and commands a significant premium.

**Clarity standards by species**: Emeralds are graded leniently on clarity — inclusions are expected and accepted. Diamonds are graded very strictly. A "VS1 emerald" and a "VS1 diamond" are not comparable clarity statements — they reflect species norms.

**Carat weight ≠ size**: Different gem species have different densities. A 1 ct ruby (corundum, SG 4.0) will appear smaller than a 1 ct diamond (SG 3.52) or much smaller than a 1 ct emerald (beryl, SG 2.71). Emerald is bought by carat but one should expect larger physical size per carat.
