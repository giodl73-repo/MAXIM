# Physical Properties of Minerals

## The Big Picture

```
+------------------------------------------------------------------+
|           MINERAL IDENTIFICATION: OBSERVABLE PROPERTIES          |
|                                                                  |
|  MECHANICAL                 OPTICAL               OTHER          |
|  Hardness (Mohs)            Color                 Magnetism      |
|  Cleavage                   Streak                Fluorescence   |
|  Fracture                   Luster                Taste/smell    |
|  Tenacity                   Transparency          Radioactivity  |
|  Specific gravity           Birefringence         Piezoelectricity|
|                             Pleochroism                          |
|                             Fluorescence                         |
+------------------------------------------------------------------+

Each property is a macroscopic expression of crystal structure + bonding
→ Cleavage follows crystallographic planes (weakest bond directions)
→ Hardness reflects bond strength and type
→ Luster reflects interaction of photons with crystal surface/structure
→ Birefringence reflects optical anisotropy of crystal structure
```

## Hardness

### Mohs Hardness Scale

```
MOHS SCALE (1812) — comparative, not linear
+--------------------------------------------------+
|  1   Talc          scratched by fingernail       |
|  2   Gypsum        scratched by fingernail       |
|  2.5 ————————————— fingernail threshold ————————|
|  3   Calcite       scratched by copper coin      |
|  3.5 ————————————— copper coin threshold ————————|
|  4   Fluorite      scratched by steel nail       |
|  5   Apatite       scratched by steel knife      |
|  5.5 ————————————— steel knife threshold ————————|
|  6   Orthoclase    scratched by window glass     |
|  6.5 ————————————— window glass threshold ————————|
|  7   Quartz        scratches window glass easily |
|  8   Topaz         scratches quartz              |
|  9   Corundum      scratches topaz               |
| 10   Diamond       scratches everything          |
+--------------------------------------------------+

Scale is ordinal, not interval:
  The jump from 9 → 10 (corundum → diamond) represents a
  3–4× jump in absolute hardness (Vickers scale)
  The jump from 1 → 2 is much smaller

Field kit: fingernail + copper coin + knife blade + piece of quartz
→ brackets any unknown mineral's hardness quickly
```

**What controls hardness**: Bond strength and type. Diamond (3D covalent carbon) = hardest. Talc (Mg phyllosilicate with only Van der Waals between layers) = softest. Ionic minerals fall in between.

### Absolute Hardness Context

| Mineral | Mohs | Vickers (GPa) |
|---------|------|---------------|
| Talc | 1 | ~0.02 |
| Calcite | 3 | ~0.09 |
| Orthoclase | 6 | ~0.80 |
| Quartz | 7 | ~1.12 |
| Topaz | 8 | ~1.43 |
| Corundum | 9 | ~2.26 |
| Diamond | 10 | ~70–100 |

The Mohs scale is logarithmic in absolute hardness terms — useful for field work but doesn't convey the true gaps.

## Cleavage and Fracture

**Cleavage** = tendency to break along specific crystallographic planes — planes where bond density is lowest or bond strength is weakest.

```
CLEAVAGE QUALITY DESCRIPTORS
  Perfect    → smooth, mirror-like planes; very easy to produce
  Good       → smooth but with some irregularity
  Distinct   → recognizable but rough
  Poor       → barely visible, mineral tends to fracture instead
  None       → no cleavage; fracture only

CLEAVAGE ANGLES (critical for ID)
  Cubic:          3 planes at 90° (halite, galena)
  Octahedral:     4 planes at 70.5°/109.5° (fluorite, diamond)
  Rhombohedral:   3 planes at ~75° (calcite)
  Prismatic:      2 planes parallel to length (feldspar, pyroxene)
  Basal:          1 plane perpendicular to c-axis (mica)
  Pinacoidal:     2 planes perpendicular to axis

KEY DIAGNOSTIC CLEAVAGES:
  Mica: 1 perfect basal {001} → peels into flat sheets
  Calcite: 3 perfect {1011} rhombohedral → rhombus fragments
  Halite: 3 perfect {001} cubic → cubes
  Pyroxene: 2 at ~90° (square cross-section in thin section)
  Amphibole: 2 at ~60°/120° (diamond cross-section in thin section)
  Feldspar: 2 at ~90° but less perfect than pyroxene
```

**Fracture** = break along irregular non-planar surfaces (when no cleavage, or cutting across cleavage planes):

| Fracture type | Appearance | Example |
|--------------|-----------|---------|
| Conchoidal | Smooth curved shell-like | Quartz, obsidian, flint |
| Uneven/irregular | Rough uneven surface | Pyrite, most minerals |
| Hackly | Jagged, sharp points | Native copper, gold |
| Splintery | Fibrous splinters | Serpentine, some amphiboles |

**Flint and chert**: Cryptocrystalline quartz (SiO₂) — no cleavage, perfect conchoidal fracture → sharp predictable edges → Paleolithic tool material. The fracture is as predictable as glass; you can control the break direction by the force angle.

## Luster

The quality and intensity of light reflected from a mineral's surface:

```
LUSTER TYPES AND WHAT CAUSES THEM
+------------------------------------------------------------------+
|  METALLIC: opaque minerals with free electrons (metals, sulfides)|
|  → Mirror-like, like polished metal                             |
|  Examples: native gold, silver, galena, chalcopyrite, pyrite    |
|                                                                  |
|  SUB-METALLIC: partially opaque, less mirror-like              |
|  Examples: magnetite, hematite (specular)                       |
|                                                                  |
|  ADAMANTINE (diamond-like): very high refractive index         |
|  → Brilliant, fire, high dispersion of light                   |
|  Examples: diamond (n=2.42), cerussite, anglesite, cassiterite  |
|                                                                  |
|  VITREOUS (glassy): most common; moderate refractive index     |
|  → Clear to glassy like glass (n=1.4–1.9)                     |
|  Examples: quartz, calcite, fluorite, garnet, most silicates   |
|                                                                  |
|  RESINOUS: amber-like; moderate-high n in non-metallic minerals |
|  Examples: sphalerite, opal, some garnets                      |
|                                                                  |
|  PEARLY: pearl-like; seen in some cleavage faces               |
|  → Interference at subsurface layers                           |
|  Examples: talc, mica (cleavage faces), dolomite               |
|                                                                  |
|  SILKY: fibrous minerals, parallel reflections                 |
|  Examples: fibrous gypsum (satin spar), asbestos, tiger's eye  |
|                                                                  |
|  WAXY/GREASY: slightly rough surface scatters light            |
|  Examples: nephrite jade, talc (greasy feel), serpentine       |
|                                                                  |
|  DULL/EARTHY: fine-grained, rough, no specular reflection      |
|  Examples: kaolinite, limonite, chalk                          |
+------------------------------------------------------------------+
```

## Color and Streak

**Color** is the least reliable property for mineral identification — often due to trace impurities, not the mineral itself.

```
IDIOCHROMATIC vs. ALLOCHROMATIC MINERALS
+------------------------------------------------------------------+
|  IDIOCHROMATIC: color is intrinsic (due to major chemistry)    |
|  Azurite: always blue (Cu²⁺)                                   |
|  Malachite: always green (Cu²⁺)                               |
|  Rhodochrosite: pink (Mn²⁺)                                    |
|  Olivine: olive green (Fe²⁺-Mg ratio)                         |
|  → Reliable color indicator for these                          |
|                                                                  |
|  ALLOCHROMATIC: color due to trace impurities                  |
|  Quartz: colorless (pure) BUT also purple (amethyst, Fe⁴⁺),   |
|    pink (rose quartz), yellow (citrine), gray-black (morion)   |
|  Corundum: colorless (pure), ruby (red, Cr³⁺),               |
|    sapphire (blue, Fe+Ti), pink, yellow, orange              |
|  Fluorite: colorless, purple, green, yellow, blue             |
|  Calcite: white to colorless (pure), any color if impure      |
|  → Color alone is unreliable for these                         |
+------------------------------------------------------------------+
```

**Streak** = color of the powdered mineral (drawn across unglazed porcelain). More reliable than color because the powder eliminates surface effects and forces a representative average:

| Mineral | Mineral color | Streak | Why streak differs |
|---------|--------------|--------|-------------------|
| Hematite | Red-black, silver | Red-brown | True Fe₂O₃ color |
| Pyrite | Brassy yellow | Greenish-black | FeS₂ powder absorbs |
| Gold | Yellow | Yellow | Doesn't change |
| Chalcopyrite | Brassy | Greenish-black | Powder absorbs |
| Galena | Lead gray | Lead-gray | Consistent |

Hematite is the classic example: can appear silver-black (specular) or red-brown (earthy) as a hand sample, but always gives a red-brown streak.

## Specific Gravity

Mass of mineral / mass of equal volume of water. Measured with a scale + Archimedes principle (weighing in air and water).

```
SPECIFIC GRAVITY RANGES
+--------------------------------------------------+
|  SG < 2.0    Extremely light (rare)             |
|  SG 2.0–3.0  Most silicates: quartz (2.65),    |
|               feldspars (2.56–2.76), calcite    |
|               (2.71), gypsum (2.32)             |
|  SG 3.0–4.0  Most sulfates, some silicates:    |
|               barite (4.48!), corundum (4.0)    |
|  SG 4.0–5.0  Most sulfides, some oxides:       |
|               pyrite (5.0), sphalerite (4.0)    |
|  SG 5.0–7.5  Dense sulfides, some oxides:      |
|               chalcopyrite (4.2), galena (7.6)  |
|  SG > 7.5    Native metals: gold (19.3),        |
|               silver (10.5), copper (8.9)       |
+--------------------------------------------------+

Field test: "heft" — a rock that feels surprisingly heavy for its size
→ Galena (SG 7.6): feels remarkably dense
→ Barite (SG 4.48): called "heavy spar" — dense for a light-colored mineral
→ Gold (SG 19.3): can concentrate in stream pans; sinks while other
   minerals wash away (placer gold panning exploits this)
```

## Optical Properties

### Transparency

- **Transparent**: can read through (fluorite, calcite, crystal quartz)
- **Translucent**: light passes but image not visible (opal, milky quartz)
- **Opaque**: no light passes (pyrite, magnetite, graphite)

### Birefringence (Double Refraction)

Birefringence is the optical consequence of an anisotropic dielectric permittivity tensor ε. In a cubic crystal, ε is a scalar (ε_xx = ε_yy = ε_zz) — one refractive index, isotropic. In a hexagonal or trigonal crystal, ε has two independent components (ε_⊥ and ε_∥ to the c-axis) — two principal refractive indices n_o and n_e, giving birefringence Δn = |n_e - n_o|. In orthorhombic, monoclinic, and triclinic crystals, all three diagonal components of ε differ — the optical indicatrix (an ellipsoid with three semiaxes proportional to n_α, n_β, n_γ) describes the angular dependence of the refractive index. The petrographic microscope exploits this: a thin crystal section in cross-polarized light produces interference colors directly related to the retardation Γ = Δn × thickness, which is read off the Michel-Lévy chart.

Light entering an anisotropic crystal splits into two rays polarized perpendicular to each other, traveling at different velocities. The difference in refractive indices = **birefringence**.

```
ICELAND SPAR (optical calcite):
  Two images seen through crystal → extreme birefringence
  n_e = 1.486, n_o = 1.658 → birefringence = 0.172 (very high)

Applications:
  Petrographic microscope: crossed polarizers + rotating stage
  → Interference colors in thin sections identify minerals
  → Birefringence color and extinction angle are diagnostic

UNIAXIAL vs. BIAXIAL CRYSTALS:
  Uniaxial: hexagonal and trigonal crystals — one optic axis
    → Positive (quartz) or negative (calcite)
  Biaxial: orthorhombic, monoclinic, triclinic — two optic axes
    → Angle between axes (2V) is diagnostic
```

### Pleochroism

Color changes when crystal is rotated under polarized light — different absorption along different crystal axes.

- **Dichroism**: 2 colors (uniaxial minerals)
- **Trichroism**: 3 colors (biaxial minerals)

Biotite: strong pleochroism yellow-brown to dark brown under plane-polarized light — diagnostic.

## Other Properties

### Magnetism

- **Ferromagnetic** (attracted to hand magnet): magnetite, pyrrhotite
- **Paramagnetic** (requires sensitive instrument): most iron-bearing silicates
- **Diamagnetic** (repels): quartz, calcite

**Field test**: magnetite is attracted to a hand magnet strongly. Pyrrhotite weakly. This distinguishes magnetite (opaque, black) from hematite (also opaque, black but red streak).

### Fluorescence

Some minerals emit visible light when irradiated with UV:

| Mineral | UV emission | Cause |
|---------|-------------|-------|
| Scheelite (CaWO₄) | Bright blue-white | Crystal field effect of W |
| Willemite (Zn₂SiO₄) | Brilliant green | Mn activator impurity |
| Calcite | Variable (red, orange) | REE activators |
| Fluorite | Blue-violet | REE activators; gave name "fluorescence" |
| Diamond | Blue | Nitrogen impurities |
| Aragonite | White | Different from calcite |

**The name "fluorescence"** comes from fluorite — first mineral noted to show the effect.

### Piezoelectricity and Pyroelectricity

**Quartz** (SiO₂): piezoelectric — mechanical stress generates electric polarization. Exploited in:
- Frequency standards: quartz oscillators are the clock reference in every digital system — every UART baud rate, every SPI clock domain, every microcontroller crystal. A 32.768 kHz watch crystal (2¹⁵ Hz, convenient for binary division to 1 Hz) is a thin quartz plate cut at a specific crystallographic angle (AT-cut at 35°15' to suppress temperature dependence of frequency). The mineral's piezoelectric response is the physical basis of digital timing.
- Pressure sensors
- Sonar transducers

**Tourmaline**: pyroelectric (electric polarization changes with temperature) and piezoelectric.

These properties require the absence of a center of symmetry in the crystal structure — only 20 of the 32 point groups can be piezoelectric.

```
HAND SPECIMEN IDENTIFICATION DECISION TREE
+------------------------------------------------------------------+
|  Step 1: LUSTER                                                  |
|   Metallic/opaque → sulfides, oxides, native metals             |
|   Non-metallic → silicates, carbonates, etc.                   |
|                    |                                             |
|  Step 2: HARDNESS                                               |
|   < 2.5 (fingernail): talc, gypsum                             |
|   2.5–3.5 (coin): calcite, halite                              |
|   3.5–5.5 (knife): apatite, fluorite                           |
|   5.5–7 (glass): pyroxene, amphibole, feldspar                 |

|                    |                                             |
|  Step 3: CLEAVAGE/FRACTURE                                      |
|   Perfect in one direction: mica                               |
|   3 planes at 90°: halite, galena                              |
|   3 planes at ~75°: calcite                                     |
|   2 planes at ~90°: pyroxene, feldspar                         |
|   2 planes at 60°/120°: amphibole                              |
|   Conchoidal fracture: quartz, obsidian                        |
|                    |                                             |
|  Step 4: STREAK (if opaque/dark)                               |
|   Red-brown streak: hematite                                    |
|   Black streak: magnetite                                       |
|   Yellow streak: gold                                          |
|   Green-black streak: pyrite, chalcopyrite                     |
|                    |                                             |
|  Step 5: SPECIAL TESTS                                         |
|   Magnetic: magnetite                                           |

|   Taste: halite (salty), sylvite (bitter salty)               |
|   UV: scheelite, willemite                                     |
+------------------------------------------------------------------+
```

## Common Confusion Points

**Mohs is ordinal, not interval**: Quartz (7) is not 1.4× harder than feldspar (6). The scale only tells you relative scratch resistance — the absolute jumps are uneven. Diamond is far harder (in absolute terms) than corundum (9) suggests.

**Cleavage vs. fracture**: Cleavage produces flat, reflective planes that repeat (you can see the flatness and the repeated angle). Fracture is irregular. A broken crystal may show both — fresh cleavage faces adjacent to fracture surfaces.

**Color is unreliable**: A beginner's biggest mistake is trusting color. Quartz comes in nearly every color. Fluorite comes in purple, green, yellow, colorless, blue. Always confirm with hardness and cleavage before trusting color.

**Streak plate hardness**: The unglazed porcelain streak plate has hardness ~7. Minerals harder than 7 (quartz and above) won't leave a streak — they'll scratch the plate instead. Use steel file or diamond plate for harder minerals.

**Specific gravity "heft"**: With practice, heavy minerals (galena, barite, magnetite) feel immediately different from typical silicates. A fist-size piece of galena weighs more than a similarly sized piece of calcite by nearly 3×. This is trainable intuition.
