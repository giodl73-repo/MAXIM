# Nuclear Deterrence

## The Big Picture

Nuclear deterrence is applied game theory: you're trying to influence another
rational actor's decisions by making certain outcomes too costly to risk. The
intellectual framework, developed 1945–1965, remains the logic governing the
world's most consequential military postures.

```
+----------------------------------------------------------------------+
|                    DETERRENCE LOGIC                                   |
|                                                                      |
|  ACTOR A (potential attacker)                                        |
|  Expected benefit of attack = pV - (1-p) × Punishment               |
|                                                                      |
|  p    = probability of attack succeeding                             |
|  V    = value of objective                                           |
|  Punishment = what you suffer in retaliation                         |
|                                                                      |
|  DETER BY DENIAL: make p small (attack can't succeed)                |
|  DETER BY PUNISHMENT: make Punishment massive (retaliation certain)  |
|                                                                      |
|  Nuclear deterrence primarily works through punishment               |
|  MAD = Punishment → infinity → expected value always negative        |
|                                                                      |
|  CREDIBILITY PROBLEM:                                                |
|  Punishment must be BELIEVED to matter                               |
|  If attacker doubts you'll retaliate → deterrence fails              |
+----------------------------------------------------------------------+
```

## Section 1: Mutual Assured Destruction (MAD)

```
  MAD LOGIC:
  ┌────────────────────────────────────────────────────────────────┐
  │  Both sides maintain SECURE SECOND-STRIKE CAPABILITY           │
  │                                                                │
  │  State A launches first strike → destroys most of B's forces  │
  │  B still has enough surviving warheads → retaliates           │
  │  A is destroyed anyway → therefore A never launches           │
  │                                                                │
  │  KEY: Second-strike survivability is the stabilizing factor   │
  │  → Missiles in hardened silos, submarines (SSBNs), bombers    │
  │  → SSBNs are most survivable: mobile, opaque                  │
  └────────────────────────────────────────────────────────────────┘

  MCNAMARA'S ASSURED DESTRUCTION CRITERIA (1960s):
  Ability to destroy ~25-33% of Soviet population and ~50-67% of industry
  in a retaliatory strike → sufficient deterrence
  (Not a targeting requirement — a minimum survivable capability)

  MAD STABILITY PARADOX:
  Stable MAD requires BOTH sides to be vulnerable to second strike
  → Missile defense is DESTABILIZING (erodes other side's confidence in retaliation)
  → ABM Treaty (1972) banned nationwide missile defense for this reason
  → SDI ("Star Wars") and US ABM withdrawal (2002) upset this logic
```

### Wohlstetter vs McNamara Debate

```
  WOHLSTETTER: "The Delicate Balance of Terror" (1958-1959)
  Argued: the appearance of second-strike capability is not the same
  as actually having it; deterrence is fragile, not robust
  Counterforce targeting (targeting enemy forces) vs countervalue (cities)
  Emphasized: accuracy, survivability, precision — not just numbers

  McNAMARA RESPONSE: Assured destruction — you don't need to win a nuclear war,
  just threaten unacceptable costs; city-busting is sufficient; numbers matter less
  than credibility

  TENSION THAT NEVER RESOLVED:
  If deterrence fails, does it matter how you fight a nuclear war?
  McNamara: no (just massive retaliation; war-fighting is irrelevant)
  Wohlstetter/Schlesinger: yes (flexible options may be more credible)
  → Led to SIOP (Single Integrated Operational Plan) and targeting debates
```

## Section 2: The Credibility Problem

Thomas Schelling's contribution — the most important nuclear theorist after Hiroshima:

```
  SCHELLING: The Strategy of Conflict (1960), Arms and Influence (1966)

  CREDIBILITY = Resolve × Capability
  Capability: can you physically deliver the retaliation?  → technical problem
  Resolve:    will you actually do it?                      → THE hard problem

  "WOULD YOU TRADE NEW YORK FOR PARIS?"
  Extended deterrence problem: US threatens to use nuclear weapons to defend allies
  If Soviets attack West Germany → would US really nuke Moscow?
  → Risks Soviet retaliation → US destroyed
  → Is this credible? The ally needs to believe it; the adversary must doubt it profitable

  WAYS TO MAKE THREATS CREDIBLE:
  ┌────────────────────────────────────────────────────────────────┐
  │ 1. COMMITMENT DEVICES: "Burning bridges"                       │
  │    Trip wires: deploy own troops so attacking ally = attacking US │
  │    (West Berlin — US troops as literal hostages)               │
  │                                                                │
  │ 2. REPUTATION: Establishing a record of following through      │
  │    Resolve in one crisis → credibility in next crisis          │
  │    "Credibility" arguments for intervention in Vietnam, etc.   │
  │                                                                │
  │ 3. AUTOMATED/DELEGATED RESPONSE: "Launch on warning"           │
  │    Remove human decision → make retaliation mechanical         │
  │    Problem: false positives (Petrov incident, 1983)            │
  │                                                                │
  │ 4. IRRATIONALITY AS STRATEGY (Schelling): appear willing to    │
  │    do the irrational → make adversary doubt your sanity        │
  │    Nixon's "madman theory" — deliberate unpredictability        │
  └────────────────────────────────────────────────────────────────┘

  POSITIVE vs NEGATIVE CONTROL:
  Positive: authorization chain → ensure weapons only used when authorized
  Negative: prevent unauthorized use → permissive action links (PALs)
  Tension: high negative control → adversary thinks you won't actually use
  High positive control → risk of unauthorized use / accident
```

## Section 3: Extended Deterrence

```
  EXTENDED DETERRENCE: Using your nuclear arsenal to deter attacks on allies

  US NUCLEAR UMBRELLA:
  ├── NATO (30+ European allies + Canada)
  ├── Japan (under US-Japan Security Treaty)
  └── South Korea (under US-South Korea Mutual Defense Treaty)

  COUPLING: Visible, credible linkage of US strategic forces to ally defense
  How to couple:
  - Forward-deployed tactical nuclear weapons (B61 bombs in Europe)
  - SACEUR (Supreme Allied Commander Europe) has nuclear planning role
  - Nuclear sharing: NATO allies fly nuclear strike missions (dual-key)
  - Statements: NPR (Nuclear Posture Review) explicitly states umbrella

  MASSIVE RETALIATION vs FLEXIBLE RESPONSE:
  Eisenhower/Dulles (1953): "Massive retaliation" — respond to any Soviet
  aggression with nuclear attack; cheap, credible, escalatory
  Kennedy (1961): "Flexible Response" — conventional capabilities must match
  conventional threats; don't want only nuclear option for every crisis
  → NATO adopted flexible response 1967 (MC 14/3)

  DECOUPLING FEAR:
  European allies fear US will NOT use nuclear weapons for them
  → Demand for consultation, transparency, participation
  → Nuclear Planning Group (NPG) created 1967
  → Extended deterrence credibility requires continuous management

  SOUTH KOREA CASE (contemporary):
  DPRK growing nuclear arsenal + ICBM capability
  → South Korea public: 70%+ want own nuclear weapons
  → Washington Declaration (2023): new nuclear consultative group
  → Credibility gap → proliferation pressure
```

## Section 4: First-Strike Stability and Crisis Stability

```
  FIRST-STRIKE STABILITY: neither side benefits from striking first

  CRISIS STABILITY: in a crisis, neither side is tempted to strike preemptively

  DESTABILIZING FACTORS:
  ┌────────────────────────────────────────────────────────────────┐
  │ SILO-BASED ICBMs:                                              │
  │ - Fixed location → targetable                                  │
  │ - Large, accurate, vulnerable                                  │
  │ - "Use them or lose them" pressure in crisis                   │
  │ - Cannot ride out first strike → pressure for launch-on-warning│
  │                                                                │
  │ COUNTERFORCE CAPABILITY:                                       │
  │ - High accuracy + high yield → can destroy enemy missiles      │
  │ - Makes first strike look "winnable" (degrade second strike)   │
  │ - Incentivizes adversary to strike first before losing missiles│
  │                                                                │
  │ MISSILE DEFENSE:                                               │
  │ - If effective → adversary's second strike fails               │
  │ - Erodes MAD stability → adversary must use MORE missiles, now │
  │ - Or strike first before defense is ready                      │
  └────────────────────────────────────────────────────────────────┘

  STABILIZING FACTORS:
  ┌────────────────────────────────────────────────────────────────┐
  │ SSBNs (ballistic missile submarines):                          │
  │ - Mobile → not targetable for first strike                     │
  │ - Opaque → adversary can't know if/where                       │
  │ - Secure second strike guaranteed                              │
  │ → Most stabilizing leg of the nuclear triad                   │
  │                                                                │
  │ HARDENING + DISPERSAL:                                         │
  │ - Hard silos resist close-in bursts                            │
  │ - Many small silos → must use many warheads to cover (expensive)│
  │                                                                │
  │ MUTUAL VULNERABILITY (countervalue targeting):                 │
  │ - No incentive to strike first if second strike still destroys │
  └────────────────────────────────────────────────────────────────┘

  LAUNCH-ON-WARNING vs RIDE-OUT:
  LaW: launch after detecting inbound missiles (before impact)
       Problem: false alarms (Petrov, 1983; US false alarms)
       Advantage: preserves second-strike capability
  Ride-out: absorb first strike → retaliate with survivors
       Problem: silo-based missiles are destroyed
       Advantage: no false alarm risk
  Most stable: SSBNs (can ride out + have guaranteed second strike)
```

## Section 5: Arms Control

```
  ARMS CONTROL LOGIC: Not disarmament — managing the competition
  to maintain stability and reduce accident/miscalculation risk

  MAJOR AGREEMENTS:
  ┌────────────────────┬────────────────────────────────────────────┐
  │ Treaty             │ Key Provisions                             │
  ├────────────────────┼────────────────────────────────────────────┤
  │ ABM Treaty (1972)  │ Banned nationwide missile defense (two sites │
  │                    │ each → zero sites by 1974 amend.); rationale:│
  │                    │ preserve MAD stability; US withdrew 2002    │
  ├────────────────────┼────────────────────────────────────────────┤
  │ SALT I (1972)      │ Freeze on ICBMs + SLBMs; first strategic   │
  │                    │ arms agreement; no reduction, just cap      │
  ├────────────────────┼────────────────────────────────────────────┤
  │ SALT II (1979)     │ Equal ceilings ~2,250 launchers; not ratified │
  │                    │ by US Senate (Afghanistan invasion)         │
  ├────────────────────┼────────────────────────────────────────────┤
  │ INF Treaty (1987)  │ Eliminated all ground-based intermediate   │
  │                    │ missiles (500-5,500 km range); US/USSR only │
  │                    │ (China not party); US withdrew 2019        │
  ├────────────────────┼────────────────────────────────────────────┤
  │ START I (1991)     │ Actual warhead reductions; 6,000 each;     │
  │                    │ verification + on-site inspection           │
  ├────────────────────┼────────────────────────────────────────────┤
  │ New START (2010)   │ 1,550 deployed strategic warheads each;    │
  │                    │ expired Feb 2026; Russia suspended 2023;    │
  │                    │ no replacement agreed                       │
  ├────────────────────┼────────────────────────────────────────────┤
  │ NPT (1968)         │ Five nuclear states pledge disarmament;    │
  │                    │ 191 non-nuclear states pledge no acquisition │
  │                    │ IAEA safeguards; 191 parties; India/Pakistan │
  │                    │ /Israel/DPRK outside                       │
  ├────────────────────┼────────────────────────────────────────────┤
  │ CTBT (1996)        │ Comprehensive test ban; not in force (US,  │
  │                    │ China, India, Pakistan, Israel not ratified)│
  └────────────────────┴────────────────────────────────────────────┘
```

## Section 6: Proliferation Logic

Why do states pursue nuclear weapons, and why do they stop?

```
  WHY PURSUE:
  ┌────────────────────────────────────────────────────────────────┐
  │ SECURITY: Existential threat that conventional forces can't    │
  │ address (Israel's Samson option, India vs China, Pakistan vs   │
  │ India, DPRK vs US overwhelming conventional superiority)       │
  │                                                                │
  │ STATUS/PRESTIGE: "Great power card" (France, UK)               │
  │ Independence from superpower guarantees                        │
  │                                                                │
  │ DOMESTIC: Military-industrial complex, national identity,      │
  │ bureaucratic momentum                                          │
  └────────────────────────────────────────────────────────────────┘

  WHY STOP:
  ┌────────────────────────────────────────────────────────────────┐
  │ SECURITY GUARANTEE: US extended deterrence replaces own weapons │
  │ (Japan, South Korea, Germany — under US umbrella)              │
  │                                                                │
  │ ECONOMIC COST: Sanctions, technology denial, investment flight  │
  │                                                                │
  │ REGIME CHANGE: Gaddafi (Libya) gave up WMD 2003 → still        │
  │ overthrown 2011; DPRK observed lesson                          │
  │                                                                │
  │ NEGATIVE SECURITY ASSURANCES: Guarantees not to attack         │
  │ non-nuclear states                                             │
  └────────────────────────────────────────────────────────────────┘

  CASE STUDIES:
  South Africa: developed 6 weapons; dismantled before black majority rule (feared
                weapons falling into ANC hands, later apartheid government)
  Ukraine: gave up ~1,900 warheads for Budapest Memorandum guarantees (US/UK/Russia);
           Russia invaded 2014 + 2022 → lesson drawn by every potential proliferant
  Libya: gave up nuclear program 2003 for sanctions relief; regime ended 2011
  Iraq: program dismantled by Gulf War + inspections; 2003 invasion premised on WMD
```

## Decision Cheat Sheet

| Concept | Key Implication |
|---|---|
| MAD stability | Requires both sides to be vulnerable to second strike |
| Credibility problem | Resolve + capability; commitment devices make threats believable |
| First-strike stability | SSBNs > silos; counterforce capabilities destabilize |
| ABM Treaty logic | Defense = bad for stability (erodes adversary's confidence in retaliation) |
| Extended deterrence | Alliance guarantees reduce proliferation pressure |
| Ukraine lesson | Conventional security guarantees (Budapest) proved insufficient |
| Proliferation prevention | Positive assurances + negative security assurances + NPT inspections |

## Common Confusion Points

**More warheads ≠ more stable**: beyond the minimum for secure second strike, additional
warheads don't add stability. They may invite more targeting of your arsenal. SALT/START
showed both sides could accept equal constraints without military disadvantage.

**Missile defense is not obviously good**: a robust national missile defense is
destabilizing if it appears able to negate the adversary's second-strike capability.
The ABM Treaty was based on this insight. Modern limited systems (against DPRK/Iran)
are less problematic; comprehensive systems aimed at Russia/China are deeply destabilizing.

**Russia's "escalate-to-deescalate" doctrine**: Russia is believed to have a doctrine
of limited nuclear use to stop a losing conventional conflict ("escalate to deescalate").
This lowers the nuclear threshold and is a major source of NATO planning concern.

**New START expiry**: New START, the last major US-Russia nuclear agreement, expired
February 5, 2026. Russia suspended cooperation 2023. There is no successor agreement.
The strategic competition is now effectively unconstrained by treaty limits for the
first time since the early 1970s.
