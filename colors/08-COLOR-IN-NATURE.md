# Color in Nature — Structural Color, Pigments, Bioluminescence

## The Big Picture

```
+------------------------------------------------------------------+
|            HOW NATURE MAKES COLOR                                |
|                                                                  |
|  PIGMENT-BASED (chemical absorption):                            |
|  Melanins: black/brown/red-yellow — UV protection, dark color    |
|  Carotenoids: red/orange/yellow — must acquire from diet         |
|  Porphyrins: heme (red blood), chlorophyll (green leaves),       |
|               biliverdin (bird eggs), bilirubin (jaundice)       |
|  Pterins: yellow/white/orange (butterfly wings)                  |
|  Flavonoids: UV-visible to insects; flower patterns in UV        |
|                                                                  |
|  STRUCTURAL (physical nanostructure):                            |
|  Thin-film interference: soap bubbles, beetle elytra, feathers   |
|  Photonic crystals: morpho butterfly, opal — periodic arrays     |
|  Quasi-ordered scatter: blue bird feathers (no blue pigment)     |
|  Diffraction: CD surface, some beetle scales                     |
|                                                                  |
|  EMISSION:                                                       |
|  Bioluminescence: luciferin/luciferase oxidation                 |
|  Fluorescence: UV → visible re-emission                          |
|                                                                  |
|  MANY ANIMALS COMBINE ALL THREE:                                 |
|  Peacock tail: melanin (dark bar) + thin-film (iridescent blue-  |
|  green) + structural matrix → the total visual effect            |
+------------------------------------------------------------------+
```

## Melanins

```
MELANIN TYPES:
  Eumelanin: black and dark brown
    Polymer of DHI (5,6-dihydroxyindole) + DHICA monomers
    Very broad-band absorption: absorbs UV strongly; broad visible absorption
    → High concentration → black; lower → brown
    Photo-protection: absorbs UV photons → dissipates as heat
    → Suntan = induced melanin production as UV protection

  Phaeomelanin: red and yellow
    Copolymer of DHI with cysteinyl-DOPA
    Less UV absorption than eumelanin (less protective)
    Red hair: higher phaeomelanin:eumelanin ratio
    → Red-haired individuals: more UV damage per dose (less protective phaeo)

  Neuromelanin: in brain (substantia nigra) — different function, unrelated to color

MELANIN SYNTHESIS PATHWAY:
  Tyrosine → DOPA → dopaquinone (tyrosinase enzyme, rate-limiting)
  → DHI/DHICA → eumelanin polymerization
  → Or + cysteine → benzothiazine → phaeomelanin
  Tyrosinase: copper-containing enzyme; albinism = tyrosinase defect
    OR OCA gene mutations preventing melanin distribution

HAIR COLOR:
  Black: high eumelanin, densely packed
  Brown: high eumelanin, less dense
  Blonde: low eumelanin, trace phaeomelanin
  Red: high phaeomelanin, low eumelanin
  Gray/White: no melanin (melanocyte death/dysfunction with age)

FEATHER COLOR FROM MELANIN:
  Eumelanin granules → black, dark gray, dark brown
  Phaeomelanin granules → chestnut, rufous, buff, yellow-brown
  Shaped/oriented melanin: nanorod arrays can produce structural color
    overlaid on pigment (see peacock feathers below)

SQUID INK:
  Sepia/cuttlefish ink: melanin + mucopolysaccharides
  Color: black → brown (diluted) → the "sepia" tones of early photography
    → Sepia tone = iron-based brown, named by association with squid ink color
```

## Carotenoids

```
CAROTENOIDS: C₄₀ isoprenoid pigments
  Built from 8 isoprene units (tetraterpenes)
  Long conjugated polyene chain → absorption in blue-green → reflects red/orange/yellow

  RULE: Animals CANNOT synthesize carotenoids de novo
        Must obtain from diet (plants, fungi, bacteria, algae, other animals)

CAROTENOID CLASSES:
  Carotenes (hydrocarbon only):
  • β-carotene: orange (carrots, orange peppers) — provitamin A
  • Lycopene: red (tomatoes, watermelon) — no vitamin A activity
  • α-carotene: orange (carrots, trace)

  Xanthophylls (contain oxygen):
  • Lutein: yellow (egg yolk, kale, marigold) — macular pigment in human eye
  • Zeaxanthin: yellow-orange — also in macula (photoprotection)
  • Astaxanthin: red (salmon, shrimp, krill, flamingo) — ketone at C4 and C4'
  • Canthaxanthin: orange-red (red flamingo feathers, some fish)

FLAMINGO PINK/RED:
  Diet: blue-green algae (Arthrospira) + small crustaceans (Artemia)
  Carotenoids extracted from food → deposited in feathers and skin
  Flamingos in captivity without proper carotenoid diet: turn white/pale
  At breeding: transfer carotenoids from feathers to egg yolk for chick nutrition
    → Breeding adults become paler (they've donated their color to eggs)
  CANTHAXANTHIN: the specific xanthophyll giving flamingo the deep pink-orange

AUTUMN LEAVES:
  Summer: chlorophyll dominates (absorbs red + blue → green) — MASKS carotenoids
  Autumn: daylight shortens → chlorophyll synthesis stops → chlorophyll breaks down
  → Carotenoids UNMASK (they were always there): yellows, oranges appear
  → Red/purple: anthocyanins ACTIVELY SYNTHESIZED in autumn (separate process)

  Why red in autumn?
  Multiple hypotheses:
  1. Photoprotection: UV shield during nitrogen resorption from leaves
  2. "Honest signal": bright autumn → healthy tree → aphids avoid (don't lay eggs)
  3. Anti-oxidant: protect leaf during nutrient export
  Best evidence: co-evolution with aphids (hypothesis 2) has support
  Maple reds require glucose: warm days + cold nights → sugar accumulation → anthocyanin
  → Cool autumns: more vivid reds; drought years: less vivid
```

## Porphyrins and Other Biological Chromophores

```
PORPHYRIN STRUCTURE:
  Four pyrrole rings linked by methine bridges → macrocycle
  Aromatic macrocycle: 18 π-electrons (Hückel: 4n+2, n=4) → intense absorption
  Soret band: ~400-430 nm (very strong)
  Q bands: ~500-650 nm (moderate)
  Metal in center changes color dramatically

HEMOGLOBIN: iron(II) porphyrin (heme b)
  Fe²⁺ in heme + globin protein
  Oxygenated (HbO₂): bright scarlet-red (absorbs ~540, ~577 nm)
  Deoxygenated (Hb): darker maroon-red (absorbs ~555 nm differently)
  Carboxyhemoglobin (HbCO): cherry red (CO locks open absorption)
  Methemoglobin (Fe³⁺): chocolate-brown (iron oxidized — can't carry O₂)

CHLOROPHYLL: magnesium porphyrin (chlorin, partially reduced ring)
  Chlorophyll a: absorbs 430 nm + 680 nm → reflects green + yellow-green
  Chlorophyll b: absorbs 460 nm + 640 nm → slight shift, more blue-green
  → Leaves are green because chlorophyll absorbs the red and blue light
    most useful for photosynthesis; green is the "wasted" band

BILIVERDIN: green (from heme degradation)
  Heme → biliverdin (green) → bilirubin (yellow-orange)
  Biliverdin in bird eggshells → responsible for blue/green egg color
    Robin's egg: biliverdin; great tit: biliverdin
    Signal to mate: egg color = female's health/carotenoid status (hypothesis)
  Biliverdin in some reptile blood (skinks with green blood)

BILIRUBIN: yellow-orange
  Terminal heme degradation product
  Jaundice: bilirubin accumulation → yellow skin/eyes
  Newborn jaundice: treated with phototherapy (blue light, ~460 nm)
    → Photoisomerizes bilirubin to excretable form

TURACIN (turacos only):
  Red copper porphyrin unique to Musophagidae (turacos)
  Water-soluble: the red literally washes off in rain (regained by preening)
  Only birds to use a copper porphyrin for vivid red (others use carotenoids)
```

## Structural Color — The Physics of Nanostructure

```
THE MORPHO BUTTERFLY (detailed):
  Wing scales have microscopic ridges with lamellar "Christmas tree" structure
  Each lamella: ~200 nm chitin shelf with ~200 nm air gap below
  Thin-film interference: 2nt = mλ
  At t ≈ 100 nm, n_chitin ≈ 1.5: constructive for λ ≈ 450 nm (blue)
  → Brilliant iridescent blue visible for 100+ meters
  → NO BLUE PIGMENT: the color is purely geometric
  → Color varies with angle (constructive interference shifts with incidence angle)
  Morpho menelaus: characteristic electric blue
  Morpho rhetenor: even more intense (more lamellae layers)

  WHY IT MATTERS FOR BIO-INSPIRED MATERIALS:
  Structural color doesn't fade (no photodegradation of chromophore)
  → Research into structural color for textiles, displays
  → Photovoltaics inspired by morpho light-trapping structures

OPAL:
  Structure: random close-packing of silica spheres (~150-300 nm diameter)
  → "Play of color" (adularescence or opalescence): different crystal domains
    diffract different wavelengths as angle changes
  Precious opal: hydrated amorphous silica with 3D sphere packing → vivid colors
  Common opal: no regular sphere packing → no color play
  Synthetic opal (grown from silica beads): reproducible structural color
  Inverse opal: fill spaces between opal spheres → remove spheres → negative structure

PEACOCK FEATHERS:
  Mechanism: melanin nanorod arrays in hexagonal packing
  + thin-film interference from melanin layers
  The specific color (blue-green vs copper vs brown) depends on:
  • Nanoscale rod spacing (determines interference wavelength)
  • Rod diameter and arrangement
  Varying those parameters across tail → the "eye" pattern
  Tail also contains ordinary melanin for dark bars

BLUE BIRD FEATHERS (Eastern bluebird, kingfisher, cotinga):
  NOT thin-film interference (no angle-dependent color)
  NOT pigment (no known blue pigment in birds)
  MECHANISM: quasi-ordered nanostructure (sponge-like air channels in keratin matrix)
  → Coherent backscattering at blue wavelengths
  "Quasi-photonic crystal": not perfectly ordered but sufficiently regular
  → Non-iridescent blue (same from all angles) — unlike morpho butterfly
  This mechanism was only understood clearly in early 2000s

HUMMINGBIRD IRIDESCENCE:
  Gorget (throat): brilliant color visible from only certain angles
  Mechanism: thin-film interference from melanin nanoplatelet layers in feather barbules
  → Platelet orientation determines angle of maximum reflectance
  → Different gorget orientations → different angle of maximum color → communication
  Color: can be anywhere in visible spectrum depending on layer thickness
  → Red gorget (ruby-throated) vs blue-green (magnificent) = same mechanism, different geometry
```

## Bioluminescence

```
CHEMISTRY: luciferin + O₂ + ATP → oxyluciferin + CO₂ + AMP + LIGHT
  Luciferase enzyme catalyzes oxidation of luciferin substrate
  → Excited oxyluciferin → ground state → emits photon
  → No heat: "cold light" (nearly 100% efficient)
  Color of emission: determined by luciferase structure (active site pocket geometry)
    → Constrains oxyluciferin's excited state → emission wavelength

LUCIFERIN TYPES (convergently evolved multiple times):
  Firefly luciferin (D-luciferin): yellow-green (λ ~560 nm)
  Coelenterazine: blue (~460 nm) — most widespread in ocean
  Bacterial luciferin (FMNH₂ + aldehyde): blue-green
  Dinoflagellate luciferin (tetrapyrrole): blue-green
  → Same function (bioluminescence) → multiple independent chemical solutions

PREVALENCE:
  ~80% of deep-sea species are bioluminescent
  → At depth below 200 m: no sunlight → bioluminescence = primary light source
  Deep ocean light ecology:
  • Counter-illumination: ventrally facing photophores match downwelling daylight
    → Animal appears invisible from below (no silhouette)
  • Lures: anglerfish (Melanocetus johnsonii) esca (lure with bacterial symbionts)
  • Communication: squid (Loligo), some fish → species-specific flash patterns
  • Startle/warning: dinoflagellate bioluminescence in bow wave → disorienting predators

FIREFLIES:
  ~2,000 species; each has species-specific flash pattern
  → Photinus pyralis (eastern US): single J-shaped flash; female responds 2s later
  → Flash rate, duration, interval = species ID signal for mate recognition
  Photinus vs Photuris: Photuris females mimic Photinus female signals → eat males
    → Called "femmes fatales"; acquire lucibufagins (toxins) this way for defense

GFP (GREEN FLUORESCENT PROTEIN):
  Aequorea victoria jellyfish: bioluminescence system
  Aequorin: emits blue light from coelenterazine oxidation
  GFP: absorbs blue (from aequorin) → re-emits green
  → Chain: ATP + O₂ → aequorin → blue → GFP → green

  OSAMU SHIMOMURA (Nobel 2008): isolated aequorin + GFP from jellyfish (1960s-1970s)
  MARTIN CHALFIE (Nobel 2008): used GFP gene to tag proteins in C. elegans (1994)
  ROGER TSIEN (Nobel 2008): engineered GFP color variants:
    • ECFP (cyan), EYFP (yellow), DsRed (red) from various organisms
    → Multicolor fluorescent protein palette for labeling multiple structures
  GFP revolutionized cell biology: non-invasive, genetically encoded
  → "Watch proteins in living cells in real time" — was not possible before GFP
```

## Camouflage Strategies

```
BACKGROUND MATCHING (crypsis):
  Appear to match local substrate
  Octopus (Octopodidae): chromatophores (pigment cells) + iridophores (structural) +
    papillae (texture) → sub-second color/texture/shape change
    Chromatophores: elastic sacs; muscles expand → large colored patch visible
    No neural signal → muscle relaxes → sac contracts → disappears
  Ground-nesting birds (plover, snipe): complex feather patterns match substrate
  Katydids: leaf mimics (Phylliidae) — match specific leaf species with veins

DISRUPTIVE COLORATION:
  Bold patterns that break up outline
  Zebra stripes: purpose DISPUTED
  Hypotheses:
  1. Anti-predator: disruptive patterning confuses predator at distance
  2. Thermoregulation: differential heating of stripes creates air movement
  3. Ectoparasite deterrent: tsetse flies avoid striped patterns (landing inhibition)
     → Experimental support: flies land less on striped horses
  Current evidence: #3 (tsetse fly deterrence) has strongest experimental support
  → Stripes evolved in Africa where tsetse flies are prevalent

COUNTERSHADING (Thayer's principle, Abbott H. Thayer, 1896):
  Dark on top, light on bottom
  → Cancels gradient of illumination (bright above, shadow below)
  → Makes 3D animal appear flat/less visible
  Applies to: most vertebrates (sharks, deer, penguins, hawks)
  Penguin: dark back (viewed from above → blends with dark ocean) +
           white belly (viewed from below → blends with bright surface)

BATESIAN vs MÜLLERIAN MIMICRY:
  Batesian: palatable species mimics unpalatable/dangerous
    Monarch (toxic) ← mimicked by → Viceroy (now known to also be somewhat toxic)
    King snake (harmless) mimics coral snake (venomous) in color pattern
    → Works because predator learned from real unpalatable → avoids mimic

  Müllerian: multiple unpalatable species converge on same warning pattern
    → Mutual reinforcement: predator learns one → avoids all that look like it
    → Poison dart frogs (Dendrobatidae): many species converge on similar patterns
    → Costs predators: every species's coloration taught the lesson to new predators
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| Why do flamingos lose color in captivity? | Diet: flamingos can't synthesize carotenoids; they get canthaxanthin/astaxanthin from algae and crustaceans. Without the right diet, no pigment, feathers go pale/white. |
| Does the morpho butterfly have blue pigment? | No — zero blue pigment. Color from periodic lamellar nanostructure on wing scales (thin-film/photonic crystal interference). Iridescent because angle-dependent. |
| What is GFP and why did it win a Nobel Prize? | Green Fluorescent Protein from Aequorea jellyfish; engineered as genetically encoded fluorescent tag for proteins in living cells. Shimomura/Chalfie/Tsien, Nobel 2008. |
| Why are leaves green? | Chlorophyll absorbs the most useful photosynthetic wavelengths (red ~680 nm + blue ~430 nm). Green is reflected because it's the least useful. Chlorophyll doesn't absorb green efficiently. |
| Why are bird eggshells blue-green? | Biliverdin — a porphyrin pigment from heme degradation, deposited in the eggshell during formation. Robin blue = biliverdin. |
| What makes structural color different from pigment color? | No chromophore: color from periodic nanostructure (thin-film interference, photonic crystal). No absorption → no fade. Often iridescent (angle-dependent). Cannot be mixed like pigments. |
| What is Thayer's countershading principle? | Animals dark on top, light on bottom cancel illumination gradient → appear less three-dimensional → harder to see against background. Explains coloration of most large vertebrates. |

## Common Confusion Points

**Blue in bird feathers is almost never pigment:** Blue and violet bird colors are almost exclusively structural (coherent scattering, thin-film interference, photonic crystals). Only a handful of specific exceptions (porphyrins in some species). "Blue feathers" are a classic test case for structural vs pigment color.

**Carotenoids in flamingos don't come from "eating pink things":** They come from eating algae and crustaceans containing specific xanthophylls (canthaxanthin, astaxanthin). The animals eating the flamingos' prey chain: algae → crustaceans → flamingo. Animals don't absorb carotenoids from random food; specific dietary sources matter.

**Bioluminescence ≠ fluorescence:** Bioluminescence = biochemical light generation (luciferin oxidation → photon). Fluorescence = absorb light → re-emit light. Completely different mechanisms. Coral reefs have BOTH: bioluminescent organisms AND fluorescent proteins (which need external UV/blue light to glow). GFP in a dark room with no UV = invisible; needs excitation light.

**Octopus color change is not conscious decision-making:** The chromatophore system is neurally controlled but the "decision" is distributed across neurons. The octopus skin samples the environment through skin photoreceptors (yes — light-sensing skin, independent of eyes) and adjusts pattern semi-autonomously. It's sophisticated but not a deliberate creative choice.
