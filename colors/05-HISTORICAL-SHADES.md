# Historical Shades — The Biography of Named Colors

## The Big Picture

```
+------------------------------------------------------------------+
|         COLOR NAMES AS HISTORICAL ARTIFACTS                      |
|                                                                  |
|  EVERY named shade has a specific origin:                        |
|  a substance, a place, an event, a person, or an organism        |
|                                                                  |
|  PATTERN: material → name (not perception → name)               |
|  Once the material has a name, the COLOR gets the same name      |
|  Later: color term decouples from material (mauve now means      |
|  a color, not necessarily the specific dye)                      |
|                                                                  |
|  TRACING ORIGINS REVEALS:                                        |
|  • Chemistry (mauveine, alizarin synthesis)                      |
|  • Trade routes (vermilion from kermes → mercury sulfide)        |
|  • Geography (ultramarine = beyond the sea)                      |
|  • Accidents (Perkin's aniline experiment gone wrong)            |
|  • Politics (magenta, Battle of 1859)                            |
|  • Botany (indigo plant, gamboge resin)                         |
|  • Zoology (cochineal insect, kermes oak scale)                  |
+------------------------------------------------------------------+
```

## The Synthetic Dye Revolution (1856–1914)

```
COAL TAR → COLORS: The central story

BEFORE 1856: Organic colorants from:
  Plants (indigo, madder, weld, saffron, woad)
  Animals (cochineal, kermes, Tyrian purple)
  Minerals (ochre, sienna, azurite, malachite, smalt)
  → All requiring cultivation, harvesting, or mining

PERKIN'S ACCIDENT (1856):
  William Henry Perkin, 18 years old
  Royal College of Chemistry, London
  August Wilhelm von Hofmann's lab: project = synthesize quinine from aniline
  Result: dark reddish-brown sludge + purple residue on flask
  → Dissolved in alcohol → brilliant purple solution → dyed silk beautifully
  → MAUVEINE (aniline purple) — first synthetic organic dye

WHY IT MATTERED:
  Coal tar (abundant waste product of gas-street-lighting industry)
  → Perkin showed: coal tar → valuable colorants via organic chemistry
  → Explosion of synthetic dye research: same decade produced magenta, azure blue
  → Founded the modern chemical industry
  → Germany saw the opportunity faster: BASF (1865), Bayer (1863), Hoechst (1863)
  All three: dye chemistry → pharmaceutical chemistry (aspirin from salicylic acid, same bench)

TIMELINE OF SYNTHETIC DYES:
+------+--------------------+-------------------------------------------+
| 1856 | Mauveine (Perkin)  | First coal tar dye; purple; Britain       |
| 1858 | Magenta (Verguin)  | Rosaniline; named after battle 1859       |
| 1860 | Aniline blue       | Triphenylmethane dye series               |
| 1861 | Crystal violet     | Widely used bacteriological stain         |
| 1862 | Bismarck brown     | Named after Bismarck; yellowish-brown     |
| 1869 | Alizarin synthetic | Same year: Perkin (UK) + Graebe/Lieb.(DE) |
| 1873 | Methylene blue     | Paul Ehrlich's staining revolution        |
| 1883 | Congo red          | First direct dye (no mordant needed)      |
| 1897 | Synthetic indigo   | BASF Heumann synthesis → destroys natural |
+------+--------------------+-------------------------------------------+
```

## Mauve (1856)

```
THE STORY:
  William Henry Perkin: August 1856, during Easter holiday experiment
  Attempting: oxidation of allyltoluidine with potassium dichromate
  (trying to synthesize quinine for anti-malarial use)
  Got: dark reddish-brown precipitate
  NOTICED: purple residue → dissolved in alcohol → intense purple solution
  TESTED: dipped silk strip → brilliant purple permanent color → washed well
  ACTED IMMEDIATELY:
  • Sent sample to Pullar's of Perth (leading Scottish dyer)
  • Pullar replied: "if your discovery does not make the goods too expensive,
    it is decidedly one of the most valuable that has come out for a long time"
  • Perkin patented the process (age 18)
  • Left school
  • Built a factory in Greenford Green with his father and brother (age 20)
  • Made fortune within 5 years

MAUVEINE CHEMISTRY:
  Not a single compound: mixture of methylated phenazine-related compounds
  Mauveine A: C₂₆H₂₃N₄⁺
  Mauveine B: C₂₇H₂₅N₄⁺
  (Phenosafranine class, later studies confirmed)
  Color: distinctive purple-mauve, slightly reddish

"MAUVE MEASLES":
  Queen Victoria wore mauve at the 1862 London International Exhibition
  Empress Eugénie (France) wore mauve → Parisian fashion followed
  "Mauve decade" (1890s): Oscar Wilde-era aesthetic movement
  The color dominated fashion for a decade → satirized as "mauve measles"

AFTERMATH:
  Perkin continued: discovered alizarin synthesis (1869), coumarin synthesis
  Received knighthood (1906), Order of the Prussian Crown, Hofmann Medal
  In 2006: 150th anniversary celebrated internationally as Perkin Centenary (Sesquicentennial)
```

## Magenta (1859)

```
THE DYE:
  Synthesized by: François-Emmanuel Verguin, Lyon (1858-1859)
  Also: August Wilhelm von Hofmann (London) independently
  Chemical: rosaniline hydrochloride (fuchsine)
  Structure: triphenylmethane dye

  COLOR: brilliant purplish-red, extremely vivid
  Soluble in alcohol → intense color
  Fastness: moderate (fades in sunlight) but popular for intensity

THE BATTLE:
  June 4, 1859: Battle of Magenta, Lombardy (then Austria, now Italy)
  French/Piedmontese forces vs Austrian Empire
  French victory (Napoleon III, Piedmont-Sardinia allied)
  Battlefield: town of Magenta, 25 km west of Milan
  Casualties: ~12,000 total

THE NAMING:
  Verguin's dye commercialized by Renard Frères & Franc, Lyon
  Named "magenta" shortly after the battle (timing disputed — some say
  the dye was named in 1858 before the battle; others say after)
  Most historians: the name attached commercially after the battle
  → One of history's more striking coincidences: major European battle and
    major chemical discovery in the same year, same name applied

MAGENTA AS SUBTRACTIVE PRIMARY:
  In CMYK printing: M = magenta (absorbs green, reflects red + blue)
  Magenta is not in the visible spectrum (non-spectral color — see colors/00)
  → A "purplish-red" that requires mixing red and blue light to perceive
  → The dye was named for the visual impression, not spectral location
```

## Gamboge

```
SOURCE: Resin of Garcinia trees
  Species: Garcinia hanburyi (Vietnam, Cambodia), G. morella (India, Sri Lanka)
  Harvest: cuts in bark → yellow latex → dried into cakes/cylinders

ETYMOLOGY:
  Portuguese: "Goma-guta" → French "gomme-gutte" → English "gamboge"
  From "Cambogia" = Cambodia (Khmer) → major source
  The color name = the place of origin

COLOR:
  Deep, saturated golden-yellow to orange-yellow
  Semi-transparent in oil → excellent for glazes
  Very permanent in manuscript illumination

USES:
  Watercolor: primary yellow in Chinese and Japanese ink painting
  Medieval European manuscripts: gold illumination
  Maps and cartography: yellow wash for land areas
  Varnish ingredient: spirit varnish tint

CHEMISTRY:
  Active compound: gambogic acid + related xanthones
  Xanthone skeleton: aromatic, extended conjugation → strong yellow absorption
  Toxic: gambogic acid is a potent purgative; chronic exposure hepatotoxic
  → Used medicinally as a purgative (dangerous, sometimes fatal doses)

STATUS TODAY:
  Largely replaced by cadmium yellow and synthetic alternatives
  Still available as artists' watercolor
  "Gamboge hue" = synthetic replacement (typically arylide yellow)
  The color: intense golden yellow-orange, more orange than chrome yellow
```

## Chartreuse

```
SOURCE: Chartreuse liqueur
  Carthusian monks, Grande Chartreuse monastery, Chartreuse mountains near Grenoble
  Secret herbal recipe: 130 herbs and flowers, macerated in grape brandy
  Two varieties:
  • Green Chartreuse: 55% alcohol, bright yellow-green
  • Yellow Chartreuse: 40% alcohol, amber-gold

COLOR:
  "Chartreuse" (the color) = the vivid yellow-green of the green liqueur
  CIE approximate: ~120-130° hue angle (midway between yellow and green)
  → Sits on yellow-green border, roughly λ ≈ 560-570 nm equivalent

  CONFUSION:
  "Chartreuse yellow" (more yellow): ~560 nm equivalent
  "Chartreuse green" (more green): ~570 nm equivalent
  Americans tend toward "chartreuse = yellow-green" (more yellow)
  Europeans tend toward "chartreuse = vivid green" (less yellow)
  Both are legitimate — the liqueur has two colors, and both are "chartreuse"

  Pantone calls out "chartreuse" as ~Pantone 375-376 range (vivid yellow-green)
  The key feature: maximum chroma at yellow-green boundary, extremely vivid

CULTURAL STATUS:
  "Chartreuse" as fashion color: 1960s psychedelic → 1980s neon → recurring trendy
  Interiors: "chartreuse green" accent walls (polarizing)
  The monastery has been making the liqueur since 1605
  The color name borrowed the liqueur's association sometime in the 19th century
```

## Puce and Other Confusing French Colors

```
PUCE:
  French: "puce" = flea (Pulex irritans)
  The color of a flea: brownish-purple to reddish-brown
  STORY: Marie Antoinette supposedly called a new fabric "puce-colored"
  (flea-colored) → it became fashionable at Versailles (~1775-1780)
  Actual color: debated. Candidates:
  • Flea body: brownish-red (dried blood?): ~RGB (122, 47, 61) — reddish-brown
  • Flea blood meal: darker, more purple
  • Modern CSS "puce": #CC8899 (dusty mauve-pink) — very different
  → No consensus; "puce" is color-identified differently by different people

CERISE:
  French: "cerise" = cherry (Prunus cerasus)
  Color: bright, vivid red, slightly cherry-like in depth
  ~RGB (222, 49, 99) — vivid cherry red
  19th century synthetic dye color (rosaniline series)
  Still used in fashion: "cerise pink" = vivid hot red-pink

ECRU:
  French: "écru" = raw, unbleached (from Old French: "escru" = raw)
  Color: off-white/beige — the color of unbleached linen
  Not quite white, not quite cream: very pale warm gray-beige
  Used in fashion for unbleached natural fiber color

TAUPE:
  French: "taupe" = mole (the animal, Talpa europaea)
  Color: mole's brown-gray fur → the color
  Modern "taupe": gray-brown, very neutral
  Extremely popular in interior design (2010s "greige" = gray + beige)

BISQUE:
  French: "bisque" = twice-baked ceramic (bisque firing before glaze)
  Color: pale cream/pinkish-white — the color of bisque-fired clay
  HTML/CSS named color: #FFE4C4 (light peach-cream)
```

## Ultramarine and Indigo

```
ULTRAMARINE:
  "Ultramarinus" (Latin/Italian): "beyond the sea"
  → The lapis lazuli came from Badakhshan (Afghanistan) — literally beyond the Mediterranean
  → The pigment's name describes its ORIGIN, not its color
  Natural ultramarine: deep blue-violet, slightly reddish
  Synthetic ultramarine (1826): identical structure, fraction of the cost
  The name now describes the color of the pigment, not the material's geography
  → But the etymology tells the whole trade-route story

INDIGO:
  Greek: "Indikon" = "Indian thing/substance"
  → The dye came from India (Indigofera tinctoria)
  Newton's spectrum: violet → indigo → blue → green → yellow → orange → red
  Newton's 7 colors matched the 7 notes of the musical scale — the indigo
  step is very narrow and many colorists consider it an artificial distinction
  "Indigo" as wavelength: ~430-450 nm
  → A blue-violet that most observers call "deep blue" or "violet" without
    the "indigo" name to anchor the category

PRUSSIAN BLUE:
  Named for Prussia (where it was accidentally synthesized in 1704)
  Berlin dye-maker Diesbach and paint-maker Zahn
  Liebermann's laboratory (actually probably Diesbach's accidental contamination)
  First synthetic pigment? No — Egyptian blue is 3000 BCE
  But: first ACCIDENTALLY discovered pigment; first modern laboratory synthesis

COBALT BLUE vs PRUSSIAN BLUE vs ULTRAMARINE:
+-------------------+---------------------+------------------------------+
| Name              | Color character      | Chemistry                   |
+-------------------+---------------------+------------------------------+
| Prussian blue     | Greenish deep blue  | Fe₄[Fe(CN)₆]₃ (cyan-blue)  |
| Cobalt blue       | Slightly warm blue  | CoAl₂O₄ (warmer than PB)   |
| Ultramarine       | Warm reddish-blue   | S₃⁻ in aluminosilicate      |
+-------------------+---------------------+------------------------------+
All three called "blue" in casual use; very different when placed side by side
```

## Vermilion, Scarlet, and Crimson

```
THREE REDS THAT CONFUSE:

VERMILION:
  Latin: vermiculus = little worm → kermes insect (crimson dye)
  Applied to: mercuric sulfide (HgS) — brilliant scarlet-orange red
  The name jumped from organic dye to mineral pigment
  Chinese vermilion = Chinese red = the same color
  The Mercury sulfide pigment: brilliant, opaque, orange-red
  Modern: cadmium red replaced vermilion for artists

SCARLET:
  Origin: "scarlat" (medieval Latin/Italian/French) — a type of fine cloth
  (not originally a color term — a fabric quality)
  → The cloth was often dyed red → color adopted the cloth's name
  Scarlet: a bright, vivid orange-red (slightly less orange than vermilion)
  Military associations: "Scarlet fever" (rash), "Scarlet Letter," British redcoats
  Chemically: "Scarlet lake" historically = kermes lake (Vermiculus pigment)

CRIMSON:
  Arabic: "qirmiz" = kermes insect → "carmine" (via Spanish carmesí)
  English: "crimson" from the same root
  Color: deep, rich red with slight blue tinge (darker, cooler than scarlet)
  Crimson is cooler and darker than scarlet; both cooler than vermilion
  Universities: "Harvard Crimson" — dark reddish (technically pantone 201C)
  "Deep crimson" = nearly burgundy; "bright crimson" = vivid saturated red

THE DISTINCTION (rough):
  Vermilion: orange-red (HgS)
  Scarlet: vivid medium red (historically kermes, now typically azo pigments)
  Crimson: deep blue-red (carminic acid from cochineal; now quinacridone)
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| What did Perkin accidentally discover in 1856? | Mauveine — the first synthetic coal tar dye. He was attempting to synthesize quinine; got a purple precipitate that dyed silk brilliantly. Founded synthetic organic chemistry. |
| What battle was magenta named after? | Battle of Magenta, June 4, 1859, Lombardy — French/Piedmontese defeat of Austria. The dye (rosaniline) was named after the battle in the same year it was commercialized. |
| What is the etymology of "ultramarine"? | Latin "ultramarinus" = "beyond the sea" — the lapis lazuli came from Badakhshan (Afghanistan), across the Mediterranean from Europe. The origin, not the color. |
| What is the color of puce, officially? | Disputed — the color of a flea (brownish-red/brownish-purple). Different dictionaries give different answers. Named allegedly by Marie Antoinette describing a fabric at Versailles. |
| Why are there so many blues with "named" differences? | Historical pigments with different sources produce distinctly different blues: Prussian blue (greenish), cobalt blue (warm), ultramarine (reddish-warm). Each source had a name; names became color categories. |
| What is gamboge? | Yellow-orange resin from Garcinia trees (Cambodia/Vietnam), used as a semi-transparent watercolor yellow. Also a violent purgative. Replaced by cadmium yellow and synthetic equivalents. |

## Common Confusion Points

**Mauveine ≠ modern purple:** Perkin's original mauveine was a reddish-purple, somewhat similar to lavender or orchid. When we say "mauve" today, we usually mean a grayish-purple or dusty lilac — the color has shifted semantically over 170 years.

**Magenta = non-spectral:** Magenta doesn't correspond to a single wavelength. It's the brain's response to simultaneous L + S cone stimulation without M stimulation — a color that "closes the circle" between red and violet. The dye appears magenta because its absorbance peaks in the green, reflecting both red and blue.

**Chartreuse yellow ≠ chartreuse green:** They're both called chartreuse and both legitimate. When precision matters, specify. In fashion, "chartreuse" often defaults to the more yellow variant; in European paint systems, often the greener variant.

**Crimson vs scarlet has cultural weight:** These color terms carry heraldic, institutional, and literary meaning. Harvard Crimson ≠ Yale Blue ≠ Oxford Blue ≠ Cambridge Blue — all are institutional reds and blues with specific Pantone specifications that clubs treat as non-negotiable.
