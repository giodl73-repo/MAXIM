# 00 — Color: Overview

## Physics → Perception → Naming → Culture → Reproduction

> **STUB** — outline only, content to be authored

**Planned coverage:**
- The layered problem of color: (1) physics — electromagnetic radiation; (2) physiology — how eye and brain process it; (3) perception — the subjective experience; (4) naming — how languages carve up color space; (5) reproduction — how to encode and transmit color; none of these layers maps cleanly onto the others
- Color is not a property of objects — it is a property of the interaction between light source, object surface, and visual system; same surface looks different under incandescent vs fluorescent vs daylight (metamerism); color is fundamentally a perception, not a physical measurement
- The visible spectrum: ~380nm (violet) to ~700nm (red); humans sensitive to ~1 octave of the electromagnetic spectrum; arbitrary — pit vipers see IR, bees see UV; the "color spectrum" has no green boundary — cyan and magenta don't appear in the rainbow (they require mixing)
- Why some colors don't have a single wavelength: magenta/pink/purple exist only in our visual system as the brain's response to simultaneous stimulation of L and S cones without M stimulation; no monochromatic "magenta" light exists
- Color science architecture: colorimetry (measurement and specification) → color appearance models → image color management; the bridge between physics (spectral power distribution) and perception (hue/saturation/brightness)
- Decision cheat sheet: which color system to use when (Munsell for paint/design, CIE Lab for measurement, Pantone for industry communication, RGB for screens, CMYK for print, RAL for industrial powder coating)
