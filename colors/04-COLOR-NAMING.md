# Color Naming — Berlin-Kay, Wine-Dark Sea, Language and Perception

## The Big Picture

```
+------------------------------------------------------------------+
|         THE UNIVERSAL SEQUENCE OF COLOR NAMING                   |
|                                                                  |
|  BERLIN & KAY (1969): across 20 languages, color terms           |
|  are acquired in a fixed order:                                  |
|                                                                  |
|  STAGE I:   black + white  (all languages have these)            |
|  STAGE II:  red                                                  |
|  STAGE III: yellow OR green (one or the other comes next)        |
|  STAGE IV:  the other (yellow or green)                          |
|  STAGE V:   blue                                                 |
|  STAGE VI:  brown                                                |
|  STAGE VII: pink, orange, purple, gray (any order)               |
|                                                                  |
|  A language with 3 basic color terms always has: black/white/red|
|  A language with 5 always has: black/white/red + yellow + green  |
|  Blue comes before brown; brown before the remaining 4           |
|                                                                  |
|  PROFOUND IMPLICATION:                                           |
|  Colors exist on a perceptual hierarchy across all human cultures|
|  The same ordering imposed on very different physical worlds     |
+------------------------------------------------------------------+
```

## Berlin and Kay (1969)

### The Study

```
BRENT BERLIN and PAUL KAY: cognitive anthropologists, UC Berkeley
Publication: "Basic Color Terms: Their Universality and Evolution" (1969)

METHODOLOGY:
  Showed Munsell color chips to native speakers of 20 languages
  Asked them to name and identify "best examples" of basic color terms
  "Basic color term" defined as:
  • Monolexemic: not "dark red" (two words) or "sky-colored" (derived)
  • Not restricted: "blond" (hair only) excluded
  • Salient: commonly known, not technical
  → 6-11 basic terms in any language

FINDING:
  Languages add color terms in a non-random sequence
  Languages with N terms always have the same N terms (with same stages above)
  No language has "green" as its first chromatic term without having "red"
  No language has "blue" before "green"

FOCAL COLORS:
  Within any color category, speakers identify a "best example" (focal color)
  Focal colors are MORE CONSISTENT across languages than category boundaries
  → The "best red" is similar across cultures; where red ends and orange begins varies

CRITICISM (extensive):
  • Original sample: many written languages in common (not fully independent)
  • Munsell chips ≠ all stimuli (pre-selected palette)
  • "Basic" criteria are circular
  • Later work: more diversity found (Regier, Kay, Cook 2005 refinements)
  CONSENSUS: The ordering is robust but not absolute; evolution model is supported
  but with more flexibility than 1969 paper suggested
```

## Homer's Wine-Dark Sea

```
ANCIENT GREEK COLOR VOCABULARY:

Homer (~8th century BCE) uses these color terms for the sea:
  οἶνοψ (oinops) — "wine-colored" or "wine-faced"
  πορφύρεος (porphyreos) — "purple" or "dark, gleaming"
  κυάνεος (kyaneos) — "dark, gleaming" (also used for metal, sky at night)
  χλωρός (chloros) — "green" but also applied to honey, sand, blood
    (clearly not a color-specific term; means "fresh, living")

NOTICEABLY ABSENT: a simple word for "blue"
  GLADSTONE (William Ewart Gladstone, PM of Britain, 1858):
  "Studies on Homer and the Homeric Age"
  Noticed: Homer never calls the sky blue
           Calls honey "green" (chloron meli)
           Has only ~5 color terms; extensive darkness/brightness vocabulary
  GLADSTONE'S CONCLUSION (wrong): Ancient Greeks were partially color-blind
  → Rejected because eye anatomy doesn't change in 3,000 years

LAZARUS GEIGER (1880):
  Surveyed ancient texts across many languages:
  Sanskrit, Chinese, Hebrew, Icelandic: none have blue color terms
  Blue appears in languages only after metalwork + dye technology developed
  → The color term follows the MATERIAL
  → Egyptian had a blue word (irtyu) because they made Egyptian blue (first synthetic, 3000 BCE)
  → Other ancient cultures: no stable blue materials → no blue term

MODERN CONSENSUS (supported by neuroscience):
  Speakers with no blue color term can still DISCRIMINATE blue from green
  (finer perceptual discrimination, not categorical)
  But: when asked to group chips into categories, they group blue with green
  Himba tribe (Namibia, studied extensively today):
  • No distinct word for blue vs green
  • Significantly slower to identify blue chip among green chips
  • Significantly faster to identify a slightly different green among greens
    (where English-speakers all see "same green")
  → Language shapes the SPEED and EASE of color discrimination, not absolute ability
  → This is the weak Sapir-Whorf effect, empirically confirmed for color
```

## Linguistic Relativity and Color (Sapir-Whorf)

```
SAPIR-WHORF HYPOTHESIS:
  Strong version: Language DETERMINES thought (linguistic determinism)
  → You cannot think thoughts your language has no words for
  → REJECTED for color: Himba can still see and distinguish blue
  → Generally rejected in strong form

  Weak version: Language INFLUENCES thought (linguistic relativity)
  → Your language makes some thoughts/discriminations easier or harder
  → Empirically SUPPORTED for color

EXPERIMENTAL EVIDENCE (weak Sapir-Whorf for color):

RUSSIAN BLUE DISCRIMINATION (Winawer et al., 2007):
  Russian has two mandatory basic blue terms:
  siniy (тёмно-синий) = dark blue (navy)
  goluboy (голубой) = light blue (sky blue)
  → Categorical perception: discrimination faster when colors straddle siniy/goluboy boundary
  → English speakers: no advantage at that boundary (it's all just "blue")
  CRITICAL TEST: gave participants verbal task simultaneously
  → Russian speakers lost their advantage (verbal task disrupted language-based category)
  → Visual task only: no difference
  → The advantage is MEDIATED BY LANGUAGE activation

GREEK DISCRIMINATION (Greek has ghalazio + ble for light/dark blue):
  Similar results to Russian study
  Left visual field (right hemisphere, less language) shows weaker effect
  Right visual field (left hemisphere, language dominant) shows stronger effect
  → Language lateralization fingerprint on color perception

HIMBA STUDY (Roberson et al.):
  Himba has no separate word for blue vs green (one term: buru)
  Himba has multiple distinct green terms where English has one
  Result:
  • Himba faster at distinguishing greens that English calls "same"
  • Himba slower at distinguishing blue from green (same category for them)
  → Categorical advantage follows language, not physics of the stimulus
```

## Focal Colors and Category Boundaries

```
FOCAL COLOR CONCEPT:
  Within any color category, some exemplars are more "central" than others
  A canonical red that everyone agrees is "good red"
  → These focal/best examples are MORE UNIVERSAL than category boundaries

  IMPLICATION:
  "Where does red end and orange begin?" → highly variable across languages
  "What is the best example of red?" → surprisingly consistent
  → Category centers are universal; edges are culturally variable

  EVOLUTIONARY HYPOTHESIS (Berlin-Kay + later work):
  Basic colors correspond to salient environmental features:
  Blood, fire → red (universally salient, survival-relevant)
  Sky → blue (appears after linguistic technology reaches that point)
  Plants → green (many plants → many green terms, but boundary differs)

  WHY NO LANGUAGE HAS "BROWN" BEFORE "GREEN":
  Brown is always late in the sequence (Stage VI)
  Hypothesis: brown is a desaturated, low-luminance orange
  → Requires the orange/yellow/red region to be already established
  → Brown is cognitively derived from ochre-yellowish-dark continuum
```

## The Origin of Color Names

```
ORANGE (the color):
  Old English: "geoluread" (yellow-red) — no word for orange
  Arabic "naranj" → Portuguese "laranja" → Spanish "naranja" → French "orange"
  → Named after the fruit (itself named from Sanskrit/Persian)
  English borrowed "orange" as color name from the fruit name (~16th century)
  → "Before oranges were known in Europe, orange-colored things were called yellow-red"
  → Etymology: Sanskrit nāranga → Persia nārang → Arabic nāranj → European languages

MAUVE (the color):
  1856: William Henry Perkin accidentally synthesizes the dye
  Names it "Tyrian Purple" / "mauveine" (after the French mallow flower — mauve)
  Queen Victoria wore mauve to the Great Exhibition 1862 → fashion epidemic
  Color name "mauve" = coined with the substance
  → This is the pattern: color name follows the material

MAGENTA:
  Dye: synthesized 1859 (Verguin, France)
  Named after: Battle of Magenta (June 4, 1859) — French/Piedmontese defeat of Austria
  The city of Magenta (Lombardy) → battle → dye named in same month
  One of history's great coincidences: major battle and major dye in same location/month

INDIGO:
  Plant origin: "Indikon" (Greek) → "Indicum" (Latin) → "indigo" (English)
  Literally: "Indian dye" (from India)
  Newton added indigo to his seven-color spectrum partly to reach 7 (match music's 7 notes)
  Many colorists: blue → indigo → violet is not three distinct categories; indigo is blue

VERMILION:
  Latin: vermiculus (little worm) → the kermes insect → crimson → then applied to HgS
  The same word used for insect-crimson (organic dye) and mercury sulfide pigment
  → Etymology preserves the color jump across different substances

CHARTREUSE:
  French: Chartreuse monastery near Grenoble → Chartreuse liqueur (since 1605)
  The liqueur: yellow-green herbal digestif
  Color chartreuse: borrowed from the liqueur → yellow-green to green-yellow
  Amusing: "chartreuse" means different color to different people
  Chartreuse yellow = more yellow; chartreuse green = more green

KHAKI:
  Urdu/Hindi: khāki = "dusty, dust-colored" (from Persian khāk = dust/soil)
  British Indian Army → British Army → World War I → brown-tan uniforms
  → The color name traveled with the garment

TEAL:
  Named after the Eurasian teal duck (Anas crecca)
  Blue-green of the wing patch → the color "teal"
  → Animal color naming (same as "scarlet" from scarlet ibis, "buff" from buffalo hide)
```

## Modern Color Naming: The XKCD Survey

```
RANDALL MUNROE (XKCD), 2010:
  Online color naming survey: show random RGB color → type its name
  Results: 222,500 survey responses
  Produced "XKCD color map": crowdsourced RGB-to-name mapping

FINDINGS:
  • English speakers applied ~2,000 distinct color names
  • Most popular: red, green, blue, purple, pink — highly consistent
  • Specific colors have high agreement
  • Large swaths of color space labeled "ugly" (olive, puce, mustard)
  • Gendered patterns: "mauve", "fuchsia", "lavender" used more by women;
    basic terms more evenly distributed
  • Blue: the most consistently named single color

NAMES FROM THE SURVEY:
  Real names that appeared: "booger green," "ugly pink," "baby poop brown"
  "Cerulean" correctly used for blue-green (educated respondents)
  "Periwinkle" for blue-purple: consistent and frequently used
  → Natural language color naming is rich but not precise

WEB COLORS (HTML/CSS 140 named colors):
  16 original VGA colors (1987) + later additions
  All have hex codes: "cornflowerblue" = #6495ED, "rebeccapurple" = #663399
  "Rebeccapurple": named in CSS specification to honor Rebecca Meyer (Eric Meyer's daughter)
  → The only person to be memorialized in CSS spec (she died at age 6; her father's grief
    shared publicly → CSS authors honored her with a named color at her father's request)
```

## Why Blue Came Last (Across Unrelated Cultures)

```
CONVERGENT EVIDENCE:
  Ancient Sanskrit: many color terms, no blue
  Classical Chinese: no basic blue term (until 青 qīng was differentiated from 绿 lǜ)
  Classical Hebrew: same situation
  Ancient Icelandic: no blue term
  → All unrelated language families; all acquired blue late

THE MATERIAL EXPLANATION (Geiger + modern synthesis):
  Step 1: Red — blood, fire, red ochre — universally available materials
  Step 2: Yellow/Green — plant-based (weld, woad), readily available
  Step 3: Blue — requires:
           • Lapis lazuli (expensive, geographically limited)
           • Azurite (unstable, not widespread)
           • Woad/Indigo (requires fermentation vat technology)
           • Egyptian Blue (requires copper + silica + lime firing — advanced technology)
  → Blue color materials required significant technological/economic development
  → Once societies had blue dyes/pigments, the term followed

THE CAUSATION DIRECTION:
  Not: "they had no blue term, so they didn't notice blue"
  But: "there were no salient, stable blue things in everyday life →
       no term developed → linguistic category absent"
  When blue materials became available → term followed within generations

  Egyptian case (exception): had stable blue material (Egyptian blue, ~3000 BCE) →
  had blue color term early (irtyu) → this supports the material-first explanation
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| What is Berlin-Kay's universal color naming sequence? | Black/white → red → yellow or green → the other → blue → brown → pink/orange/purple/gray |
| What did Homer's wine-dark sea reveal? | Ancient Greek had no basic blue term. Gladstone noticed Homer never uses blue for the sky. Geiger identified this across multiple ancient language families — blue term follows available blue materials. |
| Does lacking a blue word mean you can't see blue? | No — Himba people can discriminate blue from green. But they're slower at it and group them together in categorization tasks. Language affects ease of discrimination, not raw sensitivity. |
| Why did "orange" not exist as an English color name before ~1500? | The fruit arrived from Asia; the color borrowed the fruit's name. Before then: "yellow-red." The color name follows the naming of a familiar object of that color. |
| What is the Sapir-Whorf hypothesis (color version)? | Weak form confirmed: language shapes categorical perception speed. Russian speakers discriminate siniy/goluboy blues faster than English speakers at the siniy/goluboy boundary. Strong form (can't perceive what you can't name) rejected. |
| What is a "focal color"? | The best example of a color category — more consistent across cultures than category boundaries. Everyone agrees on "the best red" before they agree on where red ends. |

## Common Confusion Points

**Berlin-Kay is not deterministic:** The 1969 paper overstated universality. Languages show more flexibility than the strict 7-stage hierarchy implies. But the broad ordering (basic chromatic terms follow that sequence) has held up across larger, more careful studies.

**"Wine-dark sea" doesn't mean the sea was wine-colored:** Homer perceived a quality of color/sheen using a wine analogy. He was NOT saying the sea was maroon. The Greek term refers to a dark, gleaming, dynamic quality that could be applied to the sea. This is a different kind of color vocabulary — not hue-based but quality-based.

**Language effects are real but small:** The Sapir-Whorf effects on color are robust but modest. Russian speakers are faster at the siniy/goluboy boundary by ~100 ms in reaction time. This is statistically significant and scientifically interesting but does not mean English speakers fundamentally cannot perceive fine blue distinctions.

**"Chartreuse" confusion is not a mistake:** Chartreuse refers to both "chartreuse yellow" (more yellow) and "chartreuse green" (more green). Both are legitimate uses — the yellow and green Chartreuse liqueurs give different color anchors. In the US, "chartreuse" more often means yellow-green; in France/Europe, more often the green.
