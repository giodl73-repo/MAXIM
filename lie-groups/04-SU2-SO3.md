# SU(2) and SO(3) — Double Cover, Spinors, Angular Momentum

## The Big Picture

SU(2) and SO(3) have the same Lie algebra but are different groups. SU(2) is a 2-to-1 cover
of SO(3). The two sheets correspond to integer-spin vs half-integer-spin representations. This
is not a mathematical technicality — it is the reason fermions exist.

```
+----------------------------------------------------------------------+
|             THE SU(2) -> SO(3) DOUBLE COVER                          |
+----------------------------------------------------------------------+
|                                                                      |
|   SU(2) = S^3 ⊂ C^2                  SO(3) = RP^3                  |
|   unit quaternions                     rotation matrices             |
|   dim = 3, simply connected            pi_1 = Z/2Z                  |
|   center = {I, -I} = Z_2              center = {I}                  |
|                                                                      |
|         phi: SU(2) --2-to-1--> SO(3)                                |
|              {A, -A}  |-------> same rotation                        |
|                                                                      |
|   REPRESENTATIONS:                                                   |
|   SU(2) irreps:  j = 0, 1/2, 1, 3/2, 2, ...  (all half-integers)   |
|   SO(3) irreps:  j = 0, 1, 2, ...             (integers only)       |
|                                                                      |
|   Half-integer j reps: lift to SU(2) but NOT to SO(3)               |
|   Integer j reps: descend to SO(3)                                   |
|                                                                      |
|   SPINORS = the j=1/2 representation of SU(2)                       |
|   A 2pi rotation sends a spinor to its negative.                     |
|   A 4pi rotation is the identity.                                    |
|                                                                      |
+----------------------------------------------------------------------+
```

## SU(2) Explicitly

SU(2) = { [[a, -conj(b)], [b, conj(a)]] : a,b in C, |a|^2 + |b|^2 = 1 }

Writing a = x_0 + i x_3 and b = x_2 + i x_1:
```
SU(2) = { x_0 I + x_1 (i sigma_1) + x_2 (i sigma_2) + x_3 (i sigma_3) : x_0^2+x_1^2+x_2^2+x_3^2 = 1}
```
This is the unit sphere S^3 in R^4. So as a smooth manifold, SU(2) iso S^3.

**The Pauli matrices** (sigma_i) are:
```
sigma_1 = [[0,1],[1,0]]    sigma_2 = [[0,-i],[i,0]]    sigma_3 = [[1,0],[0,-1]]

Properties:
  sigma_i^2 = I
  sigma_i sigma_j = delta_{ij} I + i epsilon_{ijk} sigma_k
  [sigma_i/2, sigma_j/2] = i epsilon_{ijk} sigma_k/2
```

The Lie algebra su(2) = span_R{i sigma_1/2, i sigma_2/2, i sigma_3/2} with bracket
[i sigma_a/2, i sigma_b/2] = epsilon_{abc} (i sigma_c/2). Isomorphic to so(3).

---

## SO(3) Explicitly

SO(3) = 3x3 real orthogonal matrices with det = +1. Every element is a rotation:
```
About z-axis by angle theta:
R_z(theta) = [[cos theta, -sin theta, 0],
              [sin theta,  cos theta, 0],
              [0,          0,         1]]

General rotation by angle theta about unit axis n:
R(n,theta) = I + sin(theta) N + (1-cos theta) N^2    (Rodrigues' formula)
where N = [[0,-n_3,n_2],[n_3,0,-n_1],[-n_2,n_1,0]] is the cross-product matrix.
```

**Topology:** SO(3) iso RP^3. Every rotation is specified by (axis n, angle theta) with theta in [0,pi]
and the identification (n, pi) ~ (-n, pi). The fundamental group pi_1(SO(3)) = Z/2Z — the nontrivial
loop is a 2pi rotation. A 2pi rotation is NOT homotopic to the identity; a 4pi rotation IS.

This is demonstrable physically with the **Dirac belt trick** (or Philippine wine dance): attach
one end of a belt to a cup, make a 2pi rotation, and the belt is twisted with no untwisting possible
without further rotation. Make a 4pi rotation and you can untwist.

## The Double Cover Map phi: SU(2) -> SO(3)

**Construction:** SU(2) acts on the 3-dimensional real vector space:
```
V = span_R{i sigma_1, i sigma_2, i sigma_3} iso R^3
```
by conjugation: X |-> A X A^{-1} for A in SU(2), X in V.

Since conjugation by A preserves V (check: conjugate of a traceless skew-Hermitian matrix is
traceless skew-Hermitian) and preserves the inner product B(X,Y) = -tr(XY)/2 on V, each A
gives a rotation of R^3.

**The map:** phi(A)(n_i sigma_i) = A (n_i sigma_i) A^{-1}

**Kernel:** phi(A) = I iff A commutes with all sigma_i. Since {sigma_i} generates all of M_2(C)
as an algebra (they span the traceless Hermitian matrices), A must commute with all of M_2(C),
hence A = lambda I. Since A in SU(2): lambda^2 = 1, lambda = pm 1. So ker(phi) = {I, -I} = Z_2.

Therefore: SO(3) iso SU(2) / {I, -I} = SU(2) / Z_2.

## Why This Means Spinors Exist

**Representations must respect the kernel:** A representation rho: SO(3) -> GL(V) pulls back to
rho.phi: SU(2) -> GL(V) via the covering map. For this to be a well-defined SO(3) rep, it must
satisfy rho(phi(A)) = rho(phi(-A)), i.e., rho.phi(A) = rho.phi(-A), i.e., rho.phi(-I) = I.

The SU(2) element -I acts on the spin-j representation V_j by:
```
rho_j(-I) = (-1)^{2j} I
```
(because -I = exp(2pi i * J_3 / j) up to the right normalization — a full 2pi rotation sends
|j,m> to e^{2pi i m} |j,m> = |j,m> for integer m and -|j,m> for half-integer m).

**Consequence:**
- j in {0,1,2,...}: rho_j(-I) = I. The representation descends to SO(3). These are "tensors."
- j in {1/2, 3/2, 5/2,...}: rho_j(-I) = -I. Does NOT descend to SO(3). These are "spinors."

```
INTEGER SPIN (tensors):
  2pi rotation = identity
  These are representations of SO(3)
  Examples: scalars (j=0), vectors (j=1), symmetric tensors (j=2)

HALF-INTEGER SPIN (spinors):
  2pi rotation = multiplication by -1
  These are representations of SU(2) but NOT SO(3)
  Examples: electrons (j=1/2), protons (j=1/2), quarks (j=1/2)
```

Spinors are not a quirk of quantum mechanics — they are forced by the topology of SO(3).
The topological obstruction is pi_1(SO(3)) = Z/2Z: the double cover SU(2) exists because of this
fundamental group. Representations of the cover that do not factor through the base are spinorial.

## Quaternions and SU(2)

The unit quaternions are the group:
```
Sp(1) = { a + bi + cj + dk : a^2+b^2+c^2+d^2 = 1 }  (the 3-sphere S^3)
```
with quaternion multiplication: i^2 = j^2 = k^2 = ijk = -1.

**Identification with SU(2):**
```
a + bi + cj + dk  <-->  [[ a+di,  b+ci],
                          [-b+ci,  a-di]]

i.e., 1 <-> I,   i <-> i sigma_3,   j <-> i sigma_2,   k <-> i sigma_1
```
Under this identification, Sp(1) iso SU(2).

**Quaternion double cover of SO(3):** For unit quaternion q = cos(theta/2) + sin(theta/2)(n_1 i + n_2 j + n_3 k),
the map v |-> q v conj(q) (quaternion conjugation) for v in span{i,j,k} iso R^3 gives rotation by
angle theta about axis (n_1, n_2, n_3). This is exactly the double cover phi.

**Why quaternions make SU(2) easy to work with:**
- Composition of rotations = quaternion multiplication (no gimbal lock)
- Interpolation: SLERP (spherical linear interpolation on S^3)
- Numerical stability for cumulative rotations
- Used in robotics, game engines, 3D graphics everywhere

## Angular Momentum in Quantum Mechanics

The connection to physics: rotational symmetry of physical laws implies the Hamiltonian commutes
with the generators of rotations. By Noether's theorem, these generators are conserved — they are
the angular momentum operators.

**Angular momentum algebra:** The operators L_x, L_y, L_z (orbital), S_x, S_y, S_z (spin),
and J_i = L_i + S_i (total) satisfy:
```
[J_i, J_j] = i hbar epsilon_{ijk} J_k
```
This is the su(2) algebra (over R) with structure constants epsilon_{ijk}. The irreducible
representations of this algebra are the V_j (spin-j reps), with:
- j integer: orbital angular momentum states, describable in terms of spherical harmonics Y^m_l
- j half-integer: intrinsic spin, no classical analog

**Quantization of j:** The requirement that exp(2pi i J_3 / hbar) = I (a 2pi rotation is the identity
for a bosonic system, or = -I for fermionic) exactly selects integer or half-integer j.
The experimental fact that electrons (fermions) have j=1/2 reflects the topological fact that
SU(2) double covers SO(3).

**Spherical harmonics as SO(3) representations:** The functions Y^m_l(theta, phi) for fixed l and
m = -l,...,l transform under rotations as the (2l+1)-dimensional irrep V_l of SO(3) (integer l
only, because they are functions on S^2 = SO(3)/SO(2)).

## The SL(2,C) Double Cover of SO(3,1)

The same structure appears for the Lorentz group SO(3,1):

```
Spin group:       SL(2,C) --2-to-1--> SO^+(3,1)  (proper orthochronous Lorentz)
pi_1(SO^+(3,1)) = Z/2Z

Representations of SL(2,C): labeled by (j, j') with j, j' in {0, 1/2, 1, ...}
  (j,0):   left-handed Weyl spinors (neutrinos)
  (0,j'):  right-handed Weyl spinors
  (1/2,1/2): 4-vectors (spin-1)
  (1,0)+(0,1): antisymmetric rank-2 tensor (electromagnetic field strength F_mu nu)
```

The full spinor story of quantum field theory is the Lorentz-group version of SU(2) -> SO(3).

## Explicitly: The Spinor (j=1/2) Representation

The fundamental 2-dim representation of SU(2). A "spinor" is an element of C^2 on which SU(2)
acts by matrix multiplication.

```
State space: C^2 with basis |up> = [[1],[0]], |down> = [[0],[1]]

SU(2) action:  A in SU(2) acts by standard 2x2 matrix multiplication.

Rotation by angle theta about z-axis:
  A = exp(i theta sigma_3 / 2) = [[e^{i theta/2}, 0], [0, e^{-i theta/2}]]

Action on |up>:  A|up> = e^{i theta/2} |up>   (phase factor)
Action on |dn>:  A|dn> = e^{-i theta/2}|dn>

Under 2pi rotation (theta = 2pi):
  A = [[e^{i pi}, 0], [0, e^{-i pi}]] = [[-1,0],[0,-1]] = -I
  A|up> = -|up>,  A|dn> = -|dn>    <-- overall -1
```

A full 360-degree rotation of the coordinate frame multiplies a spinor by -1. This is not a
global sign — it is relative: the relative phase between a spinor and a tensor (scalar, vector)
changes by pi under a 2pi rotation. This relative phase has measurable consequences in neutron
interferometry experiments (confirmed experimentally in 1975, Rauch et al.).

## Decision Cheat Sheet

| Concept | SU(2) | SO(3) |
|---------|-------|-------|
| Topology | S^3, simply connected | RP^3, pi_1=Z/2Z |
| Center | {I,-I} = Z_2 | {I} |
| Irreps | j = 0, 1/2, 1, 3/2, ... | j = 0, 1, 2, ... (integer only) |
| Dimension of irrep j | 2j+1 | 2j+1 (for integer j) |
| Under 2pi rotation | -I acts as (-1)^{2j} | identity (all reps) |
| Fundamental rep | C^2 (spinors) | R^3 (vectors) |
| Quaternion model | unit quaternions | unit quaternions / {pm 1} |

## Common Confusion Points

**"-I acts as -1 on spin-1/2":** Yes. A 2pi rotation is -I in SU(2). Acting on the spin-1/2 rep
(C^2), -I acts as the scalar -1. This is a well-defined linear action, just with eigenvalue -1.
The state |psi> and -|psi> represent the same physical state in quantum mechanics (rays, not
vectors, are physical), so this has no observational consequence for a single particle. But in
interference between spin-1/2 and spin-0 states (or in relative phases), the -1 matters.

**"Spinors are complex 2-vectors":** In the SU(2) context, yes — elements of C^2. More precisely,
a Dirac spinor is a C^4 object combining both SU(2) representations. In the Lorentz context,
Weyl spinors (C^2) carry either left or right chirality; Dirac spinors (C^4) carry both.

**"SU(2) and SO(3) have the same Lie algebra so they're the same":** Same Lie algebra, different
global topology. The local structure (multiplication near identity) is the same; the global
structure (how elements in different parts of the group are related, what loops are contractible)
differs. The difference shows up exactly in the half-integer representations.

**"The double cover is a fiction":** No. The double cover arises from the nontrivial fundamental
group. It is required by the mathematics of covering spaces. The physical question is: which
objects in nature transform under SU(2) vs SO(3)? The answer from experiment: matter fields
(quarks, leptons) are spinors (SU(2)); force-mediating bosons are vectors/tensors (SO(3)-compatible).
