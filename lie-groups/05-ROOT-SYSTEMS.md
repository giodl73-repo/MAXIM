# Root Systems and Weight Lattices

## The Big Picture

Root systems encode the structure of semisimple Lie algebras in a finite combinatorial object:
a finite set of vectors in Euclidean space satisfying reflection symmetry conditions. The
classification of root systems is equivalent to the classification of semisimple Lie algebras.

```
+----------------------------------------------------------------------+
|                ROOT SYSTEMS: THE STRUCTURE                           |
+----------------------------------------------------------------------+
|                                                                      |
|  Start: semisimple Lie algebra g, Cartan subalgebra h               |
|                                                                      |
|  g = h  (+)  sum_{alpha in Phi} g_alpha                             |
|       |               |                                              |
|    Cartan        root spaces                                         |
|    subalgebra    (1-dim each, for simple g)                         |
|                                                                      |
|  Root system Phi subset h* = dual of h:                             |
|  - Finite set of nonzero vectors                                     |
|  - Spans h* (as a vector space)                                      |
|  - If alpha in Phi: -alpha in Phi (alpha and -alpha come in pairs)   |
|  - Reflection symmetry: s_alpha(Phi) = Phi for each alpha           |
|  - Integrality: <alpha, beta_check> in Z for all alpha,beta in Phi  |
|                                                                      |
|  KEY DECOMPOSITION:                                                  |
|  Phi = Phi_+ union Phi_-  (positive and negative roots)             |
|  Phi_+ = {alpha_1, ..., alpha_n} contains simple roots Pi            |
|  Every positive root = non-negative integer combination of Pi        |
|                                                                      |
+----------------------------------------------------------------------+
```

## The Cartan Decomposition

Let g be a complex semisimple Lie algebra. A **Cartan subalgebra** h is a maximal abelian
subalgebra consisting of semisimple (diagonalizable) elements.

For matrix algebras: h is the diagonal matrices.

**Root space decomposition:**
Since h is abelian and acts on g by the adjoint representation ad: h -> End(g), the
simultaneous eigenspaces are well-defined.

For each linear functional alpha: h -> C (an element of h*), define:
```
g_alpha = { X in g : [H, X] = alpha(H) X  for all H in h }
```

**Theorem:** g decomposes as:
```
g = h  (+)  sum_{alpha in Phi} g_alpha
```
where Phi = { alpha in h* \ {0} : g_alpha != 0 } is the **root system**, and dim(g_alpha) = 1
for each alpha in Phi when g is simple.

**Example for sl(2):** h = span{H}, roots are {alpha, -alpha} where alpha(H) = 2.
  g = h (+) g_alpha (+) g_{-alpha}  (the triple H, E, F from 03-REPRESENTATIONS.md).

**Example for sl(3):** h = diagonal traceless matrices (2-dimensional). 6 roots:
```
e_1-e_2, e_2-e_1, e_1-e_3, e_3-e_1, e_2-e_3, e_3-e_2
where (e_i-e_j)(diag(a_1,a_2,a_3)) = a_i - a_j
```
Positive roots: e_1-e_2, e_1-e_3, e_2-e_3. Simple roots: alpha_1 = e_1-e_2, alpha_2 = e_2-e_3.
This is the A_2 root system (SU(3) case).

## Root System Axioms

An abstract root system is a finite subset Phi of a Euclidean space (V, <.,..>) satisfying:

**Axiom 1 (Spanning):** Phi spans V as a real vector space.

**Axiom 2 (No scaling):** The only multiples of alpha in Phi are pm alpha.
  (Some sources allow 2-element Phi to be a "non-reduced" root system, but we take reduced.)

**Axiom 3 (Reflection closure):** For each alpha in Phi, the reflection:
```
s_alpha: V -> V,   s_alpha(beta) = beta - <beta, alpha_check> alpha
where alpha_check = 2*alpha / <alpha,alpha>  (the coroot)
```
maps Phi to itself: s_alpha(Phi) = Phi.

**Axiom 4 (Integrality):** For all alpha, beta in Phi:
```
<beta, alpha_check> = 2*<beta,alpha>/<alpha,alpha> in Z
```

The **Weyl group** W is the group generated by all reflections s_alpha for alpha in Phi.
It is a finite group (since Phi is finite and W permutes Phi).

## Simple Roots and the Cartan Matrix

A **simple root system** or **base** Pi = {alpha_1, ..., alpha_r} subset Phi is a subset such that:
1. Pi is a basis for V (r = rank = dim V)
2. Every root is either a non-negative or non-positive integer combination of Pi

```
Example: A_2 (sl(3) root system)
  Roots: 6 vectors in R^2 at 60-degree intervals
  Simple roots: alpha_1, alpha_2  (angle between them = 120 degrees)
  Positive roots: alpha_1, alpha_2, alpha_1+alpha_2
  Negative roots: -alpha_1, -alpha_2, -(alpha_1+alpha_2)

  +------> alpha_1
  |  . alpha_1+alpha_2
  |       .
  v alpha_2  .
```

**Cartan matrix:** A = (a_{ij}) where a_{ij} = <alpha_i, alpha_j_check> = 2<alpha_i,alpha_j>/<alpha_j,alpha_j>.

The Cartan matrix encodes all the structure of the root system:
- Diagonal entries a_{ii} = 2 always
- Off-diagonal entries a_{ij} in {0,-1,-2,-3} for i != j (for irreducible root systems)
- a_{ij} * a_{ji} in {0,1,2,3}

**Example Cartan matrices:**

A_2 (SU(3)):           B_2 (SO(5)/Sp(4)):       G_2:
  [[ 2, -1],             [[ 2, -1],               [[ 2, -1],
   [-1,  2]]              [-2,  2]]                [-3,  2]]

Note: the Cartan matrix is not symmetric (for non-simply-laced root systems), but A_T is related
to A by conjugation by the diagonal matrix of root lengths squared.

## Dynkin Diagrams

A Dynkin diagram is a graphical encoding of the Cartan matrix:
- One node per simple root
- Nodes i and j are connected by a_{ij} * a_{ji} = 0,1,2,3 lines
- For double or triple bonds, an arrow points from the longer root to the shorter root

```
A_n (n nodes, single bonds):
  o---o---o--- ... ---o

B_n (n nodes, last bond double with arrow to right):
  o---o---o--- ... ---o==>o

C_n (n nodes, last bond double with arrow to left):
  o---o---o--- ... ---o<==o

D_n (n nodes, last node forks):
  o---o---o--- ... ---o-<o
                          \o

G_2 (2 nodes, triple bond):
  o≡≡>o

F_4 (4 nodes, one double bond):
  o---o==>o---o

E_6, E_7, E_8 (Y-shaped):
  E_6:        o
              |
  o---o---o---o---o

  E_7:        o
              |
  o---o---o---o---o---o

  E_8:            o
                  |
  o---o---o---o---o---o---o
```

## The Classification Result

The connected Dynkin diagrams (equivalently, the simple root systems) are exactly:

| Series | Diagram | Corresponding group | Rank | Notes |
|--------|---------|---------------------|------|-------|
| A_n | o-o-...-o (n nodes) | SU(n+1) | n | n >= 1 |
| B_n | o-o-...-o==>o | SO(2n+1) | n | n >= 2 |
| C_n | o-o-...-o<==o | Sp(2n) | n | n >= 3 |
| D_n | o-o-...-o fork | SO(2n) | n | n >= 4 |
| G_2 | o≡≡>o | Exceptional | 2 | rank 2 |
| F_4 | o-o==>o-o | Exceptional | 4 | rank 4 |
| E_6 | Y-shape, 6 | Exceptional | 6 | |
| E_7 | Y-shape, 7 | Exceptional | 7 | |
| E_8 | Y-shape, 8 | Exceptional | 8 | |

This is the complete list. No others exist. The proof is a beautiful induction using the
allowed Cartan matrix entry products {0,1,2,3} and the fact that the Cartan matrix must be
positive definite.

## The Weyl Group

The Weyl group W is the group generated by reflections s_{alpha_i} for simple roots alpha_i.

**Properties:**
- W is finite
- W acts on h* preserving Phi (permutes the roots)
- |W| can be computed from the root system

| Root system | Weyl group W | |W| |
|-------------|-------------|-----|
| A_n | Symmetric group S_{n+1} | (n+1)! |
| B_n, C_n | (Z/2Z)^n semi-direct S_n | 2^n n! |
| D_n | (Z/2Z)^{n-1} semi-direct S_n | 2^{n-1} n! |
| G_2 | Dihedral D_6 | 12 |
| F_4 | — | 1152 |
| E_6 | — | 51840 |
| E_7 | — | 2903040 |
| E_8 | — | 696729600 |

The Weyl group of E_8 has order ~700 million. The E_8 lattice appears in 8-dimensional
sphere packing (densest known in 8 dimensions) and in string theory compactifications.

## Weight Lattices

**Weight lattice:** For a Lie algebra g with Cartan subalgebra h and simple roots Pi = {alpha_i},
the weight lattice is:
```
P = { lambda in h* : <lambda, alpha_i_check> in Z  for all i }
```
The weight lattice contains the root lattice Q = Z-span(Phi) as a subgroup.

**Fundamental weights:** omega_i defined by <omega_i, alpha_j_check> = delta_{ij}.
They form a basis for P. The dominant weights are:
```
P_+ = { lambda in P : <lambda, alpha_i_check> >= 0  for all simple roots alpha_i }
    = { sum_i n_i omega_i : n_i in Z_{>=0} }
```

**Dominant integral weights classify irreps:** By the highest weight theorem (03-REPRESENTATIONS.md),
irreducible representations of the simply-connected group G with Lie algebra g are in bijection
with P_+. The irrep V_lambda has highest weight lambda.

**For SU(2):**
- Simple root alpha, coroot alpha_check = alpha (normalized)
- Weight lattice P = Z * (alpha/2)
- Fundamental weight omega = alpha/2
- Dominant weights: P_+ = Z_{>=0} * omega, labeled by j = 0, 1/2, 1, 3/2, ...
- V_{j*omega} = spin-j representation (dim = 2j+1)

**For SU(3) (A_2):**
- Fundamental weights omega_1, omega_2
- Irreps labeled by (p,q) with p,q >= 0 (Dynkin labels)
- dim V_{(p,q)} = (p+1)(q+1)(p+q+2)/2
- V_{(1,0)} = fundamental 3-dim representation (quarks in SU(3) flavor symmetry)
- V_{(0,1)} = anti-fundamental (antiquarks)
- V_{(1,1)} = adjoint 8-dim representation (gluons in SU(3) color)

## The Highest Root and Affine Dynkin Diagrams

The **highest root** is the unique element of Phi_+ with maximal weight under the inner product.
Adding minus the highest root to Pi gives the **extended (affine) Dynkin diagram**:

```
A^(1)_2 (extended A_2):      o
                             / \
                            o   o
                             \ /
                              o

This corresponds to the affine Lie algebra sl(3)_hat, relevant to conformal field theory,
loop groups, and Kac-Moody algebras.
```

Affine Lie algebras are infinite-dimensional Lie algebras extending the simple ones by adding
a central element and a derivation. They appear in 2D CFT (Wess-Zumino-Witten models),
integrable systems, and string theory.

## Decision Cheat Sheet

| Concept | Definition | Key property |
|---------|-----------|-------------|
| Root alpha | Nonzero weight of adjoint rep | [H, E_alpha] = alpha(H) E_alpha |
| Simple root alpha_i | Basis for positive roots | Every root = +/- combo of simples |
| Coroot alpha_check | 2*alpha / <alpha,alpha> | <alpha, alpha_check> = 2 |
| Cartan matrix a_{ij} | <alpha_i, alpha_j_check> | Integer entries, a_{ii}=2 |
| Weyl group W | Generated by reflections s_{alpha_i} | Permutes Phi |
| Weight lattice P | {lambda : <lambda,alpha_i_check> in Z} | Contains root lattice Q |
| Fundamental weights omega_i | <omega_i, alpha_j_check> = delta_{ij} | Basis of P |
| Dominant weight lambda in P_+ | <lambda,alpha_i_check> >= 0 | Labels an irrep V_lambda |

## Common Confusion Points

**"Root vs weight":** Roots are weights of the adjoint representation (special case). Weights
are eigenvalues of the Cartan subalgebra action in any representation. The root lattice Q is
a sublattice of the weight lattice P; the quotient P/Q is related to the center of the simply
connected group (it is Z_{n+1} for A_n).

**"Simple roots vs positive roots":** Simple roots form a basis (r vectors). Positive roots are
all roots expressible as non-negative combinations of simple roots. For A_n with n nodes: r simple
roots, n(n+1)/2 positive roots (total |Phi| = n(n+1)).

**"B_n and C_n have the same Dynkin diagram shape but are dual":** B_n and C_n have the same
Dynkin diagram structure but with the arrow direction reversed. B_n (SO(2n+1)) has short roots
at the end; C_n (Sp(2n)) has long roots at the end. The Langlands dual of B_n is C_n and vice
versa. This matters in the geometric Langlands program.

**"D_4 triality":** D_4 (SO(8)) has an exceptionally symmetric Dynkin diagram (three legs of
equal length plus the central node). This gives the outer automorphism group S_3 of D_4, called
triality. Triality is responsible for the existence of three 8-dimensional representations of
SO(8) that are permuted by the outer automorphism: the vector, spinor, and conjugate spinor.
It also appears in string theory (Green-Schwarz superstring).
