# Planetary Interiors
## Differentiation, Structure, Seismology, Magnetic Fields, Tidal Heating

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Planetary differentiation: heat sources (accretion, ²⁶Al, ⁴⁰K, tidal), iron-silicate separation
- Interior structure: core / mantle / crust layers; PREM model for Earth
- Seismology: P, S, surface waves; shadow zones; probing core-mantle boundary
- Earth's liquid outer core and solid inner core: Lehmann discontinuity
- Geodynamo: convection + Coriolis → magnetic field; polarity reversals
- Mantle convection: plate tectonics, hotspots, subduction
- Comparative planetology: Moon (solidified), Mars (dead dynamo), Venus (no plates), Mercury (large core)
- Tidal heating: Io interior, Europa subsurface ocean
- Exoplanet interiors: mass-radius relations, super-Earth structure
