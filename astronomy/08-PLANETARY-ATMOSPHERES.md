# Planetary Atmospheres
## Structure, Escape, Greenhouse Effect, Circulation, Comparative Climatology

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Atmospheric structure: troposphere / stratosphere / mesosphere / thermosphere; scale height H = kT/mg
- Atmospheric escape: Jeans escape (thermal), non-thermal (sputtering, photodissociation), hydrodynamic escape
- Greenhouse effect: radiative transfer, optical depth, forcing vs feedback; Venus runaway vs Earth stable
- Atmospheric circulation: Hadley / Ferrel / Polar cells; Coriolis deflection; jet streams; ITCZ
- Comparative: Venus (CO₂, 92 bar, 465°C), Earth (N₂/O₂, 1 bar), Mars (CO₂, 6 mbar, frozen), Titan (N₂/CH₄, 1.5 bar, hydrocarbon rain)
- Gas giant atmospheres: zonal jets, Great Red Spot dynamics, cloud deck structure
- Atmospheric biosignatures: O₂ + CH₄ disequilibrium; JWST transmission spectra
- Venus-Earth-Mars divergence: initial conditions or feedback bifurcation?
