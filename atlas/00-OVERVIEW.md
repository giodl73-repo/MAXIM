## Master Glyph Legend

One symbol vocabulary across all 52 maps. Learn it once, read everything.

| Glyph | Meaning | Notes |
|-------|---------|-------|
| `в–І` | Mountain, volcano, high terrain | Density shows extent of range |
| `~` `вүҲ` | River, flowing water | `вүҲ` also used in ocean labels |
| `в–‘` | Desert, arid zone | Density shows extent |
| `в—Ӣ` | Lake, enclosed water body | |
| `в•‘` | Mid-ocean ridge, rift | Tectonic maps |
| `в•ұ` `в•І` `в”Ӯ` `в”Җ` | Coastline, boundary | Shape-defining lines |
| `В·` | Interior land (light fill) | NOT cities, NOT stars |
| `вҳ…` | Navigation star (primary) | Celestial maps only |
| `вҶ‘` `вҶ“` `вҶ’` `вҶҗ` | Flow, wind, current direction | Mechanism diagrams |
| `в”ңв”Җв”Җ N km в”Җв”Җв”Ө` | Scale bar | Below every geographic map |
