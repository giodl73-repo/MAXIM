## Soil Formation — The Five Factors

```
WHAT MAKES SOIL — JENNY'S EQUATION (1941)

  S = f(cl, o, r, p, t)

  S  = soil properties
  cl = CLIMATE         → temperature + moisture drive weathering
  o  = ORGANISMS       → roots, microbes, earthworms, fungi
  r  = RELIEF (slope)  → steep = erosion; flat = accumulation
  p  = PARENT MATERIAL → granite, basite, limestone, loess, alluvium
  t  = TIME            → young soils are thin; old soils are deep

  ┌─────────────────────────────────────────────────────────┐
  │  BEDROCK (parent material)                              │
  │  ════════════════════════════════════════════════════   │
  │     ↑ physical weathering (frost, roots, gravity)       │
  │     ↑ chemical weathering (water, acids, oxidation)     │
  │  ════════════════════════════════════════════════════   │
  │     ↑ biological activity (decomposition, mixing)       │
  │     ↑ leaching (water carries solutes downward)         │
  │  ════════════════════════════════════════════════════   │
  │  SOIL (living, layered, structured)                     │
  │  ← takes 500-1,000 years to form 2.5 cm of topsoil →   │
  └─────────────────────────────────────────────────────────┘

```

## Soil Horizons — The Universal Profile

```
SOIL PROFILE — CROSS-SECTION

  ┌──────────────────────────────────────────┐  ← surface
  │  O HORIZON — organic layer               │  leaf litter, humus
  │  (dark, spongy, 0-30 cm)                │  decomposing matter
  ├──────────────────────────────────────────┤
  │  A HORIZON — topsoil                     │  mineral + organic mix
  │  (dark, 10-50 cm)                        │  roots, worms, microbes
  │  ★ THIS IS THE LAYER THAT FEEDS US ★    │  highest fertility
  ├──────────────────────────────────────────┤
  │  B HORIZON — subsoil                     │  accumulated clays,
  │  (lighter, 30-100+ cm)                   │  iron oxides, minerals
  │  leached from above                      │  washed down from A
  ├──────────────────────────────────────────┤
  │  C HORIZON — weathered parent material   │  broken rock fragments
  │  (50-200+ cm)                            │  little biological
  │                                          │  activity
  ├──────────────────────────────────────────┤
  │  R HORIZON — bedrock                     │  unweathered rock
  │  (depth varies)                          │
  └──────────────────────────────────────────┘

```

## Major Soil Orders by Latitude

```
GLOBAL SOIL DISTRIBUTION — LATITUDE BANDS

  90N ┌─────────────────────────────────────────────────────┐
      │  GELISOLS — permafrost soils                        │
      │  Frozen year-round below surface. Thawing releases  │
      │  methane. Found: Siberia, Alaska, N. Canada.        │
  60N ├─────────────────────────────────────────────────────┤
      │  SPODOSOLS — acidic conifer forest soils            │
      │  Leached, sandy, acidic. Boreal forests (taiga).    │
      │  Poor for crops without heavy amendment.            │
      │  Found: Scandinavia, N. Russia, N. Canada.          │
  50N ├─────────────────────────────────────────────────────┤
      │  ★ MOLLISOLS — grassland soils (BEST FOR FARMING) ★ │
      │  Deep, dark, organic-rich topsoil. Prairie grasses  │
      │  built them over 10,000+ years. Naturally fertile.  │
      │  Found: US Great Plains, Ukrainian steppe, Pampas.  │
      │  → World's breadbaskets sit on mollisols.           │
      ├─────────────────────────────────────────────────────┤
      │  ALFISOLS — temperate deciduous forest soils        │
      │  Moderate fertility, good with management.          │
      │  Found: E. US, W. Europe, E. China.                 │
  30N ├─────────────────────────────────────────────────────┤
      │  ARIDISOLS — desert soils                           │
      │  Thin, mineral-rich but no organic matter. Dry.     │
      │  Can be productive WITH irrigation (fragile).       │
      │  Found: Sahara, Arabian, Sonoran, Gobi, Outback.   │
  15N ├─────────────────────────────────────────────────────┤
      │  OXISOLS — tropical rainforest soils                │
      │  Deeply weathered, RED, nutrient-poor (leached).    │
      │  All nutrients are in the living biomass, not soil. │
      │  Cut the forest → nutrients wash away → laterite.   │
      │  Found: Amazon, Congo, SE Asia.                     │
   EQ ├─────────────────────────────────────────────────────┤
      │  (Mirror: Oxisols → Aridisols → Alfisols → etc.)   │
  30S ├─────────────────────────────────────────────────────┤
      │  ★ MOLLISOLS again — Pampas, S. Africa grasslands ★ │
  45S └─────────────────────────────────────────────────────┘

```

## Soil Fertility Comparison

| Soil Order | Fertility | Where | Key Crops | Limitation |
|------------|-----------|-------|-----------|------------|
| **Mollisol** | ★★★★★ | Prairies, steppes, pampas | Wheat, corn, soy | None inherent — best soil on Earth |
| **Alfisol** | ★★★★ | Temperate forests | Wheat, corn, diverse | Needs organic amendment |
| **Andisol** | ★★★★ | Volcanic regions | Coffee, tea, diverse | Limited area (volcanic only) |
| **Vertisol** | ★★★ | Tropical/subtropical clay | Cotton, sorghum | Shrink/swell cracks — hard to work |
| **Ultisol** | ★★★ | Subtropical forests | Corn, tobacco, pine | Acidic, needs lime |
| **Inceptisol** | ★★ | Young, diverse | Varies | Undeveloped — needs time |
| **Aridisol** | ★★ | Deserts | Anything — WITH water | No water, no crops. Irrigation = salinization risk |
| **Oxisol** | ★ | Tropical rainforest | Slash-and-burn only | Nutrients in biomass, not soil. Deforest → laterite |
| **Spodosol** | ★ | Boreal forests | Potatoes, hardy grains | Acidic, leached, cold |
| **Gelisol** | — | Permafrost | Nothing (frozen) | Ground is ice. Thawing = structural collapse |

## The Tropical Soil Trap

One of the most important and counterintuitive facts in agriculture:

```
TROPICAL RAINFOREST ≠ FERTILE SOIL

  WHAT IT LOOKS LIKE:                WHAT'S ACTUALLY HAPPENING:
  ┌─────────────────────┐            ┌─────────────────────┐
  │  (dense canopy)    │            │  Nutrients are in    │
  │  Dense, lush,       │            │  the BIOMASS (trees, │
  │  incredibly         │            │  leaves, roots),     │
  │  productive forest  │            │  NOT in the soil.    │
  │                     │            │                      │
  │  SOIL BELOW:        │            │  Heavy rain leaches  │
  │  ┌─────────────────┐│            │  minerals constantly │
  │  │ OXISOL: deep,   ││            │  Rapid decomposition │
  │  │ red, NUTRIENT-  ││            │  → immediate uptake  │
  │  │ POOR laterite   ││            │  by roots. Nothing   │
  │  │ (iron/aluminum  ││            │  accumulates in soil │
  │  │  oxides)        ││            │                      │
  │  └─────────────────┘│            └─────────────────────┘
  └─────────────────────┘

  CUT THE FOREST:
  ┌─────────────────────┐
  │  (stumps)           │  Year 1-2: decent crops (stored nutrients)
  │  ┌─────────────────┐│  Year 3-5: declining yields
  │  │ Rain washes away││  Year 5+:  LATERITE — brick-hard iron
  │  │ everything.     ││            crust. Nothing grows.
  │  │ Soil bakes into ││            Permanent damage.
  │  │ hard red crust. ││
  │  └─────────────────┘│  This is why slash-and-burn farming
  └─────────────────────┘  ROTATES plots every 2-3 years.

  The Amazon, Congo, and SE Asian forests look fertile.
  The fertility is in the forest, not the ground.

```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| Best soil on Earth? | Mollisol (chernozem) — Ukrainian steppe, US Great Plains, Pampas |
| Can you farm tropical rainforest? | Short-term yes, long-term no. Soil turns to laterite. |
| Why are deserts infertile? | No water → no organic matter → no nutrient cycling |
| Can you irrigate desert soil? | Yes, but salinization risk. Evaporation deposits salt. |
| Fastest soil to destroy? | Loess (wind-deposited silt) — erodes catastrophically |
| Where are the most people fed? | Indo-Gangetic Plain (alluvial), N. China Plain (loess) |
| What's chernozem? | "Black earth" — the richest mollisol. Ukraine and S. Russia. |
| How long to make topsoil? | 500-1,000 years per inch. Lose it in one bad season. |
| Best volcanic soil? | Andisol — Java, Central America, East Africa highlands |
| What kills soil fastest? | Erosion (wind/water), salinization, compaction, paving |
