# Plant Propagation: Seeds, Cuttings, Division

## The Big Picture

Plant propagation is the process of creating new plants from existing ones. Every method involves a fundamental trade-off between genetic diversity and clonal fidelity:

```
PROPAGATION SPECTRUM:

  SEXUAL (seed)              ASEXUAL (clonal)
  +------------------+       +------------------+
  | Genetic variation|       | Identical to     |
  | (meiosis mixes   |       | parent plant     |
  |  alleles)        |       | (mitosis only)   |
  | New combinations |       | Preserves variety|
  | Dormancy systems |       | No dormancy in   |
  | Long-term storage|       | most methods     |
  | Slow to first    |       | Faster maturity  |
  | fruiting         |       | in most cases    |
  +------------------+       +------------------+
        |                          |
        v                          v
  Annuals, vegetables,       Fruit trees, named
  most field crops           cultivars, vegetative
  (desirable: F1 vigor)      ornamentals, potatoes

  THE FUNDAMENTAL RULE:
  Any named cultivar (variety) must be propagated clonally
  to maintain its characteristics. "Red Delicious apple trees
  from seed" do not produce Red Delicious apples — they produce
  seedlings with random genetics.
```

## Seed Biology

### Seed Structure

```
SEED ANATOMY:
  +----------------------------------+
  |  SEED COAT (testa)               |  Protection; may be hard (impermeable)
  |  +----------------------------+  |
  |  | ENDOSPERM (storage tissue) |  |  Starch, proteins, oils — food reserve
  |  |  +---------------------+  |  |
  |  |  |     EMBRYO          |  |  |
  |  |  | Radicle (root tip)  |  |  |
  |  |  | Plumule (shoot tip) |  |  |
  |  |  | Cotyledon(s)        |  |  |  Seed leaves — often storage + absorption
  |  |  +---------------------+  |  |
  |  +----------------------------+  |
  +----------------------------------+
  Monocots: 1 cotyledon (grasses, onions, palms)
  Dicots: 2 cotyledons (most vegetables, trees, shrubs)
```

### Seed Dormancy Mechanisms

Seeds don't germinate on demand — dormancy is an adaptive mechanism preventing germination under temporary favorable conditions:

```
DORMANCY TYPE     MECHANISM                HORTICULTURAL TREATMENT
──────────────────────────────────────────────────────────────────────
Physical          Impermeable seed coat    Scarification: mechanical
(e.g. Gleditsia,  blocks water uptake      (abrasion), hot water soak,
Lotus, Robinia)   and gas exchange         or acid (H₂SO₄) treatment

Physiological     Inhibitor compounds      Stratification: moist chilling
(e.g. Apple,      prevent germination      at 1–5°C for 1–3 months
Peach, Rose)      until winter-passed       (simulates winter naturally)
                                           Gibberellin (GA₃) can substitute

Morphological     Embryo underdeveloped    Warm moist stratification;
(e.g. Carrot,     at seed shed             extended germination time
Orchids)

Double dormancy   Both physical and         Sequential treatments:
(e.g. Hawthorn,   physiological            Warm stratification then
Viburnum)         dormancy present          cold stratification

Thermodormancy    Germination blocked       Cool soil temperatures;
(e.g. Lettuce)    at high temperatures      or gibberellin
──────────────────────────────────────────────────────────────────────
```

### Seed Vigor and Quality Testing

```
GERMINATION PERCENTAGE: % of seeds that germinate under optimal conditions.
  Seed lot standard: >80% for most vegetables; >85% for grasses.
  Below standard: either use more seed or source better stock.

TETRAZOLIUM (TZ) TEST:
  2,3,5-triphenyltetrazolium chloride enters living cells.
  In living tissue: reacts with dehydrogenase enzymes → red stain.
  Dead tissue: no enzyme activity → unstained.
  Assessment: embryo viability map in 24 hours.
  Advantage over germination test: 24h vs. 7–21 days.
  Used for: expensive seed lots, pre-season inventory.

SEED VIGOR (beyond germination %):
  Rate and uniformity of germination.
  Performance under suboptimal conditions (cold soil, dry, deep sowing).
  Low-vigor seed germinates if conditions are perfect; fails in field.
  ISTA and AOSA standards: cold test, accelerated aging test.

SEED STORAGE:
  Harrington's Rule: for every 1% decrease in moisture content
    above ~14%, seed life doubles.
    For every 10°F decrease in storage temperature, seed life doubles.
  Dry + cool = long storage. Most vegetable seed: 5–10 years at 40°F, <40% RH.
  Exception: some seeds are recalcitrant (cannot be dried):
    Coffee, cocoa, avocado — store fresh only; short storage life.
```

## Stem Cuttings

Stem cuttings exploit the ability of stems to produce adventitious roots (roots from non-root tissue):

```
CUTTING TYPE    TIMING              MATERIAL STATUS     EXAMPLES
──────────────────────────────────────────────────────────────────────────
Softwood        Spring–early summer  Actively growing,   Chrysanthemum,
                                     soft and flexible   Impatiens, Salvia,
                                                         Coleus, Basil

Semi-hardwood   Mid-summer to fall   Partially matured,  Camellia, Holly,
                                     base firmer but     Boxwood,
                                     tips still soft     Euonymus

Hardwood        Dormant season       Fully mature,       Roses (some),
                (winter)             dormant, woody      Willow, Grape,
                                                         Poplar, Forsythia
──────────────────────────────────────────────────────────────────────────
```

### Rooting Mechanism

```
ADVENTITIOUS ROOT FORMATION STEPS:
  1. Wounding (optional but accelerating): cut surface forms wound callus
     Physical: cutting edge, mechanical scraping of stem base
     Chemical: IBA (indole-3-butyric acid) application

  2. Dedifferentiation: cells at wound site revert to meristematic state
     Requires: auxin (internally produced or applied externally)
     Inhibited by: cytokinin (high cytokinin blocks rooting)

  3. Root initiation: cells differentiate into root initial cells

  4. Root elongation: root primordia grow out through stem cortex

  5. Emergence: roots exit the stem; cutting now functional

  TIMELINE: 10–21 days for easy-to-root species (willow, impatiens)
            4–12 weeks for difficult species (holly, conifers)

IBA (INDOLE-3-BUTYRIC ACID):
  The auxin used universally for cutting propagation.
  More stable than IAA (the natural auxin), persists in rooting medium.
  Forms: powder (talc carrier, 1,000–30,000ppm), solution (quick-dip 1,000–10,000ppm)
  Mechanism: stimulates cell dedifferentiation; promotes root initiation
  More IBA ≠ better: above optimum concentration, phytotoxic.
```

### Mist Propagation Systems

```
MIST BENCH PRINCIPLE:
  Problem: cuttings have no roots — cannot absorb water.
           But they lose water through stomata and cuticle.
           Solution: reduce water demand (shade, cool) and
           intermittently wet cutting surfaces.

  INTERMITTENT MIST:
  Mist nozzle wets cutting foliage → thin water film on leaves.
  Film evaporates → leaf cools → transpiration demand drops.
  Timer or electronic leaf ("artificial leaf") triggers mist cycles.
  Frequency: every 30–90 seconds in bright sun; less in overcast.

  ELECTRONIC LEAF:
  Resistive sensor that dries as water film evaporates.
  When dry enough: triggers mist cycle.
  Better than timer: adjusts to actual evaporation rate.

  MEDIA:
  Well-drained, low-nutrient: perlite, coarse grit, or 50:50 perlite:peat.
  No fertility — roots are stimulated by nutrient stress.
  High fertility actually inhibits rooting.
```

## Other Asexual Methods

### Leaf Cuttings

```
LEAF CUTTINGS:
  Begonia rex: entire leaf on soil surface, veins cut → plantlets
               at cut points. Exploits undifferentiated cells at veins.

  Sansevieria (snake plant): leaf sections 5-7cm, inserted upright.
               Produces shoots at base. Note: variegated types lose
               variegation from leaf cuttings — must propagate by division.

  Succulent leaves (Sedum, Echeveria): whole leaf; allowed to dry
               1–2 days; laid on surface. Adventitious plantlet from base.

  African violet (Saintpaulia): petiole cutting (leaf + leaf stalk).
               Petiole inserted at angle; plantlets form at petiole base.
               Standard commercial method for African violet production.
```

### Layering

```
LAYERING TYPES:
  SIMPLE LAYERING:
  Bend a low branch to soil; wound at bend; pin to soil; cover.
  Develops roots at wound while still attached to parent.
  When rooted (weeks-months): sever; transplant.
  Easy, low skill. Used for: forsythia, rhododendron, magnolia.

  AIR LAYERING:
  Cannot bend the branch to soil? Apply the soil to the branch.
  Wound stem; apply moist sphagnum moss; wrap in polyethylene film.
  Moisture + darkness + wound = rooting in wrapped medium.
  Sever when roots visible through film.
  Used for: rubber plant (Ficus elastica), Magnolia, fruit trees.

  MOUND LAYERING (stool layering):
  Mound soil over multi-stemmed base in spring.
  Shoots rooted in the mound by fall.
  Remove rooted shoots; replant.
  Used for: apple/quince rootstock production, gooseberry.

  STOLONS (runner) — natural layering:
  Strawberry runners: stem grows horizontally, produces plantlet at node.
  The plantlet touches soil; roots form; becomes independent.
  Commercial strawberry production: parent plants produce runners
  which are collected as transplants for new fruiting plantings.
```

### Division

```
DIVISION:
  For clump-forming herbaceous perennials.
  Dig up clump; physically separate into sections (each with roots + shoots).
  Replant sections.
  Purpose: rejuvenation (old clump center often woody/dead);
           multiplication; variety maintenance.

  TIMING: Best when plant is dormant or in early growth.
          Fall division: plant establishes over winter.
          Spring division: plant establishes before summer heat.

  SPECIES EXAMPLES:
  Hostas, daylilies, irises, ornamental grasses, phlox, asters.
```

### Bulb Propagation

```
BULB STRUCTURES:
  True bulb: compressed stem axis; fleshy scale leaves.
    Examples: tulip, narcissus (daffodil), onion, hyacinth, lily.
  Corm: solid stem storage tissue; no scales.
    Examples: gladiolus, crocus.
  Tuber: swollen storage root or stem.
    Examples: potato (stem tuber), dahlia (root tuber).
  Rhizome: horizontal stem at/below soil surface.
    Examples: iris, canna, ginger.

NARCISSUS TWIN-SCALING:
  High-tech bulb propagation for valuable cultivars.
  Cut bulb into 8-16 segments; each segment cut into twin-scales
  (two scale sections still attached to basal plate fragment).
  Treat with fungicide; place in plastic bag with moist perlite.
  Incubate 18-23°C, 8-12 weeks.
  Bulblets form at basal plate junction.
  Dramatically faster than natural offset production.
  Used for: disease-free stock, expensive cultivars.

POTATO:
  Stem tuber; each "eye" is an axillary bud.
  Certified seed potato: virus-tested, disease-free.
  Commercial production cuts seed potatoes to 40-50g pieces,
  each with at least one eye. Treated with fungicide; plant direct.
  Minitubers: disease-free microplants from tissue culture → grown
  one season to produce seed potato stock. Disease-free base.
```

## Why Clonal Propagation Matters

```
THE VARIETY PRESERVATION ARGUMENT:

  "Honeycrisp apple" is a specific clone: University of Minnesota
  apple breeding program, released 1991. It has a specific genetic
  constitution producing its characteristic flavor and texture.

  Grow Honeycrisp from seed: Mendelian genetics — genes segregate.
  None of the seedlings will be Honeycrisp.
  They will be random combinations of Honeycrisp's parents' genes.

  The variety can only be perpetuated by clonal propagation:
    Budding on rootstock → every tree is genetically Honeycrisp.
    The commercial Honeycrisp industry consists of millions
    of trees that are all clones of the original Minnesota selection.

PRECOCITY:
  A seedling apple tree takes 7–12 years to first fruit.
  A grafted tree on dwarfing rootstock: 2–4 years.
  Commercial economics require precocity.

UNIFORMITY:
  Supermarket buyers require color uniformity, size uniformity,
  flavor consistency. Clonal propagation ensures this.
  Seedling populations vary widely.
```

---

The engineering analogy is direct: clonal propagation is binary replication — copy the bits, get an identical artifact. Sexual reproduction via seed is recompiling from source with nondeterminism — the source (genome) is similar but recombination, mutation, and environmental influence during development produce a phenotype that cannot be bit-for-bit predicted. When the commercial requirement is uniform output (supermarket color standards, wine vintage consistency, pharmaceutical raw material standardization), clonal propagation is the only solution. Seed propagation is appropriate when genetic diversity is the goal, cost is the constraint, or you need large numbers with acceptable variance.

## Decision Cheat Sheet

| Propagation Method | When to Use | Key Requirement | Approximate Time to Transplant |
|-------------------|-------------|-----------------|-------------------------------|
| Seed (direct) | Annuals, vegetables, wildflowers | Dormancy treatment if needed | 3–12 weeks to transplant stage |
| Softwood cutting | Spring; easy-rooting species | Mist bench; IBA | 2–4 weeks |
| Semi-hardwood cutting | Summer-fall; woody ornamentals | IBA; mist or high humidity tent | 4–12 weeks |
| Hardwood cutting | Winter dormancy; deciduous woody | Cold storage or direct field insert | 2–6 months |
| Simple layering | Single branches to ground | Patience | 1–6 months while still attached |
| Air layering | Branches too high to ground-layer | Sphagnum + film | 6–12 weeks after wrapping |
| Division | Clump-forming herbaceous perennials | Timing (dormant/early growth) | Immediate |
| Bulb scaling | High-value bulb cultivars | Clean technique; fungicide | 1–2 seasons to flowering bulb |

## Common Confusion Points

**Stratification vs. scarification are opposite problems**: stratification (cold-moist treatment) breaks physiological dormancy in embryo. Scarification (physical or chemical seed coat abrasion) breaks physical dormancy. Many people confuse the terms — they address entirely different dormancy mechanisms.

**IBA is not a fertilizer**: applying auxin-based rooting hormone to cuttings in high-fertility media often reduces rooting. Low nutrient stress promotes root initiation; high fertility promotes shoot growth instead.

**Vegetative propagation of viruses**: cuttings, grafts, and division all transmit systemic viruses present in the parent plant. Tissue culture (meristem culture) is the only propagation method that can eliminate systemic viruses. This is why certified virus-free stock programs exist for strawberry, potato, and ornamental bulbs.

**"Running from seed" doesn't produce the parent variety**: any named cultivar (apple variety, rose, most ornamentals) must be propagated vegetatively. Seed from a 'Gravenstein' apple produces a 'Gravenstein' × some pollen donor hybrid — it will not be 'Gravenstein'. This is true even for heirloom open-pollinated vegetables that are "grown from saved seed" — only if they were not cross-pollinated with another variety.

**Dormancy is not the same as death**: a dormant seed or bulb with zero visible activity may be fully viable. The TZ test and germination tests distinguish viability from dormancy. Discarding dormant material as dead is a common error.
