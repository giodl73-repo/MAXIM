# 04 — Organizational Design

## Mintzberg Configurations, Matrix vs Hierarchy, Coordination Mechanisms

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Mintzberg's five configurations: Simple Structure (small/centralized), Machine Bureaucracy (standardized processes/Taylor's dream), Professional Bureaucracy (skilled workers/standardized skills), Divisionalized Form (autonomous divisions/market grouping), Adhocracy (project-based/mutual adjustment); plus Missionary (ideology-coordinated) and Political (conflict)
- Coordination mechanisms: mutual adjustment (informal), direct supervision, standardization of (work processes / outputs / skills / norms) — each configuration emphasizes different mechanisms
- Structural dimensions: specialization (horizontal/vertical), formalization, centralization/decentralization, unit grouping (function/product/geography/matrix/front-back)
- Functional vs divisional vs matrix: functional (efficiency, depth, silo problems), divisional (customer/product focus, redundancy), matrix (dual reporting lines, coordination cost, conflict); when each wins
- Span of control: flat (wide spans, autonomy, information overload) vs tall (narrow spans, control, slow decisions); Graicunas formula; modern flattening trends
- Conway's Law: system architecture mirrors communication structure; inverse Conway maneuver (design org to get desired architecture); team topologies (stream-aligned/enabling/complicated-subsystem/platform teams)
- Bureaucracy paradox: Weber's rational-legal authority as efficiency mechanism → rigidity → organizational pathology; adhocracy as overcorrection
- Org size and structure: Greiner growth model (5 crises: leadership/autonomy/control/red tape/growth), how structure must evolve as headcount scales
- Network organizations: outsourcing/partnerships, platform ecosystems as org form, managing across boundaries without authority
- Redesign process: diagnosis, design principles, transition management, unintended consequences
