## Section 1: Primate Taxonomy — The Phylogenetic Framework

### Strepsirrhini vs. Haplorhini

```
  STREPSIRRHINI ("wet-nosed"):
  - Moist rhinarium (like cats/dogs) — enhanced olfaction
  - Tapetum lucidum: reflective layer behind retina (eyeshine)
  - Dental comb: front lower teeth form a comb for grooming
  - More reliant on smell, less reliant on vision than anthropoids
  - Examples: ring-tailed lemur, aye-aye, slow loris, galago

  HAPLORHINI ("dry-nosed"):
  - Dry nose, no rhinarium
  - No tapetum (most species)
  - Retinal fovea: high-resolution central vision
  - Color vision elaborated (especially trichromacy)
  - Tarsiers: phylogenetically with haplorhines but retain some
    strepsirrhine features (the phylogenetic oddity)
```

### New World vs. Old World Monkeys

```
  PLATYRRHINI (New World):
  ┌──────────────────────────────────────────────────────────┐
  │ Flat nose, nostrils facing sideways                      │
  │ Prehensile tail in some species (spider, howler,         │
  │   capuchin) — NEW WORLD INNOVATION: no OW monkey has     │
  │   true prehensile tail                                   │
  │ COLOR VISION: polymorphic trichromacy                    │
  │   - Males: always dichromats (2-color)                   │
  │   - Females: either dichromats or trichromats (3-color)  │
  │   Explains: females better at discriminating ripe fruit; │
  │   males better at breaking camouflage of predators/prey  │
  │ Examples: Squirrel monkeys, Capuchins, Marmosets,        │
  │   Tamarins, Spider monkeys, Howlers                      │
  └──────────────────────────────────────────────────────────┘

  CATARRHINI (Old World — "narrow-nosed"):
  ┌──────────────────────────────────────────────────────────┐
  │ Narrow nose, nostrils facing downward                    │
  │ 32 teeth (2.1.2.3 formula, like humans)                 │
  │ ALL FEMALES ARE TRICHROMATS — consistent color vision    │
  │ No prehensile tail                                       │
  │ Ischial callosities: sitting pads (in cercopithecoids)  │
  │ Divided into: Cercopithecoidea (OW monkeys) +            │
  │               Hominoidea (apes including humans)         │
  └──────────────────────────────────────────────────────────┘
```

### The Ape Clade

```
  LESSER APES (Hylobatidae):
  Gibbons and siamangs — Southeast Asia
  Pair-bonded, territorial, brachiation specialists
  Sing elaborate duets to advertise territory and pair bond
  Diverged from great apes ~20 Ma

  GREAT APES (Hominidae):
  +----------+--------------------------------------------+
  | Orangutan| Borneo/Sumatra. Solitary. Slow life        |
  |          | history. Complex cognition. Nest-builders.  |
  |          | 97% DNA shared with humans.                |
  +----------+--------------------------------------------+
  | Gorilla  | Africa (W and E). One-male groups.         |
  |          | Silverback dominance. Knuckle-walking.     |
  |          | 98.4% DNA shared with humans.              |
  +----------+--------------------------------------------+
  | Chimp    | Pan troglodytes. Fission-fusion society.   |
  |          | Tool use. Hunting. Male coalitional        |
  |          | aggression. Lethal inter-community raids.  |
  |          | 98.7% DNA shared with humans.              |
  +----------+--------------------------------------------+
  | Bonobo   | Pan paniscus. Congo Basin only, south of  |
  |          | Congo River. Female co-dominance. Reduced  |
  |          | inter-group aggression. Sexual conflict    |
  |          | resolution. Same DNA distance as chimp.    |
  +----------+--------------------------------------------+
  | Human    | Global. Obligate bipedal. Language.        |
  |          | Extreme encephalization. Cumulative culture.|
  +----------+--------------------------------------------+

  Chimp and bonobo diverged from each other ~2 Ma.
  Human-chimp/bonobo split: ~6-7 Ma.
  Same genome distance (98.7%) — two completely different
  social structures from the same DNA.
```

## Section 2: Social Organization

### Types and Their Ecological Drivers

```
  SOCIAL SYSTEM     EXAMPLES              ECOLOGY
  ─────────────────────────────────────────────────────────
  Solitary          Orangutan             Dispersed fruits; low
                    Tarsiers              food density; large
                                         range; no group benefit

  Pair bond         Gibbons, titis        Territory defense by
                    (some)                pair; biparental care;
                                         infanticide protection

  One-male          Gorilla               Female defense by
  multi-female      Hamadryas baboon      dominant male; female
  (harem)           Gelada                clumped resources

  Multi-male        Most baboons,         Female choice; sperm
  multi-female      macaques, chimps      competition; male
                    (some)                coalition value

  Fission-fusion    Chimpanzee            Variable food patches;
                    Bonobo                flexibility in group size
                    Spider monkey         depending on resources;
                    Human                 fluid associations
```

**The Emlen-Oring hypothesis** (1977): social organization is a product of
resource distribution and consequent female distribution, which then determines
male strategies. Food is clumped -> females clump -> males can defend a group
(harem). Food is dispersed -> females disperse -> males cannot defend a group,
must compete differently.

**Dunbar's social brain hypothesis**: neocortex size across primate species
predicts group size. Larger neocortex enables tracking more social relationships
(who's allied with whom, debt, reputation). Humans: predicted group size ~150
("Dunbar's number") — confirmed in hunter-gatherer bands, Hutterite communities,
typical company size before bureaucracy sets in.

## Section 3: Communication Systems

### Vervet Alarm Calls — Proto-Semantics

```
  Seyfarth, Cheney, and Marler (1980) — Amboseli, Kenya:

  Vervet monkeys (Chlorocebus pygerythrus) have at least
  3 functionally distinct alarm calls:

  ┌──────────────────┬──────────────────────────────────────┐
  │ CALL TYPE        │ APPROPRIATE RESPONSE                 │
  ├──────────────────┼──────────────────────────────────────┤
  │ Eagle alarm       │ Look up, move into bushes            │
  │ (raptor threat)   │                                      │
  ├──────────────────┼──────────────────────────────────────┤
  │ Leopard alarm     │ Run up into trees                    │
  │ (terrestrial      │                                      │
  │  predator)        │                                      │
  ├──────────────────┼──────────────────────────────────────┤
  │ Snake alarm       │ Stand up, look at ground around you  │
  │ (python/cobra)    │                                      │
  └──────────────────┴──────────────────────────────────────┘

  Key finding: playback experiments (tape-recorded calls
  without actual predators) trigger appropriate responses —
  the call itself, not the predator's presence, carries
  the "meaning."

  But is this LANGUAGE? No:
  - No syntax (calls not combined to generate new meanings)
  - No displacement (cannot refer to absent events)
  - No intentionality (learned, but probably not "intended"
    to communicate in the human sense)
  It is the best animal analog to referential communication,
  but structurally distant from human language.
```

### Great Ape Language Studies

```
  PROJECT WASHOE (Gardner & Gardner, 1966-1970):
  Chimpanzee raised with ASL (American Sign Language).
  Acquired ~250 signs, used them spontaneously and in novel
  combinations. Taught signs to her son (Loulis) without
  human instruction — first documented non-human cultural
  transmission of a human communication system.

  PROJECT NIM CHIMPSKY (Terrace, 1974-1977):
  Chimp learned ~125 signs. Terrace RETRACTED his optimism:
  careful analysis showed Nim's "novel" combinations were
  mostly imitation and prompted responses, not true
  generative syntax. The "language" claim was overstated.

  KANZI (Savage-Rumbaugh):
  Bonobo, used lexical keyboard (Yerkish symbols).
  Acquired language incidentally by watching his mother
  being trained (critical period effect?). Understands
  novel spoken English sentences (comparable to ~2.5-year-old
  child in comprehension). Most impressive case for
  language comprehension in non-human apes.

  CURRENT CONSENSUS:
  Great apes can learn lexical items (word-like associations).
  They show limited compositional use. They do NOT show
  recursive syntax. The human faculty for open-ended,
  hierarchically structured language (the "merge" operation
  in Chomsky's minimalism) is not demonstrated in apes.
  Whether this is a quantitative or qualitative difference
  is still debated.
```

## Section 4: Cognition

### Theory of Mind Debate

```
  Theory of mind: understanding that other individuals have
  beliefs, desires, and intentions different from your own.

  Premack and Woodruff (1978): "Does the chimpanzee have a
  theory of mind?" — coined the concept, proposed chimps do.

  FALSE-BELIEF TEST (Wimmer and Perner, 1983, adapted from
  Maxi task):
  1. Subject watches object hidden in location A
  2. Object moved to location B while agent is absent
  3. Subject asked: where will agent LOOK for object?
  Young children (~4 yr): they say A (correct attribution)
  Young children (~3 yr): they say B (fail — egocentric)

  GREAT APES:
  Multiple attempts, mostly failed. But:
  - Chimps track COMPETITIVE knowledge (what can rival see?)
    better than COOPERATIVE knowledge — they understand others'
    visual access when competing for food
  - 2016 study (Krupenye et al.): eye-tracking shows chimps
    anticipate where an agent with false belief will look —
    implicit understanding without verbal response
  - 2023 replication issues — the debate continues

  CALL AND TOMASELLO FRAMEWORK:
  Human infants uniquely: point to share attention (proto-
  declarative pointing). Apes point to request (proto-
  imperative) but do not point to SHARE. The "shared
  intentionality" for joint attention to third-party objects
  may be the specific human cognitive prerequisite for language
  and culture, not intelligence per se.
```

### Tool Use — From Goodall to Matsuzawa

```
  GOODALL'S DISCOVERY (Gombe, Tanzania, 1960):
  Chimpanzees strip leaves from grass stems to "fish" for
  termites in Macrotermes mounds. First non-human tool
  manufacture recorded in wild. When Leakey heard this:
  "Now we must redefine tool, redefine Man, or accept
  chimpanzees as humans."

  TYPES OF CHIMPANZEE TOOL USE:
  ┌────────────────────────────────────────────────────────┐
  │ Termite fishing: grass stem, bark strip probe          │
  │ Ant dipping: leaf-midrib whisk for driver ants         │
  │ Nut cracking (Bossou, Guinea — Matsuzawa):            │
  │   Stone hammer + stone anvil to crack Panda nuts       │
  │   The only documented spontaneous use of two stones    │
  │   as a compound tool set in the wild                   │
  │   Juveniles watch adults for years before succeeding   │
  │   (10+ years to master) — APPRENTICESHIP               │
  │ Leaf-sponge: leaves crumpled for water absorption      │
  │ Branch club: used as weapon/intimidation display       │
  │ Spear hunting: Fongoli (Pruetz/Bertolani 2007):        │
  │   Chimps sharpen sticks with teeth, use to probe       │
  │   into cavities for small primates. ~75% female use.  │
  └────────────────────────────────────────────────────────┘

  ARCHAEOLOGICAL RECORD OF CHIMPANZEE TOOL USE:
  Toth, Schick, and colleagues: nut-cracking sites in Ivory
  Coast with stone hammer/anvils dated to ~3,500 years BP.
  A non-human primate archaeological record.

  CAPUCHIN STONE TOOL ARCHAEOLOGY (Brazil, 2016):
  Wild bearded capuchins at Serra da Capivara NP use stone
  tools to crack nuts and process quartz. Sites date to
  ~3,000 years BP. Convergent evolution of stone tool use —
  completely independent of hominin/chimp lineage.
```

### Mirror Self-Recognition (MSR)

```
  GORDON GALLUP (1970):
  Applied rouge marks to anesthetized chimps' faces.
  Chimps with prior mirror experience touched the marks on
  their OWN faces when shown a mirror — evidence of
  self-concept (using mirror as information about own body).

  PASSES MSR: Great apes (chimp, bonobo, orangutan, gorilla
  inconsistently), dolphins, elephants, magpies, possibly
  orca, rays, fish (cleaner wrasse — debated).

  FAILS MSR: Monkeys (all tested), gibbons (consistently fail
  despite being apes), most non-human animals.

  WHAT MSR INDICATES: Self-awareness of body as object.
  NOT the same as "consciousness" or full theory of mind.
  It is a COMPONENT of the broader self-model that may
  be necessary but not sufficient for higher-order cognition.
```

## Section 5: Cultural Transmission

### The Whiten Multi-Site Study

```
  Whiten et al. (1999) — published in Nature:
  Surveyed 7 long-term chimpanzee field sites across Africa:
  Gombe, Mahale, Kibale, Bossou, Tai, Budongo, Bwindi

  FINDING: 39 behavioral variants occur at some but not all
  sites. After accounting for ecology and habitat differences,
  many variants remained site-specific — the best explanation
  is SOCIAL LEARNING and CULTURAL TRADITION.

  Examples:
  - Grooming hand-clasp: some sites yes, some no
  - Nut cracking with stone tools: only West Africa, never
    observed in East African chimps despite available nuts
    and suitable stones — a cultural "blank"
  - Termite fishing: probe style varies by site

  This established chimpanzee "cultures" in the full sense:
  behavioral traditions maintained by social learning,
  geographically variable, not genetically encoded.

  IMITATION vs. EMULATION:
  Chimps tend toward EMULATION: they learn the goal/result
  of an observed action, then find their own way to achieve it.
  Humans (especially children) tend toward IMITATION: they
  copy the exact action, even when a more efficient route exists.
  This "overimitation" in humans is not stupidity — it is
  efficient copying of cultural templates even when you don't
  understand why (the "copy the master" strategy).
```

## Section 6: Bonobo vs. Chimpanzee — Same Genome, Different Society

```
  Pan troglodytes (chimp) and Pan paniscus (bonobo):
  - ~2 Ma divergence (Congo River as barrier)
  - Same genetic distance to humans
  - Profoundly different social structures

  CHIMPANZEE:              BONOBO:
  Male dominant            Female co-dominant
  Male coalitional         Female alliances
  aggression               central
  Lethal inter-community   Inter-community
  raids documented         tolerance, even sex
  Hunting, meat eating     Less hunting
  Infanticide documented   Infanticide rarely
                           if ever documented
  Sexual swelling          More prolonged sexual
  at ovulation             activity (not just
  (estrus advertising)     at ovulation)
  Tension diffused by      Tension diffused by
  dominance displays,      sexual behavior
  aggression               (the bonobo solution)

  WHAT CHANGED:
  South of the Congo River, gorillas do not compete for
  terrestrial herbaceous vegetation (THV). North of the
  river, gorillas eat the same THV as chimps -> competition.
  With gorillas as competitors, chimps must form male
  coalitions to defend food sources. Without that pressure,
  bonobo females can afford to forage together and form
  alliances. Female alliances reshape everything.

  This illustrates how a single ecological difference
  can produce radically different social structures
  from nearly identical genomes.
```

     soy, logging), particularly Sumatra/Borneo and
     Congo Basin
  2. Bushmeat trade: hunting for food; apes are high-value
     targets; orphaned infants become illegal pets
  3. Disease: Ebola outbreaks devastate gorilla and chimp
     populations (Western/Central Africa). Respiratory
     viruses transmitted from humans (no immunity).
  4. Mining and resource extraction: roads open forest
     to hunting pressure
  5. Conflict zones: conservation impossible in active war

## Common Confusion Points

**"Monkeys are apes."**
No. Apes (Hominoidea) are a distinct clade within catarrhine primates. The
defining features: no tail, more flexible shoulder joint, larger brain/body
ratio. Gibbon = ape. Baboon = monkey. Common usage conflates them because
"monkey" is informally used for any primate that isn't human.

**"Chimps are our closest relatives."**
Technically chimp and bonobo are equidistant. The clade (Pan) is our closest
relative. The chimp-human split is ~6-7 Ma; the bonobo-chimp split is ~2 Ma.
So bonobos are MORE closely related to chimps than to us.

**"Chimps and bonobos are the same species."**
They are distinct species (Pan troglodytes vs. Pan paniscus) separated by the
Congo River ~2 Ma. They do not interbreed in the wild. Zoos have occasionally
housed them together inadvertently — they do sometimes have sexual contact but
offspring are not documented.

**"Gorillas are more primitive than chimps."**
Evolutionary primitiveness is not a real concept. Gorillas diverged from the
human/chimp/bonobo lineage earlier (~10 Ma), but they have continued evolving
on their own trajectory. A gorilla is not a "less evolved human."

**"Great ape language studies proved animals can talk."**
They proved apes can learn lexical associations and use them referentially in
context. They did NOT prove syntax, displacement (talking about absent things),
or the recursive hierarchical structure that defines human language. Terrace's
Nim Chimpsky retraction is important — early enthusiasm was based on insufficient
video analysis of imitation vs. spontaneous use.
