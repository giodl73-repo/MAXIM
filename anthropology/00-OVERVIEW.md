# Anthropology — Field Overview

## The Big Picture

```
+------------------------------------------------------------------+
|                     ANTHROPOLOGY                                  |
|                                                                   |
|  "The study of humans — all of them, everywhere, across all      |
|   time — as biological, cultural, linguistic, and social beings"  |
+------------------------------------------------------------------+
          |             |              |              |
          v             v              v              v
  +--------------+ +-----------+ +----------+ +----------+
  |  BIOLOGICAL  | | CULTURAL  | |  ARCHEO- | |LINGUISTIC|
  | /PHYSICAL    | |           | |  LOGICAL | |          |
  +--------------+ +-----------+ +----------+ +----------+
  | Human        | | Ethnogr-  | | Material | | Language |
  | evolution    | | aphy,     | | culture, | | structure|
  | Primatology  | | fieldwork | | past     | | & culture|
  | Forensics    | | Kinship   | | societies| | Cognition|
  | Variation    | | Religion  | | Dating   | | & meaning|
  +--------------+ +-----------+ +----------+ +----------+

  American "Four-Field" model (Boas tradition — dominant in US)

  British "Two-Field" model:  Social Anthropology + Archaeology
  (Cultural and linguistic collapsed into social; biological separate)
```

**The core commitment** across all four fields: **holism**. Understand humans as an
integrated whole — biology shapes culture shapes language shapes cognition, all
interacting. This is what distinguishes anthropology from sociology (groups only),
psychology (mind only), or history (text records only).

## Section 2: The Holism Principle

Why anthropology insists on holism — and why it matters:

```
+-------------------------------------------------------------------+
|                      HOLISM IN PRACTICE                            |
|                                                                   |
|   BIOLOGY  ------shapes------>  CULTURE                           |
|      ^                              |                             |
|      |                              v                             |
|   selection <---creates--- LANGUAGE / COGNITION                  |
|   pressure                                                        |
|                                                                   |
|   Example: Lactase persistence (canonical case)                   |
|   - Biology: LCT gene variant allows milk digestion in adults    |
|   - Culture: cattle herding created the selection pressure        |
|   - Language: vocabulary for dairy practices, social norms        |
|   - Cannot study any of these layers in isolation                |
|                                                                   |
|   Example: Color naming                                           |
|   - Berlin/Kay: 11 basic color terms appear cross-culturally     |
|     in a predictable sequence (universal biology of vision)       |
|   - BUT: Pirahã, Dani, others have fewer distinctions            |
|   - Cultural need drives elaboration of existing biology          |
+-------------------------------------------------------------------+
```

**Contrast with neighboring disciplines:**

| Question | Sociology | Psychology | History | Anthropology |
|----------|-----------|------------|---------|--------------|
| Unit of analysis | Groups/institutions | Individual mind | Events/texts | Humans holistically |
| Time scale | Present/recent | Present | Written record | Deep time + present |
| Cross-cultural | Often WEIRD-centric | Usually WEIRD | Culture-specific | Comparative |
| Biology | Largely excluded | Individual variation | Excluded | Integrated |
| Primary method | Surveys/statistics | Experiments | Archives | Ethnography |
| Non-literate societies | Rarely | Rarely | Impossible | Core focus |

**WEIRD** = Western, Educated, Industrialized, Rich, Democratic. Most psychology
and sociology research uses WEIRD subjects and generalizes to "humans." Anthropology
has consistently pushed back on this since Boas — and the 2010 Henrich/Heine/
Norenzayan paper gave it empirical teeth: WEIRD populations are often statistical
outliers on the very dimensions social science treats as universal.

## Section 3: Key Theoretical Frameworks

### Boas and Cultural Relativism

```
  The Boasian revolution (1900-1940):

  REJECTED: "Primitive" vs. "civilized" cultures on a hierarchy
  ASSERTED: Each culture must be understood on its own terms —
            its own history, environment, and internal logic

  Methodological relativism != moral relativism
  Understanding != endorsing

  The five students who reshaped American anthropology:
  +----------+--------------------------------------------------+
  | Kroeber  | Culture history, superorganic — culture has its  |
  |          | own logic irreducible to individual psychology    |
  +----------+--------------------------------------------------+
  | Sapir    | Language and culture intertwined;                |
  |          | Sapir-Whorf hypothesis (with Whorf)              |
  +----------+--------------------------------------------------+
  | Lowie    | Kinship, political organization, critique        |
  |          | of cultural evolutionism                         |
  +----------+--------------------------------------------------+
  | Benedict | "Patterns of Culture" — cultures have            |
  |          | integrated configurations (Apollonian/Dionysian) |
  +----------+--------------------------------------------------+
  | Mead     | Samoa and New Guinea studies — gender roles      |
  |          | culturally variable (later contested by Freeman) |
  +----------+--------------------------------------------------+
```

### Malinowski and Participant Observation

The methodological revolution: before Malinowski, anthropologists worked from
armchairs — reading missionaries' reports and travelers' accounts. Malinowski
lived in the Trobriand Islands (1915-1918), learned the language, participated
in daily life. The Kula ring exchange system was invisible to any outside observer
not embedded in it.

**Functionalism**: every cultural institution serves a function for the social
system. Kula exchange maintains alliance networks. Myth charters social arrangements.
Later critiqued: claiming everything "functions" is circular and cannot explain
change, conflict, or dysfunction.

### Levi-Strauss and Structuralism

```
  Core claim: all myths are transformations of each other.
  The "myth of myths" is the same story in different registers.

  Method: Find binary oppositions embedded in myths:
    nature / culture
    raw / cooked
    self / other
    high / low

  These reflect deep structures of the human mind —
  universal cognitive architecture, not cultural specifics.

  Example — Oedipus myth breakdown:
    Overrating blood relations  vs.  Underrating blood relations
    Autochthonous origin        vs.  Denial of autochthonous origin

  The myth is about a social contradiction that cannot be resolved
  logically — only mediated symbolically through narrative.

  Critique: Elegant but unfalsifiable. Any myth can be made to fit
  the schema. The "deep structure" is imposed by the analyst, not
  discovered in the data. Largely replaced by cognitive science.
```

### Geertz and Thick Description

```
  Thin description: "He winked."

  Thick description: "He contracted his right eyelid — not a twitch
  (involuntary), not a mock-wink (practicing), but a conspiratorial
  signal to a friend that the bystander was not to know about the
  plan. The meaning is public, conventional, and requires cultural
  competence to decode."

  Culture as "texts" — public, readable, interpretable.
  The anthropologist interprets, not measures.

  Balinese cockfight: on the surface, gambling.
  Thick description reveals it as: a story Balinese tell themselves
  about fate, status, masculinity, hierarchy, and the good life.

  Limits:
  - How do you know your interpretation is correct?
  - Post-Geertz critique: the "text" metaphor hides power relations
  - Whose interpretation counts — the anthropologist's or locals'?
  - "Culture as text" risks making culture seem static
```

## Section 4: Fieldwork and Ethics

### The Ethnographic Method

```
  PRE-MALINOWSKI: Armchair anthropology
  +----------------------------------------------------------+
  | Missionaries/travelers/colonial officers write reports   |
  | Anthropologist at home reads and synthesizes             |
  | Result: decontextualized, distorted, agenda-laden        |
  +----------------------------------------------------------+

  MALINOWSKI'S INNOVATION: Participant Observation
  +----------------------------------------------------------+
  | Researcher lives with the community (1+ year typical)    |
  | Learns the language (not through interpreters)           |
  | Participates in daily activities                         |
  | Writes detailed fieldnotes                               |
  | Interviews — formal, informal, life history              |
  | Collects material culture artifacts                      |
  +----------------------------------------------------------+

  KEY TENSION: The observer affects what is observed
  Too much participation --> lose analytical distance
  Too little            --> never get past surface presentation

  The tension is productive. Reflexivity = accounting for the
  researcher's own position and effects in the analysis.
```

### Ethics: AAA Code

The American Anthropological Association Code of Ethics (2012):

```
  HIERARCHY OF OBLIGATIONS:
  1. Research participants / the people studied  (PRIMARY)
  2. Scholarship / the discipline
  3. Funding sources / institutional employers

  KEY PRINCIPLES:
  - Do no harm: anticipate consequences for research subjects
  - Informed consent: explain research; allow refusal
  - Confidentiality: protect identities when promised
  - Researcher positionality: acknowledge how your own position
    (gender, race, class, nationality) shapes what you observe
  - Reciprocity: what do you give back to the community?
```

**Sensitive historical cases**: The Yanomami controversy (Chagnon's "fierce people"
framing and alleged harm to the people he studied). Project Camelot (1960s US
military funding for anthropological research on political instability — when
exposed, it caused a discipline-wide crisis). The Phoenix Program and anthropology's
relationship to counterinsurgency.

## Section 5: Postcolonial Critique and Decolonization

```
  The uncomfortable history:
  Anthropology was born alongside colonialism. Colonial administrators
  needed to know how "natives" worked — anthropology provided that
  knowledge, often used to control, extract, or assimilate.

  "Writing Culture" (Clifford and Marcus, 1986):
  - The ethnographic monograph is a literary artifact with power
  - "Native" voices appeared only filtered through the
    anthropologist's frame
  - The "natives" were objects of study, not knowledge producers

  Decolonizing responses:
  +---------------------------------------------------------+
  | Indigenous anthropology: Vine Deloria Jr., "Custer      |
  | Died for Your Sins" (1969) — anthropologists as         |
  | extractors who parachute in and publish with no benefit |
  | to the communities studied                              |
  +---------------------------------------------------------+
  | NAGPRA (1990): Native American Graves Protection and    |
  | Repatriation Act — institutions must return human       |
  | remains and sacred objects to descendant communities   |
  +---------------------------------------------------------+
  | FPIC: Free, Prior, Informed Consent — communities have  |
  | the right to refuse research access entirely            |
  +---------------------------------------------------------+
  | CARE principles for indigenous data:                    |
  | Collective Benefit / Authority to Control /             |
  | Responsibility / Ethics                                 |
  | (counterpoint to FAIR data principles)                  |
  +---------------------------------------------------------+
  | Collaborative ethnography: research subjects become     |
  | co-authors, co-investigators                            |
  +---------------------------------------------------------+
```

## Section 6: Bridge — Anthropology at Microsoft

This is not abstract to someone running engineering organizations:

```
  ANTHROPOLOGICAL METHOD         MICROSOFT / ENGINEERING APPLICATION
  ─────────────────────────────────────────────────────────────────
  Thick description (Geertz)  -> User research: what users *do*,
                                  not what they say they do.
                                  PARC's Lucy Suchman showed the gap
                                  between plans and situated action.

  Participant observation     -> Contextual inquiry (Holtzblatt):
                                  field interviews, shadowing users
                                  in their own environment.
                                  "Follow the user home" methodology.

  Cultural models             -> Design ethnography (IDEO/PARC):
  (Bourdieu habitus)            mapping hidden practices that users
                                  cannot articulate in a survey.

  WEIRD problem               -> Global product design: features built
  (Henrich 2010)                for WEIRD psychology fail in East/
                                  South Asian, African, Latin American
                                  markets. The "universal user" is not.

  Boasian relativism          -> True localization requires understanding
                                  cultural logic, not just translating
                                  strings and reformatting dates.

  Mauss's The Gift            -> Engineering team culture: peer review
  (gift/obligation)             norms, oncall help, knowledge sharing —
                                  informal obligations that govern
                                  cooperation without explicit contract.

  Practice theory (Bourdieu)  -> Why org change programs fail:
                                  the habitus of "how we do things here"
                                  is embodied, pre-reflective, and
                                  resists top-down redesign.
```

The discipline that invented participant observation is the same one Microsoft
Research uses when studying how engineers collaborate and why global products
succeed or fail. The WEIRD problem is a direct constraint on product design
decisions made at the VP level for a global user base.

## Common Confusion Points

**"Isn't anthropology just sociology?"**
Sociology focuses on modern, usually Western, industrial societies using surveys
and statistics. Anthropology is comparative across all human societies and deep
time, uses ethnographic fieldwork, and integrates biology. Sociology rarely
engages with non-human primates or fossil hominins.

**"Cultural relativism = moral relativism?"**
No. Methodological relativism means: understand a practice on its own terms before
judging it. This is an epistemological stance (understand before evaluating). It
does not mean all practices are equally valid. Boas was an active anti-racist and
opponent of Nazi race science — the furthest thing from moral relativism.

**"Isn't the four-field model arbitrary?"**
The American model is institutionally motivated — Boas built departments housing
all four subfields. The British tradition split early (social anthropology stayed
separate from biological). Today the fields are often separate graduate programs.
"Four-field" is more disciplinary identity than intellectual necessity, and many
biological anthropologists have little professional contact with cultural ones.

**"Is thick description less rigorous than quantitative work?"**
Rigor in interpretive anthropology is about depth and coherence of interpretation,
intersubjective checking (multiple researchers in the same field site), reflexive
accounting for researcher position, and saturation (collecting until no new patterns
emerge). Not statistical power. Different epistemology, not lower standards.

**The Mead-Freeman controversy:**
Margaret Mead's "Coming of Age in Samoa" (1928) argued Samoan adolescents had no
sexual repression, proving culture determines psychology. Derek Freeman (1983)
argued Mead was deceived by informants and his own data contradicted hers. Debate
remains unresolved. Lesson: fieldwork is theory-laden, single-site ethnography is
fragile, and stakes in the nature-nurture debate make these disputes politically
explosive.

**Structuralism: can't you find binary oppositions anywhere?**
Yes, and that is the decisive critique. The method is too productive — it generates
plausible-sounding structural analyses from virtually any dataset. This is why
cognitive science and evolutionary anthropology largely replaced structuralism after
the 1980s. The brain probably does use binary contrasts extensively, but the
specific schemas Levi-Strauss identified were not empirically derived.
