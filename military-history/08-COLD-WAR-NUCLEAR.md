# Cold War and Nuclear Strategy: Deterrence, Limited War, and Doctrine

```
+------------------------------------------------------------------+

+------------------------------------------------------------------+
|                                                                  |
|  NUCLEAR LEVEL:       Strategic deterrence (MAD by 1962)        |
|                       Regulated by arms control treaties         |
|                       Key: second-strike credibility            |
```

## Deterrence Theory: The Evolution

### Massive Retaliation (1954-1961)

Eisenhower's "New Look" policy: the US would respond to any Soviet aggression with nuclear weapons. The rationale was economic — nukes were cheaper than maintaining large conventional forces ("more bang for the buck").

```
MASSIVE RETALIATION LOGIC:
  Soviet conventional advantage: ~175 divisions in Europe
  US conventional cost: enormous to match
  Nuclear deterrent: relatively cheap; deters any aggression

  PROBLEM: Credibility
  Would US use nuclear weapons over a Soviet probe in Berlin?
  Over a border incident in Korea?
  If the threat was not credible for small actions, it deters
  only strategic-scale aggression. Soviets could achieve
  objectives through incremental moves below the nuclear threshold.

  BERLIN CRISES (1948, 1958-61) demonstrated the problem:
  Soviet pressure short of direct military attack.
  US could not credibly threaten nuclear war over a blockade.
  Massive retaliation didn't deter everything — it deterred
  only those actions worth nuclear war to defend.
```

### Flexible Response (1961-1991)

Kennedy administration replaced massive retaliation with flexible response:

```
+------------------------------------------------------------------+
|                    FLEXIBLE RESPONSE ESCALATION LADDER           |
+------------------------------------------------------------------+
|                                                                  |
|  RUNGS (lowest to highest):                                      |
|                                                                  |
|  1. Subconventional: covert ops, proxies, economic pressure     |
|  2. Conventional: general purpose forces engagement             |
|  3. Tactical nuclear: battlefield/theater nuclear weapons        |
|     (Lance missiles, B61 bombs, artillery shells — NATO         |
|      deployed ~7,000 tactical nuclear weapons in Europe)        |
|  4. Intermediate: medium-range nuclear weapons                  |
|  5. Strategic: ICBM/SLBM/bomber strikes on enemy homeland       |
|                                                                  |
|  THEORY: Respond at the appropriate rung; deny enemy            |
|  escalation dominance at every level.                           |
|                                                                  |
|  PROBLEM: Nuclear escalation control                             |
|  Once tactical nukes used: does it stop there?                  |
|  NATO war games: conventional defense would fail by day 3-7;   |
|  commanders would request tactical nuclear release.             |
|  From tactical nuclear to strategic: unclear how to stop.       |
|  "Nuclear war fighting" assumed escalation could be controlled  |
|  — this was an assumption, not a demonstrated capability.       |
+------------------------------------------------------------------+
```

### Mutual Assured Destruction (MAD): The Stability Logic

By the early 1960s, both superpowers had enough nuclear weapons to destroy the other multiple times over. This produced an unexpected stability:

```
+------------------------------------------------------------------+
|                    MAD STABILITY MECHANISM                       |
+------------------------------------------------------------------+
|                                                                  |
|  FIRST STRIKE PROBLEM:                                           |
|  If State A attacks State B first:                              |
|  State A destroys B's missiles in silos/airfields.             |
|  State B launches surviving weapons.                            |
|                                                                  |
|  FOR FIRST STRIKE TO WORK:                                       |
|  A must destroy enough B weapons that B's surviving weapons     |
|  cannot cause "unacceptable damage" to A.                       |
|  Definition of "unacceptable": debated, but ~10-25% of         |
|  population and industrial capacity = 20-60 million dead.       |
|                                                                  |
|  SECOND STRIKE CAPABILITY:                                       |
|  If B has weapons that survive A's first strike and can         |
|  retaliate with "unacceptable" damage: A cannot rationally      |
|  attack first.                                                   |
|                                                                  |
|  REQUIREMENT FOR STABILITY:                                      |
|  BOTH sides must have secure second-strike capability.          |
|  If A has it and B does not: B is vulnerable, unstable.        |
|  If neither has it: first-strike advantage = pressure to       |
|  strike first in a crisis. ("Use them or lose them")           |
|                                                                  |
|  HOW SECOND STRIKE IS SECURED:                                   |
|  1. Submarines (SSBNs): hidden at sea, not in fixed silos.     |
|     Polaris/Poseidon/Trident: ~50-100+ missiles/submarine.     |
|     Cannot be found; cannot be destroyed preemptively.         |
|  2. Mobile ICBMs (Soviet SS-25, etc.): move on roads/rail.     |
|  3. Dispersal: bomber alert, rapid launch capability.           |
|  4. Hardened silos: Minuteman silos: ~300 psi overpressure.    |
|                                                                  |
|  KEY INSIGHT: Nuclear stability requires BOTH sides to be       |
|  invulnerable in their retaliatory capability. Defensive        |
|  systems that threaten second-strike (ABM systems) are          |
|  DESTABILIZING — they give one side first-strike potential      |
|  if the defense can absorb the weakened retaliation.            |
|  This is why the 1972 ABM Treaty was signed.                    |
+------------------------------------------------------------------+
```

**The credibility paradox:** MAD required both sides to commit to retaliation that would be strategically irrational to execute (since launching after absorbing a first strike accomplishes nothing — the war is already effectively over). Committing to irrational behavior rationally deters rational adversaries. This is the strategic logic of deterrence: the threat to do something catastrophically destructive prevents the need to actually do it.

## Korean War (1950-1953): Limited War Invented

### Why Korea Was Different from WWII

```
STRATEGIC CONTEXT:
  June 25, 1950: North Korea (Soviet-equipped, Chinese-trained)
  invades South Korea.
  US commitment: UN mandate + Truman's unilateral deployment.

  THE LIMITING CONSTRAINTS:
  1. Nuclear backdrop: any expansion could trigger Soviet entry
     or Soviet nuclear threat.
  2. China: 3 million PLA ground forces in Manchuria.
     If US bombs Manchurian supply bases: China's red line.
  3. Soviet Air Force: already in Korea (MiG-15 pilots in Soviet
     markings — "Honcho" fighters US pilots identified by accent).
  4. European priority: US military planners saw Korea as
     distraction from main threat in Europe.

  MacArthur repeatedly ignored these constraints:
  - Wanted to bomb Manchuria
  - Wanted to use Nationalist Chinese forces
  - Proposed using nuclear weapons
  - Communicated his disagreement with Truman publicly

  Truman fired MacArthur (April 1951):
  Civil-military relations paramount.
  "The strategy under consideration by the Joint Chiefs of Staff
  could lead to a larger war." — Truman

  This established: in the nuclear age, wars must be fought
  for limited objectives with limited means. The general serves
  the political leadership, not the reverse.
```

### The Armistice Without Victory

Armistice signed July 27, 1953. Approximate prewar line restored. No peace treaty (technically still no peace treaty). ~36,000 US dead, ~137,000 WIA, ~3 million total Korean and Chinese dead.

**The Korean War established:**
1. Limited war is possible under a nuclear umbrella
2. Civil-military relationship: generals don't set strategy
3. Nuclear weapons were not used despite military pressure to do so
4. Stalemate is an acceptable outcome if the alternative is escalation to nuclear war

## Vietnam: COIN Doctrine Rise and Fall

### The Strategic Context

```
US INVOLVEMENT:
  1955: Military Advisors (MAAG Vietnam)
  1964: Gulf of Tonkin Resolution — open-ended force authorization
  1965-68: Buildup to 543,000 US troops
  1969-72: Vietnamization + drawdown
  1973: Paris Peace Accords; US withdraws
  1975: South Vietnam falls to North Vietnamese conventional invasion

NORTH VIETNAMESE STRATEGY (Vo Nguyen Giap):
  Three-phase insurgency strategy (Maoist model):
  Phase 1: Guerrilla war — survive, build political infrastructure
  Phase 2: Mobile war — attack isolated positions, avoid battle
  Phase 3: Conventional offensive — once South Vietnam weakened

  VN UNDERSTANDING: They didn't have to WIN militarily.
  They had to outlast US political will.
  Ho Chi Minh: "You will kill ten of us for every one of you
  that we kill. But even at those odds, you will lose and we
  will win."
```

### Why US Strategy Failed: The Body Count Problem

```
+------------------------------------------------------------------+
|                    BODY COUNT METRIC FAILURE                     |
+------------------------------------------------------------------+
|                                                                  |
|  US STRATEGY (Westmoreland, 1965-68): "Attrition"              |
|  Kill more enemy than they can replace. "Crossover point"       |
|  where NLF/NVA losses exceed recruitment + infiltration.        |
|                                                                  |
|  METRIC: Enemy killed (body count).                             |
|                                                                  |
|  PERVERSE INCENTIVES:                                            |
|  Unit commanders promoted based on body count.                  |
|  Pressure: report high counts.                                  |
|  Result: inflated counts; civilians counted as combatants;      |
|  "We had to destroy the village to save it."                    |
|                                                                  |
|  MEASUREMENT PROBLEM:                                            |
|  Body count does not measure:                                   |
|  - Political control of villages                               |
|  - South Vietnamese government legitimacy                       |
|  - NLF infrastructure (tax collection, recruitment, propaganda) |
|  - Civilian support or opposition to the insurgency            |
|  - Whether US presence was creating more insurgents than        |
|    it was killing                                               |
|                                                                  |
|  ACTUAL TREND:                                                    |
|  US killed hundreds of thousands of NVA/NLF.                   |
|  NVA/NLF recruitment + Ho Chi Minh Trail infiltration          |
|  (350,000-400,000/year at peak) replaced losses.               |
|  The "crossover point" was never reached.                       |
|  Political infrastructure in the South: NLF controlled most    |
|  rural provinces after dark.                                    |
+------------------------------------------------------------------+
```

### Tet Offensive (January 31, 1968): Information Operation

```
TET OFFENSIVE:
  NVA/NLF coordinated attack on 100+ South Vietnamese cities.
  Military result: decisive NVA defeat. Viet Cong effectively
  destroyed as military force. US/ARVN repelled attacks.

  STRATEGIC RESULT: US political will broken.
  The US public had been told the war was being won.
  Seeing fighting in Saigon embassy compound, Hue taken,
  massive battles nationwide: credibility gap revealed.

  Walter Cronkite (CBS, most trusted man in America):
  "It seems now more certain than ever that the bloody experience
  in Vietnam is to end in a stalemate."
  Lyndon Johnson: "If I've lost Cronkite, I've lost Middle America."

  Johnson withdrew from 1968 presidential race.
  The Tet Offensive succeeded as an information operation
  even while failing as a military operation.
  This is a pure expression of Clausewitz: war is politics.
  The target was US political will, not US military capability.
```

## AirLand Battle Doctrine (1982)

After Vietnam, the US Army had to rebuild around the real threat: Soviet conventional forces in Europe. FM 100-5 (1982) was the most important doctrinal publication since WWII:

```
+------------------------------------------------------------------+
|                    AIRLAND BATTLE DOCTRINE                       |
+------------------------------------------------------------------+
|                                                                  |
|  PROBLEM: Warsaw Pact conventional superiority in Europe        |
|  Soviet doctrine: "Operational Maneuver Groups" (OMGs)          |
|  Echeloned attack: first echelon penetrates; second and third   |
|  echelons (following 24-48 hrs behind) exploit before NATO      |
|  can bring up reserves.                                          |
|                                                                  |
|  NATO PREVIOUS DOCTRINE: Linear defense along forward edge       |
|  Weakness: Second echelon arrives fresh to exploit gaps         |
|                                                                  |
|  AIRLAND BATTLE SOLUTION:                                        |
|  Don't just fight the first echelon — DISRUPT ALL ECHELONS     |
|  SIMULTANEOUSLY using deep attack                               |
|                                                                  |
|  DEEP ATTACK:                                                    |
|  Army Tactical Missile System (ATACMS): 150+ km range           |
|  Air Force interdiction: attack second/third echelon staging     |
|  Attack helicopter (Apache): tank killing at 8 km range         |
|  Joint Surveillance Target Attack Radar System (JSTARS): see    |
|  moving ground targets 150+ km deep                             |
|                                                                  |
|  INITIATIVE: Don't wait for Soviet attack. Seize initiative,    |
|  disrupt their timing, force them to respond to you.            |
|                                                                  |
|  KEY CONCEPTS:                                                    |
|  - Agility: OODA loop faster than adversary                     |
|  - Depth: attack rear areas, not just contact points           |
|  - Synchronization: air-ground coordination on same timeline    |
|  - Initiative: offensive spirit, not passive defense            |
|                                                                  |
|  TESTED: Gulf War 1991. Against Iraqi Army (Soviet doctrine).   |
|  AirLand Battle worked exactly as designed.                      |
+------------------------------------------------------------------+
```

## Arms Control: The Verification Problem

```
+------------------------------------------------------------------+
|                    ARMS CONTROL AGREEMENTS                       |
+------------------------------------------------------------------+
|                                                                  |
|  SALT I (Strategic Arms Limitation Treaty, 1972):               |
|  Froze number of ICBMs and SLBMs at existing levels.           |
|  Did not limit warheads (MIRV — multiple independently         |
|  targetable reentry vehicles — allowed unlimited warhead       |
|  increase on same number of missiles).                          |
|  ABM Treaty (1972): Limited ABM systems to 2 sites each.       |
|  Rationale: preserve mutual vulnerability = preserve deterrence. |
|                                                                  |
|  SALT II (1979): Limited strategic launchers.                   |
|  Never ratified by Senate (Soviet invasion of Afghanistan).     |
|  Both sides observed it anyway until 1986.                      |
|                                                                  |
|  INF Treaty (1987): Eliminated ALL intermediate-range missiles  |
|  (500-5,500 km range) — a ZERO option, not just a cap.        |
|  Destroyed: ~2,692 missiles, verified by on-site inspection.   |
|  Why possible: Reagan's SDI pressure + Gorbachev's political   |
|  need for reduced military spending.                             |
|  US withdrew 2019 (citing Russian violations with SSC-8/9M729) |
|                                                                  |
|  START I (1991): Reduced strategic warheads to 6,000 each.     |
|  START II (1993): Never entered into force.                     |
|  New START (2010): 1,550 deployed warheads each.               |
|  Russia suspended New START 2023 (Ukraine war).                |
|                                                                  |
|  VERIFICATION PROBLEM:                                           |
|  "Trust but verify" (Reagan). What can you verify?              |
|  - Delivery vehicles: large, satellites can count them         |
|  - Warheads: small, can be hidden; require intrusive inspection |
|  - Tactical weapons: short-range, dispersed, hard to count     |
|  - New systems: hypersonic gliders, nuclear torpedoes (Status-6)|
|    designed to evade existing counting rules                    |
+------------------------------------------------------------------+
```

## Proxy Wars: Geography of Cold War Competition

```
PROXY WAR MAP:
  Both superpowers fought ideological competition through
  client states and proxy forces, avoiding direct confrontation.

  Korea (1950-53): China/USSR vs US. UN proxy.
  Vietnam (1965-73): USSR/China supplied NVA. US ground war.
  Angola (1975-88): Soviet/Cuban forces vs US/South Africa-backed
    UNITA. Largest African conventional battle (Cuito Cuanavale).
  Afghanistan (1979-89, Soviet): Mujahideen (CIA-funded) vs
    Soviet Army. Stinger missiles changed air defense.
  Nicaragua (1980s): CIA-backed Contras vs Soviet-backed Sandinistas.
  El Salvador (1979-92): US-backed government vs FMLN.
  Ethiopia/Somalia: switched sides mid-Cold War (1977-78).

PROXY WAR LOGIC:
  Superpower objectives without superpower risk.
  Limited commitment: supplies, advisors, intelligence.
  Client bears the casualties.
  Deniability maintained (often thinly).
  Escalation control: if client losing, increase support;
  if winning too decisively (risk of direct confrontation): constrain.

UNINTENDED CONSEQUENCES:
  CIA-funded, ISI-trained Afghan mujahideen included Osama bin Laden.
  Stinger missiles (MANPADS given to mujahideen) created proliferation
  concern after Soviet withdrawal — where did all those missiles go?
  The proxy wars left behind armed groups with weapons and
  organization but no longer useful political patrons.
```

## Common Confusion Points

**MAD was not a strategy — it was an acknowledged condition.** Neither side wanted mutual assured destruction; both understood it was the consequence of a nuclear exchange. The "strategy" was deterrence: prevent the exchange from ever occurring by making the consequences certain and catastrophic. When critics called MAD "insane," nuclear theorists responded: the insanity of the consequence is precisely what makes the deterrence credible.

**The Korean War armistice is often called "unsuccessful" because there was no victory.** But the political objective was to restore South Korean independence, not unify Korea. That objective was achieved. MacArthur's objective (unite Korea under Rhee) was rejected because it risked nuclear escalation with China/USSR. Accepting stalemate when the alternative is catastrophic escalation is rational strategy, not failure.

**The "body count" critique is often oversimplified.** There were operational contexts in Vietnam where attrition was the correct metric — particularly in the A Shau Valley and border operations targeting NVA regular forces. The problem was applying the attrition metric to the wrong kind of warfare (counterinsurgency, village-level political competition) where killing combatants was not the decisive variable. The metric was correct for conventional war; it was wrong for COIN.

**AirLand Battle was not the 1991 Gulf War doctrine in totality.** AirLand Battle addressed the specific problem of Soviet echeloned attack in Central Europe. The Gulf War had different terrain (open desert) and different enemy (Iraqi conventional forces, not Soviet first-line doctrine). The operational concept that governed Desert Storm was actually "deep operations" — shaped by Air Force and Army contributions but not identical to FM 100-5's specific provisions. The conceptual DNA was shared; the operational plan was adapted.

**Soviet nuclear doctrine was not identical to US doctrine.** Soviet military thought did not fully accept the Western deterrence-stability framework. Soviet doctrine emphasized fighting and winning a nuclear war (using tactical nuclear weapons in Central Europe as battlefield weapons from early in a conflict), not just deterring one. This made NATO force planners nervous — the Soviets might not stay on the deterrence logic that Western analysts assumed was universal.
