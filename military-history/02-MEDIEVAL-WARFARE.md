## The Feudal Military System

### Structure and Its Inherent Limits

Feudalism was not designed as a military system — it was a political-economic response to the collapse of Carolingian central authority. Its military implications were severe:

```
+------------------------------------------------------------------+
|                    FEUDAL MILITARY SYSTEM                        |
+------------------------------------------------------------------+
|                                                                  |
|  KING                                                            |
|    |                                                             |
|    +-- GREAT LORDS (barons, dukes, bishops)                     |
|    |       Call: "Bring your knights and levy"                  |
|    |       Obligation: 40 days per year of military service     |
|    |       (English common law by ~11th century)                |
|    |                                                             |
|    +-- LESSER LORDS (knights, lesser barons)                    |
|    |       Same obligation to great lords, who aggregate        |
|    |                                                             |
|    +-- FREE PEASANTS                                            |
|            Can be called for levy (fyrd in Anglo-Saxon England) |
|            Poorly equipped, low morale, economically disruptive |
+------------------------------------------------------------------+

THE 40-DAY PROBLEM:
  A campaign beginning June 1st ends September 8th — maximum.
  Siege of a major castle might take 3-6 months.
  Strategic campaigns against distant enemies: impossible.

  Workarounds:
  - Pay knights to extend (scutage — money instead of service)
  - Hire mercenaries (routiers, condottieri) for longer campaigns
  - Use Church lands (bishops owe military service for their fiefs)
  - Military orders (Templars, Hospitallers — permanent professional
    forces, answerable to Pope not local lords)
```

### The Knight as Military System

The mounted knight (~950-1350) was not simply an individual warrior — he was a weapons platform requiring substantial support infrastructure:

```
+------------------------------------------------------------------+
|                   KNIGHT SUPPORT REQUIREMENTS                    |
+------------------------------------------------------------------+
|                                                                  |
|  HORSE:      Destrier (warhorse) cost: 100-150x daily laborer   |
|              wage. A knight needed 3-5 horses (battle, travel,  |
|              spare). Horses ate 10-12 kg grain/day. Warhorses   |
|              could not graze on campaign grass alone.            |
|                                                                  |
|  ARMOR:      Full plate (late 14th c): 15-25 kg. Articulated    |
|              to allow full movement — but requires skilled       |
|              armorer, 3-12 months to make, fitted to individual.|
|              Mail hauberk (earlier): 5-10 kg, mass-producible.  |
|                                                                  |
|  SQUIRE:     1-2 assistants to manage horses, maintain armor,   |
|              help don armor (20-30 min process for full plate),  |
|              recover horse/equipment if knight unhorsed          |
|                                                                  |
|  ESTATE:     Knight required ~10-20 villein families farming    |
|              his land to generate the income to maintain         |
|              equipment and horses                               |
|                                                                  |
|  RESULT: A single "knight" in the field represented the         |
|          agricultural surplus of perhaps 50-100 households.     |
|          Very expensive; very few of them; decisive in          |
|          the right tactical conditions.                         |
+------------------------------------------------------------------+
```

## Castle Evolution: The Cost-Benefit of Defense

The castle was the infrastructure node of medieval power. Understanding its evolution requires tracking the attack-defense competition:

```
+------------------------------------------------------------------+
|                    CASTLE EVOLUTION TIMELINE                     |
+------------------------------------------------------------------+
|                                                                  |
|  MOTTE-AND-BAILEY (~900-1100)                                   |
|                                                                  |
|     [Tower on earthen mound (motte)]                            |
|           |                                                      |
|     [Bailey — enclosed yard for garrison, animals, supplies]    |
|           |                                                      |
|     [Palisade — wooden fence]                                   |
|                                                                  |
|  Construction time: weeks. Cost: low. Materials: wood + earth.  |
|  Vulnerability: FIRE. William I built ~40 of these in England   |
|  post-1066. Rapid deployment fortification.                     |
+------------------------------------------------------------------+
|                                                                  |
|  STONE KEEP (~1100-1200)                                         |
|                                                                  |
|     [Square stone keep, 20-30m tall]                             |
|     Walls 3-5m thick. Narrow slit windows.                       |
|     Ground floor: storage (siege supplies).                      |
|     Upper floors: garrison + lord's chambers.                    |
|                                                                  |
|  Construction time: years. Cost: very high. Stone + skilled      |
|  labor. Vulnerability: square corners create dead ground —       |
|  miners can approach along wall without being shot at.           |
|  Solution: round towers (no dead ground).                        |
+------------------------------------------------------------------+
|                                                                  |
|  CONCENTRIC CURTAIN WALL (~1200-1350)                           |
|                                                                  |
|     Outer wall: lower, can be covered by inner wall              |
|     Inner wall: higher, covers outer wall                       |
|     Towers at intervals: flanking fire along walls              |
|     Gate: gatehouse, portcullis, murder holes, drawbridge       |
|                                                                  |
|  Key innovation: defenders on inner wall can cover outer wall   |
|  top. Attackers must take both walls. Towers cover wall bases.  |
|                                                                  |
|  Crusader examples: Krak des Chevaliers (Syria) — considered    |
|  most perfect example. 1100 garrison, 2-year siege provisions.  |
+------------------------------------------------------------------+
```

### Siege Techniques Against Each Generation

```
+------------------------------------------------------------------+
|                   SIEGE METHOD MATRIX                            |
+------------------------------------------------------------------+
|  Castle Type    | Assault | Starvation | Mining | Artillery      |
|-----------------|---------|------------|--------|---------------- |
|  Motte-bailey   | Fire    | Weeks      | NA     | Primitive       |
|  Stone keep     | Ladders | Months     | Corner | Mangonel        |
|  Concentric     | Rarely  | 1-3 years  | Base   | Trebuchet       |
|  After trace    | Breach  | Weeks-mo   | NA     | Cannon (fatal)  |
+------------------------------------------------------------------+

STARVATION MATH:
  Castle garrison: 200-1000 men + civilians
  Daily food: ~1.5 kg grain equivalent per person
  1000-person castle, 6-month provisions: ~270 tonnes
  With horses, cattle: provisioning a siege was an enormous
  logistical challenge for the attacker too.

  Edward I's siege of Stirling Castle (1304): 3 months, 13
  siege engines including "Warwolf" (largest trebuchet built in
  British Isles). Castle surrendered before Warwolf was tested.

MINING:
  Approach via zig-zag trench (to deny direct fire)
  Dig under corner tower
  Shore up with wood props
  Pack with combustibles
  Fire the props -> ground subsides -> tower collapses
  Counter: dig moat (water table stops mining) or sortie to
  destroy siege works
```

## Siege Machinery: Mechanical Physics

Medieval siege engines were applied physics — energy storage and release:

```
+------------------------------------------------------------------+
|              SIEGE ENGINE COMPARISON                             |
+------------------------------------------------------------------+
|                                                                  |
|  MANGONEL (TORSION):                                            |
|     Energy stored in twisted rope/sinew bundle                  |
|     Release: arm swings over, sling releases projectile         |
|     Range: ~180-400m                                            |
|     Projectile: 5-60 kg stone                                   |
|     Rate of fire: 1-2 shots/minute (slow to wind)              |
|     Flat-ish trajectory — can fire over walls if elevated       |
|                                                                  |
|  TREBUCHET (COUNTERWEIGHT):                                     |
|     Energy stored in raised counterweight (1000-10,000 kg)      |
|     Long arm with sling. Release: counterweight falls,          |
|     arm rises, sling releases at ~45-degree angle               |
|     Range: up to 300m (with counterweight calibration)          |
|     Accuracy: MUCH better than torsion — counterweight is       |
|     consistent; torsion rope stretches and weakens              |
|     Projectile: 90-150 kg stones, or diseased carcasses         |
|     (Caffa 1346 — bodies of plague victims catapulted over      |
|     walls; debated as first biological warfare incident)        |
|     Rate of fire: slow (counterweight must be re-raised)        |
|                                                                  |
|  BATTERING RAM:                                                  |
|     Suspended from frame (penthouse) to protect operators       |
|     Used against gates, weakened wall sections                  |
|     Tortoise (testudo) formation protects approach              |
|     Counter: defenders drop heavy objects on frame,             |
|     fire the penthouse, grapple/lift the ram                   |
+------------------------------------------------------------------+

GREEK FIRE:
  Byzantine incendiary weapon (~670 AD). Exact composition unknown
  (state secret — formula died with Byzantine Empire). Probably
  naphtha-based with quicklime + resins. Burns on water. Delivered
  by siphon (pressurized pump) from ship prow or from walls.
  Defeated multiple Arab siege fleets of Constantinople.
  Psychological effect beyond physical — water is supposed to
  extinguish fire. This fire spread on water.
```

## Crossbow vs Longbow: The Training Investment Question

This comparison is not just technical — it is a social and economic question about how to invest in military capability:

```
+------------------------------------------------------------------+
|                 CROSSBOW vs LONGBOW COMPARISON                   |
+------------------------------------------------------------------+
|                    CROSSBOW              LONGBOW                 |
+------------------------------------------------------------------+
| Draw weight   | 150-350+ lb (lever/    | 100-175 lb (muscle)     |
|               | windlass assisted)     |                         |
| Range (max)   | 300-380m               | 200-300m                |
| Rate of fire  | 2-4 bolts/min          | 10-12 arrows/min        |
| Armor pen.    | Excellent at close     | Excellent at close      |
|               | range (~30m+)          | range with broadhead    |
| Training time | Weeks                  | Years (childhood start  |
|               |                        | needed for draw weight) |
| Physiology    | Any able-bodied adult  | Skeletal deformation —  |
|               |                        | medieval archers show   |
|               |                        | asymmetric muscle/bone  |
| Maintenance   | Mechanical — springs,  | Wood + string, weather  |
|               | nut, trigger; complex  | sensitive; simpler      |
| Field use     | Can hold draw — sniper | Cannot hold draw;       |
|               | use from cover         | continuous effort       |
+------------------------------------------------------------------+

AGINCOURT (1415):
  French: ~12,000-30,000 (estimates vary) including ~6,000+ men-at-arms
  English: ~6,000-9,000, including ~5,000 archers (longbow)

  Terrain: muddy plowed field after night rain, funnel between woods
  French men-at-arms advanced dismounted through knee-deep mud
  English archers on flanks, staked in position (angle stakes -> horses)

  Physics: Armored knight in mud: ~0.5 m/s advance speed
  English rate: 10 arrows/min per archer × 5,000 archers = 50,000
  arrows per minute at the advancing mass

  Arrow effect on armor: direct penetration at close range;
  more importantly, arrows immobilized horses, struck visors/joints,
  fell on upward-facing faces of men bent forward in mud.
  Fallen armored men could not rise unaided -> trampled.

  Result: ~6,000+ French killed, ~1,500+ captured; English ~100-500.

KEY: This was not longbow magic — it was terrain selection, defensive
positioning, and rate-of-fire overwhelming armored opponents who had
lost mobility in mud. Same English archers lost at Patay (1429)
when French cavalry charged before archers could stake positions.
```

## Mongol Horse Archery: The Operational System

The Mongol military of 1206-1260 was the most effective military machine of the medieval period — its impact was continental, defeating every major power it engaged:

```
+------------------------------------------------------------------+
|                   MONGOL MILITARY SYSTEM                         |
+------------------------------------------------------------------+
|                                                                  |
|  ORGANIZATION (decimal system):                                 |
|     arban:  10 men (smallest tactical unit)                     |
|     zuun:   100 men (10 arbans)                                 |
|     minghan: 1,000 men (10 zuuns)                              |
|     tumen:  10,000 men (10 minghans)                           |
|                                                                  |
|  Perfect binary tree: command authority is clear at every level.|
|  Meritocratic promotion: commanders chosen by demonstrated      |
|  competence, not hereditary rank.                               |
|                                                                  |
|  INDIVIDUAL SOLDIER:                                            |
|     5 horses per man (remounts enable continuous operation)     |
|     Composite bow: recurved horn-wood-sinew construction        |
|       Draw weight: 160+ lb, range: 300m+ effective              |
|       Accuracy from gallop via "Parthian shot" (shoot backward  |
|       while retreating — feigned retreat tactic)                |
|     Light armor (lamellar/scale): speed priority over protection|
|     Each man maintains his own remounts -> no central logistics |
|     for horses; grazing + captured supplies                     |
|                                                                  |
|  OPERATIONAL CAPABILITY:                                        |
|     Strategic march rate: 80-130 km/day (using remounts)       |
|     European heavy cavalry: 30-50 km/day maximum               |
|     Mongol columns could cross Europe faster than news of       |
|     their approach. Strategic surprise was nearly guaranteed.   |
+------------------------------------------------------------------+
```

### Feigned Retreat: The Core Tactical Trick

```
FEIGNED RETREAT SEQUENCE:

  1. Mongol light cavalry engages enemy heavy cavalry
  2. Enemy cavalry charges (medieval cavalry is trained to charge)
  3. Mongols flee — apparently broken
  4. Pursuing knights: faster individually but strung out over miles
     as horses tire at different rates
  5. Strung-out pursuing cavalry hits the Mongol line of heavy
     cavalry (kheshig) waiting in prepared position
  6. Mongol light cavalry turns, encircles from flanks
  7. Trapped, surrounded, exhausted European cavalry destroyed

Mohi (1241): Batu Khan + Subutai destroyed Hungarian army
  ~100,000 total Mongol forces split to prevent relief columns
  from combining. Subutai's flanking force crossed Sajo River
  at night at a separate ford — a strategic turning movement
  Europeans had no concept of.

THE STRATEGIC MOBILITY FACTOR:
  Subutai personally planned campaign using intelligence from
  merchants and diplomats years before invasion. His forces:
  - Defeated Georgia, Russia, Poland, Hungary in one campaign
  - Operated across 1,000 km front simultaneously
  - Used deception, disinformation, psychological warfare
    (Mongols deliberately spread terror narratives to demoralize
    populations ahead of their advance)
```

### Psychological Warfare as a System

The Mongols deliberately weaponized terror through consistent policy:

- Cities that surrendered without resistance: spared, administered
- Cities that resisted: destroyed completely, population massacred
- This was not rage — it was calculated policy to minimize resistance
- Message spread ahead of armies; cities frequently capitulated without siege
- Nishapur (1221): Resisted and was razed. Every living thing killed. 1.7 million reported dead (exaggerated, but slaughter was total).
- Samarkand (1220): Surrendered peacefully. Protected, administered.

The distinction created a rational surrender calculus that reduced Mongol operational costs. Every defender knew what the choice meant. This is information warfare: shaping the adversary's decision space before the battle occurs.

## The Infantry Revolution: End of Cavalry Dominance

The period 1302-1415 saw the systematic defeat of feudal heavy cavalry by infantry — a social disruption as significant as the military disruption:

```
+------------------------------------------------------------------+
|              INFANTRY DEFEATS CAVALRY: THREE BATTLES             |
+------------------------------------------------------------------+
|                                                                  |
|  BATTLE OF GOLDEN SPURS, Courtrai (1302)                       |
|  Flemish communal infantry vs French heavy cavalry              |
|  Terrain: boggy fields with ditches                             |
|  French cavalry charged, horses hit ditches, Flemish infantry   |
|  with goedendags (spike-club hybrid) killed unhorsed knights    |
|  500+ gilded cavalry spurs collected from dead nobles -> name   |
|  Significance: first major French cavalry defeat in open field  |
|                                                                  |
|  BANNOCKBURN (1314)                                             |
|  Scottish schiltron (pike ring formation) vs English cavalry    |
|  Bruce selected boggy ground with concealed pits                |
|  English cavalry could not maneuver; Scottish pikes repelled    |
|  charges; English dismounted but lost organizational cohesion   |
|  Significance: Scottish common infantry defeated English         |
|  mounted aristocracy -> Scottish independence for decades       |
|                                                                  |
|  LAUPEN (1339) + MORGARTEN (1315)                               |
|  Swiss pikemen vs Austrian/Burgundian heavy cavalry             |
|  Swiss developed dense pike square: 30-50 men deep, ~6m pikes   |
|  Cavalry cannot charge into a pike hedge                        |
|  Significance: Swiss mercenary pike squares became the dominant |
|  European infantry system for 150+ years until gunpowder        |
+------------------------------------------------------------------+

SOCIAL IMPLICATION:
  Feudal military system was based on mounted knight as decisive
  weapon. Knight status = social-political-military privilege.
  When commons with pikes could defeat knights, the military
  justification for aristocratic privilege eroded.

  Machiavelli noted this explicitly: a prince who depends on
  mercenaries or knights he cannot rely on is weak; a prince
  whose citizens fight for him is strong.
```

### Swiss Pike Square: The Technical Solution

```
SWISS PIKE SQUARE vs CAVALRY:

  [P][P][P][P][P][P][P][P]
  [P][P][P][P][P][P][P][P]   P = pikeman, 5.5-6m pike
  [P][P][P][P][P][P][P][P]
  [P][P][P][P][P][P][P][P]
  [P][P][P][P][P][P][P][P]

  Facing cavalry:
  First 4 ranks present pikes forward (horizontal hedge)
  Rear ranks present pikes upward/angled
  A horse will not charge a wall of steel points

  Cavalry options:
  a) Charge: horse refuses the obstacle, or if forced, spears
     impale horse; rider falls into formation
  b) Skirt the flanks: disciplined square rotates to face
  c) Stand off and shoot: Swiss vulnerable to longbow/crossbow
     fire unless they keep moving

  SWISS TACTICAL RESPONSE TO FIREPOWER:
  Move fast. Swiss squares were trained to attack, not stand.
  Their offense was their defense — fix with fire, advance the
  square, close to contact before arrows/quarrels depleted them.
```

## Decision Cheat Sheet: Medieval Warfare

| Attacker's Problem | Medieval Solution | Why It Worked/Failed |
|-------------------|-------------------|---------------------|
| Reduce stone castle | Starvation or mining (until artillery) | Siege was the dominant mode; assault rarely succeeded |
| Defeat defending archers | Cavalry charge before they deploy | Agincourt failed because of mud/stakes |
| Defeat pike square | Combine missile fire + cavalry on flanks | Neither element alone worked |
| Match Mongol strategic mobility | Could not — 5 horses/man difference | Infrastructure mismatch, organizational mismatch |
| Campaign beyond 40 days | Mercenaries, scutage, military orders | Feudal system was structurally limited |
| Hold territory without castles | Cannot — castle is the infrastructure node | Castle network = administrative + military control |

## Common Confusion Points

**Medieval warfare was not all knights.** The bulk of every medieval army was infantry levy — poorly equipped freemen and villeins with spears, axes, or bows. Knights were perhaps 5-10% of a typical army by headcount. The knight was decisive in the right terrain and conditions; on bad terrain or against organized infantry with the right positioning, they were vulnerable. The myth of the invincible knight comes from chronicles written by and for the literate aristocracy.

**"Dark Ages" military technology was not primitive** — it was adapted to conditions. The motte-and-bailey castle was a rapid-deployment fortification system that enabled the Norman conquest of England in two years. The concentric curtain wall was a sophisticated engineering response to evolving siege technology. Medieval engineers understood the geometry of flanking fire and dead ground well before formal military engineering texts.

**The crossbow bolt vs longbow arrow penetration comparison is terrain-dependent.** Crossbow bolts (quarrels) had higher kinetic energy at close range; the longbow had superior rate of fire at all ranges. At Agincourt, the relevant variable was not penetration — it was volume of fire saturating a narrow approach channel. The penetration debate matters mostly at longer ranges against stationary armored targets.

**Mongol withdrawal from Europe (1241-42) was not due to military defeat.** They withdrew after Ogedei Khan died — Batu Khan had to return to participate in the election of a new Great Khan. Europe was saved by Mongol succession politics, not military resistance. No European force had been able to defeat the Mongols in pitched battle by that point. Hungary and Poland had been devastated.

**The feudal military system was financially rational** — it was free to the king when it worked. The problem was strategic: you couldn't pursue objectives that took longer than 40 days. This is a classic cost-optimization trap: a system optimized for cheapness that fails under strategic stress. When kings needed longer campaigns, they paid (scutage) or borrowed — both economically distorting.
