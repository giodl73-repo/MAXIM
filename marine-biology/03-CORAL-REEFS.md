# Coral Reef Ecosystems

## The Big Picture

Coral reefs cover ~284,000 km² — about 0.1% of ocean floor — yet host an estimated
25% of all marine species. That is the Connell Paradox: enormous biodiversity in
nutrient-poor tropical water. The mechanism is not input richness; it is tight internal
recycling and the intermediate disturbance regime.

```
CORAL REEF SYSTEM — ARCHITECTURE AND ENERGY FLOW
==================================================

OPEN OCEAN                    REEF SYSTEM                    LAND
(oligotrophic)                                               (runoff)
    ↓ Waves                 ┌──────────────────────┐
    ↓ low nutrients         │  FORE REEF           │  ←─── Wave energy
                            │  (steep drop-off)    │
                            │  Coral spurs,        │
                            │  buttresses, grooves  │
                            │  (dissipate wave      │
                            │   energy)             │
                            └──────────┬───────────┘
                                       │
                            ┌──────────┴───────────┐
                            │  REEF CREST          │
                            │  (shallowest, most   │
                            │   wave energy, most  │
                            │   light, hardiest    │
                            │   coral species)     │
                            └──────────┬───────────┘
                                       │
                            ┌──────────┴───────────┐
                            │  BACK REEF /         │
                            │  REEF FLAT           │
                            │  (calm, warm, some   │

ENERGY FLOW:
  Zooxanthellae fix CO₂ → glucose → 90%+ of coral energy needs
  Coral polyp provides: CO₂, NH₄⁺, shelter, light orientation
  Herbivorous fish (parrotfish, surgeonfish) → keep algae off coral
  Predatory fish → top-down control of herbivores
  Tight nutrient cycling: dead organisms → bacteria → nutrients
    reabsorbed by algae → eaten by animals → back to coral
  Net nutrient import from ocean: minimal
```

## Coral Biology: The Polyp

A coral "colony" is millions of genetically identical clonal polyps, each a tiny
animal related to sea anemones.

```
CORAL POLYP ANATOMY
====================

           Tentacles (nematocysts for prey capture)
            │    │    │
     ───────┤    │    ├─────────  Oral disc
            └────┼────┘
                 │   Mouth
                 │
                 │   Coelenteron (gastrovascular cavity)
                 │   lined with gastrodermis containing
                 │   ZOOXANTHELLAE (Symbiodinium)
                 │
       ┌─────────┴─────────┐
       │   Calicoblast      │      Secretes CaCO₃ skeleton below
       │   epithelium       │      (calcium carbonate deposition)
       └─────────┬─────────┘
                 │
    ┌────────────┴───────────────┐
    │  CALCIUM CARBONATE         │
    │  SKELETON (corallite)      │
    │  CaCO₃ aragonite           │
    │  Accumulated over decades  │
    │  to centuries              │
    └────────────────────────────┘
```

**Coral calcification chemistry:**
```
Ca²⁺ + CO₃²⁻ → CaCO₃ (aragonite)

Requires: high calcium (seawater Ca²⁺ ~10 mmol/kg)
          high carbonate (CO₃²⁻ — depends on ocean pH)
          aragonite saturation state (Ω_arag > 1 for growth)

Ocean acidification: CO₂ + H₂O → H₂CO₃ → H⁺ + HCO₃⁻
  H⁺ reacts with CO₃²⁻ → HCO₃⁻ (reducing CO₃²⁻ availability)
  → Ω_arag decreases → calcification slows → skeleton weakens
```

**Growth rates:**
- Branching corals (*Acropora*): up to 10-20 cm/year — fast colonizers
- Massive corals (*Porites*): 0.5-2 cm/year — form the structural core
- A 5 m diameter *Porites* head: estimated 500+ years old

## Zooxanthellae (Symbiodinium): The Critical Symbiosis

The entire reef system depends on an intracellular photosymbiosis.

```
CORAL-ZOOXANTHELLAE SYMBIOSIS
==============================

ZOOXANTHELLAE IN TISSUE
  ┌────────────────────────────────────────────────┐
  │  Symbiodinium sp. (dinoflagellate)             │
  │  ~2–5 μm diameter; lives in gastrodermal cells │
  │  ~1–2 million cells per cm² of coral tissue    │
  │                                                │
  │  Photosynthesis:                               │
  │  CO₂ + H₂O + light → glucose + O₂            │
  │                          │                     │
  │                          ▼                     │
  │              Translocates 90-99% of            │
  │              photosynthate to coral host       │
  │              (glucose, glycerol, amino acids)  │
  └────────────────────────────────────────────────┘
                  ↕ mutualism
  ┌────────────────────────────────────────────────┐
  │  CORAL HOST provides:                          │
  │  CO₂ (from respiration)                        │
  │  NH₄⁺ (nitrogen — limiting nutrient)           │
  │  PO₄³⁻ (phosphorus)                            │
  │  Physical shelter inside cell                  │
  │  Light access (thin tissue, light orientation) │
  └────────────────────────────────────────────────┘

WHY CORALS ARE COLORED:
  Coral tissue is nearly transparent
  ZOOXANTHELLAE contain chlorophyll a and c (brown/yellow)
  + coral fluorescent proteins (green/red/blue — photoprotection)
  Combined appearance: brown/tan/yellow (zooxanthellae) +
                       vivid colors (fluorescent proteins)
```

## Coral Bleaching: The Breakdown of Symbiosis

Bleaching = expulsion of zooxanthellae. Coral reverts to white skeleton color.

```
BLEACHING MECHANISM
====================

THERMAL STRESS (primary cause)
  Normal sea surface temperature + 1-2°C sustained = bleaching threshold
  (~28-30°C depending on local adaptation)

  ┌─────────────────────────────────────────────────────┐
  │  HEAT STRESS                                        │
  │       │                                             │
  │       ▼                                             │
  │  Photosystem II dysfunction in zooxanthellae        │
  │  (PSII is temperature-sensitive)                    │
  │       │                                             │
  │       ▼                                             │
  │  Excess excitation energy → reactive oxygen         │
  │  species (ROS) production                           │
  │  (O₂⁻, H₂O₂, ·OH)                                │
  │       │                                             │
  │       ▼                                             │
  │  ROS damages both zooxanthellae and coral tissue    │
  │       │                                             │
  │       ▼                                             │
  │  CORAL EXPELS ZOOXANTHELLAE                         │
  │  (adaptive response? or damage response?            │
  │   still debated — probably both)                    │
  │       │                                             │
  │       ▼                                             │
  │  WHITE SKELETON visible through transparent tissue  │
  │  Coral: ALIVE but starving                          │
  │       │                                             │
  │   ┌───┴────────────────────────────┐                │
  │   │ Stress < 4 weeks               │ Stress prolonged│
  │   ▼                                ▼                │
  │  RECOVERY                      MORTALITY            │
  │  (zooxanthellae recolonize)    (weeks-months)       │
  └─────────────────────────────────────────────────────┘

OTHER BLEACHING CAUSES:
  UV radiation (shallow water + low turbidity)
  Freshwater inundation (salinity drop — heavy rain)
  Pollution (cyanide, copper, herbicides)
  Cold water (below range minimum)
```

**Mass bleaching events:**
| Year | Event | Geographic Scale |
|------|--------|-----------------|
| 1998 | First global bleaching | El Niño; 16% of global coral died |
| 2010 | Second global bleaching | Widespread Caribbean/Indo-Pacific mortality |
| 2015-2016 | Third global bleaching | Largest recorded; ~50% of Great Barrier Reef bleached |
| 2024 | Fourth global bleaching | Confirmed by NOAA — all major reef systems affected |

## The Biodiversity Paradox: Connell's Resolution

**The question:** Why is coral reef species diversity so high when surrounding
tropical water is nearly devoid of nutrients (oligotrophic)?

```
CONNELL'S INTERMEDIATE DISTURBANCE HYPOTHESIS (1978)
=====================================================

   DIVERSITY
       │
  HIGH │           ╭─────╮
       │          ╱       ╲
       │         ╱         ╲
  LOW  │────────╱           ╲─────────
       └────────────────────────────────
            LOW   INTERMEDIATE   HIGH
                   DISTURBANCE FREQUENCY

LOW disturbance:    Competitive dominants exclude subordinates
                    → low diversity (coral monopoly)
HIGH disturbance:   Most species eliminated (storm, bleaching)
                    → low diversity
INTERMEDIATE:       Competitive dominants cannot exclude before
                    the next disturbance → all niches open
                    → maximum coexistence → HIGH DIVERSITY

Reef disturbances:
  Storms (physical damage), disease outbreaks, bleaching events,
  crown-of-thorns starfish (Acanthaster planci) outbreaks,
  predation on dominant species

The reef IS the intermediate disturbance regime at work
```

**Additional mechanism — nutrient cycling on reefs:**
- Sponges filter enormous volumes of water, remineralizing dissolved organic carbon
- Nitrogen cycling: seagrasses fix nitrogen; sponges excrete NH₄⁺
- Parrotfish: grind coral with beak-like teeth → produce CaCO₃ sand (20 tonnes/km²/year)
- Coral mucus: rich DOC → bacterial decomposition → nutrients recycled in reef matrix

## Reef Formation: Darwin's Subsidence Theory

Darwin (1842) proposed three reef types represent a temporal sequence:

```
DARWIN'S SUBSIDENCE THEORY
===========================

STAGE 1: FRINGING REEF          STAGE 2: BARRIER REEF       STAGE 3: ATOLL
  (young volcanic island)         (older, subsiding)          (sunken island)

   ISLAND                          ISLAND                      LAGOON
   ┌─────┐                         ┌───┐                       ────────
   │▲▲▲▲▲│  ←reef grows up         │ ▲ │  Lagoon               ~
   │▲▲▲▲▲│   as island             └───┘   forms               ~~~ REEF ~~~
   └──┬──┘   subsides              ═══════════                  ~
    ══╪══                          BARRIER REEF               ring of coral
    REEF                           (separated by               over drowned
                                    lagoon)                    volcanic base

Timeline: ~100,000 years per stage (island subsides ~2 cm/year tectonically)
Modern confirmation: drill cores in Pacific atolls (Bikini, Eniwetok)
  → reached basalt at ~1300 m depth under ~1300 m of CaCO₃ reef
  (reef grew up to stay in photic zone as island subsided)
Prediction confirmed: boring in 1950s found Darwin exactly right
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| What gives coral its color? | Zooxanthellae (brown) + coral fluorescent proteins (vivid colors) |
| Why is bleached coral white? | Transparent tissue over white CaCO₃ skeleton; zooxanthellae expelled |
| What temperature triggers bleaching? | Local SST + 1-2°C sustained over ~4 weeks |
| How do corals get most of their energy? | Photosynthate from zooxanthellae (90-99%) |
| Why do reefs have high diversity? | Intermediate disturbance + tight nutrient recycling |
| What is the largest coral structure? | Great Barrier Reef: ~344,400 km², visible from space |
| What is the coral triangle? | Indo-Pacific region (Indonesia, Philippines, PNG) — maximum marine biodiversity |
| Why does ocean acidification threaten corals? | Reduces CO₃²⁻ → lowers aragonite saturation → slows calcification |
| What do parrotfish do to reefs? | Graze coral algae AND scrape/digest coral → produce reef sand |

## Common Confusion Points

**Coral is not a plant.** It is an animal (phylum Cnidaria) that hosts photosynthetic algae
(zooxanthellae). The photosynthesis happens inside the coral cells, not by the coral. This
makes it functionally a solar-powered carnivore.

**Bleached coral is not dead coral.** It is a stressed, starving coral. If thermal stress
ends within ~4 weeks, zooxanthellae can recolonize and recovery occurs. Prolonged bleaching
leads to mortality from starvation + infection. Bleaching = warning; sustained bleaching = death.

**All reefs are not equivalent.** Indo-Pacific reefs have ~5–10× the species richness of
Caribbean reefs. Management lessons from one region do not directly transfer to another.

**Coral rubble ≠ dead reef.** Rubble fields exist in healthy, recovering reefs. Storm damage
creates rubble that can be colonized. The distinction between ecologically dead reef (covered
in turf algae with no coral recruitment) vs recovering rubble matters for management.

**Zooxanthellae are not all the same.** Symbiodinium is actually a genus complex with multiple
clades (A-I). Different clades have different thermal tolerances. Corals with clade D zooxanthellae
are more bleaching-resistant but grow more slowly. Natural selection is already shifting clade
composition in some populations — a form of adaptive response, but too slow for current rates.
