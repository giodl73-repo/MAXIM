# 07 — Digestive System

## GI Tract, Liver, Pancreas, Absorption

> **STUB** — outline only, content to be authored

**Planned coverage:**
- GI tract anatomy: mouth → esophagus → stomach → small intestine (duodenum/jejunum/ileum) → large intestine → rectum
- Wall layers: mucosa (epithelium/lamina propria/muscularis mucosae) / submucosa / muscularis externa / serosa
- Motility: peristalsis, segmentation, gastric emptying, migrating motor complex
- Stomach: parietal cells (HCl + intrinsic factor), chief cells (pepsinogen), G cells (gastrin)
- Pancreas (exocrine): acinar cells (digestive enzymes as zymogens), ductal cells (bicarbonate)
- Liver: hepatocyte functions — bile synthesis, gluconeogenesis, protein synthesis (albumin/clotting factors), detoxification, bilirubin metabolism
- Bile: composition, bile salts + phospholipids + cholesterol, enterohepatic circulation
- Nutrient absorption: carbohydrates (SGLT1/GLUT5), proteins (amino acid transporters), fats (micelles → chylomicrons → lymph), vitamin B12 (intrinsic factor)
- Gut hormones: secretin, CCK, GIP, GLP-1, motilin — triggers and actions
- Microbiome: ~1013 bacteria, metabolic functions (SCFA production, vitamin K, bile acid modification)
- Enteric nervous system: 500M neurons, 5-HT as key transmitter, Auerbach/Meissner plexuses
