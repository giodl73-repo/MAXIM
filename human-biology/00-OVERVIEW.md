# 00 — Human Biology Overview

## Body System Map, Levels of Organization, Homeostasis

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Levels of organization: atoms → molecules → organelles → cells → tissues → organs → systems → organism
- Four tissue types (epithelial/connective/muscle/nervous) and their distribution
- Body systems index: musculoskeletal / cardiovascular / respiratory / nervous / endocrine / immune / digestive / renal / reproductive / integumentary
- Homeostasis: negative feedback loops, set points, effectors — temperature, pH, glucose, osmolarity
- Physiological integration: how systems interact (e.g., cardiovascular-renal axis, neuro-endocrine signaling)
- Body composition: water distribution (ICF/ECF/plasma), electrolyte gradients
- Anatomical planes and directional terminology
- Key quantities: total blood volume, cardiac output, respiratory rate, GFR, metabolic rate
