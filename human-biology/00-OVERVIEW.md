## Homeostasis: The Control Systems Model

Every body system implements **negative feedback control**. The engineering analogy is exact.

```
     SET POINT
    (reference)
         |
         v
   +----------+        +-----------+
   |  ERROR   |------->|  EFFECTOR |  (actuator — muscle, gland, organ)
   | DETECTOR |        +-----------+
   +----------+               |
         ^                    v
   +----------+        +--------------+
   |  SENSOR  |<-------|  CONTROLLED  |
   | (afferent)|        |  VARIABLE    |
   +----------+        +--------------+
```

| Controlled variable | Sensor | Set point | Effectors |
|--------------------|--------|-----------|-----------|
| Core temperature | Hypothalamic thermoreceptors | ~37°C | Sweat glands, shivering, vasodilation/constriction |
| Blood glucose | Pancreatic β/α cells | ~90 mg/dL (5 mM) | Insulin (↓ glucose), glucagon (↑ glucose) |
| Blood pressure | Baroreceptors (aortic arch, carotid sinus) | ~120/80 mmHg | Heart rate, vascular resistance, renal fluid retention |
| Arterial pCO₂ | Central/peripheral chemoreceptors | ~40 mmHg | Respiratory rate and tidal volume |
| Blood pH | Renal tubules + chemoreceptors | 7.35–7.45 | H⁺ excretion, HCO₃⁻ reabsorption, ventilation rate |
| Serum Ca²⁺ | Parathyroid chief cells | 8.5–10.2 mg/dL | PTH → bone resorption, renal Ca²⁺ retention, vit D activation |
| Blood osmolality | Hypothalamic osmoreceptors | ~285–295 mOsm/kg | ADH (vasopressin) → renal water retention |

**Positive feedback** exists but is always self-terminating:
- Childbirth: oxytocin → uterine contraction → more oxytocin → delivery, then placenta delivers and loop breaks
- Blood clotting: platelet activation cascades → clot seals wound, then inhibitors (antithrombin, protein C) terminate
- Action potential: Na⁺ influx → depolarization → more Na⁺ influx → peak, then inactivation gates close

## Body Fluid Compartments

Water is ~60% of body weight (~42 L in a 70 kg reference male):

```
TOTAL BODY WATER ~42 L
┌──────────────────────────────────────────────────┐
│  INTRACELLULAR FLUID (ICF)  ~28 L  (67%)        │
│  Dominant ions: K⁺, Mg²⁺, HPO₄²⁻, proteins     │
│  Low: Na⁺, Cl⁻, Ca²⁺                            │
├──────────────────────────────────────────────────┤
│  EXTRACELLULAR FLUID (ECF)  ~14 L  (33%)        │
│  Dominant ions: Na⁺, Cl⁻, HCO₃⁻                │
│  Low: K⁺, proteins                               │
│                                                  │
│  ┌───────────────────┬──────────────────────┐    │
│  │  INTERSTITIAL     │  PLASMA              │    │
│  │  ~11 L  (78%)     │  ~3 L  (22%)        │    │
│  │  bathes cells     │  + RBCs = ~5 L blood│    │
│  └───────────────────┴──────────────────────┘    │
└──────────────────────────────────────────────────┘
```

The ICF/ECF gradient is maintained by **Na⁺/K⁺-ATPase** in every cell membrane:
- Pumps 3 Na⁺ out, 2 K⁺ in per ATP — electrogenic, slightly hyperpolarizing
- Creates the concentration gradients that power action potentials and secondary active transport
- Consumes ~20–40% of resting metabolic ATP

## Systems Integration

No system is isolated. Key coupling axes:

### Neuroendocrine Axis: Hypothalamus → Pituitary → Peripheral Gland

```
Hypothalamus
  ↓  releasing hormone (CRH, TRH, GnRH, GHRH, GHRH, dopamine)
Anterior Pituitary
  ↓  tropic hormone (ACTH, TSH, LH/FSH, GH, PRL)
Peripheral Gland
  ↓  effector hormone (cortisol, T3/T4, estrogen/testosterone, IGF-1)
Target organs
  ↓  negative feedback back to hypothalamus + pituitary
```

### Autonomic Push-Pull

```
                      ANS
           ┌──────────┴──────────┐
      SYMPATHETIC            PARASYMPATHETIC
  "fight or flight"          "rest and digest"
  NE / epinephrine           ACh transmitter
  ↑ HR, ↑ BP                 ↓ HR, ↓ BP
  bronchodilate              bronchoconstrict
  inhibit digestion          stimulate digestion
  gluconeogenesis            glycogen synthesis
  dilate pupil               constrict pupil
```

### RAAS: Kidney–Lung–Adrenal Pressure Control

```
Low BP / low Na⁺ / low renal perfusion
  ↓
Juxtaglomerular cells → Renin
  ↓
Angiotensinogen (liver) → Angiotensin I
  ↓ (ACE — in lung capillaries)
Angiotensin II
  ├── Vasoconstriction (↑ BP directly)
  └── Adrenal cortex → Aldosterone
        ↓
        Kidney: Na⁺ + H₂O retention → ↑ blood volume → ↑ BP
```

## Cellular Energy Economy

All systems run on ATP. Main production pathways:

```
GLUCOSE (6C)
  ↓ Glycolysis (cytoplasm)          → 2 ATP net, 2 pyruvate, 2 NADH
  ↓ Pyruvate dehydrogenase          → 2 Acetyl-CoA, 2 CO₂, 2 NADH
  ↓ TCA / Krebs cycle (mitoch.)     → 2 ATP, 6 NADH, 2 FADH₂, 4 CO₂
  ↓ Oxidative phosphorylation (ETC) → ~32 ATP (gradient-driven)

TOTAL: ~36–38 ATP per glucose

FATTY ACIDS: β-oxidation
  Palmitate (C16) → ~129 ATP
  Fat is ~2.3× more energy-dense than carbs per gram

ANAEROBIC (no O₂):
  Glycolysis only → 2 ATP + lactate
  pH drops, fatigue sets in
```

O₂ is the terminal electron acceptor in the ETC. This is why every system in the body is ultimately organized around O₂ delivery (cardiovascular + respiratory) and CO₂ removal.

## Common Confusion Points

**"Systemic" in medicine ≠ "a specific organ system"**
"Systemic" means whole-body (systemic lupus, systemic circulation). Not referring to any one system.

**Endocrine vs paracrine vs autocrine signaling**
- Endocrine: hormone travels in blood to distant target
- Paracrine: acts on nearby cells (not blood-borne; e.g., NO, prostaglandins)
- Autocrine: acts on the same cell that released it (e.g., some cytokines)
- Juxtacrine: direct cell-cell contact (Notch/Delta signaling)

**Negative feedback ≠ inhibition**
Negative feedback means the response opposes the deviation from set point. Insulin release after a meal is negative feedback on blood glucose — the insulin doesn't "inhibit glucose," it drives glucose into cells to restore the set point.

**Afferent vs efferent** (always flipped)
- Afferent = toward center (sensory → CNS; blood → glomerulus)
- Efferent = away from center (motor → muscle; blood → glomerulus exit)
- Mnemonic: **A**fferent **A**rrives, **E**fferent **E**xits

**Plasma vs serum**
- Plasma: blood liquid component with clotting factors intact (anticoagulated sample)
- Serum: plasma after clotting has occurred (no fibrinogen)
- Most "blood tests" are serum; coagulation panels need plasma

**Smooth vs skeletal vs cardiac muscle**
Three distinct cell types with different Ca²⁺ handling, regulation, and regenerative capacity. Not interchangeable. Skeletal regenerates (satellite cells), cardiac barely regenerates, smooth regenerates moderately.
