# 06 — Immune System

## Innate, Adaptive, Lymphocytes, Antibodies, MHC

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Innate immunity: physical barriers, pattern recognition (TLRs, NOD-like receptors), complement system, NK cells, phagocytes
- Inflammation: cardinal signs, acute vs chronic, cytokines (TNF/IL-1/IL-6/IL-8), fever
- Complement cascade: classical / lectin / alternative pathways — opsonization, MAC, chemotaxis
- Adaptive immunity: antigen presentation, clonal selection, memory
- MHC I vs MHC II: which cells express them, what they present, which T cell subtype they activate
- T cells: CD4+ helper (Th1/Th2/Th17/Treg) vs CD8+ cytotoxic — TCR, co-stimulation (CD28/B7)
- B cells: BCR, affinity maturation, somatic hypermutation, isotype switching, plasma cells
- Immunoglobulins: IgM/IgG/IgA/IgE/IgD — structure, Fab/Fc regions, complement activation, neonatal transfer
- Active vs passive immunity, primary vs secondary response, immunological memory
- Immune tolerance: central (thymus/bone marrow deletion) vs peripheral (anergy/Treg/CTLA-4/PD-1)
- Vaccines: mechanisms by type (live attenuated/killed/subunit/conjugate/mRNA/viral vector)
