# Cardiovascular System

## The Big Picture: Dual-Circuit Pump

```
PULMONARY CIRCUIT                    SYSTEMIC CIRCUIT
(right heart)                        (left heart)

    LUNGS                                BODY TISSUES
   O₂ in / CO₂ out                      O₂ delivered / CO₂ picked up
       ↑         ↓                           ↑              ↓
  Pulm veins   Pulm arteries         Systemic arteries  Systemic veins
  (O₂ rich)   (O₂ poor)             (O₂ rich)         (O₂ poor)
       |         |                           |              |
  ┌────┴─────────┴────┐              ┌───────┴──────────────┴───┐
  │   LEFT HEART      │              │   RIGHT HEART            │
  │   LA → LV         │              │   RA → RV                │
  │   High pressure   │              │   Low pressure           │
  │   ~120 mmHg sys   │              │   ~25 mmHg sys           │
  └────────┬──────────┘              └──────────────────────────┘
           │
           └──────────────────────→ Aorta → Arteries → Arterioles
                                    → Capillaries → Venules → Veins
                                    → Vena cava → RA (right atrium)
```

Both circuits are **series circuits** — output of one feeds the other. Same cardiac output flows through both per minute.

## Heart Anatomy

```
                    AORTA
                      |
              ┌───────┴──────────┐
         Pulm. veins (4)      Vena cava (SVC + IVC)
              |                      |
       LEFT ATRIUM (LA)        RIGHT ATRIUM (RA)
              |                      |
          Mitral valve           Tricuspid valve
          (bicuspid)             (3 leaflets)
              |                      |
       LEFT VENTRICLE (LV)     RIGHT VENTRICLE (RV)
       thick wall (~11mm)       thin wall (~3mm)
              |                      |
          Aortic valve           Pulm. valve
          (semilunar)            (semilunar)
              |                      |
            AORTA              PULMONARY ARTERY
```

**AV valves** (mitral, tricuspid): open in diastole, closed in systole — prevent backflow to atria.
**Semilunar valves** (aortic, pulmonary): open in systole, closed in diastole — prevent backflow from arteries.
Chordae tendineae + papillary muscles anchor AV valve leaflets — prevent prolapse during systole.

### Coronary Arteries

```
AORTIC ROOT → Coronary ostia
  Left main → LAD (left anterior descending) → anterior wall, septum, apex
            → Circumflex → lateral + posterior LV wall
  Right coronary artery (RCA) → RV, posterior septum (usually), SA+AV nodes

Right dominant anatomy (85%): RCA supplies posterior descending artery
Left dominant (8%): circumflex supplies posterior descending
Codominant (7%): both contribute
```

## Cardiac Conduction System

```
SA NODE (sinoatrial)
  Right atrium, near SVC entry
  Intrinsic rate: 60–100 bpm (fastest pacemaker → dominates)
  Phase 4 spontaneous depolarization (funny current If, + Ca²⁺ channels)
        |
        v  (spreads through atrial myocardium → atrial contraction)
        v  (internodal pathways)
AV NODE (atrioventricular)
  Floor of right atrium
  Intrinsic rate: 40–60 bpm (backup pacemaker)
  KEY DELAY: ~120 ms — allows atria to empty before ventricles contract
  Only normal electrical pathway between atria and ventricles (fibrous annulus insulates)
        |
        v
BUNDLE OF HIS
  Passes through interventricular septum
        |
        ├── LEFT BUNDLE BRANCH (left fascicles: anterior + posterior)
        └── RIGHT BUNDLE BRANCH
                |
                v
        PURKINJE FIBERS
          Subendocardial network
          Intrinsic rate: 20–40 bpm (last resort pacemaker)
          Rapid conduction (4 m/s) → synchronized ventricular contraction
```

If SA node fails: AV junction escapes at 40–60 bpm.
If AV node blocked (3rd degree heart block): ventricles pace themselves at 20–40 bpm.

## ECG: Electrical Correlates

```
P wave:       Atrial depolarization (SA → AV node spread)
PR interval:  AV nodal delay (~120–200 ms) — prolonged in heart block
QRS complex:  Ventricular depolarization (Purkinje spread) (~80–100 ms)
ST segment:   Ventricular plateau (action potential phase 2) — elevated in STEMI
T wave:       Ventricular repolarization (apex to base)
QT interval:  Total ventricular electrical systole — prolonged by drugs/electrolytes → arrhythmia risk
```

ECG axis: normal is −30° to +90° (vectors point down-left toward LV mass). Left axis deviation in left bundle branch block / left ventricular hypertrophy. Right axis deviation in RVH / right bundle branch block.

## Cardiac Cycle

```
EVENTS IN ONE BEAT (~800 ms at 75 bpm):

DIASTOLE (filling):
  ┌─────────────────────────────────────────────────────┐
  │  Isovolumetric relaxation: AV + semilunar valves    │
  │  both closed, pressure falls rapidly                │
  │  → Mitral/tricuspid open when ventricular P < atrial P │
  │  Passive rapid filling → slow filling → atrial kick │
  │  End-diastolic volume (EDV) ~130 mL                 │
  └─────────────────────────────────────────────────────┘

SYSTOLE (ejection):
  ┌─────────────────────────────────────────────────────┐
  │  Isovolumetric contraction: all valves closed,      │
  │  pressure rises until LV > aorta (~80 mmHg)        │
  │  → Aortic valve opens                               │
  │  Rapid ejection → reduced ejection                  │
  │  End-systolic volume (ESV) ~60 mL                   │
  │  Stroke volume = EDV − ESV = ~70 mL                 │
  └─────────────────────────────────────────────────────┘

HEART SOUNDS:
  S1 ("lubb") — mitral + tricuspid closure (start of systole)
  S2 ("dubb") — aortic + pulmonary closure (start of diastole)
  S3 — rapid ventricular filling in diastole (normal in young; HF in adults)
  S4 — atrial contraction against stiff ventricle (hypertension, HCM)
```

### Pressure-Volume Loop

```
LV Pressure
  ^
  |           C ─────── D
  |          /          \
  |         /            \
  |        /              \
  |   B──/                \──A
  |
  +──────────────────────────→ LV Volume

  A: mitral opens (diastolic filling begins)
  B: mitral closes (isovolumetric contraction begins)
  C: aortic opens (ejection begins)
  D: aortic closes (isovolumetric relaxation begins)

  Width of loop = stroke volume
  Area of loop = stroke work
```

## Cardiac Output and its Determinants

```
CO = HR × SV

Stroke Volume determined by:
  PRELOAD  ────→ SV (Frank-Starling: ↑ EDV → ↑ stretch → ↑ force)
  AFTERLOAD ───→ SV (↑ aortic pressure → ↑ ESV → ↓ SV)
  CONTRACTILITY → SV (intrinsic force at given preload/afterload)
```

**Frank-Starling law**: Stretching sarcomeres increases overlap between actin and myosin, optimizing cross-bridge formation. Also increases troponin C Ca²⁺ sensitivity. Works up to ~2.2 μm sarcomere length, then decreases.

**Contractility** altered by:
- ↑ Sympathetic stimulation (NE → β₁) → ↑ contractility (positive inotropy), ↑ HR (chronotropy)
- ↑ Intracellular Ca²⁺ (cardiac glycosides: digoxin inhibits Na/K-ATPase → ↑ intracellular Na⁺ → ↓ Ca²⁺ extrusion via NCX)
- Acidosis, hypoxia, β-blockers → ↓ contractility

Normal resting CO: ~5 L/min. Maximum (trained athlete during exercise): ~25–35 L/min (5–7× increase).

```
(capacitance vessel)     (post-capillary)

```

Endothelium is **not passive** — it produces:
- NO (nitric oxide): vasodilator, anti-aggregant, anti-inflammatory
- Prostacyclin (PGI₂): vasodilator, anti-platelet
- Endothelin-1: potent vasoconstrictor
- vWF: platelet adhesion
- tPA: fibrinolysis

### Poiseuille's Law

```
Flow (Q) = (ΔP × π × r⁴) / (8 × η × L)

Q  = blood flow
ΔP = pressure gradient
r  = vessel radius (4th power — dominant factor)
η  = blood viscosity
L  = vessel length

Resistance R = 8ηL / πr⁴

Key insight: doubling radius → 16× increase in flow (or 16× decrease in resistance)
Arterioles control TPR: small radius changes → huge flow changes
```

### Mean Arterial Pressure

```
MAP = CO × TPR (total peripheral resistance)
MAP ≈ DBP + 1/3 (PP)  where PP = SBP − DBP

Normal: MAP ~93 mmHg (120/80 → 80 + 40/3 ≈ 93)

Regulation of MAP:
  Acute: baroreceptors → ANS (seconds)
  Medium: RAAS, ADH (minutes-hours)
  Long-term: renal pressure natriuresis (days)
```

## Microcirculation: Starling Forces

Fluid movement across capillary walls governed by Starling equation:

```
Net filtration = Kf [(Pc − Pi) − σ(πc − πi)]

Pc  = capillary hydrostatic pressure (pushes fluid OUT)
Pi  = interstitial hydrostatic pressure (pushes fluid IN, usually negative)
πc  = plasma oncotic pressure (pulls fluid IN) — albumin dominant
πi  = interstitial oncotic pressure (pulls fluid OUT, normally low)
Kf  = filtration coefficient
σ   = reflection coefficient
```

At arteriolar end: Pc high → net filtration out (into interstitium).
At venular end: Pc lower, πc unchanged → net reabsorption back in.
~90% of filtered fluid reabsorbed by capillaries; remaining 10% returned by lymphatics.

**Edema mechanisms**:
| Cause | Mechanism |
|-------|-----------|
| Heart failure | ↑ venous hydrostatic pressure → ↑ Pc |
| Cirrhosis/nephrotic syndrome | ↓ albumin → ↓ πc → ↓ oncotic reabsorption |
| Inflammation | ↑ capillary permeability → protein leaks to interstitium → ↑ πi |
| Lymphatic obstruction | Fluid not returned from interstitium |

## Blood Pressure Regulation

### Baroreceptor Reflex (acute, seconds)

```
↑ BP → baroreceptors (aortic arch, carotid sinus) stretch
  → CN IX/X to NTS (nucleus tractus solitarius, medulla)
  → ↑ parasympathetic (CN X) → ↓ HR
  → ↓ sympathetic → ↓ vasoconstriction, ↓ HR, ↓ contractility
  → BP corrects
```

### Renin-Angiotensin-Aldosterone System (medium-term, minutes–hours)

See overview module. RAAS is the dominant long-term pressure regulator via blood volume.

### ANP/BNP (atrial/B-type natriuretic peptide)

Released by cardiomyocytes in response to stretch (volume overload):
- ↑ GFR → ↑ Na⁺ + water excretion (natriuresis, diuresis)
- Vasodilation
- Suppress RAAS and sympathetic
Counter-regulatory: opposes RAAS. Elevated BNP = sensitive marker of heart failure.

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| Why is heart rate fastest at SA node? | Steepest phase 4 spontaneous depolarization slope (fastest funny current / If) |
| What determines stroke volume? | Preload (EDV), afterload (resistance), contractility |
| Why does arteriole tone control MAP? | r⁴ relationship — small radius changes → enormous resistance changes; arterioles are the resistance vessels |
| Systolic vs diastolic heart failure? | Systolic (HFrEF): can't eject (low EF); diastolic (HFpEF): can't fill (stiff wall, normal EF) |
| What closes heart valves? | Pressure gradients — no active mechanism. Valves are passive. |
| Why does exercise increase CO so much? | ↑ HR (sympathetic) + ↑ SV (↑ venous return → Frank-Starling + ↑ contractility) |
| S3 vs S4 significance? | S3: rapid filling, compliance issue (HF in adults); S4: stiff ventricle, atrial kick against resistance |

## Common Confusion Points

**Preload ≠ EDV, but EDV is its best surrogate**
Preload technically = end-diastolic wall tension / ventricular filling pressure (LVEDP). Clinically approximated by EDV or central venous pressure.

**Ejection fraction (EF) vs stroke volume**
EF = SV/EDV = ~60% normally. A dilated failing heart can have normal SV but low EF (large EDV masks dysfunction). A hypertrophied stiff heart can have low SV but normal EF.

**Pulse pressure reflects aortic compliance**
Pulse pressure = SBP − DBP (~40 mmHg normal). Widened PP (>60) with age = arterial stiffness (calcification reduces Windkessel buffering). Widened PP + bounding pulse = aortic regurgitation.

**ECG is electrical; echo is mechanical**
An ECG shows depolarization/repolarization patterns. A normal ECG doesn't mean normal mechanical function. Pulseless electrical activity (PEA) = ECG looks normal but heart isn't pumping.

**Left vs right heart failure: where the blood backs up**
- LV failure → blood backs up into pulmonary veins → pulmonary edema (dyspnea)
- RV failure → blood backs up into systemic veins → peripheral edema, JVD, hepatomegaly
