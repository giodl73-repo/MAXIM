# 08 — Renal System

## Nephron Function, GFR, Acid-Base, Electrolytes

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Kidney anatomy: cortex/medulla/pelvis, ~1M nephrons per kidney, renal vasculature
- Nephron segments: glomerulus → Bowman's capsule → PCT → loop of Henle → DCT → collecting duct
- Glomerular filtration: GFR (~120 mL/min), Starling forces at glomerulus, filtration fraction
- Tubular reabsorption: PCT (67% of Na/water, all glucose/amino acids), loop (countercurrent multiplier), DCT/CD (fine-tuning)
- Countercurrent mechanism: descending (water permeable) vs ascending (ion pump), medullary gradient, urea recycling
- Hormonal regulation: aldosterone (Na reabsorption/K secretion), ADH (water permeability), ANP (natriuresis)
- RAAS: renin → angiotensin I → ACE → Ang II → aldosterone + vasoconstriction
- Acid-base regulation: bicarbonate reabsorption, titratable acid, ammonium excretion
- Electrolyte homeostasis: Na+/K+/Ca2+/Mg2+/phosphate — each with regulatory axis
- Clearance concept: creatinine clearance, GFR estimation equations (CKD-EPI, MDRD)
- Urinalysis: specific gravity, protein, glucose, casts, sediment interpretation
