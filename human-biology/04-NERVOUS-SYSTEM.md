## Brain Regions

```
CEREBRAL CORTEX (4 lobes per hemisphere)
┌─────────────────────────────────────────────────────────────┐
│  FRONTAL           │  PARIETAL                              │
│  Primary motor     │  Primary somatosensory (postcentral)   │
│  (precentral gyrus)│  Spatial processing                    │
│  Prefrontal:       │  Body image                            │
│  executive, working│                                        │
│  memory, inhibition│                                        │
├─────────────────────┤─────────────────────────────────────  │
│  TEMPORAL          │  OCCIPITAL                             │
│  Primary auditory  │  Primary visual (V1: striate cortex)   │
│  Wernicke's area   │  V2–V5: motion, color, object          │
│  (language compreh)│  recognition                           │
│  Hippocampus       │                                        │
│  (memory encoding) │                                        │
└─────────────────────────────────────────────────────────────┘

DEEP STRUCTURES:
  Basal ganglia (striatum, globus pallidus, substantia nigra, STN)
    → movement initiation and suppression, habit formation
  Thalamus: relay station — almost all sensory info passes through
    (except olfaction → goes direct to cortex)
  Hypothalamus: master homeostasis controller + circadian clock (SCN)
    + appetite (arcuate nucleus), temperature, osmoregulation
  Limbic system: emotion + memory
    Amygdala: threat detection, fear conditioning, emotional salience
    Hippocampus: spatial maps, episodic memory encoding
    Cingulate cortex: conflict monitoring, pain affect, social cognition

CEREBELLUM:
  Timing + coordination of movement, balance, motor learning
  Receives: sensory info (proprioception, vestibular) + motor commands
  Outputs: corrective signals to motor cortex (via thalamus)
  Damage → ataxia, dysmetria, intention tremor (opposite limb side NOT affected — ipsilateral)

BRAINSTEM (midbrain / pons / medulla):
  Houses cranial nerve nuclei III–XII
  Controls: respiration, cardiovascular, consciousness (reticular formation)
  Crossroads: all descending motor + ascending sensory tracts pass through
```

## Spinal Cord: Organization

```
CROSS-SECTION (cervical level):
          Dorsal horn (sensory)
               ↑↑↑
    ┌──────────────────────────┐
    │   Gray matter (H shape)  │
    │   Dorsal horn: sensory   │
    │   processing, relay      │
    │   Ventral horn: motor    │
    │   neuron cell bodies     │
    │   Lateral horn: ANS      │
    │   (thoracic/lumbar)      │
    └──────────────────────────┘
               ↓↓↓
          Ventral horn (motor)

WHITE MATTER (myelinated tracts surrounding gray):
  Dorsal columns (posterior):
    Ipsilateral fine touch, vibration, proprioception
    → ascends ipsilateral → decussates in medulla → thalamus → cortex

  Spinothalamic tracts (anterolateral):
    Pain, temperature, crude touch
    → synapse in dorsal horn → IMMEDIATELY decussate → ascend contralateral
    → thalamus → cortex (S1)

  Corticospinal tract (lateral white matter, descending):
    Voluntary motor
    → decussates in medulla (pyramidal decussation)
    → descends contralateral → synapses on ventral horn motor neurons
```

### Spinal Cord Lesion Pattern Logic

```
Brown-Séquard (half cord):
  Ipsilateral: ↓ motor (corticospinal) + ↓ dorsal column sensation
  Contralateral: ↓ pain/temp (spinothalamic)

Complete cord transection:
  Below level: bilateral loss of all sensation + motor
  At level: LMN signs (flaccid, fasciculations)
  Below level: UMN signs (spastic, hyperreflexia) — after spinal shock resolves

Anterior cord syndrome (anterior spinal artery):
  ↓ Motor + pain/temp; preserved proprioception/vibration (dorsal columns spared)

Central cord syndrome (hyperextension in elderly):
  Arms > legs weakness (cervical somatotopic: arms center, legs periphery of cord)
```

## Upper vs Lower Motor Neuron Signs

```
UPPER MOTOR NEURON (UMN)           LOWER MOTOR NEURON (LMN)
(lesion in cortex or               (lesion in ventral horn,
 corticospinal tract)               peripheral nerve, or NMJ)

↑ Muscle tone (spasticity)         ↓ Muscle tone (flaccidity)
Hyperreflexia                      Hyporeflexia / areflexia
Babinski sign (upgoing toe)        No Babinski
No fasciculations                  Fasciculations (denervation)
No atrophy (initially)             Muscle atrophy (rapid)

  Stroke, MS, ALS (UMN component)   ALS (LMN component), polio,
```

## Neuron Electrophysiology

### Action Potential

```
MEMBRANE POTENTIAL (mV)
 +40 ─── threshold (~−55 mV) ─── peak (+40) ─── repolarization ─── hyperpolarization
  |                    ↑                 ↑                ↑                ↑
  |          Na⁺ channels open   Na⁺ inactivate  K⁺ channels open  K⁺ slowly close
  −70
  (resting)

ALL-OR-NOTHING: sub-threshold → no AP; at threshold → full AP
Refractory period: absolute (Na⁺ inactivated, impossible to fire) + relative (hyperpolarized)
```

### Synaptic Transmission

```
AP in presynaptic axon terminal
  ↓
Voltage-gated Ca²⁺ channels open
  ↓
Ca²⁺ influx → SNARE complex → vesicle fusion → neurotransmitter release
  ↓
NT diffuses across synaptic cleft (~20 nm)
  ↓
Binds postsynaptic receptors:
  Ionotropic (AMPA, NMDA, GABA-A, nAChR): direct channel opening → fast (ms)
  Metabotropic (mGluR, GABA-B, adrenergic, muscarinic): G protein → 2nd messenger → slower (sec-min)
  ↓
EPSP (excitatory) or IPSP (inhibitory)
  ↓
Spatial + temporal summation at axon hillock
  ↓
If threshold reached → AP fires in postsynaptic neuron
```

## Blood-Brain Barrier

```
BRAIN CAPILLARY:
  Endothelial cells: TIGHT JUNCTIONS (no fenestrae, minimal pinocytosis)
  Astrocytic end-feet: enwrap capillaries, maintain barrier, signaling
  Pericytes: regulate permeability, contractile

WHAT CROSSES:
  Freely: O₂, CO₂, lipid-soluble molecules (anesthetics, ethanol, many CNS drugs)
  By carrier: glucose (GLUT1), amino acids (LAT transporters), some ions
  NOT: large proteins, most polar molecules, many antibiotics (hence CNS infections hard to treat)
  Active efflux: P-glycoprotein (MDR1) pumps many drugs OUT — major drug resistance mechanism

CIRCUMVENTRICULAR ORGANS (no BBB — need blood sensing):
  Area postrema (vomiting trigger), subfornical organ, OVLT (osmolality)
  Posterior pituitary (ADH/oxytocin release into blood)
```

## CSF

```
Production: choroid plexus (lateral + 3rd + 4th ventricles)
Volume: ~150 mL; produced ~500 mL/day → complete turnover 3–4×/day
Circulation: lateral ventricles → 3rd ventricle → cerebral aqueduct → 4th ventricle
             → subarachnoid space (via foramina of Magendie + Luschka)
             → surrounds brain + spinal cord → absorbed at arachnoid granulations → venous sinuses

Normal CSF:
  Pressure: 7–18 cmH₂O (lumbar puncture opening pressure)
  Protein: 15–45 mg/dL (low — BBB keeps proteins out)
  Glucose: 50–80 mg/dL (~2/3 serum glucose)
  Cells: < 5 white cells/μL

Lumbar puncture: L3-L4 or L4-L5 space (below L1-2 where cord ends → no cord injury risk)
```

## Major Sensory Pathways

```
DORSAL COLUMN–MEDIAL LEMNISCAL PATHWAY:
  Fine touch, vibration, proprioception, 2-point discrimination
  1° neuron: DRG → dorsal column (ipsilateral)
  2° neuron: nucleus gracilis/cuneatus (medulla) → decussate → medial lemniscus → thalamus (VPL)
  3° neuron: thalamus → somatosensory cortex (S1, postcentral gyrus)

SPINOTHALAMIC PATHWAY:
  Pain, temperature, crude touch
  1° neuron: DRG → dorsal horn
  2° neuron: dorsal horn → immediate decussation → anterolateral funiculus → thalamus (VPL)
  3° neuron: thalamus → S1

VISUAL PATHWAY:
  Retina → optic nerve → optic chiasm (nasal fibers cross)
  → optic tract → LGN (thalamus) → optic radiation → V1 (occipital)
  Meyer's loop: inferior retina fibers loop through temporal lobe
  (temporal lobe lesion → contralateral superior quadrantanopia "pie in the sky")
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| Where do spinal reflexes integrate? | Spinal cord — monosynaptic stretch reflex entirely in cord, no brain needed |
| Why is cerebellar ataxia ipsilateral? | Cerebellum controls same-side body (ipsilateral pathways, unlike corticospinal) |
| What does the hypothalamus NOT control? | Voluntary movement, language, perception — it's for homeostasis/autonomic/endocrine |
| Sympathetic: one ganglion or many? | Sympathetic has short preganglionic to paravertebral chain → diverges widely ("mass" response); parasympathetic has long preganglionic to local ganglia → targeted |
| BBB drug design rule? | Lipophilic, low molecular weight, not P-gp substrate → crosses. CNS drugs need this. |
| Which sense doesn't relay through thalamus? | Olfaction — goes directly to olfactory cortex (piriform) and amygdala |

## Common Confusion Points

**Broca vs Wernicke's aphasia**
- Broca (frontal, inferior): expressive aphasia — understands, can't produce fluent speech ("knows what wants to say, can't say it")
- Wernicke (posterior temporal): receptive aphasia — fluent speech, doesn't understand, "word salad"
Both in dominant (usually left) hemisphere.

**Dermatome vs myotome**
- Dermatome: skin area innervated by one spinal root (sensory)
- Myotome: muscle group innervated by one spinal root (motor)
Different roots. Disc herniation at L4-5 affects L5 root (not L4) — compresses the exiting root below.

**Spinal shock vs neurogenic shock**
- Spinal shock: transient flaccid paralysis + areflexia immediately after cord transection (lasts days-weeks, then UMN signs emerge)
- Neurogenic shock: cardiovascular instability from loss of sympathetic tone (hypotension + bradycardia — cervical/high thoracic injuries)

**NMDA receptor and memory**
NMDA receptors require both ligand (glutamate) AND depolarization (Mg²⁺ block removed) to open — "coincidence detector." This temporal coincidence is the molecular basis of Hebbian synaptic plasticity and LTP (long-term potentiation), the cellular correlate of memory formation.
