# 09 — Reproductive System

## Anatomy, Gametogenesis, Hormonal Cycles, Development

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Male anatomy: testes (seminiferous tubules + Leydig/Sertoli cells), epididymis, vas deferens, prostate
- Spermatogenesis: spermatogonia → primary spermatocyte → secondary → spermatid → sperm; 74-day cycle
- Male hormonal axis: GnRH → LH (testosterone) + FSH (spermatogenesis); inhibin feedback
- Female anatomy: ovaries, fallopian tubes, uterus (endometrium/myometrium), cervix, vagina
- Oogenesis: primordial follicles (fixed pool from fetal life), primary oocyte arrested in prophase I until ovulation
- Menstrual cycle (28 days): follicular phase (FSH → follicle growth → estrogen) → ovulation (LH surge) → luteal phase (corpus luteum → progesterone) → menstruation if no implantation
- Fertilization: acrosome reaction, zona pellucida, cortical granule reaction (block to polyspermy)
- Implantation and placentation: blastocyst implantation (day 6-10), trophoblast invasion, hCG as pregnancy signal
- Embryonic development: germ layers, organogenesis, teratogen-sensitive periods
- Parturition: oxytocin/prostaglandins, positive feedback loop of labor
- Menopause: follicle depletion → estrogen loss → FSH/LH surge → symptoms
