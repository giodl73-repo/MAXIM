# Ethology: Tinbergen's Four Questions

## The Big Picture

Ethology is the biology of animal behavior — understanding behavior with the same rigor applied to anatomy or physiology. The field was formalized in the 1930s-1950s by Konrad Lorenz and Nikolaas Tinbergen (Nobel Prize 1973, shared with Karl von Frisch). Tinbergen's Four Questions remain the most powerful organizing framework for behavioral biology: they specify that any behavioral phenomenon requires four distinct, non-redundant levels of explanation.

```
TINBERGEN'S FOUR QUESTIONS (1963)
====================================

  Any animal behavior can be asked about at 4 levels:

  CAUSATION               DEVELOPMENT
  (proximate)             (ontogeny)
  "How does it work?"     "How did it develop in this individual?"
  Neural mechanism         Genetic program + experience
  Hormonal control         Critical periods, learning
  Sensory trigger          Epigenetics
       |                        |
       v                        v
  BEHAVIOR  <-----------  BEHAVIOR

       ^                        ^
       |                        |
  EVOLUTION               FUNCTION (ADAPTATION)
  (phylogeny)             (ultimate)
  "How did it evolve?"    "Why does it increase fitness?"
  Ancestral state          Survival advantage
  Phylogenetic history     Reproductive success
  Natural selection        Ecological context

  KEY INSIGHT: all 4 questions are independent and all 4 are required.
  Answering one doesn't answer another.
```

## Question 1: Causation (Mechanism)

### Neural and Hormonal Mechanisms

```
PROXIMATE MECHANISMS OF BEHAVIOR
===================================

SENSORY TRIGGERING:
  Sign stimulus (Tinbergen): specific stimulus that releases fixed behavior
  Supernormal stimulus: exaggerated sign stimulus; stronger response
    Herring gull chick: pecks red spot on parent's bill (IRM -- innate releasing mechanism)
    Experimental: model bill with larger, brighter spot -> more pecking than real bill
    Oystercatcher + huge egg: roll huge egg back to nest in preference to own egg
    -> supernormal stimulus overrides normal behavior

COMMAND NEURONS (invertebrate models):
  Mauthner cell (fish, Xenopus): single neuron triggers C-start escape reflex
    Acoustic/visual stimulus -> depolarize Mauthner cell -> crossed inhibition
    -> all trunk muscles on one side contract -> tail flip -> escape
    Bilateral pair; one per side; lateral inhibition ensures asymmetric response
  Crayfish tail flip: homologous system (LG, MG cells)
  Speed: 5-15ms from stimulus to full muscle activation

HORMONAL REGULATION:
  Testosterone -> aggressive behavior + song in male songbirds
    Spring: rising T -> territory establishment + singing
    Manipulation: T implant in females -> female-typical aggression
    Castrated males: reduced song + aggression; T implant restores
  Oxytocin/vasopressin: pair-bonding in voles
    Prairie vole (monogamous): high V1a receptor density in reward circuits
    Meadow vole (promiscuous): low V1a receptor density
    Experimental: V1a receptor expression in reward circuit -> monogamy-like behavior
```

### Fixed Action Patterns

```
FIXED ACTION PATTERNS (FAPs)
================================

DEFINITION: stereotyped, species-typical motor sequence
  Once initiated by sign stimulus: runs to completion even if stimulus removed
  Not learned; wired in neural circuitry
  Varies by arousal/motivation state (internal) + external sign stimulus

CLASSIC EXAMPLES:
  Greylag goose egg retrieval:
    Sign stimulus: egg outside nest (any rounded object works)
    FAP: extend neck, roll egg back with bill
    Remove egg mid-retrieval: continues rolling motion to completion
    -> Motor program runs independently once triggered

  Three-spined stickleback (Gasterosteus) aggression:
    Sign stimulus: red belly of rival male
    FAP: threat display + attack
    Experimental: model with red belly > realistic model without red
    -> Red is the key sign stimulus, not "fish" shape

  Spider web building:
    Motor sequence for web construction species-typical, stereotyped
    Modified by caffeine, amphetamines in famous NASA study
    -> pharmacological manipulation disrupts FAP sequence

DISTINCTION FROM REFLEXES:
  Reflex: fast, stereotyped, local (segmental)
  FAP: complex, species-typical, centrally coordinated
  Overlap: both involve fixed neural circuits; FAPs more extensive
```

## Question 2: Function (Adaptation)

### Cost-Benefit Analysis of Behavior

```
FUNCTION: FITNESS CONSEQUENCES
=================================

BEHAVIORAL ECOLOGY FRAMEWORK:
  Selection acts on PHENOTYPE (behavior = phenotype)
  Beneficial behavior: increases direct fitness (own reproduction)
                    OR increases inclusive fitness (kin reproduction)

OPTIMAL FORAGING (function analysis):
  Question: "Why does this animal forage this way?"
  Answer: Natural selection favored individuals that maximized energy intake rate
    (or another fitness-relevant currency under the specific constraints)
  Function: maximize fitness through energy acquisition
  Mechanism (Causation Q): sensory evaluation + decision circuits
  These are DIFFERENT questions (see Decision Cheat Sheet)

PREDATOR AVOIDANCE TRADE-OFF (function):
  Why do birds stop foraging to scan for predators?
  Function: reduce predation risk
  Trade-off: scanning reduces food intake
  Optimal vigilance: increases with predation risk; decreases with group size
  Hamilton's "selfish herd" (1971): stay in center of group -> safer
    -> herd structure emerges from individual selfish behavioral rule
```

### Sexual Selection

```
SEXUAL SELECTION (Darwin 1871)
=================================

INTRASEXUAL (same-sex competition):
  Male-male competition for mates or resources
  Results in: weapons (antlers, horns, large body size)
  Examples: elk (Cervus elaphus): antlers for male-male combat
            elephant seals: extreme sexual dimorphism (male 4x female)

INTERSEXUAL (mate choice):
  Typically female chooses among males
  Results in: ornaments, displays, songs, nuptial gifts
  Examples: peacock tail; bird of paradise display; firefly flash

RUNAWAY SELECTION (Fisher 1930):
  Initial female preference + heritable male trait -> coevolution
  Gene for preference + gene for trait linked -> autocatalytic
  -> exaggeration beyond "natural selection" optimum
  Result: extreme ornaments that reduce survival (peacock tail)

ZAHAVIAN HANDICAP (1975):
  Honest signal: only high-quality males can afford costly ornament
  Cheap ornaments: can be faked -> not informative
  Expensive handicap: survival cost is the honesty mechanism
  Examples:
    Peacock tail: costly to grow + maintain; predation risk
    Bright plumage: carotenoids in diet indicate foraging ability
    Loud calling: energy + predation risk; honest signal of condition

GOOD GENES HYPOTHESIS:
  Female choice for genes improving offspring survival (not just attractiveness)
  MHC-based mate choice: choose dissimilar MHC partners
    -> offspring have more diverse immune repertoire
  Evidence: experimental studies in fish, birds, humans
    (sweaty T-shirt experiments: women prefer MHC-dissimilar men's scent)
```

## Question 3: Development (Ontogeny)

### Learning and Instinct

```
INSTINCT vs LEARNING (a false dichotomy)
==========================================

INSTINCT (genetic programming):
  Behavior pattern appears without specific experience
  Examples:
    Newly hatched turtle heading to sea
    Spider building species-typical web (never seen one)
    Bird displaying species-typical song syllables (innate template)

LEARNING: behavior modified by experience
  Examples:
    Song learning in oscines (see ornithology/05)
    Fear conditioning
    Operant conditioning

MODERN VIEW: most behavior is BOTH
  Genes specify:
    Learning rules (what can be learned and how easily)
    Sensitive periods (when learning occurs)
    Constraints (what can and cannot be learned)
  Experience specifies:
    What specific information is encoded
    Calibration of sensory systems

  Example: Bird song
    Genes: specify learning period, species-recognition filters, motor circuits
    Experience: determines which specific song is copied
    Remove tutor: bird develops simple genetically-specified song (template only)
```

### Critical Periods

```
CRITICAL PERIODS IN BEHAVIORAL DEVELOPMENT
============================================

IMPRINTING (Lorenz):
  Young ducks/geese follow first moving object seen (hours after hatch)
  Lorenz: hand-reared goslings followed him
  Tested: model object, human, species-typical conspecific
  Critical window: first 36 hours post-hatch (approx)
  After window: imprinting no longer possible (or much harder)
  Filial imprinting: follow mother figure
  Sexual imprinting: later; determines preferred mate

SONG LEARNING CRITICAL PERIOD (see ornithology/05):
  Sensory (memorization): first months post-hatch
  Motor (sensorimotor): subsequent months
  Crystallization: fixes adult song

FEAR LEARNING:
  Juvenile: acquire fears of predators
  Mouse: pup does not show ultrasonic vocalizations to cat
    Pup exposed to cat odor + mother's fear response -> learns cat fear
    Parent teaches fear -> cultural transmission of fear
  Critical period + parental teaching = adaptive fear learning

OLFACTORY IMPRINTING (salmon):
  Salmon imprint on odor of natal stream as juveniles
  Return as adults to same stream (guided by remembered odor)
  Experimental evidence: plug olfactory organ = random stream choice
  -> Chemical fingerprint of each stream learned in critical period
```

## Question 4: Evolution (Phylogeny)

### Comparative Ethology

```
EVOLUTIONARY ANALYSIS OF BEHAVIOR
=====================================

HOMOLOGY vs ANALOGY in behavior:
  Behavioral homologs: shared behavior due to common ancestry
  Behavioral analogs: similar behavior due to convergent evolution
  Problem: behavior doesn't fossilize -> phylogenetic inference needed
  Method: map behavior on molecular phylogeny -> infer ancestral states

EXAMPLE: Maternal care in vertebrates
  Phylogenetic reconstruction: maternal care in some fish + all birds + mammals
  Was ancestral vertebrate maternal? Or multiple independent origins?
  Crocodilian parental care (guarding + transport): suggests early origin
  Dinosaur nesting (Maiasaura, Citipati): parental care predates birds
  Hormonal mechanism (prolactin): conserved across birds + mammals + some fish
  -> Homologous hormonal mechanism; convergent ecological expression

RITUALIZATION:
  Behavior that evolved from a functional original into a display
  Examples:
    Greeting ceremonies: derived from food-begging behaviors (Lorenz)
    Threat displays: derived from attack + escape motor patterns
    Courtship feeding: functional feeding -> ritualized courtship
  Mechanism: exaggeration, stereotyping, emancipation from original context
```

### Phylogenetic Methods in Behavior

```
BEHAVIORAL PHYLOGENETICS
=========================

METHOD: Map behavioral characters onto molecular phylogeny
        Use parsimony or maximum likelihood to infer ancestral states
        + number of independent origins

EXAMPLE: Vocal learning in birds
  Map "vocal learning present/absent" on avian phylogeny
  Result: vocal learning in 3 distantly related groups (songbirds, parrots, hummingbirds)
  At 3 widely separated phylogenetic positions
  INFERENCE: 3 independent evolutionary origins (not one shared ancestor)
  Convergent evolution of both: the behavior + the neural circuits

EXAMPLE: Eusociality in insects
  Map "eusocial present/absent" on insect phylogeny
  Result: ~15+ independent origins in Hymenoptera + 1 in Blattodea (termites)
  -> Multiple origins of most complex social system in nature

COMPARATIVE ANALYSIS (PICs - Phylogenetically Independent Contrasts):
  Statistical method: correct for phylogenetic non-independence
  Trait A correlated with Trait B -> is this independent evolution or
    just shared ancestry?
  PICs: use only divergences at phylogenetic nodes -> true independent tests
  Essential for any comparative behavioral analysis
```

## Tinbergen's Four Questions Applied

```
EXAMPLE APPLICATION: STICKLEBACK AGGRESSION
=============================================

BEHAVIOR: Male three-spined stickleback (Gasterosteus aculeatus)
          attacks other males with red bellies during breeding season

Q1 CAUSATION (Mechanism):
  Neural: visual system detects red; releases aggression circuitry
  Hormonal: testosterone elevated in spring -> increases sensitivity to red
  Sign stimulus: red belly (not shape; not motion specifically)
  Supernormal: red model > realistic model without red

Q2 FUNCTION (Adaptation):
  Defend breeding territory -> exclusive access to females -> fitness
  Male with own territory -> attracts female -> builds nest -> fertilizes eggs
  Selection pressure: males that defended territory more reproductively successful

Q3 DEVELOPMENT (Ontogeny):
  Red sensitivity: genetic (naive fish show response first time)
  Hormonal: testosterone rises with photoperiod in spring
  Territorial behavior: context-dependent; only when reproductively primed
  NOT learned (genetic basis confirmed by cross-breeding experiments)

Q4 EVOLUTION (Phylogeny):
  Related species (Pungitius): similar but some variation in coloration signal
  Phylogenetic analysis: red belly signal in multiple Gasterosteus lineages
  -> likely ancestral to this group; sexual selection + species recognition functions
  Defense of territory: widespread in teleost fish; ancestral character
```

## Decision Cheat Sheet

| Question | Level | Example question | Discipline |
|----------|-------|-----------------|------------|
| Causation | Proximate | "What neural pathway triggers this?" | Neuroscience; neuroethology |
| Function | Ultimate | "How does this increase fitness?" | Behavioral ecology |
| Development | Proximate | "How did this behavior develop in the individual?" | Developmental biology; learning |
| Evolution | Ultimate | "How did this behavior originate in the lineage?" | Evolutionary biology; phylogenetics |

## Common Confusion Points

**Causation and function are different questions**: The most common error in behavioral biology is conflating mechanism (Q1) with function (Q2). "Birds sing because testosterone drives the singing circuit" (Q1) is not the same as "birds sing because it helps them defend territories and attract mates" (Q2). Both are correct, necessary, and non-redundant.

**Instinct and learning are not mutually exclusive**: All "instincts" require appropriate developmental conditions; all "learned" behaviors are constrained by genetic predispositions. The dichotomy is heuristic. Modern view: "nature via nurture" — genes set up the architecture for learning to operate within.

**Tinbergen vs Lorenz**: Lorenz focused primarily on instinct and its neural organization (causation, development). Tinbergen broadened to include function and evolution as equally important explanatory levels. The 1963 paper specifying the four questions was Tinbergen's explicit synthesis.

**Fixed action patterns are not fully rigid**: FAPs vary with motivational state. A well-fed animal shows less intense food-related FAPs; a highly aroused animal shows more intense/frequent FAPs. "Fixed" refers to the motor program structure (stereotyped sequence) not to the absolute threshold or intensity.

**Supernormal stimuli in humans**: Human visual system responds to supernormal stimuli too: infantile facial features (large eyes, round face, small nose) trigger caregiving responses; exaggerated human proportions in advertising exploit these preferences. The concept generalizes across vertebrates.
