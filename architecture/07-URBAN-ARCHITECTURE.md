# Architecture in the Urban Context

## The Big Picture

No building exists in isolation. Every building is simultaneously a private artifact (serving its program and client) and a public artifact (contributing to or degrading the city around it). Urban architecture is the discipline of managing both dimensions at once.

```
+--------------------------------------------------------------------+
|                   URBAN ARCHITECTURE SCALES                        |
|                                                                    |
|  CITY SCALE (1:10,000–1:50,000)                                    |
|  Urban structure, transit networks, land use, zoning               |
|  → Urban planning, not architecture per se                         |
|                       │                                            |
|  DISTRICT / NEIGHBORHOOD SCALE (1:2,000–1:10,000)                  |
|  Urban fabric, block structure, street network,                    |
|  TOD (transit-oriented development)                                |
|  → Architecture + urban design overlap                             |
|                       │                                            |
|  BLOCK SCALE (1:500–1:2,000)                                       |
|  Building typology: perimeter block / tower / slab / podium        |
|  Figure-ground relationship, street wall, setbacks                 |
|  → Primary architecture + urban design territory                   |
|                       │                                            |
|  STREET SECTION (1:100–1:500)                                      |
|  Ground floor activation, building edge condition,                 |
|  public vs private threshold, sunlight access                      |
|  → Architecture's primary urban responsibility                     |
|                       │                                            |
|  BUILDING SCALE (1:1–1:200)                                        |
|  The building's contribution to urban experience:                  |
|  facade quality, entry, materials, scale articulation              |
|  → Architecture proper                                             |
+--------------------------------------------------------------------+
```

## Urban Typologies

### Perimeter Block

The dominant typology of European cities: buildings form a continuous street wall around the perimeter of a block, enclosing a private interior courtyard.

```
  PERIMETER BLOCK
  =================

  STREET           STREET
  ─────────────────────────
  │████████████████████████│  building
  │█                     █│
  │█   INTERIOR          █│  ← courtyard
  │█    COURTYARD        █│     (private, quiet,
  │█                     █│      green, shared)
  │████████████████████████│  building
  ─────────────────────────
  STREET           STREET

  ADVANTAGES:
  + Continuous street wall → active, safe street (Jacobs)
  + Protected interior → quiet zone, children's play, community
  + Mixed use possible: retail ground floor, residential above
  + Historic European city character (Paris, Amsterdam, Barcelona)
  + Density achievable without towers

  DISADVANTAGES:
  - All units cannot have optimal orientation
  - Ground floor units may have limited light
  - Requires land assembly (difficult in US market)
  - Requires parking to be within block or below grade

  TYPICAL BUILDING HEIGHT: 4–8 stories in European context
  American equivalent: row houses, brownstones (2–4 stories)
```

### Tower in Park

Le Corbusier's Radiant City (1930s): demolish the perimeter block, replace with towers standing in continuous parkland at high density.

```
  TOWER IN PARK TYPOLOGY
  =======================

  STREET
  ──────────────────────────────────────────────
          .          .          .
         ┌┐         ┌┐         ┌┐
         │││        │││        │││
         │││        │││        │││    towers
         │││        │││        │││    (20–40 stories)
         │││        │││        │││
         └┘         └┘         └┘

         ░░░ park ░░░░░░░░░░ park ░░░ ground plane
  ──────────────────────────────────────────────
  STREET

  LE CORBUSIER'S ARGUMENT:
  High density via height → ground liberated for parkland
  Sun and air for all units (no interior courtyards)
  Modernist hygiene — light, ventilation, greenery

  JANE JACOBS' CRITIQUE (The Death and Life of Great American Cities, 1961):
  ─────────────────────────────────────────────────────────────────────────
  The ground plane is inert — "no eyes on the street"
  Street is residual, not activated
  Towers have no surveillance of the park below
  Parks become unsafe (no natural surveillance)
  Community destroyed: no street-level interaction
  Mixed use eliminated: towers are mono-use residential

  THE EVIDENCE:
  Pruitt-Igoe (St. Louis, 1954–1972):
  33 towers, 2,870 units. Demolished 1972–1976.
  Failure attributed to: social policy + architectural
  typology + maintenance. Not just the architecture.
  But the typology cannot be separated from the outcome.

  SUCCESSFUL TOWER-IN-PARK:
  Barbican, London (Chamberlin Powell Bon, 1965–1976):
  Works because: internal streets, high-density social mix,
  maintained management, arts center (ground activation).
  Exception that proves the rule.
```

### Podium + Tower

The dominant contemporary typology for high-density urban development, particularly in East Asian cities.

```
  PODIUM-TOWER TYPOLOGY
  =======================

                    ┌────┐
                    │    │
                    │    │   tower
                    │    │   (residential / hotel / office)
                    │    │
              ──────┤    ├──────
              │ P  O  D  I  U  M │   (4–8 stories)
              │ retail / office   │
              │ amenity / parking │
              └───────────────────┘
              ░░░░░ ground activation ░░░░░

  RATIONALE:
  Podium: large floor plates (commercial, parking, amenity)
          Street-level activation
          Scale transition between tower and street
  Tower: thin floor plates (residential, hotel — smaller programs)
         Views, light
         Maximum density with minimum ground footprint

  ADVANTAGES:
  + Ground floor activation (podium activates street)
  + Parking concealed in podium (no surface lots)
  + Scale transition: podium at human scale, tower above
  + Mixed use: retail/office podium + residential tower

  TYPICAL IN:
  Hong Kong, Singapore, Seoul, Vancouver, Manhattan
  Increasingly common in US TOD developments.
```

### Slab Block

The long, thin residential slab: maximizes units with direct sunlight on each facade.

```
  SLAB BLOCK TYPOLOGY
  ====================

  PLAN VIEW:

  ─────────────────────────────────────────────  (south-facing units)
  │                                             │
  │              SLAB BLOCK                    │
  │                                             │
  ─────────────────────────────────────────────  (north-facing units)

  Often oriented east-west (long axis)
  → south-facing units get solar gain
  → north-facing units get diffuse light

  ADVANTAGES:
  + Efficient unit layout (double-loaded corridor)
  + Maximizes daylight access (no interior units)
  + Simple structure (repetitive bays)

  DISADVANTAGES:
  - Ends face urban context awkwardly (blank end walls)
  - Urban grain: slab blocks create a very different
    street texture than perimeter blocks
  - Scale: typical slab is 100–200m long — much longer
    than a perimeter block face
  - "Wall of housing" criticism: monolithic at neighborhood scale
```

## Jane Jacobs and Street Life

Jacobs's *The Death and Life of Great American Cities* (1961) remains the foundational text for understanding what makes urban streets safe, vital, and economically healthy.

### The Four Generators of Urban Diversity

```
  JACOBS' FOUR CONDITIONS FOR URBAN VITALITY
  =============================================

  1. MIXED PRIMARY USES
     Different functions attract people at different hours.
     Offices (daytime) + residences (evening/weekend) + schools
     → 18-hour activity, not just 9-5
     → Eyes on the street across the day

  2. SHORT BLOCKS
     Frequent corners → multiple routes → pedestrian choice
     Long blocks = dead middle zones
     Target: 200–300ft (60–90m) maximum block face
     Manhattan's 250ft grid vs Brooklyn's 900ft superblocks

  3. OLD BUILDINGS
     New buildings cost more → high rent → chain stores + banks
     Old buildings → cheap rent → independent businesses, artists
     The mix of building ages creates economic diversity
     "Lively districts are not all new" — they mix eras

  4. SUFFICIENT DENSITY
     Enough people per block to support local commerce
     Minimum for viable retail: ~75 residential units/acre
     Suburbia at 4 units/acre = car dependency
     Urban vitality requires 40–200+ units/acre

  THE OPPOSITE OF JACOBS = LE CORBUSIER'S RADIANT CITY:
  Single use (residential towers)
  Long blocks (towers in park)
  All new buildings (clearance of old neighborhoods)
  Insufficient street-level density (ground plane unused)
```

### Eyes on the Street

```
  NATURAL SURVEILLANCE PRINCIPLE
  =================================

  Safe streets require "eyes on the street":
  People who can see the street from windows and stoops.
  Ground-floor activation creates these eyes.

  UNSAFE:                        SAFE:
  ┌────────────────┐             ┌────────────────┐
  │ BLANK WALL     │             │ RETAIL / CAFE  │ ← eyes
  │                │             │ ────────────── │
  │ (parking,      │             │ STOOP / ENTRY  │ ← eyes
  │  loading,      │             │ ────────────── │
  │  service)      │             │ RESIDENTIAL    │ ← eyes
  │                │             │ WINDOWS        │
  └────────────────┘             └────────────────┘
       ↓ STREET ↓                      ↓ STREET ↓
  No surveillance.               Continuous surveillance.
  No activity.                   Activity at multiple hours.
  Unsafe at night.               Safer at all hours.

  ARCHITECTURAL DESIGN IMPLICATION:
  Ground floor facing street: MUST be active (retail, lobby,
  residential stoop, public amenity — never blank wall, parking,
  or mechanical).
  This is now codified in many form-based codes as
  "active frontage" requirements (% of street-facing facade
  that must be glazed / occupied).
```

## Public Space Design

### Whyte's Street Life Lessons

William H. Whyte studied New York plaza behavior using time-lapse photography (*The Social Life of Small Urban Spaces*, 1980). His findings are empirical and counter-intuitive.

```
  WHYTE'S DESIGN VARIABLES FOR SUCCESSFUL PUBLIC SPACE
  ======================================================

  FOOD:
  People go where food is.
  A coffee cart transforms an empty plaza.
  Fixed food vendors > food carts > food trucks.
  The strongest predictor of plaza use.

  SUN:
  People seek sun in cold climates.
  Move seasonally — track sun.
  Southern exposure in northern hemisphere is prized.
  NYC's famous "full sun" plazas are crowded; shadowed ones empty.

  MOVEABLE SEATING:
  Fixed benches: people sit where they must.
  Moveable chairs: people sit where they choose → they CHOOSE.
  The act of moving a chair to position = personalization.
  Average chair movement: 1–2 feet from where they found it.
  Moveable chairs increase use vs fixed.

  TRIANGULATION:
  Something that causes strangers to interact.
  Art, performer, fountain, unusual event.
  "Without triangulation, there is no reason to talk to a stranger."

  WATER:
  Moving water → sound that masks city noise.
  Misting in summer → thermal comfort.
  Children play; adults watch children play.
  Even a small fountain significantly increases plaza use.

  TREES:
  Shade in summer.
  Enclosure and definition.
  Falling leaves = temporal experience of seasons.

  FAILED PLAZA CHARACTERISTICS:
  - Hidden from street (recessed, raised, steps down)
  - No food
  - Fixed, anti-social seating (single-person benches, spikes)
  - No sun
  - No shelter
  - Blank walls on surrounding building faces
```

### Sunlight Rights

```
  SUN ACCESS IN URBAN DESIGN
  ===========================

  NEW YORK 1916 ZONING:
  The first major US zoning law addressed daylight.
  Setback requirements (wedding-cake profiles):
  Buildings must step back from street as they rise
  to allow daylight to reach the street.

  EFFECT: NYC's Art Deco wedding cake skyline.
  Empire State Building, Chrysler Building profiles
  are direct expressions of the 1916 zoning envelope.

  LONDON DAYLIGHT RIGHTS:
  "Ancient lights" doctrine: owners of existing buildings
  have right to daylight they have historically received.
  New buildings can be challenged on daylight grounds.
  BRE (Building Research Establishment) daylight assessments
  required for planning approval of tall buildings.
  Vertical Sky Component (VSC) and Average Daylight Factor (ADF)
  are the metrics used.

  SHADOW IMPACT ANALYSIS:
  Part of planning applications for new buildings.
  Hourly sun / shadow diagrams for equinoxes and solstices.
  Computer-modeled in Rhino, Revit, SketchUp.
  Quantifies shadow impact on streets, parks, neighboring buildings.
```

## Transit-Oriented Development (TOD)

### The Half-Mile Walkshed

```
  TOD PRINCIPLES
  ===============

  WALKSHED CONCEPT:
  Most people will walk 1/4 mile (400m) to transit.
  Some will walk 1/2 mile (800m).
  Beyond 1/2 mile = car trip to transit (defeats the purpose).

  ┌──────────────────────────────────────────────┐
  │                                              │
  │           1/2 mile radius                    │
  │        ┌─────────────────────────────┐       │
  │        │                            │        │
  │        │       1/4 mile radius      │        │
  │        │    ┌──────────────┐        │        │
  │        │    │              │        │        │
  │        │    │   [STATION]  │        │        │
  │        │    │              │        │        │
  │        │    └──────────────┘        │        │
  │        │        high density        │        │
  │        │        mixed use           │        │
  │        └────────────────────────────┘        │
  │                medium density                │
  └──────────────────────────────────────────────┘
               lower density at edge

  DENSITY GRADIENT:
  Highest density immediately at station (podium towers)
  → Medium density within 1/4 mile (mid-rise mixed use)
  → Lower density at 1/4–1/2 mile (townhouses, garden apartments)
  → Single-family beyond (transition to neighborhood fabric)

  SUCCESSFUL US TOD EXAMPLES:
  Pearl District, Portland (streetcar + mixed use)
  Rosslyn-Ballston Corridor, Arlington VA (Metro + density)
  Del Mar Village, Pasadena (Metro + retail + residential)

  FAILED TOD PATTERNS:
  Station with surface parking → not TOD, just P&R
  Station with single-use office park → 9-5 activity only
  Station surrounded by car-oriented retail → hostile pedestrian
```

### Parking Minimums vs Maximums

```
  THE PARKING PARADIGM SHIFT
  ============================

  CONVENTIONAL ZONING: minimum parking requirements
  "Office: 4 spaces per 1,000 sqft"
  This mandates that every project provides abundant parking,
  which mandates car dependency, which reduces walkability,
  which reduces transit viability → self-reinforcing loop.

  TOD APPROACH: parking maximums (not minimums) at transit
  "Office within 1/4 mile of transit: max 1 space per 1,000 sqft"
  Forces development to be transit-served, not car-dependent.
  Frees land from parking (12% of typical US cities is parking lots).

  LAND VALUE:
  Surface parking: ~$5,000/space land value
  Structured parking: $20,000–40,000/space construction cost
  That land at TOD density: $200–500+/sqft (multiply by area)
  Replacing a surface parking lot with TOD development:
  typically 100–500× land value created.
```

## Climate Resilience

### Urban Heat Island

```
  URBAN HEAT ISLAND (UHI) EFFECT
  ================================

  Urban areas are 2–10°C warmer than surrounding rural areas.

  CAUSES:
  - Impervious surfaces (asphalt, concrete) absorb heat
  - Low albedo (dark surfaces) → more solar absorption
  - Reduced evapotranspiration (no vegetation)
  - Waste heat from buildings, cars, AC units
  - Canyon effect: buildings trap re-radiated heat

  UHI MITIGATION STRATEGIES:
  ────────────────────────────────────────────────────────────────
  Green roofs:      Evapotranspiration + albedo improvement
                    Up to 3°C local cooling
  Cool pavements:   Reflective asphalt (albedo 0.30 vs 0.05)
                    Permeable pavement (infiltration → cooling)
  Street trees:     Shade (reduces surface temperature 10–20°C)
                    Evapotranspiration
                    Wind comfort
  Water features:   Evaporative cooling (~3–5°C local effect)
  Albedo of roofs:  White/reflective roofs: albedo 0.70–0.85
                    vs black roof: albedo 0.05–0.10
                    "Cool roof" requirement in several US energy codes

  TEMPERATURE IMPACT OF TREE CANOPY:
  Surface under tree: 10–20°C cooler than exposed asphalt
  Ambient temperature: 1–3°C cooler with 30%+ canopy cover
  City-wide: measurable mortality reduction during heat events
```

### Flood Resilience and Sponge Cities

```
  FLOOD RESILIENCE STRATEGIES
  =============================

  CONVENTIONAL (gray infrastructure):
  Rain hits impervious surface → runoff → storm drain → overwhelm
  Design event: 25-year or 100-year flood
  Pipe everything to combined sewer → sewer overflow in large events

  SPONGE CITY (green + gray integrated):
  Distribute water retention throughout urban fabric.
  Reduce peak flows by absorbing at source.

  ┌──────────────────────────────────────────────────────────────┐
  │  SPONGE CITY COMPONENTS                                      │
  │                                                              │
  │  GREEN ROOFS          Retain 60–80% of rainfall event        │
  │  BIOSWALES            Linear infiltration channels           │
  │  RAIN GARDENS         Planted infiltration basins            │
  │  PERMEABLE PAVEMENT   Infiltration through surface           │
  │  URBAN WETLANDS       Large-scale storage + infiltration     │
  │  RETENTION PONDS      Storage + delayed release              │
  │  TREE PITS            Subsurface infiltration                │
  └──────────────────────────────────────────────────────────────┘

  BEIJING CASE (2021 floods): massive flooding exposed failure
  of conventional drainage for extreme events.
  Post-event: accelerated sponge city implementation.
  Target: 80% of Beijing's urban area absorbs 70% of rainfall.

  BUILDING-SCALE CONTRIBUTION:
  Green roof → significant retention
  Disconnecting downspouts → rain garden (not combined sewer)
  Underground cistern → reuse for irrigation
  These are architectural decisions, not just civil engineering.
```

### Sea Level Rise

```
  COASTAL ADAPTATION TYPOLOGIES
  ================================

  PROTECT:                    ACCOMMODATE:
  ────────────                ─────────────────────
  Seawall                     Elevated ground floors
  Levee system                Flood-proof construction below
  Storm surge barrier         habitable level
  (Thames Barrier, Rotterdam) Amphibious foundations
                              (float on flood, sit on ground
  RETREAT:                     normally)
  ──────
  Managed coastal retreat     Living shorelines:
  Buyout of flood-prone areas → Natural systems (salt marsh,
  Relocate structures back        mangroves, oyster reefs) as
  from coast                      coastal protection
                                  More resilient than hard walls
                                  + habitat value

  FLORIDA REALITY (2024):
  Miami Beach: pumps + road raising (accommodate)
  FEMA flood maps: determine building elevation requirements
  IBC flood provisions: minimum first floor elevation (FFE)
  NFIP (National Flood Insurance Program):
  Insurance rates calibrated to elevation above BFE
  (Base Flood Elevation = 100-year flood level)

  ARCHITECTURAL IMPLICATION:
  In coastal Flood Zone A or V: first floor elevated on
  piers, columns, or fill.
  In AE or VE zones: specific elevation certificate required.
  All mechanical/electrical above BFE.
```

## Tall Buildings

### Why Buildings Became Tall

```
  ENABLING TECHNOLOGIES FOR TALL BUILDINGS
  ==========================================

  1852:  Elisha Otis invents the safety elevator (brake mechanism)
         → Vertical transit becomes safe → upper floors valuable
         → Economics of height possible

  1880s: Steel skeleton construction (Home Insurance Building, 1885)
         → Load-bearing masonry required very thick walls at base
         → Steel frame: slender columns, curtain wall can fill
         → Height no longer limited by base wall thickness

  1900s: Reinforced concrete → economical for floors, shear walls
  1960s: Computer structural analysis → complex systems optimized
  1970s: Lightweight high-strength glass curtain wall systems
  1990s: GPS/laser surveying → tall structures built plumb at scale
  2000s: Computational fluid dynamics → wind tunnel equivalent

  ECONOMICS OF HEIGHT:
  Elevator core efficiency drops with height:
  At 10 stories:  elevators = 10–12% of floor plate
  At 50 stories:  elevators = 20–25% of floor plate
  At 100 stories: elevators = 30–35% of floor plate
  → Sky lobbies (double-deck elevators, express/local separation)
    allow some efficiency recovery above ~50 stories
```

### Structural Systems for Tall Buildings

```
  STRUCTURAL SYSTEMS BY HEIGHT
  ==============================

  1–15 stories:   Moment frame / shear walls
                  Standard office / residential
                  Wind: minor factor

  15–40 stories:  Core + moment frame or braced frame
                  Wind begins to govern lateral design

  40–80 stories:  Core + outrigger
                  Central core (elevator + stair + shear walls)
                  + Outrigger trusses at mechanical floors
                  + Belt trusses at perimeter
                  Very efficient (WTC, Hancock Chicago)

  80–150 stories: Tube systems
                  Framed tube: dense perimeter columns + beams
                              (WTC originator: Fazlur Khan)
                  Bundled tube: multiple tubes (Sears Tower)
                  Tube-in-tube: inner core tube + outer shell tube

  150+ stories:   Diagrid, supertall systems
                  Core + mega-columns + outriggers
                  (Burj Khalifa: Y-shaped plan reduces wind)

  WIND IS THE GOVERNING STRUCTURAL FORCE ABOVE ~40 STORIES.
  Not gravity loads — the structure is large enough that
  wind overturning moment dominates. This drives:
  - Core sizing (moment arm to perimeter)
  - Damping (TMD — tuned mass damper in Taipei 101, 660 tonnes)
  - Form optimization (tapered, aerodynamic profiles)
```

### Supertalls and Urban Impact

```
  TALL BUILDING URBAN IMPACT
  ============================

  POSITIVE:
  + Skyline identity (Chrysler, Empire State, Sears, Burj)
  + Land use efficiency (vertical density over small footprint)
  + Economic activity concentration at transit nodes

  NEGATIVE:
  - Shadow: 1,000 ft tower casts 1/4 mile shadow
    (impacts street life, parks, neighboring buildings)
  - Wind: tall buildings create downdrafts at base
    "Wind streets" in Midtown Manhattan (6th Ave corridor)
  - Visual intrusion: visible from miles away
  - Infrastructure demand: 50,000-person tower requires
    subway, power, water infrastructure at building base

  PODIUM AS URBAN RESPONSE:
  Podium at human scale (4–6 stories) provides:
  - Continuity with street wall
  - Ground-floor activation (retail, lobby, amenity)
  - Scale transition between tower and pedestrian
  Tower above: set back, less impactful at street level

  CONTEXTUAL vs ICONIC:
  Contextual: building references surrounding scale, material,
  typology. Fits into the street fabric.
  Iconic: building announces itself as singular, different.
  Neither is inherently right — depends on urban context.
  An iconic building that ignores its street context
  undermines the urban fabric it occupies.
```

## Decision Cheat Sheet

| Urban design situation | Typological response | Key principle |
|-----------------------|---------------------|---------------|
| High-density residential without towers | Perimeter block (European tradition) | Street wall activation + courtyard relief |
| Mixed-use development at transit station | Podium + tower hybrid | Podium activates street; tower achieves density |
| Revitalizing a dead plaza | Whyte: add food, moveable seating, sun access, water | Activity begets activity |
| Reducing urban heat island | Green roofs + trees + cool pavement | 30%+ tree canopy is most effective single measure |
| New development in flood zone | Accommodate: elevated FFE, flood-proof below grade, sponge elements | FEMA NFIP governs minimum; exceed it for resilience |
| TOD development zone | Park minimums → park maximums; density gradient from station | 1/4 mile walkshed is the economic unit |
| Long blank building wall | "Active frontage" design: windows, stoops, retail bays | Eyes on the street = Jane Jacobs' principle |
| Tall building context | Podium at street scale + tower setback above | Human scale at grade; iconic above |

## Common Confusion Points

**Urban design vs urban planning**: Urban design is the physical design of cities — building form, street sections, public space. Urban planning is the policy, regulatory, and economic framework — zoning, land use, transportation planning. They overlap but are different professions (MArch + urban design vs. MUP planning degree).

**Setback vs build-to line**: A setback is a minimum distance from the street (building must be at least X feet from curb). A build-to line is where the building MUST be (zero setback — building goes right to the line). Build-to lines create continuous street walls; setbacks create fragmented suburban frontages. Seemingly a small regulatory difference; enormous urban design consequence.

**TOD vs transit-adjacent development**: TOD is walkable, mixed-use, dense development organized around transit. Transit-adjacent development (TAD) is development that happens to be near transit but is still car-oriented (surface parking, big setbacks, single use). Many developments claiming to be TOD are actually TAD.

**Starchitecture critique**: The "iconic building" that ignores its street context — all attention on the sculptural building form, none on how it meets the ground, activates the street, or relates to neighbors. The Guggenheim Bilbao is celebrated as an urban catalyst. But a building that succeeds as sculpture while failing as urban architecture is a trophy, not a contribution.

**Wind at tall building base**: Wind impacts from tall buildings are a real engineering and urban design problem. When wind hits a tall building, it is deflected downward (the building acts as a bluff body). The ground-level wind can be 150–200% of free-stream wind speed. This has killed pedestrians in Chicago's Loop. Wind tunnel testing and CFD analysis of pedestrian wind comfort are required for tall buildings in many jurisdictions.
