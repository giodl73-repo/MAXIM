# The Kraft Process: Chemical Pulping

## The Big Picture

```
+------------------------------------------------------------------+
|              THE KRAFT MILL — FULL SYSTEM VIEW                   |
|                                                                  |
|  WOOD      CHIP BIN   DIGESTER    BLOW      BROWN STOCK         |
|  YARD  →  (screened) → (cooking) → TANK  →  WASHING →          |
|                           |                                      |
|                    BLACK LIQUOR           SCREENING              |
|                           |                   |                  |
|                    EVAPORATION         BLEACH PLANT              |
|                           |                   |                  |
|                    RECOVERY BOILER     BLEACHED PULP             |
|                           |                   |                  |
|                    CAUSTICIZING       TO PAPER MACHINE           |
|                           |          or DRYING MACHINE           |
|                    WHITE LIQUOR                                  |
|                    (recycled)                                    |
|                                                                  |
|  CHEMICAL LOOP: NaOH + Na2S → Black liquor → Recovery → White L.|
+------------------------------------------------------------------+
```

The kraft process is 45–50% yield on dry wood, but it's a complete chemical
recovery system. The combustion of dissolved organics in the recovery boiler
generates ~70% of the mill's energy. Modern kraft mills are essentially
energy self-sufficient and often net electricity exporters.

## Wood Preparation

### Debarking and Chipping

```
   LOG → DRUM DEBARKER → CHIPPER → CHIP SCREEN

   Drum debarker: logs tumble against each other → bark removed
   Disk chipper: rotating disk with 4–8 blades
   Chip size target: 25 × 20 × 4 mm (length × width × thickness)

   WHY CHIP UNIFORMITY MATTERS:
   ──────────────────────────────
   Cooking liquor penetrates from chip surfaces
   Thick chip center: under-cooked (high kappa number, fiber damage)
   Thin chip: over-cooked (cellulose degradation, low yield)
   Target: all chips cooked to same degree in same time

   CHIP BULK DENSITY: ~200 kg/m³ (bone dry equivalent)
```

## The Cooking Stage (Digestion)

### White Liquor Composition

```
   WHITE LIQUOR (fresh cooking chemical):
   ────────────────────────────────────────
   NaOH    (sodium hydroxide):    active delignification agent
   Na2S    (sodium sulfide):      accelerates delignification
   Na2CO3  (sodium carbonate):    inactive alkali
   Na2SO4  (sodium sulfate):      sodium balance
   NaHS    (sodium bisulfide):    reductant

   EFFECTIVE ALKALI (EA) = NaOH + ½Na2S  [as Na2O equivalents]
   SULFIDITY = Na2S / (NaOH + Na2S) × 100%
   Typical sulfidity: 25–30%

   Why sulfide matters:
   Na2S → HS⁻ + OH⁻   (in alkaline solution)
   HS⁻ accelerates dissolution of lignin fragments
   Prevents "condensation" reactions that would lock lignin in fiber
   Kraft = 2–3× faster lignin removal vs. soda process (NaOH only)
```

### The Kraft Cooking Reaction

```
   TEMPERATURE PROFILE (batch digester, simplified):
   ──────────────────────────────────────────────────
   Impregnation: 80–115°C   — liquor penetrates chip pores
                             — no significant delignification yet

   Initial cooking: 115–140°C — rapid dissolution of extractives
                               — some lignin removal begins

   Main cooking: 150–175°C  — primary delignification
   Time at max T: 30–90 min — depends on target kappa number

   THE CLEAVAGE REACTION:
   ───────────────────────
   Lignin β-O-4 aryl ether bonds: ~50% of all lignin bonds
   HS⁻ attacks → cleaves → smaller fragments dissolve in alkali
   Cellulose: peeling reaction at chain ends (yield loss)
   Target: minimize cellulose degradation while maximizing lignin removal
```

### Kappa Number — Residual Lignin

```
   KAPPA NUMBER:
   ──────────────
   Standard test: permanganate oxidation of residual lignin
   1 kappa number ≈ 0.15% residual lignin (softwood)
   1 kappa number ≈ 0.13% residual lignin (hardwood)

   TARGET KAPPA BY APPLICATION:
   ──────────────────────────────
   Unbleached kraft liner: kappa 30–45  (brown paper, high yield)
   Bleachable SW kraft:    kappa 25–30  (max bleachability)
   Bleachable HW kraft:    kappa 16–20  (hardwood reaches lower)
   Dissolving pulp:        kappa 10–15  (viscose/lyocell)

   MODERN TARGETS: Extended cooking → kappa 18–22 (SW) while
   maintaining viscosity — reduces bleaching chemical demand
```

### Continuous vs. Batch Digesters

```
   CONTINUOUS DIGESTER (Kamyr-type):
   ───────────────────────────────────
   Chips fed from top, cook liquor counter-current
   Continuous withdrawal at bottom
   20–30 m tall vessels
   Steady-state operation — consistent kappa number
   70–90% of new installations

   BATCH DIGESTER:
   ───────────────
   Fill → cook → blow → wash
   Cycle time: 2–5 hours
   More flexible — easy grade changes
   "SuperBatch" / "RDH": heat recovered between batches
   Still used in specialty operations
```

## Black Liquor and the Recovery Cycle

### What Is Black Liquor?

After cooking, the liquor contains:
- Dissolved lignin fragments (~35% of wood input)
- Dissolved hemicelluloses (~20% of wood input)
- Cooking chemicals (Na salts)
- Water (~85% of black liquor by weight)

```
   BLACK LIQUOR COMPOSITION (dry solids):
   ────────────────────────────────────────
   Organic materials: ~65–70%
     Lignin fragments, carboxylates, extractive derivatives
   Inorganic:         ~30–35%
     Na2S, Na2SO4, Na2CO3, Na2SO3

   Heating value: ~14,000–15,000 kJ/kg dry solids
   (similar to low-grade coal)
```

### Evaporation

Black liquor exits washing at ~15–18% dry solids. Must be concentrated to
65–70% for safe combustion in recovery boiler.

```
   EVAPORATION TRAIN:
   ───────────────────
   Weak black liquor (15%) → Multiple-effect evaporators
   5–7 effects: each successive stage uses vapor from previous
   → Concentrated black liquor (65–70% dry solids)
   → Energy-efficient: ~3–4 kg steam to evaporate 1 kg water

   Soap (tall oil precursor) separated by decanting during evaporation
   → Acidified → crude tall oil → refinery byproduct ($0.15–0.25/kg)
```

### Recovery Boiler — The Heart of the Kraft Mill

```
   RECOVERY BOILER FUNCTION:
   ──────────────────────────
   1. BURN organic matter → heat → steam → electricity (energy)
   2. REDUCE Na2SO4 → Na2S (chemical recovery)
   3. Na2S + Na2CO3 = smelt → dissolve in water → green liquor

   COMBUSTION ZONES:
   ──────────────────
   Upper: oxidizing zone — burns organic compounds → CO2 + H2O
   Lower: reducing zone — Na2SO4 reduced to Na2S
   Char bed: Na2S + Na2CO3 smelt accumulates

   STEAM PRODUCTION:
   ──────────────────
   Modern recovery boiler: 3–5 MPa steam at 450–480°C
   Steam turbine: generates electricity for mill
   A large kraft mill (500,000 t/yr) recovery boiler:
   Steam: 3–4 million kg/day
   Electricity: 100–150 MW (often net exporter to grid)

   CRITICAL HAZARD — SMELT-WATER EXPLOSION:
   ──────────────────────────────────────────
   Molten smelt (800°C+) contact with water → violent steam explosion
   Boiler floor water leaks: catastrophic risk
   Annual inspections + leak detection systems mandatory
```

### Causticizing — Regenerating White Liquor

```
   GREEN LIQUOR → CAUSTICIZING → WHITE LIQUOR
   ─────────────────────────────────────────────
   Green liquor: Na2S + Na2CO3 dissolved in water

   Add Ca(OH)2 (slaked lime):
   Na2CO3 + Ca(OH)2 → 2NaOH + CaCO3↓

   Result:
   White liquor: NaOH + Na2S (back to cooking chemical)
   CaCO3 (lime mud): filter off → lime kiln

   LIME KILN:
   ──────────
   CaCO3 → CaO + CO2  at 900–1100°C (rotary kiln)
   CaO + H2O → Ca(OH)2  (slaker — regenerates lime)
   Loop: lime mud → kiln → quicklime → slaker → milk of lime → causticizing
```

## Brown Stock Washing

After cooking, cellulose fibers are entangled with black liquor. Washing
extracts as much black liquor as possible (to recover chemicals and heat).

```
   COUNTER-CURRENT DRUM WASHING:
   ──────────────────────────────
   Wash water flows counter to pulp flow
   3–5 washing stages
   Dilution factor: 2–4 kg water per kg dry pulp

   DILUTION FACTOR / WASH EFFICIENCY:
   ────────────────────────────────────
   Higher wash water → better chemical recovery but more dilute BL
   → More evaporation energy needed
   Optimization: balance chemical loss vs. evaporation cost

   Ideal: wash loss < 5 kg Na2O per ton pulp
```

## Screening and Cleaning

```
   KNOTS: incompletely cooked chip bundles → pressure screens remove
   SHIVES: fiber bundles (fiber-in-fiber aggregates) → fine screens
   DIRT: bark, sand, mineral → centrifugal cleaners
```

## Bleaching

Unbleached kraft pulp is brown (kappa ~25–30). To make white paper, residual
lignin must be removed without degrading cellulose.

### Bleaching Stages — The Modern Sequence

```
   ELEMENTAL CHLORINE FREE (ECF) — DOMINANT GLOBALLY:
   ────────────────────────────────────────────────────
   D  — ClO2 (chlorine dioxide): most effective lignin oxidant
   EOP — alkaline extraction + O2 + H2O2: remove oxidized lignin
   D  — ClO2 second stage: final brightness increase
   Example 3-stage: D(EOP)D → ISO 90–92

   WHY NOT ELEMENTAL CHLORINE (Cl2):
   ───────────────────────────────────
   Cl2 + lignin fragments → chlorinated organics (AOX)
   Dioxins, furans detectable in some effluents pre-1990
   ECF: ClO2 generates far lower AOX, comparable brightness
   Transition: ~1990s in Scandinavia, US; now global standard

   TOTALLY CHLORINE FREE (TCF):
   ─────────────────────────────
   O → (A) → Q → (OP) → Q → (PO)
   Ozone (O), peracetic acid (A), chelation (Q), H2O2 (P)
   No chlorine compounds at all
   Slightly lower brightness ceiling, higher chemical cost
   Used for tissue, food contact, or Nordic environmental branding

   PRE-BLEACHING (oxygen delignification):
   ────────────────────────────────────────
   Now standard before chlorine-based bleaching
   O2 + NaOH at 100°C removes 40–50% of residual lignin
   → Reduces bleaching chemical demand 40–50%
   → Less AOX in effluent
   → Improves chemical recovery efficiency
```

### Target Brightness Levels

| Grade | ISO Brightness |
|-------|---------------|
| Brown kraft liner | 25–35% |
| Unbleached bag | 35–50% |
| Bleached SW kraft | 88–92% |
| Bleached HW kraft | 88–93% |
| Optical brightened (OBA) | 100–115% (fluorescent) |

## Mill Effluent and Environmental Controls

```
   EFFLUENT SOURCES:
   ──────────────────
   Condenser water from evaporation
   Bleach plant effluent
   Washing filtrates
   Spills (controlled)

   TREATMENT:
   ──────────
   Primary: clarifier (settle suspended solids)
   Secondary: activated sludge (biological oxidation of BOD)
   Tertiary: optional for specific compounds

   KEY METRICS:
   ────────────
   BOD7 (biological oxygen demand): <2–5 kg/ton pulp (modern mill)
   AOX: <0.1 kg/ton pulp (ECF)
   Color: brown from lignin residuals
   Modern mills: large reduction vs. 1980s — closed water loops, better retention
```

## Closed-Loop Process Architecture Bridge

The kraft chemical recovery cycle is a textbook closed-loop system — the architectural pattern that makes the process economically viable:

```
KRAFT RECOVERY CYCLE — CLOSED-LOOP ARCHITECTURE
=================================================

  Wood chips + white liquor (NaOH + Na2S)
              │
              ▼
        DIGESTER (batch, 170°C, 2 hr)
         Lignin dissolved → "black liquor" (pulp + spent chemicals)
              │
       ┌──────┴──────┐
       │             │
    PULP           BLACK LIQUOR
    (washed,        (25% solids: lignin + Na compounds)
    bleached)              │
                           ▼
                  MULTIPLE EFFECT EVAPORATORS
                  Concentrate to 65–80% solids
                           │
                           ▼
                    RECOVERY BOILER  ◄── THE ARCHITECTURAL INSIGHT
                  ┌─────────────────────────────────────────────┐
                  │  DUAL ROLE:                                  │
                  │  1. Burn organics (lignin) → STEAM → POWER  │
                  │     (mill is energy self-sufficient or net   │
                  │      exporter — pulp mill as power plant)   │
                  │  2. Na2SO4 reduced to Na2S in smelt →       │
                  │     CHEMICAL REGENERATION                    │
                  └─────────────────────────────────────────────┘
                           │ (smelt: Na2S + Na2CO3)
                           ▼
                    CAUSTICIZER
                  Na2CO3 + Ca(OH)2 → NaOH + CaCO3
                  CaCO3 → CaO (lime kiln) → Ca(OH)2 (slaker)
                           │
                           ▼
                    WHITE LIQUOR (NaOH + Na2S)
                           │
                           └─────────────────────► back to digester

ENGINEERING SIGNIFICANCE:
  Without recovery: NaOH and Na2S are consumables → process is uneconomical
  With recovery: chemicals cycle indefinitely; only makeup additions needed
  Energy balance: kraft mill burns lignin → self-sufficient; sells surplus
  This is the same principle as any closed chemical loop in process engineering
  (chlor-alkali, ammonia-urea, petroleum refining) — waste stream as feedstock
  is the architectural decision that determines economic viability.
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| Why sulfide in kraft? | Accelerates lignin removal, prevents condensation |
| What is kappa number? | Measure of residual lignin (1 kappa ≈ 0.15% lignin) |
| What is the recovery boiler's dual role? | Burn organic for energy + reduce Na2SO4 → Na2S |
| Why not use chlorine for bleaching today? | AOX (dioxin risk), ECF (ClO2) achieves same brightness |
| Why is kraft yield only ~50%? | Lignin + some hemicellulose dissolved in cooking |
| What is causticizing? | Na2CO3 + Ca(OH)2 → regenerate NaOH for white liquor |
| How does the mill achieve energy self-sufficiency? | Recovery boiler burns black liquor organics |

## Common Confusion Points

**Black liquor is not waste**: It is the carrier of all dissolved organics and
the primary fuel for the mill's energy system. A kraft mill that cannot
efficiently recover black liquor chemicals and heat is not economically viable.
The recovery boiler + chemical loop is what makes kraft economical despite
only 50% yield.

**"Kraft" means strength, not sulfate**: The German/Swedish "kraft" (= strength)
refers to the fiber quality. The process is also called "sulfate process"
because Na2SO4 was historically the sulfur makeup chemical (it gets reduced to
Na2S in the recovery boiler). Both names refer to the same process.

**Oxygen delignification is not optional in modern mills**: Early mills went
from unbleached to bleached in one step. Modern mills run O-stage first,
reducing chlorine chemical consumption by ~40% and dramatically cutting AOX.
Without O-stage, ECF mills would use 3–4× more ClO2.
