# Drain, Waste & Vent (DWV)

## Gravity, Traps, P-Traps, Vent Stacks, AAVs

## The Big Picture

```
DWV SYSTEM: THREE INTERACTING FUNCTIONS

  ┌──────────────────────────────────────────────────────────────┐
  │  DRAIN: moves wastewater from fixture to sewer               │
  │  WASTE: carries non-toilet fixture waste (sinks, tubs,      │
  │         dishwashers, washing machines)                       │
  │  VENT: maintains atmospheric pressure throughout system      │
  │        → prevents trap siphon                               │
  └──────────────────────────────────────────────────────────────┘

  WHY VENTING IS MANDATORY:
    Moving water column in pipe displaces air ahead, creates
    negative pressure (vacuum) behind it.
    Without vent: negative pressure siphons the trap water out
    → empty trap → sewer gas enters building.
    Vent connects drain to atmosphere → pressure equilibrium →
    trap water stays in place.

  THE FULL FLOW PATH:
  Fixture
     ↓ (trap: water seal here)
  Branch drain (sloped 1/4"/ft toward stack)
     ↓
  Vertical stack
     ↓
  Building drain (near-horizontal, slope to sewer)
     ↓
  Sewer lateral
     ↓
  Municipal sewer or septic system

  Vent pipe runs alongside/from drain system
  Terminates ABOVE ROOF → atmospheric pressure maintained
```

## The Physics of Drain Flow

```
  ALL DRAINS FLOW BY GRAVITY (no pump)

  HORIZONTAL DRAIN SLOPE:
    Standard: 1/4" drop per foot of horizontal run
    → 2% grade
    → water flows freely, carries solids

  WHY SLOPE MATTERS:
  ┌──────────────────────────────────────────────────────────┐
  │  TOO SHALLOW (< 1/8" per foot):                         │
  │    Water moves slowly                                    │
  │    Solids settle, accumulate → chronic clogging         │
  │                                                          │
  │  JUST RIGHT (1/4" per foot):                            │
  │    Water and solids move together                        │
  │    Self-scouring velocity                                │
  │                                                          │
  │  TOO STEEP (> 1/2" per foot on large pipe):              │
  │    Water runs ahead of solids                            │
  │    Solids stranded in pipe → chronic clogging            │
  │    (also called "overscouring" effect)                   │
  └──────────────────────────────────────────────────────────┘

  EXCEPTION: vertical stacks
    Fall by gravity, no slope needed
    Designed for controlled splashdown at base

  STACK BASE FITTING:
    Long-turn 90° (not short 90°): at base of vertical stack
    where it transitions to horizontal building drain
    Allows water to decelerate and flow horizontally
    Short 90° at base = turbulence, blockage risk
```

## The Trap: Life-Safety Device

```
  EVERY FIXTURE REQUIRES A TRAP

  FUNCTION:
    Holds approximately 2" of water (code minimum)
    Water seal = physical barrier against sewer gases

  ┌──────────────────────────────────────────────────────────┐
  │                  P-TRAP ANATOMY                          │
  │                                                          │
  │  Inlet (from fixture)                                    │
  │          ↓                                               │
  │      ──────                                              │
  │     /  water  \  ← water seal (2" min depth)            │
  │    │  SEALED  │                                          │
  │     ────────                                             │
  │          ↓                                               │
  │      To drain line (and vent above)                      │
  └──────────────────────────────────────────────────────────┘

  THE GASES BEING BLOCKED:
  ┌─────────────────────────────────────────────────────────┐
  │  Gas            │ Risk               │ Threshold        │
  ├─────────────────────────────────────────────────────────┤
  │  Methane (CH₄)  │ Explosive          │ 5-15% LEL        │
  │  H₂S            │ Toxic, lethal      │ Toxic at 50 ppm; │
  │                 │                    │ lethal at 300 ppm │
  │  CO₂            │ Asphyxiant at high │ —                │
  │  Ammonia        │ Toxic, irritant    │ —                │
  │  Organic VOCs   │ Health effects     │ —                │
  └─────────────────────────────────────────────────────────┘

  SEWER GAS FATALITIES:
    Workers entering manholes (H₂S accumulates)
    Homeowners with dried traps (especially floor drains)
    Drain cleaning workers disturbing plugged systems
    ALL of these are preventable → trap seals are life-safety
```

## P-Trap vs S-Trap

```
  P-TRAP (required for all new installations):

    Side view:
         │
    ─────┘    Water seals here
         └──→ To drain line (exits to SIDE → vent pipe)
              Vent prevents siphon; seal maintained

  S-TRAP (BANNED in new construction, US codes):

    Side view:
         │
    ─────┘    Water seals here
         │
         ↓    Exits DOWNWARD directly
              ↓ Siphon creates vacuum behind draining water
              ↓ Vacuum pulls water from trap
              EMPTY TRAP → sewer gas entry

  WHY S-TRAPS FAIL:
    When water drains rapidly through an S-trap, the momentum
    of the water column creates a siphon that pulls water from
    the trap behind it. Nothing maintains the seal.
    P-trap: the vent breaks the siphon before it can form.

  IN THE FIELD:
    S-traps common under old bathroom sinks (pre-1970s)
    Hard to identify at a glance vs P-trap
    Look at the outlet: exits to side wall (P-trap) or
    straight down through floor (S-trap = problem)
    Code doesn't require immediate retrofit of existing S-traps,
    but they should be noted and eventually replaced

  TRAP SIZING:
    1-1/4" P-trap: lavatories (bathroom sinks)
    1-1/2" P-trap: kitchen sinks, most fixtures
    2" P-trap: shower stalls (body waste + volume)
    3" trap: toilets (integral siphon trap in toilet bowl)
```

## Venting: How It Works and Configuration Types

```
  THE CORE PHYSICS:
    Water draining in pipe → column of water moving
    → air displaced ahead → negative pressure behind
    Vent pipe: open path for air to enter behind moving water
    → atmospheric pressure maintained → no siphon possible

  VENT CONFIGURATIONS:
  ┌───────────────────────────────────────────────────────────┐
  │  INDIVIDUAL VENT (true vent):                            │
  │    Each fixture gets its own vent pipe back to stack     │
  │    Most reliable; most pipe required                      │
  │    Standard for toilets                                   │
  ├───────────────────────────────────────────────────────────┤
  │  COMMON VENT:                                            │
  │    Two fixtures back-to-back (opposite sides of wall)    │
  │    Share one vent; vent between them                     │
  │    Common for bathroom pairs (two sinks, two baths)      │
  ├───────────────────────────────────────────────────────────┤
  │  WET VENT:                                               │
  │    Single pipe serves as BOTH drain for upper fixture    │
  │    AND vent for lower fixture simultaneously             │
  │    Pipe must be sized to handle both functions           │
  │    Code-governed: specific fixture combinations allowed  │
  ├───────────────────────────────────────────────────────────┤
  │  STACK VENT:                                             │
  │    Vertical drain stack extends above highest fixture    │
  │    Extension serves as vent (no additional pipe needed   │
  │    above highest fixture on that stack)                  │
  ├───────────────────────────────────────────────────────────┤
  │  CIRCUIT VENT:                                           │
  │    Commercial: vents a series of fixtures on one branch  │
  │    Single vent at end of branch serving multiple         │
  └───────────────────────────────────────────────────────────┘

  VENT TERMINAL:
    Must extend ABOVE ROOF (typically 6" min above roof surface)
    Must not terminate near air intakes or windows
    (sewer gases must escape to outdoor atmosphere)
    Frost caps in cold climates (prevent frost closure)
    Minimum 4" above grade if through wall (some AAV applications)
```

## Air Admittance Valves (AAVs)

```
  AAV (also: Studor valve, after brand name):
    One-way mechanical valve
    Opens: negative pressure (suction when drain flows)
             → admits air to maintain pressure balance
    Closes: pressure equalized or positive pressure
             → prevents sewer gas from entering building

  ┌───────────────────────────────────────────────────────┐
  │  DRAIN FLOWING:                                       │
  │    Negative pressure below valve                      │
  │    Valve opens → air enters from room                 │
  │    Trap seal maintained → no siphon                   │
  │                                                       │
  │  DRAIN AT REST:                                       │
  │    Valve closed by gravity + pressure                 │
  │    Sewer gases cannot exit through valve              │
  └───────────────────────────────────────────────────────┘

  WHERE AAVs ARE USEFUL:
    Kitchen islands (no easy route to roof vent)
    Basement laundry (adding drain without major vent work)
    Anywhere running vent pipe to roof is impractical
    Remodel additions where opening walls is costly

  CODE ACCEPTANCE (varies by jurisdiction):
    Most US codes: YES for individual fixtures and wet vent
    configurations (IPC 917, IRC P3114)
    NOT allowed:
      Below grade (below ground level)
      In inaccessible locations (must be accessible for replacement)
      Where manufacturer instructions prohibit
    CRITICAL: Building STILL requires at least one stack vented
    to atmosphere for positive pressure relief
    AAVs only handle negative pressure; positive pressure
    (from sewer backup or gas accumulation) needs roof vent

  INSTALLATION:
    Minimum 4" above drain connection
    In a location with air access (inside cabinet = OK)
    Not in sealed wall (needs oxygen to function)
    Replace every 10-20 years (rubber diaphragm ages)
```

## Drain Sizing

```
  DRAINAGE FIXTURE UNITS (DFU):
    Each fixture assigned a DFU value representing drain load
    Unlike supply fixture units (SFU) — separate calculation

  ┌───────────────────────────────────────────────────────┐
  │  Fixture           │ DFU Value                        │
  ├───────────────────────────────────────────────────────┤
  │  Toilet            │ 4 DFU                           │
  │  Lavatory (sink)   │ 1 DFU                           │
  │  Bathtub           │ 2 DFU                           │
  │  Shower stall      │ 2 DFU                           │
  │  Kitchen sink      │ 2 DFU                           │
  │  Clothes washer    │ 3 DFU                           │
  │  Dishwasher        │ 2 DFU                           │
  │  Floor drain       │ 2 DFU                           │
  └───────────────────────────────────────────────────────┘

  PIPE SIZING (horizontal drain, IPC Table 704.1):
  ┌───────────────────────────────────────────────────────┐
  │  Pipe size │ Max DFU (horizontal) │ Max DFU (stack)   │
  ├───────────────────────────────────────────────────────┤
  │  1-1/2"    │ 3                   │ 4                  │
  │  2"        │ 6                   │ 10                 │
  │  3"        │ 20 (no toilets)     │ 48                 │
  │  4"        │ 160                 │ 500                │
  └───────────────────────────────────────────────────────┘

  TOILET MINIMUM:
    3" minimum drain for toilet (IRC/IPC)
    4" if on horizontal branch with other fixtures (some jurisdictions)
    Never install toilet on 2" or smaller pipe

  VENT SIZING (based on drain size):
    1/2 the drain pipe diameter, minimum 1-1/4"
    Larger vents for longer developed lengths
```

## Cleanouts

```
  CLEANOUT = access port for clearing blockages

  CODE-REQUIRED LOCATIONS:
    Base of each stack (before it transitions horizontal)
    At each direction change > 45° on horizontal drain
    Every 100 ft on horizontal runs (more common: every 50 ft)
    Near building sewer exit
    Two-way cleanout at building sewer for bidirectional rodding

  TYPES:
    Floor-level cleanout plug: ABS/PVC plug in threaded fitting
    Wall-accessible cleanout: plug in wall pocket
    Two-way cleanout: allows rodding in either direction

  CAMERA INSPECTION:
    Modern drain inspection: camera down cleanout
    Video of entire pipe interior to sewer
    Diagnoses: root intrusion, pipe sag, offset joints,
    scale buildup, broken pipe
    Standard practice before buying older home

  CLEANOUT INSTALLATION:
    Must be accessible (not buried, sealed in wall permanently)
    Above floor: preferred for major stacks
    At grade or accessible from basement: for building drain
```

## Common Problems and Diagnosis

```
SYMPTOM → LIKELY CAUSE:

  Slow drain (one fixture):
    → Partial clog at or near trap (hair, grease)
    → Solution: clear trap or snake 6-10 feet

  Slow drain (multiple fixtures on same stack):
    → Blockage in shared drain or stack
    → Solution: snake or jet the main

  Gurgling after toilet flushes:
    → VENTING PROBLEM (most diagnostic gurgle)
    → Negative pressure siphoning nearby trap (bubbling)
    → Check for blocked/inadequate vent
    → Verify roof vent not frost-blocked or debris-blocked

  Sewer smell (one fixture):
    → Dry trap (evaporated or siphoned)
    → Infrequently used floor drain, bathroom
    → Solution: pour water in drain; check trap seal depth

  Sewer smell throughout house:
    → Multiple dry traps, OR
    → Missing trap somewhere (direct connection to drain)
    → Check floor drains, utility sinks, rarely-used fixtures

  Slow drain + gurgling + smell:
    → Classic venting failure trifecta
    → Diagnose vent system before assuming blockage

  Root intrusion:
    → Slow or complete blockage of sewer lateral
    → Roots enter through joints in older clay/cast iron pipe
    → Camera confirms; options: root cutting (annual), CIPP
       liner (trenchless rehabilitation), or replacement
```

## Decision Cheat Sheet

| Situation | Solution |
|---|---|
| Adding fixture in kitchen island | AAV (check local code acceptance) |
| Toilet drain minimum pipe size | 3" minimum (4" with other fixtures) |
| Drain slope for 3/4 bath horizontal run | 1/4" per foot |
| Gurgling after toilet flush | Check/unblock vent stack |
| Sewer smell, infrequently used drain | Pour water to recharge trap |
| Older home, occasional sewer smell | Check for S-traps, dry floor drains |
| Camera inspection access | Cleanout at base of stack |
| Multiple fixtures, one drain line | Sum DFUs → size from IRC/IPC table |
| Below-grade laundry drain | Ejector pump (not gravity) + check valve |

## Common Confusion Points

**Venting prevents trap loss, not sewer entry**: The trap blocks sewer gases. The vent prevents the trap from being siphoned empty by drain flow. Both are essential. People think vent pipes "let gases out" — they do, but the more critical function is maintaining atmospheric pressure to protect trap seals.

**Gurgling = venting problem, not clog**: A toilet that gurgles nearby fixtures after flushing is a classic vent failure signal. The negative pressure created by the flush is siphoning adjacent trap water. Check the vent stack before snaking any drain.

**S-traps don't fail immediately**: An S-trap may work for years before it siphons. Partial siphon (losing some but not all trap water) creates intermittent sewer smell that's hard to trace. If a fixture has unexplained intermittent smell and you can see a downward drain exit = S-trap → replace.

**Not all jurisdictions accept AAVs everywhere**: Some local codes restrict AAVs to specific applications. Always verify with AHJ. An inspector can reject an AAV installation even if the code book allows it if the local amendment restricts it.

**1/4" per foot is a minimum, not a target for large pipes**: For 3" and 4" drains, steeper slopes cause the liquid/solid separation problem. 1/4" per foot is specifically sized for proper self-scouring velocity. More is sometimes required for smaller pipes; more can be detrimental for larger pipes carrying solids.
