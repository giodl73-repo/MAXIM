# Fittings & Joints

## Eight Ways to Connect Pipes

## The Big Picture

```
JOINT TYPE MAP:
+------------------------------------------------------------------+
|                                                                  |
|  PERMANENT                      SEMI-PERMANENT / SERVICEABLE    |
|  =========                      ================================ |
|                                                                  |
|  Sweat solder   (copper)        Compression    (copper/rigid)   |
|  Press-fit      (copper/SS)     Flare          (soft copper)    |
|  Expansion      (PEX-a)         Threaded NPT   (any compatible) |
|  Solvent weld   (PVC/CPVC/ABS)  Push-fit       (copper/PEX/CPVC)|
|  Crimp/Clamp    (PEX)                                           |
|  Grooved        (Victaulic)                                     |
+------------------------------------------------------------------+

STRENGTH HIERARCHY (for supply pressure service):
  Expansion (PEX-a) > Sweat solder > Press-fit > Crimp > Clamp >
  Solvent weld > Threaded > Compression > Push-fit

  Note: all properly made joints exceed pipe pressure rating —
  "weakest link" in plumbing is almost always an improperly made
  joint, not the joint type itself
```

## Sweat/Solder (Copper)

```
PRINCIPLE: capillary action draws molten solder into gap between
           fitting socket and pipe

CAPILLARY ACTION REQUIREMENTS:
  Clearance: 0.001" to 0.004" (too tight = no capillary draw;
             too loose = cold joint, excessive solder use)
  Clean surfaces: oxides block solder wetting → flux dissolves oxides
  Heat: fitting (not pipe) heated to solder melting point (~450°F/230°C)
  No water: steam prevents solder draw-in → guaranteed cold joint

PROCESS:
  1. Cut pipe square (pipe cutter preferred over saw — cleaner)
  2. Ream inside of pipe (cutter burr restricts flow)
  3. Clean pipe OD + fitting ID with emery cloth/abrasive pad
     (bright copper = clean; dark/oxidized = will reject solder)
  4. Apply flux to both cleaned surfaces (thin, even coat)
  5. Assemble joint; rotate fitting to distribute flux
  6. Heat FITTING with torch, not the pipe
     → heat conducts from fitting socket into pipe end
     → more uniform heating
  7. Test: touch solder to joint — if it melts, joint is ready
     (NOT torch to solder)
  8. Apply solder; capillary action draws it in around joint
     STOP when solder beads appear at bottom of fitting cup
  9. Remove heat; wipe excess solder (optional)
  10. Allow to cool undisturbed — don't move joint while solidifying

TORCH FUEL:
  Propane: adequate for 1" and under; safe, widely available
  MAPP Gas: hotter, faster; essential for 1.5"+ joints in
             tight spaces; more expensive
  Oxyacetylene: for 2"+ pipe or very large commercial joints

DRY PIPE TRICKS:
  Freeze fitting: "Freeze-It" spray or pipe freeze kit
                  Creates ice plug in water column upstream
  Bread plug: white bread in pipe absorbs residual water
              Dissolves in water when pipe is put back in service
  Best: drain the system
```

## Press-Fit (ProPress / Viega)

```
PRINCIPLE: hydraulic tool compresses EPDM O-ring into annular groove
           in fitting; O-ring compressed against pipe OD creates seal

FITTING ANATOMY:
  ┌──────────────────────────────────┐
  │  Press fitting body              │
  │  ┌──────────────────────────┐   │
  │  │  O-ring groove           │   │
  │  │  [EPDM O-ring ─────────] │   │
  │  │                          │   │
  │  │  ← pipe inserted here    │   │
  │  └──────────────────────────┘   │
  │  Press jaw compresses body       │
  │  → O-ring deforms, seals OD     │
  └──────────────────────────────────┘

LEAK-BEFORE-PRESS TECHNOLOGY:
  Some fittings (ProPress "Smart Connect"):
  Before pressing: fitting has small G-shaped leak path
  → If system pressurized before all joints pressed → leak reveals
  → After pressing: leak path eliminated
  Allows pressure testing to identify unpressed joints

COMPATIBLE MATERIALS:
  Copper (ProPress for copper)
  Stainless steel (SS ProPress)
  Carbon steel (MegaPress) — for mechanical piping, carbon steel systems
  NOT for PEX, CPVC

ADVANTAGES:
  No open flame → safe in occupied/retrofit/spray foam areas
  Speed: ~3× faster than soldering per joint
  Consistent: tool ensures full compression regardless of skill
  Auditable: press record logs in tool (commercial requirements)

DISADVANTAGES:
  Higher fitting cost (~3-5× solder fittings)
  Tool rental/purchase required
  Must be inspected/pressed (some O-ring failures if not pressed)
  Not reusable (cold-pressed O-ring doesn't recover)
```

## Compression Fittings

```
PRINCIPLE: brass ferrule (olive/ring) compressed against pipe OD
           by fitting nut → creates mechanical seal

ANATOMY:
  ┌──────────────────────────────────────────────┐
  │  Body nut  │ ferrule │  Body fitting          │
  │  ────────► │ (olive) │ ◄─ pipe →             │
  │            │         │                        │
  │  Tighten nut → ferrule compresses into pipe OD│
  │  → indentation creates seal                   │
  └──────────────────────────────────────────────┘

FERRULE TYPES:
  Two-piece: separate front and back ferrule (more forgiving)
  Single olive: most common in residential
  Flared (for harder materials): see flare section

LIMITATIONS:
  Rigid pipe only (copper, CPVC, PVC — not PEX directly)
  Not for buried applications (corrosion, no access)
  Not for long runs (too many leak-point joints)
  Reusable only with new ferrule each time

PRIMARY USES:
  Angle stops (shutoffs at fixtures): standard compression
  Appliance connections (dishwasher, refrigerator icemaker)
  Faucet supply connections
  Drain cleanout connections
```

## Flare Fittings

```
PRINCIPLE: pipe end expanded to 45° cone (flare) by flaring tool
           Fitting nut compresses flared end against conical seat
           → metal-to-metal seal

COMPATIBLE PIPE: soft-drawn copper (Type K or L only)
                 Stainless steel tubing
                 NOT Type M (too thin — cracks at flare)
                 NOT PEX, PVC, CPVC

FLARE ANGLE: 45° SAE flare (plumbing, HVAC, fuel oil)
             37° JIC flare (hydraulics, aerospace) — not for plumbing

FLARING TOOL:
  Yoke-and-cone type: pipe held in die block; cone screwed in
  Standard: pipe deburr → insert in die block → tighten cone → flare

APPLICATIONS:
  LP gas connections (copper LP gas in some jurisdictions)
  HVAC refrigerant connections
  Fuel oil supply
  Laboratory gas

WHY NOT FOR WATER:
  Metal-to-metal flare seal is susceptible to distortion from
  over/under-tightening; difficult to get right
  Overtightening → crack at flare → sudden failure
  Compression fittings are easier and more reliable for water
```

## Push-Fit (SharkBite and Equivalents)

```
PRINCIPLE: stainless steel grab ring + EPDM O-ring
           Insert pipe → teeth bite into OD (can't pull out)
           → O-ring seals → instant connection

ANATOMY:
  ┌──────────────────────────────────────────────────────────┐
  │                                                          │
  │  ← push pipe in this direction                          │
  │                                                          │
  │  Release collar   Stainless teeth   EPDM O-ring          │
  │  ┌─────────────┬──────────────────┬────────────────────┐ │
  │  │  ▼          │  \\\\\\\\\\\\\\  │  ○                 │ │
  │  │  Push this  │  teeth grab pipe │  seals against OD  │ │
  │  │  to release │  on pull-out     │                    │ │
  │  └─────────────┴──────────────────┴────────────────────┘ │
  │                                                          │
  │  Push release collar with removal tool → teeth release → │
  │  pull pipe out                                           │
  └──────────────────────────────────────────────────────────┘

COMPATIBLE MATERIALS:
  Copper (hard or soft, all types)
  CPVC
  PEX-a, PEX-b, PEX-c
  NOT galvanized steel, threaded pipe, ABS, PVC DWV

REMOVAL:
  SharkBite removal tool (or tube cutter collar as makeshift)
  Slide over pipe, press release ring, pull pipe out
  Fitting can be reused (check O-ring and teeth condition)

LONGEVITY DEBATE:
  SharkBite/manufacturer: O-ring rated 25+ years in normal conditions
  Critics: less long-term confidence than solder or crimp
  Anecdotal: many 2000s-era SharkBite fittings still functioning
  Code: accepted for permanent installation in most jurisdictions
         EXCEPT buried or encased in concrete (access required)

APPLICATIONS:
  Repairs (no torch needed — even with water in line, often works)
  Retrofit connections (joining dissimilar materials)
  Temporary repairs until permanent fix scheduled
  Under-pressure temporary fix (some plumbers use for immediate
  shutoff of active leak)
```

## Threaded NPT

```
NATIONAL PIPE TAPER (NPT) THREAD:
  Tapered thread: diameter increases toward fitting end
  As nut/fitting threads together → thread flanks wedge
  → self-sealing at thread flanks (not at thread crests)

  Taper: 1:16 per side (3/4" per foot taper included angle)
  Male NPT: on pipe and male fitting ends
  Female NPT: on fitting bodies, valves, appliances

THREAD SEALANT:
  REQUIRED: thread flanks seal but crest/root leaks without sealant
  PTFE tape ("Teflon tape"):
    White: standard; 2-3 wraps clockwise (from male thread end)
    Yellow: gas-specific (thicker/denser); for gas lines
  Pipe dope/compound:
    Superior to tape for large threads and male brass
    Thread compound seeps into thread voids
    Required by some codes in addition to tape for gas
    Use pipe dope rated for gas on gas lines

  TECHNIQUE:
    Wrap tape clockwise (when looking at thread end)
    So wrapping doesn't unwind as fitting threads on
    Hand-tight + 2-3 wrench turns (not more)
    Overtightening cracks plastic fittings, splits cast fittings

NOT FOR:
  Copper pipe (no NPT copper pipe — use NPT×copper-solder adapters)
  PEX directly (use PEX×NPT fittings)
```

## Solvent Welding (PVC / CPVC / ABS)

```
PRINCIPLE: solvent chemically dissolves both surfaces; surfaces
           fuse together when solvent evaporates → molecular bond

THIS IS NOT GLUING:
  No adhesive layer between surfaces
  The surfaces themselves become one material
  True fusion — stronger than mechanical joints

  Before: two separate pieces of PVC
  During: solvent dissolves surface → thick semi-liquid polymer
          Joint inserted → surfaces contact → blend together
  After: solvent evaporates → single piece of PVC

MATERIALS AND CEMENTS (CRITICAL: use correct product):
  ┌──────────────────────────────────────────────────────────┐
  │ Material  │ Primer                    │ Cement           │
  ├──────────────────────────────────────────────────────────┤
  │ PVC       │ Purple primer (required   │ PVC cement       │
  │           │ in most jurisdictions     │ (grey/clear/blue)│
  │           │ for pressure service)     │                  │
  ├──────────────────────────────────────────────────────────┤
  │ CPVC      │ CPVC primer (yellow/clear)│ CPVC cement      │
  │           │                           │ (orange/yellow)  │
  ├──────────────────────────────────────────────────────────┤
  │ ABS       │ ABS primer or cleaner     │ ABS cement (black│
  │           │ (some brands primer-free) │ can)             │
  ├──────────────────────────────────────────────────────────┤
  │ ABS→PVC   │ Purple primer             │ Transition cement│
  │ (transition│                          │ (specified for   │
  │ joint)    │                           │ ABS/PVC combo)  │
  └──────────────────────────────────────────────────────────┘

PROCESS:
  1. Cut pipe square; deburr inside and outside
  2. Test dry fit first (pipe should go ~1/3 depth by hand)
  3. Apply primer to pipe OD and fitting socket ID
     (purple color confirms application for inspector)
  4. Apply cement to both surfaces quickly
  5. Insert and ROTATE 1/4 turn while inserting
     (distributes cement, eliminates dry spots)
  6. Hold firmly 30-60 seconds while initial set begins
  7. Wipe excess cement at joint line
  8. DO NOT MOVE joint for cure time:
     1.5" PVC at 60°F+: 30 minutes before pressure
     Full cure for sustained pressure: 24 hours
     Cold weather = longer cure times (use cold-weather formula)
```

## PEX Fittings

```
CRIMP (PEX-b standard):
  Copper or stainless ring crimped over barbed brass fitting
  with crimping tool
  Go/no-go gauge: verify correct crimp (ring overlap must be correct)
  Inexpensive; most widely used method

CLAMP/CINCH:
  Stainless steel ear clamp squeezed with single-ear crimper
  Cheaper tool than crimp; same performance
  Can be used on all PEX types
  Easier in tight spaces

EXPANSION (PEX-a only — see pipe materials):
  Full-bore; strongest; most expensive

PUSH-FIT (SharkBite):
  Works on all PEX types; see push-fit section above
```

## Decision Cheat Sheet

| Situation | Joint type |
|---|---|
| Standard copper supply, new construction | Sweat solder |
| Copper in occupied building, no flame | Press-fit |
| Quick repair, no torch | Push-fit (SharkBite) |
| Under-sink shutoffs | Compression angle stop |
| PEX new construction | Crimp or expansion |
| PVC drain, residential | Solvent weld |
| Gas piping | Threaded NPT |
| Commercial pipe 2"+ | Grooved (Victaulic) |

## Common Confusion Points

**Sweat soldering: heat the fitting, not the pipe**: Heating the pipe first makes the pipe draw heat away from the joint. Heating the fitting ensures the joint socket reaches temperature and heat conducts evenly to the pipe end. Most failed solder joints from beginners come from heating the pipe.

**Solvent welding is not reversible**: Once a PVC joint is welded, it cannot be separated without cutting. Plan DWV runs with cleanouts where you'll need future access. Pressure joints cannot be reused.

**Push-fit requires correct pipe OD**: SharkBite works on nominal IPS pipe sizes (copper, CPVC, PEX). It does NOT work on CPVC pipe that has CTS (copper tube size) vs IPS dimensions — verify which CPVC you're buying.

**Pipe dope AND tape for gas is often required**: Some jurisdictions require both PTFE tape AND pipe dope on gas threads. The tape provides initial thread lubrication and gap-filling; the dope provides secondary sealing and hardens to lock the joint. Check local code and gas company requirements.
