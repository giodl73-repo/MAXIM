### Annex Principle

> **"Would a reader 200 years from now need this to understand the field?"**
>
> **Yes → stays in the deck.** Type system taxonomy, language genealogy, algorithm theory, how the web works, what databases are, CS foundations.
>
> **No, it's a reference card for a specific tool → annex.** PostgreSQL syntax, Docker CLI flags, NumPy API, Bash builtins, Windows registry.
>
> The overviews and cheatsheets stay. The per-item cards go. The deck teaches the field. The annex is the lookup table.
