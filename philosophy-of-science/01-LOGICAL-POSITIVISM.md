# Logical Positivism: Vienna Circle, Verification, Carnap, Hempel

## The Big Picture

Logical positivism was the attempt to make philosophy scientific by applying the tools of modern symbolic logic (Frege, Russell) to the problems of empiricism (Hume, Mach). The Vienna Circle's central project: eliminate metaphysics by showing that metaphysical claims are literally meaningless — not false, but nonsensical.

```
+------------------------------------------------------------------+

+------------------------------------------------------------------+

|  |  as foundation   |  |  positivism     |  |  scientific     |   |
|  |  of mathematics |  |  sensation-base |  |  rigor         |   |
|  +-----------------+  +-----------------+  +----------------+   |
|          |                    |                    |             |

|  |            VIENNA CIRCLE (1920s–1938)                        |   |
```

## The Vienna Circle

### Membership and Context

The Circle formed around Moritz Schlick at the University of Vienna in 1922, though the informal group began meeting circa 1907 around Ernst Mach's ideas. Key figures:

| Figure | Role | Key Contribution |
|--------|------|-----------------|
| **Moritz Schlick** | Chair | General theory of knowledge; "philosophy as activity of clarification" |
| **Rudolf Carnap** | Main theorist | Aufbau (1928), logical syntax, formal reconstruction of science |
| **Otto Neurath** | Sociologist | Protocol sentences, physicalism, encyclopedia of science |
| **Friedrich Waismann** | Mathematics | Transmitted Wittgenstein's ideas; later broke with positivism |
| **Hans Hahn** | Mathematician | Co-authored the Circle's 1929 manifesto |
| **A.J. Ayer** | UK exponent | Language, Truth and Logic (1936) — accessible popularization |
| **Hans Reichenbach** | Berlin group | Probability and induction; philosophy of physics |

Wittgenstein was not a member but his *Tractatus* (1921) heavily influenced them — particularly "Whereof one cannot speak, thereof one must be silent."

## The Verificationist Criterion of Meaning

### The Basic Claim

A declarative sentence is **cognitively meaningful** if and only if it is either:
1. **Analytically true or false** — true by the meanings of words alone ("All bachelors are unmarried"), or
2. **Empirically verifiable in principle** — there exist possible observations that would confirm or disconfirm it

Metaphysical claims like "The Absolute is beyond time and space," "Existence precedes essence," or "God exists" are not false — they are cognitively meaningless. They generate no observable predictions.

```
              SENTENCE S — IS IT MEANINGFUL?
                          |
              +-----------+-----------+
              |                       |
         ANALYTIC?               EMPIRICALLY
     (true/false by          VERIFIABLE?
      meaning alone)         (possible obs.)
         YES                     YES
          |                       |
      MEANINGFUL               MEANINGFUL
      (logic, math)            (science)
                                   NO
                                    |
                               MEANINGLESS
                            (metaphysics, theology,
                             much of traditional
                             philosophy)
```

### Ayer's Formulation (Language, Truth and Logic, 1936)

Ayer distinguished strong and weak verification:
- **Strong**: Statement is conclusively verifiable by finite observations
- **Weak**: Observations could make the statement probable

Ayer used weak verification — no finite set of observations conclusively verifies universal laws like "All copper conducts electricity," but observations can make them probable.

## Protocol Sentences and the Observation Base

### The Problem

The verificationist criterion requires that claims ultimately connect to observation. But what exactly is an observation? The Circle debated this intensely.

**Protocol sentences** (Protokollsätze) are the basic reports of immediate experience — the bedrock on which scientific knowledge rests. But there was disagreement:

| Position | Advocate | Claim |
|----------|----------|-------|
| **Phenomenalism** | Early Carnap | Protocol sentences report private sense data ("I now see red here") |
| **Physicalism** | Neurath | Protocol sentences describe public physical events ("The pointer is at 5") |
| **Coherentism** | Neurath | Protocol sentences are provisional; no incorrigible given |

Neurath won the day with physicalism: science is intersubjective, so the observation base must be publicly accessible physical events, not private sense data.

## Carnap's Aufbau (Der logische Aufbau der Welt, 1928)

### The Project

Carnap attempted to show that all concepts of science could be logically constructed (aufgebaut) from a minimal base of **elementary experiences** and **remembered similarity**. The reconstruction would show that science is nothing but a logical structure built on experience.

```
+------------------------------------------------------------------+
|                  CARNAP'S AUFBAU STRUCTURE                       |
+------------------------------------------------------------------+
|                                                                  |
|  LEVEL 4:  Cultural objects — institutions, values              |
|            (constructed from psychological objects)              |
|                                                                  |
|  LEVEL 3:  Physical objects — material things                   |
|            (constructed from sense qualities at world points)    |
|                                                                  |
|  LEVEL 2:  Psychological objects (my own + others')             |
|            (constructed from elementary experiences)             |
|                                                                  |
|  LEVEL 1:  Elementary experiences — cross-section of            |
|            total experience at a moment                          |
|            BASIS: "erinnerungsgleich" (similarity in memory)    |
|                                                                  |
|  LOGIC:    All construction steps are explicit logical           |
|            definitions using Principia Mathematica notation      |
|                                                                  |
+------------------------------------------------------------------+
```

This is phenomenalist: the base is private experience. The Russell-Whitehead logic of Principia provides the construction machinery. The result is meant to demonstrate that empiricism is viable — the entire conceptual structure of science can be grounded in experience via logic.

### Why the Aufbau Matters

The Aufbau is one of the most ambitious projects in analytic philosophy. It fails, but its failure is instructive:
- **Goodman**: The similarity relation at the base is not well-defined
- **Quine**: The analytic/synthetic distinction collapses — you cannot cleanly separate logical construction from empirical content
- **Sellars**: The "myth of the given" — there are no theory-neutral raw experiences

## Carnap's Logical Syntax of Language (1934)

After the Aufbau, Carnap shifted from phenomenalism to **logical syntax**: the analysis of language as a formal calculus. Two languages:
- **Language I** (finitist): restricted to finite, constructible claims
- **Language II** (classical): full classical mathematics + physics

The **Principle of Tolerance**: "In logic, there are no morals. Everyone is at liberty to build up his own logic, i.e., his own form of language, as he wishes." — there is no single correct logic; different linguistic frameworks suit different purposes.

This is a direct predecessor to the semantic view of theories and to Carnapian pragmatics.

## Hempel's Confirmation Theory

### The Hypothetico-Deductive Picture

Carl Hempel formalized the relationship between evidence and hypothesis. The HD model: hypothesis H is confirmed by observation O if H (with auxiliaries) logically entails O, and O is true.

Hempel developed a formal satisfaction criterion: observation report e confirms hypothesis h iff the development of h to the vocabulary of e is entailed by e.

### The Raven Paradox

Hempel's confirmation theory generates a notorious paradox:

```
+------------------------------------------------------------------+
|                    THE RAVEN PARADOX                             |
+------------------------------------------------------------------+
|                                                                  |
|  Hypothesis H: "All ravens are black"                            |
|  Logically equivalent to H': "All non-black things are non-ravens"|
|                                                                  |
|  Nicod's criterion: an instance of "All Fs are G" is confirmed  |
|  by observing an F that is G.                                    |
|                                                                  |
|  H' = "All non-black things are non-ravens"                     |
|  Instance: observing a green leaf (non-black, non-raven)        |
|                                                                  |
|  Therefore: observing a green leaf confirms H' (which = H)      |
|  Therefore: observing a green leaf confirms "All ravens are black"|
|                                                                  |
|  This seems absurd. You're in your office, never saw a raven,   |
|  and you've confirmed "All ravens are black" by looking at      |
|  your houseplant.                                               |
|                                                                  |
|  HEMPEL'S RESPONSE: Not absurd — just a tiny amount of          |
|  confirmation. The green leaf does confirm H, but only           |
|  infinitesimally. Ravens themselves confirm it much more.       |
|                                                                  |
|  The Bayesian response: confirmation depends on prior            |
|  probability. The world contains vastly more non-black          |
|  objects than ravens. A green leaf gives negligible Bayesian    |
|  update; a black raven gives substantial update.                 |
|                                                                  |
+------------------------------------------------------------------+
```

The paradox exposes that confirmation is relative to background knowledge and prior probabilities — the formal satisfaction criterion alone is insufficient.

## Internal Tensions and Collapse

### Mathematics is Not Empirically Verifiable

The verificationist criterion has a fatal problem: mathematics and logic are not empirically verifiable (they are not contingent empirical claims), yet they are not meaningless. The positivists' answer: mathematical truths are **analytic** — true by definition, contentless, merely expressing linguistic conventions.

But:
- Quine (1951, "Two Dogmas of Empiricism"): there is no clean analytic/synthetic distinction. Every statement has both meaning-component and fact-component. The boundary is not sharp.
- If math is "merely analytic," why is it applicable to empirical science? (Wigner's "unreasonable effectiveness")

### Self-Refutation

The verificationist criterion itself is neither analytically true nor empirically verifiable. By its own lights, it is meaningless. The Circle tried various fixes (treating it as a "proposal" or "convention" rather than a truth claim), but the problem persists.

### Neurath's Boat

Otto Neurath's image captures the anti-foundationalist insight: there is no Archimedean point outside the ship of knowledge from which to judge it. We are like sailors who must rebuild the ship at sea, plank by plank, never able to take it apart entirely:

```
   We are at sea. Our ship is science.
   We want to rebuild it on secure foundations.
   But we cannot take it to dry dock.
   We must rebuild it plank by plank
   while staying afloat.

   No plank is sacred. No plank is unrevisable.
   But we cannot replace all planks at once.

   → No protocol sentence is incorrigible.
   → No protocol sentence is beyond revision.
   → But we cannot doubt everything at once.

   Contrast: Descartes tried to demolish the ship,
   start from scratch on solid ground.
   Neurath says: no solid ground exists.
```

This anticipates Quine's web of belief and is one of logical positivism's most durable contributions.

## Ayer and Language, Truth and Logic (1936)

Ayer's book made logical positivism accessible to English-speaking audiences. His formulations are sharper but less qualified than the Vienna Circle's internal debates:

- Ethical statements are not truth-apt — they express emotions (emotivism / non-cognitivism)
- Theological claims are meaningless (neither analytic nor empirically verifiable)
- The self is a logical construction from sense impressions

Ayer later admitted the book was "mostly wrong" — the verificationist criterion is too restrictive (rules out well-formed scientific counterfactuals) and too permissive (can be gerrymandered to include any claim).

## Legacy and What Survived

What logical positivism got right:
- **The linguistic turn**: philosophical problems often dissolve when you clarify the language
- **Formal methods applied to philosophy**: using logic to be precise about inference
- **Science vs. pseudoscience**: the demarcation question is real and important
- **Unified science program**: the encyclopedist vision of integrated knowledge

What collapsed:
- Verificationist criterion of meaning: too tight (rules out lawlike universals, counterfactuals) and self-refuting
- Protocol sentence foundationalism: Neurath's boat is right — no incorrigible base
- Analytic/synthetic distinction: Quine demolishes it
- Phenomenalist reduction: Goodman, Sellars, and others show it cannot work

## Common Confusion Points

**"Meaningless" ≠ "false"**: The positivists say metaphysical claims are not even in the running — they are nonsense, not false. Saying "God exists" is neither true nor false; it is cognitively empty, like saying "colorless green ideas sleep furiously."

**Ayer ≠ the whole Circle**: Ayer's Language, Truth and Logic is an accessible but simplified version. The actual Circle debates were far more nuanced — especially Neurath's anti-foundationalism, which contradicts Ayer's cleaner picture.

**Analyticity was the loadbearing wall**: The positivist edifice rests on the analytic/synthetic distinction (logic and math are analytic; empirical claims are synthetic). Quine's demolition of this distinction is what collapses the whole structure.

**Carnap's later work is more sophisticated**: The Aufbau is his early work. The later Carnap (logical syntax, inductive logic, semantic analysis) is more defensible and more influential on formal epistemology and formal semantics.
