# Popper and Falsificationism: Demarcation, Corroboration, Critical Rationalism

## The Big Picture

Karl Popper's core move: flip the positivists' question. They asked "what makes a claim meaningful?" Popper asked "what makes a claim *scientific*?" His answer: a claim is scientific if it is falsifiable — if there exist possible observations that would refute it. This reorients epistemology entirely: science doesn't grow by accumulating confirmations; it grows by surviving attempts at refutation.

```
+------------------------------------------------------------------+
|                   POPPER'S CENTRAL MOVES                         |
+------------------------------------------------------------------+
|                                                                  |
|  PROBLEM OF INDUCTION (Hume)                                     |
|  No number of white swan observations entails "all swans white." |
|  Induction is not logically valid. How can science work?         |
|                          |                                       |
|                          v                                       |
|  POPPER'S RESPONSE: Don't solve induction — dissolve it.        |
|  Science doesn't confirm. Science conjectures + refutes.         |
|                          |                                       |
|                          v                                       |
|  ASYMMETRY OF FALSIFICATION                                      |
|  "All swans are white" CANNOT be confirmed by finite obs.        |
|  "All swans are white" CAN be falsified by ONE black swan.       |
|  Existential claims: cannot falsify / can confirm                 |
|  Universal claims: can falsify / cannot confirm                  |
|                          |                                       |
|                          v                                       |
|  DEMARCATION CRITERION                                           |
|  Scientific claim: makes risky predictions (falsifiable)         |
|  Pseudoscientific claim: immunized against refutation            |
|                          |                                       |
|                          v                                       |
|  CRITICAL RATIONALISM                                            |
|  Rationality = willingness to expose beliefs to refutation       |
|  Science = the institution of organized refutation               |
|                                                                  |
+------------------------------------------------------------------+
```

## The Problem of Induction

### Hume's Challenge

David Hume (1739): No inference from "All observed Fs have been G" to "All Fs are G" is logically valid. The future need not resemble the past. The problem of induction: induction cannot be justified by logic (circular) or by experience (that experience confirms induction is itself an inductive claim).

Logical positivists tried to answer Hume via probability: induction is inductively justified (Reichenbach) or is constitutive of meaning (Schlick). Popper says both fail. His response is more radical: **abandon induction entirely**.

### Popper's Dissolution

Scientists don't and shouldn't try to confirm theories. They should try to refute them. The growth of knowledge is not accumulation of confirmations — it is elimination of false theories.

```
CONFIRMATORY PICTURE (wrong, per Popper):
  Observe O1, O2, O3... → induce → Theory T

POPPERIAN PICTURE:
  Conjecture T (creative act, no logic)
      |
      v
  Derive predictions P1, P2, P3... from T
      |
      v
  Test: attempt to falsify P1, P2, P3...
      |
      +--- Pi falsified → T refuted → revise/replace T
      |
      +--- Pi not falsified → T "corroborated" (not confirmed)
                              → try harder to falsify
```

Note: "corroboration" is not confirmation. A corroborated theory is one that has survived rigorous testing. It doesn't become more *probable* — it acquires a track record of resisting refutation.

## Falsifiability as Demarcation

### The Criterion

A theory T is **scientific** if and only if it is **falsifiable**: there exists at least one possible basic statement (a potential observation report) that contradicts T.

Not: T has been falsified. Not: T is probably false. Just: there is a possible observation that would, if true, refute T.

```
FALSIFIABLE (scientific):
  "All copper conducts electricity"
  → "This copper rod does not conduct electricity" would falsify it
  → This observation is possible → Scientific

NOT FALSIFIABLE (pseudoscientific by Popper's lights):
  "Everything happens for a reason"
  → No possible observation could refute this
  → Unfalsifiable → Not scientific

BORDERLINE:
  "The universe came into existence 5 minutes ago with all memory
   traces and evidence intact"
  → Unfalsifiable by design → Not scientific
  → Not false — just not science
```

### The Demarcation Problem Applied

Popper's key targets were **Marxist historical theory** and **Freudian psychoanalysis**. His argument:

| Theory | The Problem |
|--------|------------|
| **Freudian psychoanalysis** | Every human behavior confirms Freudian theory. Generosity = sublimated aggression. Aggression = aggression. Nothing could falsify it. |
| **Adlerian psychology** | Same structure: every case confirms the theory by explanation after the fact. |
| **Marxism** | Originally made falsifiable predictions (revolution in industrialized countries first). When falsified, revised ad hoc. The revision immunized it against further falsification. |
| **Einsteinian relativity** | Risky: predicted exact gravitational deflection of light. Could have been falsified by Eddington's 1919 eclipse expedition. Was not → corroborated. |

The contrast: Einstein's theory was endangered by each prediction. The psychoanalytic theories were safe — they explained everything by accounting for it afterward.

## The Asymmetry of Falsification

The logical asymmetry is crisp:

```
MODUS TOLLENS (valid deductive inference):
  Premise 1: If T, then O   (T predicts O)
  Premise 2: Not-O          (O not observed)
  Conclusion: Not-T         (T is false)

This is logically valid. One counterexample refutes "All swans are white."

MODUS PONENS (confirmation — invalid generalization):
  Premise 1: If T, then O   (T predicts O)
  Premise 2: O              (O observed)
  Conclusion: T             (T is true)

INVALID. Affirming the consequent. Infinitely many other theories also predict O.
```

This asymmetry is the technical foundation for why falsification is epistemically privileged over confirmation.

## Corroboration

Popper replaces "probability" and "confirmation" with **corroboration**: a measure of how well a theory has survived rigorous testing. Key properties:

- Corroboration is **not a probability** and cannot be combined by the probability calculus
- Theories with bolder, more precise predictions have higher **corroborating power** per test
- Past corroboration gives no guarantee of future performance (no induction)
- A theory with high corroboration is one that has been exposed to severe tests and survived

The **severity of a test** matters: a test that the theory was unlikely to survive (given background knowledge) and survived is more corroborating than an easy test. This anticipates the Bayesian notion of "surprising" evidence being more informative.

## Critical Rationalism

Popper extends falsificationism into a general epistemological stance: **critical rationalism**. The title of his 1963 collection is *Conjectures and Refutations*. Key claims:

- All knowledge is conjectural — we never have certainty
- The rationality of belief consists in its openness to criticism
- Progress happens by eliminating errors: "bold conjectures and sincere attempts at refutation"
- Democratic institutions should be organized on the same principle — "open society" vs. "closed society"

The political application: totalitarian ideologies (Marxism, Nazism) are epistemically closed, immune to refutation, and practically dangerous.

## The Duhem-Quine Problem for Popper

### Holism

Pierre Duhem (1906) noticed: a physical hypothesis is never tested in isolation. When you test "planets orbit the sun in ellipses," you also rely on:
- Newtonian mechanics
- Optics (telescope theory)
- Assumptions about atmospheric refraction
- Calibration of instruments

If an anomaly appears, you don't know which of these to reject. You can always save the central theory by modifying auxiliaries.

Quine (1951) generalized this to all of language: no statement faces the "tribunal of experience" alone. The whole "web of belief" meets experience collectively.

```
WHAT FALSIFICATION ACTUALLY LOOKS LIKE:

  Test T: [Core theory H] + [Auxiliary A1] + [Auxiliary A2] + ...
                                    |
                            Prediction P derived
                                    |
                           Observation: not-P
                                    |
                     +---------------------------+
                     |                           |
                Revise H                    Revise A1 or A2
              (Popper's intent)             (Duhem-Quine escape)
```

### Implication for Popper

This is a serious problem. If you can always save the core theory by adjusting auxiliaries, nothing is ever unambiguously falsified. Popper's response:

1. **Methodological decision**: accept the auxiliary hypotheses as unproblematic unless there is independent reason to doubt them
2. **Pragmatic constraint**: revisions must themselves be testable (no ad hoc immunization)

But "no ad hoc revisions" is a normative rule, not a logical constraint. Kuhn and Lakatos argue that scientists routinely and reasonably protect their core theories. This leads to Lakatos's research programs (04-LAKATOS.md).

## Does Science Actually Work This Way?

### Kuhn's Counter

Kuhn's historical studies suggest that scientists do NOT immediately abandon theories when they encounter anomalies. Normal science dismisses anomalies as experimental error or unsolved puzzles. Scientists protect their paradigm. This is rational, says Kuhn — you wouldn't want to abandon a well-developed theory every time something didn't fit.

### Lakatos on Naive vs. Sophisticated Falsificationism

Lakatos distinguishes:
- **Naive falsificationism**: one counterexample refutes a theory → historically false and scientifically suicidal
- **Sophisticated falsificationism**: a research program is abandoned only when a better one is available that accounts for all prior successes plus the anomalies

Popper's reply: he always meant sophisticated falsificationism. But the ambiguity in Popper's texts is real.

## Popper on Probability and Quantum Mechanics

Popper interpreted quantum mechanical probabilities as **propensities** — objective tendencies or dispositions of physical systems. This is Popper's propensity interpretation of probability (1957).

Against frequentism (probability = long-run frequency): propensities are single-case objective probabilities — the probability that this uranium atom decays in the next hour is an objective feature of the atom, not a frequency in an ensemble.

This was important for Popper's account of corroboration: the higher the probability that a test would have falsified the theory if it were wrong, the more corroborating a passed test is.

| Why does Popper reject induction? | Hume shows induction is invalid; Popper: dissolve the problem by replacing confirmation with falsification |
| What is corroboration? | Not probability — a track record of surviving rigorous testing; severe tests + survival = high corroboration |

## Common Confusion Points

**Falsifiability ≠ falsified**: A theory is scientific if it *could* be falsified by possible observations. It doesn't have to have been falsified. Unfalsified but falsifiable theories are still science in good standing.

**Corroboration ≠ confirmation**: Corroboration measures a track record. It gives no inductive warrant for the future — Popper is consistently anti-inductivist. High corroboration means "has survived severe tests," not "probably true."

**One counterexample ≠ immediate rejection**: Popper (when pressed) did not say scientists should immediately abandon theories on first counterexample. Methodological decisions about which auxiiliaries to revise are legitimate.

**The asymmetry only holds logically for the formal case**: In practice, observation reports themselves are theory-laden. "I observed a non-black raven" relies on color theory, species taxonomy, lighting assumptions. Pure modus tollens is an idealization.

**Popper is normative, not descriptive**: He is prescribing how science ought to work, not describing how it does work. Kuhn describes. Popper prescribes. Lakatos tries to do both.
