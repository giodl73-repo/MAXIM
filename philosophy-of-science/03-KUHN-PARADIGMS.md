# Kuhn and Paradigms: Normal Science, Crisis, Revolution, Incommensurability

## The Big Picture

Thomas Kuhn's *The Structure of Scientific Revolutions* (1962) is the most cited academic work of the 20th century across all fields. Its central thesis: science does not grow by smooth accumulation of knowledge. Instead it alternates between periods of **normal science** (routine puzzle-solving within an established framework) and **revolutionary episodes** (framework replacement). The framework itself — the paradigm — sets what counts as a problem, what counts as a solution, and what the world is made of.

```
+------------------------------------------------------------------+
|              KUHN'S SCIENTIFIC DEVELOPMENT MODEL                 |
+------------------------------------------------------------------+
|                                                                  |
|  PRE-SCIENCE                                                     |
|  Competing schools, no consensus on fundamentals               |
|  e.g., pre-paradigm optics: dozens of incompatible theories     |
|              |                                                   |
|              v                                                   |
|  NORMAL SCIENCE (paradigm established)                          |
|  Puzzle-solving within the paradigm framework                   |
|  Anomalies dismissed, explained away, or set aside              |
|              |                                                   |
|              v (anomalies accumulate, resist solution)          |
|  CRISIS                                                          |
|  Growing awareness that something is fundamentally wrong        |
|  Proliferation of rival theories; foundational debate           |
|              |                                                   |
|              v (new paradigm emerges and wins adherents)        |
|  REVOLUTION                                                      |
|  Gestalt switch: world looks different through new paradigm     |
|  Incommensurability with old paradigm                           |
|              |                                                   |
|              v                                                   |
|  NEW NORMAL SCIENCE (new paradigm established)                  |
|  New generation of puzzle-solvers; old anomalies become         |
|  solved problems; old paradigm problems disappear               |
|                                                                  |
+------------------------------------------------------------------+
```

## The Paradigm Concept

"Paradigm" is the most contested term in *Structure*. Margaret Masterman famously counted 21 different senses Kuhn uses. Kuhn later distinguished two main senses:

### Disciplinary Matrix

The broader sociological sense: the entire constellation of beliefs, values, techniques, and shared examples that a scientific community holds. Includes:
- **Symbolic generalizations**: equations and formal rules (F = ma, PV = nRT)
- **Models and analogies**: how to think about the domain (heat as a fluid, molecules as billiard balls)
- **Values**: accuracy, consistency, scope, simplicity, fruitfulness
- **Exemplars**: specific solved problems that serve as models for future problem-solving

### Exemplar (the core sense)

The concrete problem-solution pair that new students learn as the model for how the field proceeds. When you learn physics, you solve the inclined plane, the pendulum, the two-body problem. These exemplars teach you not just the solution but *how to see and approach* problems in physics. They are more fundamental than rules or theories.

```
+------------------------------------------+
|  EXEMPLAR = PROBLEM + SOLUTION + HOW     |
|  TO SEE THE DOMAIN                       |
+------------------------------------------+
|                                          |
|  Student learns to solve:                |
|  → Simple pendulum (Period = 2π√(L/g))  |
|  → Not just the math                     |
|  → How to RECOGNIZE a pendulum problem  |
|  → What to treat as given vs. derived   |
|  → What constitutes a satisfactory      |
|     solution                             |
|  → What questions are worth asking      |
|                                          |
+------------------------------------------+
```

## Normal Science as Puzzle-Solving

Normal science is the dominant mode of scientific work. It is not primarily about testing the paradigm or trying to overthrow it. Normal scientists are puzzle-solvers: the paradigm defines which puzzles are worth solving and what counts as a solution.

### Why "Puzzle" Not "Problem"?

Kuhn uses "puzzle" deliberately. A puzzle:
- Has a guaranteed solution (the puzzle-maker wouldn't give you unsolvable puzzles)
- Has rules constraining what counts as a solution
- Is defined by the community's shared exemplars

When a scientist fails to solve a puzzle, the fault is attributed to the **scientist**, not the paradigm. Normal science is not testing the paradigm — it is demonstrating its power.

```
NORMAL SCIENCE ATTITUDE:
  "The theory is right. The anomaly is my problem.
   I need better instruments, better technique,
   better calculation, better experimental design.
   The theory doesn't need fixing."

POPPERIAN ATTITUDE (that Kuhn says scientists don't actually have):
  "The anomaly is a potential falsification.
   I should consider revising the theory."
```

This is not irrationality — it is rational to protect a well-developed, successful framework from frivolous revision.

## Anomalies and Crisis

### What Is an Anomaly?

An anomaly is an observation or result that resists assimilation into the paradigm. Under normal science, anomalies are typically:
1. Set aside as instrument error or experimental artifact
2. Filed as unsolved puzzles (to be resolved later)
3. Explained by ad hoc modification

Not all anomalies are treated equally. Anomalies that threaten core aspects of the paradigm or accumulate despite sustained effort eventually become **crisis-inducing**.

### Conditions for Crisis

A field enters crisis when:
1. A particularly troubling anomaly has resisted solution for a long time
2. The anomaly strikes at the foundations of the paradigm
3. Multiple alternative theories begin to proliferate (a sign that the field has lost confidence)
4. Leaders of the field begin to acknowledge the problem in print

Historical examples Kuhn analyzes:
- **Copernican Revolution**: accumulated problems with Ptolemaic epicycles + calendar reform need → crisis
- **Lavoisier/Oxygen**: anomalies in phlogiston theory (negative weight of phlogiston, Priestley's puzzles)
- **Relativity**: failure of ether-drift experiments + theoretical tensions in Maxwell + Lorentz

## Scientific Revolution

### The Gestalt Switch

Kuhn's most provocative claim: during a revolution, scientists don't just adopt new theories — they see the world differently. He uses the duck-rabbit gestalt figure: you cannot see duck and rabbit simultaneously. Similarly, Priestley (working in the phlogiston paradigm) and Lavoisier (working in the oxygen paradigm) literally saw different things when they looked at the same experimental setup.

```
+------------------------------------------------------------------+
|                THE GESTALT SWITCH IN SCIENCE                     |
+------------------------------------------------------------------+
|                                                                  |
|  BEFORE REVOLUTION          |  AFTER REVOLUTION                 |
|                             |                                    |
|  Priestley sees:            |  Lavoisier sees:                  |
|  "dephlogisticated air"     |  "oxygen"                         |
|  Gas that supports burning  |  An element that combines with    |
|  by accepting phlogiston    |  materials during combustion      |
|                             |                                    |
|  Same experiment.           |  Different ontology.              |
|  Different paradigm.        |  Different questions asked.       |
|                             |  Different research program.      |
|                                                                  |
|  Neither is irrational within their paradigm.                   |
|  The paradigm frames what counts as data.                       |
|                                                                  |
+------------------------------------------------------------------+
```

### What Changes in a Revolution

| Dimension | Before | After |
|-----------|--------|-------|
| **Ontology** | Different fundamental entities (phlogiston vs. elements) | New catalog of what exists |
| **Questions** | Which problems are worth solving | Different set of legitimate questions |
| **Standards** | What counts as a good solution | Different criteria of adequacy |
| **Meaning** | Key terms shift meaning (mass in Newton vs. Einstein) | Same words, different content |

## Incommensurability

### The Strong Claim

Kuhn's most controversial claim: paradigms before and after a revolution are **incommensurable** — there is no neutral standpoint from which to compare them, no common measure that determines which is better.

Three dimensions of incommensurability:
1. **Methodological**: what counts as evidence and good scientific practice differs
2. **Observational**: what scientists "see" differs — data is paradigm-laden
3. **Semantic/conceptual**: key theoretical terms change meaning across paradigms

Newtonian "mass" and Einsteinian "mass" are not the same concept. "Genus" in pre-Darwinian and post-Darwinian biology have different meanings. A full translation between paradigms is impossible.

### The Strong Charge: "Mob Psychology"

Critics (Scheffler, Shapere, Laudan) charged that incommensurability collapses into relativism: if paradigms are incommensurable, rational choice between them is impossible. The choice becomes a matter of social psychology, aesthetic preference, or mob rule — not reason.

### Kuhn's Later Clarifications

In response to the "mob psychology" charge, Kuhn clarified:
- Incommensurability is **not total incomparability**. Partial comparison is possible.
- Scientists have **shared values** across paradigms: accuracy, consistency, scope, simplicity, fruitfulness (though these are weighted differently)
- The choice of paradigm involves **judgment** that cannot be algorithmized — not irrationality

Later Kuhn (after 1969) moved toward a more semantic focus: incommensurability is primarily **untranslatability** of key terms, not complete incomparability of theories.

## The Social and Sociological Dimension

### Kuhn's Contribution to Sociology of Science

Before Kuhn, the sociology of science (Merton) studied institutional norms (universalism, communism, disinterestedness, organized skepticism). Kuhn shifted attention to the cognitive content: the paradigm determines what scientists do, and paradigms are acquired and maintained through social processes — textbook education, community gatekeeping, peer review.

The scientific community becomes the unit of analysis, not the individual scientist or the theory alone. Community consensus, not individual judgment, determines what is a puzzle and what is a crisis.

### Post-Kuhnian Science Studies

Kuhn opened the door to:
- **SSK** (Edinburgh school, 1970s): sociological explanation of scientific knowledge itself
- **Laboratory studies** (Latour, Knorr-Cetina): ethnographic study of how facts are made
- **Science, Technology and Society** (STS): science as a social institution

Kuhn himself was not a radical relativist or constructivist. He found the SSK extension of his work too strong. But *Structure* provided the conceptual vocabulary that made these moves possible.

## Connection to Computer Science

Kuhn's paradigm concept has been influential in CS and software engineering, sometimes usefully and sometimes by analogy-stretching:

```
USEFUL CONNECTIONS:
  "Programming paradigm" (OOP, functional, logic)
  IS a legitimate Kuhnian concept — communities,
  exemplars (LISP as FP exemplar, Simula as OOP),
  normal work, occasional "revolutions" (Agile vs. Waterfall)

  Normal science = using the language/framework as designed,
  not questioning its foundations

  Revolution = paradigm shift from FORTRAN to structured
  programming; from structured to OOP; from monolith to
  distributed systems

INCOMMENSURABILITY ANALOGY:
  OOP and pure FP programmers talking past each other
  about "encapsulation" or "state" — partial semantic
  incommensurability is real in software communities

DISANALOGY:
  Software paradigms coexist more easily than scientific
  paradigms — polyglot programming is common.
  Scientific paradigms are more mutually exclusive about
  what the world actually contains.
```

## Decision Cheat Sheet

| Question | Answer |
|----------|--------|
| What is a paradigm? | Disciplinary matrix (all shared beliefs/values) or more specifically an exemplar (concrete problem-solution teaching the community's methods) |
| What is normal science? | Puzzle-solving within the paradigm; not testing it, extending and applying it |
| When does crisis begin? | When anomalies accumulate, resist solution despite sustained effort, and strike at the paradigm's core |
| What is a scientific revolution? | Paradigm replacement: gestalt switch, new ontology, new exemplars, changed meaning of key terms |
| What is incommensurability? | Pre- and post-revolution paradigms lack common measure; partial untranslatability of key terms |
| Is incommensurability relativism? | Kuhn says no — values like accuracy/consistency persist; choice requires judgment, not mob rule |
| How does Kuhn differ from Popper? | Kuhn: descriptive, historical; scientists protect paradigms rationally. Popper: normative; scientists should seek falsification |

## Common Confusion Points

**"Paradigm shift" is overused**: In popular usage, "paradigm shift" means any big change. Kuhn's usage is specific: a transformation in the fundamental framework of a scientific community, involving new ontology, exemplars, and semantic change — not just a new technology or policy.

**Normal science is not stagnant**: Normal science is where most of science actually happens and where most progress is made. It is not inferior to revolutionary science — the vast majority of scientific work is puzzle-solving, and paradigms enable rapid progress precisely because they are not constantly questioned.

**Kuhn is not a relativist**: He believes some theories are genuinely better (more accurate, with greater scope, more consistent). He denies there is an algorithm that establishes this independently of all paradigm-relative judgments. These are not the same claim.

**Incommensurability ≠ incomparability**: Kuhn's later view: paradigms are partially untranslatable, but comparison is possible by bilingual scientists who know both paradigms. The comparison is just not a neutral logical reduction.
