# Settings — Prong, Bezel, Pavé, Channel, Tension

## The Big Picture

A setting is the mechanical interface between metal and stone. Every setting is a tradeoff among four variables: security, light exposure, stone protection, and aesthetic.

## Prong Setting (Claw Setting)

The default for solitaire diamonds. Maximizes light exposure, minimizes metal interference.

```
4-PRONG                        6-PRONG
=======                        =======
     ___                            ___
  /|   |\                        /|   |\
 / |   | \                      / |   | \
/  |   |  \                    /  |   |  \
|  |   |  |                   |   |   |   |
|  |   |  |                   |   |   |   |
 \ | O | /                     \  | O |  /
  \|___| /                      \ |___| /
    | |                            | |
  prongs                         prongs

4-prong: makes round stone appear slightly squarish
6-prong: appears rounder; more secure for large stones;
         more metal → slightly less light

PRONG TIP SHAPES
 Round:    Classic; smooth; most common
 Square:   More contemporary; heavier look
 V-tip:    For pointed corners (princess, marquise, pear — protects
            the vulnerable pointed corner from chipping)
 Pointed:  Traditional claw; maximum grip
```

### The Tiffany Setting (1886)

Charles Lewis Tiffany's patent: 6-prong solitaire raised high on knife-edge band. Revolutionary because it lifted the diamond off the band to expose it to light from all angles. Before this, most engagement rings used closed-back "collet" settings that blocked light from below.

```
TIFFANY SETTING
===============
                 ┌──────────────────┐
                 │   6-PRONG CROWN  │
                /│   Raised high    │\
               / │   above band     │ \
              /  │                  │  \
             /   └──────────────────┘   \
            /         KNIFE-EDGE         \
           /═══════════════════════════════\
                         SHANK

Height allows light to enter stone from below through open gallery.
```

### Prong Maintenance

Prongs wear down from daily abrasion. Typical wear cycle: 5–10 years for yellow/rose gold prongs; 10–15 years for white gold; 20+ years for platinum prongs.

```
Signs needing retipping: prongs look thin, flat, or bent
                         stone has movement when pushed with fingernail
                         stone catches on fabric (prong bent out)
Retipping: add small ball of solder to prong tip and reshape
Re-prong: fabricate entirely new prong when too worn to tip
```

## Bezel Setting

Metal strip (bezel) surrounds the stone's girdle. Most secure setting type.

```
FULL BEZEL                      PARTIAL BEZEL (Half-bezel)
==========                      ==========================
  ┌──────┐                        ┌──────┐
  │      │                        │  ○   │
  │  ○   │  Metal strip           │      │  Open on
  │      │  surrounds             └──────┘  sides
  └──────┘  entire               (sides open)
            stone

TUBE SETTING: cylindrical bezel drilled into metal; stone drops in;
              top edge burnished over girdle
              Clean, modern; suits round, oval stones

ADVANTAGES:
+ Most secure — stone can't fall out
+ Protects stone edges from chipping (suits soft stones, opal, etc.)
+ Low profile — doesn't snag
+ Suits active-wear

DISADVANTAGES:
- Obscures part of stone's circumference
- Less light enters from sides
- Can look heavy for small stones
- Resizing is harder (affects bezel geometry)

BURNISHING PROCESS:
 Metal bezel strip slightly smaller than stone diameter
 Stone pressed into bezel (seat must be accurately cut to stone's pavilion angle)
 Bezel pusher rolled around circumference pushing metal over girdle
 Burnisher smooths and work-hardens the metal fold
```

## Pavé and Micro-Pavé

Small stones set tightly together to create a surface of continuous sparkle. The French "pavé" = paved (like cobblestones).

```
PAVÉ CONSTRUCTION
==================

Top view:
 ○ ○ ○ ○ ○ ○
 ○ ○ ○ ○ ○ ○
 ○ ○ ○ ○ ○ ○
 Metal barely visible between stones.

Cross-section:
  ┌─────────┐
  │o o o o o│
  ├─────────┤
  │  METAL  │
  └─────────┘
  Top row: stones. Bottom: metal.
 Tiny beads raised from metal between stones hold each stone (bead setting).

STONE SIZES
 Pavé:       0.02–0.05 ct per stone (roughly 1.7–2.3mm diameter)
 Micro-pavé: 0.005–0.01 ct per stone (1.0–1.5mm) — requires magnification
             to set; highest labor cost

TECHNIQUES
 Hand pavé: setters use gravers and bead raisers; most expensive
 Machine/CAD pavé: prong positions designed in CAD, cast in place,
                    stone seats cut by CNC, stones set by hand
                    More consistent; still labor intensive

WEAKNESS: stones can fall out over time as tiny beads wear;
          micro-pavé rings common complaint in 5-10 year range;
          avoid for sentimental stones
```

## Channel Setting

Stones set in a groove between two parallel rails. No metal between stones.

```
CHANNEL SETTING
================
Side view:
 ┌──────────────────────┐
 │  ┌────────────────┐  │
 │  │                │  │
 │  └────────────────┘  │
 └──────────────────────┘

Top view:
 ─────────────────────
      □  □  □  □  □
 ─────────────────────

 Stones flush in channel; no prongs visible.

SUITABLE STONE SHAPES:
 Baguettes: rectangular step-cut; classic channel set in Art Deco
 Princess: square brilliant; popular modern channel set
 Rounds: round brilliants in curved channel (eternity bands)

ADVANTAGES:
+ Clean, streamlined appearance
+ Stones relatively secure (held by both rails)
+ No prongs to snag
+ Modern, minimalist look

DISADVANTAGES:
- Resizing is expensive/difficult (ring shank must be cut, band integrity compromised)
- Stones must be precisely matched in size (calibrated)
- Limited light enters stones from sides
- Repair/replacement if one stone is damaged = complex
```

## Tension Setting

Stone held by compression from the metal band. No prongs, no bezel — stone appears to float.

```
TENSION SETTING
================

Cross-section:
   ┌────┐  ┌────┐
   │    │  │    │
   │ ←  │○ │  → │
   │    │  │    │
   └────┘  └────┘
    Metal springback holds stone by compression.

Face view:
                ──┐
              ○○ │← stone floating
                ──┘

MECHANICS: metal band has precise slot cut for stone;
           band is opened slightly to insert stone;
           band's springback clamps the stone's girdle;
           must exert exactly correct pressure (too little = falls out;
           too much = stone damage)

MATERIAL REQUIREMENT:
 Only suitable for hardened metals with predictable spring properties:
 Platinum (Pt950): preferred — high springback, no creep
 18k white gold (hardened): acceptable
 Titanium: used in contemporary tension designs

 Stone must be: ≥8.5 Mohs — diamond, sapphire, ruby are safe
               Emerald (7.5), opal (5.5-6.5) = NOT suitable —
               clamp pressure can crack the stone

MAINTENANCE CONCERN: if band is bent (e.g., dropped), stone can fall out;
                     cannot be resized; limited repair options
```

## Flush / Gypsy / Burnish Setting

Stone sits flush with the surrounding metal surface.

```
FLUSH SETTING PROCESS
======================
1. Drill hole in metal slightly smaller than stone's girdle diameter
2. Bur (setting bur) cut at correct angle to create seat for pavilion
3. Stone placed in hole — girdle sits slightly above surface
4. Burnisher or hammer handpiece spreads metal over stone's girdle
5. Stone sits flush or below surface of metal

APPEARANCE: stone appears embedded in metal; very low profile
            Popular for: men's rings, wedding bands, contemporary design
            Protection: maximum — only crown above metal, pavilion protected
```

## Other Setting Types

### Illusion Setting

```
Small diamond surrounded by shaped white metal plate that visually
extends the stone's apparent size.

  ╔══╦══╗
  ║  ║  ║  ← metal plate with mirror finish
  ╠══╬══╣
  ║  ║  ║  ← extends apparent diamond diameter
  ╚══╩══╝
       ●   ← actual small diamond in center

Common in mid-century engagement rings (0.25 ct appears as ~0.5 ct).
Can appear dated; diamond sizes changed since 1960s-70s examples.
```

### Cluster Setting

Multiple stones arranged to appear as one larger stone. Victorian diamond clusters; flower clusters; calibré-cut colored stone outlines in Art Deco.

### Star Setting

Small stone within a star of engraved lines radiating outward — visually expands the stone. Indian and Middle Eastern tradition.

### Invisible Setting (Van Cleef & Arpels "Mystery Setting")

Calibré-cut stones with grooves on their girdles, placed on a metal grid rail system — no prongs visible from top.

```
  ┌────────────────┐
  │ □□□□□□□□□□□□□  │
  │ □□□□□□□□□□□□□  │
  │ □□□□□□□□□□□□□  │
  └────────────────┘
  Stones appear seamless from top; metal hidden below.

Patented by Van Cleef & Arpels; requires precision calibration;
stones must be cut to exact tolerances; extremely expensive labor
```

## CAD/CAM in Modern Settings

```
CAD WORKFLOW
=============
3D design (MatrixGold, Rhino + GemVision, JewelCAD, ZBrush)
     |
     v
Stone placement: CAD software simulates stone dimensions;
                 seat depth calculated from stone's pavilion angle;
                 prong positions optimize security vs aesthetics
     |
     v
3D print (SLA resin or direct wax print) → prototype
     |
     v
Investment cast or direct metal print (DMLS)
     |
     v
Stones drop into pre-cut seats; minimal adjustment needed

ADVANTAGE OVER HAND FABRICATION:
 Ensures stones fit before casting (not after)
 Parametric ring sizing (increase/decrease size in software)
 Consistent production across multiple pieces
 Client can visualize in rendering before committing
```

## Setting Selection Summary

| Stone Type | Best Setting | Avoid |
|------------|-------------|-------|
| Diamond solitaire | 4 or 6-prong, or bezel | None — most settings work |
| Engagement ring (active life) | Bezel or low-prong | Tension (damage risk if hit) |
| Large diamond, maximum visibility | 6-prong raised Tiffany | Bezel (hides perimeter) |
| Emerald | Bezel or 4-prong with V-tips | Tension, pavé |
| Opal, pearl, turquoise | Bezel | Prong, tension (too fragile) |
| Eternity band with rounds | Pavé or channel | Large single prongs |
| Eternity band with baguettes | Channel | Pavé (baguettes don't suit) |
| Men's ring | Flush/gypsy, bezel | Tension, pavé |
| Points: marquise/pear corner | V-tip prong | Round prong (corner chips) |

## Common Confusion Points

**Pavé vs micro-pavé vs bead setting**:
Bead setting is the technique (raising tiny metal beads around each stone). Pavé describes the resulting appearance when the technique is applied densely. Micro-pavé = same technique with stones under 1.5mm diameter.

**Tension setting vs spring setting**:
Tension setting relies on the metal's compression around the girdle. Spring settings (used in watch crystals, some industrial applications) use actual spring mechanisms. Jewelry tension settings rely on the metal's elastic properties.

**"Cathedral" setting**:
Refers to the design of the shank — metal arches rise on the sides like cathedral buttresses to support the center stone setting. Not a stone-holding technique; any crown type can be cathedral-style.

**Resizing pavé and channel rings**:
Both are expensive to resize. Channel: must cut shank, spread or compress, potentially re-cut the channel. Pavé: stones at the resize point may need to be removed, repositioned, or may be lost in the process. Get these right the first time.

## Decision Cheat Sheet

| Goal | Setting Choice |
|------|---------------|
| Maximum stone visibility | Prong (4 or 6) or tension |
| Most durable for daily wear | Bezel |
| Modern "floating" look | Tension (if stone is hard enough) |
| Continuous sparkle band | Pavé or channel |
| Low-profile active-lifestyle ring | Flush/gypsy or bezel |
| Protect soft stone (opal, pearl) | Bezel with protective metal edge |
| Art Deco or vintage look | Calibré channel or invisible setting |
| Men's band | Flush, channel, or bezel |
| Vintage reproduction | V-prong, illusion, or closed-back bezel |
