# Gemstones — Classification, Crystal Systems, the 4Cs

## The Big Picture

Gemstones are valued minerals. The science is mineralogy; the commerce is gemology. The bridge between them is understanding why the same atomic-scale structure produces optical phenomena visible from across a room.

```
GEMSTONE CLASSIFICATION TREE
==============================

ALL GEMSTONES
├── MINERALS (crystalline inorganic solids — most gems)
│   ├── Native elements
│   │   └── Diamond (C, cubic, sp³ covalent bonds, hardness 10)
│   ├── Oxides
│   │   ├── Corundum (Al₂O₃, trigonal) → ruby (Cr³⁺), sapphire (Fe²⁺/Ti⁴⁺)
│   │   └── Spinel (MgAl₂O₄) — often mistaken for ruby historically
│   ├── Silicates (Si-O backbone, most gem species)
│   │   ├── Cyclosilicates
│   │   │   ├── Beryl (Be₃Al₂Si₆O₁₈) → emerald (Cr), aquamarine (Fe)
│   │   │   └── Tourmaline (complex borosilicate) — widest color range
│   │   ├── Nesosilicates
│   │   │   ├── Garnet group (pyrope, almandine, spessartine, grossular,
│   │   │   │    andradite demantoid, uvarovite tsavorite)
│   │   │   └── Topaz (Al₂SiO₄(F,OH)₂)
│   │   ├── Inosilicates
│   │   │   └── Jadeite (NaAlSi₂O₆) — "true jade"
│   │   └── Tectosilicates
│   │       ├── Quartz (SiO₂) → amethyst, citrine, rose quartz
│   │       └── Feldspar → moonstone, labradorite
│   ├── Phosphates
│   │   └── Turquoise (CuAl₆(PO₄)₄(OH)₈·4H₂O)
│   └── Sulfates / Carbonates
│       └── Malachite (Cu₂(CO₃)(OH)₂)
├── ROCKS (polycrystalline / multi-mineral aggregates)
│   └── Lapis lazuli (lazurite + pyrite + calcite)
└── ORGANICS (biogenic materials)
    ├── Pearl (nacre: CaCO₃ aragonite + conchiolin protein)
    ├── Amber (fossilized tree resin — polymerized terpenes)
    ├── Coral (CaCO₃ calcite/aragonite)
    └── Jet (fossilized wood — lignite coal)
```

## The Seven Crystal Systems

Crystal system determines: cleavage planes, optical class, and how light behaves inside the stone.

```
CRYSTAL SYSTEM    AXES/ANGLES           GEM EXAMPLES
==============    ===========           ============
Cubic (isometric) a=b=c, all 90°       Diamond, spinel, garnet (most),
                                        fluorite, pyrite
                                        → ISOTROPIC (single refraction)

Tetragonal        a=b≠c, all 90°       Zircon, apophyllite
                                        → anisotropic

Orthorhombic      a≠b≠c, all 90°       Topaz, tanzanite (when classified
                                        here), peridot
                                        → anisotropic

Hexagonal         a=b=c≠d, 120°/90°    Beryl (emerald, aquamarine),
                                        apatite
                                        → uniaxial

Trigonal          subset of hexagonal  Corundum (ruby/sapphire), quartz,
(rhombohedral)                          tourmaline, calcite
                                        → uniaxial; dichroism common

Monoclinic        a≠b≠c, one 90°       Jadeite, orthoclase feldspar,
                  angle ≠ 90°           malachite
                                        → biaxial

Triclinic         a≠b≠c, no 90°        Labradorite, kyanite, turquoise
                                        → biaxial, pleochroism

AMORPHOUS (no crystal structure): opal (hydrated SiO₂), glass, amber
```

**Why this matters for cutting:**
- Cubic gems (diamond): no birefringence, cleavage in 4 octahedral directions
- Trigonal gems (corundum): dichroic — ruby looks red from one axis, brownish from another → cut orientation matters
- Tanzanite (orthorhombic): trichroic — different colors on three axes; oriented for best face-up color

## Hardness, Toughness, and Cleavage

These three are independent and all affect wearability:

```
HARDNESS (Mohs) — resistance to scratching
=================
10  Diamond
 9  Corundum (ruby, sapphire)
 8  Topaz, spinel, alexandrite
7.5 Beryl (emerald, aquamarine)
 7  Quartz (amethyst, citrine, tourmaline ~7–7.5)
 6  Feldspar (moonstone, labradorite)
5.5 Opal (6.5 max), glass
 3  Calcite, coral
2.5 Pearl, amber

TOUGHNESS — resistance to fracture (different from hardness!)
==========
Excellent:   Nephrite jade, jadeite (interlocking crystals = tough)
Very Good:   Ruby, sapphire, alexandrite
Good:        Diamond (hard but perfect octahedral cleavage → can split)
Fair:        Emerald (heavily included, fractures easily)
Poor:        Topaz (perfect basal cleavage — splits with single blow)
             Tanzanite (multiple cleavages — vulnerable to impact)

CLEAVAGE — preferred splitting directions along crystal planes
=========
Diamond: perfect in 4 directions (octahedral) — survives tools
         because hardness prevents scratching, NOT because tough
Topaz: 1 perfect direction — avoid bezel settings with pressure
Corundum: none (conchoidal fracture) — excellent durability
Quartz: none (conchoidal) — reliable wearability
```

## The Four "Precious" Gemstones

### Diamond

```
Composition:    Carbon (C), sp³ covalent bonds, cubic crystal
Hardness:       10 Mohs (4× harder than corundum — non-linear scale!)
Density:        3.52 g/cm³
Refractive index: 2.417 (very high — enables total internal reflection
                  at shallow angles, gives extraordinary brilliance)
Dispersion:     0.044 (rainbow fire)

Color types:
 Type Ia:   ~98% of all diamonds; N (nitrogen) aggregates; absorption
            in UV; near-colorless to faint yellow
 Type IIa:  Nitrogen-free; colorless or faint pink; rarest; highest
            clarity potential (Cullinan, Koh-i-Noor)
 Type IIb:  Boron impurity → blue color + semiconductor properties
            (Hope Diamond = Type IIb)
 Type Ib:   Isolated N atoms → vivid yellow (canary diamonds)
 Fancy colors: pink (plastic deformation or N centers), red (rarest),
               green (natural irradiation), orange, chameleon

GIA grading scale:
 Color:   D (colorless) → Z (light yellow/brown), 23 grades
          Fancy color scale separate (more valuable above Z)
 Clarity: FL → IF → VVS1/2 → VS1/2 → SI1/2 → I1/2/3
 Cut:     Excellent → Very Good → Good → Fair → Poor
 Carat:   weight (1 ct = 0.2 g); price per carat rises non-linearly
          at 0.5, 1.0, 1.5, 2.0 ct thresholds (psychological pricing)
```

### Ruby

```
Mineral species:   Corundum (Al₂O₃), trigonal crystal
Color agent:       Cr³⁺ (chromium, ~1–3%) substituting for Al³⁺
                   Cr³⁺ absorbs blue and green → transmits red
                   Also causes red fluorescence under UV (adds brightness)
Hardness:          9 Mohs
Density:           ~4.00 g/cm³

Value drivers:
 1. Geographic origin: Burmese (Mogok Valley) = "pigeon blood" red,
    most prized; Thailand = slightly darker; Sri Lanka = lighter pink-red;
    Mozambique (recent major source); Tanzania; Afghanistan
 2. Color: pigeon blood = pure red, high saturation, medium tone;
    pink corundum = pink sapphire, not ruby (boundary contested)
 3. Treatment: ~95% of rubies on market are heat-treated to improve
    color and clarity; "unheated" premium significant (3–5×+);
    fracture-filled (glass, lead) = low quality, must be disclosed
 4. Inclusions: "silk" (rutile needles), fingerprints, needles;
    "star ruby" = asterism from silk oriented on 3 axes (cabochon cut)
```

### Sapphire

```
Mineral species:   Corundum (Al₂O₃) — same as ruby
Color agents:      Blue: Fe²⁺/Ti⁴⁺ charge transfer
                   Yellow: Fe³⁺
                   Orange: Fe²⁺ + Cr³⁺ (padparadscha)
                   Pink: Cr³⁺ (low concentration — below ruby threshold)
                   "Fancy sapphires": yellow, orange, green, purple, colorless

Key origins:
 Kashmir:     Cornflower blue; velvety appearance (silk inclusions diffuse light);
              mines exhausted ~1930s; highest prices; certification essential
 Ceylon/SL:   Blue, pink, padparadscha; lighter tones; classic source
 Burmese:     Royal blue; similar to Kashmir but different growth signature
 Montana (US): Steel blue/teal; small market
 Madagascar:  Major modern source; range of blues

Padparadscha: pink-orange sapphire from Sri Lanka; named for lotus blossom;
              one of the most controversial gem certifications — labs disagree
              on boundary between "pink sapphire" and "padparadscha"
```

### Emerald

```
Mineral species:   Beryl (Be₃Al₂Si₆O₁₈), hexagonal crystal
Color agents:      Cr³⁺ or V³⁺ (vanadium — debate about V emerald)
Hardness:          7.5–8 Mohs
Density:           2.72 g/cm³

Characteristic inclusions: "jardin" (French: garden) — extensive internal
                            fractures and inclusions; accepted in trade where
                            unacceptable in diamond; type I (heavily included)
                            is normal for emerald

Origins:
 Colombia: Muzo/Chivor mines; gold standard; three-phase inclusions
           (liquid + vapor + solid) with calcite/halite crystals
 Zambia:   Darker, more blue-green; excellent clarity; strong market
 Brazil:   Volume producer; lighter, more yellow-green
 Zimbabwe: Sandawana mine; small but intense saturation

Treatment: "oiling" = filling surface fractures with cedar oil,
           polymers (Opticon), or resin; industry standard;
           "no oil," "minor," "moderate," "significant" = disclosure levels
           Minor oiling: acceptable; significant treatment = large discount
```

```
INSTRUMENT             WHAT IT MEASURES           USE
==========             ================           ===
Refractometer          Refractive index (RI)       ID mineral species;
                                                    birefringence (DR)

                                                    polymorph differentiation
```

## Common Confusion Points

**"Semi-precious" is commercial, not scientific**:
The division between "precious" (diamond, ruby, sapphire, emerald) and "semi-precious" (everything else) is a trade convention, not a mineralogical category. Alexandrite and fine tanzanite can exceed mediocre rubies and sapphires in value.

**Hardness vs toughness in daily wear**:
Topaz (8 Mohs, harder than emerald) is less practical for daily wear because it has perfect basal cleavage — a sharp blow splits it cleanly. Emerald (7.5–8 Mohs) is also fragile due to jardin inclusions. Sapphire (9 Mohs, no cleavage) is the most practical colored stone for rings.

**Ruby vs pink sapphire**:
Same mineral (corundum), same chromophore (Cr³⁺). The distinction is saturation threshold. No universal standard — GIA and AGL can certify the same stone differently. Color grading is inherently subjective for colored stones.

**Natural vs treated**:
Heat treatment is so standard for corundum (~95% treated) that unheated stones command large premiums. For emeralds, minor oiling is considered acceptable. For diamonds, clarity enhancement (fracture filling, HPHT color improvement) must be fully disclosed and is significantly discounted.

## Decision Cheat Sheet

| Scenario | Recommendation |
|----------|---------------|
| Ring for daily wear — durability priority | Ruby, sapphire (9 Mohs, no cleavage) |
| Ring with green stone — toughness concern | Tsavorite garnet > emerald for durability |
| Engagement ring alternative to diamond | Sapphire (hard, durable, variety of colors) |
| Color-change effect | Alexandrite (natural) or lab alexandrite |
| Optical phenomenon stone (star/cat's eye) | Cabochon cut required |
| Budget ring with opal | Bezel or protective setting; avoid rings entirely |
| Most durable everyday colored gem | Sapphire or ruby |
| Emerald purchase — what to check | Oil treatment grade; country of origin; jardin acceptable |
| Diamond grading priority | Cut → Color → Clarity → Carat (in that order for appearance) |
