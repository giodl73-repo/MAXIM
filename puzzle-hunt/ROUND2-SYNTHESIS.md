## Systemic Concerns (from Gottlieb — must address)

### 1. Crossword-in-crossword collision
If Arts is a crossword puzzle AND the meta is a crossword, it's confusing. **Resolved: Arts is Anamorphic Drawing, not a crossword.**

### 2. Suit distribution imbalance
Current archetype selections are 8♣, 2♦, 1♥, 1♠. Should be more balanced. **Action: review suit assignments in V3-CHANGES.md and rebalance.**

### 3. Crossword grid construction risk
EQUINOX, SYNTAX, TORQUE have Q and X — rare letters that constrain grid design. **Action: verify grid is buildable before committing answer words. May need to swap EQUINOX → ECLIPSE or TORQUE → THRUST if grid fails.**

### 4. The Grid edge cases (Black Joker)
Five specific risks:
- **Archetype name ambiguity**: "architect" appears naturally in many contexts. Need clear signal.
- **"Outside home section" boundary**: what counts as "home"?
- **Keyword extraction**: "the Architect of [what exactly]" — multi-word nouns?
- **Length uniqueness**: can two cells in the same row have the same dash count?
- **Bootstrapping**: solver only knows 13 of 52 archetype names from the Red Joker. How do they learn the other 39?

**Proposed solutions:**
- The Red Joker uses all 13 archetype names prominently → solver learns 13
- The card backs or ROLES.md lists all 52 → solver can look them up
- Keyword is always a SINGLE word (no multi-word nouns)
- Length uniqueness within rows is a design constraint we enforce
- Context + length confirmation handles ambiguity

### 5. Answer word feasibility
Several puzzles have answer words that haven't been verified against their mechanism:
- SYMMETRY (8 letters) for proof completion — needs 8 blanks in a coherent proof
- INFLECTION (10 letters) for multi-cipher — needs 10 encoding systems (verified: pool has 10+)
- CASTING (7 letters) for element ID — verified: Carbon, Aluminum, Silver, Tin, Iron, Nickel, Gold
- GENETIC (7 letters) for codons — verified: G-E-N-E-T-I-C all valid amino acids
- NICHE (5 letters) for dichotomous key — needs 5 specimens with diagonal cooperation

**Action: verify SYMMETRY, NICHE, and POLYMATH against their mechanisms before construction.**
