# Criminology — Landscape and Taxonomy

## The Big Picture

**Systems-thinking bridge:** Criminology's theory-to-policy pipeline is a control loop: a theory specifies a causal model (e.g., blocked opportunity causes crime), measurement instruments operationalize variables (self-report surveys, arrest records), and intervention evaluation closes the loop (does expanding opportunity reduce crime?). The field's perpetual debate between theories is a model-selection problem under observational data constraints — causal identification is hard precisely because randomized experiments on criminal justice policy are rare and ethically constrained. Comparative criminology (09) partially compensates by treating nations as natural experiments.

Criminology spans explanatory theory (why crime occurs), institutional analysis (how justice systems respond), and policy evaluation (what interventions actually work). The field sits at the intersection of sociology, psychology, economics, and law.

## Core Theoretical Families

| Family | Key Claim | Level of Analysis | Policy Implication |
|--------|-----------|-------------------|--------------------|
| **Classical** | Crime = rational calculation of cost/benefit | Individual | Increase deterrence, swift punishment |
| **Strain** | Crime = frustrated access to legitimate goals | Social structure | Expand opportunity, reduce inequality |
| **Social Control** | Crime = weak bonds to conventional society | Social bonds | Strengthen family, school, community |
| **Labeling** | Crime = created by societal reaction | Interaction/labels | Divert, decriminalize, reduce stigma |
| **Differential Association** | Crime = learned in intimate groups | Social learning | Disrupt criminal networks, mentorship |
| **Conflict** | Crime = power and class struggle | Macro/political | Challenge power structures |
| **Life-Course** | Crime = contingent on life events and turning points | Longitudinal | Target critical transitions |
| **Routine Activity** | Crime = motivated offender + suitable target + absent guardian | Situational | Environmental design, guardianship |

## Crime Measurement: The Dark Figure Problem

```
+----------------------------------------------------------+
|                  CRIME ICEBERG                           |
+----------------------------------------------------------+
|                                                          |
|  ~~~~ WATERLINE ~~~~                                     |
|                                                          |
|  OFFICIAL STATISTICS (UCR, NIBRS, police data)          |
|  [visible tip]                                           |
|                                                          |
|  DARK FIGURE                                             |
|  - Not reported to police                                |
|  - Reported but not recorded                             |
|  - Crimes police don't discover (white-collar, drugs)    |
|  [submerged mass]                                        |
|                                                          |
|  SELF-REPORT SURVEYS (NCVS, BCS, ICVS)                  |
|  [attempts to measure submerged portion]                 |
+----------------------------------------------------------+
```

**Key measurement sources:**
- **UCR (Uniform Crime Reports)** -- police agency reports, FBI-compiled, hierarchy rule problem
- **NIBRS (National Incident-Based Reporting)** -- replaces UCR, incident-level data
- **NCVS (National Crime Victimization Survey)** -- household survey, catches unreported crime
- **Self-report studies** -- captures offending behavior directly, good for drug/minor crime
- **ICVS (International Crime Victimization Survey)** -- cross-national comparison

## Key Distinctions

### Crime vs. Deviance

```
  DEVIANCE                    CRIME
  --------                    -----
  Violation of social norms   Violation of legal norms
  Context-dependent           State-enforced
  Examples: being rude,       Examples: theft, assault,
  tattoos in some contexts    fraud

  OVERLAP: Most crime is deviant;
           not all deviance is crime;
           not all crime is morally deviant (victimless)
```

### Mala in Se vs. Mala Prohibita

```
  MALA IN SE              MALA PROHIBITA
  (wrong in itself)       (wrong because prohibited)
  ---------------         ----------------------
  Murder, rape, theft     Drug possession, jaywalking,
                          tax evasion, speeding
  Near-universal          Varies by jurisdiction,
  consensus               era, political climate
```

### Primary vs. Secondary Deviance (Lemert)

```
  PRIMARY DEVIANCE         SECONDARY DEVIANCE
  ----------------         ------------------
  Initial rule violation   Deviance adopted as
  (anyone can do it)       master identity in
                           response to labeling

  You shoplift once  -->  Police/courts label you
                          "criminal"  -->
                          You internalize label  -->
                          Criminal career begins
```

## Common Confusion Points

**Criminology vs. Criminal Justice**
Criminology is the academic study of crime causation, patterns, and social responses. Criminal justice is the applied institutional system (police + courts + corrections). Criminology informs CJ policy but is analytically prior -- like epidemiology vs. public health administration.

**Consensus vs. Conflict View of Law**
Classical and most mainstream criminology assumes criminal law reflects societal consensus. Conflict/critical criminology argues law reflects the interests of dominant classes -- criminalization is a political act. Both views partially hold; the tension is empirically productive.

**Correlation vs. Causation in Crime Research**
Poverty correlates with crime. Poverty does not straightforwardly cause crime -- most poor people do not offend. The causal mechanisms run through social disorganization, strain, and weakened bonds, not poverty itself.

**Punishment vs. Deterrence**
Deterrence has two levers: *certainty* of punishment and *severity* of punishment. Research consistently shows certainty matters far more than severity. Mandatory minimums increase severity without increasing certainty -- limited deterrent effect.
