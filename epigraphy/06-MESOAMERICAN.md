# Mesoamerican Writing: Maya Glyphs, Aztec Pictography, Zapotec

## The Big Picture

```
MESOAMERICAN WRITING SYSTEMS
==============================

+------------------------------------------------------------------+
|  WRITING SYSTEM    APPROX DATE    STATUS       LANGUAGE          |
|  ---------------   -----------    --------     --------          |
|  Zapotec           500 BCE -       Partially    Zapotecan        |
|  (Monte Alban)     900 CE          read         (Otomanguean)    |
|                                                                  |
|  Maya Classic      200-900 CE      ~75-80%      Classic Maya     |
|  (fully developed) (to ~1700 CE)   deciphered   (Mayan family)   |
|                                                                  |
|  Epi-Olmec         ~150-450 CE     Partially    Mixe-Zoquean?    |
|  (La Mojarra)                      read         (debated)        |
|                                                                  |
|  Aztec/Nahuatl     ~1300-1521 CE   Not a        Nahuatl          |
|  pictography       (colonial too)   true writing (Uto-Aztecan)   |
|                                    system                        |
|                                                                  |
|  Olmec             ~900-400 BCE     Undeciphered Unknown         |
|  (Cascajal block)                                                |
+------------------------------------------------------------------+
```

**The Americas developed writing independently from the Old World.** These are genuine independent inventions, not diffusions from Mesopotamia or Egypt. The case for independent invention is strong: geographic isolation, different materials (stone stelae, bark-paper codices, not clay tablets), and distinctive sign inventories.

## Maya Writing: System Architecture

### Sign Count and Script Type

```
MAYA SIGN INVENTORY
====================

Total signs catalogued: ~800-1000 (including rare and variant)
Signs in regular use: ~200-300

SCRIPT TYPE: Mixed logographic-syllabic
  Logograms (word-signs): represent morphemes directly
  Syllabic signs: represent CV or V syllables
    -> Syllabary embedded within the larger system
    -> ~120 syllabic signs known

PARALLEL TO EGYPTIAN:
  Hieroglyphic: logograms + phonetic signs + determinatives
  Maya: logograms + syllabic signs + semantic classifiers
  Both systems use phonetic complementation
  Both are mixed systems, not pure phonetic or pure logographic

DIRECTION:
  Left to right within "glyph block"
  Glyph blocks arranged in double columns
  Double columns read left-to-right (paired)
  Then down to next level of double columns
  Path: col-pair 1 top, col-pair 1 middle, col-pair 1 bottom,
        then col-pair 2 top, etc.
```

### Glyph Block Structure

```
GLYPH BLOCK ANATOMY
====================

A "glyph block" = the unit of Maya writing (not a single sign)
Each block = typically 2-3 signs combined

+-------+
|  MAIN | <- Main sign: usually the logogram or main syllable
| SIGN  |
|  [T]  |
+--+--+-+
|Af|  |Af| <- Affixes: smaller signs attached to main sign
+--+--+-+
  Prefix (left/above)  Suffix (right/below)

Reading order within a glyph block:
  Prefix -> Main sign -> Suffix

Example: K'UH-na
  K'UH (logogram: "holy/divine")
  -na  (phonetic suffix: sound "na")
  Together: "k'uhna" = "holy place"

Phonetic complementation (same as Egyptian):
  Logogram + phonetic sign repeating part of its sound
  Confirms reading and disambiguates homophones
```

## Maya Long Count Calendar

```
LONG COUNT CALENDAR
====================

The Maya developed a positional numbering system
(base-20 with a modified 18 for the second position)
used to record dates from a fixed starting point.

UNITS:
  1 kin      =  1 day
  1 uinal    = 20 days (20 kins)
  1 tun      = 360 days (18 uinals -- the modification)
  1 katun    = 7,200 days (20 tuns)
  1 baktun   = 144,000 days (20 katuns)

DATE FORMAT: baktun.katun.tun.uinal.kin
Example: 9.16.4.1.1 (typical Classic Maya date)

CORRELATION WITH WESTERN CALENDAR:
  GMT correlation (Goodman-Martinez-Thompson):
  Maya "0 point" = August 11, 3114 BCE (Julian) or
                   September 6, 3114 BCE (Gregorian)
  This is the standard used in most scholarship
  The famous "2012 date" = 13.0.0.0.0 in Long Count
    = December 21, 2012 Gregorian
    = completion of 13th baktun, NOT "end of Maya calendar"
    (the calendar continues: 14.0.0.0.0, etc.)

SIGNIFICANCE FOR EPIGRAPHY:
  The Long Count dates on stelae allow absolute dating
  Maya inscriptions from ~250-900 CE (Classic period)
    have known dates accurate to the day
  This is unusually precise for ancient history
  Example: Stela 29 at Tikal = 9.0.0.0.0 = 292 CE exactly
```

## What Maya Inscriptions Actually Say

Before Proskouriakoff (1960), scholars thought Maya texts were purely astronomical and calendrical. They are not:

```
MAYA INSCRIPTION CONTENT
==========================

HISTORICAL NARRATIVE:
  Birth (sak nik nahal = "white flower first")
  Accession to power (chum ajaw = "seating of the lord")
  Warfare and capture (ch'ak = "axe event" = battle/conquest)
  Death (och b'ih = "road entering" = death metaphor)

POLITICAL GENEALOGY:
  Mother, father, patron deity
  Dynastic succession

BUILDING DEDICATIONS:
  When a monument was commissioned
  Who commissioned it
  What ceremony was performed

ASTRONOMICAL RECORDS:
  Venus table in Dresden Codex
  Eclipse predictions (Saros cycle tables)
  Mars stations

RELIGIOUS TEXTS:
  Creation narratives (Popol Vuh content paralleled in inscriptions)
  Deity invocations
  Sacrificial context records

WHAT IS NOT IN MAYA INSCRIPTIONS:
  Economics and commerce (no price lists)
  Personal letters
  Literary fiction
  -> Elite political and religious content dominates
  -> This is the survivorship bias of monumental media
```

## The Four Surviving Maya Codices

```
MAYA CODICES
=============

Only 4 pre-Columbian Maya screenfold books survive:
  De Landa (1562) burned hundreds at Mani, Yucatan.

DRESDEN CODEX (Sachsische Landesbibliothek, Dresden):
  Acquired by Dresden library 1739; origin unclear
  Astronomical: Venus tables, eclipse tables, almanacs
  Date: possibly 13th c. CE copy of earlier original
  Condition: good until 1945 (bomb damage: some pages)
  Most studied and best preserved

MADRID CODEX (Museo de America, Madrid):
  Almanacs, agricultural calendars, divination tables
  Longest codex (112 pages)
  Post-Classic period; cruder style
  Divided into two halves (Troano and Cortesianus) until recognized

PARIS CODEX (Bibliotheque Nationale, France):
  Katun prophecies, deity information
  Badly deteriorated (discovery in waste paper basket, 1859)
  Many pages illegible

GROLIER CODEX (now Maya Codex of Mexico):
  Authenticity disputed until 2016 (now accepted as genuine)
  Venus almanac; oldest surviving Maya codex (~1230 CE)
  Originally sold privately to Grolier Club, New York
```

## Aztec Pictography: Not a True Writing System

```
AZTEC PICTOGRAPHIC SYSTEM
===========================

The Aztec (Mexica) used pictographic records for:
  Historical chronicles (codices)
  Tax/tribute records (Matricula de Tributos)
  Ritual calendars (tonalamatl)
  Land records (maps with notations)

WHY NOT "TRUE WRITING":
  No phonetic decoding possible without external glosses
  Each image = a concept or event, not a sound
  You cannot "read" an Aztec codex aloud in Nahuatl
    without knowing what the image is supposed to represent
  -> Pictography, not script

PHONETIC ELEMENTS THAT EXIST:
  Rebus principle used for names:
  Place names often spell out phonetically:
  Coatepec = coatl (snake) + tepetl (hill) = Snake Hill
    -> picture of a snake + a hill = "Coatepec" phonetically
  Personal names similarly spelled phonetically
  THESE ARE PHONETIC -- but only for proper nouns

COMPARISON WITH MAYA:
  Maya: full phonetic writing of Mayan language
        (any Mayan sentence can be written and read)
  Aztec: primarily pictographic with rebus for names
         (Nahuatl sentences cannot be fully encoded)

The Aztec system is a mnemonic/pictographic system,
not a full writing system by standard linguistic criteria.
```

## Zapotec: Oldest Writing in the Americas

```
ZAPOTEC WRITING
================

Location: Monte Alban, Oaxaca, Mexico
Period: ~500 BCE - 900 CE
Language: Zapotecan (Otomanguean family)

Discovery: Danzante ("dancer") slabs at Monte Alban
  ~300+ carved stone monuments
  Figures in unusual poses (formerly called "dancers")
  Now recognized: conquest victims (mutilated figures)
  With inscriptions: dates, captive names

Status: partially read
  The calendar system (260-day ritual calendar) is understood
  Some glyphs have been read as logographic
  Many remain uncertain
  No bilingual text has been found

Significance:
  Predates Maya writing by ~500-700 years
  Suggests writing developed in Oaxaca region
  then spread to or developed independently in Maya area
  Maya may have been influenced by Zapotec
  (current scholarly debate; not resolved)
  -> Writing may have been invented once in Mesoamerica
     and spread, OR may have been independently invented
     in multiple Mesoamerican cultures
```

|--------|------|--------|--------|
| Maya Classic | Logo-syllabic (full writing) | 250-900 CE (continued to ~1700) | ~75-80% deciphered |
| Aztec | Pictographic (not full writing) | 1300-1521 CE | Readable with context; no phonetic decoding |
| Zapotec | Partially logo-syllabic | 500 BCE - 900 CE | Partially read |
| Olmec | Unknown | 900-400 BCE | Undeciphered |

| Term | Meaning |
|------|---------|
| Long Count | Maya vigesimal calendar; dates inscriptions to the day |
| Glyph block | Maya text unit (main sign + affixes combined) |
| Screenfold | Folded bark-paper book (codex); accordion-fold |
| Chum ajaw | "Seating of the lord" = accession formula |
| Rebus principle | Writing a sound by using a picture of something with that sound |

## Common Confusion Points

**Aztec pictography is not a writing system.** It cannot encode any Nahuatl sentence phonetically. It is a sophisticated mnemonic and documentary system — admirable on its own terms — but it is categorically different from Maya script, which can write any utterance in a Mayan language.

**The Maya 2012 "end of the world" was a popular misconception.** The Long Count reached 13.0.0.0.0 on December 21, 2012 — the completion of the 13th baktun. The Maya calendar continues; the Maya themselves made no apocalyptic predictions for this date. This is a Western projection onto the Maya calendar.

**Thompson's suppression of Knorozov was a significant scholarly harm.** J. Eric Thompson dominated Maya epigraphy from the 1940s through the 1970s. His opposition to phonetic decipherment — based partly on Cold War antipathy and partly on aesthetic objection — delayed the decipherment by ~20 years. The 1986 breakthrough could have come in the late 1960s.

**Not all Maya codices were destroyed.** De Landa burned hundreds; four survive (three definitively, one recently authenticated). Additionally, many Post-Classic inscriptions survive on stone. The Classic Maya corpus is primarily in stone (stelae, lintels, stairways, buildings).

**Zapotec may be older than Maya.** Monte Alban inscriptions (~500 BCE) predate the earliest clearly dated Maya inscriptions (~300 BCE). The relationship between Zapotec and Maya writing is an active research area.
