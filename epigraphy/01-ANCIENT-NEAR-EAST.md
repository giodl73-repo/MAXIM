# Ancient Near East: Cuneiform, Sumerian, Akkadian, Ugaritic

## The Big Picture

```
ANCIENT NEAR EAST WRITING TIMELINE
=====================================

3400-3000 BCE: Proto-Cuneiform (Uruk IV-III)
  Logographic: pictographs on clay, administrative
  No phonetic component yet

3000-2500 BCE: Early Cuneiform
  Signs become more abstract (stylus rotation)
  Phonetic principle emerging (rebus writing)
  Still primarily administrative

2500-2000 BCE: Classic Sumerian Cuneiform
  Full writing system: logograms + syllabic signs
  Literature: Gilgamesh precursors, hymns
  600+ signs in use

2400-2000 BCE: Akkadian Adaptation
  Semitic language using Sumerian sign system
  Syllabic use expands; some Sumerian logograms retained
  ~400 signs active in Akkadian

1400 BCE: Ugaritic Alphabet
  30-letter cuneiform abjad on clay
  Radical simplification: one sign per consonant
  Used for local alphabet not Mesopotamian system

1200 BCE: Bronze Age Collapse
  Many scribal traditions disrupted
  Cuneiform survives in reduced form

500s BCE: Achaemenid Persian Empire
  Old Persian cuneiform (36 signs, quasi-alphabetic)
  Used alongside Akkadian and Elamite on trilingual texts
  Behistun Inscription: key to Akkadian decipherment

75 CE: Last known cuneiform text
```

## How Cuneiform Works

### Physical Mechanics

```
CUNEIFORM PRODUCTION
=====================

TOOL: reed stylus (calamus)
  Cut at an angle to produce a triangular tip
  Pressed into wet clay at ~45 degrees
  Two basic strokes:
    Vertical wedge: tip pressed straight in -> head down
    Horizontal wedge: tip pressed at angle -> head left
    Diagonal: intermediate angles

MEDIUM: clay tablet
  Sizes: thumbnail (single account) to large biscuit (~30cm)
  Tablet orientation: landscape or portrait
  Writing surface: rows left-to-right, top-to-bottom
  Multiple columns possible for long texts
  Reverse side used when front full
  Colophon on final column: scribe's name, date, purpose

SIGN EVOLUTION (pictographic -> abstract):
  Original pictograph: head of ox = aleph
  Rotation: signs rotated 90 degrees (reading direction change)
  Simplification: fewer strokes, more stylized
  Result: "cuneiform" (Latin cuneus = wedge)
  The wedge shape is an artifact of the stylus + clay medium

FIRING:
  Administrative tablets: air-dried (permanent enough)
  Important documents: sometimes fired (permanent)
  Lucky: Nineveh fire (612 BCE) fired the library tablets,
  preserving them when the palace was destroyed
  Archive = 30,000 fragments recovered by Layard, 1840s-1850s
```

### Sign Count and Complexity

```
SIGN COUNT BY PERIOD
=====================

Uruk IV (3400-3200 BCE):    ~900 signs (proto-cuneiform)
Uruk III (3200-3000 BCE):   ~800 signs
Early Dynastic (3000-2500): 600-700 signs
Old Babylonian (2000-1600): ~400 signs in regular use
Neo-Assyrian (900-600 BCE): ~300 signs in regular use
Late Babylonian (600-0 BCE): ~200 signs in regular use

WHY THE DECLINE:
  Administrative needs drove sign reduction
  Scribes preferred efficiency
  Phonetic use expanded; logographic use concentrated
  in traditional/religious texts (prestige conservatism)
```

## Sumerian: The Original Language of Cuneiform

Sumerian is:
- A **language isolate**: no demonstrable genetic relationship to any other language
- The first language for which we have writing
- An **ergative-absolutive** language (subject of intransitive = object of transitive)
- Agglutinative: builds words by stacking suffixes

```
SUMERIAN WRITING FEATURES
===========================

LOGOGRAMS (word-signs):
  One sign = one word or morpheme
  Sign for "head" + sign for "mouth" = "speak"
  Sign for "mouth" alone = "mouth" or "speak" or "eat"
  Context determines which reading

DETERMINATIVES (semantic classifiers):
  Signs prepended/appended to indicate category
  DINGIR (deity marker): precedes god names
  KI (place marker): follows place names
  GIS (wood/tree marker): precedes wooden objects
  These are not pronounced -- they are category flags
  Analogy: an invisible semantic tag on each word

PHONETIC COMPLEMENTATION:
  Logogram + phonetic sign indicating how to read it
  Helps distinguish multiple readings of same sign
  Critical in later decipherment work:
  Akkadian scribes added phonetic clues for their own use

SYLLABIC SIGNS:
  Sumerian: V, CV, VC, CVC types
  e.g.: a, an, na, nun, la, ta, etc.
  These can spell out words phonetically
  Used for grammatical particles, foreign names,
  words without a dedicated logogram
```

## Akkadian Cuneiform: Semitic Language, Sumerian Script

Akkadian (including its dialects Babylonian and Assyrian) was a Semitic language that adopted the Sumerian writing system around 2400 BCE:

```
AKKADIAN ADAPTATION CHALLENGES
================================

Sumerian: monosyllabic and agglutinative
           simple consonant + vowel syllable structure
Akkadian: Semitic language with complex consonant clusters
          (Semitic roots are typically 3 consonants)
          e.g.: root K-T-B (writing) -> katbu, aktub, niktub

Problem: Sumerian syllabic signs didn't cover Akkadian phonology
  Akkadian has emphatic consonants (q, t, s with pharyngeal backing)
  not present in Sumerian
  Solution: repurpose existing signs; add new signs; tolerate ambiguity

AKKADIAN SYLLABARY:
  ~300 signs in regular Babylonian use
  Each sign may have multiple readings (polyphony):
    sign can be read as a syllable, a Sumerian logogram,
    or an Akkadian logogram depending on context
  This is why cuneiform is hard: the same sign has
  multiple possible readings; context disambiguates

SUMEROGRAMS IN AKKADIAN TEXT:
  Scribes retained Sumerian logograms for prestige vocabulary
  Read them in Akkadian not Sumerian
  (Like writing "£" in an English text -- the sign is not English)
  This creates "scribal Sumerian" stratum in Akkadian texts
  Understanding Akkadian cuneiform requires knowing both systems
```

## The Behistun Inscription and Akkadian Decipherment

```
BEHISTUN INSCRIPTION
=====================

Location: cliff face on Mount Behistun (Bisotun), western Iran
          on the royal road from Babylon to Ecbatana
          at ~100m elevation, deliberately inaccessible
Commissioned by: Darius I (reigned 522-486 BCE)
Date: ~520 BCE

Content:
  Darius's account of his seizure of power after Cambyses
  List of rebel kings defeated
  27 paragraphs, 1200 lines

Languages:
  Old Persian (39 columns)
  Elamite (67 columns)
  Babylonian Akkadian (112 columns)
  -> Trilingual like the Rosetta Stone, but in cuneiform

Henry Rawlinson (1810-1895):
  British army officer, Major in Persian army
  Copied the inscription 1835-1847 (rope descents)
  1847: published complete Babylonian column transcription
  Worked from Old Persian (partially known from Zoroastrian texts)
  -> identified proper names -> bootstrapped Babylonian reading

Decipherment validation:
  1857: Rawlinson, Hincks, Fox Talbot, Oppert
  Each independently translated same new Akkadian text
  Royal Asiatic Society compared results
  Substantial agreement -> decipherment confirmed
```

## Ugaritic: Cuneiform Alphabet

```
UGARITIC ALPHABET (c. 1400-1200 BCE)
======================================

Location: Ugarit (Ras Shamra), coastal Syria
          Port city, Bronze Age trade hub

System: 30 signs + 3 additional for /'a/, /'i/, /'u/
        (marking initial vowel with aleph)
        One sign per consonant (abjad)
        Written on clay with stylus (cuneiform technique)
        But: no Sumerian or Akkadian logograms
             purely phonetic system

Discovery: French excavations 1929-
Decipherment: 1930 (Hans Bauer, Edouard Dhorme, Charles Virolleaud)
  -- extremely rapid because:
  1. Sign count was small (30 signs)
  2. Sign inventory could be counted
  3. Short texts with single characters = alphabet
  4. Related to known Semitic languages
  5. Bilingual tablets at Ugarit (Akkadian + Ugaritic)

Content:
  Baal Cycle (mythology): storm god vs. sea god; vs. death
  Kirta Epic (heroic narrative)
  Aqhat Epic (heroic narrative)
  Administrative texts
  Religious and ritual texts

Significance:
  Earliest known alphabet of any kind
  Shows that alphabetic writing was already being
  adapted to cuneiform medium by 1400 BCE
  Predecessor to Phoenician abjad -> Greek alphabet
```

## Cuneiform Phase Diagram

```
WRITING SYSTEM COMPLEXITY (approximate sign count)
=====================================================

Uruk Proto-Cuneiform  900 |  ####################################
Early Dynastic        700 |  ############################
Old Babylonian        400 |  ################
Neo-Assyrian          300 |  ############
Late Babylonian       200 |  ########
Ugaritic abjad         30 |  #
Greek alphabet         24 |  #

Note: the compression from 600+ signs (full cuneiform)
      to 30 signs (abjad) is a radical cognitive simplification.
      Learning cuneiform required years of scribal school.
      Learning the Ugaritic or Phoenician abjad:
      probably a few weeks.

This is why alphabets spread: they are learnable
by non-specialists. Cuneiform required a professional
scribal class to maintain.
```

## Scribal Education

```
MESOPOTAMIAN SCRIBAL SCHOOL (edubba = "tablet house")
======================================================

Duration: childhood to young adulthood (10-20 years)
Curriculum:
  Year 1-3: sign lists (lists of signs, not texts)
  Year 3-7: standard literary texts (canonical repertoire)
  Year 7+: specialized (law, mathematics, divination, medicine)

Teaching method:
  Teacher writes exemplar on one half of tablet
  Student copies on other half
  "Model contracts" -- formulaic legal texts
  Student corrected by teacher -- error-correction process

Output: trained scribe could:
  Write in multiple languages (Sumerian + Akkadian at minimum)
  Read sign lists, literary texts, administrative forms
  Produce legally valid documents

Social status:
  High -- but subservient to priests and kings
  The scribe is an employee of an institution
  Personal name + title preserved in colophons
  Some scribes became scholars (tupsar: "tablet writer")
  -> Astronomy, medicine, divination, mathematics
```

## Common Confusion Points

**Cuneiform is a script, not a language.** Multiple languages were written in cuneiform: Sumerian, Akkadian, Elamite, Hittite, Old Persian, Ugaritic. The script is a technology; the languages are distinct. When someone says "cuneiform text," they must specify which language.

**Sumerian and Akkadian coexisted in the same documents.** Sumerograms (Sumerian logographic signs) appear in Akkadian texts. Scribes read them in Akkadian. It is like using "£" in an English sentence — the sign is Sumerian, the reading is Akkadian.

**Ugaritic is not "simplified cuneiform."** It is an independent abjad adapted to the cuneiform writing technique (clay + stylus). It has no genetic relationship to the Mesopotamian logosyllabic system — it just uses the same physical inscription method.

**Clay tablets survived because they were fired.** Many tablets survived through accidental firing when the buildings housing them burned. Intentional firing was rarer; most tablets were just air-dried. The Nineveh "library" survived because the palace burned.

**Decipherment of Akkadian was a collaborative and competitive process.** Rawlinson, Hincks, Oppert, and Fox Talbot were simultaneously working on different aspects. The 1857 "test" (each independently translating the same new text) is a scientific validation event, not a race.
