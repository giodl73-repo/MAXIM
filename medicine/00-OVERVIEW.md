# 00 — Medicine & Pharmacology Overview

## PK/PD, Drug Development, Clinical Fundamentals

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Pharmacokinetics (PK): ADME — Absorption (bioavailability, first-pass effect, routes), Distribution (Vd, protein binding, BBB penetration), Metabolism (CYP450 system, Phase I/II reactions, inducers/inhibitors), Elimination (renal clearance, half-life, steady state)
- Pharmacodynamics (PD): receptor types, dose-response curves, EC50/Emax, therapeutic index (TI = TD50/ED50), agonist/antagonist/partial agonist/inverse agonist
- Drug-drug interactions: PK interactions (CYP induction/inhibition), PD interactions (synergy/antagonism)
- Drug development pipeline: target ID → lead compound → IND → Phase I (safety/dose) → Phase II (efficacy/dose) → Phase III (RCT) → NDA → Phase IV (post-marketing)
- Regulatory: FDA approval pathways, accelerated approval, biosimilars, generic substitution (bioequivalence)
- Special populations: renal/hepatic dosing adjustment, pediatric PK differences, pregnancy (FDA risk categories → PLLR), elderly (Beer's criteria)
- Drug naming conventions: generic (INN), brand, chemical; −mab / −nib / −umab / −zumab / −olol / −pril naming stems
- Index of all drug class modules in this directory
