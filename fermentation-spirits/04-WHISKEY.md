## Scotch Whisky

### The Legal Framework (Scotch Whisky Regulations 2009)

```
SCOTCH WHISKY CATEGORIES

SINGLE MALT                     SINGLE GRAIN
  One distillery                  One distillery
  Malted barley only              Any cereal grain
  Pot still                       Column still (Coffey/patent)
  Scotland                        Scotland
  Min 3 years oak                 Min 3 years oak
  Example: Glenfiddich, Lagavulin Example: Cameronbridge, Girvan

BLENDED MALT                    BLENDED GRAIN
  Multiple distilleries           Multiple distilleries
  All single malt                 All single grain
  Example: Monkey Shoulder        Rare; specialty product

BLENDED SCOTCH
  Mix of single malts + single grains from multiple distilleries
  ~90% of all Scotch sold
  Examples: Johnnie Walker, Chivas Regal, Dewar's, Ballantine's
  Blender's art: consistency across years with limited age-statement stocks
```

### Peat

```
PEAT FORMATION:
  Dead sphagnum moss + plant matter -> compressed over millennia
  -> Peat bogs in Scotland, Ireland, Islay, Orkney

PEAT IN MALTING:
  Kilning with burning peat fires -> smoke permeates malt
  Phenolic compounds (guaiacol, cresol, syringol) adsorb to malt

MEASUREMENT: PPM phenols (parts per million in finished malt)
  Unpeated:    0-5 PPM
  Lightly peated: 5-15 PPM
  Moderately:  15-30 PPM
  Heavily:     30-50 PPM (Caol Ila, Bowmore)
  Very heavy:  50+ PPM (Ardbeg ~55, Bruichladdich Octomore ~80-167)

NOTE: Malt PPM != whisky PPM (some phenols lost in distillation/aging)
      Ardbeg malt ~55 PPM -> finished whisky ~17-20 PPM phenols

SENSORY: Peat smoke, iodine, medicinal, campfire, seaweed, ash
```

### Cask Maturation

```
CASK TYPES AND FLAVOR CONTRIBUTION

EX-BOURBON (American white oak, Quercus alba)
  Most common cask for Scotch
  Source: bourbon legally requires new oak, so bourbon barrels are available
          after one use; sold to Scotch industry
  Character: vanilla, coconut, caramel, lighter spice
  Size: standard barrel (190L) or hogshead (250L, re-built from barrel staves)

SHERRY CASK (European oak, Quercus robur or sessilflora)
  PX (Pedro Ximenez): very sweet sherry -> dried fruit, chocolate, Christmas cake
  Oloroso: oxidized dry sherry -> nut, orange peel, complexity
  First-fill vs second-fill: first-fill more flavor extraction
  Expensive: sherry butts (500L) command premium; some European casks made new
             and seasoned with sherry for purpose
  Examples: Macallan (sherry cask dominant), Glendronach

PORT PIPE / WINE CASKS
  Port: rich fruit, jammy
  Sauternes: honey, apricot (Glenmorangie Nectar D'Or)
  Bordeaux: red fruit
  "Finish": secondary maturation in specialty cask (months)

VIRGIN OAK
  No prior use -> maximum wood extraction
  Spicy, tannic; common in bourbon/Texan whiskey

MIZUNARA OAK (Quercus mongolica) -- Japanese specialty
  Rare, difficult wood: very porous, hard to cooperage
  Extremely long seasoning required
  Character: sandalwood, incense, coconut, oriental spice
  Premium pricing due to scarcity

COOPERAGE SCIENCE:
  Toast level (light/medium/heavy): Maillard + caramelization at stave surface
  Entry proof: high ABV extracts more wood compounds
  Warehouse: temperature swing -> "breathing" in/out of wood
  Position: top floor warehouse hotter -> faster extraction (Kentucky rickhouse)
```

## Irish Whiskey

```
IRISH WHISKEY CATEGORIES

SINGLE POT STILL WHISKEY (uniquely Irish)
  Mix of malted AND unmalted barley in same mash
  Copper pot still distillation
  Unmalted barley: grainy, spicy, oily character ("pot still character")
  Triple distillation standard (though not legally required)
  Most revived post-Troubles of Irish distilling industry
  Examples: Redbreast, Green Spot, Yellow Spot, Powers (Jameson blend)

SINGLE MALT
  Same as Scotch category but made in Ireland
  Examples: Teeling Single Malt, Bushmills single malts

SINGLE GRAIN
  Column still, any cereal grain
  Light, delicate

BLENDED IRISH WHISKEY
  Mix of the above categories
  Examples: Jameson (world's #1 Irish), Tullamore DEW, Powers Gold Label
  Jameson formula: triple-distilled pot still + grain whisky, sherry + bourbon casks

KEY DIFFERENCES FROM SCOTCH:
  Triple distillation (typically): lighter, smoother character
  Unmalted barley in pot still: distinctive Irish style
  Generally no peat (Connemara single malt is exception)
  3-year minimum age
  Monopoly era: Irish Distillers Ltd controlled most Irish whiskey 1960s-2000s
               -> innovation suppressed; now 40+ distilleries
```

## Bourbon

### Legal Requirements (27 CFR Part 5)

```
BOURBON LEGAL DEFINITION

+------------------------------------------------------------------+
| MUST be:                                                          |
|   Produced in the UNITED STATES                                   |
|   Mash bill: 51%+ CORN (remainder: rye, wheat, malted barley)    |
|   Distilled to no more than 160 proof (80% ABV)                  |
|   Barreled at no more than 125 proof (62.5% ABV)                 |
|   Aged in NEW, CHARRED OAK CONTAINERS                            |
|   Bottled at no less than 80 proof (40% ABV)                     |
|                                                                   |
| NO AGE REQUIREMENT for plain "bourbon"                           |
|                                                                   |
| "STRAIGHT bourbon": minimum 2 years in new charred oak           |
|   (age statement required if under 4 years)                      |
|                                                                   |
| "BOTTLED IN BOND": 4 years minimum, one distillation season,     |
|   one distillery, 100 proof (50% ABV) -- Bottled in Bond Act 1897|
+------------------------------------------------------------------+
```

**Why new charred oak?**
Partly historical accident -- American coopers had abundant virgin wood.
Also: barrel industry lobbied successfully for this regulation.
The char layer is critical: charcoal filtration layer removes sulfurs,
lactones from wood provide vanilla/caramel, and the char adds smoky complexity.

### Mash Bill Families

```
BOURBON MASH BILL COMPARISON

HIGH RYE:                    HIGH WHEAT:                 WHEATER:
  Corn: 70-75%                 Corn: 68-75%               Corn: 65-70%
  Rye: 20-25%                  Wheat: 18-20%              Wheat: 18-20%
  Barley: 5-10%                Barley: 8-10%              Barley: 8-10%
  Spicy, dry, complex          Soft, fruity, smooth       Softer rye influence
  Four Roses (high), Bulleit   Maker's Mark,              W.L. Weller,
  Woodford Reserve             Larceny, Bernheim           Pappy Van Winkle

Note: Pappy Van Winkle is made from the same mash as W.L. Weller
      at the same distillery (Buffalo Trace) -- just different barrel selection and age
```

### Sour Mash Process

```
SOUR MASH (nearly universal in American whiskey)

PURPOSE:
  1. pH control: acidic backset lowers mash pH -> inhibits bacterial growth
  2. Consistency: same microbial environment batch to batch
  3. Flavor continuity: maintains yeast population profile

PROCESS:
  Backset = stillage (spent liquid from bottom of column still)
  Typically 25% of mash bill replaced with backset
  "Sour" refers to the acidic nature, NOT lactic sourness in the finished whiskey

YEAST: Each distillery jealously guards its yeast strain(s)
  Buffalo Trace: proprietary strain; stored in multiple locations
  Four Roses: OESK, OESV, OBSK, OBSV, etc. (5 mash bills x 2 yeast = 10 recipes)
  Jim Beam: descended from original Beam family strain

SWEET MASH (rare exception):
  Fresh water, no backset; less flavor control
  Used by some craft distilleries for a different profile
```

### Kentucky Geography

```
WHY KENTUCKY?

Limestone shelf:
  Water filters through limestone -> removes iron, adds calcium/magnesium
  Iron-free water critical: iron causes off-colors in spirit

Climate:
  Hot summers (barrel expansion, wood absorption)
  Cold winters (barrel contraction, extraction ceases)
  -> Dramatic seasonal cycling -> faster maturation than Scotland

Rickhouses:
  Multi-story warehouse; top floors hotter, more temp swing, faster maturation
  Floor selection = flavor decision
  Fungal blackening (Baudoinia compniacensis) on exterior = "warehouse fungus"
  Lives on ethanol vapors; harmless

Bourbon belt: 95% of world's bourbon made in Kentucky
  Not legally required (any US state)
  Colorado, Texas, Indiana, New York: growing bourbon production
  "Kentucky Straight Bourbon" = specific geographic designation
```

## Rye Whiskey

```
RYE WHISKEY (American)

LEGAL: 51%+ rye in mash bill (otherwise same as bourbon)
  Typical: 51-53% rye (low rye) or 95% rye (high rye, MGP of Indiana)
  Aged in new charred oak
  Can be "straight" (2yr+) or not

CHARACTER vs BOURBON:
  Rye: spicy, dry, herbal, less sweet
  Rye grain contains more complex carbohydrates -> more fusel oils -> spice
  Pre-Prohibition: rye was the dominant American whiskey (Maryland, Pennsylvania)
  Bourbon displaced rye post-Prohibition; rye revival 2010s+

CANADIAN WHISKY confusion:
  "Canadian rye" does NOT require 51% rye
  Canadian labeling uses "rye" colloquially for any blended Canadian whisky
  Actual Canadian rye (some craft): separate legal category
  Most commercial Canadian: light blended grain whisky with small rye addition
```

## Tennessee Whiskey and the Lincoln County Process

```
LINCOLN COUNTY PROCESS (Jack Daniel's, George Dickel)

  New-make corn whiskey
      |
      | CHARCOAL FILTRATION through 10 feet of sugar maple charcoal
      | (made by burning hard maple, then dousing with flame)
      |
      v
  Filtered spirit -> new charred oak barrel -> aging

This step REMOVES the spirit from the bourbon legal category
(bourbon cannot have its character altered by filtering before barreling)
-> Tennessee whiskey is not bourbon by law (though distilled similarly)
-> Jack Daniel's is not bourbon (despite looking, tasting, and being made like bourbon)

LEGAL STATUS:
  Tennessee whiskey is a separate legal category (US law, Tennessee state law)
  Must be produced in Tennessee, from 51%+ corn, filtered through maple charcoal,
  and aged in new charred oak
  Jack Daniel's and George Dickel are the two major producers

FLAVOR EFFECT:
  The charcoal filtration removes some fusel oils and congeners
  Adds a subtly sweet, slightly smoky character distinct from standard bourbon
  The difference is subtle but measurable in side-by-side tasting
```

## Japanese Whisky

```
JAPANESE WHISKY

ORIGINS:
  Masataka Taketsuru studied at Hazelburn, Glasgow 1919
  Returned to Japan -> founded Yoichi distillery 1934 (Hokkaido)
  Shinjiro Torii founded Suntory -> Yamazaki distillery 1923 (near Kyoto)
  -> Both companies are major conglomerates today
  (Nikka = former Taketsuru company, now owned by Asahi)

SCOTCH-INSPIRED BUT DIFFERENT:
  Multiple still shapes within single distillery (unusual in Scotland)
  -> Creates different spirit types for in-house blending
  Traditional Scotch approach: buy from other distilleries to blend
  Japanese approach: self-sufficient; Japanese distilleries rarely trade stocks

MIZUNARA OAK:
  Japanese oak (Quercus mongolica/crispula)
  Very porous: difficult cooperage; leaks; expensive
  Long seasoning time (decades) before suitable
  Character: sandalwood, incense, coconut -- distinctly "Japanese" profile
  Premium: Hibiki 21 Year Mizunara Edition commands $1,000+

SHORTAGE CRISIS 2010s:
  Surge in global demand 2010-2018 (largely driven by whisky press awards)
  Whisky takes 10-20+ years to mature
  -> Stocks dramatically depleted
  -> Most 12yr, 17yr, 21yr expressions discontinued or severely restricted
  -> No-age-statement (NAS) products proliferated
  -> Prices for aged expressions 10x+ in 5 years
  Lesson: whisky production capacity planned decades ahead -- demand spike cannot
          be met short-term

LEGAL SITUATION (historically lax):
  Until 2021: no Japanese legal definition for "Japanese Whisky"
  Any spirit could be labeled "Japanese" -- including imported Scottish blends
  2021: Japan Spirits & Liqueurs Makers Association voluntary standards
         -> Malted grain, distilled in Japan, oak aged in Japan 3yr min, 40% ABV min
         -> Not legally binding but market pressure applies
```

## Age Statements and NAS

```
AGE STATEMENT GUIDE

  "12 Year" = youngest whisky in the blend is 12 years old
  Older whiskies can be included; stated age is the minimum

  "No Age Statement (NAS)": growing trend driven by shortage economics
    Pro-NAS: allows master blender to optimize flavor without age constraints
    Anti-NAS: removes consumer's information; enables price inflation

  SCOTCH AGING RULES:
    3yr minimum for any Scotch
    Age statement = youngest in vatting
    Cask type: first-fill counts differently than third-fill for flavor
               (but legally both count toward age)

  BOURBON:
    No minimum age (unlike "straight")
    "Straight" = 2yr minimum
    "Bottled in Bond" = 4yr minimum, bonded warehouse, 100 proof
    Many premium bourbons don't state age (Buffalo Trace Antique Collection)

  JAPANESE:
    Before 2021 voluntary standards: no rules
    After 2021 JSLMA guidelines: 3yr minimum for "Japanese Whisky" label
```

## Common Confusion Points

**"Single malt = better than blended."**
Marketing mythology. Many top blends (Johnnie Walker Blue, Compass Box) are
technically superior to mass-market single malts. "Single" means one distillery,
not one cask or batch. "Malt" means malted barley, not quality.

**"Scotch and bourbon are both aged in oak; why taste so different?"**
Three factors: new vs used oak (bourbon: new = heavy extraction; Scotch: used = subtle),
grain bill (corn vs malted barley), and distillation proof (bourbon max 80% = more
congeners than high-column grain neutral at 94%).

**"Bourbon must be made in Kentucky."**
False. Any US state. "Kentucky Straight Bourbon" is a geographic designation.
Virginia (A. Smith Bowman), Texas (Garrison Bros), New York (Tuthilltown) all
make legal bourbon.

**"Higher proof bourbon is better."**
"Cask strength" (uncut) bottles allow personal dilution to preference. They are not
inherently "better," just undiluted. Some drinkers prefer the intensity; others find
high-proof harsh. Water opens up aroma compounds.

**"Japanese whisky is rare and expensive because of quality."**
Quality is high. But the scarcity/price is driven by demand shock meeting supply that
takes a decade to replenish. The shortage is a production economics problem,
not a reflection of intrinsic worth vs other world whiskies.
