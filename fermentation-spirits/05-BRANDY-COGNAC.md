## Cognac

The world's most prestigious brandy and the defining example of grape distillate.

### The Geography (AOC Cognac)

```
COGNAC APPELLATION CONTROLLEE
  Departments: Charente + Charente-Maritime
  City of Cognac at center

COGNAC CRUS (sub-zones, best to least regarded):

  +----------------------------------------------+
  |  GRANDE CHAMPAGNE (35,000 ha)                |
  |  Cretaceous chalky limestone (tufa/chalk)    |
  |  Slowest-maturing, most delicate, finest     |
  |  "Fine Champagne" = blend Grande + Petite    |
  |                                              |
  |  PETITE CHAMPAGNE (Surrounds Grande)         |
  |  Similar chalk soils, slightly less fine     |
  |                                              |
  |  BORDERIES (northwest, smallest cru)         |
  |  Clay + flint; round, violet/nutty notes     |
  |  Fastest-maturing; blender's secret weapon   |
  |                                              |
  |  FINS BOIS (surrounding ring)                |
  |  Clay + limestone; quicker to mature         |
  |                                              |
  |  BONS BOIS + BOIS ORDINAIRES (outer zones)   |
  |  Sandy/clay; less distinctive                |
  +----------------------------------------------+
```

**Why the chalk soils matter:**
Free-draining limestone forces shallow root expansion and restricts yield.
Grapes develop high acidity and low alcohol -- ideal distillation substrate.
The mineral character from calcareous soils contributes to what cognac traders
call "rancio" -- the nutty, cheese-rind, mushroom character that develops with age.

### The Grape: Ugni Blanc (Trebbiano)

```
UGNI BLANC PROFILE:
  ABV at harvest: 7-9% (very low alcohol -- stressed, acidic)
  Acidity: Very high (pH 2.8-3.1; TA 8-10 g/L tartaric)
  Flavor as wine: Thin, harsh, barely drinkable
  As distillation substrate: IDEAL

  Why ideal:
    High acid -> microbial stability during fermentation (no SO2 needed)
    Low alcohol -> more distillation runs per volume (economics)
    Neutral flavor -> distillation can develop terroir character

  Also permitted: Folle Blanche, Colombard, Montils, Sémillon
  But Ugni Blanc = 90%+ of production
```

### Double Distillation: The Charentais Process

```
COGNAC DISTILLATION (regulated, specific equipment)

CHARENTAIS POT STILL:
  Copper onion-shaped pot (chapiteau = head)
  Heated by direct flame (or steam)
  Small lyne arm with optional chauffe-vin (heat exchanger)
  Condenser

TWO RUNS:

RUN 1: "Premiere chauffe" (first heating)
  Input: Wine (~8-9% ABV)
  Output: Brouillis (~28-35% ABV)
  Duration: ~8-9 hours per charge
  Cut: Simple; some heads/tails removed

RUN 2: "Bonne chauffe" (good heating)
  Input: Three batches of brouillis combined
  Output: Eau-de-vie at 68-72% ABV (the target range)
  Duration: ~12 hours per charge
  Cut: Tetes (heads) removed: ~1%
       Coeur (heart): 68-72% -- the kept portion
       Secondes (seconds): added to next bonne chauffe
       Queue (tails): added to next premiere chauffe

LEGAL REQUIREMENT: Must be in copper Charentais still
  Continuous distillation prohibited for Cognac
  No Coffey still, no column still
```

### Cognac Aging

```
COGNAC AGING

OAK: Limousin (Quercus petraea, wider grain, more tannin)
     Tronçais (tight grain, less tannin, more elegant)
     New or old oak

PROCESS:
  New cognac placed in new oak barrels (225-350L) initially
  -> Rapid extraction of oak tannins, lactones
  Then moved to older barrels (50-100yr old) -- "demi-muids"
  -> Slow micro-oxidation without excess wood
  Finally: large "foudres" (old large vats) for extended aging
          -> Very slow oxidation, rancio development, no new wood

"PARADIS":
  Oldest cognac stocks stored in glass demijohns (not wood)
  -> Aging arrested; these are the blending components for XO, Louis XIII
  Some paradis stocks 100+ years old

ANGEL'S SHARE in Cognac:
  ~2.5-3% per year
  "The angels' portion" builds up as black wall mold
  (Baudoinia compniacensis -- same fungus as Kentucky rickhouses)

AGE DESIGNATIONS (minimum age of youngest component):
  VS (Very Special): minimum 2 years
  VSOP (Very Superior Old Pale): minimum 4 years
  XO (Extra Old): minimum 10 years (raised from 6 years in 2018)
  Hors d'Age / Extra: beyond official classification; often 20-30+ years
  Vintage Cognac: single year; rare; not a legal category but practiced
```

### Major Cognac Houses

```
THE "BIG FOUR" (control ~80% of Cognac production):

HENNESSY (LVMH): Largest. Volume leader. "Paradis" range.
REMY MARTIN: Grande Champagne and Fine Champagne specialist.
MARTELL: Oldest (1715). Borderies specialist; distinctive profile.
COURVOISIER: "Emperor's brandy" (Napoleon legend, somewhat apocryphal).

ARTISAN/QUALITY HOUSES:
  Hine, Delamain (old rare cognac specialists)
  Pierre Ferrand, Tesseron (quality mid-size)
  Camus (family-owned)

COGNAC BUSINESS MODEL:
  Negociant model: houses buy grapes/wine/eau-de-vie from growers
  Small growers (vignerons) sell to houses or bottle own brand
  Cooperatives: collect and sell to houses
  "Recoltant" = estate cognac (growing, making, bottling same place)
```

## Armagnac

```
ARMAGNAC vs COGNAC COMPARISON

+-----------------------------------------------------------+
|                   ARMAGNAC          COGNAC                |
|                   --------          ------                |
| Region            Gascony (Gers,    Charente region       |
|                   Lot-et-Garonne,                         |
|                   Landes)                                 |
| Appellation       Bas-Armagnac,     Grande Champagne,     |
|                   Armagnac-         Petite Champagne,     |
|                   Tenareze,         Borderies, etc.       |
|                   Haut-Armagnac                           |
| Grapes            Ugni Blanc,       Ugni Blanc dominant   |
|                   Baco Blanc (unique),                    |
|                   Folle Blanche,                          |
|                   Colombard                               |
| Still type        Alembic armagnacais  Charentais pot still|
|                   (continuous         (double distillation)|
|                   column still)                           |
| ABV out           52-72%             68-72%               |
| Character         Robust, rustic,    Refined, delicate    |
|                   earthy, prunes     floral, complex      |
| Single vintage    Common practice    Not a legal category |
| Aging minimum     VS: 1yr            VS: 2yr              |
|                   VSOP: 4yr          VSOP: 4yr            |
|                   XO: 10yr           XO: 10yr             |
| Scale             ~6M bottles/yr     ~200M bottles/yr     |
+-----------------------------------------------------------+

ALEMBIC ARMAGNACAIS:
  Semi-continuous column still (single distillation)
  Lower output ABV than Cognac double-distillation
  -> More congeners retained -> more rustic, complex character
  Mobile alembic (alambic ambulant): historic tradition -- still trailer
  moves from farm to farm during campaign season

BACO BLANC (Baco 22A):
  Hybrid grape: Folle Blanche x American Vitis riparia
  Only grape hybrid permitted in any French AOC brandy
  Armagnac grandfathered it due to resistance to phylloxera
  Character: distinctive plum, spice -- contributes to Armagnac identity
```

## Calvados

```
CALVADOS (Norman Apple Brandy)

REGION: Normandy, France
  AOC Calvados: entire Normandy
  AOC Calvados Pays d'Auge: higher standard, pot still required
  AOC Calvados Domfrontais: pear-apple blend (60%+ pear)

APPLES (200+ permitted varieties):
  Bitter apples: high tannin, low sugar -> structure
  Bittersweet: balance
  Sweet: fermentable sugar
  Sharp (acidic): freshness
  Blend provides complete profile; no single variety makes good Calvados

PRODUCTION:
  Harvest -> Crush -> Ferment (2-6 weeks, ambient yeast often) -> Cider (~5% ABV)
  -> Distillation (pot still for Pays d'Auge; column for basic Calvados)
  -> Minimum 2 years oak aging
  -> Pays d'Auge minimum 2 years double-distilled pot still

POMMEAU:
  Calvados AOC: unfermented apple juice + young Calvados (16-18% ABV)
  Mutage (adding spirit to prevent fermentation, like port)
  Aperitif; sweet, rich

APPLEJACK (American):
  Traditional method: FREEZE CONCENTRATION ("jacking")
  Freeze cider -> ice crystals = water -> concentrated cider flows out
  Not distillation -- concentration
  Modern American applejack (Laird's): actual distilled apple brandy
  Laird's: oldest continually operating licensed distillery in US (1780)
```

## Pisco

```
PISCO (South American Grape Spirit)

THE DISPUTE: Peru and Chile both claim pisco.
  Pisco, Peru: city founded 1573; may predate Chilean production

PERUVIAN PISCO:
  AOC: Ica, Lima, Arequipa, Moquegua, Tacna regions
  8 permitted grape varieties (Quebranta, Italia, Torontel, Albilla, etc.)
  Pure grape spirit; NO water addition, NO oak aging
  Single distillation in copper pot still
  Pisco Puro: single variety; Acholado: blend; Mosto Verde: partial ferment
  Bottled at distillation strength (38-48%)

CHILEAN PISCO:
  Aconcagua + Atacama regions (north Chile)
  Pedro Jimenez, Moscatel, Torontel, Muscat varieties
  Can be aged in oak
  Water addition permitted (for strength adjustment)
  Often higher alcohol than Peruvian

INTERNATIONAL LEGAL BATTLE:
  Both countries protect as GI in their own jurisdiction
  EU recognition: granted to Peru (2013) -- Chile disputes
  WTO rules are complex; no global resolution
  USA: does not recognize either country exclusively
  The "Pisco Sour" cocktail: both Peru and Chile claim it

SENSORY DIFFERENCE:
  Peruvian: bold, floral, sometimes musty (Quebranta grape)
  Chilean: lighter, more refined, sometimes oaky
```

## Grappa and Pomace Brandy

```
GRAPPA (Italian)

SOURCE: Vinaccia (grape pomace -- skins, seeds, stems after winemaking)
  Pomace is 20-30% residual juice by weight
  -> Hydrated with water and fermented (if dry) or fermented directly (if wet)
  -> Distilled in steam-heated alembic or continuous still

LEGAL:
  "Grappa" is an Italian geographic indicator (GI) -- Italy only
  Minimum 37.5% ABV
  Stills must be discontinuous (batch)

STYLES:
  Young (giovane): unaged, clear, raw, intense grape character
  Invecchiata/Affinata: aged in wood; amber, mellower
  Aromatica: aromatic grape varieties (Moscato, Gewurztraminer) -- intense fruit
  Monovitigno: single variety
  Riserva: extended aging (12+ months)

MARC DE BOURGOGNE (French equivalent):
  Burgundy pomace brandy
  Less prominent internationally than Grappa
  AOC geographic protection

BAGACEIRA (Portuguese grappa equivalent from Portuguese wine pomace)

CHARACTER OF GRAPPA:
  Can be rough and harsh (mass production) or extraordinary (artisan)
  Key: freshness of pomace (must be distilled immediately after pressing)
  Italian culture: digestivo after meal; Espresso corretto (coffee + grappa)
```

## Common Confusion Points

**"Cognac is aged in Limousin oak, which is what makes it special."**
Limousin oak is one factor. The full picture: soil/terroir of Charente,
Ugni Blanc grape, Charentais double distillation, and patient aging across
multiple cask types. Modern cognac uses both Limousin and Tronçais oak
depending on style goal.

**"VSOP means Very Special Old Product."**
"Very Superior Old Pale." The "Pale" is historical -- early 19th century
when caramel coloring was added to cognac to suggest age, "pale" cognac
was a quality signal (no artificial color). Modern Pale simply means
minimum 4-year aging; caramel (E150a) is still permitted for color.

**"Pisco and brandy are different things."**
Pisco IS a brandy -- distilled fermented grape juice. The distinction from
Cognac or Armagnac: no oak aging (Peruvian style), different grape varieties,
different terroir. Calling it "brandy" rather than "pisco" is commercially
and politically incorrect in South America.

**"Grappa is rotgut."**
Industrial grappa can be harsh. Artisan grappa from single-variety monovitigno
(Moscato grappa, Amarone grappa) can be extraordinary. The key variable is
pomace freshness: grappa made from spent dry pomace days after pressing will
be rough. Grappa made from freshly pressed wet pomace is entirely different.
