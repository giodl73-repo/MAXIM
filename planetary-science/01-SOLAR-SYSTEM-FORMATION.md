# Solar System Formation: Nice Model and Grand Tack

## The Big Picture

The solar system formed from a collapsing molecular cloud ~4.567 billion years ago. The timeline runs from dust to planets in about 100 million years — then 500 million years of dynamical reorganization that shaped what we see today.

```
+------------------------------------------------------------------+
|              SOLAR SYSTEM FORMATION TIMELINE                      |
+------------------------------------------------------------------+
|                                                                  |
|  t=0          Molecular cloud collapse; solar nebula forms       |
|               (triggered by nearby supernova shock?)             |
|                                                                  |
|  t ~ 0.1 Myr  Proto-Sun ignites; disk forms (100-200 AU radius)  |
|                                                                  |
|  t ~ 1 Myr    Dust grains → cm pebbles (coagulation)            |
|               Streaming instability → planetesimals (km-scale)   |
|                                                                  |
|  t ~ 3-5 Myr  Planetary embryos (Moon–Mars mass) via oligarchic  |
|               accretion; Jupiter's core forms beyond frost line   |
|                                                                  |
|  t ~ 3-10 Myr Gas disk dissipates (photoevaporation + accretion) |
|               Jupiter and Saturn reach full mass                  |
|                                                                  |
|  t ~ 10-50 Myr Giant impact phase: Moon-forming impact ~50 Myr  |
|               Terrestrial planets reach final masses              |
|                                                                  |
|  t ~ 700 Myr  Late Heavy Bombardment (Nice model instability)    |
|               ~3.9 Ga — heavy cratering on inner solar system    |
|                                                                  |
|  NOW          4.567 Gyr stable configuration                     |
+------------------------------------------------------------------+
```

## Stage 1: The Protoplanetary Disk

A collapsing molecular cloud conserves angular momentum → flattens into a disk. The proto-Sun sits at the center; temperature decreases outward.

```
DISK STRUCTURE
==============

Hot interior                            Cold outer disk
(>1500 K near Sun)                     (<150 K beyond frost line)

  [Sun]---silicates---metals---[FROST LINE ~2.7 AU]---ices---
         condense here                                condense here

  refractory grains only                water ice + other ices
  (olivine, pyroxene, Fe/Ni)            dramatically increase
                                        solid surface density

  Solid surface density:                ~4× higher beyond frost line
  ~1-2 g/cm² at 1 AU                   because ices add to solids
```

The minimum-mass solar nebula (MMSN) is the reconstruction of the disk from current planetary masses: smear the planets out into rings and you get ~0.01-0.02 M_Sun of solid + gas.

## Stage 2: Planetesimal Formation

The "meter-size barrier" is the crux of planet formation theory. Grain growth from microns to kilometers must cross this:

```
SIZE PROGRESSION AND BARRIERS
==============================

  Dust grains    Pebbles      Meter-size    Planetesimals   Embryos
  (microns)  --> (cm-dm)  --> BARRIER   --> (km-100s km) --> (Moon-Mars)
                              |
                              Problem: meter-size bodies drift inward
                              rapidly (gas drag) AND fragment on collision
                              (bouncing/fragmentation barriers)

  RESOLUTION: Streaming Instability
  ----------------------------------
  Pebbles concentrate in pressure bumps in the disk.
  Local solid/gas density ratio exceeds ~1.
  Self-gravity of the pebble cloud collapses directly to ~100 km body.
  Bypasses the meter-size barrier entirely.
  Evidence: many planetesimals are "pebble-pile" aggregates (Arrokoth!)
```

Streaming instability is now the consensus mechanism for planetesimal formation. It produces a size distribution peaked at ~100 km — matching the observed break in asteroid size distributions.

## Stage 3: From Planetesimals to Planets

Two phases:

```
OLIGARCHIC ACCRETION               GIANT IMPACT PHASE
--------------------               ------------------
  Planetesimals collide and grow   Embryos on crossing orbits collide
  via direct impacts + gravitational and merge in large events
  focusing                         (planet-scale collisions)

  Result: ~100 embryos,            Result: 4 terrestrial planets
  Moon-to-Mars mass each,
  in well-separated orbits          Timescale: 10-100 Myr
  within the terrestrial zone

  "Isolation mass" — each embryo   Moon-forming impact:
  sweeps up its feeding zone        ~50 Myr after t=0
  and then stalls                   Theia (Mars-size) hits proto-Earth
                                    Giant impactor + Earth debris
                                    → Moon
```

**Pebble accretion** (post-2012 mechanism): gas-drag assisted capture of cm-dm pebbles. Far more efficient than planetesimal accretion for growing giant planet cores fast enough before disk gas dissipates.

```
PEBBLE ACCRETION ADVANTAGE
===========================
  Pebbles experience gas drag → slow down relative to the growing body
  → huge capture cross-section (orders of magnitude > geometric)
  → Jupiter's core (~10-20 M_Earth) can form in ~1 Myr (fast enough!)

  Classic planetesimal accretion alone: too slow to form Jupiter before
  disk gas dissipates (~3-5 Myr)
```

## The Grand Tack Model

Proposed 2011 (Walsh et al.). Jupiter's formation and inward migration explains why Mars is so small and the asteroid belt so depleted.

```
GRAND TACK SEQUENCE
===================

Phase 1: Jupiter forms at ~3.5 AU; Saturn forms at ~8-9 AU
         |
         v
Phase 2: Jupiter migrates INWARD to ~1.5 AU
         (Type II migration: planet in gap in disk, dragged inward)
         |
         v
Phase 3: Saturn catches up with Jupiter; they enter 2:3 resonance
         Saturn's outward torque now EXCEEDS Jupiter's inward torque
         |
         v
Phase 4: Both planets reverse course — migrate OUTWARD
         ("tack" like a sailboat — hence the name)
         |
         v
Phase 5: Jupiter settles near 5.2 AU; Saturn near 9.5 AU
         The gas disk dissipates; migration stops

CONSEQUENCES:
  - Jupiter's inward sweep TRUNCATES the disk at ~1 AU
    → Mars grows small (limited feeding zone)
  - Asteroid belt is sculpted:
    Inner belt: S-type (rocky) — always there
    Outer belt: C-type (carbonaceous) — scattered in from beyond snow line
  - C-type asteroids → possibly delivered water to Earth
```

The Grand Tack elegantly explains the small mass of Mars (a longstanding problem) and the compositional mixing in the asteroid belt.

## The Nice Model and the Late Heavy Bombardment

The Nice model (2005, Gomes, Tsiganis, Morbidelli, Levison) explains the Late Heavy Bombardment (LHB) ~3.9 Ga.

```
NICE MODEL SEQUENCE
===================

INITIAL STATE (after disk dissipates):
  Giant planets in compact configuration, near-circular orbits
  Jupiter: ~5.45 AU  Saturn: ~8.18 AU  Uranus: ~11.5 AU  Neptune: ~14 AU
  Kuiper Belt: ~15-30 AU (massive — ~20-35 Earth masses)
         |
         v
SECULAR EVOLUTION (~500 Myr):
  Neptune slowly migrates outward, exciting the Kuiper Belt
  Planetesimals scattered in → planets gain angular momentum
         |
         v
INSTABILITY TRIGGER:
  Jupiter and Saturn cross their 2:1 mean-motion resonance
  (occurs when Saturn's period = 2× Jupiter's period)
  → Orbital eccentricities of Uranus and Neptune suddenly increase
         |
         v
NICE MODEL CATASTROPHE:
  Neptune and Uranus scatter wildly; Neptune may swap position with Uranus
  Jupiter scattered inward → Jupiter Trojans captured in this epoch
  Massive KBO scattering into inner solar system
         |
         v
LATE HEAVY BOMBARDMENT:
  Inner solar system bombarded ~3.9 Ga
  Evidence: Apollo lunar samples show spike in impact ages at 3.9 Ga
  Lunar basins (Imbrium, Orientale, etc.) dated to this period
         |
         v
FINAL STATE:
  Giant planets in current configuration with current orbits
  Kuiper Belt depleted by orders of magnitude from original mass
```

**Grand Tack vs Nice model**: Both are needed. Grand Tack operates during disk gas phase (<10 Myr); Nice model operates ~500 Myr later. They address different epochs.

## Volatile Delivery to Earth

Earth is drier than its formation zone should produce. Where did Earth's water come from?

```
VOLATILE DELIVERY MECHANISMS
=============================

  Source             Isotope signature    Timing
  ------             -----------------    ------
  Carbonaceous chond.  D/H ~1.5×SMOW     Late accretion (~50-100 Myr)
  (CC asteroids)       ³⁶Ar/³⁸Ar matches
                       well to Earth

  Comets (Oort Cloud)  D/H ~2× SMOW      Mostly ruled out (too deuterium-rich)
                       (Rosetta: 67P has
                       3× Earth D/H)

  Enstatite chond.     D/H matches!       Early accretion
  (EC — inner solar    But water content
  system)              very low

  CONSENSUS: Mix of CC asteroids (~80-90%) + enstatite chondrites
  Grand Tack scattered C-type asteroids into the inner belt →
  they become the source population for late-accreting volatiles
```

## Condensation Sequence

Not all minerals condense at the same temperature. The sequence determines what goes into each planet:

```
TEMPERATURE (K)    CONDENSATE          LOCATION IN DISK
-----------        ----------          ----------------
>1500              Corundum (Al₂O₃)   Very inner disk
~1400              Ca-Al-Ti oxides     Inner disk (calcium-rich inclusions)
~1350              Mg-Fe silicates     Silicate condensation zone
~1300              Metallic iron/Ni    Metal condensation zone
~400               Troilite (FeS)      Moderate T zone
~180               Water ice           Frost line (~2.7 AU)
~80                CO₂ ice             Outer disk
~40                CO ice              Very outer disk
~30                N₂ ice              Trans-Neptunian

  This sequence explains:
  - Why CAIs (Ca-Al inclusions) are the oldest solar system solids (4.567 Ga)
  - Why inner planets are rock+metal (high-T condensates)
  - Why outer planets/moons are ice-rich (low-T condensates)
```

CAIs (calcium-aluminum-rich inclusions) in carbonaceous chondrites are the oldest dateable material in the solar system — the 4.567 Ga reference age.

| Question | Answer |
|---|---|
| How long does planet formation take? | Planetesimals: 1 Myr; giant planets: 3-10 Myr; terrestrial planets: 30-100 Myr |
| Why is Mars so small? | Grand Tack — Jupiter's inward migration truncated Mars's feeding zone |
| Why is the asteroid belt depleted? | Grand Tack swept through it; also mean-motion resonances with Jupiter continuously scatter bodies out |

| What are CAIs? | Ca-Al-rich inclusions — first solids to condense; dated to 4.567 Ga, defining solar system age |

| How did Earth get water? | Primarily late accretion of C-type asteroids (D/H matches Earth's); Grand Tack delivered them to asteroid belt crossing zone |
| What is pebble accretion? | Gas-drag assisted capture of cm-scale pebbles; huge geometric cross-section; explains fast giant planet core growth |

## Common Confusion Points

**MMSN is a minimum**: The minimum-mass solar nebula is reconstructed by spreading planetary masses into annuli. Real disk was heavier; planets formed inefficiently.

**Nice model vs original vs Nice 2**: The original Nice model (2005) had specific initial conditions. "Nice 2" and subsequent variants explore different initial configurations. The key physics (resonance crossing → instability) is robust; the exact timing and trigger are still debated.

**Moon-forming impact timing**: ~50 Myr is the canonical estimate, but Hf-W chronometry gives ~30-60 Myr; there's genuine uncertainty. The impactor (Theia) was likely isotopically similar to Earth (same feeding zone), explaining why Moon and Earth have nearly identical isotope ratios.

**LHB vs cataclysm**: Some researchers argue the LHB was a smooth tail of declining impact rates, not a sudden spike (the "sawtooth" vs "cataclysm" debate). The Apollo evidence may be biased toward Imbrium basin ejecta. The story is still evolving.

**Grand Tack is a model, not consensus**: It elegantly solves several problems but requires specific assumptions about disk torques and timing. Alternative explanations for Mars's small size exist (e.g., empty gap scenario).
