# 05 — Process Design & Safety

## Flowsheets, P&IDs, Economics, Process Control, HAZOP

> **STUB** — outline only, content to be authored

**Planned coverage:**
- Process design hierarchy: Douglas doctrine (batch vs continuous, input/output, recycle, separation)
- Block flow diagrams (BFD), process flow diagrams (PFD), piping & instrumentation diagrams (P&ID)
- P&ID symbology: ISA-5.1 instrumentation symbols, equipment symbols, line types
- Mass balance: degree-of-freedom analysis, recycle streams, purge, bypass
- Energy balance: heat duties, utility selection (steam, cooling water, refrigeration, fired heaters)
- Heat integration: pinch analysis, composite curves, minimum utility targets, heat exchanger networks
- Equipment sizing: vessels (residence time), heat exchangers (LMTD/NTU), columns (diameter+height)
- Piping hydraulics: pump selection, NPSH, control valve sizing, pressure drop networks
- Process economics: capital cost estimation (Lang factors, Guthrie method), operating costs
- Profitability analysis: NPV, IRR, payback period, sensitivity analysis
- Process simulation: ASPEN Plus/HYSYS overview, equation of state selection, convergence methods
- Process control fundamentals: control objectives, feedback, feedforward, ratio control
- PID tuning: Ziegler-Nichols, IMC-based, direct synthesis — tuning trade-offs
- Control loop design: cascade control, override control, split-range, inferential control
- Plantwide control: inventory control, Buckley procedure, level control philosophy
- Dynamic simulation: disturbance rejection, startup procedures, control valve stiction
- Process safety: inherently safer design (Kletz), Layers of Protection Analysis (LOPA)
- HAZOP (Hazard and Operability Study): guide words (No/More/Less/Reverse/Other), deviation-cause-consequence
- Fault tree analysis (FTA) and event tree analysis (ETA): quantitative risk assessment
- Relief systems: API 520/521, two-phase relief, rupture disk sizing, flare system design
- Environmental regulations: VOC emissions, wastewater treatment, life cycle assessment (LCA)
