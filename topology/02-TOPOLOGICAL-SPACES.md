# Topological Spaces

## The Big Picture

```
+====================================================================+
|           TOPOLOGICAL SPACES — THE ABSTRACT FRAMEWORK              |
+====================================================================+
|                                                                    |
|  TOPOLOGY τ on set X:                                             |
|  Collection of subsets (called "open") satisfying:               |
|  1. ∅ ∈ τ and X ∈ τ.                                             |
|  2. Arbitrary union of opens is open: if Uα ∈ τ, then ∪Uα ∈ τ.  |
|  3. Finite intersection of opens is open: U,V ∈ τ → U∩V ∈ τ.   |
|                                                                    |
|  TOPOLOGICAL SPACE: (X, τ).                                       |
|                                                                    |
|  WHY THESE AXIOMS?                                               |
|  They are the MINIMAL axioms that make continuous functions work. |
|  f: X→Y continuous ↔ f^{-1}(U) open in X for every open U in Y. |
|                                                                    |
|  KEY CONSTRUCTIONS:                                               |
|  Subspace topology: on Y ⊆ X, opens = {Y∩U : U open in X}.      |
|  Product topology: on ∏Xα, opens generated by ∏Uα.              |
|  Quotient topology: on X/~, opens = sets whose preimage is open. |
+====================================================================+
```

---

## Why Open Sets?

The key insight: continuity is entirely open-set-theoretic (f continuous iff f^{-1}(open) = open), and the open set axioms are exactly what makes this work. The axiom restricting to *finite* intersections is load-bearing: ∩_n B(x,1/n) = {x} would force every singleton to be open, collapsing to the discrete topology. Infinite intersections destroy the structure.

## Examples of Topologies

```
DISCRETE TOPOLOGY: τ = P(X) (all subsets are open).
  Every set is open. Every set is also closed.
  Every function from X is continuous (trivially).
  Metric realization: discrete metric d(x,y) = 0 if x=y, else 1.

INDISCRETE (TRIVIAL) TOPOLOGY: τ = {∅, X}.
  Only ∅ and X are open.
  Every function TO X is continuous (f^{-1}(∅)=∅, f^{-1}(X)=domain, both open).
  Very few functions FROM X are continuous.

COFINITE TOPOLOGY on X:
  Open = ∅ or sets with finite complement.
  Interesting for infinite X: satisfies all axioms.
  On R: not the same as Euclidean topology. Not Hausdorff!

EUCLIDEAN TOPOLOGY on Rⁿ: generated by open balls.
  The "standard" topology of analysis.

ZARISKI TOPOLOGY on Rⁿ (or kⁿ):
  Closed sets = algebraic varieties {x : f₁(x)=...=fₖ(x)=0}.
  Open = complements of varieties.
  NOT Hausdorff: any two nonempty open sets intersect.
  Used in algebraic geometry. Fundamentally different from Euclidean.

ORDER TOPOLOGY on a total order:
  Generated by rays (a,∞) and (-∞,b).
  On R: same as Euclidean.
  On ordinals: gives ordinal topology (interesting for transfinite limits).

PRODUCT TOPOLOGY on ∏_{α∈A} Xα:
  Base = {∏Uα : Uα open in Xα, Uα = Xα for all but finitely many α}.
  (NOT: all products of open sets — that's the BOX topology, which is coarser.)
  Product topology is the "right" one for Tychonoff's theorem.
  Box topology: {(x₁,x₂,...) : xₙ ∈ Uₙ for all n}. Finer than product.
```

## Bases and Sub-bases

```
BASIS for topology τ:
  Collection B of open sets such that every open U = union of elements of B.
  B is a basis iff: (1) X = ∪B (covers X) and (2) B₁, B₂ ∈ B, x ∈ B₁∩B₂ → ∃B₃ ∈ B: x ∈ B₃ ⊆ B₁∩B₂.

  Generating a topology from a basis:
  τ = {∪ of elements of B} (all unions of basis elements).

  Examples:
  R: B = {(a,b) : a < b} generates Euclidean topology.
  Rⁿ: B = {open balls} generates Euclidean topology.
  Order topology: B = {(a,b), (a,∞), (-∞,b)} generates order topology.

SUB-BASE S:
  Any collection S generates a topology: base = finite intersections of S.
  Product topology: sub-base = {π_α^{-1}(Uα) : Uα open in Xα}.
  Tychonoff theorem proves: any product of compact spaces is compact.
  Proof (Alexander): If X is a product and has an open cover with no finite subcover,
    restrict to the sub-basic open sets — get a contradiction.

SECOND COUNTABLE: X has a countable basis.
  Rⁿ: yes (open balls with rational center and radius).
  Uncountable discrete space: no.
  Second countable → separable (has countable dense subset).
  Metric + separable ↔ second countable.
```

## Separation Axioms

```
The Tₙ axioms measure how well points can be "separated."

T₀ (Kolmogorov): For x≠y, ∃ open set containing one but not the other.
T₁: For x≠y, each is in some open set not containing the other.
  Equivalently: every singleton {x} is closed.
T₂ (Hausdorff): Distinct points have disjoint neighborhoods.
  This is the "useful" one — sequences have at most one limit.
T₃ (Regular + T₁): A closed set and a point not in it have disjoint open sets.
T₄ (Normal + T₁): Two disjoint closed sets have disjoint open sets.
  Urysohn's lemma holds. Used to construct partitions of unity.

HIERARCHY: T₄ → T₃ → T₂ → T₁ → T₀.

EXAMPLES:
  Hausdorff (T₂): Rⁿ, metric spaces, manifolds.
  T₁ but not T₂: cofinite topology on infinite set.
  T₀ but not T₁: Sierpiński space ({0,1}, τ = {∅,{1},{0,1}}).
  Not T₀: indiscrete topology on any set with ≥ 2 elements.
  Zariski topology: T₁ (affine n-space) but generally not T₂.

METRIC SPACES ARE NORMAL (T₄):
  Urysohn's metrization theorem: regular + second countable → metrizable.
  Nagata-Smirnov: normal + second countable → metrizable.

STONE DUALITY (TCS CONNECTION):
  Boolean algebras ↔ Stone spaces (compact Hausdorff + totally disconnected).
  Every Boolean algebra B has a Stone space: Spec(B) = {ultrafilters on B}.
  Topology: clopen sets of Spec(B) are in bijection with elements of B.
  Generalization: distributive lattices ↔ spectral spaces.
  The Zariski topology on Spec(R) (prime ideals of a ring R) is exactly
    the Stone-type topology for the lattice of ideals of R.
  Sober spaces / locale theory: topology defined without points —
    the lattice of opens IS the fundamental object, not the set X.
    This bridges topology and lattice theory (domain theory, order theory).

SCOTT TOPOLOGY AND DOMAIN THEORY:
  For a dcpo (directed-complete partial order) D:
  Scott-open sets = upward-closed sets U such that every directed set
    with sup in U already has an element in U.
  Scott topology is T₀ but generally not T₁ or Hausdorff.
  This is the foundational topology for denotational semantics:
    continuous functions in the Scott topology = computable functions
    (in the domain-theoretic sense). A function f: D → E is Scott-continuous
    iff it preserves directed sups — exactly the monotone functions that
    are compatible with the "approximation" order.
  Dana Scott's domain theory (1969–): models of the λ-calculus live
    in categories of dcpos with Scott-continuous functions.
```

## Quotient Topology

```
QUOTIENT SPACE X/~ :
  Equivalence relation ~ on X.
  X/~ = X/{equivalence classes}.
  Quotient map q: X → X/~, q(x) = [x].

QUOTIENT TOPOLOGY: U ⊆ X/~ is open iff q^{-1}(U) is open in X.
  "Finest topology making q continuous."

EXAMPLES:
  Circle: S¹ = [0,1]/{0~1}. Identify endpoints of interval.
    q^{-1}({[x]}) = {x} for x ∈ (0,1), {0,1} for [0]=[1].

  Torus: T² = [0,1]²/{(x,0)~(x,1) and (0,y)~(1,y)}.
    Identify opposite edges of square.

  Möbius band: [0,1]×[0,1]/{(0,y)~(1,1-y)}.
    Identify with a half-twist.

  RP² (projective plane): S²/{x~-x} (identify antipodal points).
    Non-orientable surface.

  Lens spaces: S³/Z_n (quotient by cyclic group acting on S³).
    3-manifolds with π₁ = Z/n.

CONE on X: CX = X×[0,1] / {(x,1)~(y,1) for all x,y}.
  Cone the interval [0,1]: get a triangle. Cone S¹: get a disk.
  Cone Sⁿ: get the ball D^{n+1}.

SUSPENSION ΣX = X×[-1,1] / {(x,1)~(y,1), (x,-1)~(y,-1)}.
  Suspension of Sⁿ is S^{n+1}: ΣSⁿ = S^{n+1}.
  Suspension of circle is sphere: ΣS¹ = S².
```

## Compactness Briefly (Preview)

```
COMPACT SPACE: every open cover has a finite subcover.

EXAMPLES:
  Compact: [a,b] ⊆ R, Sⁿ, T², any closed bounded subset of Rⁿ.
  Not compact: R, (0,1), Z (discrete).

WHY COMPACTNESS MATTERS:
  Continuous functions on compact spaces attain their max and min.
  Compact + Hausdorff → very nice: closed = compact, continuous bijection = homeomorphism.
  Tychonoff: product of compact spaces is compact.
  Compactness is to topology what "finite" is to algebra:
    the one condition that turns infinitary arguments into finitistic ones.
```

## Common Confusion Points

**"Product topology = box topology."**
For finite products: they agree. For infinite products: the product topology has
open sets specified by finitely many coordinates; the box topology by all coordinates.
Product topology is compact (Tychonoff); box topology is not. For analysis, product
topology is always the right one.

**"Every topological space is Hausdorff."**
Many non-Hausdorff spaces are important: Zariski topology (algebraic geometry),
Sierpiński space (logical spaces), spectral spaces (from commutative rings in
Stone duality). Non-Hausdorff spaces model situations where "points are not
distinguishable by open sets" — which happens naturally in algebraic settings.

**"Quotient topology is just the subspace topology."**
Subspace topology = restriction of open sets from ambient space (inclusion map is continuous).
Quotient topology = finest topology making the quotient map continuous (q: X → X/~).
These are dual constructions: subspace is a right adjoint (initial topology for inclusion);
quotient is a left adjoint (final topology for quotient map).

**"Open sets = neighborhoods."**
Not quite: a neighborhood of x is any set containing an open set containing x.
Neighborhoods need not be open. The "neighborhood filter" at x = all neighborhoods of x.
Topology can be defined via neighborhood filters instead of open sets (equivalent but different axioms).
