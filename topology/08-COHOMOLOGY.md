# Cohomology

## The Big Picture

```
+====================================================================+
|        COHOMOLOGY — THE DUAL THEORY WITH RING STRUCTURE           |
+====================================================================+
|                                                                    |
|  HOMOLOGY: chains (add up geometric pieces) → groups Hₙ(X).      |
|  COHOMOLOGY: cochains (assign numbers to pieces) → groups Hⁿ(X). |
|                                                                    |
|  RELATIONSHIP: Hⁿ(X;Z) ≅ Hom(Hₙ(X),Z) ⊕ Ext(Hₙ₋₁(X),Z)        |
|  (Universal Coefficient Theorem — cohomology ≈ dual of homology)  |
|                                                                    |
|  WHY COHOMOLOGY IS RICHER:                                        |
|  Cohomology groups Hⁿ(X;R) form a graded RING under cup product: |
|  ⌣: Hᵖ(X) ⊗ Hq(X) → Hᵖ⁺q(X)                                    |
|  This ring structure distinguishes spaces with the same groups.   |
|                                                                    |
|  DE RHAM COHOMOLOGY (for smooth manifolds):                       |
|  HᵈᴿⁿR(M) = {closed n-forms} / {exact n-forms}                   |
|  De Rham's theorem: HᵈᴿⁿR(M;R) ≅ Hⁿ(M;R).                       |
|  Bridges topology, analysis, and differential geometry.           |
+====================================================================+
```

## Singular Cohomology

### From Homology to Cohomology

```
COCHAIN GROUP: Cⁿ(X; R) = Hom(Cₙ(X), R)
  = { functions f: {singular n-simplices} → R }

  For R = Z: integer-valued functions on chains.
  For R = R: real-valued functions (relevant for de Rham).

COBOUNDARY OPERATOR δⁿ: Cⁿ → Cⁿ⁺¹:
  (δf)(σ) = f(∂σ)  for f ∈ Cⁿ, σ a singular (n+1)-simplex.
  δδ = 0 follows from ∂∂ = 0.

COCHAIN COMPLEX: 0 → C⁰ --δ⁰--> C¹ --δ¹--> C² --δ²--> ...

COCYCLES: Zⁿ = ker(δⁿ) = {cochains f with δf = 0}
COBOUNDARIES: Bⁿ = im(δⁿ⁻¹) = {cochains of the form f = δg}

COHOMOLOGY GROUP: Hⁿ(X; R) = Zⁿ/Bⁿ

UNIVERSAL COEFFICIENT THEOREM:
  0 → Ext¹(Hₙ₋₁(X;Z), Z) → Hⁿ(X;Z) → Hom(Hₙ(X;Z), Z) → 0.
  This sequence splits (non-naturally).

  Ext¹(Z, Z) = 0, Ext¹(Z/n, Z) = Z/n, Ext¹(free, Z) = 0.

  KEY CONSEQUENCE: If all homology groups are torsion-free (free abelian),
  then Hⁿ(X;Z) ≅ Hₙ(X;Z). (Homology and cohomology groups agree when free.)

  When there's torsion: cohomology can have torsion one degree higher.
  Example: RP²:
    H₁(RP²;Z) = Z/2 → Ext¹(Z/2, Z) = Z/2 appears in H²(RP²;Z).
    So H²(RP²;Z) = Z/2, even though H₂(RP²;Z) = 0.
```

## The Cup Product

### Construction

```
CUP PRODUCT: ⌣ : Hᵖ(X;R) ⊗ Hq(X;R) → Hᵖ⁺q(X;R)

For cochains:
  (f ⌣ g)(σ) = f(σ|[v₀,...,vₚ]) · g(σ|[vₚ,...,vₚ₊q])
  for σ: Δᵖ⁺q → X a singular simplex, with the front-p and back-q face maps.

WELL-DEFINED: The cup product of cocycles is a cocycle; coboundary times anything
  is a coboundary. So ⌣ descends to cohomology.

GRADED RING STRUCTURE:
  H*(X;R) = ⊕ₙ Hⁿ(X;R) with the cup product is a GRADED-COMMUTATIVE ring:
  α ⌣ β = (-1)^{pq} β ⌣ α  for α ∈ Hᵖ, β ∈ Hq.
  [NOT simply commutative — the sign is essential for complex coefficients.]

  The graded unit: 1 ∈ H⁰(X;R) (the constant cochain with value 1).

WHY THIS IS POWERFUL:
  Two spaces can have identical cohomology groups but different cup products.
  The cohomology RING distinguishes them.
```

### Computed Examples

```
EXAMPLE 1: H*(S¹; Z)
  H⁰(S¹) = Z (generated by 1)
  H¹(S¹) = Z (generator α, representing the "winding 1-cocycle")
  All higher = 0.
  Cup product: α ⌣ α ∈ H² = 0. So H*(S¹) = Z[α]/(α²) (as a ring).
  (Since H² = 0, α² = 0.)

EXAMPLE 2: H*(S²; Z)
  H⁰=Z (gen 1), H²=Z (gen β), rest 0.
  β ⌣ β ∈ H⁴ = 0. Ring: Z[β]/(β²).

EXAMPLE 3: H*(T²; Z) — the key example
  H⁰=Z (gen 1), H¹=Z² (gens α,β), H²=Z (gen γ).
  Cup products:
    α ⌣ β = γ (the fundamental class)
    β ⌣ α = -γ  (graded commutativity: (-1)^{1·1} = -1)
    α ⌣ α = 0, β ⌣ β = 0.
  Ring: H*(T²) = Z[α,β]/(α²,β²,αβ+βα).

EXAMPLE 4: H*(S¹ × S¹) via Künneth
  Künneth for cohomology rings (when torsion-free):
  H*(X×Y;R) ≅ H*(X;R) ⊗ H*(Y;R)  as graded rings.
  H*(S¹×S¹) = Z[α]/(α²) ⊗ Z[β]/(β²) = Z[α,β]/(α²,β²).
  This matches the direct computation above. ✓

EXAMPLE 5: S² vs T² — groups same? No, but example of ring distinguishing spaces.
  RP²×RP²  vs  a different space with same groups but different ring.
```

### Distinguished Spaces via Cohomology Ring

```
CP² vs S² ∨ S⁴  (complex projective plane vs wedge of spheres):

  H*(CP²; Z):  H⁰=Z, H²=Z, H⁴=Z, rest 0.  Generator α of H².
                α ⌣ α = generator of H⁴.  So α² ≠ 0.
                RING: Z[α]/(α³).

  H*(S² ∨ S⁴; Z):  H⁰=Z, H²=Z, H⁴=Z, rest 0.  Same groups!
                α ⌣ α = 0 (generator of H² crossed with itself is zero,
                since the α generator "comes from" the S² piece and
                the H⁴ "comes from" the S⁴ piece — no interaction).

  Groups are the same. Rings are different.
  CP² ≇ S² ∨ S⁴ is detected by the cup product.
  (CP² is homeomorphic to no such simple space — it's a true non-trivial manifold.)

COHOMOLOGY RING AS COMPLETE INVARIANT?
  Not quite — spaces exist with the same cohomology ring but not homeomorphic.
  But cohomology ring + extra structure (Steenrod operations, etc.) often classify.
```

## de Rham Cohomology

### Differential Forms

```
SETUP: M a smooth n-manifold. Local coordinates x₁,...,xₙ.

0-FORM: a smooth function f: M → R.
1-FORM: a smooth section of T*M (cotangent bundle).
  Locally: ω = Σᵢ fᵢ dxᵢ.
  dxᵢ is the "differential of the i-th coordinate function."

k-FORM: a smooth antisymmetric (0,k) tensor field.
  Locally: ω = Σ_{i₁<...<iₖ} fᵢ₁...ᵢₖ dxᵢ₁ ∧ ... ∧ dxᵢₖ
  The wedge product ∧ is antisymmetric: dxᵢ ∧ dxⱼ = -dxⱼ ∧ dxᵢ.

EXTERIOR DERIVATIVE d: Ωᵏ(M) → Ωᵏ⁺¹(M):
  For a function f: df = Σᵢ (∂f/∂xᵢ) dxᵢ.
  For a k-form: dω = Σ (df_{...}) ∧ dx_{...}.

  d² = 0: d(dω) = 0 for all ω. (Antisymmetry kills second partials.)

KEY EXAMPLES:
  On R³: 0-form f, 1-form ω = P dx + Q dy + R dz, 2-form η = A dy∧dz + B dz∧dx + C dx∧dy.

  d(f) = (∂f/∂x)dx + (∂f/∂y)dy + (∂f/∂z)dz   ↔  grad f
  d(ω) = (∂R/∂y - ∂Q/∂z)dy∧dz + ...            ↔  curl ω
  d(η) = (∂A/∂x + ∂B/∂y + ∂C/∂z)dx∧dy∧dz      ↔  div η

  The identities curl(grad f) = 0 and div(curl F) = 0 are d²=0!
```

### de Rham Complex and Cohomology

```
DE RHAM COMPLEX:
  0 → Ω⁰(M) --d--> Ω¹(M) --d--> Ω²(M) --d--> ... --d--> Ωⁿ(M) → 0

CLOSED FORM: dω = 0 (ω is a cocycle).
EXACT FORM: ω = dη for some (k-1)-form η (ω is a coboundary).
Exact → closed (d² = 0), but closed ≠ exact in general.

DE RHAM COHOMOLOGY:
  HᵈᴿᴿR^k(M) = ker(d: Ωᵏ → Ωᵏ⁺¹) / im(d: Ωᵏ⁻¹ → Ωᵏ)
              = {closed k-forms} / {exact k-forms}

KEY EXAMPLE: R²\{0}.
  Consider the 1-form: ω = (-y dx + x dy) / (x² + y²).
  dω = 0 (closed — check directly).
  Is ω exact? If ω = df for some f, then df = ω → f = arctan(y/x) + const.
  But arctan(y/x) is multi-valued around the origin — no single-valued f exists.
  So ω is closed but NOT exact. H¹_dR(R²\{0}) = R. ✓
  (This is the "winding form" — its integral around the origin = 2π.)

CONNECTION TO PHYSICS:
  Maxwell's equations in 4D: F = dA (F is closed since d²=0), dF=0 (Bianchi identity).
  The electromagnetic potential A is the "exact" part.
  H² of spacetime measures whether A exists globally (gauge field = exact 2-form).
```

### de Rham's Theorem

```
DE RHAM'S THEOREM:
  For a smooth manifold M:
  HᵈᴿᴿR^k(M; R) ≅ Hᵏ(M; R)  (singular cohomology with real coefficients).

  The isomorphism is given by integration:
  [ω] ↦ ([σ] ↦ ∫_σ ω)
  This is the PERIOD MAP of ω over cycles σ.

STOKES' THEOREM (generalized):
  ∫_∂σ ω = ∫_σ dω.
  This is why the integration pairing Hᵏ_dR × Hₖ → R is well-defined:
  - ω closed: ∫_∂σ ω = ∫_σ dω = 0 (boundary terms vanish for cycles).
  - σ a boundary: ∫_∂τ ω = ∫_τ dω = 0 (integral over boundary is zero for closed forms).

EXAMPLES:
  H⁰_dR(M) = R^(# connected components).
  H^k_dR(Rⁿ) = 0 for k ≥ 1 (Poincaré lemma: closed → exact on contractible domains).
  H^k_dR(Sⁿ) = R if k=0 or k=n, else 0.
  H^1_dR(S¹) = R, generated by dθ (the angle form).
```

## Poincaré Duality

```
POINCARÉ DUALITY:
  Let M be a compact, oriented, smooth n-manifold (no boundary).
  Then there is a natural isomorphism:
  Hᵏ(M; Z) ≅ Hₙ₋ₖ(M; Z)  [if M is torsion-free]
  More precisely:
  Hᵏ(M; R) ≅ Hₙ₋ₖ(M; R)  for any ring R.

  The isomorphism is given by "cap product with the fundamental class [M]":
  Hᵏ(M) --∩[M]--> Hₙ₋ₖ(M).

FUNDAMENTAL CLASS [M]:
  For a compact oriented n-manifold: Hₙ(M;Z) = Z, generated by [M].
  [M] represents "the entire manifold" as an n-cycle.
  Non-orientable manifolds have Hₙ(M;Z) = 0 (no Z fundamental class).
  (They do have Hₙ(M;Z/2) = Z/2.)

EXAMPLES:
  S²: H⁰ ≅ H₂ = Z ✓, H¹ ≅ H₁ = 0 ✓. (n=2, duality k ↔ 2-k)
  T² (n=2): H⁰≅H₂=Z ✓, H¹≅H₁=Z² ✓.
  Σ_g (genus-g): H⁰≅H₂=Z, H¹≅H₁=Z^{2g}. ✓

POINCARÉ DUALITY AND CUP PRODUCT:
  On M compact oriented n-manifold, the pairing:
  Hᵏ(M;R) × Hⁿ⁻ᵏ(M;R) → Hⁿ(M;R) ≅ R
  given by (α,β) ↦ ⟨α⌣β, [M]⟩ is a perfect (non-degenerate) bilinear pairing.

  The cup product structure combined with Poincaré duality gives:
  Signature of Hⁿ/²(M) (for n=4k): the "intersection form" with index theory.

NON-ORIENTABLE MANIFOLDS:
  RP² is compact but not orientable.
  H₂(RP²;Z) = 0, not Z → Poincaré duality fails over Z.
  H₂(RP²;Z/2) = Z/2 → Poincaré duality holds over Z/2 for all compact manifolds.

POINCARÉ DUALITY FOR MANIFOLDS WITH BOUNDARY:
  Poincaré-Lefschetz duality: Hᵏ(M,∂M) ≅ Hₙ₋ₖ(M).
  Key in surgery theory and 4-manifold topology.
```

## Characteristic Classes

```
CHARACTERISTIC CLASSES: cohomology classes of a vector bundle E → B
  that measure the "twist" of the bundle.

CHERN CLASSES cᵢ(E) ∈ H²ⁱ(B;Z):
  For complex vector bundles E → B.
  c₀ = 1, cᵢ = 0 for i > rank(E).
  Total Chern class: c(E) = 1 + c₁ + c₂ + ... ∈ H*(B;Z).

  c₁(line bundle) = first Chern class = element of H²(B;Z).
  For a complex line bundle, c₁ classifies the bundle completely.
  H²(B;Z) = group of complex line bundles = Picard group of B.

STIEFEL-WHITNEY CLASSES wᵢ(E) ∈ Hⁱ(B;Z/2):
  For real vector bundles.
  w₁(M) = 0 iff M is orientable.
  w₂(M) = 0 iff M has a spin structure (crucial for spinors/Dirac operators).

PONTRYAGIN CLASSES pᵢ(E) ∈ H⁴ⁱ(B;Z):
  For real vector bundles.
  Related to Chern classes by: pᵢ(E) = (-1)ⁱ c₂ᵢ(E ⊗ C).
  Pontryagin classes appear in Hirzebruch's signature theorem and the
  Atiyah-Singer index theorem.

EULER CLASS e(E) ∈ Hⁿ(B;Z) for oriented rank-n bundle:
  e(TM) ∈ Hⁿ(M;Z): its integral over M = χ(M) (Gauss-Bonnet-Chern).

CHERN-WEIL THEORY:
  Characteristic classes can be computed analytically via curvature:
  cₖ(E) = [Tr(iΩ/2π)^k / k!] ∈ HᵈᴿᴿR^{2k}(M) ≅ H²ᵏ(M;R).
  (Ω = curvature 2-form of a connection on E.)
  Bridge between topology (characteristic classes) and differential geometry (curvature).
```

  For a fibration F → E → B:
  Eᵣᵖ·q ⟹ Hᵖ⁺q(E; R).

  d₂: E₂⁰·¹ = H⁰(S²)⊗H¹(S¹) = R → E₂²·⁰ = H²(S²) = R is an isomorphism.

WHY SPECTRAL SEQUENCES?
  Computing Hₙ(E) from pieces Hₚ(B) and Hq(F) — the long exact sequence in
  cohomology of a fibration is infinite; the spectral sequence keeps track of it.
  Same idea as how a Fourier series approximates a function iteratively.

## de Rham vs Singular Cohomology: Comparison

```
+------------------------------------------------------------------+
| ASPECT          | SINGULAR H*(X;R)    | DE RHAM H*_dR(M;R)      |
+------------------------------------------------------------------+
| Domain          | Any top. space      | Smooth manifolds only    |
| Construction    | Dual of chains      | Differential forms       |
| Computation     | Algebraic (CW/simp) | Analytic (differential)  |
| Functoriality   | f*: H*(Y)→H*(X)     | f*: H*(Y)→H*(X) via pullback|
| Cup product     | Abstract formula    | Wedge product ∧          |
| Stokes' theorem | Boundary operator  | Integral = pairing       |
| Connection      | De Rham's theorem  | De Rham's theorem        |
+------------------------------------------------------------------+

De Rham's theorem makes these equivalent (over R coefficients).
The bridge: periods ∫_σ ω pair de Rham cohomology with singular homology.
```

## Common Confusion Points

**"Cohomology is just the dual of homology."**
As groups over Q or R (where Hom(−,R) is exact), yes: Hⁿ(X;R) ≅ Hₙ(X;R)* (dual).
Over Z, the Universal Coefficient Theorem introduces Ext terms — torsion in Hₙ₋₁
contributes to Hⁿ in the degree above. Example: H²(RP²;Z) = Z/2 even though
H₂(RP²;Z) = 0. The Z/2 torsion in H₁ "shifts up" to H².

**"The cup product is commutative."**
Graded-commutative: α ⌣ β = (-1)^{|α||β|} β ⌣ α. In even degrees and over Z/2
this looks commutative, but in general it's not. The sign is essential for
the Leibniz rule for Steenrod operations and for getting the correct algebra structure.

**"de Rham cohomology requires integration — it's just calculus."**
The computation often uses calculus (differential forms), but the result is
purely topological (de Rham's theorem). The period integrals ∫_γ ω depend
only on the homology class [γ] (by Stokes), and the de Rham class [ω]
(exact forms integrate to 0). So periods give topological invariants.

**"Poincaré duality holds for all manifolds."**
Compact + oriented: yes. Drop orientability: only over Z/2. Drop compactness:
need to use cohomology with compact support Hᵏ_c(M). Non-compact manifolds
satisfy Hᵏ_c(M;R) ≅ Hₙ₋ₖ(M;R). For example, H¹_c(R) = R but H¹(R) = 0.

**"Characteristic classes are just cohomology classes."**
They're natural cohomology classes of classifying spaces. c₁(E) ∈ H²(B;Z)
is the pullback of the fundamental class of CP^∞ = BU(1) via the classifying
map B → BU(1). This is why characteristic classes satisfy naturality:
f*(cᵢ(E)) = cᵢ(f*E). The abstract machine is classifying space theory;
the practical computation uses Chern-Weil (curvature = characteristic class).
